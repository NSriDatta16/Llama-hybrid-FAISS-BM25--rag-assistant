See the chapter on video in Dive into HTML 5 for the long answer. The short answer is: How is it possible to play a video stored on the web through HTML5? In supporting browsers, use the element and set the src to point to the URI For example, this video http://www.cybertechmedia.com/samplevideos.html But not those videos as they aren't stored in formats that browsers support.
