Its better to use awk or some programming language that does file processing as well as floating maths all in one. Bash does not support floating point maths. The problem with your script is , you call external sed command for every "real" lines you find. This is a performance hit. awk '/real/{s+=$2;c++}END{print "average is: " s/c}' file
