Google's app engine is free for low-volume use, and includes some amount of storage. If you get really popular, you can scale up easily. More info here: http://code.google.com/appengine/ The documentation on what is available for free is here: http://code.google.com/appengine/docs/quotas.html In a nutshell, if you serve fewer than 5 million pageviews a month, you are unlikely to run out of free processing power.
