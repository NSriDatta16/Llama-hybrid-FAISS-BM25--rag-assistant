https://en.wikipedia.org/wiki/Overfitting This model is over-fitting. Better train accuracy (and validation accuracy that gets worse with successive iterations) indicates over-fit. For CNN Next steps should be to reduce complexity of the model and adding droputs / batch normalization. Few articles on this : https://towardsdatascience.com/handling-overfitting-in-deep-learning-models-c760ee047c6e https://machinelearningmastery.com/how-to-reduce-overfitting-with-dropout-regularization-in-keras/
