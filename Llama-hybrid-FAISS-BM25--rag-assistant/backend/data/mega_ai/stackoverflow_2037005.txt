with JUnit 4 I like to use an annotated AllTests class: @RunWith(Suite.class) @Suite.SuiteClasses({ // package1 Class1Test.class, Class2test.class, ... // package2 Class3Test.class, Class4test.class, ... }) public class AllTests { // Junit tests } and, to be sure that we don't forget to add a TestCase to it, I have a coverage Test (also checks if every public method is being tested).
