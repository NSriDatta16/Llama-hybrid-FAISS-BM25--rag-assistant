This WORKS I wrote this function to reduce number of arguments from 2 to 1 by specifying the second argument. public static Expression > Bind2nd (Expression > source, T2 argument) { Expression arg2 = Expression.Constant(argument, typeof(T2)); var arg1 = Expression.Parameter(typeof(T1)); return Expression.Lambda >(Expression.Invoke(source, arg1, arg2), arg1); } Usage: IQueryable cars = ...; Driver driver = ...; cars.Where(Bind2nd(CanBeDrivenBy, driver)); arg1 is temporary storage between calls.
