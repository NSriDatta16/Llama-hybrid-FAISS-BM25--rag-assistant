You need a grouping, otherwise you will only get one result, also you want to order by an aggregate column (sum of all amounts for a given organization) so you have to give it a name: SELECT sum(amount) as total, org FROM iis GROUP BY org ORDER BY total desc LIMIT 0, 20;
