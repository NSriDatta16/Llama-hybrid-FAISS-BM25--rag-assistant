You can't have multiple appbase but there are multiple ways to get around this issue. If you have an Apache front-end, you can make multiple hosts look like one by mapping the URLs. You can create symlink in the appBase. You can use context fragment so the docBase can be pointed to anywhere, effectively having multiple appBase. To use context fragment, you need to place a xml file in conf/Catalina/[host] directory. The file should contain something like this, The xml file name will be the context/app name.
