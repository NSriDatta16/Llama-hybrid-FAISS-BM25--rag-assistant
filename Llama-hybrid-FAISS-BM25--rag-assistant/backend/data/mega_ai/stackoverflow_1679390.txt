Example; kinda pseudo but you should get the concept. dim cnn as connection dim cmd as command dim cmd2 as command dim str1 as string dim str2 as string cnn.open cmd.connection = cnn cmd.command = str1 cmd.execute cmd2.connection = cnn cmd2.command = str2 cmd2.execute cnn.close
