The equivalent type for varbinary in C# is byte[], not string. Also, as Hogan said, AddWithValue tries to assume a data type. Instead, you can make it explicit: sqlCmd.Parameters.Add("buF", SqlDbType.VarBinary, -1).Value = yourByteArray; The -1 for length corresponds to varbinary(max).
