It doesn't have to be a single source. If a property of a WMi class has the MappingStrings qualifier, you can find the source of the property there. For example: Win32_LogicalDisk.VolumeName - Win32API|File System Functions|GetVolumeInformation Win32_LogicalDisk.MediaType - Win32API|Device Input and Output Functions|DeviceIoControl Win32_LogicalDisk.NumberOfBlocks - MIB.IETF|HOST-RESOURCES-MIB.hrStorageSize So it seems that it is mostly Win32 API and MIB
