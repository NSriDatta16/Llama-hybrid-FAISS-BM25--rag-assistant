UPDATE. Only if you find a way to send email with an HTML body . So far there is no way to send HTML email from a BlackBerry device: You cannot send HTML email messages from a BlackBerry device. RIM API 5.0 Messaging - Email messages - Accessing HTML email messages However if you will find a way to send an HTML email body (using TCP or if there will be this feature in the newer RIM API) then you can add an image as a part of the message content and use its content id in the HTML markup.
