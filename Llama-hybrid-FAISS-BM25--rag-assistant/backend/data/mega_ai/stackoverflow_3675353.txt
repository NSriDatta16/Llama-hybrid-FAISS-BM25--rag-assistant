You have to add the session ID as jsessionid fragment of callback URL. Twitter has to callback to http://example.com/callbackservlet;jsessionid=1E6FEC0D14D044541DD84D2D013D29ED (note: the jsessionid value is here just an example). The HttpServletResponse#encodeRedirectURL() (and encodeURL() ) won't encode the URL when the client already supports cookies. You need to hard-encode it yourself. String url = "http://example.com/callbackservlet"; String encodedURL = url + ";jsessionid=" + request.getSession().getId();
