To hide all the rows but the first: $("table tbody tr:not(:first-child)").hide(); To make them visible when you click on the first row: $("table tbody tr:first-child").click(function() { $(this).siblings().show(); }); Alternatively you may want to organise your tables slightly differently (if possible): table tbody tr { display: none; } $(function() { $("table thead tr").click(function() { $("tbody tr", $(this).parents("table")[0]).show(); }); }); ... first row is in thead not tbody ... ... row1 .. ... row2 .. ... row3 .. There are lots of ways to skin this particular cat.
