To do some processing on a tree node after an invalide drop: There is a method in Ext.tree.TreeDragZone, override that method : Ext.override(Ext.tree.TreeDragZone, { afterRepair:function(){ // Code to be processed on node after an invalid drop this.dragging = false; }, getRepairXY : function(e){ return false; } }); Hope that helps somebody.
