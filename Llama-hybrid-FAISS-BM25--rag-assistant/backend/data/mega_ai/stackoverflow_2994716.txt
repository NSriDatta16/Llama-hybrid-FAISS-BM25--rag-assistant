Here's a stab at a performant way to create a generic list from an ArrayList. List myList = new List (source.Count); myList.AddRange(source.OfType ()); By calling the constructor that accepts an int , the backing storage is allocated only once. As always, you should measure the performance using whatever tools you normally use.
