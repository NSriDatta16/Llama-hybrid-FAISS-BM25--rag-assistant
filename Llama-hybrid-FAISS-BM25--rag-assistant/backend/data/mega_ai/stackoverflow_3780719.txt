The event you want to listen is the one used in your example "drag", what you can do to make it look less "bogus" is add a delay, so the maps follows the marker instead of being immediately updated. Try this: google.maps.event.addListener(marker, "drag", function(event) { var point = marker.getPosition(); window.setTimeout(function(){ map.panTo(point); }, 100); });
