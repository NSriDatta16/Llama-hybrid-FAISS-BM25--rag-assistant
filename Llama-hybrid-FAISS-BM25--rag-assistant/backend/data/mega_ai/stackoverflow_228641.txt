It's because you're returning zero when you reach a leaf node. You should be returning the size stored in that leaf node. In addition, if your non-leaf nodes also have a size, you'll need to process them as well thus: private long sum(Node thisNode) { if (thisNode.Left == null && thisNode.Right == null) return thisNode.Size; if (node.Right == null) return thisNode.Size + sum(thisNode.Left); if (node.Left == null) return thisNode.Size + sum(thisNode.Right); return thisNode.Size + sum(thisNode.Left) + sum(thisNode.Right); } If your non-leaf nodes don't have size, use: private long sum(Node thisNode) { if (thisNode.Left == null && thisNode.Right == null) return thisNode.Size; if (node.Right == null) return sum(thisNode.Left); if (node.Left == null) return sum(thisNode.Right); return sum(thisNode.Left) + sum(thisNode.Right); } A more elegant version of the first one is: private long sum(Node thisNode) { if (thisNode == null) return 0; return thisNode.Size + sum(thisNode.Left) + sum(thisNode.Right); }
