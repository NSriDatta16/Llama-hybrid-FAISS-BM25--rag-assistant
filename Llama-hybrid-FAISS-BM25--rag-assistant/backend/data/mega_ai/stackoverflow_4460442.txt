disableDoubleClickZoom (and the other properties listed) are not public properties on the Map class - they are properties of the MapOptions class. To change their value you need something similar to the following: lock: function() { this.map.setOptions({ disableDoubleClickZoom: true, draggable: false }); }, unlock: function() { this.map.setOptions({ disableDoubleClickZoom: false, draggable: true }); } This creates a MapOptions object (inside the {}'s) and passes it to SetOptions, which updates the current state of the Map based on the values passed in.
