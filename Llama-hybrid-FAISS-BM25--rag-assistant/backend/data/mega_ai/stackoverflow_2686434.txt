Your data arrives in sets of 3 lines, so one approach is to organize the parsing that way: use strict; use warnings; # Ignore header junk. while (<>){ last unless /\S/; } until (eof) { # Read in a set of 3 lines. my @lines; push @lines, scalar <> for 1 .. 3; # Filter and clean. next if $lines[0] =~ /\sNULL\s/; $lines[2] =~ s/.+BtsSiteMgr=//; print @lines[0,2]; }
