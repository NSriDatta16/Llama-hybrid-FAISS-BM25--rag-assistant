AJAX is for rendering data in a Web browser, it doesn't affect the back-end data storage. The books and authors tables have a many-to-many relationship, using ORMs such as DBIx-Class can help. In the Web form, there is a new book and a collection of authors . At the server side, create a book , then add authors to book . Browse the DBIx::Class cookbook to get some ideas.
