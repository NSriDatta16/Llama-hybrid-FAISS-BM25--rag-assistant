JFreeChart has a Gantt chart and PrimeFaces has an image component which allows you to dynamically stream content in. Here is an example in combination with JFreeChart. Alternatively you can also just grab h:graphicImage and let it point to a simple servlet which streams the JFreeChart result to the response based on the request parameters or pathinfo.
