if you want to increase you search performance, you should to use duplicate data for search. for example: to separate your data by "key". try to use different storage collection for each "key". Something like this: //for key is Location LocationCollection { Value: value // reference to root entity ID:"some _id", StaticKey:"Some static", } //for key is Name of person FullNameCollection { Value: value // reference to root entity ID:"some _id", StaticKey:"Some static", }
