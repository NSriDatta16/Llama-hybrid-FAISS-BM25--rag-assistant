It depends on the relationship among the three tables. But if your current query is guaranteed to return the records you want to average, all you have to do is: SELECT AVG(QScore) FROM CustPrimaryQ cp JOIN Customer c ON cp.CxID = c.CustomerID JOIN SacCode sc ON sc.SacCode = c.SacCode WHERE sc.ResellerCorporateID = 392
