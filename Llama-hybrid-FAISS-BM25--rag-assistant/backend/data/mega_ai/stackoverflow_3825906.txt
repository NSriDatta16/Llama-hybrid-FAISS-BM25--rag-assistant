How about if you started by putting as many of the largest files you can onto one DVD and then filling it up with as many of the smallest files that you can (starting with the smallest). Repeat this process with the remaining files for each disk. I'm not sure that's going to give you perfect coverage/distribution but I think it might go some way to solving your needs.
