The solution to the index & search step is to use an inverted index data structure, and the best available open source package that implements this for indexing & search is Lucence . There are also open source projects that provide a composite solution to the crawling, indexing & searching steps which may be of interest, e.g. nutch This free online book on information retrieval may help you (see chapter on constructing an inverted index ).
