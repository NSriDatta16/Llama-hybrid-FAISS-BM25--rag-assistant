How about a brute force approach? Put it in an infinite loop and exit it when done. Pseudocode(haven't checked it): create or replace procedure execDDL(ddl in varchar2) is myexp EXCEPTION; pragma exception_init (myexp, -54); begin loop begin execute immediate ddl; exit; exception when myexp then null; end loop; end;
