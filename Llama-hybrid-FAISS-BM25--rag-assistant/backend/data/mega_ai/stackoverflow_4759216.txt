My idea is integrating once over all possible background colors and averaging the square error. i.e. for each component calculate(using red channel as example here) Integral from 0 to 1 ((r1*a1+rB*(1-a1))-(r2*a2+rB*(1-a2)))^2*drB which if I calculated correctly evaluates to: dA=a1-a2 dRA=r1*a1-r2*a2 errorR=dRA^2+dA*dRA+dA^2/3 And then sum these over R, G and B.
