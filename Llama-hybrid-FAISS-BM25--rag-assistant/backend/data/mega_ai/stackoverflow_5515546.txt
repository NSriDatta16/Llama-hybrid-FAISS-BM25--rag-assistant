1) Animation in Flash can either be done in a visual click-and-drag way using the Timeline or by writing ActionScript to move the objects, whereas animation in JavaScript has to be written in code. 2) Flash combines graphics from the .fla with code from .as to create the swf. This process is often referred to as "compiling the swf" but note that it is very different from compiling C code.
