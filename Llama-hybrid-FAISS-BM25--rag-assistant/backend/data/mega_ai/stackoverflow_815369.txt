The magic CPAN index file that maps a module to a distribution that contains it is modules/02packages.details.txt.gz . If you're ever just stuck with a copy of CPAN with access to sites such as CPAN Search , that's the place to look. Also, the cpan -D command will tell you: % cpan -D LWP::Parallel::UserAgent LWP::Parallel::UserAgent ------------------------------------------------------------------------- (no description) M/MA/MARCLANG/ParallelUserAgent-2.57.tar.gz Installed: CPAN: undef Not up to date Marc Langheinrich (MARCLANG) marclang@cpan.org
