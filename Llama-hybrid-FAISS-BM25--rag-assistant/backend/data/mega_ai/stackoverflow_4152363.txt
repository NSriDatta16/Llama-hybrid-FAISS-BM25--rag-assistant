It's pretty easy: string text = CloudStorageAccount.Parse(" ").CreateCloudBlobClient().GetBlobReference("path/to/the/blob.txt").DownloadText(); Of course, if the blob is in a public container, you can just do: string text = new WebClient().DownloadString("http://youraccount.blob.core.windows.net/path/to/blob.txt");
