By default, they all use std::allocator , which uses standard memory routines to get free heap blocks. There is no pooling involved on this layer. (However, most heap implementations use a dedicated low-fragmentation heap to serve small allocations, and strings are most likely to fall into this category. But this is implementation dependent and not exclusive to or optimized for std::string s ...).
