If you dont want to download the same files from the server over and over again (while debugging), download it from the server once using WebClient then store it on IsolatedStorage. The following code will get you started: // read/write from/to IsolatedStorage IO.IsolatedStorage.IsolatedStorageFile.GetUserStoreForSite.OpenFile
