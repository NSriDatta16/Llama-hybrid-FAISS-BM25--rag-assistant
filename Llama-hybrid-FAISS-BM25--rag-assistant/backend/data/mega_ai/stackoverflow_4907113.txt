Try removing the scheme entirely, and start with //netloc, i.e.: >>> SCHEME="qqqq" >>> url="qqqq://base/id#hint"[len(SCHEME)+1:] >>> url '//base/id#hint' >>> urlparse.urlparse(url) ('', 'base', '/id', '', '', 'hint') You won't have the scheme in the urlparse result, but you know the scheme anyway. Also note that Python 2.6 seems to handle this url just fine (aside from the fragment): $ python2.6 -c 'import urlparse; print urlparse.urlparse("qqqq://base/id#hint")' ParseResult(scheme='qqqq', netloc='base', path='/id#hint', params='', query='', fragment='')
