I had this exactly problem. In my case, I am running Windows 7 32bits. I found solution: Modify ddms.bat in C:\android-sdk-windows\tools folder. The last line from call %java_exe% %java_debug% -Djava.ext.dirs=%javaextdirs% -Dcom.android.ddms.bindir= -jar %jarpath% %* to: call %java_exe% %java_debug% -Djava.ext.dirs=%javaextdirs% -Dcom.android.ddms.bindir= -jar %jarpath% ping emulator 0012 It works for me.
