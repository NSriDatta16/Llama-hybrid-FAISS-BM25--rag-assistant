It is not recommended to use PCA or Factor analysis as a step before running linear models. Your results will be less easy to interpret and you will lose the potential predictive power that is in specific variables. A better way to go about this, is to use some sort of stepwise variables selection technique. This is explained in this topic: Choosing variables to include in a multiple linear regression model
