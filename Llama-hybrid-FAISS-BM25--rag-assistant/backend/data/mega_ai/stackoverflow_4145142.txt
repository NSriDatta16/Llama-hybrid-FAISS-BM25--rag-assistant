You can use this function to unescape HTML (stolen from the Prototype source code): function unescapeHTML(html) { return html .replace(/&lt;/g,' ') .replace(/&amp;/g,'&'); } You can then use jQuery's parsing capability to get the text from the tags: for(var x=0;x ' + $el.text() + ' '; // get the text from the jQuery object and insert it into the fragment x++; }
