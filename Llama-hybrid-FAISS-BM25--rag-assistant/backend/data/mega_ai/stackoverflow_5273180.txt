Ok, the answer was that CursorAt will default to the parent of the dragged item, which in this case had changed to a div. So changing the draggable() to $("draggable").draggable({ helper: "clone", appendTo:'body', cursorAt: { left: -10, top: -10} }) fixed the problem.
