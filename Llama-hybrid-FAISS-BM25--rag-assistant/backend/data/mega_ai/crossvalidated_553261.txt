Yes, logistic regression is a good choice for the structural model . However, you may also consider using a measurement model such as factor analysis (FA) or item response theory (IRT) instead of using summed scores. Nowadays, there are plenty of software tools for both types of measurement models (e.g., in R the lavaan package in for FA and the mirt package for IRT).
