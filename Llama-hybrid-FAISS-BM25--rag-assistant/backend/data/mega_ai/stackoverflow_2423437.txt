For a widget these four methods control the behaviour of a dragged widget : virtual void dragEnterEvent ( QDragEnterEvent * event ) virtual void dragLeaveEvent ( QDragLeaveEvent * event ) virtual void dragMoveEvent ( QDragMoveEvent * event ) virtual void dropEvent ( QDropEvent * event ) I guess you have to reimplement the dragMoveEvent to achieve what you want.
