You might consider putting everything into an external .js file. Then the browser only has to download the code once. If the functions cause errors on pages where they're not not needed, you should only include them on pages where they are needed. Otherwise, it depends on how long it takes the functions to execute. If it's not noticeably long, I would say don't worry about it. It's not worth the fragility you'd introduce by adding a bunch of if statements.
