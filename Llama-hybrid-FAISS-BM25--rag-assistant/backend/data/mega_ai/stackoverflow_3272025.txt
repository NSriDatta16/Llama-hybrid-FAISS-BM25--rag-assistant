You've got to aggregate x in some fashion: SELECT Y, MAX(X) AS Max_X, AVG(X) AS Avg_X FROM myTable GROUP BY Y ORDER BY Y Minimum, Maximum, Average, First, Last, etc. If you don't care which x you get, perhaps just choose the first or last: SELECT Y, First(X) AS X, FROM myTable GROUP BY Y
