You'll probably need to organize a group of elements into something like the following: html: css: #element { // tileable background image, repeat-y; } #elementTop { // top, fixed portion of the background image, no-repeat } #elementBottom { // bottom, fixed portion of the background image, no-repeat, set to: background-position: 0 100%; }
