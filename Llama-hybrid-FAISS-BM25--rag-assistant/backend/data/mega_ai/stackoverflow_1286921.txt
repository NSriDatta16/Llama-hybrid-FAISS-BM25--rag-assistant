There is a JS client implementation for OAuth here: https://developers.google.com/identity/protocols/OAuth2UserAgent It contains example code to get you running. Basically, what you do is this: var url = "..."; var accessor = { token: "...", tokenSecret: "...", consumerKey : "...", consumerSecret: "..." }; var message = { action: url, method: "GET", parameters: {...} }; OAuth.completeRequest(message, accessor); OAuth.SignatureMethod.sign(message, accessor); url = url + '?' + OAuth.formEncode(message.parameters); // send request to 'url' ... Cheers, Matthias
