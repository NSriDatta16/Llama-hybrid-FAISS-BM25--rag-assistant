I think it uses wifi when the wifi information scanned from the wifi chipser matches data the remote Google database (I don't know the coverage of this database, but it should include all cities that have StreetView). Else it uses the far less accurate cell triangulation. To check which kind is used, you might try LocationProvider.requireCell( ) (haven't tested it...)
