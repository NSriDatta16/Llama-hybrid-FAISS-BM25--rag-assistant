You can schedule a DTS by creating SQL Server agent job, and adding the DTS execution as a scheduled task. By your use of the term 'DTS' I'm inferring that you are using the SQL Server 2000 tool. Later versions of SQL have SSIS (SQL Server Itegrated Services), which might be a better option for you. With this you'll need to Add a data flow task Drill down into the data flow task. Add two database connections, your 2000 and 2008 databases Add a data source and destination. Connect the two. Give it a shot.
