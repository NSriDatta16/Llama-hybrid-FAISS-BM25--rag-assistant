For performance and memory considerations you should consider issuing a single SELECT SUM(total_size) statement, e.g. Widget.sum( :total_size, :conditions => [ 'widget_type = ? AND organization_id = ?', Widget::IMAGE', self.id ], :joins => [ :pages, :phases, :trips ] )
