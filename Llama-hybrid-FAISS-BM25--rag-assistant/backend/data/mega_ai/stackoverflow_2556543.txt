It depends on your table structure. If you have all fixed width fields you can get pretty close. If you have variable width fields the best you will do is an estimate. This is for postgres. SELECT pg_total_relation_size('my_table') * i / cnt FROM ( SELECT SUM(CASE WHEN organization_id = 1 THEN 1 ELSE 0 END) AS i, COUNT(1) AS cnt FROM my_table ) sub
