Obviously, it depends on what you want to do with it (and what you mean be "logging"). I'd recommend a flexible database storage. That way you can query it reasonably easily, and also make it flexible to changes later on. Also, from a privacy point of view, it's appropriate to be able to easily associate items with certain entities so they can be removed, if so requested.
