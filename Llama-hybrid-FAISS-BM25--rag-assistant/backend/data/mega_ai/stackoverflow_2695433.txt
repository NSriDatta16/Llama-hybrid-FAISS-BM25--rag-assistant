Use YUI's storage utility and force it to use the SWF storage engine. All computers and browsers would still have to have Flash installed, but you wouldn't have to write your own Flash app. You would benefit from using the one maintained by the YUI team. As others have said, this is not very portable, but in a controlled environment, it might work for you.
