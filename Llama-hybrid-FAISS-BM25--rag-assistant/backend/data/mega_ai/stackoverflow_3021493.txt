You need to re-organize your code a little, and use WriteString to write the values, as this encodes "special" characters: use http://msdn.microsoft.com/en-us/library/system.xml.xmltextwriter.writestring.aspx When the Xml is read back (with an Xml reader) the encoded characters will be decoded.
