FTP used to be the One True system to move data around. That's pretty much fragmented now: for public data distribution: HTTP, BitTorrent for sharing data inside an organization: web-based tools, SMB and other native filesharing platforms for moving data between boxes: scp, rsync for sending data to an individual: email, web-based tools
