I would make sure the url starts with http:// or https:// : if(preg_match("`^https?://`i", $_GET['url'])) // do stuff You may also want to make sure it isn't pointing anywhere internal: if(preg_match('`^https?://(?!localhost|127\.|192\.|10\.0\.)`i', $_GET['url'])) // do stuff Rather than a big dirty regex, you could go for a more elegant host black-list approach, but you get my drift...
