em.persist doesn't persist the object, it marks it as persistable and at the commit the object will be persisted. I assume from your code fragment you don't have a commit? EntityTransaction tx = em.getTransaction(); tx.begin(); dispositivoMedico = new DispositivoMedico(); dispositivoMedico.setCategoria("prova"); dispositivoMedico.setCodice("323"); em.persist(dispositivoMedico); tx.commit(); em.close();
