Using the CSS jQuery function like you did: http://docs.jquery.com/CSS/css#name But read this paragraph: Shorthand CSS properties (e.g. margin, background, border) are not supported. For example, if you want to retrieve the rendered margin, use: $(elem).css('marginTop') and $(elem).css('marginRight'), and so on.
