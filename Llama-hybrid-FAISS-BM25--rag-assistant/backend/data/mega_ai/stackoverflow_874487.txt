There is no such thing. There are dinosaurs, and there are people who need to get work done. If something like 'Shadow IT' happens, it is because 'Official IT' is not doing its job. Software developers have hundreds of little and not so little applications they need to get their work done. The IT governance organisation should learn how to handle tens of updates a day, and switch to releasing daily (and patching a few times a day). Development has learned how to do that, they are next.
