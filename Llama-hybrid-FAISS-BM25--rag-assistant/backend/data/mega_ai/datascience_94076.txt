The area of combining machine learning and human knowledge is called expert-augmented machine learning. One common method is called RuleFit. RuleFit has two steps: Fit a tree-based model to make rules. Fits a linear model with these rules as input. The result is highly interpretable which can be filtered or applied by humans.
