For some inspiration, check out the Alternate Hard and Soft Layers pattern described on the C2 wiki. As for my two cents, why embed a scripting language? Some reasons that I've experienced include, REPL easy string manipulation tools leverage the power of loops, macros, and recursion within your data set create dynamically generated content wrappers to fetch content from the web logic to provide default values if data is missing unit tests written at the data set level
