Calculate growth rate in iPhone

Ref URL; http://en.wikipedia.org/wiki/Future_value Earlier similar question; calculate Future value in iPhone Like we hvae PV (Present value0, FV (Future value), No. of years (N) and growth rate (%) How can we calculate growth rate (or interest rate) using the other 3 variables in iPhone? Code for FV below; -(float) calcFVFromPresentValue: (float) pv interest_rate: (float) interest_rate time: (float) time { float fv; //pv = 200000.0; // present value //i = 0.012; // interest rate (1.2%) //t = 5.0; // time period fv = pv * pow (1.0 + interest_rate, time); return fv; }
