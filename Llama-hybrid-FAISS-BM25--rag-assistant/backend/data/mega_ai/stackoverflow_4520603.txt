Just a Generalized Approach Running The selectRandom() procedure every 5seconds will be too messy. instead run selectRandomGroups() every 120 seconds. selectRandomGroups() executes selectRandom() 120/5 times in every execution. and stores the resulting groups in some storage (cache). next use grabRandomGroup() every 5 seconds that retrieves the last Group from the cache.and remove it.
