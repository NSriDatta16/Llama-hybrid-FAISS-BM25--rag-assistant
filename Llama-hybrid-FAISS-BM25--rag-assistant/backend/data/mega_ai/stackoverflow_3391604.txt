I experienced this problem using the following configuration: Windows 2008 32-bit PHP 5.2.14 MySQL 5.1.49 phpMyAdmin 3.3.5 Here is what I did to get rid of the "Your PHP MySQL library version 5.0.51a differs from your MySQL server version 5.1.36. This may cause unpredictable behavior." message: I stopped IIS I deleted "C:\Program Files\PHP\libmysql.dll" file I copied "C:\Program Files\MySQL\MySQL Server 5.1\bin\libmySQL.dll" file into "C:\Program Files\PHP\" directory I started IIS It somewhat worked, but php-cgi began crashing...
