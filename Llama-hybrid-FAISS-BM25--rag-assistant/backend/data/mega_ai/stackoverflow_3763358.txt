Here is my inelegant crack at it: public static void CalculateWeek (out DateTime WeekStart, out DateTime WeekEnd, DateTime InputDate) { DateTime tempDT = InputDate; while (tempDT.DayOfWeek != DayOfWeek.Wednesday) { tempDT = tempDT.AddDays(-1); } WeekStart = tempDT.Date; while (tempDT.DayOfWeek != DayOfWeek.Tuesday) { tempDT = tempDT.AddDays(1); } WeekEnd = tempDT.Date; } By no means clever or superefficient, although the loops will iterate no more than a trivial 7 times.
