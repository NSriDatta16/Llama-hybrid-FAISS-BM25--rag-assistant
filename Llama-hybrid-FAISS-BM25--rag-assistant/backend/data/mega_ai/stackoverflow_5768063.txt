One way to get the list of processes running is shown on this tutorial: Win32 APIs for Process Retrieval . Another way is through EnumDesktopWindows . If at this point you have access to the window's handle then you can move it with SetWindowPos() . But if you only have access to it's title, then you'll need to use FindWindow() first and obtain a handle to that window.
