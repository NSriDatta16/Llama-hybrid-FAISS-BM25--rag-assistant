img { border: 0 } in your CSS will fix it. What you're seeing is the default border in Internet Explorer for images inside links that have an href attribute that has been previously visited . The reason this happens in the first place is that Internet Explorer presumably has something like this in its User Agent Stylesheet: a:visited img { border: 2px solid purple }
