Follow Using TopLink JPA with Dali to get started with the Toplink Essentials (JPA reference implementation). To get the mentioned jars ( toplink-essentials.jar and toplink-essentials-agent.jar ), download them from there or, if you use Maven, from the Java.Net Maven Repository . PS: Please note that after 2.0-b36, the toplink-essentials-agent.jar file is no longer needed as the agent facility is now integrated into toplink-essentials.jar . So you can do like below. $ java -javaagent:toplink-essentials.jar client.Client
