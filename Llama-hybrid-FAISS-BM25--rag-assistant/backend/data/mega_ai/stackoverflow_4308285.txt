Here's a powershell fragment to list all the generic types in the system assemblies [AppDomain]::CurrentDomain.GetAssemblies() | ? { $_.FullName -match "^System" -or $_.FullName -match "mscorlib"} | % { $_.GetTypes() | ? { $_.ContainsGenericParameters } } This could be adapted to give a more comprehensive list. If you load powershell w/ a .Net 4 config , you'll get the 4.0 list. Count Name ----- ---- 1 System.Xml 12 System.Data 67 System 187 mscorlib 325 System.Core
