Your query is producing a scalar rather than a set of rows. If you want to get the average rate per item then you should do something like: SELECT avg(r.rate) FROM rate r GROUP BY ItemIdColumn And yes, if you want to fetch the value by column name, you should use an alias, like knittl mentioned.
