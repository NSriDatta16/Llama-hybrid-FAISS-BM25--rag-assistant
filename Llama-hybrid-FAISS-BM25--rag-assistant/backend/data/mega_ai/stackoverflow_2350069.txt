Continue after exception in RSpec

Is there any way to have RSpec continue processing specifications after an exception is raised? This is what my spec task looks like: SPEC_PATTERN = "spec/**/*_spec.rb" Spec::Rake::SpecTask.new() do |t| t.spec_files = FileList[SPEC_PATTERN] t.verbose = true t.spec_opts = ["--format", "html:spec/spec_report.html"] t.fail_on_error = false t.rcov = true t.rcov_dir = 'coverage' t.rcov_opts = ['--exclude', 'spec'] end
