It's not possible to know for sure in all cases, but if the user is using IE, then the user agent string will contain the CLR version(s) installed. e.g. Mozilla/4.0 (compatible; MSIE 7.0; Windows NT 5.1; Trident/4.0; .NET CLR 1.1.4322; .NET CLR 2.0.50727; .NET CLR 3.0.4506 If they're using Firefox, it might also be possible to tell as the .Net Framework sometimes installs a Firefox extension that adds this to the user-agent string , e.g. Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13 (.NET CLR 3.5.30729) You can access the user-agent string on a webpage using the Javascript navigator.userAgent variable.
