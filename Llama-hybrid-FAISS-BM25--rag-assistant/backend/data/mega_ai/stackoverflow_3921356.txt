Use flashVars to send data to the datagrid flex app Create an xml based structure with your data Pass the xml url as flashvars to the flex app. The app should read the xml url from params, load the xml, parse and display it. How to parse xml and fill dg http://blog.flexexamples.com/2007/07/26/displaying-xml-data-in-a-datagrid/ How to read flashvars http://blog.flexexamples.com/2007/08/07/grabbing-flashvars-from-the-embedding-html-template/
