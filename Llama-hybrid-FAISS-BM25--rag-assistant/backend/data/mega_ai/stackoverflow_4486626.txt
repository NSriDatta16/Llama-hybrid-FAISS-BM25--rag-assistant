Most bots send out a special HTTP Header called User-Agent . Google's is User-Agent: Googlebot/2.1 for example. Also Google has a published range of IP addresses from which their spiders crawl. A smart web server module can customize your content based upon if it detects googlebot or not. Some consider this to be blackhat SEO . Showing the search crawler something different than what the user sees. See Googlebot Spoofer for more.
