This is easily accomplished by using the TextRange 's Characters , Words , Sentences , Runs and Paragraphs objects and then it's Font object to set Bold, Underline and Italic (amongst other properties). For example: Sub setTextDetails() Dim tr As TextRange Set tr = ActiveWindow.Selection.SlideRange.Shapes(1).TextFrame.TextRange With tr .Text = "Hi There Buddy!" .Words(1).Font.Bold = msoTrue .Runs(1).Font.Italic = msoTrue .Paragraphs(1).Font.Underline = msoTrue End With End Sub
