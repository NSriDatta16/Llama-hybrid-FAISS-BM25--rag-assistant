As you note in your question, PCA attempts to reduce dimensionality while maximizing variance accounted for. LPP on the other hand attempts to reduce the number of dimensions while maximizing a sense of nearness to one's neighbors. I located the following website ( https://notebook.community/jakevdp/lpproj/Example ) which appears to give a nice explanation with some sample code in Python.
