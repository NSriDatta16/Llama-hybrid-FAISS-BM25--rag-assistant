Given WorkflowDesigner designer , Activity activity , and drag source dataSource , use this code (credits to tilovell ) ModelItem mi = designer.Context.Services.GetService () .CreateModelItem(null, activity); DataObject data = new DataObject(DragDropHelper.ModelItemDataFormat, mi); DragDrop.DoDragDrop(dataSource, data, DragDropEffects.Copy); The application will go into drag and drop mode. The designer will receive the drop as an Activity.
