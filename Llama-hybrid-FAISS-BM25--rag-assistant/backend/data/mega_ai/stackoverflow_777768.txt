The index will need some reorganisation. This is part of the C in ACID: When your UPDATE completes everything is done and dusted. Also, any indexed views using the data will need updated too, again part of the "C". If it's not clustered then this is about it. I wouldn't worry about it too much unless it's happening many time a second...
