If you're talking about TCP fragments, those should be hidden (reassembled) by the OS upon receipt, unless you use a low level packet capture facility e.g. Ethereal To force sending of TCP fragments, decrease the maximum packet/segment size on some router, and/or configure the sending OS to use a larger MSS than will fit.
