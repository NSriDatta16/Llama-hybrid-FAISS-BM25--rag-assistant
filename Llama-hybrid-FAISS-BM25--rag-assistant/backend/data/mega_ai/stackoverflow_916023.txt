I think you have more flexibility if you construct your IsolatedStorageFileStream using an IsolatedStorageFile instance, because they can be created on for example an app-specific basis: using (IsolatedStorageFile isf = IsolatedStorageFile.GetMachineStoreForApplication()) using (IsolatedStorageFileStream isfs = new IsolatedStorageFileStream("filename", FileMode.Create, isf)) { // Code using the file stream here... }
