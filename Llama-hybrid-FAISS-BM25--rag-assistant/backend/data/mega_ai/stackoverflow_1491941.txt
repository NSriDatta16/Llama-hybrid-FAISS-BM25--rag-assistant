If you derive from System.Windows.Forms.Button, for instance, your new control will be available in the toolbox. A new tab should appear at the top and will be named [Project Name] Components. You can just drag and drop it on a form. As far as changing the type of a control already on a form, I've had success with changing the type in the designer generated code .cs file. Here's a great article on creating custom .net controls: The article discusses evey possible scenario, including using inheritance, and provides lots of sample code. EDIT: Add this to your project: public class DerivedTextBox: System.Windows.Forms.TextBox { } And then build the project. DerivedTextBox should then be available in the ToolBox.
