Show us some code. You should be able to pass any events you are not interested in back to the parent view. For example, after you detect a two finger tap, and do whatever you want, pass that same event back to mapview and have it zoom itself. Here is what you call once you are done with your event detection: [self.nextResponder touchesBegan:touches withEvent:event];
