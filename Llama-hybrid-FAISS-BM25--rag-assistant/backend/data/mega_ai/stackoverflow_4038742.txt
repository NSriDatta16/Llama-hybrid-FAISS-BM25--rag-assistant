To force Hyperic HQ Agent to detect JBoss, you have to edit conf/agent.properties jboss.installpath=C:/jboss Then, log into your Hyperic HQ Server and manually add the jboss server under inventory. Give it few minutes, the local agent will pick up the instruction and look for the jboss server.
