Browse-sniffing is tricky to do because many user agents try to mimic others to get past programmer-induced restrictions. If you're specifically trying to make a mobile-friendly stylesheet, using the "media" attribute of the LINK tag is a good way to target specific devices. i.e. The first stylesheet will be applied to all media types, while the second will only get applied to "mobile" devices. If you're looking for a way to target only IE, check out Conditional Comments: http://www.quirksmode.org/css/condcom.html Aaaand for a bit more information on controlling the viewport of the iPhone, check out this article from Apple: https://developer.apple.com/library/content/documentation/AppleApplications/Reference/SafariHTMLRef/Articles/MetaTags.html
