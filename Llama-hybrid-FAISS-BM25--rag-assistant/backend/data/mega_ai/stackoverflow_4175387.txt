You can just use .load() on the dialog and event.preventDefault() on the anchor (to prevent the normal go-to-href behavior), like this: $("a").click(function(e) { dialogDiv.load(this.href); e.preventDefault(); }); Just change the $("a") to be a bit more specific...only selecting whatever links you want to load in the dialog. Also those pages should only be fragments, or use a different selector, for example if you want the from the page in the href , it'd look like this: dialogDiv.load(this.href + " #content");
