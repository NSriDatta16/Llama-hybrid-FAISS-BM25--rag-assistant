From http://hsivonen.iki.fi/xhtml2-html5-q-and-a/ : If I can use any doctype for XHTML5, how can browsers tell XHTML 1.0 and XHTML5 apart? They can’t and they don’t need to. By design, a user agent that implements XHTML5 will process inputs authored as XHTML 1.0 appropriately.
