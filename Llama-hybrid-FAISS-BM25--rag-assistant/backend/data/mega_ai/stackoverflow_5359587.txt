If you use a reasonably modern compiler, you can replace these guards with more elegant directive #pragma once To modify the header's template in Eclipse CDT, go to "Window/Preferences/C++/Code Templates/Files/C++ Header File/Default C++ header template" and put there ${filecomment} #pragma once ${typecomment} ${declarations} After that your new h files will start from something like this: /* * FileServer.h * * Created on: Feb 26, 2011 * Author: krit */ #pragma once
