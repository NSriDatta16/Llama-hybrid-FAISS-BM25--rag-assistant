I'd avoid inter-language calls if possible; fragility and massive increases in dependencies await you down this road. However, there is... Inline::Python If python must be used, the Inline::* series of modules have been generally well received. This lets you write python inside a perl script. You still have to write perl but it would let you use python libraries inside perl scripts. It will make things more difficult to debug, though.
