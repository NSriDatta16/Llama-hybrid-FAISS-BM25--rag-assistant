An elegant way to do that is using the analytic function row_number : SELECT barcode, price, startdate FROM ( SELECT * , ROW_NUMBER() OVER (PARTITION BY barcode ORDER BY startdate DESC) as rn FROM YourTable ) subquery WHERE rn = 1 If performance is an issue, check out some more complex options in this blog post .
