This should be able to get the maximum temperatures for you, you would just have to change the where filter to get the minimums using System.Xml.Linq; \\... XDocument xmlDoc = XDocument.Load("YourXml.xml"); var maximums = from tempvalue in xmlDoc.Descendants("temperature").Elements("value") where tempvalue.Parent.Attribute("type").Value == "maximum" select (string)tempvalue; List returnme = maximums.ToList (); return returnme; Hope this help. Note: I'm a bit rusty with my Linq to Xml in C# so it may not be the most elegant solution.
