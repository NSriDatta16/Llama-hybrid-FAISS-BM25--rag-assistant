This is equivalent to Chris Dodd's answer , but uses a different naming convention (and coincidentally doesn't require the sed magic. Copied from a later duplicate . If you are using a GNU compiler, the compiler can assemble a list of dependencies for you. Makefile fragment: depend: .depend .depend: $(SOURCES) rm -f ./.depend $(CC) $(CFLAGS) -MM $^>>./.depend; include .depend There is also the tool makedepend , but I never liked it as much as gcc -MM
