you can require rake files from other rake files and run the tasks from the required file. put this in rakefile.rb: require 'foo/rakefile.rb' task :default => [:bar] put this in foo/rakefile.rb: task :bar do puts "baz" end run the root level rakefile and it will print "baz" to the screen. you should be able to organize your rake files and common code however you want using this idea
