Following up on Ragepotato's answer. The easiest way to do this is to create your own Binding that inherits from Binding and then set the things you need, like NotifyOnValidationError="True" and ValidatesOnDataErrors="True" in the constructor. public class ExBinding : Binding { public ExBinding() { NotifyOnValidationError = true; ValidatesOnDataErrors = true; } } And then you use this Binding instead
