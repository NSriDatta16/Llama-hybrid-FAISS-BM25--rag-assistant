I would just use the type Dictionary > . To build this structure from a list of lists, you could have code like this: var alternateNames = new string[][] { new string[] { "Elizabeth", "Liz", "Betty" }, new string[] { "Bob", "Robert", "Rob" }, }; var altNameLookup = ( from nameList in alternateNames from name in nameList select new { Name = name, NameList = nameList.Except(new string[] { name } ) } ).ToDictionary(o => o.Name, o => o.NameList);
