If you'd like that /../test would mean the same as /test like paths in a file system then you could use normpath() : >>> url = urlparse.urljoin("http://example.com/", "../test") >>> p = urlparse.urlparse(url) >>> path = posixpath.normpath(p.path) >>> urlparse.urlunparse((p.scheme, p.netloc, path, p.params, p.query,p.fragment)) 'http://example.com/test'
