You can use bit-fields and let the compiler do the bit twiddling. For example: struct PropertySet { unsigned internal_storage : 1; unsigned format : 4; }; int main(void) { struct PropertySet x; struct PropertySet y[10]; /* array of structures containing bit-fields */ if (x.internal_storage) x.format |= 2; if (y[2].internal_storage) y[2].format |= 2; return 0; } Edited to add array of structures
