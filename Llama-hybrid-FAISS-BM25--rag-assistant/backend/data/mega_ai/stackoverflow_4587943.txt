Using both Lennart Regebro 's and Adam Vendenberg 's suggestions, you could do something like this: import urllib import urlparse query=urllib.urlencode({"test":1}) urls=['http://www.aaa.com', 'http://www.aaa.com/home', 'http://www.aaa.com/?location=home', 'http://www.aaa.com/?location=home&page=1'] for url in urls: x=urlparse.urlparse(url) new_query=(x.query+'&'+query) if x.query else query y=urlparse.urlunparse((x.scheme,x.netloc,x.path,x.params,new_query,x.fragment)) print(y) yields http://www.aaa.com?test=1 http://www.aaa.com/home?test=1 http://www.aaa.com/?location=home&test=1 http://www.aaa.com/?location=home&page=1&test=1
