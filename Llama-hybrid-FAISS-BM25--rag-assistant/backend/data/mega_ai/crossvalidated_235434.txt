Bayesian: Poisson with constant prior

Need some help. Assume the sampling distribution is Poisson with sample size $n$. Assume constant prior distribution. Likelihood: $L(\lambda | \mathbb{x})=\prod_{i=1}^{n}\dfrac{e^{-\lambda}\lambda^{x_{i}}}{x_{i}!}=\dfrac{e^{-n\lambda}\lambda^{\sum_{i=1}^{n}x_{i}}}{\prod_{i=1}^{n}x_{i}!}$ Prior: $p(\lambda)=\frac{1}{a}$, $a$ is some real constant. Therefore, posterior: $\pi(\lambda | \mathbb{x})=\dfrac{e^{-n\lambda}\lambda^{\sum_{i=1}^{n}x_{i}}}{\prod_{i=1}^{n}x_{i}!}\cdot\dfrac{1}{a}$ $\pi(\lambda | \mathbb{x})\propto e^{-n\lambda}\lambda^{\sum_{i=1}^{n}x_{i}}$ Is this correct? And if so, is there no nice posterior for the Poisson/constant?
