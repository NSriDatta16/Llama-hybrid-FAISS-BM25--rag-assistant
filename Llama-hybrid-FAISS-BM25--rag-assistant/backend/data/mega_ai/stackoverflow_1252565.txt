iframes have access to certain properties of the parent document, e.g. redirect the parent frame to a new location using parent.location.href or parent.window.location ( IE allows to restrict that). This is great for phishing attacks when embedding content from other servers (even if you trust that server it might be compromised). iframes can also be used for a variety of other attacks: IFrames security summary .
