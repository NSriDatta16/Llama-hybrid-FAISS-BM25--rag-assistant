This is a standalone URI parsing library that can handle any URI you can imagine (according to the author) and allows you to work with URI's like you would a JS object (which is nice) Examples: uri = new jsUri('http://user:pass@www.test.com:81/index.html?q=books#fragment'); uri.protocol(); // http uri.userInfo(); // user:pass uri.host(); // www.test.com uri.port(); // 81 uri.path(); // /index.html uri.query(); // q=books uri.anchor(); // fragment
