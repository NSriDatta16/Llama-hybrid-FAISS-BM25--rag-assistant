Here's a snippet to load a local html web page contained inside your app bundle: UIWebView *webView = [[UIWebView alloc] initWithFrame:[[UIScreen mainScreen] bounds]]; NSString *path = [[NSBundle mainBundle] pathForResource:@"mywebpage" ofType:@"html"]; [webView loadRequest:[NSURLRequest requestWithURL:[NSURL fileURLWithPath:path]]]; Another alternative is to embed a local web server running in a background thread, and use a URL to localhost. A UIWebview supports HTML5 local storage.
