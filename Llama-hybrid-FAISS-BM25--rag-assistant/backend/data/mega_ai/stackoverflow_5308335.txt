Essentially it should be something like this. Google for "Blocking by user agents htaccess". Options +FollowSymlinks RewriteEngine On RewriteCond %{HTTP_USER_AGENT} "Download Accelerator Plus" RewriteRule ^(.*)$ http://mypage.org/error.html [R=301,L] See the Apache Docs on mod_rewrite for more complex conditions.
