T think the syntax of the case is not correct. Do something like: SELECT last_name, commission_pct, (CASE commission_pct WHEN 0.1 THEN ‘Low’ WHEN 0.15 THEN ‘Average’ WHEN 0.2 THEN ‘High’ ELSE ‘N/A’ END ) Commission FROM employees ORDER BY last_name;
