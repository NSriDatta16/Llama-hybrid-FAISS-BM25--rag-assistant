karoberts' solution doesn't work on Mac OS X (Snow Leopard). Neither does another command I found on a blog I get xargs: illegal option -- i and xargs: illegal option -- d respectively. This works on Mac (adapted from karoberts'): svn status | awk '$1=="?"||$1=="I" { print $2 }' | xargs rm -rf
