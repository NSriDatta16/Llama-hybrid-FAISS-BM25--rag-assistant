My suggestion is to use Internet Explorer's user agent. And IE's user agent is depending on what software you have installed, by installing download manager, the software will appen its ID into the user agent, for IE & firefox, (depending on the download manager). To know what is your useragent, in address bar of IE or FF type : javascript:alert(navigator.userAgent) In the dialog box press Ctrl+C to copy the content
