I don't think you can add resources to an app after you create it. You need to store them using other methods. For example using externalStorage, or the hidden storage available where the app was installed. Edit: The Data storage developer doc is located here . It gives you a great explanation of how you might go about storing your data. Since you are using images, some methods won't be appropriate for you.
