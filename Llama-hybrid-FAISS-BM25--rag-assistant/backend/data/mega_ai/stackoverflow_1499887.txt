Warning create_function() has been DEPRECATED as of PHP 7.2.0. Relying on this function is highly discouraged. To follow up on Alex Barrett's post, create_function() returns a value that you can actually use to call the function, thusly: $function = create_function('$echo', 'echo $echo;' ); $function('hello world');
