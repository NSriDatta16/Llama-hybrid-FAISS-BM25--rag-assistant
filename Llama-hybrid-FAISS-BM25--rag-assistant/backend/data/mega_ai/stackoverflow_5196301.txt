It can certainly be done but to what extent you need is slightly up to you. Pete Brown posted an example on creating a simplistic report writer which may suffice based on your needs and is free; allowing you to build on top of the example as needed. If you moved up to .NET 4 you could leverage the Silverlight Reporting project on codeplex. telerik also offers a reporting package ; however it is not free and requires the full .NET runtime for generation. Don't confuse the .NET runtime as an assumption that it must be server side generation as that is not the case.
