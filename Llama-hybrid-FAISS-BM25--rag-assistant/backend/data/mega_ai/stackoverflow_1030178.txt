OLEtoDisk "This version saves the entire contents of a table containing OLE Objects to disk. Does NOT require the original application that served as the OLE server to insert the object. Supports all MS Office documents, PDF, All images inserted by MS Photo Editor, MS Paint, and Paint Shop Pro. Also supports extraction of PACKAGE class including original Filename. Contains function to produce a full Inventory of the OLE field including LINKED path and Filenames. Uses Structured Storage API's to read the actual contents of the field" http://lebans.com/oletodisk.htm
