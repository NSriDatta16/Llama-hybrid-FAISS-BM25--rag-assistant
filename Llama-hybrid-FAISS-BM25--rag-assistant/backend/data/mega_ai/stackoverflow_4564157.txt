Alternatively, please also consider using the Manchester OWL API - it is extremely powerful and very close to syntax of owl. This, in combination with any number of reasoners, will allow you to leverage your ontology into an application. The website describing the api also has an extensive set of tutorials that will get you up and running in no time!
