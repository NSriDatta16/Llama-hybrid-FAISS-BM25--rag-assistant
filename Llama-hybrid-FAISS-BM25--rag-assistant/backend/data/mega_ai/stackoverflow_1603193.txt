There is an answer in this article about how to use [System.Diagnostics.DebuggerHidden] or [System.Diagnostics.DebuggerNonUserCode] Attributes to exclude methods from code coverage. Update as per David's comment: As of .NET 4.0 there is a dedicated attribute for this: [ExcludeFromCodeCoverage]
