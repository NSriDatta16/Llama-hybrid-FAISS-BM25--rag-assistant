I figured out how to do this. It's called an InlineUIContainer you can do something like this to get it working. Although it doesn't save it into the Xaml var p = new Paragraph(); var inlineUIContainer = new InlineUIContainer() { Child = new Button() { Content = "This is a Button!" } }; p.Inlines.Add(inlineUIContainer); _richTextBox.Blocks.Add(p);
