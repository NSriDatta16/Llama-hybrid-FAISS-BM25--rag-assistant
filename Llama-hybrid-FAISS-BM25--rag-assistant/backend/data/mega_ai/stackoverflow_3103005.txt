The project default setting can have higher precedence than the pragma - see this MSDN article . Structure packing interacts with compiler alignment behavior as follows. If the packsize is set equal to or greater than the default alignment, the packsize is ignored. If the packsize is set smaller than the default alignment, the compiler aligns according to the packsize value.
