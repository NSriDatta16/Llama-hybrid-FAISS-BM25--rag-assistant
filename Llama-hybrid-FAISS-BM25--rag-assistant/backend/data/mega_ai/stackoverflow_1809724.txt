Quickly look on Web GUI recognition principles in QTP. Get to the descriptive programming part. Find Help for Link object in QTP. Define your target Link object using descriptive programming. It should be something like Set objTargetLink = Browser("title:=...").Page("title:=...").Frame("title:=...").Link("id:=target_id") Use boolRC = objTargetLink.Exist(0) To check if your link exists. Use sURL = objTargetLink.GetTOProperty("url") to retrieve the actual url. You may get other properties the same way. Use objTargetLink.Click to click on the link. PS.
