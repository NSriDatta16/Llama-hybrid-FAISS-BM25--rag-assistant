Replacing user portion of email address in java

You have user.nick@domain.com and result should be: ******@domain.com Currently I'm doing it this way: public static String removeUserFromEmail(String email) { StringBuffer sbEmail = new StringBuffer(email); int start = sbEmail.indexOf("@"); sbEmail.delete(0, start); return "******" + sbEmail.toString(); } Is there something simpler or more elegant?
