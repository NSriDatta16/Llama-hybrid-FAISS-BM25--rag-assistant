You can change it in Entity Designer. Enity Data Model (EDM) contains 3 main parts: Conceptual model. What objects do you want to save. Storage model. What database structure do you have. Mapping. How to map objects to database. POCOs are actually 'defined' in your conceptual model. Open Edmx file in Model Designer. Locate User entity there and change the name of the property.
