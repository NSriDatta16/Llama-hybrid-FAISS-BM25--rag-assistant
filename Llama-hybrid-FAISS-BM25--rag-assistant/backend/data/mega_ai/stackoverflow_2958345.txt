I would create my own "MyStats.nsf" database on the server. This db will contain an LotusScript Agent "UpdateAll" triggered each n hours. The agent basically get a list of database paths. For each path open the database. The NotesDatabase.AllDocuments.Count gives you the total number of docs. Open the view "($All)" and retrive NotesView.AllEntries.Count the nr. of docs. for "All Documents". Take this information and create a new NotesDocument to save db name and the retrieved number info. Last but not least create a view in MyStat to display the result.
