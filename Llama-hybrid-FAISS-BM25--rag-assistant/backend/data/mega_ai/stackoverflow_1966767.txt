Algorithms are, quite simply, the steps by which you do something. For instance the Coffee Maker Algorithm would run something like Turn on Coffee Maker Grind Coffee Beans Put in filter and place coffee in filter Add Water Start brewing process Drink coffee A data structure is a means by which we store information in a organized fashion.
