You need to call the function on the this instance, like this: gDebug.call(this, "Hi!"); The correct way to do this is to put the function in the class prototype: (This should be done after declaring the constructor function) ganttChart.prototype.gDebug = function(debugMessage) { alert(this.gDebugPannel.innerHTML); if (this.isDebugMode) { this.gDebugPannel.innerHTML = debugMessage; } } this.gDebug("Hi!");
