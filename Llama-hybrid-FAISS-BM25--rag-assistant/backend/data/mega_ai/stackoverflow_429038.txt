That kind of thing is done all the time. The trick is to tell both sides to use a "C"-style calling protocol for the routine. In C++ this is done with extern "C" declarations, and in the Ada side with pragma Export ("C", ... Look those up in your favorite respective reference sources for details. Watch out for pointer and reference paramters!
