Here's a list I've compiled for my own reference, along with some of the legal values that can be used. Note that these are passed into MSBuild using the /p: = syntax. DeployOnBuild True False DeployTarget MsDeployPublish Package Configuration Name of a valid solution configuration CreatePackageOnPublish True False DeployIisAppPath / MsDeployServiceUrl Location of MSDeploy installation you want to use MsDeployPublishMethod WMSVC (Web Management Service) RemoteAgent AllowUntrustedCertificate (used with self-signed SSL certificates) True False UserName Password SkipExtraFilesOnServer (leave existing non-conflicting files alone) True False
