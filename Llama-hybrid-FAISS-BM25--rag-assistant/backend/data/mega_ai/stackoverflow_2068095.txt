I would not send every letter over the network , this will needlessly generate a large amount of network traffic and database/file writes. Instead send the message when the user has committed to it by pressing a Send button or the Enter key. Storing in a database makes much more sense than a file, as you will need to be able to index the data for quick retrieval. Consider that you will need to retrieve data in groups, not as it was sequentially added to the file.
