You could do something like this: double GetAverageAge (IEnumerable persons, Func propertyAccessor) { double acc = 0.0; int count = 0; foreach (var person in persons) { acc += propertyAccessor(person); ++count; } return acc / count; } As an alternative, consider using LINQ , it already provides something like this.
