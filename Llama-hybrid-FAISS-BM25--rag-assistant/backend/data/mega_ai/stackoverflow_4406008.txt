Rationale: if the height() is equal to the line-height, the paragraph (or any block element) gains the class "oneliner" (which is centered with a css rule). css: /* all Ps should be { text-align: justified; height: auto; } by default */ .oneliner {text-align: center} js: $(document).ready(function(){ $('p').each(function(){ // change the "p" selector at will var $this = $(this); if(Math.round(parseFloat($this.css('lineHeight'), 10)) == $this.height()) $this.addClass('oneliner'); }); }); Note that this will not work if you have nested images or block elements taller than the paragraph's line-height, or if the height is not auto.
