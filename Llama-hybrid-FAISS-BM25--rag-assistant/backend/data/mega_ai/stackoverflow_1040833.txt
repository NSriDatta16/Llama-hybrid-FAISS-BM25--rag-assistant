The code sample in this answer provides an elegant way around this problem. Code quoted from that answer: public void UpdateTestBox(string newText) { BeginInvoke((MethodInvoker) delegate { tb_output.Text = newText; }); } ...although in your case you would want to call BeginInvoke on the controls themselves: public void UpdateTestBox(string newText) { tb_output.BeginInvoke((MethodInvoker) delegate { tb_output.Text = newText; }); }
