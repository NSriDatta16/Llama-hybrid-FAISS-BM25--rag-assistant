It's not possible to specify expiration. It's completely up to the user. https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage Of course, it's possible that something your application stores on the client may not be there later. The user can explicitly get rid of local storage, or the browser may run into space considerations. It's good to program defensively. Generally however things remain "forever" based on some practical definition of that word. edit â€” obviously, your own application can actively remove stuff if it decides it's too old.
