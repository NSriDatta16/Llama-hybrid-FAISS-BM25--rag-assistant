In case someone else comes across this problem, here's a solution that I've found The Cause I was pulling data from my SqlDataSource in the section of my Repeater control. This is what caused the problem. I'm not sure why, but the point is that doing this will not work. The Solution All data retrieval (i.e. block) must be used in the section of the Repeater control.
