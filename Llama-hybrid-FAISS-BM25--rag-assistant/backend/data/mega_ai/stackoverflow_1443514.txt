Note that the window resize function only fires once, after the window has been resized. It does not update during the resize operation, so if you are dragging the window border to test this, no changes will happen before you let go of the mouse button. Also, make sure you do this within $(document).ready() , like this: $(function() { $('#main-content') .css({'height': (($(window).height()) - 361)+'px'}); $(window).resize(function(){ $('#main-content') .css({'height': (($(window).height()) - 361)+'px'}); alert('resized'); }); }); Here is a working demo .
