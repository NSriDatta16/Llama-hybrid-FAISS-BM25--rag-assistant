I don't know if BeautifulSoup can do it more elegantly, but you could merge the two loops like so: for tag in soup.findAll(['script', 'form']) + soup.findAll(id="footer"): tag.extract() You can find classes like so ( Documentation ): for tag in soup.findAll(attrs={'class': 'noprint'}): tag.extract()
