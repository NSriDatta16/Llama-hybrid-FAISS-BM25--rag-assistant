You select your texture as active and bind it - like normally. But you must pass your texture unit as uniform to the shader program. So you can use the gl_PointCoord as texture coordinates in your fragment shader. glUniform1i(MY_TEXTURE_UNIFORM_LOCATION, 0); glActiveTexture(GL_TEXTURE0); glBindTexture(GL_TEXTURE_2D, yourTextureId); glEnable(GL_TEXTURE_2D); glDrawArrays(GL_POINTS, 0, pointCount); glDisable(GL_TEXTURE_2D); And your fragment shader should look like: uniform sampler2D texture; void main ( ) { gl_FragColor = texture2D(texture, gl_PointCoord); }
