You can also use ordinal forests. In this case it is not better than the standard random forest, at least for the test dataset. For a better evaluation you should use cross-validation (for example via mlr package) instead of a simple test dataset, which randomly can provide better results for the standard random forest. Example code for ordinal forest: library(ordinalForest) ordforest
