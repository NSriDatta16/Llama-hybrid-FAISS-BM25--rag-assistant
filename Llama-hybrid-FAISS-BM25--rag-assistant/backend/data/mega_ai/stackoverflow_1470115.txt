You are also missing the headers: Accept Cache-Control Pragma The Accept header alone might be enough for the server to respond with something meaningful. Try something like this for httpclient 4: httpclient.addRequestInterceptor(new HttpRequestInterceptor() { public void process(final HttpRequest request, final HttpContext context) throws HttpException, IOException { request.setHeader("Pragma","no-cache"); request.setHeader("Cache-Control","no-cache"); request.setHeader("Accept","text/html, image/gif, image/jpeg, *; q=.2, */*; q=.2"); } }); Or something like this for httpclient 3.x: method.setRequestHeader("Pragma","no-cache"); method.setRequestHeader("Cache-Control","no-cache"); method.setRequestHeader("Accept","text/html, image/gif, image/jpeg, *; q=.2, */*;
