My Workarond: use a boolean member variable to detect if you call navigate from inside code or navigating is called from ouside code. Implement the "Navigating" (BeforeNavigating) event and check if it was a call from outside. Example: call navigate from inside: private void NavigateLocal(string htmlFileName) { this.localNavigate = true; this.webBrowser.Navigate(htmlFileName); } event method: private void WebBrowser_Navigating(object sender, WebBrowserNavigatingEventArgs e) { if (this.localNavigate)//inside call { this.localNavigate = false; } else //call from outside { this.DoDragDrop(e.Url.ToString()); } }
