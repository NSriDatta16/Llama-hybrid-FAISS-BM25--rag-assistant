There are typically 2 ways to achieve this. Option1 : Add another parameter to IGarrage representing the T which should be passed into the IGenericCar constraint: interface IGarrage where TCar : IGenericCar { ... } Option2 : Define a base interface for IGenericCar which is not generic and constrain against that interface interface IGenericCar { ... } interface IGenericCar : IGenericCar { ... } interface IGarrage where TCar : IGenericCar { ... }
