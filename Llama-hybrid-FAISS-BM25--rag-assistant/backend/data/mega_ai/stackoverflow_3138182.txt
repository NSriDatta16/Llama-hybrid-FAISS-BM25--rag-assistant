The link on the web page is just a link to jnlp resource, that is when clicked you will alway get jnlp downloaded to your machine and then executed by Java Web Start module. There is nothing you can do about it. What you can do is you can add the following lines to your jnlp : My App That will create a nicely branded shortcut on user's desktop that user can click on to start your app instead of going to the web page and downloading jnlp file again.
