Start from an existing content management system , done in PHP and MySQL if those are your requirements, such as eZ Publish , or Knowledgetree . For rapid testing of these applications, Bitnami provides quick-to-install " stacks " of these as well (WAMP-stacks on steroids). Then you can tailor these applications to your organizations needs, and stay up-to-date with the changes upstream.
