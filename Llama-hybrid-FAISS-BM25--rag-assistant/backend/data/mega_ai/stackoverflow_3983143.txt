JS Lint incorporates what Douglas Crockford considers to be the best practices for JavaScript. One of the functions he strongly discourages the use of is eval . I believe he considers it to be slow and insecure. There could be many potential alternatives, depending on the code in question. If you'd like to post the section of your code which uses eval , we can give more specific advice.
