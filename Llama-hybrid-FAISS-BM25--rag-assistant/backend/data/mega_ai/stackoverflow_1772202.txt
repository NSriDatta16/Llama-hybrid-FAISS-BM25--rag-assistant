You can add multiple formats of your data to the DataObject you pass into the DoDragDrop call, so just add another call to SetData to add the new formats. This is the most appropriate implementation, this way the Drop target can query for available formats and choose the one it likes best. DataObject d = new DataObject(); d.SetData(DataFormats.Serializable, myObject); d.SetData(DataFormats.Text, myObject.ToString()); myForm.DoDragDrop(d, DragDropEffects.Copy);
