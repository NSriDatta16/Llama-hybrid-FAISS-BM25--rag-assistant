For what it's worth, here's another way to extract tokens from an input string, relying only on standard library facilities. It's an example of the power and elegance behind the design of the STL. #include #include #include #include #include int main() { using namespace std; string sentence = "And I feel fine..."; istringstream iss(sentence); copy(istream_iterator (iss), istream_iterator (), ostream_iterator (cout, "\n")); } Instead of copying the extracted tokens to an output stream, one could insert them into a container, using the same generic copy algorithm. vector tokens; copy(istream_iterator (iss), istream_iterator (), back_inserter(tokens)); ... or create the vector directly: vector tokens{istream_iterator {iss}, istream_iterator {}};
