Pragmas swp,ivdep,prefetch support in various compilers

In good Altix manual (altix_single_cpu_optimization http://wwwuser.gwdg.de/~parallel/parallelrechner/altix_documentation/altix_single_cpu_optimization.pdf ) I've found this list of pragmas-hints for optimizing compilers #pragma ivdep // no aliasing #pragma swp // try to software-pipeline #pragma noswp // disable software-pipelining #pragma loop count (NN) // hint for SWP #pragma distribute point // split this large loop #pragma unroll (n) // unroll n times #pragma nounroll // do not unroll #pragma prefetch a // prefetch array “a” #pragma noprefetch c // do not prefetch array “c” What compilers does support these pragmas? And what does implement linked optimizations (software pipelining is one of the rather hard optimization for low ipc and low reg count processors) ?
