The BulkMutateJobService only allows for mutates, or changes, to the account. It does not provide the bulk retrieval of information. You can fetch monthly search volume for keywords using the TargetingIdeaService . If you use it in STATS mode you can include up to 2500 keywords per request. Estimates CPC values are obtained from the TrafficEstimatorService . You can request up to 500 keywords per request. FYI, there is an official AdWords API Forum that you can ask questions on.
