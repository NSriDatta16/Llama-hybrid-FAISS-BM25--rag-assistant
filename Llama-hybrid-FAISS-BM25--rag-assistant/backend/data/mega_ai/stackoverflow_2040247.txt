A tidy and Javaesque way to return multiple values is to create an object to return them in, e.g. public class Whatever { public String getUn() { return m_un; } public String setUn(String un) { m_un = un; } public String getPwd() { return m_pwd; } public String setPwd(String pwd) { m_pwd = pwd; } }; public Whatever getWhatever() { Whatever ret = new Whatever(); ... ret.setPwd(...); ret.setUn(...); ... return ret; }
