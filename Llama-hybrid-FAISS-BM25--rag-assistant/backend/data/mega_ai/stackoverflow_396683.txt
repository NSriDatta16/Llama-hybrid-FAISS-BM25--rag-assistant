SELECT DocNum, STUFF((SELECT ', ' + CAST(DocEntry AS VARCHAR(MAX)) AS [text()] FROM Temp5 b WHERE a.DocNum = b.DocNum FOR XML PATH('')), 1, 2, '') AS DocEntry FROM Temp5 a GROUP BY DocNum Itzik Ben-Gan in his excellent book T-SQL QUERYING has some specialized solutions for aggregate string concatenation. The query screams for itself.
