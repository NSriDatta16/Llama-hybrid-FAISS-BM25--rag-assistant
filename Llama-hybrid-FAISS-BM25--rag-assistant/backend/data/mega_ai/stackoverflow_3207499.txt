Almost certainly, Google is analysing the User Agent string of the request, deciding that you are not a browser, meaning you are probably some form of web spider, and decides it doesn't want to waste web server time helping some other serach engine. You can set the User Agent of your request via WebClient Headers property. client.Headers.Add ("user-agent", "Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.2; .NET CLR 1.0.3705;)");
