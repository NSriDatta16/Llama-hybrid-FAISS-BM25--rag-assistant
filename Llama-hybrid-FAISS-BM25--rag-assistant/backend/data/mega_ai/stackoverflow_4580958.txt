I think what you are trying to achieve is to get the result from a dialog box. If that is the case you want to do the following: private void ShowDialogAndDoSomethingBasedOnTheResult() { DialogResult result = MessageBox.Show( "Dialog text", "Caption to go in title bar", MessageBoxButtons.OK); if (result == DialogResult.OK) { //Do work } } See http://msdn.microsoft.com/en-gb/library/0x49kd7z.aspx for more examples.
