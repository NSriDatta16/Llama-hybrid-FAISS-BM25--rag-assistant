Why linear regression one variable uses squared cost function?

This is about linear regression course given by Andrew Ng on Coursea about machine learning. why cost function is $$ \frac{1}{m} \sum _{i=1}^m \left(h_\theta(X^{(i)})-Y^{(i)}\right)^2 $$ and not simply: $$ \frac{1}{m} \sum _{i=1}^m \lvert h_\theta(X^{(i)})-Y^{(i)} \rvert $$
