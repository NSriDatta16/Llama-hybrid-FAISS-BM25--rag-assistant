Run the commands using Process.Start . Example Using the override Process.Start(string fileName, string arguments) Process.Start("vsinstr", "-coverage hello.exe"); Process.Start("vsperfcmd", "/start:coverage /output:run.coverage"); Process.Start("hello"); Process.Start("vsperfcmd", "/shutdown");
