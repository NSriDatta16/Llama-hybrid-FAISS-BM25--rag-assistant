For anyone interested I sort of found a workaround for this myself. Not the most elegant solution, but in any case here's what worked for me: @user.image = params[:user][:image] if @user.image_integrity_error flash[:error] = "Changes could not be saved." @user.errors.add(:upload, @user.image_integrity_error) render 'edit' else @user.update_attribute(:image, @user.image) flash[:success] = "Profile updated successfully!" redirect_to user_path(@user) end
