You can assign a valid drop target selector in this way. $('.drag').drag(function( event, dd ){ // move the dragged element },{ drop:'#drop1' }); // assign drop selector You can also return some drop event handlers "false" $('.drop').bind('dropinit drop',function( event, dd ){ if ( $( dd.drag ).is('#drag1') ) return false; });
