If you type $ make all it will build all the things to the right of all : , such as svm-train, svm-predict, and svm-scale $(CXX) -shared -dynamiclib svm.o -o libsvm.so.$(SHVER) is the specific rule to build the target lib and it will translate to g++ -shared -dynamiclib svm.o -o libsvm.so.2 depending on the values of the variables CXX and SHVER.
