Your quotes are off.... also don't forget to set the URL you are sending the request to! ( $.ajax({ url: ... , type: ... ). Problem: $("ul#wall").prepend(" "+message_box+" "); You need to pay attention to which quotes will be rendered as part of the string and which quotes denote the string boundaries. Solution: $("ul#wall").prepend(' ' + message_box + ' '); Look at the difference in the syntax highlighting of these 2 code snippets. Try it out with this jsFiddle
