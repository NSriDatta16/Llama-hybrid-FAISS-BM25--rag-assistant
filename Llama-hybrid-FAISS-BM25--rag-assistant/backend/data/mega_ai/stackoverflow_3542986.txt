If this is only for your development machines then a "back to the future" solution might be a RAM Disk , aka RAM Drive. A RAM disk or RAM drive is a block of RAM (primary storage or volatile memory) that a computer's software is treating as if the memory were a disk drive (secondary storage). One product for example. Search for RAM Disk or RAM drive and you'll get a cornucopia of choices.
