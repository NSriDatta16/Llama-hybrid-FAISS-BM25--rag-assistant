Not LINQ syntax, but at least more elegant... List mergedNotes = notes .GroupBy(x => x.UserID) .SelectMany(g => GetMergedNotesFor(g.OrderBy(x => x.CreatedOn))) .ToList(); With my test data it creates the same result as your original code.
