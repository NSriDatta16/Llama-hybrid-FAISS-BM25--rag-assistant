The Fast Artificial Neural Network library (FANN) is a free open source neural network library, which implements multilayer artificial neural networks in C and supports both fully and sparsely connected networks. FANNTool is a GUI to the FANN library which allows its easy usage without the need o programming http://code.google.com/p/fanntool/
