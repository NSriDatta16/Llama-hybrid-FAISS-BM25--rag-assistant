It depends on the distro and the storage mechanism. MyISAM databases are often stored in one file per table under /var/lib/mysql/[databasename] All InnoDB databases are stored in the same file by default under, for example, /var/lib/mysql. An optimized installation may have the database files live in a special filesystem optimized for speed or reliability, or both.
