The first line of the trace warns you that your entities were not enhanced at build time or at class load time with a javaagent . And while my understanding is that running the enhancer is not is not strictly required, the following part of the trace is still an hint that OpenJPA is not aware of any entity : Error message: The name "Offeringattribute" is not a recognized entity or identifier. Known entity names: [] It looks like that something went wrong and/or that OpenJPA didn't use any of the fallback mechanisms . Look for a previous warning in the logs. Or setup build time enhancement. The following resources might help: Entity Enhancement OpenJPA User's Guide Section 2. Enhancement
