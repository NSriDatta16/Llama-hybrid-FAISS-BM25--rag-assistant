Try a UIImageView within a UIScrollView ! To detect your touchs in a UIImageView you have to rewrite a UIImageView and set UserInteractionEnabled to YES . Then you can handle events like touchesBegan and touchesEnded . UIImageWithTouchControl.h @interface UIImageWithTouchControl : UIImageView UIImageWithTouchControl.m -(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event { } -(void)touchesEnded:(NSSet *)touches withEvent:(UIEvent *)event { }
