you could use removeClass() to remove class values but the widget might not work if you do. example: $('.myitem').removeClass('ui-widget'); I suggest you add new class values using addClass() to overide the styles example: $('.myitem').addClass('newClass'); or target the element(s) $('p').addCLass('newClass'); and call the function at the end of your widget declaration. $(document).ready(function() { $("#MyItem").draggable(); //here $("#MyItem").addClass("newClass"); });
