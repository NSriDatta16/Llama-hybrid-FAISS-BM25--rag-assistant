Jets3t - http://jets3t.s3.amazonaws.com/index.html We've used this and it works like a charm for S3. I'm not sure I understand if you are looking for a framework that will work for both file-system storage and S3, but as unique as S3 is, I'm not sure that such a thing would exist. Obviously with S3, backups and maintenance are handled for you.
