H Declares the interface to a library - including functions, structures, and constants. Written in the C language. LIB Either declares the binary interface to a dynamic library (DLL) or contains the binary code of a library. DLL A dynamic library - your application shares these with the system or you use them to keep your code base organized. DEF A textual description of functions exported by a DLL.
