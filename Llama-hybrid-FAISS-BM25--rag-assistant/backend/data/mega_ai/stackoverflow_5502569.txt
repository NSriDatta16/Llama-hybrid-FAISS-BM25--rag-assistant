you can also use the std::string::find_first_of() method that allows you to search for character (delimiters) starting from a given position, e.g. size_t tok_end = linia.find_first_of(";", prev_tok_end+1); token = linia.substr(prev_tok_end+1, prev_tok_end+1 - tok_end); However, the boost solution is the most elegant.
