I think you should install the very good express framework. I really simplifies node.js webdevelopment. You could install it using npm npm install express This snippet shows you how to set headers and read headers var express = require('express'); var app = express.createServer(); app.get('/', function(req, res){ console.log(req.header('a')); res.header('time', 12345); res.send('Hello World'); }); app.listen(3000); Curl from command line $curl http://localhost:3000/ -H "a:3434" -v * About to connect() to localhost port 3000 (#0) * Trying ::1...
