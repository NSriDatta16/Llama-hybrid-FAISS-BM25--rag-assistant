"Contents" is where you tell the package manager which files you want to include in your package. You can include The path to a specific file The path to a specific directory, which will recursively include all the files in that directory Paths are relative based on type. For example, a Magento Local module file will be relative from ./app/code/local See this article for a full list of types/base-directories (self-link) http://alanstorm.com/magento_connect_role_directories
