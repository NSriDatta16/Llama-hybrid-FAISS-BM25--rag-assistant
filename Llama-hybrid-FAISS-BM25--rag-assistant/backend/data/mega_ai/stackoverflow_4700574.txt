The urlparse module has nothing to do with parsing HTML. All it does is break a URL up into bits: protocol, network address, path, etc. For example: >>> urlparse.urlparse("http://www.stackoverflow.com/questions/4699888") ParseResult(scheme='http', netloc='www.stackoverflow.com', path='/questions/4699888', params='', query='', fragment='') For parsing HTML, try BeautifulSoup .
