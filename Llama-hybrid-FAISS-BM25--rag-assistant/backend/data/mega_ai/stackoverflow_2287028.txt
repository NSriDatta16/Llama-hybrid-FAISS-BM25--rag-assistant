I have two nicely absurd but elegant examples that I love public static IEnumerable Information(this byte x) { return Enumerable.Range(0, 8).Select(i => ((x >> i) & 1) == 1); } public static IEnumerable Information(this IEnumerable xs) { return xs.SelectMany(Information); } Albeit these are encapsulated as query operators so you can reuse them, e.g. for binary parsing var n = bytes.Information().Skip(3).Take(16).ToInt();
