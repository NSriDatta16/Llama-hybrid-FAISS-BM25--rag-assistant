I'm guessing here but it seems that you want to swap the CSS styles of two elements after some drag&drop event. How about this solution: Create four CSS classes: .parentDIV_1 { ... } .parentDIV_2 { ... } .horiz { ... } .vert { ... } Assign two classes to the div s: ... ... Now you can toggle them: $('.horiz').toggleClass('vert').toggleClass('horiz'); $('.vert').toggleClass('vert').toggleClass('horiz');
