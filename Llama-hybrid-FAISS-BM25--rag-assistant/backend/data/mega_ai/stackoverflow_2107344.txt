From RFC 3986, Uniform Resource Identifiers (URI): Generic Syntax , appendix B (p. 50): ^(([^:/?#]+):)?(//([^/?#]*))?([^?#]*)(\?([^#]*))?(#(.*))? If the URI matches this regular expression, it's well formed. The match groups give you the various pieces, which are: scheme = $2 authority = $4 path = $5 query = $7 fragment = $9
