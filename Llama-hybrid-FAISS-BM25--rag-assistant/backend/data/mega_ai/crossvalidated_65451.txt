Random effect estimator (GLS estimator) is a weighted average of between and within estimators. In Stata, the default is random effect and you need to use R-squared: overall. As specified here , R-sq: within is not correct for fixed effect and there are alternatives to correct that in Stata. For example you need to use R-square from the one provided by either regress or areg .
