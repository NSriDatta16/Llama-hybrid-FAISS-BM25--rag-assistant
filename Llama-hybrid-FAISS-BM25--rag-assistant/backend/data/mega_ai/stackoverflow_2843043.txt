I'm not sure about 'zend', but for deal with number in lucene, you need use following technique: To place int to document use following: document.Add(new Field(FIELD_SPEC, NumberTools.LongToString(YOUR_INT), Field.Store.YES, Field.Index.UN_TOKENIZED)); To locate value use Term: Term(FIELD_SPEC, NumberTools.LongToString(YOUR_INT))
