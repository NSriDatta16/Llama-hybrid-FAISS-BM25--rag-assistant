Leveraging jQuery ... The $.getJSON() function parses a local JSON file and returns it as an object. $.getJSON("myJSON.js", function(json){ alert(json.summarydata.rows[0].name); }); You could also just do this, using the JSON library for javascript (the object is also standard in most recent browsers). alert(JSON.parse(myJSONString).summarydata.rows[0].name);
