If you have to use CreateSqlQuery, you can use the following: Product GetSpecificProduct() { ISQLQuery query = session.CreateSQLQuery("SELECT * FROM Products WHERE price = $500"); Product p = query.SetResultTransformer(Transformers.AliasToBean ()).UniqueResult (); } I suggest you better use ICriteria as in: Product GetSpecificProduct() { ICriteria c = session.CreateCriteria(); c.Add(Expression.Eq("Price", 500)); return c.UniqueResult (); }
