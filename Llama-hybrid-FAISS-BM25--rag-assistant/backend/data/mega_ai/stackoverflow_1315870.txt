Yes it is valid. This increases readability. But in your release version it would be better to minify your js file to reduce the bandwidth usage. Jsmin is a javascript minifer. JSMin is a filter which removes comments and unnecessary whitespace from JavaScript files. It typically reduces filesize by half, resulting in faster downloads. It also encourages a more expressive programming style because it eliminates the download cost of clean, literate self-documentation.
