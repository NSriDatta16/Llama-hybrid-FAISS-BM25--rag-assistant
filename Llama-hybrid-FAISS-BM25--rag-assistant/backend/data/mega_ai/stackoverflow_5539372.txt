This will work, although it is discouraged. _G["for"] = 42 print(_G["for"]) Edit to answer comment : As AdrianMar said, they are reserved for a reason. Plus, why do you want to do that ? It's very unlikely you have to. And finally, quoting the Programming in Lua book (Chapter 14) : Beware, however: some programmers get a little excited with these facilities and end up writing code like _G["a"] = _G["var1"] , which is just a complicated way to write a = var1 .
