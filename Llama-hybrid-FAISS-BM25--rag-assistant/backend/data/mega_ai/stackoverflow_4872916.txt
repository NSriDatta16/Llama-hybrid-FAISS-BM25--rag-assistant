Andrew, I'm afraid that your namespace is not correct. I think that version 2.5 should be under javaee namespace (JDK, Java 1.5 and 1.6), not j2ee (J2SDK, Java 1.4). So replace http://java.sun.com/xml/ns/j2ee/web-app_2_5.xsd with http://java.sun.com/xml/ns/javaee/web-app_2_5.xsd
