You're not very specific about which properties you want to bring back, but you can always pull back more properties by adding this to you MDX query (assuming this is how you are querying, as it seems you are using a drag and drop editor): SELECT {[YourDimension]} DIMENSION PROPERTIES KEY0,MEMBER_CAPTION ON COLUMNS
