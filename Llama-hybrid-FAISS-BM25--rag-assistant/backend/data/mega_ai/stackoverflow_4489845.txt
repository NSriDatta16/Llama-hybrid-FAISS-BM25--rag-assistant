You need at least two instances if you want to have an SLA. Depending on the load you expect I would assume that a Small instance variant would be enough for you. making use of a SQL CE database That would rather become either SQL Azure or table storage instead. SQL Azure costs quite some more money than table/blob storage so try to get more information about what you want to use if costs matter to you. To calculate the costs I suggest you check out this great tool: http://azureroi.cloudapp.net/ .
