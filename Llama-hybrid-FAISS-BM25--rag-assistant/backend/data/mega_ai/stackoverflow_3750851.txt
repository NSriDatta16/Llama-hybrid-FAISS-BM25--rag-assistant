To do that you set up a constraint between the tables with cascading delete. You can do that in a diagram by dragging a connection between fields and editing the properties, or using a query: alter table SomeTable add constraint SomeConstraint foreign key (SomeField) references SomeOtherTable (SomeOtherField) on delete cascade
