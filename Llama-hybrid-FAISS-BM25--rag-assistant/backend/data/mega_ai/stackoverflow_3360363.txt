There are a number of ways to do what you want; the simplest might be to use ' scatter.smooth ' which more or less packs everything you need (data plot, loess fit, and curve plot) in a single functional call. data(AirPassengers) # a monthly time series supplied w/ base R install scatter.smooth( x=1:length(AP), y = as.vector(AP), pch=20, # this line and lines below are just aesthetics col="orange", lty="dotted", lwd=1.5, xlab="")
