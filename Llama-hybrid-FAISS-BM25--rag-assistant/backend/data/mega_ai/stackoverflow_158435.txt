This is very simple to do with my TGpTextFile unit. I'll put together a short sample and post it here. It should also be very simple with the new Delphi 2009 - are you maybe using it? EDIT: This his how you can do it using my stuff in pre-2009 Delphis. var strAnsi : TGpTextFile; strUnicode: TGpTextFile; begin strAnsi := TGpTextFile.Create('c:\0\test.udl'); try strAnsi.Reset; // you can also specify non-default 8-bit codepage here strUnicode := TGpTextFile.Create('c:\0\test-out.udl'); try strUnicode.Rewrite([cfUnicode]); while not strAnsi.Eof do strUnicode.Writeln(strAnsi.Readln); finally FreeAndNil(strUnicode); end; finally FreeAndNil(strAnsi); end; end; License: The code fragment above belongs to public domain. Use it anyway you like.
