If you want SQLite to not use temporary files as journals, e.g, you don't want any file activity other than when you manually requests a connect, disconnect, attach or detach. Then use the following two pragmas at runtime after you connect to your ":memory:" database. PRAGMA temp_store=MEMORY; PRAGMA journal_mode=MEMORY; From the docs.
