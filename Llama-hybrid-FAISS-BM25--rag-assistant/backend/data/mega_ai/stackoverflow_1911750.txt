Assuming that you have 3 radiobuttons like this: When you POST, you will get just 1 value, the one checked, like this: var radio = Convert.ToInt16(Request.Form["radio"]); The formCollection will contain just one key with the respective value. I encourage you to use the model binders either by direct parameter binding: public ActionResult Action(int radio) or using a ViewModel.
