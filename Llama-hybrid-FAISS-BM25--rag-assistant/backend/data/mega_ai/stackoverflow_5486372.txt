$mesg will have array of entries. You are trying to print the first entry from the search result. Try, print Dumper $mesg also change your filter to filter => '(objectClass=organizationalUnit)' ldapsearch start the search from the base dn and including basedn. Here OU=test,OU=company,DC=example,DC=com is also organizationalunit so this entry is coming as a first entry in the result for you and you are printing only that.
