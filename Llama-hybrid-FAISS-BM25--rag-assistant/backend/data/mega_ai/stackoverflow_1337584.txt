None that I know of. But unless you're doing something really extravagant, it really isn't a lot of effort to override OnRender and draw it yourself: public class CustomBorder : Border { protected override void OnRender(DrawingContext dc) { base.OnRender(dc); dc.DrawLine(new Pen(BorderBrush, BorderThickness.Top), new Point(0, 0), new Point(ActualWidth, ActualHeight)); } } Result:
