All addresses of every single place in the world (or the US)... That takes a lot of space, more than the average iOS device can handle. So, definitely not an offline version. However, you can simply use Open Street Maps which can help you get some data, store this in some kind of cache (store only everything in a 5 miles radius, for example) and refresh this cache when you leave the area.
