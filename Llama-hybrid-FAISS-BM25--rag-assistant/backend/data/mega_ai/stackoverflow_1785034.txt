You just need to pass the average up the stack with the return value: let foo ls = let rec foo xs sumAcc lenAcc = match xs with | x::xs -> let avg,s = foo xs (x + sumAcc) (1 + lenAcc) in if x (sumAcc / lenAcc),0 in let avg,res = foo ls 0 0 in res
