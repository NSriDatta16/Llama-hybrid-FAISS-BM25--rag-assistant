Google internally use C++, Java and Python. See Rhino on Rails : One of the (hundreds of) cool things about working for Google is that they let teams experiment, as long as it's done within certain broad and well-defined boundaries. One of the fences in this big playground is your choice of programming language. You have to play inside the fence defined by C++, Java, Python, and JavaScript. Google's search algorithm is essentially MapReduce , which stems from functional programming techniques, implemented in C++. Google has its own storage mechanism for this called the Google File System .
