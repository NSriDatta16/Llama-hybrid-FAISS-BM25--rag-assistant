The Channel hack did not work for me. So I just added above all other content in my PHP file some code that 301 redirects to the URL without the fb_xd_fragment addition: $url = "http://".$_SERVER['SERVER_NAME'].$_SERVER['REQUEST_URI']; if (isset($_GET['fb_xd_fragment'])) { $url = str_replace("?fb_xd_fragment=","",$url); header( "HTTP/1.1 301 Moved Permanently" ); header("Location: {$url}"); exit(); }
