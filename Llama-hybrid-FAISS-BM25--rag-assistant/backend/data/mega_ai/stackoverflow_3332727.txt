You'll need to set some headers in php to ensure the file is cached. At the top of js.php put: ob_start("ob_gzhandler"); $expires = 2678400; // 1 month in seconds header("Pragma: public"); header("Cache-Control: maxage=".$expires); header('Expires: ' . gmdate('D, d M Y H:i:s', time()+$expires) . ' GMT'); That will add both basic caching + gzip compression on the fly.
