this post was really helpful for me, maybe it will be helpful for you too http://hacks.mozilla.org/2010/06/html5-adoption-stories-box-net-and-html5-drag-and-drop/ I ended up changing my javascript to be: holder.ondrop = function(e) { e.preventDefault(); var file = e.dataTransfer.files[0]; var request = new XMLHttpRequest(); request.open("POST", upload_url, true); // open asynchronous post request request.setRequestHeader("X-Requested-With", "XMLHttpRequest"); request.send(file); }; and voila.
