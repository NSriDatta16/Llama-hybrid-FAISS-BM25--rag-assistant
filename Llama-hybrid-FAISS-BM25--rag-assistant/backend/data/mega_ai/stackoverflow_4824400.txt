Your phone can be set to debug on connecting through USB. Then, it will always debug irrespective of whether you are tethering, charging or connecting on USB. If you want to debug and have access to USB through the phone, set it on charging. If you want to debug and have access to the USB storage on your computer, use Mass storage.
