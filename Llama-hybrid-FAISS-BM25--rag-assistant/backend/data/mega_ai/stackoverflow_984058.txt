Does this seem like an alright approach to implementing tabs in ASP.NET MVC?

After looking at how the Orange Tabs ASP.NET MVC demo handles tabs, they have something like: View: and a corresponding helper class: namespace Helpers { public static class IsCurrentActionHelper { public static bool IsCurrentAction(this HtmlHelper helper, string actionName, string controllerName) { string currentControllerName = (string)helper.ViewContext.RouteData.Values["controller"]; string currentActionName = (string)helper.ViewContext.RouteData.Values["action"]; if (currentControllerName.Equals(controllerName, StringComparison.CurrentCultureIgnoreCase) && currentActionName.Equals(actionName, StringComparison.CurrentCultureIgnoreCase)) return true; return false; } } } Does this seem like an elegant way to approach this problem?
