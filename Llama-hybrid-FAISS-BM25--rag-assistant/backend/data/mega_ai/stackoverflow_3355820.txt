you can use apache http components httpclient to do this, it will automatically follow redirects HttpClient client=new HttpClient(); GetMethod get=new GetMethod("graph.facebook.com/oauth/authorize?client_id=...& redirect_uri=www.facebook.com/connect/login_success.html&type=user_agent&display=popup"); int status=client.exectueMethod(get); Then you will have the information you need in the Location header of the response which you can access by using: String location=get.getResponseHeader("location").getValue(); and parse the location-header for the url fragment you want. hope that helped
