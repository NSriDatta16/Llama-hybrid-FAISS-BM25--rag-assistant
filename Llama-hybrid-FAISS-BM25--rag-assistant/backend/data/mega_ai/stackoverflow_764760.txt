Assuming you choose the Lego path, for Mindstorms I'd definitely recommend NQC . NQC is a programming language that you can use to program the Lego Mindstorms robots, and I found it much easier than dragging nodes around in a WYSIWYG-style environment (which is how the bundled software and LabView-based version worked). I haven't had any experience with the NXT robot, but if NQC sounds interesting, there's an equivalent for the NXT: NBC/NXC . Steve
