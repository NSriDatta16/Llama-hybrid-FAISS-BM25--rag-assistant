I've updated my script including the suggestion provided by dipetkov (thank you very much) and included a Bootstrap simulation to evaluate whether the standard deviation estimates were consistent with the values defined by the simulation. Attention to the standard deviation (theta) values for the random effects that need to be scaled by the residual SD. rm(list=ls(all=TRUE)) library(lme4) library(ggplot2) library(reshape2) set.seed(1) # Set up parameters for simulation SD1 $Y = Y$ sim_1 head(data) str(data) #Parametric Bootstrap simulation SD.DF = NULL for (i in 1:50){ Y = simulate( ~ 1 + (1 | X1) + (1 | X2 / X3), newdata = data, newparams = list( beta = c(`(Intercept)` = 0), theta = c(`X1.(Intercept)` = SD1/SDResidual, `X2.(Intercept)` = SD2/SDResidual, `X3:X2.(Intercept)` = SD3/SDResidual), sigma = SDResidual ) ) data $Y = Y$ sim_1 LM $varcor)[,5] SD.DF = rbind(SD.DF,VARCORR.) } colnames(SD.DF) = data.frame(summary(LM)$ varcor)$grp SD.DF DF.MELT = melt(SD.DF) ggplot(DF.MELT, aes(x = value)) + geom_histogram() + facet_wrap(.~Var2, scale = "free_x") + geom_vline(data = data.frame(Var2 = colnames(SD.DF), value = c(SD3, SD2, SD1, SDResidual)), aes(xintercept = value),size = 2, color = "gold") + theme_bw(base_size = 13)
