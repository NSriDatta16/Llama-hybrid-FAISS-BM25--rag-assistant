Note: The posterior probablity: $P(M_i|X)= \displaystyle\frac{P(X|M_i)P(M_i)}{\sum_kP(X|M_K)P(M_K)}$ So when i=1 we get that it's equal to 1, so if there is only one M, then it will happen for sure. More inf on model averaging: http://www.stat.colostate.edu/~jah/papers/statsci.pdf
