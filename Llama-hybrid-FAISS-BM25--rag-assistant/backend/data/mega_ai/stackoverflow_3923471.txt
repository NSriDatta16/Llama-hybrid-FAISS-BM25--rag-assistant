Good literature on unit testing?

Where can I find good literature on unit testing? Book titles and links are welcome. Update: Here is a list of books mentioned in answers below xUnit Test Patterns: Refactoring Test Code Growing Object-Oriented Software Guided by Tests The Art Of Unit Testing The real challenge of software testing is solving the puzzle of test design. Testing Object-Oriented Systems: Models, Patterns, and Tools provides three dozen test design patterns applicable to unit test design. It also provides many design patterns for test automation. These patterns distill many hard-won best practices and research insights. Pragmatic Unit Testing Test Driven Development: By Example
