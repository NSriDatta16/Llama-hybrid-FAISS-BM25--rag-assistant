You can create an IPA file in Xcode with "Build -> Build and Archive". Now open "Window -> Organizer" and you find the IPA in the section "Archived Applications". Save the IPA to a folder now. IMPORTANT: You have to be an Apple developer to create an IPA file. The IPA file have to be signed with certificates, etc. ...
