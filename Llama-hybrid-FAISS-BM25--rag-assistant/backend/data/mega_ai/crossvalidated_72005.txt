You could fit a two-component mixture model using http://cran.r-project.org/web/packages/mixtools/index.html . Try using normalmixEM. You could then follow Erich Schubert's suggestions and find the region where Pr[data point was generated from the component with the smaller mean] >= 0.50. Edit: example R code: library(mixtools) simulate
