Use the simple_format helper. It's replace all \n by or Examples from API : my_text = "Here is some basic text...\n...with a line break." simple_format(my_text) # => " Here is some basic text...\n ...with a line break. " more_text = "We want to put a paragraph...\n\n...right there." simple_format(more_text) # => " We want to put a paragraph... \n\n ...right there. " simple_format("Look ma! A class!", :class => 'description') # => " Look ma! A class! "
