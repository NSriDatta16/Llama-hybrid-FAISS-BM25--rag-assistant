Any floating point type, but probably double precision as a compromise between accuracy and storage. float is generic floating point type that you can specify the precision of. Data Type | Range | Storage float -1.79E+308 to -2.23E-308, 0 and 2.23E-308 to 1.79E+308 Depends on the value of n real -3.40E + 38 to -1.18E - 38, 0 and 1.18E - 38 to 3.40E + 38 4 Bytes The synonym for real is float(24). The synonym for double precision is float(53). Source If you want more accuracy you should use decimal .
