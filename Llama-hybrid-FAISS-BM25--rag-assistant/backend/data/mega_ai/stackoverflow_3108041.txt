When the type attribute has the value "text" or "password", this attribute specifies the maximum number of characters the user may enter. This number may exceed the specified size, in which case the user agent should offer a scrolling mechanism. The default value for this attribute is an unlimited number. Source: http://www.w3.org/TR/html401/interact/forms.html#adef-maxlength
