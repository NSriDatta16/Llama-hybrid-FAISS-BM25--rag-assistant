Key part to understand that code is the replace(/[\^H\!9X]/g, '') parts. if the 2nd argument for the replace is '' , then it's merely removing stuff from the previous string. Really inelegant way to obfuscate things. Probably the aim is just to be random for each user and avoid Bayesian spam filters.
