You could leverage wiring up the UnhandledException event from the AppDomain and routing all unhandled exceptions to whatever you like. AppDomain currentDomain = AppDomain.CurrentDomain; currentDomain.UnhandledException += (sender, args) => { /* Write To Wherever */ }; I would recommend fleshing that out to using a real handler etc, but it gives you an idea.
