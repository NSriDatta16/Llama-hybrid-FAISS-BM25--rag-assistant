The .draggable() syntax is a little different than you have, you need to set .draggable() on the element that you want draggable ( #container in this case) then provide a selector to the handle option , like this: $("#container").draggable({ containment: 'window', scroll: false, handle: '#titlebar' });​​​ You can see a working demo here .
