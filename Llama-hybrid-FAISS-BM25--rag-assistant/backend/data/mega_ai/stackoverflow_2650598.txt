MSSQL answer: If you can get the files automatically (residing on a share, ready to copy, ...) you can use SSIS to create a package that does the required stept and then schedule it with SQL Server Agent. General answer: You can automate repeating tasks, specific way to do this differs at least as much as the involved product names.
