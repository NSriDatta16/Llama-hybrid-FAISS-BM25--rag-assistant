The results you get are due to the following facts : A sequence of nodes contains the nodes as they are in their respective document -- not copies of the nodes. creates a copy of each node selected by the expression in the select attribute. creates a new XML fragment (temporary tree) and this is the value of the variable $someElementA2 $someElementA1/a does not select any node, because none of the elements in $someElementA1 have any children named a . $someElementA2/a selects all elements a that are children of the document node ( / ).
