The way you'd want to do this is to make a new DBManager object that creates the database in its onCreate method--that way when the DatabaseHelper is first instantiated, the table will be created for you. Then you'd instantiate one and call getReadableDatabase() on it to get a DB object which you can then query . This tutorial may help you more, it's more succinct and up to date: http://developer.android.com/guide/topics/data/data-storage.html#db After that, to set up the list view in a ListActivity , you can call setListAdapter and pass in a SimpleCursorAdapter . Here's a tutorial on that: http://developer.android.com/guide/appendix/faq/commontasks.html#binding
