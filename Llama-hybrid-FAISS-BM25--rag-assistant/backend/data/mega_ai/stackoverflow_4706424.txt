Just change your model's data storage, in between beginInsertRows() and endInsertRows(). For instance, let's say you have a flat list model and your model stores the data internally in a QVector m_data. You want to prepend the list, i.e. insert a row at position 0: beginInsertRows( QModelIndex(), 0, 0 ); //notify views and proxy models that a line will be inserted m_data.prepend( somedata ); // do the modification to the model data endInsertRows(); //finish insertion, notify views/models
