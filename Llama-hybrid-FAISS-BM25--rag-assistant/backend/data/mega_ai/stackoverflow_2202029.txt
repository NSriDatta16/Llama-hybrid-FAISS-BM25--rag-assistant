Python uses these characters for splitting in unicode.splitlines() : U+000A LINE FEED (\n) U+000D CARRIAGE RETURN (\r) U+001C FILE SEPARATOR U+001D GROUP SEPARATOR U+001E RECORD SEPARATOR U+0085 NEXT LINE U+2028 LINE SEPARATOR U+2029 PARAGRAPH SEPARATOR As Hank says, using splitlines() will let Python take care of all of the details for you, but if you need to do it manually, then this should be the complete list.
