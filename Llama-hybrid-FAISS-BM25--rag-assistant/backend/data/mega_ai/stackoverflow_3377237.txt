Assuming you are using jQuery Draggable, you could use the drag part of it: $("#draggable").draggable({ drag: function() { var offset = $(this).offset(); $(this).html(offset.left + ', ' + offset.top); } }); Example: http://jsfiddle.net/QH6QP/
