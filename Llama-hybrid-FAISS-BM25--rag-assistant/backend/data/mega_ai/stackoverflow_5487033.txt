Use drupal_goto with the path to redirect to as parameter: $items['adminQuestion/create'] = array( 'title' => t('Crear Cuestionarios'), 'page callback' => 'drupal_goto', 'page arguments' => array('node/add/webform'), 'access arguments' => array('access questionnaires'), 'type' => MENU_NORMAL_ITEM, ); Also note that $items['adminQuestions'] is bad practice: URLS and paths should never be case-sensitive: in fact: in Drupal CamelCase is highly discouraged in any code.
