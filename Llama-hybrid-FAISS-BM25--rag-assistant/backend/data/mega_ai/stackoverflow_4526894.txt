I am not sure if it's supported by all major browsers. It is. jQuery brings its own data storage implementation that is independent from browsers. Additionally, HTML 5's data-* properties get pulled in when present (i.e. they are accessible through .data() ). According to the manual, the only exception is XML documents: Note that this method currently does not provide cross-platform support for setting data on XML documents, as Internet Explorer does not allow data to be attached via expando properties. jQuery docs on .data()
