I think what you need is a result transformer. Create a model class for your query. Then add null checks or other checks into your setter methods. Something like this, String hql="select count(distinct ip) as ip, sum(bytes) as bytes from Entity en where .... Query q=Session.createQuery(hql).setResultTransformer(Transformers.aliasToBean(Model.class)); And in your Model class, class Model { private String ip = ""; private BigDecimal bytes = BigDecimal.ZERO; public void setIp(String anIp) { if ( aString != null) { this.ip = anIp; } } public void setBytes(String aByte) { if ( aByte != null) { this.byte = aByte; } } . . . }
