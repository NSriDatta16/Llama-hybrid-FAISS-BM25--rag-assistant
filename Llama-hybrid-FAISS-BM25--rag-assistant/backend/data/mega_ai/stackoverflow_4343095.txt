If you want to selectively remove links without having to copy/edit entire xml files, a nice solution can be found in this post in the magento forums In this solution, you override the Mage_Customer_Block_Account_Navigation block with a local version, that adds a removeLinkByName method, which you then use in your layout.xml files, like so: recurring_profiles billing_agreements
