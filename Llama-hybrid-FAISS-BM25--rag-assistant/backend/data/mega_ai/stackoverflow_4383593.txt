That is just the way lag() for ts objects is defined. From help(lag) : lag package:stats R Documentation Lag a Time Series Description: Compute a lagged version of a time series, shifting the time base back by a given number of observations. Use xts() or zoo() if you'd rather have NAs added while keeping start and end times.
