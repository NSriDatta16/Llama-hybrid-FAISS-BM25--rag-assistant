Consider making a Webmin plugin. Webmin: Comes with its own web server which can do SSL Will probably squeeze in 16MB no sweat Is specifically made for being a GUI front-end to modify Unix and Linux systems Has a well-documented plugin API Has a ton of example modules (plugins) you can learn from It's no silver bullet, but for the niche it serves, it satisfies it 100%. The only downside is that the API, having a long legacy, isn't exactly elegant and you won't win any buzzword awards with it.
