I think the most elegant approach would be to use NumberFormat.parse(String,ParsePosition) , something like this: public class MyNumberEditor extends PropertyEditorSupport { private NumberFormat f; public MyNumberEditor(NumberFormat f) { this.f = f; } public void setAsText(String s) throws IllegalArgumentException { String t = s.trim(); try { ParsePosition pp = new ParsePosition(0); Number n = f.parse(t, pp); if (pp.getIndex() != t.length()) throw new IllegalArgumentException(); setValue((Float) n.floatValue()); } catch (ParseException ex) { throw new IllegalArgumentException(ex); } } ... }
