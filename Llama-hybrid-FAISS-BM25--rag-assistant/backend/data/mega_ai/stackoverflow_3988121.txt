The Same Origin Policy prevents Ajax calls to external domains. Popular workarounds include JSONP Embedding the data in an iframe instead Using a server-side proxy the does the fetching (see @BrunoLM's answer for a PHP example; it is possible in any server-side language) YUI's Get as shown in @Alex's answer depending on what your use case is.
