One alternative is to use the universal selector to make a more specific rule which will take effect if the nodes aren't direct children: div p {color: red;} // Takes effect if there's a child at some level div * p {color: black;} // .. but this'll be true if it's not a direct child However, you need to watch out for specificity conflicts. div * p will be more specific than another rule that works on paragraphs in general, for example.
