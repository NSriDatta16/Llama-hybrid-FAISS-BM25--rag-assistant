Use Request.UserAgent instead. Beware that it includes the word Mobile as well: Mozilla/5.0 (iPad; U; CPU OS 3_2 like Mac OS X; en-us) AppleWebKit/531.21.10 (KHTML, like Gecko) Version/4.0.4 Mobile/7B367 Safari/531.21.10 From Apple on preparing content for iPad: https://developer.apple.com/library/content/technotes/tn2010/tn2262/_index.html
