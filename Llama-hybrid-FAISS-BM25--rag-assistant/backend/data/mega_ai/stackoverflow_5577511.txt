make your table row clickable and attach an onClick listener tableRow.setClickable(true) tableRow.setOnClickListener(new View.OnClickListener(){ public void onClick(View view){ System.out.println("Row Clicked with tag " + view.getTag()); } }) Attention: view.getTag() does not have any argument! this returns just an object. You added an integer as a tag for this view so you can do something like this if (((int) tableRow.getTag()) == 1) { bla bla }
