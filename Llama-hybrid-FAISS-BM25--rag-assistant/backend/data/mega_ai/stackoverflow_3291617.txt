Disable locale considerations for the block and use cmp as usual: sub mycmp { no locale; $_[0] cmp $_[1]; } The perlop documentation provides lt , le , ge , gt and cmp use the collation (sort) order specified by the current locale if use locale is in effect. See perllocale . and then in perllocale The default behavior is restored with the no locale pragma, or upon reaching the end of block enclosing use locale . For example, running my($one,$two) = map pack("N", $_) => 1, 2; say mycmp($one, $two); say mycmp($two, $one); outputs -1 1
