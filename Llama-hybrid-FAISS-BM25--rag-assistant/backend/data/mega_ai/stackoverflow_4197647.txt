PST format specification Wiki article on PST and OST file formats lipff - opensource C library and tools to access the Personal Folder File (PFF) and the Offline Folder File (OFF) format. PFF is used in PAB (Personal Address Book), PST (Personal Storage Table) and OST (Offline Storage Table) files. PST file format SDK C++. Can read PST and OST files.
