You can pass many values back to your jQuery script in the form of an array. For instance, I could pass the following: session_start(); // necessary for getting/setting SESSION data print json_encode(array( "username" => $_SESSION["username"], "msgCount" => 0, "userHTML" => " There are no new messages "; )); Then from your jQuery code, you'll use the JSON method of data-retrieval: $.post("page.php", {}, function(results) { alert(results.username + " has " + results.msgCount + " messages."); $("messages").append(results.userHTML); }, "json");
