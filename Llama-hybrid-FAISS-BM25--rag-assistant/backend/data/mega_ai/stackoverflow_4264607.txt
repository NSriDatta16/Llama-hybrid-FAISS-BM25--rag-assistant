Your syntax is invalid, you're missing some closing quotes ' . Also, look at the documentation for the possible .bind syntax: .bind( eventType, [ eventData ], handler(eventObject) ) .bind( eventType, [ eventData ], false ) You'll have to use $('#drop_zone').bind('dragover', handleDragOver) , otherwise handleDragOver is interpreted to be the eventData parameter, not as the handler parameter.
