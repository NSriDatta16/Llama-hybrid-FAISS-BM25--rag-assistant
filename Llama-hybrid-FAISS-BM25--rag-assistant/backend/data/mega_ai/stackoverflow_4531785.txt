No, it is not possible for 3rd party developers to run application in the background as well as startup for Windows Phone 7.0. What you can do is update the Tile(icon) using Push Notification. UDPATE: WP7 7.5 (Mango) has Background Agent to mimic most of the behavior of background processing.
