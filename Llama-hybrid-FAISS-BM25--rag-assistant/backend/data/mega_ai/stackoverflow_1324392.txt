EDI stands for Electronic Data Interchange and is a generic term to describe the transmission of certain structured data between organizations. Gentran is an EDI tool that can parse and convert data from one format to another (such as XML, AS2 , or X12 ). It has a Business Process Modeling Language component where drag & drop programming can take place. Mapping is a generic term that describes matching up data points from one document format to another. For example, if my company received an EDI request in AS2 format and we used XML internally, I would map the fields from the AS2 document to a new XML document that my systems could read. Mapped fields could include things like the part numbers that the company is ordering, the shipping address, etc...
