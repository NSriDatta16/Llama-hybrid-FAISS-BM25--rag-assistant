We can detect with touchesBegan -(void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event { NSUInteger numTaps = [[touches anyObject] tapCount]; UITouch* t; if([[event allTouches] count]==2)//double tap { t=[[[event allTouches] allObjects] objectAtIndex:0]; CGPoint p1=[t locationInView:self.view]; } numTaps gives the nuber of taps . P1 has the point where it is tapped. All the best.
