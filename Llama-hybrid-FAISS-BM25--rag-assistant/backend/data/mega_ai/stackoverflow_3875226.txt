I would recommend compiling with -std=gnu99 . To elaborate on this. By default, gcc compiles with -std=gnu89. Here are the results for the following source code. #include int main() { struct timespec asdf; return 0; } [1:25pm][wlynch@cardiff /tmp] gcc -std=gnu89 foo.c [1:26pm][wlynch@cardiff /tmp] gcc -std=gnu99 foo.c [1:25pm][wlynch@cardiff /tmp] gcc -std=c89 foo.c foo.c: In function ‘main’: foo.c:4: error: storage size of ‘asdf’ isn’t known [1:26pm][wlynch@cardiff /tmp] gcc -std=c99 foo.c foo.c: In function ‘main’: foo.c:4: error: storage size of ‘asdf’ isn’t known
