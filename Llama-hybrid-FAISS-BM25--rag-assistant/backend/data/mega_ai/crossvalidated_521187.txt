Keeping aside other assumptions. If strict exogeneity hold the OLS parameter are unbiased and consistent. If weak form hold, but not strict, the OLS parameters are consistent but not unbiased. However pay attention about the fact that exogeneity is a causal concept. About the last point read here: Strict exogeneity and lagged variables
