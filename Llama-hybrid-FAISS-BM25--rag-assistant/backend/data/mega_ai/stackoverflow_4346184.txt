I use this to handle swf-files stored in as blobs. You should easily be able to modify it to suit your needs and type of files. class SwfHandler(webapp.RequestHandler): def get(self): query = cgi.FieldStorage() swf = BlobStorage.get_by_id(int(query.getvalue('id'))) if swf: self.response.headers['Content-Type'] = 'application/x-shockwave-flash' self.response.out.write(swf.fileData) else: self.error(404) ## add "('/swfhandler/', SwfHandler)" to your application : application = webapp.WSGIApplication([('/swfhandler/', SwfHandler), ****) I request the file with: /swfhandler/?id=12
