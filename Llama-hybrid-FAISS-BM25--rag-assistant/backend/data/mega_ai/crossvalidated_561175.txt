The standard logistic function is just $$ \sigma(x) = \frac{1}{1 + e^{-x}} = \frac{e^x}{e^x + 1} $$ so you don't need any $\beta$ 's. The $\beta$ 's appear in logistic regression. See also the Softmax vs Sigmoid function in Logistic classifier? thread. So you've proven what you wanted to show.
