Embedding javascript in the page is bad practice . Add the submit to the item using a selector. This also enables you to add the functionality to a basic link, as you have requested. HTML Upload file jQuery $(document).ready(function() { $("#uploadLink").click(function() { $("#myForm").submit(); } }); Additonal resources Here is a link to the jQuery Form plugin .
