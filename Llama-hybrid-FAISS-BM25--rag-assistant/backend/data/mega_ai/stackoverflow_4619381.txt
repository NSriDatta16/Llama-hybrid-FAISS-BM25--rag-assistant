How about using Threading? MSDN has a good write-up from back in 2007 on the basics: WPF Threads: Build More Responsive Apps with The Dispatcher Make sure you don't only pay attention to the Dispatcher but also the second on BackgroundWorker . If you're not familiar with Threading at all (which seems like it might be the case), then you should probably start out with the plain Threading basics from MSDN: Threading (C# and Visual Basic)
