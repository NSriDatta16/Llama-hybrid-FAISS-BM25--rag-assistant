It sounds like you might have Sharepoint already deployed in your environment. Consider integrating your application with Sharepoint, and have it be your datastore. No need to recreate all the things you like about Sharepoint, when you could leverage it. It'd take a bit of configuring, but you could call SP web services to CRUD your list data for you. inserting list data into Sharepoint via web services reading SP lists via web services Sharepoint 2010 can also expose lists via OData, which would be simple to consume from any application.
