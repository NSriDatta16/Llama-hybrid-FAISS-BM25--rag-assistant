To the best of my knowledge, you can prevent padding in structures by using #pragma pack ( gcc , vc++ ) #pragma pack(push, 1) struct Example { int a; char b; short c; }; #pragma pack(pop) Printing sizeof(Example) shows it is 7 bytes. Without the #pragma pack, the size is 8 bytes.
