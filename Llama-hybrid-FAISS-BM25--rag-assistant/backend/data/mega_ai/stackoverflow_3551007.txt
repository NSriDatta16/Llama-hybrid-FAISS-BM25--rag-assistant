File sdCard = Environment.getExternalStorageDirectory(); File file = new File(sdCard, "image.jpg"); FileOutputStream fos = new FileOutputStream(file); Use fos reference as a 3rd parameter of b.compress() method in Moncader's answer. The image will be stored as image.jpg in root directory of your sd card.
