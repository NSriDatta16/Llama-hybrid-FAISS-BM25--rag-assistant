Here are my implementations; I will benchmark and accept faster (or sufficiently more elegant) solutions: # Implementation 1 class Hash def slice(*keys) Hash[keys.zip(values_at *keys)] end end # Implementation 2 class Hash def slice(*keys) {}.tap{ |h| keys.each{ |k| h[k]=self[k] } } end end # Implementation 3 - silently ignore keys not in the original class Hash def slice(*keys) {}.tap{ |h| keys.each{ |k| h[k]=self[k] if has_key?(k) } } end end
