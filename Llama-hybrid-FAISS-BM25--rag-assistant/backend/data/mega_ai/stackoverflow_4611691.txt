If you view your REST request response, the JSON might be something like this { "users": [ {"first": "Peter", "last": "Griffin"} ], "books": [ {"title": "Design Patterns", "author": "The Gang of Four", "year": 1995} ] } This rest query result can be loaded as a javascript array object var resultObj = ; Then, you access it in your html with javascript blocks like document.write(resultObj.users[0].first) which would return 'Peter'
