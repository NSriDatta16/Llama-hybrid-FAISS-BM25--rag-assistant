Do you mean the alpha value of the fragment you're drawing on (which would explain why you want alpha_ * current_alpha )? If so, remember that reading a fragment back from the pipeline is expensive . If you're rendering back to front, consider using the GL_SRC_ALPHA + GL_ONE_MINUS_SRC_ALPHA trick.
