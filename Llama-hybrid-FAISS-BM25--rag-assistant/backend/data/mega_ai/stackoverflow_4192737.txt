From rcov manual : --[no-]callsites Show callsites in generated XHTML report. (somewhat slower; disabled by default) --[no-]xrefs Generate fully cross-referenced report. (includes --callsites) From Rcov CallSiteAnalyzer Class A CallSiteAnalyzer can be used to obtain information about: * where a method is defined ("defsite") * where a method was called from ("callsite") Having this analyze rcov can provide more accurate coverage information in cost of longer execution.
