You can use Perl to help you complete your task. use strict; use LWP; my $browser = LWP::UserAgent->new; my $responce = $browser->get("http://google.com"); print $responce->content; Your responce object can tell you if it suceeded as well as returning the content of the page.You can also use this same library to post to a page. Here is some documentation. http://metacpan.org/pod/LWP::UserAgent
