The -l flag automatically adds the lib prefix and the .a extension- you want: g++ -LC:/mingw/lib main.o -o test.exe -lopengl32 Note you don't need the quotes around the path either. You could also just specify the whole library name & path: g++ main.o -o test.exe C:/mingw/lib/libopengl32.a As regards your installation problems, use either http://tdragon.net/recentgcc/ or http://nuwen.net/mingw.html - using the MinGW site itself is a recipe for pain.
