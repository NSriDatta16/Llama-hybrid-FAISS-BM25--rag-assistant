Sadly JAXB (bundled with Java6) seems to rely on a non-public class "com.sun.xml.internal.bind.marshaller.NamespacePrefixMapper" to allow you to specify namespace prefixes when marshalling to xml. You have to really go out of your way to get this compiling with ant: http://pragmaticintegration.blogspot.com/ Summary: Option 1. Add jre libs as bootclasspathref Add property: includeJavaRuntime="yes" Option 2. Use JAXB-RI libs - change property to "com.sun.xml.bind.marshaller.NamespacePrefixMapper" Also mentioned here: Define Spring JAXB namespaces without using NamespacePrefixMapper
