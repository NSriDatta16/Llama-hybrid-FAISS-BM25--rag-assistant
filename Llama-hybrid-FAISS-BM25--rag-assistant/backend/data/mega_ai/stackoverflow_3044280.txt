The query part of the URL needs to be before the # . Browsers only send the part before the # to the server. The part after is for auto-scrolling to elements via their id or name attribute, e.g. /index.php?contact_added=1#tabletabs2 See also the " Syntax " section of Wikipedia's "Universal Resource Locator" article. Especially the description of fragment identifiers .
