An applet is basically just a JAR file which you put on your web server and then you add a JNLP description so the browser knows what to do. Think of it as a complex HTML page because it doesn't go into WEB-INF but besides the other files for the browser (HTML, external JavaScript, images, CSS, ...) This article from Oracle describes the steps.
