i'd say it's you. i'd close both resources in a separate try/catch block. i'd create static methods to help me: public static void sourceXmlToBeautifiedXml(File input, File output) throws TransformerException, IOException, JAXBException { FileReader fileReader = new FileReader(input); FileWriter fileWriter = new FileWriter(output); try { // may throw something sourceXmlToBeautifiedXml(fileReader, fileWriter); } finally { close(fileReader); close(fileWriter); } } // same for reader & writer public static void close(InputStream s) { try { if (s != null) { s.close(); } } catch (IOException e) { e.printStackTrace(); } }
