You could use the NHibernate Disjunction class to do this in a more elegant (IMHO) fashion: var disjunction= new Disjunction(); disjunction.Add(Restrictions.On (e => e.Code).IsLike(codePart)); disjunction.Add(Restrictions.On (e => e.Description).IsLike(codePart)); //(and so on) and then: query.Where(disjunction) Each "OR" is a separate instruction, which helps if you want to add the predicates conditionally.
