If you need to get rid of the .. portion, you can use a System.IO.DirectoryInfo object. Use 'fred\frog..\frag' in the constructor. The FullName property will give you the normalized directory name. The only drawback is that it will give you the entire path (e.g. c:\test\fred\frag).
