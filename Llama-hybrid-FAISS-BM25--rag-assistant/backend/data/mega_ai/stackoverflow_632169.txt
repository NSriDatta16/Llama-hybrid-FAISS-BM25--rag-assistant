In some situations, office will create an implicit checkout . This definitely throws users and they have some difficulty correcting this. I really cannot help with the cost/benefit analysis on this one, but I do know that users consitently take a while to get the issue. When dealing with a large organisation, it is definitely worth while limited the checkin feature of libraries to only those libraries that require it.
