For JMX you need to enable JMX-remoting: java -Dcom.sun.management.jmxremote Depending on from where you want to access the jmx-server, you also need to specify a port: -Dcom.sun.management.jmxremote.port=12345 and set or disable passwords. Have a look at http://download.oracle.com/javase/1.5.0/docs/guide/management/agent.html for more details.
