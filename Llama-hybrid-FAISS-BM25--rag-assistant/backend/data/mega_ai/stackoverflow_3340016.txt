This is just a get function to the pointer of a static variable. There is nothing illegal about it. It is not intrinsically any more dangerous than any other type of static data. But static data: creates tight coupling hampers re-entrancy creates the opportunity for subtle bugs is often a sign of weak design should be used very judiciously The static storage class modifier means the memory will be reserved for this variable for the life of the process.
