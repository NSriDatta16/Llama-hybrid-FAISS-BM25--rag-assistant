Mark Pilgrim advises to use httplib2 in " Dive Into Python3 " as it handles many things (including redirects) in a smarter way. >>> import httplib2 >>> h = httplib2.Http() >>> response, content = h.request("http://garagaeband.com/3252243") >>> response["content-location"] "http://garageband.com/michael_jackson4.mp3"
