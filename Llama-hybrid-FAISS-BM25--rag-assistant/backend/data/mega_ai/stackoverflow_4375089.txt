You can prevent the browser cache by calling (before_filter :prevent_browser_cache ) the following code: def prevent_browser_cache headers["Pragma"] = "no-cache" headers["Cache-Control"] = "must-revalidate" headers["Cache-Control"] = "no-cache" headers["Cache-Control"] = "no-store" end
