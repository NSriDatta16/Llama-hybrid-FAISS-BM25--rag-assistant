Simple... Create a file called functions and insert the following: session_start(); $_GLOBALS['users'] = array( //'username' => 'password' 'robert' => 'my_pass' ); function isAuthed() { if(empty($_SESSION['logged_in'])) { if(!empty($_REQUEST['username']) || !empty($_REQUEST['password'])) { if(isset($_GLOBALS['users']) && is_array($_GLOBALS['users'])) { if(isset($_GLOBALS['users'][$_REQUEST['username']]) && $_GLOBALS['users'][$_REQUEST['username']] === $_REQUEST['password']) { $_SESSION['logged_in'] = true; return true; } } } }else { return true; } return false; } and then in your secured pages just do: if(!isAuthed()) { die("You're not authorized to see this page"); } and on your login page just create a form that sends the username, password to the an area of your site that your authorizing Note: This is not copy and past'able code, this is for example purposes only.
