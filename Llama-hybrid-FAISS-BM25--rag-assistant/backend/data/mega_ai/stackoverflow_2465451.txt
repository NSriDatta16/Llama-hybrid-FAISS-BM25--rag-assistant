I think #pragma message needs C strings only. IIRC, _MSC_FULL_VER is a number, while __FILE__ and __DATE__ are C strings. Try this // BEWARE! Untested macro hackery ahead! #define STRINGIFY( L ) #L #define MAKESTRING( M, L ) M(L) #define STRINGIZE(X) MAKESTRING( STRINGIFY, X ) #pragma message( "You have " STRINGIZE(_MSC_FULL_VER) ) (On a side note, this allows #define SHOWORIGIN __FILE__ "(" STRINGIZE(__LINE__) "): " #pragma message( SHOWORIGIN "your message here" ) which allows you to double-click on a message in VS's output pane and be taken to its file/line.)
