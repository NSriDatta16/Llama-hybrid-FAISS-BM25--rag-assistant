Here is an elegant way of doing it using a custom class: First, define a SmartArray class: class SmartArray extends ArrayObject { public static function convert(&$array) { $array = new self($array); } public function offsetGet($index) { return $this->offsetExists($index)? parent::offsetGet($index):NULL; } } Then simply call: SmartArray::convert($_GET); Now your if(isNum($_GET['id'])) { ... } won't complain anymore if $_GET doesn't have the key 'id' :)
