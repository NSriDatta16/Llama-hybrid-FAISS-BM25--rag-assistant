It's mostlikely that you have a circular reference somewhere. A.parent = B; B.parent = C; C.parent = A; You could try to pass a set of previous visited nodes and check whether or not you've visited this node before. The thing with recursion is that you have to be sure that it will end and an unchecked circular reference is a situation where it wouldn't end.
