For those interested, I found out in the meantime that a Bayesian weighted regression is not that straightforward - according to the Bayesian approach, propensity scores would have to be treated as random variables and accordingly possessing a distribution. One possible implementation is discussed here: Uncertainty in Propensity Score Estimation: Bayesian Methods for Variable Selection and Model Averaged Causal Effects
