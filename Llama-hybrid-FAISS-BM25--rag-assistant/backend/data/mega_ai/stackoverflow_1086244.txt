See some examples in Embedding the Dynamic Language Runtime . A simple example, setting-and-fetching-variables : SourceCodeKind st = SourceCodeKind.Statements; string source = "print 'Hello World'"; script = eng.CreateScriptSourceFromString(source, st); scope = eng.CreateScope(); script.Execute(scope); // The namespace holds the variables that the code creates in the process of executing it. int value = 3; scope.SetVariable("name", value); script.Execute(scope); int result = scope.GetVariable ("name");
