There are elegant answers posted already, but since you requested regex, try this: var regex = @"^(? .{4}\s(? .*)$)"; var matches = Regex.Matches (textInput, regex, RegexOptions.IgnoreWhiteSpace | RegexOptions.Singleline); then you parse through matches object to get whatever you want.
