I have tried this VB.NET code on IE and Firefox. Response.Cache.SetAllowResponseInBrowserHistory(False) Response.Cache.SetCacheability(HttpCacheability.NoCache) Response.Cache.SetNoStore() Response.Expires = 0 This does the trick, but I agree with the other answers where you can't guarantee what a client browser will behave.
