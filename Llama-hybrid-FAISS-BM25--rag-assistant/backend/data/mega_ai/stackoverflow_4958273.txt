According to MSDN on Code Coverage Data Overview : Code coverage data is calculated for code blocks, lines of code, and partial lines if they are executed by a test run. A code block is a code path with a single entry point, a single exit point, and a set of instructions that are all run in sequence . A code block ends when it reaches a decision point such as a new conditional statement block, a function call, exception throw, enter, leave, try, catch, or a finally construct.
