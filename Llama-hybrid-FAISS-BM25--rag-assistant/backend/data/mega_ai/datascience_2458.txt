There are really so many good resources now. If you want to stay away from textbooks, both O'Reilly Media and Packt Publishing offer much lighter but effective reading on a lot of great topics. These books are much more applied in practice. As far as learning the languages go, Coursera, Udacity, Code Acadmey, and Code School have great tutorials.
