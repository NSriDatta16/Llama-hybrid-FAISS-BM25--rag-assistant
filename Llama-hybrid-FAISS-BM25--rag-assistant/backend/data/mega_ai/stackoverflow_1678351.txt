It's impossible to give you a complete answer without your html and css. But I do know that a lot of my IE6 issues when implementing jQuery UI's sortable/draggable functionality were solved by making sure the elements had "layout" in IE6. Here's a good article on the layout issue in IE . For my purposes, I ended up adding a conditional css script for IE6 with the following css that is applied to my sortable list: /* Gives layout to elements in IE6, similar to zoom: 1; in IE7 */ #fields, #fields li { height: 0; }
