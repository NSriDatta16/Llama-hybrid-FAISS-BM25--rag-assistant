When you do $('#txtName').val() before submit, it should hold the value entered by the user. $('form').submit(function() { alert($('#txtName').val()); }); If that doesn't work, check if the form element has really the id 'txtName'. If not, this jquery selector works: $("form [name=txtName]") The data can be stored before submit in a cookie/ web storage
