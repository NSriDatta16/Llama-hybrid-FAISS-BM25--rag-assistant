As Michael Burr noted in Carl Smotricz ' answer , for Visual Studio 2008+ you can use __pragma : #define MYMACRO(f,g) \ __pragma(warning(push)) \ __pragma(warning(disable:4127)) \ do { f; g; } while (0) \ __pragma(warning(pop)) You can put it on one line (without the \ s) if you prefer macros to be unreadable.
