On iOS 4.0 and up you can do the following: - (NSUInteger) indexOfObjectWithName: (NSString*) name inArray: (NSArray*) array { return [array indexOfObjectPassingTest: ^BOOL(id dictionary, NSUInteger idx, BOOL *stop) { return [[dictionary objectForKey: @"Name"] isEqualToString: name]; }]; } Elegant, no?
