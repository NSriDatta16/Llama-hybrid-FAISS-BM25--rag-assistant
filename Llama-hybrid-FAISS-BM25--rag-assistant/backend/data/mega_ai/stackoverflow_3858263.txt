This wil work for you You can get rid of the dateby using convert(varchar,Value,108) select name, Reference1, CONVERT(varchar, CAST(AVG(CAST(CompletedTime AS float) - CAST(CompletedTime AS int)) AS datetime),108) averageTime From tblOrderRouteStops where Name not like 'Cta%' and Reference1 <> '000000' and Name <> '' and CompletedTime is not Null Group By name, Reference1
