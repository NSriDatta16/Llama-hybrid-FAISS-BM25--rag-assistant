To display your two buffers vertically, you can use : :vsplit # or the short version :vs # To reorganize your layout from horizontal to vertical, you must use: K : will move the current windows to to the top with a horizontal split J : will move the current windows to to the bottom with a horizontal split H : will move the current window to the the left with a vertical split L : will move the current window to the the rigth with a vertical split See :help windows.txt for more information
