Without an explanation of why you're trying to do this, I'm going to guess that it's for better code organization. If that's the case, I would split your JavaScript up into multiple files something along the lines of this, and to be included on your page in this order: // namespace.js if (typeof Cart === 'undefined') { Cart = {}; } // checkoutnow.js Cart.CheckoutNow = { // module code here } // alias.js alias = function() { // alias code here } // domready.js onDocumentReadyFunction() { alias(); }
