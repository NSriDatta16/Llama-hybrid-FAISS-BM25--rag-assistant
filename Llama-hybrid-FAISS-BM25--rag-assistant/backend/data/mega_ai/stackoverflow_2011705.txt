A 3-tier architecture usually has the following components: Client Browser Web server hosting the ASP.NET application Some backend storage such as database that is being accessed by the ASP.NET application So to answer your question on how to write code for a 3-tier architecture, you develop an ASP.NET application that communicates with a data storage.
