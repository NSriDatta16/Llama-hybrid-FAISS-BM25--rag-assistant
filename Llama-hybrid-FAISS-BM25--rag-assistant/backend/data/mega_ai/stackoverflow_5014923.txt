I would replace your .htaccess file with a known good copy. Error 500's can be caused by the wrong permissions as well. I doubt this is the case with you since there is no actual admin folder it is done with the url rewrites. None the less here is a post I made for people who are getting error 500: link Also here is my URL rewrites from my magento site incase you do not have a backup copy: ############################################ ## workaround for HTTP authorization ## in CGI environment RewriteRule .* - [E=HTTP_AUTHORIZATION:%{HTTP:Authorization}] ############################################ ## always send 404 on missing files in these folders RewriteCond %{REQUEST_URI} !^/(media|skin|js)/ ############################################ ## never rewrite for existing files, directories and links RewriteCond %{REQUEST_FILENAME} !-f RewriteCond %{REQUEST_FILENAME} !-d RewriteCond %{REQUEST_FILENAME} !-l ############################################ ## rewrite everything else to index.php RewriteRule .* index.php [L] ############################################
