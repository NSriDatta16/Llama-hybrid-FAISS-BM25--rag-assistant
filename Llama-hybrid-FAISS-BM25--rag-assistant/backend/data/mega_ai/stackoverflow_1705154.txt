The way I have done it is as follows (not all my own code but I can't find a referance to the original author). interleave Skippable = Whitespace | Comment; interleave Comment = CommentToken; @{Classification["Comment"]} token CommentToken = CommentDelimited | CommentLine; token CommentDelimited = "/*" CommentDelimitedContent* "*/"; token CommentDelimitedContent = ^('*') | '*' ^('/'); token CommentLine = "//" CommentLineContent*; token CommentLineContent = ^( '\u000A' // New Line | '\u000D' // Carriage Return | '\u0085' // Next Line | '\u2028' // Line Separator | '\u2029' // Paragraph Separator ); This allows for both single line ( // ) comments as well as multiline ( /* */ ) comments.
