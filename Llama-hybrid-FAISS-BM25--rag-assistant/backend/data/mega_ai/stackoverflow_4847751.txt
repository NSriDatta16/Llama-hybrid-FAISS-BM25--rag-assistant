In the first case ( MethodA(this) ), a copy of the reference is passed by value to the method. This is blazingly fast. I guarantee it's not a bottleneck in your application. In the second case ( MethodB(ref y) ), the storage location of the reference is passed to the method. This is blazingly fast. I guarantee it's not a bottleneck in your application.
