Try this... First, override the Equals() (and GetHashCode() ) methods in your entity objects. The two entity objects should be equal if they are the same type and their IDs are equal. Next, use this operation public IQueryable GetAgentListByUserProfile(UserProfile userProfile) { var agentObjects = this.DataContext.AgentSet; var results = (from p in agentObjects where p.UserProfile == userProfile select p); return results; }
