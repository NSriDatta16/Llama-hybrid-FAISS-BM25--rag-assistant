SELECT person_id, AVG(mpg) AS mpg_avg FROM car GROUP BY person_id WITH ROLLUP The WITH ROLLUP -modifier will add a line to the result set where persion_id is NULL and mpg_avg is the average over the whole result set (MySQL >= 4.1.1): person_id | mpg ------------------ 1 | 32.5 2 | 19.0 3 | 33.0 NULL | 27.2
