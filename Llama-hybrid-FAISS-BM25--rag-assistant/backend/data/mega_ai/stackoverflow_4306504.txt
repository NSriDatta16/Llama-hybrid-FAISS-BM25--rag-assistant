Adobe has an object embedding reference that lists some of the possible parameters . Notably missing is the allowFullScreen parameter. The following assumes " Satay Method ": What parameters can be used to embed a flash video? Parameter | Type | How ======================================================= align | tag | stage.align α allowFullScreen | tag | --- base | tag | --- bgcolor | tag | --- classid | attribute | --- codebase | attribute | --- data | attribute | --- flashvars | tag | stage.loaderInfo.parameters or documentObject.loaderInfo.parameters fullScreenAspectRatio | tag | --- height | attribute | stage.stageHeight † id | attribute | ExternalInterface.objectID ‡ loop | tag | not directly accessible ◊ menu | tag | stage.showDefaultContextMenu movie | tag | --- name | attribute | ExternalInterface.objectID ‡ play | tag | not directly accessible ◊ quality | tag | stage.quality salign | tag | stage.align α scale | tag | stage.scaleMode seamlesstabbing | tag | --- swliveconnect | tag | --- width | attribute | stage.stageWidth † wmode | tag | --- † only if scale is set to no_scale ‡ for cross-browser compatibility both the id and name attributes must be present, and set to the same value. ◊ see this question on how to indirectly access the value. α NOTE: not sure whether align , salign or both are available via stage.align
