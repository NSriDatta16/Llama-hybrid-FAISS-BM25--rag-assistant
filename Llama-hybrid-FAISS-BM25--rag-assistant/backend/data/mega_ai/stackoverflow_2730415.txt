Have a look at this tutorial , it has all you need (with examples). If you want to transform XML that comes in in String format, use something like: Templates template = ...; String xml = ...; Transformer transformer = template.newTransformer(); Writer out = new StringWriter(); transformer.transform(new StreamSource(new StringReader(xml)), new StreamResult(out));
