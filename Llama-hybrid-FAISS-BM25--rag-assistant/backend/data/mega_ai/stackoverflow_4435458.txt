If you're using the Facebook Graph API, there is some documentation for how to do that dialog here . Basically, you can create an iframe with its source set to this URL: https://graph.facebook.com/oauth/authorize?client_id=...&redirect_uri=...&type=user_agent&display=popup Set client_id to your application ID and redirect_uri to the URL of a server-side handler that can retrieve a user's auth token.
