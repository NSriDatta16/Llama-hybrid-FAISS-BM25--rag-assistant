Date is a bit hard to work with, you should use Time. Try converting the Dates into Times: require 'time' foo_time = Time.parse(foo.to_s) bar_time = Time.parse(bar.to_s) Convert them to timestamps, then calculate the average, then convert back to Time: avg = Time.at((foo_time.to_f + bar_time.to_f) / 2) You can convert that back to Date: avg_date = Date.parse(avg.to_s)
