Your approach is probably the simplest way of achieving your goal, however it's slightly more elegant in Selenium 2 (WebDriver). An example in Java is below: List links = driver.findElements(By.xpath("//a")); for (WebElement link : links) { System.out.println(link.getText()); } This would output the link text for every link on the page to the console.
