You can check the Masapi framework to see if it fulfills your needs. You can also investigate the source code to learn how they approached the problem. http://code.google.com/p/masapi/ It's well written and maintained. I used it successfully in a desktop RSS client I developed with Air. It worked very well assuming you pay attention to the overhead while loading too many resources in parallel.
