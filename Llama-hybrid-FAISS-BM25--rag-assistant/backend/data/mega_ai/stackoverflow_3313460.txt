.add(html) just adds the element to the jQuery object (it's a document fragment at this point), it doesn't insert it into the DOM :) .append() is for a different purpose, it actually adds the content to the matched elements , as their children, whereas .add() it's a sibling, and only inside that jQuery object, not in the DOM yet.
