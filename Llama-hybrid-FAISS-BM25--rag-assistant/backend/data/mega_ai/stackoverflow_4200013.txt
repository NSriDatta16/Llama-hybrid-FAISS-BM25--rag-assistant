"Partial" answer for changing the display order in runtime: http://plugins.jquery.com/project/reverseorder This is a jQuery plugin. You can reverse the order of the elements in your example with $('#extMenu li').reverseOrder(); Not a really elegant way, but it works (if JavaScript is enabled of course...)
