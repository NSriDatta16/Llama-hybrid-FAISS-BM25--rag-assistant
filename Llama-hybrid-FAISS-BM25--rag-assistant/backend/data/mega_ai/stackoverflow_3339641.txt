The pound sign ( # ) refers to an ID which needs to be unique for the page. The period ( . ) refers to a class which can be used many times. People would use p#mytext if they wanted a unique styling for one (just one) paragraph tag. You can read up about it here . Wanted to add that some web developers seem to gravitate towards declaring everything as classes. If you use a layout generator of any kind more often than not every element will be a class.
