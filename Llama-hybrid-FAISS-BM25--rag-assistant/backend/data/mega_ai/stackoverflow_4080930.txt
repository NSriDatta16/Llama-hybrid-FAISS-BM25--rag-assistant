The HTML5 specification introduced the concept of Global Attributes. These are attributes that may be specified on all HTML elements. The list should be the following: accesskey class contenteditable contextmenu dir draggable hidden id lang spellcheck style tabindex title Source and further reading: W3.ORG: HTML5 - 3.2.3 Global attributes
