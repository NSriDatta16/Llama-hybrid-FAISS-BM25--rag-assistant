I"m not sure you need core text, you can use CGSize cs = [theString sizeWithFont:[UIFont systemFontOfSize:fontSize] constrainedToSize:CGSizeMake(width,FLT_MAX) lineBreakMode:UILineBreakModeWordWrap]; to get the height of the paragraph. Aligning it with that info should be pretty easy.
