Check the registry keys [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Internet Settings\User Agent\Post Platform] and [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\ Internet Settings\5.0\User Agent\Post Platform] Some pieces of software will add additional values here, which is fine, unless you specify a user agent string. In that case, most browser detects will fire off and detect the last value they find. Typically, these values will either be in a "User Agent" key or "Post Platform" key.
