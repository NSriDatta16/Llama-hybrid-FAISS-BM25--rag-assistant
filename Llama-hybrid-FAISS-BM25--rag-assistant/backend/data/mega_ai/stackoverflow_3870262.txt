googlebot will not even be able to touch the sitemapindex.xml the robots.txt is a crawler directive. the sitemap.xml is fetched via the googlebot crawler. googlebot will not access the sitemapindex.xml no crawl coverage, no indexing, no SERP listing you can test this with google webmaster tools robots.txt verification tool and fetch as googlebot (in the labs section) feature.
