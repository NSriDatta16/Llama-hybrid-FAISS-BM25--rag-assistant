You could probably work eval() into a simple solution. For instance: var myNum = 10; function EQ(eq) { return eval(myNum+eq); } alert( EQ("*100/10") ); // outputs 100 I'd encourage you to expand upon this by implementing a try-catch and handling exceptions.
