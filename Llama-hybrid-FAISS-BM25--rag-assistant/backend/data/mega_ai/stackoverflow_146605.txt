It uses a resource fork-based binary format. Valid workarounds: Have the user drag a URL from your application (NSURLPboardType) to the Finder. The Finder will create a webloc for you. Create a Windows Web Shortcut (.URL file). These have a INI-like data fork-based format and should be documented somewhere on the Internet; the OS supports them as it supports weblocs.
