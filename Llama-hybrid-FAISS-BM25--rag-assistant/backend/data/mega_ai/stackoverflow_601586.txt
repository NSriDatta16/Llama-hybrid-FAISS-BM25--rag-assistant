Here's another, more elegant solution that works around escaping spaces in the filenames git diff --name-only 0abc 1def | \ tr '\n' '\0' | \ xargs -0 tar -rjvf ~/deploy.tar.bz2 As jpalecek pointed out, one wold not want to run "tar -c..." multiple times, it's better to use -r.
