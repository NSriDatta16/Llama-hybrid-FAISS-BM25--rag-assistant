The specification for GL_pixel_buffer_object gives an example demonstrating how to render to a vertex array under "Usage Examples". The following extensions are helpful for solving this problem: GL_texture_float - floating point internal formats to use for the color buffer attachment GL_color_buffer_float - disable automatic clamping for fragment colors and glReadPixels GL_pixel_buffer_object - operations for transferring pixel data to buffer objects
