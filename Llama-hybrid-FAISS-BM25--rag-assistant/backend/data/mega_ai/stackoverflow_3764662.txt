Eval() should allow you to do something similar. This expression returns -1 (True): Debug.Print Eval("""g"" Not In (""a"",""b"",""c"")") I wouldn't call that elegant, though. Consider using the Like operator. This expression returns True: Debug.Print Not "g" Like "[abc]"
