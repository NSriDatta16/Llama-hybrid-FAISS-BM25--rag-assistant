First, you need to parse the entire document, not just a section of the document. Trying to pull a section out of an XML document without parsing (at least) the stuff preceding it is liable to be fragile ... especially if there are changes in the way that the XML is created. Second, you need to read up on the technologies for parsing XML and decide which one best suits your use-case. Here are some resources: The Sun XML Tutorial stream. Wikipedia material on XML processing and JAXP . There are various other approaches, but I don't want to confuse you with a million choices.
