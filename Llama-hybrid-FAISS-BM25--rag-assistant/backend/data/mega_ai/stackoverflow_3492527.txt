i needed the same, solution was to use cycle and swipe plugins: "#gallery" is a container with all img tags in it. $(function() { $('#gallery').cycle({ fx: 'scrollHorz', timeout: 0, next: '', prev: '', speed: 300, nowrap: 0 }); }); $(function() { $('#gallery').swipe({ swipeLeft: function() { $('#gallery').cycle("next"); }, swipeRight: function() { $('#gallery').cycle("prev") }, threshold: { x: 15, y: 99999 }, preventDefaultEvents: false }); }); it works, but not exactly like Photo library, because images doesnt drag to 50% of width before changing. just like in crmunro's solution, but based on jQuery and plugins.
