You should be able to create a new instance of the UltraGridRow class and pass it to the UltraWebGrid 's InsertDBRow method. Here's an example of inserting a row using InsertDBRow. // Create new UltraGridRow (using the object[] constructor) var newRow = new UltraGridRow( new[] { "My First Value" , "My Second Value" } ); UltraWebGrid1.InsertDBRow( newRow );
