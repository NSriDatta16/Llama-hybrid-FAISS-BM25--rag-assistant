If you have it in an arraylist, you could do something like: HashMap result = new HashMap (); // for(String line: arraylist) { String[] items = line.split(" "); String agent = items[3]; double amount = Double.valueOf(items[2]); if(!result.containsKey(agent) result.put(agent, amount); else result.put(agent, result.get(agent) + amount); } You can iterate over the hash map's keys to get the results.
