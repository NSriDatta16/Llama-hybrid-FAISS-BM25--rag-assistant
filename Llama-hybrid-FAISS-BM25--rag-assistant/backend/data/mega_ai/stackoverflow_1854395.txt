I think this is what you're after: SELECT AVG(items.price) FROM (SELECT t.price FROM TABLE t WHERE t.price > '0' AND t.item_id = '$id' ORDER BY t.price LIMIT 5) items It will return the average of the 5 lowest prices - a single answer.
