You should be able to do this with Flash, I don't know for sure about HTML5. In windows I should suggest you create a directshow graph to do this. You'll want a screen capture source filter, h264 encoder and a flash streaming output filter. A directshow screen capture filter. http://www.mainmediasoft.com/products/screencapture/ ffdshow can encode h264 http://ffdshow-tryout.sourceforge.net/ flash streaming output filter. http://www.nanocosmos.de/v3/en/rtmp-streaming.html In theory you should be able to drag all these parts together in graphedit and make this work.
