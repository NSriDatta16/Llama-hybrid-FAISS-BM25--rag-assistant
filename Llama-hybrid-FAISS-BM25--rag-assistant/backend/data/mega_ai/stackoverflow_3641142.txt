Inside the drop handler, $(this) is the droppable, ui.draggable is the draggable (as a jQuery object). So for the droppable: $(this).attr('id'); // or this.id; And for the draggable: ui.draggable.attr('id'); // or, for the sake of symmetry: ui.draggable[0].id;
