To remove by object, such as removing a specific triangle from a triangle list, use List::remove(Object) For the above purposes, you might want to use List::listIterator() , though. Something like: ListIterator it = triangles.listIterator(); while (it.hasNext()) { if (it.next().contains(startDrag)) { it.remove(); break; } }
