The trick here is to ALSO accept the event in the QGraphicsScene::dragMoveEvent()! The reason is the DEFAULT implementation which ignores drag and drop events if there is no item under the mouse! Also refer to: http://www.qtcentre.org/threads/8022-QGraphicsScene-doesn-t-accept-Drops Cheers
