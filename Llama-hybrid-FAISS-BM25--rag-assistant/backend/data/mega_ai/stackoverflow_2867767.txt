That sounds about right, I've done something similar. You'll probably want something like: class YourWrapperClass extends SoapClient { private $session; private function mMakeSureSessionIsStarted() { if(!$this->session) { // code to start session here... $this->session = parent::startSession(); } } public function __soapCall($method,array $arguments,array $options,$input_headers=null, array &$output_headers=null) { $this->mMakeSureSessionIsStarted(); $arguments['session'] = $this->session; return parent::__soapCall($method,$arguments,$options,$input_headers,$output_headers); } } (Not 100% positive that's correct syntax, but I think it's close...)
