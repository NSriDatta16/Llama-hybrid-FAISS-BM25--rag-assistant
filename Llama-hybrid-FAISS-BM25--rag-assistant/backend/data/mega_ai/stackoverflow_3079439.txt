Your xpath expression should already return exactly what you want. Quoting from http://www.w3.org/TR/xpath/#booleans , 5th paragraph: If one object to be compared is a node-set and the other is a string, then the comparison will be true if and only if there is a node in the node-set such that the result of performing the comparison on the string-value of the node and the other string is true
