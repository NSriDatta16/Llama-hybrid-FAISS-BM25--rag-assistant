You can also use jQuery's data function to store groups for easy retrieval. var group1 = { name: "Group 1", users: { uid1: { name: "User 1" }, uid2: { name: "User 2" }, uid3: { name: "User 3" } } } $('#ArbitraryElementID').data('Group 1', group1); then you can use: var group1 = $('#ArbitraryElementID').data('Group 1'); to retrieve the object.
