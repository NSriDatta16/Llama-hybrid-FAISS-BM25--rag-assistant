System.Data.SQLite is an ADO.NET provider for the SQLite embedded database. Its old homepage was http://sqlite.phxsoftware.com/ and currently this is where the bulk of knowledge/discussion is found, but it has recently been forked by some new folks (with blessings from the original author, Robert Simpson) and now lives at: http://system.data.sqlite.org/ Two features that make System.Data.SQLite stand out are that it is free, and that it works with things designed around ADO.NET, such as Entity Framework. http://sqlite.org/
