You need to escape the apostrophe in the input string (basically replace a single ' with two '') as you build your SQL string You need to pay attention to this anywhere you choose to pass user input to a SQL server database as its a security issue (SQL Injection Attacks) c.f. Bobby Tables if Rtrim(@LastName) <> '' select @Where = @Where + ' and a.LastName like '+ Replace(Rtrim(@LastName),'''','''''') + ''
