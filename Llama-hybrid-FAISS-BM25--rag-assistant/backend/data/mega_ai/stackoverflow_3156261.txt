Have a look at IsolatedStorageSettings . You have ApplicationSettings (specific to a single application on a site) and SiteSettings (for all applications at a particular domain). Note that in both cases, settings are specific to a user. A quick example: private void StoreSetting(string key, string value) { IsolatedStorageSettings.ApplicationSettings[key] = value; } private string GetSetting(string key) { return IsolatedStorageSettings.ApplicationSettings[key]; }
