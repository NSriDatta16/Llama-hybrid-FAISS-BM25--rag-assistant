You cannot do that if you want to continue to use the XML Schema simple types. decimal and the types derived from it are locked down to using a period. As you say: the spec here . If you want to use a comma as a separator, you need to define your own simple type, for example: Before you do that, though, be careful; XML is a data storage format, not a presentation format.
