Not elegant, but this will work: find -name "list.xml" | xargs cat | tr -d "\n" | sed 's/ /\n/g' | sed 's/ /: /g' | sed 's/ ]*>//g' | egrep "Name:|ID:" | sed 's/Name: /---\nName: /g' Basically it does this: remove all newlines put each key value pair on its own line add : separator remove all element content (between ) only save Name and ID fields (drop all others) add --- separator Sample Output: --- Name: Greg ID: 52 --- Name: Amy ID: 53 --- Name: Mr Jones ID: 51
