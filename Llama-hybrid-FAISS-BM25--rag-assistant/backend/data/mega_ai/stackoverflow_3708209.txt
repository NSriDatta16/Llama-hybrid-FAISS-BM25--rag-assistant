It happens bacause your callback function leave in global context. You can choose 2 ways Use .bind function to bind context to callback function http://www.robertsosinski.com/2009/04/28/binding-scope-in-javascript/ jQuery's AJAX supports transfer some data to callback function. You can write smth like this: : this.tictac = function () { $.ajax ({ type: 'POST', context:this, url: '/rap/rapClock.php', complete: function (data) { this.setClockTime(data); } }); } }
