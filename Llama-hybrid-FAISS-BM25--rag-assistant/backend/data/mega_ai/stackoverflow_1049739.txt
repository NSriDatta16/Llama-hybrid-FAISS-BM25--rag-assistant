You always need to expect nulls, because of your outer join: select records.id, contacts.name + CASE WHEN contacts.organization IS NULL OR contacts.organization='' THEN '' ELSE ' (' + contacts.organization + ')' END as contact, from records left join contacts on records.contact = contacts.contactid
