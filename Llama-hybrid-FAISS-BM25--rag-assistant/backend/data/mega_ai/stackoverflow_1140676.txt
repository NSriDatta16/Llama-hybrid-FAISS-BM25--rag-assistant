This isn't the fastest code in the world, but it might work for a while: Type type = request.Headers.GetType(); MethodInfo method = type.GetMethod("AddWithoutValidate", BindingFlags.Instance | BindingFlags.NonPublic); method.Invoke(request.Headers, new [] { "Date", DateTime.Now.ToString() }) Just be aware that using reflection to access private or protected members is extremely fragile and can break with any update to .NET that MS does...
