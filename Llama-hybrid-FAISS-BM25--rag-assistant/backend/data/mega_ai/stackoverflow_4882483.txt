You can use mouseDragged() method to do this. Take two global variable X and Y . now in paint method of JPanel draw you image like this: public void paintComponent(Graphics g) { .... g.drawImage(image,X,Y,this); .... } and in your mouseDragged method do this: public void mouseDragged(MouseEvent e) { X = e.getX(); Y = e.getY(); repaint(); } Hope this helps.
