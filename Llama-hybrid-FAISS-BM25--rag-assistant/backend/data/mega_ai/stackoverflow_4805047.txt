I used a Form / UserControl with a Panel. Then I used the Panel as the new parent and set the new position and size of the panel with SetWindowPos to the child window. NOTE: MS recommends to use SetWindowPos after using SetWindowLong (or SetWindowLongPtr) to activate the new styles.
