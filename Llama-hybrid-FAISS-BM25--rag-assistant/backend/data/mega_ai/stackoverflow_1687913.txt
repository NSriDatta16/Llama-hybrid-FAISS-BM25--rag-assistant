The javascript alternatives are far easier to work with that sIFR. I use cufon, but there are a number of alternatives. Another option is to use the @font-face css property. This is more difficult to initially set up but has the huge advantage of using CSS instead of javascript. Here's a good article on how to achieve this http://snook.ca/archives/html_and_css/becoming-a-font-embedding-master
