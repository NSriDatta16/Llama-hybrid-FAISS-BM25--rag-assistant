JavaScript libraries do not require installation like other libraries. You just download, unzip if the download is zipped, and include the JavaScript files in the tag. With Mootools, it's even easier because you can leverage the Google-hosted Mootools files: or If you need to know the difference between the two, that should probably be another question by itself. Here's the reference for Mootools hosted on Google: http://code.google.com/apis/ajaxlibs/documentation/index.html#mootools
