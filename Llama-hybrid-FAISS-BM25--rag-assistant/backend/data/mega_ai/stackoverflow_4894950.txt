As a start: // Given a, return a 'raw a', being two integers below a. // Those two access by the following calls. #define stb__sbraw(a) ((int *) (a) - 2) // The current number opf elements in the array; same as count #define stb__sbm(a) stb__sbraw(a)[0] // The number of elements available to the current storage #define stb__sbn(a) stb__sbraw(a)[1] So: // Free the raw storage #define sbfree(a) ((a) ? free(stb__sbraw(a)),0 : 0) stb__sbgrowf does all the allocation; if initially passed a pointer to null it will create the two integer header automatically.
