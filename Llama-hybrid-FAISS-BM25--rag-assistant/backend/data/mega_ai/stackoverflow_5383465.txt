You can do this with events, there's some basic information on how to create events in the magento wiki: http://www.magentocommerce.com/wiki/5_-_modules_and_development/0_-_module_development_in_magento/customizing_magento_using_event-observer_method The event you need is probably catalog_product_save_before or catalog_product_save_after, depending on how you setup the code.
