Suppose you have an image with id dragme in DragHandleTemplate ; you can do this in the ItemDataBound handler: Image dragMe = (Image)((TableRow)e.Item.Controls[0].Controls[0]).Cells[0].Controls[0].FindControl("dragMe"); dragMe.Style.Add(HtmlTextWriterStyle.Visibility, "hidden"); This way you preserve the alignment.
