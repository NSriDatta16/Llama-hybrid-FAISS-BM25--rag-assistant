According to the documentation, the HttpWebRequest.Date property is DateTime.MinValue by default which will not send the Date header with the request. So to set it all you should have to do is... HttpWebRequest webRequest; webRequest = (HttpWebRequest)WebRequest.Create("http://commondatastorage.googleapis.com"); webRequest.Date = DateTime.UtcNow;
