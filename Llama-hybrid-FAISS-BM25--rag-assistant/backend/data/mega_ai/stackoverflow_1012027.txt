Martin Fowler has a nice article dealing with things that change with time . It might give you some inspiration. From the first sight storage seems pretty obvious: a table with ID, GTIN (or whatever else you're using to track products), date and price. Hence extracting data for charting is trivial: select range of prices for one or more GTINs and plot them onto graph.
