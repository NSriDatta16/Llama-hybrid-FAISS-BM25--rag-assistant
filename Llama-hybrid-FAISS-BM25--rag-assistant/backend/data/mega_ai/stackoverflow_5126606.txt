Modularity with RequireJS : module1.js define( ["dependency"] , function( dep ){ return { speak: function(){ alert( dep.message ); } } } ); dependency.js define( { message: "Hello world!" } ); impl.js if ( $(".someCssExpression").length ) { require( [ "module1" ] , function( mod ){ mod.speak(); }); } index.html ... ... Your file structure will be modular. Your implementation will be modular. And no clunky namespacing or weird constructs to make it feel organised. Takes some getting used to, but totally worth it.
