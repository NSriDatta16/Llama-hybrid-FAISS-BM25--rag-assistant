It appears that the best way would be to use generics to store the correct type of delegate in the record. No hacks required. program Project51; {$APPTYPE CONSOLE} uses SysUtils; type TMyProc = reference to procedure(var param: integer); TStorage = record FDelegate: T; end; var a : TMyProc; b : TMyProc; p : TProc; param: integer; stg : TStorage ; begin b := procedure (var param: integer) begin param := 2*param; end; stg.FDelegate := b; param := 21; stg.FDelegate(param); Writeln(param); Readln; end.
