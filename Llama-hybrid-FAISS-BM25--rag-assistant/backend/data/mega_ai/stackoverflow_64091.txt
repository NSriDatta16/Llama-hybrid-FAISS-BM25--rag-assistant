I don't know how to do it in ASP.NET but in PHP I would do something like: header("Expires: Mon, 26 Jul 1997 05:00:00 GMT"); header("Cache-Control: no-cache"); header("Pragma: no-cache"); Which forces the browser to recheck that the item, so your authentication checking should be triggered, denying the user access.
