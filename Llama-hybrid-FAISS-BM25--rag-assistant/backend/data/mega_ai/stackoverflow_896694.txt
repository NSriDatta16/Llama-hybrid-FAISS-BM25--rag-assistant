"Can we open server-side files?" - Yes, but requires a web service (edited) "Can we open client-side files?" - Yes, only via an Open File window or files in isolated sotrage "Can we get change notification on files (client or server side)?" - Yes, but only on server side. "Can we do the normal open/seek/write/append operations?" With System.IO.FileStream class, which is available in Silverlight, you can manipulate textual streams in memory. However, you need to fetch the stream by web services, opening it with file open window, or from isolated storage.
