You can try Compelson SyncML server from Compelson labs . It has an SDK for custom Storage. By default it ships with SqlServer data store for sync clients. The SDK also has a sample project which uses File System as data store for sync clients. Chk this link: http://www.mobiledit.com/downloads.asp?show=syncml_developers
