Use GROUP BY : select a, b, c, max(n) from table group by a, b, c; This will show only unique or distinct sets of a, b, c and show the maximum n found in that set. MAX is an aggregate function designed for use with GROUP BY . Other potentially useful aggregate functions include MIN , AVERAGE , and COUNT .
