It works similar to internal and external Javascript if you allow me that comparison. The SWC option is like embedding Javascript in the host HTML file (assets from the SWC are statically linked (in)to your main SWF). The RSL option is like having an external JS file which is loaded via from an HTML file (assets from the RSL are dynamically linked to your main SWF.
