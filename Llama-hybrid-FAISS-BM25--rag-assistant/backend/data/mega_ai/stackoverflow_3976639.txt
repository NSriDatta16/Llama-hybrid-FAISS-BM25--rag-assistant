You probably want to use Readonly for creating "true" constants. Constants created using the constant pragma are actually inlinable subroutines . It means that at compile time the appropriate scalar constant is inserted directly in place of some subroutine call. If the constant is a reference, nothing prevents you from changing the data it points to.
