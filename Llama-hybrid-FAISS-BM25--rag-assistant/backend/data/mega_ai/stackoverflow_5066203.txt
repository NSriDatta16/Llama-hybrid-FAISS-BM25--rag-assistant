No, this quote refers to the inclusion of ZODB "blob support" (http://en.wikipedia.org/wiki/Binary_large_object) in Plone 4. Prior to this release, objects like files and images were stored in the (flat file) Data.fs file (which is part of the ZODB). Now, they are stored on the filesystem in files (still managed by the ZODB) that look like this: var/blobstorage var/blobstorage/.layout var/blobstorage/0x00 var/blobstorage/0x00/0x00 var/blobstorage/0x00/0x00/0x00 var/blobstorage/0x00/0x00/0x00/0x00 var/blobstorage/0x00/0x00/0x00/0x00/0x00 var/blobstorage/0x00/0x00/0x00/0x00/0x00/0x00 var/blobstorage/0x00/0x00/0x00/0x00/0x00/0x00/0x3b var/blobstorage/0x00/0x00/0x00/0x00/0x00/0x00/0x3b/0xa5 var/blobstorage/0x00/0x00/0x00/0x00/0x00/0x00/0x3b/0xa5/0x038ba9d72acbdcdd.blob var/blobstorage/0x00/0x00/0x00/0x00/0x00/0x00/0x3b/0xa9 var/blobstorage/0x00/0x00/0x00/0x00/0x00/0x00/0x3b/0xa9/0x038ba9d836b5cdaa.blob
