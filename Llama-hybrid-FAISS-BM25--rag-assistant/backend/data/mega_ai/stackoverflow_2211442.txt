It’s possible to perform cross-domain requests using jQuery . Simply include this file after loading jQuery, and then you can do funky stuff like: $('html').load('http://helixagent.com/todo/'); Note that this of course won’t load the embedded CSS and JS files. You’ll need to use an iframe for that: $(function() { $('body').append(' '); }); Live example: http://jsbin.com/uruka
