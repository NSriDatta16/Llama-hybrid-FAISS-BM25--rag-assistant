That is the simplest way to do this, the only way to get the tabs of the window is by doing getAllInWindow from the tab API. Once you do that, you can organize it in any way you want within the popup . To get page previews, you can use the screenshot API everytime, or use a third party to generate previews, and you can cache them locally, you can view the source for the speed dial extension to see how it is done.
