Here's a SO question/answer on Calculating Standard Deviation: Standard Deviation in LINQ For the list of items just do items = grp.ToList() or grp.Select(c => c.Field...).ToList() var results = from res in dt.AsEnumerable() group res by res.Field (key) into grp orderby Convert.ToInt32(grp.Key) select new { date = grp.Key, sum = grp.Average(r => Convert.ToDouble(r.Field (average))), items = grp.ToList(), sd = CalcSD(grp.Select(c => Convert.ToDouble(r.Field (average)))) };
