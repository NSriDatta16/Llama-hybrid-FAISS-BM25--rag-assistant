The Value of in your example is "One StringAnother string" . You need to select the Value from each individually: var query = from paymenttype in offer.Element(myns + "paymentTypes") .Elements(myns + "paymentType") select paymenttype.Value; List PaymentType = query.ToList(); Or, if there are multiple , the Value from each in each : var query = from paymenttypes in offer.Elements(myns + "paymentTypes") from paymenttype in paymenttypes.Elements(myns + "paymentType") select paymenttype.Value; List PaymentType = query.ToList();
