While it's perhaps not the most elegant solution, you can always call external scripts using the Process support in SBT. import sbt._ import Process._ class Project(info: ProjectInfo) extends DefaultProject(info) { lazy val runSomething = task { "ruby somescript.rb" ! log None } } There's a bit more info about the external process support available here: http://code.google.com/p/simple-build-tool/wiki/Process
