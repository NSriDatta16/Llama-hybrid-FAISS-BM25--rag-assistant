If your already using Mootools, a good way to do this is using the element's data storage . window.addEvent('domready', function() { $$('a.classname').each(function(el) { el.store('productID', /*get this however you want*/); el.addEvent('click', function(e) { var productID = el.retrieve('productID'); } } } And here's one method for getting the productID's (assuming you have control over URL formatting): link //in your js (above) var pid = el.get('href').split('#')[1].split(':')[1]; el.store('productID', pid);
