The duplicate private structure is a bit odd in your example. The below compiles fine, for instance: class TestFather { protected: struct myStruct { // struct definition }; vector myStructVect; public: virtual vector get_myStructVect() = 0; }; class TestSon : public TestFather{ public: vector get_myStructVect(); }; int main(int argc, char**argv) { TestSon testSon; } Note the replacement of private with protected, allowing derived classes access to the parent structure definition and myStructVect storage.
