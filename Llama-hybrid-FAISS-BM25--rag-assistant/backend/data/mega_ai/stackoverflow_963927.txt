if a == b -> expr1; true -> exprElse end There is no real else in Erlang, you just match with true. More information at the Erlang documentation . You have to pay close attention to the ';' terminator : if a == b -> clauseN, expr1; cond2 -> clause1, clause2, expr2; cond3 -> expr3; true -> exprElse end It's not a copy-paste friendly language.
