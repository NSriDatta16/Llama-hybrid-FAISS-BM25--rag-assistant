See here: Stopwords and Stoplists . The syntax is: CREATE FULLTEXT STOPLIST MyList [FROM SYSTEM STOPLIST] ALTER FULLTEXT STOPLIST MyList ADD 'above' LANGUAGE 'English' ALTER FULLTEXT STOPLIST MyList ADD 'accordingly' LANGUAGE 'English' And so on. You can also manage all of this through SSMS - it's in [Your database] > Storage > Full Text Stoplists .
