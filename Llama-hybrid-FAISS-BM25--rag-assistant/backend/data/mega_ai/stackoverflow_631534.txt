You can use SQLMetal from the command line instead of using the IDE. Or you can even use this UI for running SQLMetal. You could also use Visual Studio Express . The feature set is a bit limited in comparison to the other VS editions, but it still is a very good product and includes support for Linq2SQL. In this case, getting Linq2SQL up and running boils down to creating a new dbml file and dragging some database tables into it.
