From JavaMail 3rd Party Products. jASEN is a pure Java Anti Spam ENgine combining bayesian-like scanning with intelligent email inspection and classification. jASEN is best suited to developers wishing to integrate anti-spam services into an existing server based Java email application, but can be used for client applications. Available at: http://www.jasen.org More JavaMail 3rd Party Products here: http://java.sun.com/products/javamail/Third_Party.html
