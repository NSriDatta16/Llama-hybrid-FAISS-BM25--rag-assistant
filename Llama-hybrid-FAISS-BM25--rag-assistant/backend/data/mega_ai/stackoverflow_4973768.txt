George, I guess what you need is a java library which implements DNS protocol. Take a look at dnsjava This is very good in terms of complete spec coverage of all types of records and class. But the issue which you might face with a java based library is performance. DNS servers would be expected to have a high throughput. But yes, you can solve that by throwing more hardware. If performance is a concern for you , I would suggest to look into unbound
