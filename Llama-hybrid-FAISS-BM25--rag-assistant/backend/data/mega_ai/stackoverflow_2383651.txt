You can use the localStorage API. See http://code.google.com/chrome/extensions/options.html for an example of building an options page with this, but most simply you can do: localStorage['foo'] = 'bar'; // Store data var foo = localStorage['foo']; // Get data This data won't be wiped out on extension update, either.
