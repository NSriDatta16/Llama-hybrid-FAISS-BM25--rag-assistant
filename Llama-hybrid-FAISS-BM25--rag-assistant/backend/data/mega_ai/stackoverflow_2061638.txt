I just gave Mechanize a try with my company's web site. The home page is HTTP, but it contains a link, "customer login," which sends the browser to an HTTPS page. It worked fine. The code is: #!/usr/bin/ruby1.8 require 'rubygems' require 'mechanize' agent = WWW::Mechanize.new page = agent.get("http://www.not_the_real_url.com") link = page.link_with(:text=>"CUSTOMER LOGIN") page = link.click form = page.forms.first form['user_login'] = 'not my real login name' form['user_password'] = 'not my real password' page = form.submit
