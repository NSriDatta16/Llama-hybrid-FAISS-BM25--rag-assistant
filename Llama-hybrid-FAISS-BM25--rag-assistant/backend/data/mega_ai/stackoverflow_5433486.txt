To get the number of countries: SELECT COUNT(DISTINCT country) AS NumCountries FROM tableName To get the number of individuals, or the number of organisations: SELECT COUNT(*) AS NumIndividuals FROM tableName WHERE type = 'individual' SELECT COUNT(*) AS NumOrganisations FROM tableName WHERE type = 'organisation'
