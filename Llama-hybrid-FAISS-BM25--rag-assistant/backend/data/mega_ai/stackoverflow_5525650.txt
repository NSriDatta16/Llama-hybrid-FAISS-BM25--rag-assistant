pushState : window . history . pushState(data, title [, url ] ) Pushes the given data onto the session history, with the given title, and, if provided, the given URL... the user agent must run the following steps... Add a state object entry to the session history, after the current entry , with cloned data as the state object, the given title as the title, and, if the third argument is present, the absolute URL that was found earlier in this algorithm as the URL of the entry. Update the current entry to be the this newly added entry...
