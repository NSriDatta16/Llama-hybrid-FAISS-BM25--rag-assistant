Yes, you can do this using the externals feature. Note that this is not quite the same thing as what was available in VSS, because in Subversion you can only include an entire subtree of a repository, instead of picking and choosing specific files. The method VSS used was fragile and commits to one project could affect an unknown number of other projects. Don't even think about trying to get a compilable historical build out of VSS with shared files.
