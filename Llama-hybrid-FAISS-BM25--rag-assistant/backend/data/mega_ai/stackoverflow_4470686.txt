T.J. Crowder's answer does what you want. This is just an alternative (using the jQuery data storage methods): $('#first').click(function() { // Assign this element to have data with the key 'myVar' $(this).data('myVar', $(this).next().val()); }); $('#second').blur(function() { // Read the data from the other element console.log($('#first').data('myVar')); });
