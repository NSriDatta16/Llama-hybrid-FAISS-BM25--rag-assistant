A naive yet simple way is to compute a convex combination of data and moving average, using a small coefficient 0 for the data and i-1 for the moving average. data = [386 379 295 238 159 190 384 402 552 347 463 259 412 481 523 589 611 479 320 403 642]; for i = .1:.01:.4 plot(movmean(data,7)) hold on plot(data) plot(data*i+movmean(data,7)*(1-i)) legend('movmean','data',"data*"+i+"+movmean*"+(1-i),'Location','northwest') pause(.1) clf end
