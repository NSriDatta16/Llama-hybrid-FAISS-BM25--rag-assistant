Say, if you have data like this A B --------- ---------- xyz.com http://abc.com yyy.com In column B, use a function similar to this. =IF(ISERR(SEARCH("http:",A1,1)),CONCATENATE("http://",A1),A1) Drag down to fill. This will put an http prefix for all items in A where the prefix doesn't exist. Tweak this a little bit for your specific case if needed.
