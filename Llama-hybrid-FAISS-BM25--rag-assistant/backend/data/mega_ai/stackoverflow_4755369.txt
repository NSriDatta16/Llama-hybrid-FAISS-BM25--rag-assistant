Learn PCRE ( preg_* functions in PHP). POSIX regular expressions ( ereg*, split functions in PHP) have been deprecated as of PHP 5.3, and should not be used anymore. This is the official recommendation. To quote the manual page: This function has been DEPRECATED as of PHP 5.3.0. Relying on this feature is highly discouraged.
