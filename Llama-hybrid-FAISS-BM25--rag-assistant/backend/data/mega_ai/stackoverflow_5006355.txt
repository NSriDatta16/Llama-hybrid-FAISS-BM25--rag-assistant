Considering the way lshistory (man page) , the simplest way would be to still use -minor , and " grep attribute " the result. –min/or Includes less important events in the listing: attaching of attributes , version labels, and so on. For type objects and storage pools, minor events include rename operations and changes to pool parameters ( mkpool –update ). That option is the only one with attribute-related event mentioned.
