Generally, no, you can't use an agent from one with a policy server from the other. The Agent Policy Server protocol is not a standard protocol. However, if you are looking to set up federation between a SiteMinder instance and an Open SSO instance you might have more luck. Here is an article from Sun describing one scenario for setting it up. Open SSO supports both SAML 2.0 and SAML 1.1 if this helps.
