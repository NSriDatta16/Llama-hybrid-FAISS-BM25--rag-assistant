One would need to write a function that would save the current selection, then try increasingly wide selections, until the new selection exceeds the saved one or selects all text. Some possible selections are: viW - select word vis - select sentence vip - select paragraph viB - select text within the innermost brackets v2iB - select text within the next most innermost brackets ggVG - select all text
