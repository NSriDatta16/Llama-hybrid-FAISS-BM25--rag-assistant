Your logic is correct. I recommend you don't use wurfl to switch from mobile to desktop version. It penalize the application performance. It is better to use a simple switcher based on a set of heuristics to switch from mobile to desktop, and then use wurfl only ont he mobile version to obtain the devices capabilities. There are a lot of code useful to recognize if a user-agent is a mobile user-agent.
