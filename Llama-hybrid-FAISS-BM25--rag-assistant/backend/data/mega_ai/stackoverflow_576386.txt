In fact, C# has exactly this. double sqrt(double x) { var goodEnough = new Func (guess => Math.Abs(square(guess) - x) (guess => average(guess, x / guess) ); var sqrtIter = default(Func ); sqrtIter = new Func (guess => goodEnough(guess) ? guess : sqrtIter(improve(guess)) ); return sqrtIter(1.0); }
