The clipboard should contain data in the format of CF_HDROP . This can be converted to a file list by calling the DragQueryFile API. Then it's just a list of fully-qualified filenames of whatever files were copied, and now you can operate on those files. A Visual Basic example is in How To Obtain Files Copied from Windows Explorer .
