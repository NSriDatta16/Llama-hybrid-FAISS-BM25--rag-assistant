Every object created where the fields pragma is in use will have those fields (and only those fields) defined, even if you don't initialize them. So you don't have to worry about the %FIELDS table being deprecated. DB $apple = Fruit->new( {qw(color red shape apple taste like-an-apple)} ) DB p join' ',keys %$apple color taste shape DB $kiwi = Fruit->new() DB p join' ',keys %$kiwi color taste shape
