Interesting problem. First let me say that I think the best strategy is to enclose the whole answer in a div and then the problem becomes trivial to solve: Question here Answer here Next Question ... with: $(function() { $("h2").click(function() { $(this).next().toggleClass("highlighted"); }); }); But that being said, it is solvable without that. $(function() { $("h2").click(function() { $(this).nextAll().each(function() { if (this.tagName == 'H2') { return false; // stop execution } $(this).toggleClass("highlighted"); }); }); }); Not supremely elegant but it'll work. Note: This assumes the questions are siblings. If they are not it gets much more complicated.
