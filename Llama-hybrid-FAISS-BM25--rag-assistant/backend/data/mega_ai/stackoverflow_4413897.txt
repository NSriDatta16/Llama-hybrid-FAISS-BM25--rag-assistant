One important thing to bear in mind, is that you should not be recreating your dialog multiple times. (See: http://blog.nemikor.com/2009/04/08/basic-usage-of-the-jquery-ui-dialog/ for a more in depth explanation) In your existing code, your dialog object will be recreated every time you click the show_myform item. $("#myform").dialog({ modal : true, draggable : false, resizable : false, open: function(event, ui) { $("#myform :text:first").focus(); }, autoOpen: false }); $("#show_myform").click(function(event) { $('#myform').dialog('open'); return false; });
