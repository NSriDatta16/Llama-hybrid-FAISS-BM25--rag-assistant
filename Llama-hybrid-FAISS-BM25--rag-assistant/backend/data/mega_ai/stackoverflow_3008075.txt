Do you even need them to be input elements in the first place? You can use Javascript to dynamically create divs or paragraphs or list items or whatever that contain the information you want to present. But if the interactive element is important and it's a pain in the butt to place those elements outside the block, it ought to be possible to remove those elements from the form when the page gets submitted.
