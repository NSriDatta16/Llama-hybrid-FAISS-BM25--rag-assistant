The quick and dirty way would be something like this. 1. create a cachefilename for this specific url 2. check if such a file exists in the cache directory and if it is younger than n minutes 2.1 if not: 2.2 // business logic goes here => Save output e.g. in a variable $output 2.3 save contents of $output in cachefilename 2.4 echo $output 3. if so: 3.1 $output = file_get_contents(cachefilename) 3.2 echo $output It might not be as elegant as memcache or memcached, but you can use it virtually everywhere.
