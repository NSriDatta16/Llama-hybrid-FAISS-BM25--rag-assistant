The constructor public StreamSource(String systemId) Construct a StreamSource from a URL. I think you're passing the content of the XSLT instead. Try this: File xslFile = new File("path/to/your/xslt"); TransformerFactory factory = TransformerFactory.newInstance(); Templates xsl = factory.newTemplates(new StreamSource(xslFile)); You must also set the OutputStream that your StreamResult will write to: ByteArrayOutputStream baos = new ByteArrayOutputStream(); Result result = new StreamResult(baos); transformer.transform(xmlInput, result); return baos.toString();
