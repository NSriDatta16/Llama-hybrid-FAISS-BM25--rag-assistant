This works for me, but it's fairly fragile, so use pixels or ems, not percentages: #div1 { height: 20em; width: 30em; border: 1px dashed #000; text-align:center; } #div2 { height: 5em; width: 5em; border: 1px solid #000; margin: auto; margin-top: 25%; margin-bottom: -25%; line-height: 5em; } Woot! I just set the top margin to 25% and the bottom to -25%. Seems to work pretty well.
