First you should reorganize your HTML code to represent each row in one piece. If you organize the table in columns then it's not easy to create that code from your source XML data. After you reorganized your HTML you can write a new template for the node "studentDetails". Move the code for each row into that template. In the template that matches the node "student" insert to insert the information for each Student.
