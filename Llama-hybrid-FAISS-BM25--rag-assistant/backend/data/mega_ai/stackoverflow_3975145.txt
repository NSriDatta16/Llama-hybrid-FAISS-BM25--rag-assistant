Verify that your browser is not sending a Cache-control: no-cache header with its request. According to the documentation, cached results are not sent if the user agent has specified no-cache. Based on the .ajax call you posted, you should be good, but double checking what is actually sent to the server will let you be sure. A tool like fiddler is invaluable for debugging exactly what is going on over the wire for browser/web service interaction.
