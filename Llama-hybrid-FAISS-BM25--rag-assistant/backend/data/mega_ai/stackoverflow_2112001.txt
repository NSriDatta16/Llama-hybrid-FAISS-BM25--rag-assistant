Tables in RAM are very fast. You can change your existing tables by altering their storage engine. But make a backup if you want to use the data after a server restart. ALTER TABLE test ENGINE = MEMORY; Keep in mind, that MEMORY tables cannot contain BLOB or TEXT columns and data and index size is limited to available memory.
