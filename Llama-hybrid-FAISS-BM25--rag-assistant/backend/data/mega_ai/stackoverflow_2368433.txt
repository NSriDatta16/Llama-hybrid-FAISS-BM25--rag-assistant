You need to customize the toolbars. Specifically, the "Context Menus" toolbar. Select "Customize..." from the "Tools" menu to display the "Customize" dialog. Select the "Toolbars" tab. Check the "Context Menus" item. Modify the context menu you need. You will probably have to create a macro to open the specific item, then "drag" this macro to the context menu. Macros can be found on the "Commands" tab. Select the "Macros" category to display all the available macros in the "Commands" list box. http://social.msdn.microsoft.com/Search/en-US?query=How+to%3A+Customize+Toolbars+%28Visual+Studio%29&ac=8
