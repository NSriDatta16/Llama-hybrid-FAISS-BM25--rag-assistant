You can also just set the UA on the LWP::Simple module - just import the $ua variable, and it'll allow you to modify the underlying UserAgent: use LWP::Simple qw/get $ua/; $ua->agent("WikiBot/0.1"); print get("http://en.wikipedia.org/wiki/Stack_overflow");
