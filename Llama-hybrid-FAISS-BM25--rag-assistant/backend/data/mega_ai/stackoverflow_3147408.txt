Based on Michael Gilbert's answer, you can edit the description in powershell like so: $newCLFormat = p4 change -o | select-string -pattern change, client, status $newCLFormat += "Description: " + $myDescription $newCLFormat | p4 change -i To pull out the new changelist number: $newCLFormat | p4 change -i | select-string "\b(\d)+" | %{$_.matches[0].value} Now there's got to be a quicker, neater way to pull out that number? edit: refactored findstr to select-string edit: neater way to retrieve the changelist: $newCLFormat | p4 change -i | %{ $_.split()[1] }
