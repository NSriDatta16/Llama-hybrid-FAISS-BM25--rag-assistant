The following should give you the day of the week for today (ex. tuesday): or for a specific date you can use something like this: For more info: http://www.php.net/manual/en/function.date.php If you have just a number that you are trying to convert to a day of the week, you could use the following: function convertNumberToDay($number) { $days = array('Sunday','Monday', 'Tuesday', 'Wednesday','Thursday','Friday','Saturday'); return $days[$number-1]; // we subtract 1 to make "1" match "Sunday", "2" match "Monday" } echo convertNumberToDay(3); // prints "Tuesday"
