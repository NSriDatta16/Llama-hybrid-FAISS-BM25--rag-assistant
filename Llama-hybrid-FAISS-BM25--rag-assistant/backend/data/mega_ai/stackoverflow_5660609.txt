You got the syntax almost correct: [0-9]{1,7} . You can make your solution a bit more elegant (and culture-sensitive) by replacing [0-9] with the generic character group "decimal digit": \d (remember that other languages might use different characters for digits than 0-9). And here's the documentation for future reference: .NET Framework Regular Expressions
