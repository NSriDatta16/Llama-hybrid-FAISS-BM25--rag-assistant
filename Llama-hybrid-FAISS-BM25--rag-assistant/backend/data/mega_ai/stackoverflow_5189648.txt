sinatra-authentication assumes you're using a layout unless the current request is an XMLHttpRequest (see the code ). You have two options: Create a layout for your application in views/layout.haml as described a couple paragraphs down at http://sinatra-book.gittr.com/#templates Override sinatra-authentication's use_layout? method as such: module Sinatra module Helpers def use_layout? false end end end
