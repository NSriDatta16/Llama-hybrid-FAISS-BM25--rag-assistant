it adds better control over what is transferred over the wire - cookies are always transferred to the server on each request, the app can choose what data is transferred from the HTML5 storage, when it needs it. it adds support for larger amounts of data - cookies are limited in size. it adds support for caching scenarios - an example is a web app that deals with huge blobs of data - it is better to download the data once to the local storage and subsequently download only incremental updates. it adds support for offline scenarios - an example is a web app that allows you to work with your data even without persistent connection to their servers and uploads when it detects connection.
