This is what the C++03 standard says (3.9.1/2) : There are four signed integer types: “signed char”, “short int”, “int”, and “long int.” In this list, each type provides at least as much storage as those preceding it in the list. So : sizeof(char) This is what the C++0x (3.9.1/2) and C99 (6.2.5/4) standards say : There are five standard signed integer types, designated as signed char, short int, int, long int, and long long int. long is synonym of long int long long doesn't exist in C++03, but will in C++0x.
