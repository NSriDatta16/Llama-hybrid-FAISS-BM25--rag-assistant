The plugin you are using JEditable monitors the elements you've tagged with a specific class for the 'click` event and then enabled the editing of the element once you've clicked it. You can easily fire the click event from another element like this: $("a#click_to_edit").click(function(){ $("#editme").triggerHandler("click"); return false; }) .
