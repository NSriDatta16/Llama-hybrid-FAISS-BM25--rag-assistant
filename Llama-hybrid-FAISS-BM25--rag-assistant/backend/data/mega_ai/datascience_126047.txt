You could use vector autoregressive models to do this, please see the following resource: 11.2 Vector Autoregressive models VAR(p) models This model is used for multivariate time series, and works by using each variable as a previous lag of itself as well as other variables. In this regard, you could choose to model the multiple time series using statsmodels as per this example: statsmodels: Vector Autoregressions tsa.vector_ar
