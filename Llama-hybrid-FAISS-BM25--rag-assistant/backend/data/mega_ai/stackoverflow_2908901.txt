Just an additional consideration: Depending on the content you store in the blobs and what kind of processing you need to do, you might be able to reference data directly from the web page. The advantages of this model is that it offloads the web server from tunneling the content. You can also take advatnage of Windows Azure CDN. BTW, this post explains this in more detail: Azure storage + shared access signatures
