You want to send a content type header to make the browser download the file. If you aren't' generating it dynamically, you will need to read it off the disk first. $fullPath = "/path/to/file/on/server.pdf"; $fsize = filesize($fullPath); $content = file_get_contents($fullPath); header("Pragma: public"); // required header("Expires: 0"); header("Cache-Control: must-revalidate, post-check=0, pre-check=0"); header("Cache-Control: private",false); // required for certain browsers header("Content-Type: application/pdf"); header("Content-Disposition: attachment; filename=\"".basename($fullPath)."\";" ); header("Content-Transfer-Encoding: binary"); header("Content-Length: ".$fsize); echo $content;
