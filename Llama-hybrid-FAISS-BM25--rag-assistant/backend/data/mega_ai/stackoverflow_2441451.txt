I followed your line of thinking, but there were a few bugs using this approach (sometimes the element being dropped just disappeared when using droppable + sortable). Here's something that worked out for me: $(".draglist").draggable({ start: function(e, ui) { draggedItem = ui.item; } }); $(".sortlist").sortable({ receive: function(event, ui) { //do something with the 'draggedItem' here... var droppedElemTxt = draggedItem.text(); var droppedElemId = draggedItem.attr('id'); } , start: function(e, ui) { draggedItem = ui.item; } });
