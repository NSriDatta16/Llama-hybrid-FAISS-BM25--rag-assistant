The problem here is that you removed the django.contrib.syndication.views.feed reference from your urls.py. Instead of using reverse to redirect to a different URL, try just returning the feed from your existing view: from django.http import HttpResponseRedirect from django.core.urlresolvers import reverse from django.contrib.syndication.views import feed def redirect(request, **kwargs): if request.META['HTTP_USER_AGENT'].startswith('FeedBurner'): return feed(request, **kwargs) else: return HttpResponseRedirect('http://feeds2.feedburner.com/MyFeed')
