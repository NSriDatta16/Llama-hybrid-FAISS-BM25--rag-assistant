Set your editor to save in a known encoding (UTF-8 unless you have a good reason to use something else) Specify that encoding in your HTTP Headers Make sure the encoding doesn't get changed by any program code or database storage along the way. Do not use an XML prolog ( ), it triggers Quirks mode in IE6 See also: Character encodings for beginners Handling character encodings in HTML and CSS
