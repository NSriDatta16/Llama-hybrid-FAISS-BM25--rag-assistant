You could set the .height() and .width() using the start and stop events, something like this: $(".icon").draggable({ start: function() { $(this).height(100).width(100); //drag dimensions }, stop: function() { $(this).height(50).width(50); //original icon size } });​ You can give it a try here , or a bit more compact : $(".icon").draggable({ start: function() { $(this).css({ height: 100, width: 100 }); }, stop: function() { $(this).css({ height: 50, width: 50 }); } });​
