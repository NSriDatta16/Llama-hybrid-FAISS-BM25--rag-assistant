Because JAXB default the access type to PUBLIC_MEMBER, make sure you annotate the property (getter) and not the field: @XmlElement(name="organisation") public getOrganisationList() { return organisationList; } If you want to annotate the field then add the following annotation to your class: @XmlAccessorType(XmlAccessType.FIELD)
