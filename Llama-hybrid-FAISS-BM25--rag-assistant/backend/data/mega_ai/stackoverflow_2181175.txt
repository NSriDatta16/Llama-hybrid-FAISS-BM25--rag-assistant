Here's an example of how you might do what Tony has suggested: [DllImport("test.dll", EntryPoint="my_test_func", CharSet=CharSet.Ansi, ExactSpelling=true, CallingConvention=CallingConvention.Cdecl)] private static extern IntPtr my_test_func_imported(); public static string my_test_func() { return Marshal.PtrToStringAnsi(my_test_func_imported()); } On the unmanaged side: const char *my_test_func(void); (this is importing a C function from "test.dll") P.S. As pointed out by ctacke - this is likely unsuitable for .NET CF.
