The default handler for drag_drop in Gtk# is Widget.OnDragDrop. So you would need to subclass TreeView and override that virtual method to preempt TreeView's built-in TreeDragDest mechanism. This signature for OnDragDrop is: protected virtual bool OnDragDrop (Gdk.DragContext context, int x, int y, uint time_)
