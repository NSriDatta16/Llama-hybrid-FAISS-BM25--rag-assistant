Update: Since Android 10,11 Storage updates , Google has restricted Storage access through standard programming language file operations. For applications targeting only Android 10 (API 29) and above, you need to declare "requestLegacyExternalStorage="true" " in your android manifest file to use programming language based file operations. ========== You want to be sure you are correctly finding the address of your SDCard, you can't be sure its always at any particular address. You will want to do the following! File directory = new File(Environment.getExternalStorageDirectory()+File.separator+"images"); directory.mkdirs(); Let me know if this works for you!
