DOM Storage or Web Storage is the collective name given to the following types of client storage options available in HTML5. It includes: localStorage sessionStorage Local storage is persistent meaning the stored data will still be there when you close and re-open the browser window. Session storage is temporary and is available as long as the page session lasts. There is really no comparison between the two since technically they both are the same.
