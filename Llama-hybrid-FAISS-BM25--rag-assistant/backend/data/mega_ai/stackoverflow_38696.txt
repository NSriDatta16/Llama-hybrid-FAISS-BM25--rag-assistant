It would probably be easier and faster to use strpos() to find the position of the first and first then use substr() to extract the paragraph. $paragraph_start = strpos($blog_post, ' '); $paragraph_end = strpos($blog_post, ' ', $paragraph_start); $paragraph = substr($blog_post, $paragraph_start + strlen(' '), $paragraph_end - $paragraph_start - strlen(' ')); Edit: Actually the regex in others' answers will be easier and faster... your big complex regex in the question confused me...
