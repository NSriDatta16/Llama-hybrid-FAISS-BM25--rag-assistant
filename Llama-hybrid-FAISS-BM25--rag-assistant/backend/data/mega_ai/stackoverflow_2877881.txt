Use \n for a line break and \t if you want to insert a tab. You can also use some XML tags for basic formatting: for bold text, for italics, and for underlined text. Other formatting options are shown in this article on the Android Developers' site: https://developer.android.com/guide/topics/resources/string-resource.html#FormattingAndStyling
