dvcolgan’s answer is right, but it sounds like you might be trying to break your array into groups of 100. If that’s the case, there’s a convenient built-in method for that: nums = (1..500).to_a nums.each_slice(100) do |slice| puts slice.size end # => 100, 100, 100, 100, 100
