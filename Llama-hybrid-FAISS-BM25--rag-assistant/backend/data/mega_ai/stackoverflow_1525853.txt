Your only options are to use CssClass or inline styles. As ASP.NET auto-generates the ID's of server side controls you should never try to second guess what these will be. It can be a major pain getting Webforms to work with elegant CSS layouts. ASP.NET 4.0 will introduce the ClientID property that should make it easier to work with ID attributes in the future.
