What do you want to do with the obtained HTML? Parse it? It may be good to know that a bit decent HTML parser can already have a constructor or method argument which takes straight an URL or InputStream so that you don't need to worry about streaming performance like that. Assuming that all you want to do is described in your previous question , with for example Jsoup you could obtain all those news links extraordinary easy like follows: Document document = Jsoup.connect("http://news.google.com.ar/nwshp?hl=es&tab=wn").get(); Elements newsLinks = document.select("h2.title a:eq(0)"); for (Element newsLink : newsLinks) { System.out.println(newsLink.attr("href")); } This yields the following after only a few seconds:
