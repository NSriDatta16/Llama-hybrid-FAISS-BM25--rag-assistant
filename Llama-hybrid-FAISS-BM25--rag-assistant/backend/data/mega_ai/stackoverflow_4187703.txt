' n ' represents support for unicode characters. char - specifies string with fixed length storage. Space allocated with or without data present. varchar - Varying length storage. Space is allocated as much as length of data in column. text - To store huge data. The space allocated is 16 bytes for column storage. Additionally - text and ntext have been deprecated for varchar(max) and nvarchar(max)
