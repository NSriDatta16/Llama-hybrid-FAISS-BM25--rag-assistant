Don't worry. You can run anything in Emacs with a single keystroke. Others have noted ways you can do this elegantly with TRAMP, but if that doesn't suit (for whatever reason) you could also just cook up your own function using: (defun arbitrary-remote-command () (interactive) (shell-command "ssh user@remotehost arbitrary-command")) (global-set-key (kbd " ") 'arbitrary-remote-command)
