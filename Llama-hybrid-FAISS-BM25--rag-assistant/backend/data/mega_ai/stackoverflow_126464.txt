I would recommend disregarding the W3C standards and writing out HTML-properties on the elements that need handlers attached to them: Note: this will not break the rendering of the page! Declare a few functions: function doAlertOne() { } function doAlertTwo() { } function doAlertThree() { } And then using jQuery like so: $("ul li").each(function () { switch($(this).attr("handler")) { case "doAlertOne": doAlertOne(); break; case ... etc. } }); Be pragmatic.
