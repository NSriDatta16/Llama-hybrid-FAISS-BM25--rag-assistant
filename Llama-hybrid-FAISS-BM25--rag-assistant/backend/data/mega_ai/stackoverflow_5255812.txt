Microsoft have not provided any mechanisms for accessing a database in the first release of Windows Phone 7, so right now you can access a SQL CE database. That said there are other options (web services and 3rd party databases using Isolated Storage on the phone). For more information, see this question: How to use the class SqlCeConnection in the Window Phone 7 application ?
