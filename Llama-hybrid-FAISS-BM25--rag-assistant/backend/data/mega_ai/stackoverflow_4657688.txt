I guess you were using the script from the following page In order to fix the bug occuring in the Chrome change the following lines: if (document.all && document.getElementById && !window.opera) this.initIEengine(); if (!document.all && document.getElementById && !window.opera) this.initFFengine(); to this: var is_chrome = navigator.userAgent.toLowerCase().indexOf('chrome') > -1; if (is_chrome = true) { this.initIEengine() } else { if (document.all && document.getElementById && !window.opera) this.initIEengine(); if (!document.all && document.getElementById && !window.opera) this.initFFengine(); } Works perfectly then
