Your inheritance tree looks something like this: ref2 is-a int[][] ref2 is-a Object[] ref2 is-a Object Here's a code fragment that illustrates what I mean: Object ref2 = new int[][]{{1,2,3}, {4,5,6}}; System.err.println("ref2: " + (ref2 instanceof int[][]) + " " + (ref2 instanceof Object[])); You should see something like: ref2: true true
