You are not doing anything wrong, bitbucket does some user agent detection (to detect mercurial clients for example). Just changing the user agent fixes it (if it doesn't have urllib as a substring). You should fill an issue regarding this: http://bitbucket.org/jespern/bitbucket/issues/new/
