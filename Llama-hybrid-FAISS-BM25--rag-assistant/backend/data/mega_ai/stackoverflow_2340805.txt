You should prepend the base URL (most of the time forward slash, but not always). You can get it as follows (general expression that will work even if your site resides in a directory): $baseUrl = rtrim((string)dirname($_SERVER['SCRIPT_NAME']), '/\\') . '/'; And then you'll write: includes/NAV.css" type="text/css" /> Another possibility is to use the base tag in head: " /> This has the advantage of changing only one line in your code. You can go furthur and make utility method to do this: function baseUrl($url = '', array $query = null, $fragment = null) { static $baseUrl = rtrim((string)dirname($_SERVER['SCRIPT_NAME']), '/\\') . '/'; if ($query !== null) { $url .= '?' . http_build_query($query); } if ($fragment !== null) { $url .= '#' . (string)$fragment; } $url = $baseUrl . $url; $url = trim(str_replace('\\', '/', (string)$url)); return $url; }
