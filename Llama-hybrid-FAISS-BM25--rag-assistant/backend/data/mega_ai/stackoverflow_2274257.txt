You can do this using tab stops in the Format -> Text menu 1) Create a calc field with the following definition (the character in the quotes is a tab): title & " " & page 2) Add this field to your layout (it needs to be an actual field, not a merge field) 3) Highlight the field and choose format -> text -> paragraph -> tabs 4) Create a new Tab with a position of 6 inches and a Fill Character of "." or "â€¦" Now when viewed, any space from the end of the title up to the tab stop 6 inches away is filled with the fill character. No monospace font required.
