If a 3000 byte packet enters an Ethernet network with a default MTU size of 1500 (for ethernet), it will be fragmented into two packets of each 1500 bytes in length. That is the only time I can think of. Wireshark is your best bet for checking this. I have been using it for a while and am totally impressed
