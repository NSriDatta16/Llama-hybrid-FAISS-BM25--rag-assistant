Your function scope doesn't have access to the $conn variable, one way to work around this issues is by using globals . Here is an example: function query(){ global $conn; // your code here... } Search StackOverflow for Singleton , Factory , Registry and Dependency Injection Patterns for other more advanced and elegant alternatives.
