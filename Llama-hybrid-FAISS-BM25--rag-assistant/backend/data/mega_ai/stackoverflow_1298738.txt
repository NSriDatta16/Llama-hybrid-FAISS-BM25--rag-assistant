The $.browser obejct is populated using a concept know as Browser-Sniffing . Put simply, this is the processes scanning data out of the user-agent string which is sent by browsers, robots, and anything else that accesses the page. Due to the susceptiblity of the user agent to faking, this object is deprecated in later versions of jQuery. Object Detection and Feature Detection (see $.support ) are now used in preference.
