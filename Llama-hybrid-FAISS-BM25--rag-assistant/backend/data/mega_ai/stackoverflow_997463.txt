What do you mean by numpy solution ? Numpy has some sort facilities, which are instantenous for those reasonably small arrays: import numpy as np a = np.random.randn(100000) # Take a few ms on a decent computer np.sort(a) There are 3 sorts of sort algorithms available, all are Nlog(N) on average.
