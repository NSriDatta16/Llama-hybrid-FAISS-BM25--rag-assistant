Your HttpWebRequest.GetResponse call will throw a WebException when; Abort was previously called. -or- The time-out period for the request expired. -or- An error occurred while processing the request. Catch this exception. I used to just pull my network cable out to test this sort of thing although you could be more elegant and use a proxy tool and block that particular request.
