Yes, You have to use the http headers to instruct browser not to cache the page. This page () from OWASP contains the information about how to do this. As per the above article you can set the following header to instruct browser not to cache the page: HTTP/1.1: Cache-Control: no-cache or HTTP/1.0: Pragma: no-cache Expires: Hope this helps.
