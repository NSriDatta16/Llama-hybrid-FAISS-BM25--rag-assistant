If your goal is to send messages from a .NET language, consider leveraging the Apache ActiveMQ NMS library for .NET . They claim to use a single API to connect to multiple different providers. Currently, the following providers are available: ActiveMQ which connects using OpenWire to an ActiveMQ Message Broker. STOMP which connects to any STOMP Broker. Their site linked above has the downloads, and links to articles on how to get started on the common messaging scenarios.
