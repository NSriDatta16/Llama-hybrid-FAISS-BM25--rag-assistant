This line: inmdptr inmodptr; declares a pointer inmodptr , but doesn't initialise it - it's pointing at some random memory location. You need to make it point somewhere that can hold an actual instance of the struct inmod_struct object - for example: inmdtyp inmod; inmdptr inmodptr = &inmod;
