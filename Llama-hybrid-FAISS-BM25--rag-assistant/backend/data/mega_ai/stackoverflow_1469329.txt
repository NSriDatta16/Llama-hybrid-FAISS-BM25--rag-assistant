You can do this using the "application/octet-stream" type of Embed: http://flexscript.wordpress.com/2008/08/02/embedding-text-file-into-flex/ The relevant code: [Bindable] [Embed(source="MyFile.txt", mimeType="application/octet-stream")] private var myFileClass:Class; ... var MyFileByteArray:ByteArrayAsset = ByteArrayAsset(new myFileClass()); var story:String = MyFileByteArray.readUTFBytes(MyFileByteArray.length)
