Ordinal logistic regression is what you need in this case. Multinomial is used when the DV is nominal. See this page for more info. To answer your second question, logistic regression requires the dependent variable to be categorical while the independent variable(s) may be continuous or categorical, so the ordinal logistic regression will be fine.
