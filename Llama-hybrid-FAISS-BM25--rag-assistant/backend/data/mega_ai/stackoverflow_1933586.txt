Column order is irrelevant to a database, so there isn't any benefit beyond readability to reorganizing column order. The only means of reordering a tables columns is to: Rename the current table to something else, like [tablename]_ORIG. A database won't allow identically named tables Create a new table with the column order you desire Copy the data from the old table into the new one Delete the old table
