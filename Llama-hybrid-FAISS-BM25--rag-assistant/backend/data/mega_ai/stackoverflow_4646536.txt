You need to create your own module. Never edit files in app/code/core/ folder . If you want to add functionality to Magento, you need to rewrite the base classes. Alan Storm has good tutorials to follow: How to create a simple 'Hello World' module in Magento? To rewrite a controller ( AccountController in your case), and after you create you own module, you can follow this tutorial. Configure the Layout In your app/code/local/MyCompany/Module/etc/config.xml : myfile.xml (...) Then you could update your layout in app/design/frontend/default/mydesign/layout/myfile.xml .
