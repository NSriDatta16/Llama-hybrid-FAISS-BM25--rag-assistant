Get up the Toolpart (Modify Shared Web Part) then Right Click > View Source. Do a search for MSOSPWebPartManager_OldSelectedStorageKeyName and there is the ID of your web part ( Microsoft.SharePoint.WebPartPages.WebPart.ID ) Tested with WSS3 (SharePoint 2007) and SharePoint Foundation 2010.
