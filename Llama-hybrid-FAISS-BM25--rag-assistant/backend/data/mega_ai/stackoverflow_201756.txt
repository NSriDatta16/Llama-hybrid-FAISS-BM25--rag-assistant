urllib.urlopen() throws the following request at the server: GET / HTTP/1.0 Host: 127.0.0.1 User-Agent: Python-urllib/1.17 while urllib2.urlopen() throws this: GET / HTTP/1.1 Accept-Encoding: identity Host: 127.0.0.1 Connection: close User-Agent: Python-urllib/2.5 So, your server either doesn't understand HTTP/1.1 or the extra header fields.
