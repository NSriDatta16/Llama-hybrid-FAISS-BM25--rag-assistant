Your CSS for #handler matches only id="handler" , not id="handlerXX" , you should use a class and a class selector instead, like this: ' ' + ' ' (Note the trailing space on the id is also removed, to be valid) Also with matching CSS: .handler { /* styles */ } The same applies to #drag -> .drag , etc. You can test it here .
