You can attach it in node creation time (see Menu fieldset), or use module like Auto Menu . Some more details about this module (from its project page): The Auto Menu module automatically generates menu entries on node creation/edition. Parent menu item can be specified on a per content type basis. This module acts when the menu section of a node is left empty only. So, users can still organize menus manually. Moreover, default setting for content types is to not create menu items automatically.
