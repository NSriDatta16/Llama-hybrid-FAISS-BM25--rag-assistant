If you can install pecl_http then this is a really elegant solution. It can allow you to check if you are overriding a get variable they may have set. $urlComps = parse_url($url); // Get the current query string $queryString = isset($urlComps['query']) ? $urlComps['query'] : ''; // Turn it into an array for easy manipulation parse_str($queryString, $queryVars); // Make changes to the query vars $queryVars['q2'] = 'two'; // Empty paths return relative URLs. $urlComps['path'] = isset($urlComps['path']) ? $urlComps['path'] : '/'; // Make the pecl_http call $newURL = http_build_url($urlComps, array('query' => http_build_query($queryVars))); Note: if you can't install pecl_http, the only function that comes from that is the last one with the function http_build_url . You can fairly easily build your own function to rebuild the URL from its components... parse_url parse_str http_build_query http_build_url * uses pecl_http
