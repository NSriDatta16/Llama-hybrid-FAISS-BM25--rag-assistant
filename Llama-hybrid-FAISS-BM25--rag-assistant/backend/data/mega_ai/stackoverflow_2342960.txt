In SQL Server Management Studio, go expand the SQL Server Agent node under the DB server, right click the Jobs folder and select New Job... Jobs -> New Job"> (If the SQL Server Agent node does not appear, you may be missing the required permissions ) That will take you through a wizard to schedule a sproc to run on whatever schedule you want. In terms of how to get a sproc to run on db startup, see this article .
