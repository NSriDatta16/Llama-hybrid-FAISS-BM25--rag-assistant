You should be able to leverage Generic Constraints to help you out. First, change your interface definition: public interface ILookupRepository where T : ILookup // ^^^^^^^^^^^^^^^^^ Second, change your class definition to match the constraints: public class LookupRepository : ILookupRepository where T : ILookup // ^^^^^^^^^^^^^^^^^ The constraint will require the generic type parameter to implement ILookup . This will let you use the interface members in your GetList method.
