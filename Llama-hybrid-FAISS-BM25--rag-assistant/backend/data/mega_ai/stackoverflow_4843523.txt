OWL statements are just RDF statements that use predicates from the defined OWL vocabulary to (meta-)describe a vocabulary. In RDF/XML, which seems to be your target, here is an example of the definition of an Object Property. withinOrganization Determines in which organization is the position Here is a portion of the above as raw triples, which are in some ways clearer. . "withinOrganization" . "Determines in which organization is the position" . In practical terms, download Protege or TopBraid if you want to make up an Ontology, or (e.g.) Jena if you want to do this in Java code.
