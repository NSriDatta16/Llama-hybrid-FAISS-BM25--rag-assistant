This is a bug in Mysql .net connector check this bug report for more details http://bugs.mysql.com/bug.php?id=52747 UPDATE: After version 6.1.1 you should add "old guids=true" to your connection string whenever you use BINARY(16) as your storage type. Else you should use CHAR(36)
