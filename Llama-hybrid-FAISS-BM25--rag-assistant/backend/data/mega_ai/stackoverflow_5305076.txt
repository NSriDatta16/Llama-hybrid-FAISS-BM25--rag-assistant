With the help of Qt Undocumented qmake I figured out a custom configuration in qmake. However, it is not flawless, it produces now a libagent.so and a agent.o which is not needed. SOURCES_AGENT = agent.cpp agent.name = agent agent.input = SOURCES_AGENT agent.dependency_type = TYPE_C agent.variable_out = OBJECTS agent.output = libagent.so agent.commands = $${QMAKE_CXX} $(CXXFLAGS) -fPIC -shared -o libagent.so $(INCPATH) ${QMAKE_FILE_IN} QMAKE_EXTRA_COMPILERS += agent
