If you're looking to disallow the user to drag-out from a touch, look at the UIControl class. Specifically take note of: - (void)addTarget:(id)target action:(SEL)action forControlEvents:(UIControlEvents)controlEvents If you're using InterfaceBuilder, you should look at the options available for Events (in the Inspector window, second tab).
