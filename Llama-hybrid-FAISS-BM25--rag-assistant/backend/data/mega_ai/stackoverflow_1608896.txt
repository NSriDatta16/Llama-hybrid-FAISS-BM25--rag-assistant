IDs identify , classes classify . This point has been mentioned several times here, more or less. An example would be: [...] From a CSS specificity standpoint, targeting IDs will make CSS rules more specific than rules with classes, even several classes. Thus: p#recent {} is more specific than body>p.news.recent {} and when a user agent finds both of these, the first will prevail.
