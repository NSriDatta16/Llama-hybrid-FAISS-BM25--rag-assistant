Ok, I mis-read the question initially. :-) If you change the insert lines from: INSERT [foo] ([a], [b], [OrganizationId]) VALUES (N'a', N'b', N'1'); to: exec('INSERT [foo] ([a], [b], [OrganizationId]) VALUES (''a'', ''b'', ''1'')'); You shouldn't have that problem, since the SQL "text" inside the exec won't be parsed until the exec() is actually called.
