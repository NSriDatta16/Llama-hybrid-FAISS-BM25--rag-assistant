There are 4 things that we look for: The user agent string. It is very easy to fake, but often crawlers will use their own unique user agent string. The speed of access of pages, if they access more than one every half second or so, that's usually a good indication If they request just the HTML, or if they request the entire page. Some crawlers will only ask for the HTML structure. This is usually a good tip off. The incoming url
