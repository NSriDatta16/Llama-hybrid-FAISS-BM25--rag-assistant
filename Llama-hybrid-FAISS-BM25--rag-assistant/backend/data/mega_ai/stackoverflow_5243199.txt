in the end I used VFS library of apache and my code looks like this: public static void main(String[] args) { FileSystemManager fsManager = null; String path = "\\\\myip\\storage\\ogg\\VX-276.ogg"; try { fsManager = VFS.getManager(); FileObject basePath; basePath = fsManager.resolveFile("file:" + path); if (basePath.exists()) System.out.println("exists"); else { System.out.println("missing" + basePath.getURL()); } } catch (FileSystemException e) { // TODO Auto-generated catch block e.printStackTrace(); } } In this way, I don't need to create a driver for each user of the system and it allows me not to depend on operation system!
