Two things I can think of: 1: Your SD Card may not be mounted. if (android.os.Environment.getExternalStorageState() != android.os.Environment.MEDIA_MOUNTED) then you have a problem. 2: All of the directories in path to the file may not exist. filepath = getFilePath(locationName); File file = new File(filepath); File parentDir = file.getParentFile(); if (!parentDir.exists()) { parentDir.mkdirs(); }
