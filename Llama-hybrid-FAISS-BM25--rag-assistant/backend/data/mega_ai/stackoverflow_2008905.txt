Yes you can using a partial file. The syntax is it will then look for a file called Resources.spark in the View/ > directory and if that isn't found in the shared directory. If you name the file _Resources.spark then you could also use it as a custom element (sans the underscore) like so for more information see http://sparkviewengine.com/documentation/organizingcontent#Parsingandrenderingpartialfiles
