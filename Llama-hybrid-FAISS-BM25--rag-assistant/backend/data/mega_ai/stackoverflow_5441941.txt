You haven't allocated any storage, only a pointer. The pointer needs something to point at. You should do it like this: #include int main(void) { int i; printf("Enter a no \n"); scanf("%d", &i); printf("i=%d", i); return 0; } I also took the liberty of making the declaration of main() valid, and returning a value. If you want to learn about pointers, then try like this: #include int main(void) { int i; int* ip; ip = &i; printf("Enter a no \n"); scanf("%d", ip); printf("*ip=%d", *ip); return 0; } Or if you want heap allocation rather than stack allocation: #include #include int main(void) { int* ip; ip = malloc(sizeof(int)); printf("Enter a no \n"); scanf("%d", ip); printf("*ip=%d", *ip); free(ip); return 0; }
