Merging all of your css files into one will absolutely gain performance. Whether that performance is noticeable depends on your load, number of requests etc. For the average blog, this will have close to zero impact. Read Best practices for speeding up your web site at Yahoo! Developer. It'll explain things way better than i can. As you say, a reason not to merge css files is for maintainability. However, there are many tools out there which automatically merge and minify your css files into one. You should check out YUI Compressor , this will help you with merging and minifing your css files.
