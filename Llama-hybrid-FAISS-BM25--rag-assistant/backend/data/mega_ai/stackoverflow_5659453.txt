Using Win32_Process , You are able to get the UserModeTime and the KernelModeTime (given in 100 nanosecond units) which allows you to compute each CPU average during a certain time. Assuming you get two Win32_Process informations separate by a TimeInterval (given in 100 nanosecond units). UserTimeRate = ((UserModeTime2 - UserModeTime1) / TimeInterval) * 100; KernelTimeRate =((KernelModeTime2 - KernelModeTime1) / TimeInterval) * 100; CPU = (((UserModeTime2 - UserModeTime1) + (KernelModeTime2 - KernelModeTime1)) / TimeInterval) * 100; If you want the rate from the begining, you can compute TimeInterval from CreationDate to now.
