Welcome to Rails, Your best approach here is probably not to use a loop, but to chain a find to your first finder. For example: @user.stores.find(store_id) This will leverage the db and be much faster. Check out the API If you do want to loop you do something like the following @user.stores.each do |store| @my_store = store if store.id == store_id end or @my_store = @user.stores.select{|s| s.id == store_id} or @contains_store = @user.stores.include?{|s| s.id == store_id} Good luck,
