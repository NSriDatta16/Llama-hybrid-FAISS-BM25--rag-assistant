Classification accuracy is not a good metric when your dataset is unbalanced. You should use AUC. The default feature importance techniques in random forests are based on classification accuracy. You should use feature importances based on AUCs: An AUC-based permutation variable importance measure for random forests . If your use R you can find their method implemented in the package party .
