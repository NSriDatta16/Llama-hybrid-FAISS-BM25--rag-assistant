Since you need some real code to start with, here are a few hints. Drag and drop the ASIHTTPRequest package into your project. Since what you want is download images from the internet, you only need to deal with one class in the ASI package, the ASIHTTPRequest class. Read the ASIHTTPRequest.h file closely before you start. It has every interface you need. Specifically, pay attention to these two methods, you will have to override them to get real-time download statistics. . - (void)request:(ASIHTTPRequest *)request incrementDownloadSizeBy:(long long)newLength; - (void)request:(ASIHTTPRequest *)request didReceiveBytes:(long long)bytes;
