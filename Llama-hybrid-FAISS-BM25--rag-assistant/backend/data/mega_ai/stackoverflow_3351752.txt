If you look at the Gang of Four design patterns, there's a pattern called FlyWeight . Here's a quote from wikipedia: A flyweight is an object that minimizes memory use by sharing as much data as possible with other similar objects; it is a way to use objects in large numbers when a simple repeated representation would use an unacceptable amount of memory. This would be an acceptable use of a shallow copy.
