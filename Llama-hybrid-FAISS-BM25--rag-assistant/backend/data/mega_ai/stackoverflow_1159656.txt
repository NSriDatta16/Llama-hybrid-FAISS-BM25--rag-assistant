As far as I know, there's no direct information in the drag drop structure that indicates the originating application. See * Shell Clipboard Formats (MSDN). If you're only interested in finding out if it's a drop from Internet Explorer, the presence of CFSTR_UNTRUSTEDDRAGDROP is a clue; AFAIK, only Internet Explorer and Web Browser Controls will put this format on the clipboard.
