In Javascript you can achieve this vis hash = window.location.hash; In PHP you can access it using parse_url() parse_url() : Return Values On seriously malformed URLs, parse_url() may return FALSE. Otherwise an associative array is returned, whose components may be (at least one): * scheme - e.g. http * host * port * user * pass * path * query - after the question mark ? * fragment - after the hashmark #
