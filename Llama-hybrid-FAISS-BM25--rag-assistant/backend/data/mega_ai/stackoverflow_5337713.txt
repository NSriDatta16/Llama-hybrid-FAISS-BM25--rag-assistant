You'd prevent the default behaviour from being actioned on links, buttons and the like by using preventDefault() ; $( ".selector" ).draggable({ stop: function(event, ui) { // Do some stuff when you've finished dragging your element event.preventDefault(); } }); Check the jQuery API for more information.
