You can use a CASE expression to test whether the cost is 0: SELECT *, CASE cost WHEN 0 THEN (7.87 + (kwh * 8.19) / 100) * 1.03 ELSE cost END calc_cost FROM tablename You can then use this expression with AVG to get the average: SELECT AVG(CASE cost WHEN 0 THEN (7.87 + (kwh * 8.19) / 100) * 1.03 ELSE cost END) FROM tablename
