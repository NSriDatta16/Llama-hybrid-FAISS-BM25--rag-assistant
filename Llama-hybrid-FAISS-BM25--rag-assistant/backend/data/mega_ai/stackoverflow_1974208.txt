If you really want to fetch from a SQL database, there are a number of wrapper libraries you could use. FMDB (SQLite only) FDO (SQLite only) CocoaMySQL (MySQL only) Depending on your data needs, I might actually leverage a scripting language and call that (somehow) from your PrefPane. For example, create a bundle using PyObjC or MacRuby and use SQLAlchemy or ActiveRecord as appropriate.
