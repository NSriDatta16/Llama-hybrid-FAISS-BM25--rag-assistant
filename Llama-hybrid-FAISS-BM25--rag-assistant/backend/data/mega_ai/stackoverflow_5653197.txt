ASP.Net MVC : Get query values with no key

I have URL: http://site.com/page.aspx?update how do I check if that update value is present? HttpValueCollection treats that as an entity with null key. I have tried: var noKeyValues = Request.QueryString.GetValues(null); if (noKeyValues != null && noKeyValues.Any(v=>v==update)) ... but it gives me a frowny line, because GetValues' argument is decorated with [NotNull]. so I end up doing: var queryValuesWithNoKey = Request.QueryString.AllKeys.Select((key, index) => new { key, value = Request.QueryString.GetValues(index) }).Where( item => item.key == null).Select(item => item.value).SingleOrDefault(); if (queryValuesWithNoKey != null && queryValuesWithNoKey.Any(v => v.ToLower() == "update")) live = true; not the most elegant workaround. Is there a better way to get key-less value from query string?
