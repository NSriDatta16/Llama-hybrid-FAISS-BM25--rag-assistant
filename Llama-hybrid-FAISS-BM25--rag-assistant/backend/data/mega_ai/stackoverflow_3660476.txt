Magento has no built in way of doing an order import export with imp/exp profiles. If you want orders from an different system imported into a different system, it means Programmatically exporting the orders, customers, shipments, invoices, etc. from one system Programmatically importing the orders into the new system It's not a trivial task. If you're not up for a long learning process you'll need a developer or firm that's done it before to do it for you.
