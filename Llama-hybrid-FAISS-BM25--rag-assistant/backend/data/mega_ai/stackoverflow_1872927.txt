If you want to reuse your native code you could expose a function that takes the server name and port as input parameters and that returns the data read. This native function could then be invoked in C#: [DllImport("yourlib.dll")] static extern string FetchData(string host, int port); Another option is to use .NET built-in classes to achieve similar functionality: var request = (HttpWebRequest)WebRequest.Create("http://s-syncml.geodesic.net:444"); request.UserAgent = "CeHttp"; var response = request.GetResponse(); using (var stream = response.GetResponseStream()) using (var reader = new StreamReader(stream)) { string result = reader.ReadToEnd(); }
