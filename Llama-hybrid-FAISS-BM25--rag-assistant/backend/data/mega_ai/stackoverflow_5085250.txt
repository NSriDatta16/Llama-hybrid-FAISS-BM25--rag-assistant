Berkeley DB XML is part of the Oracle Berkeley DB family of products. It is a C++ library that links into your application, enabling XML data storage and XQuery and XPath queries. It provides C++ & Java APIs and support for several scripting languages. It supports transactional and non-transactional operations, XML and/or key-value data storage and High Availability via replication. It's designed to be small, highly performant, reliable and scalable. You can find more information about Berkeley DB XML in the following locations: Product Homepage Downloads FAQ Documentation Primary technical discussion forum .NET Implementation
