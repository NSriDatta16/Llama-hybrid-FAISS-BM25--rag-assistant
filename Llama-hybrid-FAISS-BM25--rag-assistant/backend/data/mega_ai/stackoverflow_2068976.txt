strip_tags() removes HTML from the value of a variable. The second parameter is useful if you would like to make exceptions, and leave certain tags in, like the p aragraph tag. $text = ' Paragraph. Other text '; echo strip_tags($text); // Paragraph. Other text echo strip_tags($text, ' '); // Paragraph.
