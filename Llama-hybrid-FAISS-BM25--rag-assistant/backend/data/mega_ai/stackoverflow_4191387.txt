The remote server might be checking your user agent. Try setting it: $ua = 'Mozilla/5.0 (X11; U; Linux x86_64; en-US; rv:1.9.2.12) '. 'Gecko/20101027 Ubuntu/9.10 (karmic) Firefox/3.6.12'; curl_setopt($ch, CURLOPT_USERAGENT, $ua);
