Try to reduce the actions on the DOM as much as possible. Every appendChild on unorderedList forces the browser to re-render the complete page. Use documentFragement for that sort of action. var frag = document.createDocumentFragment(); for (var i = alphabet.length; i--; ) { var li = document.createElement("li"); li.appendChild(document.createTextNode(alphabet[indexNum++])); frag.appendChild(li); } unorderedList.appendChild(frag); So there will be only one DOM action which forces a complete redraw instead of alphabet.length redraws
