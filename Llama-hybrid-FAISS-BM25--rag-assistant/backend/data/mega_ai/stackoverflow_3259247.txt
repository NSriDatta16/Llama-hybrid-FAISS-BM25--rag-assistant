You could use the Set::combine() utility method to do this. I've used it for similar means as so: public function groupByMenu() { return Set::combine ( $this->find ( 'all', array ( 'conditions' => array ( 'NavItem.active' => 1 ), 'order' => 'NavMenuItem.display_order' ) ), '{n}.NavItem.title', '{n}', '{n}.NavMenu.id' ); } The code above takes a set of navigation items and reorganizes them so that they're grouped by the menu(s) they are displayed within.
