The Caja parser uses libhtmlparser, an HTML5 parser that deals well with tag soup containing embedded XML subtrees producing an org.w3c.dom.DocumentFragment , and has a renderer that produces well formed HTML. The parser code is at http://code.google.com/p/google-caja/source/browse/trunk/src/com/google/caja/parser/html/DomParser.java The renderer code is at http://code.google.com/p/google-caja/source/browse/trunk/src/com/google/caja/parser/html/Nodes.java
