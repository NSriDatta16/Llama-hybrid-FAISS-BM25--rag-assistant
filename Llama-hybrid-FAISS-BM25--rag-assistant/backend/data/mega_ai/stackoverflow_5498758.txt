I'm not exactly sure why you would want to do this, but depending on the field name you can consider using loops to help automate the entire process. $field_map = array( 'element_3' => 'Name', 'element_2' => 'E-mail', 'element_14' => 'City', 'element_15' => 'Country' ); $submitted_data = array('fields' => array()); foreach ( $field_map as $key => $label) { $submitted_data['fields'][] = array( 'key' => $key, // e.g. element_2 'label' => $label, // e.g. E-mail 'value' => $post_data[$key] // e.g. john@example.com ); } This separates the storage/mapping of key/label pairs from the part which processes it, making it easier to maintain and modify in the future.
