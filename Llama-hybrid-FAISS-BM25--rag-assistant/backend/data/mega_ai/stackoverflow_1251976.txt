You can implement drag and drop on most (all?) Swing components. See this tutorial to get started. Update: based on your updated question. have a look at JFrameBuilder (note that its not free). JFrameBuilder is an easy-to-use visual Java GUI builder. JFrameBuilder provides the application GUI solution for Java developers. It enables Java developers to create sophisticated Swing GUI applications using drag-and-drop interface without spending a lot of time writing code. Personally I find it a lot simpler to write pretty UIs in SWT (the toolkit used by Eclipse), it has more access to the underlying OS and provides a richer experience that's closer to what you're after in my opinion.
