Though i'm not sure what tools you're using for unit-testing, you could try to move the attribute on top of the get method declaration: int GetValue { [ExcludeFromCodeCoverage] get { this.SomeMethod(); return this.Find(x => x.Value == this._Value); } } This, because on compile time, the GetValue property is converted into a method, with signature get_GetValue : int
