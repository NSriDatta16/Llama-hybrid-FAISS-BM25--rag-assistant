You can create an "Event Oberserver" to trigger an event. Create the following folders : app/code/local/[myMagento]/EventTest/etc app/code/local/[myMagento]/EventTest/Model create a file under /EventTest/Model directory like myMagento_EventTest.xml and put the following : true local create config.xml file under /EventTest/etc folder nad put the following : 0.1.0 singleton myMagento_EventTest_Model_Observer MyTestMethod create model class file observer.php under the /EventTest/Model directory and put following getEvent(); //Fetches the current event $product = $event->getProduct(); $eventmsg = "Current Event Triggered : " . $event->getName() . " Currently Added Product : " . $product->getName(); //Adds Custom message to shopping cart echo Mage::getSingleton("checkout/session")->addSuccess($eventmsg); //Your Custom Logic Here //you can use print_r($product) here to get more details } }
