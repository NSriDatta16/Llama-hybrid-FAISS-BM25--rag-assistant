You can use an ORDER BY with a LIMIT : SELECT id, average, name FROM videos ORDER BY average DESC LIMIT 1 ORDER BY average DESC orders the rows in order of descending average (i.e. the first row will have an average equal to MAX(average) ). LIMIT 1 causes only the first row to be returned.
