For all who are interested, I figured it out. By changing the HeaderFooter parameter to Phrase, from Paragraph, the problem was solved. Change this line: HeaderFooter header = new HeaderFooter(new Paragraph(headString, iTextHeadingFont), true); To this: HeaderFooter header = new HeaderFooter(new Phrase(headString, iTextHeadingFont), true); Issue closed!
