A JavaScript library allows you to do these things very easily jQuery $('#myElement').css({ font-size: '12px', left: '200px', top: '100px' }); Object and a for-in-loop Or, a much more elegant method is a basic object & for-loop var el = document.getElementById('#myElement'), css = { font-size: '12px', left: '200px', top: '100px' }; for(i in css){ el.style[i] = css[i]; }
