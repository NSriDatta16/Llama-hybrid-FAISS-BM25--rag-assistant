I hacked something together, but you will have to look at $_SERVER['HTTP_USER_AGENT'] to see if they come from a search-engine domain. function is_crawlers() { $sites = 'Google|Yahoo|msnbot|'; // Add the rest of the search-engines return (preg_match("/$sites/", $_SERVER['HTTP_USER_AGENT']) > 0) ? true : false; }
