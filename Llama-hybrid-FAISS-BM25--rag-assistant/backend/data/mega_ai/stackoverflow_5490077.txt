What do you mean by agile development framework? By wikipedia : "Agile software development is a group of software development methodologies based on iterative and incremental development, where requirements and solutions evolve through collaboration between self-organizing, cross-functional teams" So agile is a way how you manage your project, not a framework in which you work (you can develop in agile way in any language). Of course by agile you can understand sets of software development practices as: Unit testing TDD Solid principles which you can easier implemented using additional frameworks/tools
