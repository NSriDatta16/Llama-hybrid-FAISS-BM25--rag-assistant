Put it in a common location that you like in source control. Prior to using the targets, do a get from your TFSBuild.proj file like this: &quot;C:\Program Files\Microsoft Visual Studio 9.0\Common7\IDE\tf&quot; C:\YourPathHere &quot;$/YourProj/YourCustomProjPath&quot; The key to this working is to setup a Workspace for your build agent user that points to your custom stuff. This directory should NOT be in the same space as your normal builds. Vaccano
