In the jQuery UI droppable documentation it shows you that it provides additional jQuery objects: * ui.draggable - current draggable element, a jQuery object. * ui.helper - current draggable helper, a jQuery object * ui.position - current position of the draggable helper { top: , left: } * ui.offset - current absolute position of the draggable helper { top: , left: } So, changing your code as follows should work: $gallery.droppable({ accept: '#suitcase li', activeClass: 'custom-state-active', drop: function(ev, ui) { var $unid = ui.draggable.attr('id'); alert($unid); recycleImage(ui.draggable, $unid); } });
