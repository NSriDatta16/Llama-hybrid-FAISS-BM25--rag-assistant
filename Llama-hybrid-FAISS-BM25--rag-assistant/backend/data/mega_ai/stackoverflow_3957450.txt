If your XML file only contains ASCII characters it will be valid ASCII/ANSI as well as valid UTF8, so don't worry about Notepad++ recognizing the file as ANSI. While you can use a BOM for UTF8, it is discouraged because it will break a lot of Unix programs and you really shouldn't do it.
