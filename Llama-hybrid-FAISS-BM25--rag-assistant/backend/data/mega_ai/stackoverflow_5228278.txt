You can use Fragments now in the API 11. There's also support for the API 4 or greater, but I'm not sure if all features are supported. Take a look: http://developer.android.com/guide/topics/fundamentals/fragments.html http://developer.android.com/reference/android/app/Fragment.html For API 4: http://android-developers.blogspot.com/2011/03/fragments-for-all.html
