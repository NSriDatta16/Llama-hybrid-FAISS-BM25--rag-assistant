I have not heard of anything like that. It sounds like you have a model but that it is a black box and you want to be able to explain it to some audience. If this is the case I would: Try a linear model like linear regression. This will give you a formula that can be implemented. Try a ‘decision tree’ which will also give you a set of rules and can also output the importance of the inputs to the model. Keep your black box model, admit that it is black box, but use a method like PCA to demonstrate the importance of different variables (not model parameters) to the results.
