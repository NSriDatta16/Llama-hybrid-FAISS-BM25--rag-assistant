:visible is a selector and needs to be un quotes, like this: if ($("#BrokContent").is(":visible")) { $("#brokerageBanner").show(); $("#bannerTop").hide(); } else { $("#bannerTop").show(); $("#brokerageBanner").hide(); } Or a bit more concise using .toggle(bool) like this: var bcVisible = $("#BrokContent").is(":visible"); $("#brokerageBanner").toggle(bcVisible); $("#bannerTop").toggle(!bcVisible);
