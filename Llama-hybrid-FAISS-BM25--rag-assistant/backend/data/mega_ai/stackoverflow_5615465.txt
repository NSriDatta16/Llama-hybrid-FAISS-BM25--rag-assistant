If you're able to choose the delimiters, you might be able to use String.interpolate from the Facets gem: one_boy = "hello".upcase str = "\#{one_boy}!!!" String.interpolate{ str } #=> "HELLO!!!" But I'd first check that Facets doesn't cause any conflicts with Rails.
