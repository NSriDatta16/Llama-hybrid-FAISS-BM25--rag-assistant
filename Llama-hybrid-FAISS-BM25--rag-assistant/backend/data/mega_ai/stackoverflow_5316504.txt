If you look under \Common7\IDE\Templates\LoadTest\Browser you will find a bunch of .browser files. You can add to the list of browsers by createing a new .browser file with the appropriate useragent entries. You might be able to copy the ones from 2010 back to 2008, but I haven't tried it.
