Detecting local file drag'n'drop with HTML/JavaScript

There is a HTML textarea. I'm able to catch that event when a local file is dragged and dropped onto the textarea. But how to obtain the name of the dropped file? (To be modified and inserted into the textarea finally.) The following expressions returns None in that case: event.dataTransfer.files event.dataTransfer.getData('text/plain') I made a short example for Firefox 3 that is my target platform currently. function init() { document.getElementById('x').addEventListener('drop', onDrop, true) } function onDrop(event) { var data = event.dataTransfer.getData('text/plain') event.preventDefault() alert('files: ' + event.dataTransfer.files + ' && data: ' + data + '.') }
