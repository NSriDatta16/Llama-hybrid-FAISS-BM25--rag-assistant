We were able to figure out how to do this using an iterative Bayesian approach like so: $$ P[N \ge H_0 | X] = \frac{ \int_{\hat{x}}^{\infty}\prod_i P[x_i|N=H_0]dH_0 * P[N \ge H_0] }{ \int_{0}^{\hat{x}}\prod_i P[x_i|N=H_0]dH_0 * (1-P[N \ge H_0]) + \int_{\hat{x}}^{\infty}\prod_i P[x_i|N=H_0]dH_0 * P[N \ge H_0] } $$ Where $$ P[x|N=H_0]: \text{PDF of Neg Bin with parameters determined with Johnson-SB model} \\ P[N\ge H_0]: \text{prior probability} \\ P[N \ge H_0 | x]: \text{posterior probability} $$ Starting with a prior of 0.5, and then using the posterior of the last data point as the prior for the next data point.
