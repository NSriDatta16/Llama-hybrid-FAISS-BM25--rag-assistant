You can use XamlReader to read your Xaml string and convert it to a control: string templateString = " This text is blue and bold. "; StringReader stringReader = new StringReader(templateString); XmlReader xmlReader = XmlReader.Create(stringReader); Paragraph template = (Paragraph)XamlReader.Load(xmlReader); Just make sure you include the following tag in your template: xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" HTH
