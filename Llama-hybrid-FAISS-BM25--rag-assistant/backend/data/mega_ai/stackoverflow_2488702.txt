You cant't create Date object by passing string argument shows like date. Create Like this way. var str_date = "2010/03/21"; var date = new Date(); var date_elements = str_date.split("/"); date.setYear(date_elements[0]); date.setMonth(date_elements[1]); date.setDay(date_elements[2]); alert(date.toString); give attention on date.toString method. it returns full date String. date.getDate will only give you day of that day. for more info of Date object, refer www.w3school.com
