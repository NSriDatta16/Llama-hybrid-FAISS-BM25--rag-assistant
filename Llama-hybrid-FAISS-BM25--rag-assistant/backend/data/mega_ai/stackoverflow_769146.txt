maybe not the most efficient, but fairly intuitive is $('textarea').css('border', '1px solid green'); $('div.red textarea').css('border', '1px solid red'); although something like this may belong more in css if you are trying to apply it to every textarea at all times: textarea { border: 1px solid green; } div.red textarea { border: 1px solid red; } Edit: As Paolo Bergantino mentions, if you want to do something more complex than one css command on the textareas you can use var redAreas = $('div.red textarea'); var otherAreas = $('textarea').not(redAreas);
