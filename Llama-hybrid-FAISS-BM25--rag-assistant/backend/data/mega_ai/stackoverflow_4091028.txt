Instead of using imp , you can assign different names to imported modules. import mypackage_xxyyzzww as mypackage If you then create a __init__.py file inside of Storage , you can add several of the above lines to make importing easier. Storage/__init__.py : import mypackage_xxyyzzww as mypackage import otherpackage_xxyyzzww as otherpackage Interpreter: >>> from Storage import mypackage, otherpackage
