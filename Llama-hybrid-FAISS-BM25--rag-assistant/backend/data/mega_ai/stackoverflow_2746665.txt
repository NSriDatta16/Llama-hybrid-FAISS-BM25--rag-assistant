You need to declare the name-method virtual and the override it in the derived class. I also took the liberty of "fixing" your code with the most common naming conventions. abstract class FuzzyContainer : IContainer{ protected virtual string GetName(){ return "Fuzzy Container"; } } class SpecialContainer: FuzzyContainer{ override string GetName(){ return base.GetName() + " Special Container"; } }
