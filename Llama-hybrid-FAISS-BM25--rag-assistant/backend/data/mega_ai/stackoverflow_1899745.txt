You can get id by control.GetID() but you do not need to get ID to bind, you can just do this self.btn.Bind(wx.EVT_BUTTON, self._onBtnClick) It is preferable(id should be implementation detail) and short e.g. in this sample import wx app = wx.PySimpleApp() frame = wx.Frame(None) btn = wx.Button(frame, -1, "anurag") btn.Bind(wx.EVT_BUTTON, lambda e:wx.MessageBox("Wow it works")) frame.Show() app.SetTopWindow(frame) app.MainLoop()
