Further research: http://msdn.microsoft.com/en-us/library/cc295545.aspx Controls that are not compatible with UpdatePanel controls The following ASP.NET controls are not compatible with partial-page updates, and are therefore not designed to work inside an UpdatePanel control: GridView and DetailsView controls when their EnableSortingAndPagingCallbacks property is set to true. The default is false.
