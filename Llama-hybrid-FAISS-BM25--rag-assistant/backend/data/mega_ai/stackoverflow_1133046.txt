According to the Gant site, no: Gant is a tool for scripting Ant tasks using Groovy instead of XML to specify the logic. A Gant specification is a Groovy script ... A Gant build script uses Groovy script, not XML, but it uses the Ant tasks. Therefore if you have any custom Ant tasks you will still be able to use those.
