If you want do to this from the ground up, then all you need is the agile manifesto and recurring retrospectives each week. But I guess that is not enough, so here is my start-up-list: Convert your existing project tasks/points/todos into User Stories Pair program on everything. Switch pairs often! Use Test Driven Development. Strive for 100% coverage! Use one week iterations. Repetition is learning! Deliver valuable software to the customer in each iteration.
