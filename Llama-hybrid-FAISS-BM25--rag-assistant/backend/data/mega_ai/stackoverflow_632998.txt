Are you unsure about how to convert the input to numbers? If so use the CInt function. Public Sub OnAverageClick(ByVal sender as Object, ByVal e As EventArgs) Handles AverageButton.Click Dim input1 as Integer = CInt(textBox1.Text) Dim input2 as Integer = CInt(textBox2.Text) Dim input3 as Integer = CInt(textBox3.Text) Dim average = (input1 + input2 + input3) / 3 End Sub
