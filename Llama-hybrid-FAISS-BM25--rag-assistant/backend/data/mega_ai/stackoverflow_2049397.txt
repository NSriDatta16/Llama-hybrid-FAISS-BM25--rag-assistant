Using transform makes this compact and elegant: transform(a, Result = ifelse(Control, ifelse(Pf, ifelse(Pv, "Mixed", "Pf"), ifelse(Pv, "Pv", "Negative")), "Invalid")) Yields Control Pf Pv Result 1 TRUE TRUE FALSE Pf 2 TRUE FALSE TRUE Pv 3 FALSE FALSE FALSE Invalid 4 TRUE TRUE TRUE Mixed 5 TRUE FALSE FALSE Negative Alternatively, building on Marek's version we can use logical vectors to calculate the index slightly more compactly: a$Result = apply(a,1, function(x){ c(rep("Invalid", 4), "Negative", "Pv", "Pf", "Mixed") [1+sum(c(4,2,1)[x])]})
