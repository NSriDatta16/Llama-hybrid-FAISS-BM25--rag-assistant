I guess you use LSB storage order. Try this: procedure SetBit(const a,b: cardinal; var dest); var d: Int64 absolute dest; begin d := $F000000000+(Int64(a) shl 21)+b shl 6; end; It will change the 8 bytes value pointed by dest, storing the data into the first 5 bytes.
