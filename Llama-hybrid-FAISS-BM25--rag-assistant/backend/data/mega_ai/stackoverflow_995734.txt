If I understand correctly, you have a document where the markup looks something like this: paragraph 1 par1 div1 par1 div2 paragraph 2 par2 div1 par2 div2 paragraph 3 par3 div1 paragraph 4 par4 div1 And you want to end up with something like this: paragraph1 par1 div1 par1 div2 paragraph2 par2 div1 par2 div1> ..... If this is the case, then this jQuery snippet will do the trick: $(function(){ $('p').each(function(index,item){ var nextDiv = $(item).next(); var followingDiv = $(nextDiv).next('div'); $(item).wrap(" "); $("#fieldset"+index).append(nextDiv).append(followingDiv); }); });
