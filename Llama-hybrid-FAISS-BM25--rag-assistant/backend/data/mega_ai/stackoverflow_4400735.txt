You could check if the query string is equal to lang=en : RewriteCond %{HTTP_USER_AGENT} !(FeedBurner|FeedValidator) [NC] RewriteCond %{QUERY_STRING} !=lang=en RewriteRule ^feed/?$ http://feeds.feedburner.com/SimoBlog [L,NC,R=302] RewriteCond %{HTTP_USER_AGENT} !(FeedBurner|FeedValidator) [NC] RewriteCond %{QUERY_STRING} =lang=en RewriteRule ^feed/?$ http://feeds.feedburner.com/digitalking/haPl [L,NC,R=302]
