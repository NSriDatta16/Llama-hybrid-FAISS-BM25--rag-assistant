tinyMCE offers a method under DOMUtils which is tinymce.dom.DOMUtils/remove // Removes all paragraphs in the active editor tinyMCE.activeEditor.dom.remove(tinyMCE.activeEditor.dom.select('p')); // Removes a element by id in the document tinyMCE.DOM.remove('mydiv'); So in your case since you want to remove and what's inside then you should write something like : // Removes all paragraphs in the active editor tinyMCE.activeEditor.dom.remove(tinyMCE.activeEditor.dom.select('info'));
