You need to use the classes in the Microsoft.FSharp.Reflection namespace so: open Microsoft.FSharp.Reflection ///Returns the case name of the object with union type 'ty. let GetUnionCaseName (x:'a) = match FSharpValue.GetUnionFields(x, typeof ) with | case, _ -> case.Name ///Returns the case names of union type 'ty. let GetUnionCaseNames () = FSharpType.GetUnionCases(typeof ) |> Array.map (fun info -> info.Name) // Example type Beverage = | Coffee | Tea let t = Tea > val t : Beverage = Tea GetUnionCaseName(t) > val it : string = "Tea" GetUnionCaseNames () > val it : string array = [|"Coffee"; "Tea"|]
