you can design a custom header for your packet transmission, which specifies packet length, indexing info (if packet fragmentation is implemented) and some hashing if you need. some rough pseudocode as follows : recv(socket, headerBuf, headerSize, MSG_WAITALL); nPacketSize = headerBuf[16]; //sample nByteRead = 0; while (nByteRead != nPacketSize) { nByteToRead = nPacketSize - nByteRead; nCurRead = recv(socket, someBuf, nByteToRead, MSG_PARTIAL); nByteRead += nCurRead; Sleep(0); // Yield processor }
