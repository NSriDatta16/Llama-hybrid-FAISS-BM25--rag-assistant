You can scope the #pragma warning around that line of code only by using #pragma warning(push) #pragma warning(disable:XXXX) your code here; #pragma warning(pop) or #pragma warning(disable:XXXX) your code here; #pragma warning(default:XXXX) You can also use UNREFERENCED_PARAMETER(close_guard); after the line of code above.
