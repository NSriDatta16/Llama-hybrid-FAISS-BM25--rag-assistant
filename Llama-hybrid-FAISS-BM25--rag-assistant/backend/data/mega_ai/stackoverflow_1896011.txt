You can leverage the GridView's DataKeys property for this: In your code you can loop through the rows and read the data key for that row and get the checkbox like this: foreach (GridViewRow gvr in gvImages.Rows) { int CustomerID = (int)gvImages.DataKeys[gvr.RowIndex].Value; CheckBox CheckBox1 = (CheckBox)gvr.FindControl("CheckBox1"); }
