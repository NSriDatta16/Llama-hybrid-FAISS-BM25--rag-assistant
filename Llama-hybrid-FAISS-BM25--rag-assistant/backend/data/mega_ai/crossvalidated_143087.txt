If you want to compute the AUC for the combination of two variables, you can include both as predictors in a logistic regression model and compute the AUC using the predictions from the model. See some sample code: library(pROC) data(infert) auc(case ~ age, data=infert) #Compute AUC for predicting case with the variable age mod1
