I think those ones are better (and they look fine on retina display too) : .png generated from this very nice .psd file : http://www.teehanlax.com/blog/2010/08/12/iphone-4-gui-psd-retina-display/ And then use it as a strechable image for the background of your UIButton : UIImage* greenButton = [UIImage imageNamed:@"UIButton_green.png"]; UIImage *newImage = [greenButton stretchableImageWithLeftCapWidth:greenButton.size.width/2 topCapHeight:greenButton.size.height/2]; [callButton setBackgroundImage:newImage forState:UIControlStateNormal];
