You could use cURL to serve as a pass-thru for the content. This would conceal the source of the actual data, and allow you to secure it any way you choose. It would also take a lot of bandwidth, roughly 2 times the size of all downloaded files. Give this a shot and let me know if it works? if ($passesallyoursecurity) { set_time_limit(0); header("Pragma: public"); header("Expires: 0"); header("Cache-Control: must-revalidate, post-check=0, pre-check=0"); header("Cache-Control: private",false); header("Content-Type: application/download"); header("Content-Disposition: filename=filetheyget.ext"); $ch = curl_init("http://remotedomain.com/dir/file.ext"); curl_exec($ch); curl_close($ch); exit(); }
