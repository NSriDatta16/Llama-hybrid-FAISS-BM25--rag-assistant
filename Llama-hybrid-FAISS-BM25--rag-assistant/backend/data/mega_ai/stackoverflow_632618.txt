Problem is, you're trying to upcast to a richer object. You simply need to add the items to a new list: if (myObject is IEnumerable) { List list = new List (); var enumerator = ((IEnumerable) myObject).GetEnumerator(); while (enumerator.MoveNext()) { list.Add(enumerator.Current); } }
