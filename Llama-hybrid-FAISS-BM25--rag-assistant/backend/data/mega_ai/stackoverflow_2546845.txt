Try: $("#dialogApprove").dialog({ autoOpen: false, draggable: true, resizable: false, buttons: { "Cancel": function() { $(this).dialog("close") }, "Approve": function() { $.post($("#UserApprove").attr('action'), $("#UserApprove").serialize(), function(result) { // result will be your partial view $(this).dialog("close"); } ); } } }); This will post your form and return the partial view in the callback. `
