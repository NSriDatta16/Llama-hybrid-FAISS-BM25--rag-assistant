One option is to use array_slice() To show each element followed by a line break: echo implode(" ", array_slice($array, 0, 5)); not suitable for arrays containing huge amounts of data because the slice will be a copy, but elegant for normal everyday use. For a resource-conscious approach, see @Svisstack's answer (And now @Gordon's).
