Because you developed those two games, it makes using their data so much easier. You have a couple of options now. But for all of those options, I'd recommend enforcing a permission: Security and Permissions so that applications that you don't want accessing your data can't access it. Here are your options: SharedPreferences but using getSharedPreferences("filename", MODE_WORLD_READABLE); Internal Storage External Storage Database with a Content Provider
