There are a couple of ways you can do this. Dynamic File Linking or using the Files and Folders view. With Dynamic File linking you can point your project to a directory or directories and IS will refresh the list of files at build time and add them to your install. The files and folder view can be used to drag and drop a whole load of files and folders in to your project in one go. If you choose to use Dynamic File Linking be sure to read up on the implications first. Check out the Installshield help file
