Problem solved... when you want to use Telerik Components in your views, you need to register your scripts with script manager like this : group .Add("jquery-1.4.2.js") .Add("jquery.ui.core.js") .Add("jquery.ui.widget.js") .Add("jquery.ui.mouse.js") .Add("jquery.ui.draggable.js") .Add("jquery.ui.button.js") .Add("jquery.ui.resizable.js") .Add("jquery.ui.dialog.js") .Add("jquery.ui.position.js") ); %> Regards
