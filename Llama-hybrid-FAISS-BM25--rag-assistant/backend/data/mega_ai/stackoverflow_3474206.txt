Hope I understood it right. Basically you want a mapping from primitive class types to their wrapper methods. A static utility method implemented in some Utility class would be an elegant solution, because you would use the conversion like this: Class wrapper = convertToWrapper(int.class); Alternatively, declare and populate a static map: public final static Map , Class > map = new HashMap , Class >(); static { map.put(boolean.class, Boolean.class); map.put(byte.class, Byte.class); map.put(short.class, Short.class); map.put(char.class, Character.class); map.put(int.class, Integer.class); map.put(long.class, Long.class); map.put(float.class, Float.class); map.put(double.class, Double.class); } private Class clazz = map.get(int.class); // usage
