Alt + Cmd + Up to toggle between .h and .m files of a class Cmd + ' to move to next error or warning Cmd + Shift + ' to move to previous error or warning Column Selection : Holding down the option key while dragging allows you to perform a column based selection of text. Might be useful in limited circumstances.
