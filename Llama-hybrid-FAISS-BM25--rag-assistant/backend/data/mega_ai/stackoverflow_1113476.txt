Facebook.com MySQL with MyRocks . Used to store user info and social activities such as likes, comments, and shares. Hive (Data warehouse for Hadoop, supports tables and a variant of SQL called hiveQL). Used for "simple summarization jobs, business intelligence and machine learning and many other applications" Cassandra (Multi-dimensional, distributed key-value store). Currently used for Facebook's private messaging. Currently running 610 (soon to be 1000) Hadoop nodes in a single cluster with Hive datastore. Both Hive and Cassandra have been open-sourced by Facebook.
