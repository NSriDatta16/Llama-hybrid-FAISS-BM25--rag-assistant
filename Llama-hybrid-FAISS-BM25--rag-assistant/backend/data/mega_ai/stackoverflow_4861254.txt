Text in newer versions of IE should support ClearType, however some animations in jQuery and JavaScript cause dodgy opacity issues with ClearType. If you are talking about IE6 and Windows XP - could I suggest this: http://www.useragentman.com/blog/2009/11/29/how-to-detect-font-smoothing-using-javascript/ This blog is about detecting whether the client is using ClearType.
