Looking at the sample code it looks like you start coding in column 1. Trad mainframe compilers expect code in particular columns: ----+----1----+----- * comment star goes in column 7 PROCEDURE DIVISION. PARAGRAPH_NAME. * All start between cols 8 and 11 * ADD +1 TO COBVER. * Other operations start in col 12 or after. * Anything after column 71 will be ignored.
