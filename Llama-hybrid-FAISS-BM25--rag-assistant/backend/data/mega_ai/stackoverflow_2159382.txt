There is an internal LaTeX macro to perform this function called \@hangfrom . Here is an example: \documentclass{article} \usepackage{lipsum} \begin{document} \makeatletter \newcommand*\objectiveresheading[1]{% \@hangfrom{\Large\bfseries Objective: }% {\large\bfseries #1\par}% } \makeatother \objectiveresheading{\lipsum[1]} \end{document} Note that the wrapped text can only be a single paragraph. If you need multiple paragraphs, something more similar to a list environment would be better.
