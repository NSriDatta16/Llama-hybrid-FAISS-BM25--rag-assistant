You can't read a local file in your HTML page unless you use the FileAPI (using a fileReader) But it is only implemented in firefox 3.6+ yet. May be it will be also in chrome 5.0 Here are some articles on this subject and a demo You can also use OfflineAPI to store information about the file selection and DragDrop to extend user experience
