There gamlss package fits regressions to simultaneously estimate the conditional distributions of shape, scale, and location parameters for many sampling distributions, include several parameterizations of the beta. This package also allows for the fitting of varying intercept and varying slope models. While it is not a pre-packaged multi level Bayesian beta regression, it's pretty close.
