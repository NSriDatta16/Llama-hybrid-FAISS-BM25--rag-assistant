Since you asked about another storage system in a comment, you might want to look into SQLite. Even though SQLite behaves the same in appending to data first, it offers the vacuum command to recover unused storage space. From the Python API , you'll can either use the vacuum pragma to do it automatically, or you can just execute the vacuum command .
