memcached is a popular way to solve this problem. You just need to get it up and running (easy) and update your php.ini file to tell it to use memcached as the session storage. In php.ini you would modify: session.save_handler = memcache session.save_path = "" For the general idea: PHP Sessions in Memcached . There are any number of tutorials on setting up the Zend session handler to work with memcached. Take your pick.
