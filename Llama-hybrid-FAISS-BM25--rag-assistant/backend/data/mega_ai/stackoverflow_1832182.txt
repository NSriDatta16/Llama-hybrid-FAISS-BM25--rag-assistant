For instance, very often training a neural network is supervised learning: you're telling the network to which class corresponds the feature vector you're feeding. Clustering is unsupervised learning: you let the algorithm decide how to group samples into classes that share common properties. Another example of unsupervised learning is Kohonen's self organizing maps .
