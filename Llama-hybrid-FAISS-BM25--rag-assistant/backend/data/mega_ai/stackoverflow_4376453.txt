I think you cannot modify an existing dialog in this way. To accomplish the task you're faced with just include the button in the first call, but immediately hide it. Once the ajax request has finished just unhide the button. For a proper visual experience just use hide/unhide the whole button pane: var $dialog = jQuery(' Wait ') .html('Sending your message... ') .dialog({ modal: true, width: 160, autoOpen: false, resizable: false, draggable: false, buttons: { "Ok": function() { jQuery(this).dialog('close'); } }) ; // hide buttons $dialog.children('.ui-dialog-buttonpane').hide(); In the event handler for ajax complete just do // unhide buttons $dialog.children('.ui-dialog-buttonpane').show(); Optionally you may provide an easing identifier to animate the unhiding process.
