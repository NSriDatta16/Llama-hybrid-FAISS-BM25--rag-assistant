You should never use string manipulation to create XML. If you use the XML APIs of .NET, they will take care of all the special characters for you. Try: XmlElement linkElement = doc.CreateElement("Link"); XmlAttribute urlAttribute = doc.CreateAttribute("URL"); urlAttribute.Value = token; linkElement.SetAttributeNode(urlAttribute); fragment.AppendChild(linkElement);
