You can use CommandBinding, and inside that binding, you have options to handle EditingCommands such as increasing font size, aligning text, and etc. This would let you handle your own implementation instead of the default behavior. This way, you can say, well if MoveDownByParagraph was triggered by the agent, then, do this instead. Examples: http://msdn.microsoft.com/en-us/library/ms752308.aspx http://www.switchonthecode.com/tutorials/wpf-tutorial-command-bindings-and-custom-commands
