Looks to be duplicate of GLSL convert gl_FragCoord.z into eye-space z . Edit (complete answer): // input: x_coord, y_coord, samplerDepth vec2 xy = vec2(x_coord,y_coord); //in [0,1] range vec4 v_screen = vec4(xy, texture(samplerDepth,xy), 1.0 ); vec4 v_homo = inverse(gl_ProjectionMatrix) * 2.0*(v_screen-vec4(0.5)); vec3 v_eye = v_homo.xyz / v_homo.w; //transfer from homogeneous coordinates
