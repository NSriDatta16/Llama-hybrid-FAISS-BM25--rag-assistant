Maybe you can try running a principal components analysis (PCA) first for you data set, and then use these components as variables to build your tree. Therefore, at each split, the tree algorithm will be selecting from specific combinations of your original data. PCA will build components that describe features present in your data, such as contrasts between variables, overall size, ...
