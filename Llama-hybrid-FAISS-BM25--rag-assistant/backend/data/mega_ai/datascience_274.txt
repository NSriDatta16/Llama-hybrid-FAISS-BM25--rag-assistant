One benefit is wide support - C, Java, Perl, Python, and R all have HDF5 bindings. Another benefit is speed. I haven't ever seen it benchmarked, but HDF is supposed to be faster than SQL databases. I understand that it is very good when used with both large sets of scientific data and time series data - network monitoring, usage tracking, etc. I don't believe there is a size limitation for HDF files (although OS limits would still apply.
