You're looking for the vacuum_indexes command. http://code.google.com/appengine/docs/python/tools/uploadinganapp.html#Deleting_Unused_Indexes This will work for both Java and Python apps. In fact, Java's appcfg.sh does not yet support this at all, so you'll need the Python SDK if you're running a Java app.
