very helpful thread. I used this to hide the declaration for IE6. Turns out IE6 only checks the first line of the document for doctype sniffing, which means that if the string is present, IE6 will render in quirks mode regardless of the doctype following the declaration I'm not sure if its critical to include an opera check yet, but have included it anyway. //check if IE 6 or less $not_lte_ie6 = true; $ua = strtolower($_SERVER['HTTP_USER_AGENT']); if(strpos($ua,'msie') !== FALSE) { if(strpos($ua,'opera') === FALSE) { if(preg_match('/(?i)msie [1-6]/',$ua)) $not_lte_ie6 = false; } } if ($not_lte_ie6) echo " \n";
