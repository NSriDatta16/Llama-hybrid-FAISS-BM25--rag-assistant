This is often called a Template-Hook pair (a.k.a Hotspot), coined by Wolfgang Pree. See this PDF , PowerPoint , HTML One reason for doing the indirection as you call it is that things often can/has to be setup prior a method, and some cleaing post a method call. In the subclasses you only need to supply the necessary behaviour without doing the setup and cleaning ...
