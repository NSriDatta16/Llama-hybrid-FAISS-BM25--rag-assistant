You could return a single JSON string parameter and parse it on the server within your webmethod. This is just one approach. Another might be to take a look at your overall design and think about how you might improve it so that you don't have to pass variable parameters around. they can be nasty, as you have found. Often, stepping back and refactoring can lead to a simpler, more elegant design that is a lot easier to program.
