Just wrap your tokenization/POS-tagging pipeline in a singleton . If the underlying OpenNLP code isn't thread safe, put the calls in synchronization blocks, e.g.: // the singletons tokenization/POS-tagging pipeline String[] tokens; synchronized(tokenizer) { tokens = tokenizer.tokenize(text); } String[] tags; synchronized(posTagger) { tags = posTagger.tag(tokens); }
