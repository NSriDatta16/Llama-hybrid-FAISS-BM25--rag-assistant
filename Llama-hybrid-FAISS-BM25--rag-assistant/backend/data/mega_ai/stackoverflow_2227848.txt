Try this post . This will help you set the company name attribute permanently via registry. Answer: Set the value of the registry key for HKLM\Software\Microsoft\Windows NT\CurrentVersion\RegisteredOrganization For 64 bit OS, the registry key is HKLM\Software\Wow6432Node\Microsoft\Windows NT\CurrentVersion\RegisteredOrganization
