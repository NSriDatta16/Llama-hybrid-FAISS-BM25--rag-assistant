The partial effect is defined as $$\frac{\partial f(x_t \theta)}{\partial x_{tk}}$$ so the average partial effect is simply $$\frac{1}{T} \sum_t \frac{\partial f(x_t, \theta)}{\partial x_{tk}}$$ calculated using the estimate $\hat \theta$ .
