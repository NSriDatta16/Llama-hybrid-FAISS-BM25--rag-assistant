MD5 is not reversible, and you shouldn't autofill a password field for security reasons. It's showing longer because MD5ed strings are always 32 characters. edit: You also shouldn't use a reversible password storage method. Please see Jeff Atwood's "You're Probably Storing Passwords Incorrectly" post for details.
