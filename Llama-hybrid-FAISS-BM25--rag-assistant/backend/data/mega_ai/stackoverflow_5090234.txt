Well, if it's a different view for mobile devices but all desktop versions are the same then you could use JQtouch. http://railscasts.com/episodes/199-mobile-devices # config/initializers/mime_types.rb Mime::Type.register_alias "text/html", :mobile # application_controller.rb before_filter :prepare_for_mobile private def mobile_device? if session[:mobile_param] session[:mobile_param] == "1" else request.user_agent =~ /Mobile|webOS/ end end helper_method :mobile_device? def prepare_for_mobile session[:mobile_param] = params[:mobile] if params[:mobile] request.format = :mobile if mobile_device? end The above code is taken from the Railscasts example.
