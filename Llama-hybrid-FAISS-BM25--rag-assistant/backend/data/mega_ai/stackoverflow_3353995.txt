This isn't elegant by any means, but it certainly works: Add a debug sampler into your test plan, and give it the same name as your regex reference. This will write out the time value into the results file. Example if you have different pages: regex reference = X-Execution-Time Debug Sampler Name = PageName - Execution: ${X-Execution-Time}
