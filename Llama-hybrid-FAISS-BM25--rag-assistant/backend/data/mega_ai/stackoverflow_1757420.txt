It sounds like the relatively benign undocumented UIAlertView API's aren't being allowed anymore, so whether you're using a text field or a text label, you're going to need to use a different mechanism. I'd recommend the following blog posts are good examples of how to address this issue: Custom UIAlertView (Color chooser) Embedding UITables into UIAlertViews Alert View with Prompt These techniques appear to be legitimate from an App Store perspective and address how to embed any sorts of controls into alert views.
