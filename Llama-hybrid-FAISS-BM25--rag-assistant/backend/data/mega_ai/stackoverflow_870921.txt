In your server explorer, drag your stored procedure into your DBML file. Once this is done, you can create an instance of your DataContext object and call that stored procedure directly as if it were a class level method. For example: MyDataContext db = new MyDataContext(); var storedProcedureResultSet = db.NameOfMyStoredProcedure(parameter);
