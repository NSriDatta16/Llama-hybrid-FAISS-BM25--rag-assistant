Oracle: SELECT ... START WITH ... CONNECT BY Oracle has an extension to SELECT that allows easy tree-based retrieval. Perhaps SQL Server has some similar extension? This query will traverse a table where the nesting relationship is stored in parent and child columns. select * from my_table start with parent = :TOP connect by prior child = parent; http://www.adp-gmbh.ch/ora/sql/connect_by.html
