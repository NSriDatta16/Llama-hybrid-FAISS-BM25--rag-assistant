try it this way: $device = ''; iOSDetect(); if($device == 'default') { // Do something } else { /* Do something else */ } Or, you can use a better way: function iOSDetect() { global $device; $browser = $_SERVER['HTTP_USER_AGENT']; if(strstr($browser, 'iPhone') || strstr($browser, 'iPod')) { $device = 'iPhone'; } else { $device = 'default'; } return $device; } and then, use it this way: if(iOSDetect() == 'default') { // Do something } else { /* Do something else */ }
