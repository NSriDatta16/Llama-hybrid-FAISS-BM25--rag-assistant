It's CS1591. Just press F1 on the warning location to view MSDN documentation for it. You can resolve this specific warning either by: Reducing warning level (it's a level 4 warning) Removing /doc switch (which will stop XML file generation) Disable the warning using #pragma directive Disable the warning using /nowarn:1591 switch
