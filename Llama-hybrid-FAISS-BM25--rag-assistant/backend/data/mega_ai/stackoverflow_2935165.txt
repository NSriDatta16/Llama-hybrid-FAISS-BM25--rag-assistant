A good solution in this case might be js-build-tools which supports 'conditional compilation'. In short you can use comments such as // #ifdef debug var trace = debug.getTracer("easyXDM.Rpc"); trace("constructor"); // #endif where you define a pragma such as debug . Then when building it (it has an ant-task) //this file will not have the debug code //this file will
