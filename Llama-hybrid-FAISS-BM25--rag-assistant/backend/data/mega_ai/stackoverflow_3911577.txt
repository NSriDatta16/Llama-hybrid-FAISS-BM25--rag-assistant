IF you are using TLF TextField and events are attached to TextField and a is the name of the TextField then try this code import flash.events.MouseEvent; a.addEventListener(MouseEvent.MOUSE_DOWN , startdrag); stage.addEventListener(MouseEvent.MOUSE_UP, stopdrag); function startdrag(evt:MouseEvent):void { a.startDrag(true); } function stopdrag(evt:MouseEvent):void { a.stopDrag(); }
