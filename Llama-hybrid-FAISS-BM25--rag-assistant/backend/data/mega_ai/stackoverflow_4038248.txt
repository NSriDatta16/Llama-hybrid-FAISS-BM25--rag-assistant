When you navigate to next page the previous page is destroyed (if it does not run the background thread). You have a sevral ways to display settings on page nr 1. When user log in and go to page 2 save your setting to the Isolated Storage and when he presses the back button use protected override void OnNavigatedTo(System.Windows.Navigation.NavigationEventArgs e) { var settings = LoadMySettingsFromIS(); if (settings =! null) { // update it here } base.OnNavigatedTo(e); }
