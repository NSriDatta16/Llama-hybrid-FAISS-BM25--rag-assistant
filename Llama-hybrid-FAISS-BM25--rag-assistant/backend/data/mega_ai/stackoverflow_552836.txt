Even simpler than Mitch Wheat's answer, since the fragment in question is a well-formed XML document: using System.Xml; using System.IO; ... string xml = @" stockholm sweden 51.1 67.98 "; XmlDocument d = new XmlDocument(); d.Load(new StringReader(xml)); Console.WriteLine(d.SelectSingleNode("/poi/gpoint/lat").InnerText); Console.WriteLine(d.SelectSingleNode("/poi/gpoint/lng").InnerText);
