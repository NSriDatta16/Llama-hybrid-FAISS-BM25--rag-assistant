Magento is technically Open Source. However, it does not embrace the concept to its fullest and include open/transparent development. All development and direction is done behind closed doors. They release snapshots of the code at their leisure and have zero feedback on bug fixes. We've been trying to get some sort of dialog going for core development fixes and additions, but nothing has happened so far.
