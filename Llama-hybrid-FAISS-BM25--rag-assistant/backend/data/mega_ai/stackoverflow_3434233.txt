Here's an 'always on' version that fixes as you type. It will let you type all uppercase words, but as soon as it detects mixed case it will capitalize. (defun blah (s e l) (let ((letter (string-to-char (word-before-point)))) (if (and (eq letter (upcase letter)) (not (eq (char-before) (upcase (char-before))))) (capitalize-word -1)))) (add-to-list 'after-change-functions 'blah)
