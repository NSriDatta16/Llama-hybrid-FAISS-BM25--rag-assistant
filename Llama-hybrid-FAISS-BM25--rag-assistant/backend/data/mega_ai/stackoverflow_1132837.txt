Would something like this work? get_git_dirty() { set __MT_OK__=0 if [[ -d .git ]]; then __MT_OK__=1 fi while [[ ! -d .git ]]; do cd .. if [[ -d .git ]]; then __MT_OK__=1 break fi if [[ $PWD = "/" ]]; then break fi done if [[ __MT_OK__ -eq 1 ]]; then git diff --quiet || echo '*' fi } This may not be the most elegant solution, but it should work.
