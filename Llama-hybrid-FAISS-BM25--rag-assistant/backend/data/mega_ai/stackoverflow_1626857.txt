You can trick it. If you are running PostGRESQL on a UNIXlike system, then you should be able to create a RAMDISK and use that for the database storage. Here's a pretty good step by step guide for RAMdisks on Linux . In general though, I would suggest using SQLITE for an SQL db in RAM type of application.
