If your config file can be turned into a python file, just make it a dictionary and import the module. Job = { "Name" : "host.domain.com-foo", "Client" : "host.domain.com-fd", "JobDefs" : "DefaultJob", "FileSet" : "local", "Write BootStrap" : "/etc/foo/host.domain.com-foo.bsr", "Pool" : "storage-disk1" } You can access the options by simply calling Job["Name"]..etc. The ConfigParser is easy to use as well. You can create a text file that looks like this: [Job] Name=host.domain.com-foo Client=host.domain.com-fd JobDefs=DefaultJob FileSet=local Write BootStrap=/etc/foo/host.domain.com-foo.bsr Pool=storage-disk1 Just keep it simple like one of the above.
