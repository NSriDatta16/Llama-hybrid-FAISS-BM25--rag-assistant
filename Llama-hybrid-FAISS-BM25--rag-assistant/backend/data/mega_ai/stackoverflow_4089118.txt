This is what I would do. Have the all the game logic done on the client side. Running an opponent AI on your server would be very costly. Save the state using window.localStorage (perhaps have a fallback using ajax for older browsers) Send off the steps to your server to validate the moves, log a win etc. This isn't the only way and may not be the best for your situation.
