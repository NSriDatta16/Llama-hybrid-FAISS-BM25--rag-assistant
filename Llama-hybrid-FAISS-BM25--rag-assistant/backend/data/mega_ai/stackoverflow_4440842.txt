Test if a variable is read-only

To test if a variable is read-only, there are the following ugly hacks: # True if readonly readonly -p | egrep "declare -[:lower:]+ ${var}=" # False if readonly temp="$var"; eval $var=x 2>/dev/null && eval $var=\$temp Is there a more elegant solution?
