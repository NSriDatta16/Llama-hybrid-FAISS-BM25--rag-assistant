A possible way would be to normalize the RMSE with the standard deviation of $Y$: $NRMSE = \frac{RMSE}{\sigma(Y)}$ If this value is larger than 1, you'd obtain a better model by simply generating a random time series of the same mean and standard deviation as $Y$.
