Yes, loading the file locally means that it doesn't have an origin. Since localStorage is uses the same-origin policy to determine access to stored data, it is undefined what happens when you use it with local files, and likely that it won't be persisted. You will need to host your file on a web server in order to have a proper origin; you can just run Apache or any other server locally and access it via localhost .
