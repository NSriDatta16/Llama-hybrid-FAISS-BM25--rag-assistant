You can use the builtin csv module: import csv csvReader = csv.reader(open('input.txt'), delimiter=' ') headers = csvReader.next() values = [map(int, row) for row in csvReader] def average(l): return float(sum(l)) / len(l) averages = [int(round(average(trial))) for trial in zip(*values)] print ' '.join(headers) print ' '.join(str(x) for x in averages) Result: Trial1 Trial2 Trial3 1 1 1
