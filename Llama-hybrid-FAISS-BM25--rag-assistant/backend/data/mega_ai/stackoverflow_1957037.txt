parse_url() can get the hash from a url string. Note the signature: mixed parse_url ( string $url [, int $component = -1 ] ) You must already know the string. For instance, from the docs: Outputs Array ( [scheme] => http [host] => hostname [user] => username [pass] => password [path] => /path [query] => arg=value [fragment] => anchor ) /path Note the entry under key "fragment".
