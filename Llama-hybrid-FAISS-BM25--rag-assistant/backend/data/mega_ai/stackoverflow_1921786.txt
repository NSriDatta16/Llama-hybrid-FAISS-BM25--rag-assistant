You can use a function like this: def numberToPrettyString(n): """Converts a number to a nicely formatted string. Example: 6874 => '6,874'.""" l = [] for i, c in enumerate(str(n)[::-1]): if i%3==0 and i!=0: l += ',' l += c return "".join(l[::-1])
