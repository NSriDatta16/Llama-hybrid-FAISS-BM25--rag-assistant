Try using FontSelector. You can add multiple fonts and it will process your text and use the different fonts where appropriate. FontSelector selector = new FontSelector(); selector.addFont(FontFactory.getFont(FontFactory.TIMES_ROMAN, 12)); selector.addFont(FontFactory.getFont("MSung-Light", "UniCNS-UCS2-H", BaseFont.NOT_EMBEDDED)); Phrase ph = selector.process(TEXT); document.add(new Paragraph(ph)); from http://itextpdf.com/examples/iia.php?id=214
