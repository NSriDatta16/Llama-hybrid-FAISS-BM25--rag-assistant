Not sure if this is the most elegant solution but you could do a case statement with your @offset like this: SET @offset = CASE WHEN DATEPART(weekday, @today) >= 2 THEN -(DATEPART(weekday, @today) - 2) ELSE -(DATEPART(weekday, @today) + 5) END I believe it works for all cases.
