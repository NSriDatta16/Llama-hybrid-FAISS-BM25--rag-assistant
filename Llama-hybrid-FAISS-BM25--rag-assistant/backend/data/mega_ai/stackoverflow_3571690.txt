I'm not sure if this is the best way, but I got it to work using SqlProjection: Session.CreateCriteria ("foo") .CreateAlias("foo.Bar", "bar") .SetProjection(Projections.SqlProjection("{alias}.*", new string[] {}, new IType[] {})) .SetResultTransformer(new AliasToBeanResultTransformer(typeof(Foo))) .List (); This produces the following SQL: SELECT this_.* FROM Foo this_ inner join Bar b1_ on this_.BarId=b1_.Id
