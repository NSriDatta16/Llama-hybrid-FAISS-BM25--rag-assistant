Alright, here's my solution. Not terribly elegant, but not a terrible hack either. def foo(self): self.subplot.clear() hb = self.subplot.hexbin(...) if self.cb: self.figure.delaxes(self.figure.axes[1]) self.figure.subplots_adjust(right=0.90) #default right padding self.cb = self.figure.colorbar(hb) This works for my needs since I only ever have a single subplot. People who run into the same problem when using multiple subplots or when drawing the colorbar in a different position will need to tweak.
