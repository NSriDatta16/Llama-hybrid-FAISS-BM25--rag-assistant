You can disable specific warnings using the #pragma directive, this might suit your purpose. The syntax for disabling the 'missing xml comment' warning is: #pragma warning disable 1591 // Disable 'missing xml comment' warning You can re-enable it using the following syntax: #pragma warning restore 1591 // Restore 'missing xml comment' warning This allows you to selectively ignore warnings for either part of a file or a complete file.
