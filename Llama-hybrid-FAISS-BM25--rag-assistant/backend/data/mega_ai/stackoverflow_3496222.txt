Smooth the data using a lowpass filter (or average each sample with a number of surrounding samples). Find the peak in the center by looking for the highest sample value. find the valley to the right of the peak by searching for the first sample that has a higher value than its predecessor. Find the peak to the right of the valley by searching for the first sample to a smaller value than its predecessor.
