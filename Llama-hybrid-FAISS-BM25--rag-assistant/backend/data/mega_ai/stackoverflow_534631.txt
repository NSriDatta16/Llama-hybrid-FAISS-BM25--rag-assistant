Use inheritance to centralise your own customisations e.g. Add a class called MyButton, say: using System; using System.Drawing; using System.Windows.Forms; namespace MyApp { class MyButton : Button { public MyButton() { this.BackColor = Color.Pink; this.Font = new Font(SystemFonts.DialogFont.FontFamily, 12, FontStyle.Bold, GraphicsUnit.Point); } protected override Size DefaultSize { get { return new Size(150, 50); } } } } And it will appear as a control in the Toolbox that you can drag and drop onto the Form instead of the regular button.
