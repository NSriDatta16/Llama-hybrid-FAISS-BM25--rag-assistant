You can use Activator.CreateInstance() (which allows you to pass in arguments) to create an instance of TException . Then, you could throw the created TException . For example: private void LogAndThrow (string message, params object[] args) where TException : Exception, new() { message = string.Format(message, args); Logger.Error(message); TException exception = (TException)Activator.CreateInstance(typeof(TException), message); throw exception; }
