Why write something that has already been built and is available to you with the product you already have installed? You should look into Log Shipping over your DSL line. It is a very reliable SQL Server product and has been tested extensively. http://msdn.microsoft.com/en-us/library/ms187103.aspx Although the express edition does not have SQL Server Agent, there are workarounds that you can use: http://blog.willbeattie.net/2009/07/log-shipping-in-sql-server-express-2008.html
