It's not mandatory. Removing stopwords can sometimes help and sometimes not. You should try both. A case for not using stopwords : Using stopwords will provide context to the user's intent. So when you use a contextual model like BERT, all stopwords are kept to provide enough context information like the negation words (not, nor, never) which are considered to be stopwords. According to this paper : Surprisingly, the stopwords received as much attention as non-stop words, but removing them has no effect in MRR performances.
