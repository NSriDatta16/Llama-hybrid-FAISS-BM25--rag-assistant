It returns an Entity object, so you'll need to call ToEntity() if you want a System User object. The following should work for you: var context = new OrganizationServiceContext(crmService); SystemUser x = (from c in context.CreateQuery () where (string)c["DomainName"] == @"pfgc\" + Environment.UserName select c).FirstOrDefault().ToEntity ();
