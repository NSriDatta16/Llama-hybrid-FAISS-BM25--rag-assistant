In a more C-like pseudo code this could be rewritten as: NSInteger returnNumberOfRowsInTableViewSelection(UITableView* tableView, NSInteger section) { ... } Contrast with a similar function using simple types: int add(int a, int b) NSInteger is the return type, tableView and section are the parameters. You may find the named parameter syntax in ObjC longwinded and confusing at first but in my opinion it leads to much more readable and maintainable code :)
