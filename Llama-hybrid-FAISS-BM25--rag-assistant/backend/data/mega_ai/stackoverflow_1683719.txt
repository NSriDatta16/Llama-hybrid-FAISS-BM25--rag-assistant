You need to pass an appropriate user-agent value to fool Facebook (e.g. "Mozilla 4") via the -A option, and also instruct curl to follow redirects via the -L option: $ curl -A 'Mozilla 4' -L "http://www.facebook.com/feeds/notifications.php?id=..." ...
