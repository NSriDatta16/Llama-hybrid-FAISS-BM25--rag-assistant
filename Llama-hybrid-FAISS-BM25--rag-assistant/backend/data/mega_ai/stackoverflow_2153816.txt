This is going to be painful, but to really do this correctly for production pushes you should drop vsewss 1.2, re-organize your projects inside of Visual Studio and use WSPBuilder. WSPBuilder is great because it takes a lot of the manual work out of creating manifest, ddf and compiling the CAB.
