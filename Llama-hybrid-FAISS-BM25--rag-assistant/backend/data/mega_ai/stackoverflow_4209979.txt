Simply change your query to be var query = from size in ( from catalogItem in cardigan.Descendants("catalog_item") where catalogItem.Attribute("gender").Value == "Men's" select catalogItem.Descendants("size") ) where size.Attribute("description") == "Large" select size.Elements("color_swatch"); This will work because you first get all "Size" items that apply for Men, then from that list it filters those only grabbing the "Large" ones.
