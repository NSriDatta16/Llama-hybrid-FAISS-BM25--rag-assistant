What happens if you set the size of your width property to be the width of the next label? Something like (in your scroll view delegate) : - (void)scrollViewDidEndDragging:(UIScrollView *)scrollView willDecelerate:(BOOL)decelerate { // Get the label that's going to slide into view UIView *label = [self getCurrentLabel]; // Get it's width float width = [label frame].size.width; // Set our size CGRect frame = [scrollView frame]; frame.size.width = width; [scrollView setFrame:frame]; } PS I've not tried this so it might just fail horribly!
