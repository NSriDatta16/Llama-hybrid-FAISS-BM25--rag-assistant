calling a cdef in a cdef class

is their any way to make this work, without sacrificing the cdef in cdef caller? (no use of cpdef either) from array import * from numpy import * cdef class Agents: cdef public caller(self): print "caller" A[1].called() cdef called(self): print "called" A = [Agents() for i in range(2)] def main(): A[0].caller()
