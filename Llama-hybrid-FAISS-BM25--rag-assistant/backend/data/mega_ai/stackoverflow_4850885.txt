This is a good practice when defining array on multiple lines. It's also encouraged by ZendFramework's coding standards : When using this latter declaration, we encourage using a trailing comma for the last item in the array; this minimizes the impact of adding new items on successive lines, and helps to ensure no parse errors occur due to a missing comma.
