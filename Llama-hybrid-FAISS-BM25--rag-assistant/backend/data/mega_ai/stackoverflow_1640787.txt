You can either post them on the page before you include other scripts, or use something like this and include the php file like a js file: ob_start(); //all variables here echo 'var someJson = ' . json_enode($someArray); $content = ob_get_clean(); header('Content-Type: text/javascript'); header('Content-Length: ' . strlen($content)); //or mb_strlen with utf8 echo $content; exit; You can add headers to provide caching with etag (hashing/id), last modified or expire/pragma.
