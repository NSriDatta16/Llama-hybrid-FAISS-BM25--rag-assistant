Yes, you can make the same device appear as two (or more) logical devices. I did this for a project a couple of years ago, but it's been a while now and I've forgotten the details. I think I had mass storage, video capture, and serial port all running simultaneously. Yes, Composite device sounds familiar. The project I worked on didn't use any third party OS or USB libraries; it was all "bare metal" code for the Freescale i.MX21 CPU. Jan Axelson's USB Complete was invaluable, as was the Packet-Master USB analyser.
