You could try using the HttpWebRequestClass , or WebClient . Here's the slightly complicated web request example. It's advantage over WebClient is it gives you more control over how the request is made: HttpWebRequest httpRequest = (HttpWebRequest) WebRequest.Create(lcUrl); httpRequest.Timeout = 10000; // 10 secs httpRequest.UserAgent = "Code Sample Web Client"; HttpWebResponse webResponse = (HttpWebResponse) httpRequest.GetResponse(); StreamReader responseStream = new StreamReader(webResponse.GetResponseStream()); string content = responseStream.ReadToEnd();
