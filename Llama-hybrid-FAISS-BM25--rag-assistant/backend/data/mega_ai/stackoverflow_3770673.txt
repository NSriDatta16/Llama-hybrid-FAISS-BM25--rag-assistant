cat /proc/stat you will see something like this cpu 178877 11039 58012 5027374 22025 2616 1298 0 0 cpu0 122532 8808 34213 2438147 10881 1050 448 0 0 cpu1 56344 2230 23799 2589227 11143 1565 850 0 0 Simply take the sums of first three numbers and divide them with sums of first four integer The first 4 numbers are user, nice, system, and idle times note: This gives overall average. If you want to take spontaneous average, you should take two samples and subtract them from each other before the divide.
