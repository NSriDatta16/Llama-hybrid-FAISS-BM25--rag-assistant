Not as elegant as a simple re.sub() , but try this: import re s='a.b.c. test@test.com http://www.test.com' m=re.search('(.*?)(([a-zA-Z]\.){2,})(.*)', s) if m: replacement=''.join(m.group(2).split('.')) s=m.group(1)+replacement+m.group(4) print s It assumes that there's no more than one acronym per string, but you could always run it repeatedly.
