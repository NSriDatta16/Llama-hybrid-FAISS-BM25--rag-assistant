1) Specify this option on remote JVM. -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005 Note that now the debug server is running at this remote host and port 5005. 2) In Eclipse, create and run remote debug config under Run > Debug> Debug Configurations... > Remote Java Application .
