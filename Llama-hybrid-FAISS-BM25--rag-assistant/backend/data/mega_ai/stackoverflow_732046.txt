For Eclipse Ganymede: You can use the following menu to disable HTML validation all together or parts of it by project. Right Click on the project | Project Properties | Validation | Enable Project Specific Settings. Then you can disable HTML Validation. Or, from Project Properties, click open Validation and select HTML Validation. Select Enable Project Specific Settings and set any of the ten element checks to "Ignore." You can also turn these settings off for the entire workspace using the Windows (Eclipse) Preferences | Validation settings.
