I strongly encourage you to use Dalli . config.cache_store = :dalli_store, 'cache-1.example.com', 'cache-2.example.com', { :namespace => NAME_OF_RAILS_APP, :expires_in => 1.day, :compress => true, :compress_threshold => 64*1024 } Otherwise, use the :memcached_store option to configure Memcached. config.action_controller.cache_store = :memcached_store To debug the error, open a new Rails console # rails 3 $ rails c # rails 2 $ ruby script/console and try to access Rails.cache . See if it works.
