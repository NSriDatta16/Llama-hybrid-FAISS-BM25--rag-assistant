There's no "easy" way to ensure that functionality hasn't been changed. You have to have complete unit tests that cover all possibilities. It's impossible to test absolutely everything but you can make sure that your most important user cases have thorough tests. You can also use a coverage tool to ensure you have good test coverage: http://covtool.sourceforge.net/
