It's a kind of scripting language for defining data format, something similar to XDR . You just have to write a parser for it (don't try to use the script in the runtime). Write some functions like get_WORD_BE() or get_DWORD_LE() , etc so that you don't depend on endiannes. Yes, and if you want to use Tony's approach, add some #pragma pack(1) .
