You should make sure that your query/criteria uses a 'resultransformer'. In this case, you'll need the DistinctRootEntityResultTransformer. query.SetResultTransformer(Transformers.DistinctRootEntity); Next to that, how is that association mapped ? Is it mapped as a bag, or as a set ?
