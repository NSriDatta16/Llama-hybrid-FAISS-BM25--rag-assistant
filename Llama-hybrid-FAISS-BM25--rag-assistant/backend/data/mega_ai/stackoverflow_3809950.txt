16.6/1- "A preprocessing directive of the form # pragma pp-tokensopt new-line causes the implementation to behave in an implementation-defined manner. Any pragma that is not recognized by the implementation is ignored." An implemention defined behavior is supposed to be documented by the vendor. So you need to look into the documentation.
