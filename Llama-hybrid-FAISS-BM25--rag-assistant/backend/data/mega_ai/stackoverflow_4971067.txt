I would use 2 stylesheets to theme the page, and detect the browser and serve up the right css depending on what the browser is detected as. Have a look at the 'get_browser' function of PHP http://php.net/manual/en/function.get-browser.php then just have an if statement: if (preg_match('/mobile/i', $u_agent)) { $css = 'mobile.css'; } and then just load the $css Hope that makes some sense.
