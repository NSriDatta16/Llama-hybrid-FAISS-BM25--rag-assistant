ElementTree is a good module for reading xml and writing too e.g. from xml.etree.ElementTree import Element, SubElement, tostring root = Element('root') child = SubElement(root, "child") child.text = "I am a child" print(tostring(root)) Output: I am a child See this tutorial for more details and how to pretty print. Alternatively if your XML is simple, do not underestimate the power of string formatting :) xmlTemplate = """ %(name)s %(address)s """ data = {'name':'anurag', 'address':'Pune, india'} print xmlTemplate%data Output: anurag Pune, india You can use string.Template or some template engine too, for complex formatting.
