I think there is a better way than writing : criteria .CreateCriteria("Operation") // add other expressions .CreateCriteria("OperationTypes", "operationTypes", JoinType.LeftOuterJoin) .Add(Expression.In("operationTypes.Id", operationTypeId)) because additional join is performed on "Operation" You can simply write: criteria.Add(Expression.In("OperationTypes", operationTypeId)) .CreateCriteria("Operation") .Add(Expression.Eq("IsPublished", true)) .Add(Expression.Eq("IsActive", true)) .Add(Expression.Le("AdvertisingStartDate", DateTime.Now)) .Add(Expression.Ge("AdvertisingEndDate", DateTime.Now)) .SetResultTransformer(new DistinctRootEntityResultTransformer())
