Well, you've already verified that you can't! Here's why: From: http://www.ietf.org/rfc/rfc2818.txt Names may contain the wildcard character * which is considered to match any single domain name component or component fragment. E.g., *.a.com matches foo.a.com but not bar.foo.a.com . f*.com matches foo.com but not bar.com .
