OOP lets you extend classes, reusing your the code. Here's a simple example. Check out the PHP docs for Inheritance . class Monster { public $size = 'average'; public $color = 'grey'; public function scare() { return "Bo!"; } } class Rat extends Monster { public $size = 'small'; public function scare() { return parent::scare() . " (runs away)"; } } class Mouse extends Rat { public $color = 'white'; } And the output would be: $rat = new Rat(); echo $rat->scare(); //output: Bo! (runs away) echo $rat->size; //output: small echo $rat->color; //output: grey
