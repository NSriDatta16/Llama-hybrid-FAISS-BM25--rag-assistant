Ugly, but works. select st.uid ,st.uname ,group_concat(concat(sc.sid,'=',sc.sname)) as example1 ,group_concat(sc.sid) as example2 ,group_concat(sc.sname) as example3 from students st left join links l on l.uid = st.uid left join schools sc on sc.sid = l.sid where st.uid = 3 group by st.uid ,st.uname; example_1 gives you value pairs, such as (1=Cambridge,2=Oxford,3=Haganässkolan). example_2 contains a csv string of school ids (1,2,3) example_3 contains a csv string of school names (Cambridge,Oxford,Haganässkolan)
