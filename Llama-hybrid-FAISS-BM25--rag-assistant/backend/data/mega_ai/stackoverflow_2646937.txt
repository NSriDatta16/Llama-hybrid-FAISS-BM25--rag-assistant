If you are embedding ViewController1 in an UINavigationController , this is a pretty common use-case. Inside ViewController1, add this code where you want to show the ViewController2 (in an action for example): ViewController2 *controller = [[ViewController2 alloc] initWithNibName: bundle:nil]; [controller setData: ]; [self.navigationController pushViewController:controller animated:YES]; [controller release]; The navigation controller will take care of the rest.
