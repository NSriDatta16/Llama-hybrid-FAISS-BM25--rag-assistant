I guess the point of the interview question basically is the fact that innerHTML is way faster than any DOM operation in every browser. So, don't use DocumentFragment , use a simple string instead: var ul = document.getElementById("list"); var lstLi = ul.childNodes; var str = ''; for (var i=lstLi.length - 1; i >= 0; i--) { str += ' ' + lstLi[i].innerHTML + ' '; } ul.innerHTML = str; http://jsfiddle.net/bKeuD/
