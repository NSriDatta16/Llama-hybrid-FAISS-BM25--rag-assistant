Try the static methods on the ControlPaint class. Drawn controls may not be skinned like the rest of the GUI, but the effect will be pretty believable. Below is a stripped-down version of some of my code. It is overriding the DrawItem method of an ownerdrawn ListBox to make the list items look like buttons, using ControlPaint.DrawButton method. There are more goodies on that class for checkboxes, combos, even drag handles. protected override void OnDrawItem(System.Windows.Forms.DrawItemEventArgs e) { e.DrawBackground(); if (e.Index > -1) { String itemText = String.Format("{0}", this.Items.Count > 0 ? this.Items[e.Index] : this.Name); //Snip System.Windows.Forms.ControlPaint.DrawButton(e.Graphics, e.Bounds, ButtonState.Normal); e.Graphics.DrawString(itemText, this.Font, SystemBrushes.ControlText, e.Bounds); } }
