As you would be aware that, logistic regression is a standard linear model , following these steps would be helpful : Plot the input data, just observe how would probably the decision boundary look like. If it's completely non-linear then other models will give you a good result than logistic regression. If there is a high chance of linearity possible then you can go with logistic regression
