Right now, it's not clear what you intend the template parameter to mean. It appears that a non-template function should work fine: float average(Triple const &arg) { return (arg.getElem(1) + arg.getElem(2) + arg.getElem(3)) / 3.0f; } If Triple is itself a template that can be instantiated over different possible types, you could do something like this: template T average(Triple const &arg) { return (arg.getElem(1) + arg.getElem(2) + arg.getElem(3)) / T(3.0); }
