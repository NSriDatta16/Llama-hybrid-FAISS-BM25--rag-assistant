in the "File System (setup project)" window, click right mouse button on directory you want to create shortcut (lets say "Application Folder" inside "File System on target machine". Choose "Create Shortcut to Application folder". Shortcut will be created in this directory, and than you simply drag&drop it on "User's Programs Menu" or where ever you want it to have.
