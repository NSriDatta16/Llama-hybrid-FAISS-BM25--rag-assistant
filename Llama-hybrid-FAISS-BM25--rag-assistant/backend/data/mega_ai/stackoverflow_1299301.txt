What your user should be looking at is up to you, the Web designer, to figure out. To achieve precedence you have many tools at your disposal: * Position — Where something is on a page clearly influences in what order the user sees it. * Color — Using bold and subtle colors is a simple way to tell your user where to look. * Contrast — Being different makes things stand out, while being the same makes them secondary. * Size — Big takes precedence over little (unless everything is big, in which case little might stand out thanks to Contrast) * Design Elements — if there is a gigantic arrow pointing at something, guess where the user will look?
