You don't embed it, you link to it. Usually, the link is a thumbnail from the video itself. iPhones don't support embedding of movie files directly in a site. Clicking the link will open Quicktime on the user's iPhone, then return them to the web page when they're done. Even if embed works, a linked image is going to be easier to remember:
