Turbo Delphi Explorer is free, can open Delphi 5 projects and contains a Model View (Together) which allows to drag and drop classes to diagrams in the Delphi IDE, with automatic connections for class inheritance. As always when using tools which work on your precious code, do not remember to use backups or source control.
