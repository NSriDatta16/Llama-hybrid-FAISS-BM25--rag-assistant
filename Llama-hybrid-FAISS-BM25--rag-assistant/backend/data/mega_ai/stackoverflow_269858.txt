For what it is worth is supported in Internet Explorer 8. There are a lot of problems with concerning multiple nested elements (nested quotes) and "proper" quoting styles per language. This has hampered both the user-agent support as well as adoption rate. All is not rosy in Firefox either, if go the CSS generated quotes route, when you an end-user cuts & pastes text with generated characters those characters don't make it to your clipboard. ( see mozilla bug 12460 ) Looking forward, there was an interesting thread in the W3C HTML working group on what HTML 5 should do with in October of 2008 .
