Multiple options here, but as a starting point ... can you just use a PHP try/catch and echo your error? While not the most elegant way, for this simple issue, would most likely be the simplest. try { if ($client) { $client->getBlob('setup', 'folders.txt', $filename); echo 'Blob Load ok!'; } else { echo 'Blob Storage Error; } echo ', END'; } catch(Exception $ex) { echo $ex->getMessage(); }
