You should add element to your application manifest as described in this article (See section “Building and Embedding a Manifest with Microsoft Visual Studio® 2005 for Windows XP and Windows Vista Applications” ; this method will work both in Vista/7 and in XP/2003 server), and set its level attribute to asInvoker value. Thus you'll eliminate the option 2 from the installer detection algorithm quoted in this answer .
