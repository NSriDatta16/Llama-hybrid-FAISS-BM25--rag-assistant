Go to admin>sales>orders Write down your test orders incremental ids, for example 100000001,100000002,100000003,100000111,100000112,100000199 now create a php file in magento root directory and name it: remove_test_orders.php Paste the code below: require 'app/Mage.php'; Mage::app('admin')->setUseSessionInUrl(false); //replace your own orders numbers here: $test_order_ids=array( '100000001', '100000002', '100000003', '100000111', '100000112', '100000199', ); foreach($test_order_ids as $id){ try{ Mage::getModel('sales/order')->loadByIncrementId($id)->delete(); echo "order #".$id." is removed".PHP_EOL; }catch(Exception $e){ echo "order #".$id." could not be remvoved: ".$e->getMessage().PHP_EOL; } } echo "complete."; Now go to command line and run: php remove_test_orders.php At the end, delete remove_test_orders.php.
