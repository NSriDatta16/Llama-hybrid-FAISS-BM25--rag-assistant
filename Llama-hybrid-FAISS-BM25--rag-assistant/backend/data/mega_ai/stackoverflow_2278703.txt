I've used the built-in style "Table Grid" to remove the paragraph spacing style in the cells (The Word 2007 default, Insert > Table uses the same style): Word.Document Doc = Globals.ThisDocument.Application.ActiveDocument; Word.Table WordTable = Doc.Tables.Add(curSel.Range, 8, 5, ref missing, ref missing); //Table Style object tableStyle = "Table Grid"; WordTable.set_Style(ref tableStyle);
