Use isset() or array_key_exists() . The two functions do have a very slight difference in that if the value is null, even if the key exists, isset returns false. However, it shouldn't matter in this case if ( isset($_POST['branding']) || isset($_POST['print']) ){ //... } Or possibly better $ops = array('branding', 'print'); $hasSomethingSet = false; foreach ( $ops as $val ){ if ( isset($_POST[$val]) ){ $hasSomethingSet = true; break; } } if ( $hasSomethingSet ){ //... } If you have PHP 5.3, a slightly slower but more elegant solution is (untested): $ops = array('branding', 'print'); $hasSomethingSet = array_reduce($ops, function($x, $y){ return $x || isset($_POST[$y]; }, false); It depends on how happy with functional programming you are as to if you prefer it.
