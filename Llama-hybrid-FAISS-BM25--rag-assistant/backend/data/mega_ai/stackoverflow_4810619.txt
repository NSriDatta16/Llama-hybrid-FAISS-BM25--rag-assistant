Line breaks in VB.NET can be achieved with minimal code by using vbcrlf : twoParagraphs = "Paragraph One..." & vbcrlf & vbcrlf & "Paragraph Two..." aTextBox.Text = twoParagraphs vbcrlf is essentially the same as Environment.NewLine but is specific to VB.NET
