It doesn't work because AverageHelper<>::Average() doesn't override TrivialList::Average() . In order to override a virtual function, the overriding class must inherit from the class containing the function to be overridden. You could change your template thus: template class AverageHelper : public Base { public: template AverageHelper(T arg) : Base(arg) {} // ... }; class IndexableList : public AverageHelper { public: IndexableList(int count) : AverageHelper (count) {} // ... }; You might want to virtually derive from TrivialList : template class AverageHelper : virtual public Base { // ... };
