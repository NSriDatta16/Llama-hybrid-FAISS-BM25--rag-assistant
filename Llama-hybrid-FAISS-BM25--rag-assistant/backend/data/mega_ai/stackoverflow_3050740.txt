This isn't with regex, and I don't know if this is what you consider "elegant," but... var sequence = 'ATGAACATAGGACATGAGGAGTCA'; var matches = []; do { matches.push('ATG' + (sequence = sequence.slice(sequence.indexOf('ATG') + 3))); } while (sequence.indexOf('ATG') > 0); I'm not completely sure if this is what you're looking for. For example, with an input string of ATGabcdefghijATGklmnoATGpqrs , this returns ATGabcdefghijATGklmnoATGpqrs , ATGklmnoATGpqrs , and ATGpqrs .
