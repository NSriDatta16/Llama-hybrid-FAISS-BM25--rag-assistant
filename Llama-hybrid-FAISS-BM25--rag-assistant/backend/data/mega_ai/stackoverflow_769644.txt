you cannot , java Arrays doesn't implements Iterable: public static int sum(Iterable elements) { int s = 0; for (int i : elements) { s += i; } return s; } public static void main(String[] args) { L1: System.out.println(sum(1,2,3)); L2: System.out.println(sum(Arrays.asList(1,2,3))); L3: System.out.println(sum(new int[] { 1,2,3 })); } this results in two compile-time errors in (L1 and L3); so you must design your method to accept an Iterable (Collections) and/or an Array, at least one method must perform some conversion (to/from array) WORKAROUND : you may be try with an adapter: public class ArrayIterator implements Iterator { private final T[] array; private int i; public ArrayIterator(T[] anArray) { array = anArray; i = 0; } public boolean hasNext() { return i () { public Iterator iterator() { return new ArrayIterator (elements); } }); } you should pay attention only when dealing with primitive arrays; when you use only reference object (your case) ArrayIterator + anonymous class are cool hope it helps
