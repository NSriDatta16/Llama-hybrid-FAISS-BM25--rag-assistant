The reason is I didn't load mysql settings in django's database configuration, which is not related to unicode problem I presented. The configuration should be like this: DATABASES = { 'default': { 'ENGINE': 'django.db.backends.mysql', 'NAME': 'db', 'USER': 'xxx', 'PASSWORD': 'xxx', 'HOST': 'localhost', 'PORT': '3306', 'OPTIONS' : { "init_command": "SET storage_engine=INNODB", 'read_default_file': '/etc/my.cnf', } } }
