The graph has a source filter, demux and then decoders and renderers. The source filter is just a wrapper around the file access APIs. If you replace the stock source filter with a custom one, you can play from memory, or from whatever storage your custom filter accesses. There is a sample source filter in the sdk which reads the whole file into memory and uses that. G
