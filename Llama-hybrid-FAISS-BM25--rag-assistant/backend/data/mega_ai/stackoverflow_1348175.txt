You should be able to put the password in the $HOME/.my.cnf file, avoiding this issue as well as increasing the security . Another option to using $HOME/.my.cnf is embedding the configuration file into the script. A comment on the documentation gives examples on this. For your case it should be: $ mysql --user=XXX --defaults-file
