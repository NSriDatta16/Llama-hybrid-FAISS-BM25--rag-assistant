When running edu.stanford.nlp.parser.lexparser.LexicalizedParser on the command line, you want to use: -outputFormat "wordsAndTags" Programatically, use the TreePrint class constructed with formatString="wordsAndTags" and call printTree, like this: TreePrint posPrinter = new TreePrint("wordsAndTags", yourPrintWriter); posPrinter.printTree(yourLexParser.getBestParse());
