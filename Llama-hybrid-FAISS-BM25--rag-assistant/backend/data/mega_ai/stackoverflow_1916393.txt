A more elegant approach IMHO : foreach ($arrays as $array) { if (array_product(array_map('is_numeric', $array)) == true) { echo "TRUE\n"; } else { echo "FALSE\n"; } } This will return true if all the values are numeric and false if any of the values is not numeric.
