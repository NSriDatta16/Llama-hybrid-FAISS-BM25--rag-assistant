Depending on the data you receive you can gather evidents that the caller is a user. Only ask for the CAPTCHA, if the caller seems to be a bot. Evidents can be: the user agent (not reliable, because Opera can fake this) the interval the submit is triggered (the more exact the call hits the i.e. every 2 minutes the more likely it is a bot) if your data allows checking for returning patterns it can be an evident, too ...
