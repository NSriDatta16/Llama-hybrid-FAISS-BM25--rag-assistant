I had this problem and it came from not returning the clf from my classifier functions so when I was doing: def random_for(): from sklearn.ensemble import RandomForestClassifier clf = RandomForestClassifier(max_depth = 5) clf.fit(Input, Labels) y_pred = clf.predict(Test_input) predictions = [round(value) for value in y_pred] accuracy = accuracy_score(Test_Labels, predictions) print("Random Forest accuracy: %.2f%%" % (accuracy*100.0)) rclf = random_for() I got the same error as the OP. Whereas; def random_for(): from sklearn.ensemble import RandomForestClassifier clf = RandomForestClassifier(max_depth = 5) clf.fit(Input, Labels) y_pred = clf.predict(Test_input) predictions = [round(value) for value in y_pred] accuracy = accuracy_score(Test_Labels, predictions) print("Random Forest accuracy: %.2f%%" % (accuracy*100.0)) return clf rclf = random_for() solved the problem.
