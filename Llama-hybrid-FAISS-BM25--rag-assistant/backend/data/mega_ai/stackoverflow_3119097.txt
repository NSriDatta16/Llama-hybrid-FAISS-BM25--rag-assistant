If you have both res/drawable and family of res/drawable-... (where ... is ldpi, mdpi or hdpi) folders then for standard emulator configuration (HVGA skin) file from res/drawble-mdpi has precedence over the one in res/drawable . edit: see: DevGuide http://developer.android.com/guide/topics/resources/providing-resources.html#AlternativeResources
