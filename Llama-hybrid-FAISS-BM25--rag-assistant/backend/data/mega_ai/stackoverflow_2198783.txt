What do you want irp.AgentPolicyVersion.Version and irp.ResourcePolicyEnabled to be when rp is null? I would expect them to be set to some appropriate default values rather than left as they are. Otherwise, I would have written it like this: IResourcePolicy irp = (IResourcePolicy)dmo; irp.AgentVersion = agentVersion; irp.ResourcePolicy = rp; if (rp != null) { irp.AgentPolicyVersion.Version = Convert.ToInt64(policyVersion); irp.ResourcePolicyEnabled = Convert.ToBoolean(enabled); }
