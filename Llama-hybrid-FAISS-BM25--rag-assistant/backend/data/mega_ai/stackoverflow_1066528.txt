Use the viewstate, is the best option for you situation: // Storing a student in view state. Student stud = new Student("John", "Doe"); ViewState["CurrentStudent"] = stud; // Retrieve a student from view state Student stud = (Student) ViewState["CurrentStudent"]; All you need to pay attention is that your classes must be serializable to be stored on the viewstate. This is a nice tutorial to help you through.
