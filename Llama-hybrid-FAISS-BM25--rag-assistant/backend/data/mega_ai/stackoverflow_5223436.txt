There is a native extension, too. The SplEnum SplEnum gives the ability to emulate and create enumeration objects natively in PHP. http://www.php.net/manual/en/class.splenum.php Attention: https://www.php.net/manual/en/spl-types.installation.php The PECL extension is not bundled with PHP. A DLL for this PECL extension is currently unavailable.
