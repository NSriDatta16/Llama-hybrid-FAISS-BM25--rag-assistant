You can read more about foreign key support in sqlite here: http://www.sqlite.org/foreignkeys.html but you should be able to turn it on: sqlite> PRAGMA foreign_keys = ON; and then setup the database schema with the deletes cascading: -- Database schema CREATE TABLE Employee( E_id INTEGER PRIMARY KEY, name TEXT ); CREATE TABLE Department( Dept_id INTEGER PRIMARY KEY, name TEXT, E_id INTEGER REFERENCES Employee(E_id) ON DELETE CASCADE ); CREATE TABLE Electronics( Elec_id INTEGER PRIMARY KEY, name TEXT, Dept_id INTEGER REFERENCES Department(Dept_id) ON DELETE CASCADE ); With all this in place and the data in the tables: DELETE FROM Employee WHERE E_id = 2;
