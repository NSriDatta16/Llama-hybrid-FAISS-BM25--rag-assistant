I achieved this by adding a custom attribute server side to the controls stating if I wanted them disabled, I called this MappedControl, I then used the following Javascript to do the disabling function Level1Populated() { if ($("*[id$='ddl_OrganisationalLevel4Id']").attr("MappedControl")) { $("*[id$='ddl_OrganisationalLevel4Id']").disabled = true; } } function pageLoad(sender, args) { $find("ccd_Level1BID").add_populated(Level1Populated); }
