pocketwit is open source and has a cool interface http://code.google.com/p/pocketwit/ Kinetic List - The main list implements "kinetic scrolling". Side Menus - Action menus are located to the sides of the main menu. So if you drag the list to the left or right, you will see the a menu of actions beside the list.
