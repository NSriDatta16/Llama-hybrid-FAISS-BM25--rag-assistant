If you had a MouseEventHandler for the UltraGrid then you can do the following: UltraGrid grid = (UltraGrid)sender; UIElement element = grid.DisplayLayout.UIElement.ElementFromPoint(new Point(e.X, e.Y)); You can then cast the element depending on its expected type using element.GetContext(): UltraGridCell cell = (UltraGridCell)element.GetContext(typeof(UltraGridCell));
