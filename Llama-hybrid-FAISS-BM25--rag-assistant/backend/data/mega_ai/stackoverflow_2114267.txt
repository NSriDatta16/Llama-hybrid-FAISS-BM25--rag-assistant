Robert Hurst is correct in theory, but if you like FancyBox, it supports both modes via configuration If you look at bottom of the http://fancybox.net/howto page, it has an option centerOnScroll If true, content is centered when user scrolls page here is an example of how you would invoke it: jQuery(document).ready(function(){ $('#a').fancybox({ centerOnScroll: false }); });
