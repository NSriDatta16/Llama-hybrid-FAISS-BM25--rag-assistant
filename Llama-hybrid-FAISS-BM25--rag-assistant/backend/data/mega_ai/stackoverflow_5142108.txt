You can test that with var_dump(!$result = FALSE); // TRUE var_dump(!$result = array()); // TRUE var_dump(!$result = array(1)); // FALSE So if SimpleXml::xpath returns an empty array or FALSE , the Magento function will return FALSE . Negating an array will typecast it to boolean first. Empty Arrays typecasted to Boolean will be FALSE . Which means, when your XPath is syntactically correct, but did not find any results, Magento will return FALSE .
