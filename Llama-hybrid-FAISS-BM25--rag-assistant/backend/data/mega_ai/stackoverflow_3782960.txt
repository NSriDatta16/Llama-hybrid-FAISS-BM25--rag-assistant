You could use the BitConverter class has Robert mentioned. A simple example: byte[] byte_arr = new byte[] { 0x12, 0x23, 0x34, 0x45, 0x56, 0x67, 0x78, 0x89 }; UInt16 uint1 = BitConverter.ToUInt16(byte_arr, 0); // 0x2312 UInt16 uint2 = BitConverter.ToUInt16(byte_arr, 4); // 0x6756
