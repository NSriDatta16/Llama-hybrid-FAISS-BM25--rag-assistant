You should always send documents that are still usable when JavaScript support is not available. And as most of such navigation use the fragment identifier of the URI to identify the part of a document that should be displayed, you could use this too and target your form to this fragment: Or you use JavaScript to send the data, fetch the response and display it inside the current tab.
