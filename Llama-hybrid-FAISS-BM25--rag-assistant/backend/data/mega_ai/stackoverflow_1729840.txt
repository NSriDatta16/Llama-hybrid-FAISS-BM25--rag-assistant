You can have a procedure that writes messages to a table using an autonomous transaction something like: procedure log (p_message) is pragma autonomous_transaction; begin insert into message_log (user, datetime, message) values (user, sysdate, p_message); commit; end; Then monitor the table from another Oracle session.
