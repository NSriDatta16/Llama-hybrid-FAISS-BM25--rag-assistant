I'd do it like this: var result = new List (); while (num != 0) { result.Insert(0, num % 10); num = num / 10; } return result.ToArray(); Slightly less performant but possibly more elegant is: return num.ToString().Select(c => Convert.ToInt32(c.ToString())).ToArray(); Note that these both return 1,2,3,4,5,6 rather than 49,50,51,52,53,54 (i.e. the byte codes for the characters '1','2','3','4','5','6') as your code does. I assume this is the actual intent?
