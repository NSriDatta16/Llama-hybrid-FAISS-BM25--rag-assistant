This seems to work: $(function() { $('body').click(function() { alert('clicked the page'); return false; }); $('*:not(body)').click(function() { alert('clicked an item!'); return false; }); }); One "problem" you'll have is that paragraphs are block elements by default, so clicking to the side of a sentence will produce an item click even if it looks like you haven't clicked on an item.
