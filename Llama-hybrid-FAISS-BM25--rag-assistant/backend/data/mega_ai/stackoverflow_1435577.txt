Yes, it's reentrant as it only modifies its arguments Wikipedia provides some nice points on what have to be provided to be reentrant: To be reentrant, a computer program or routine: Must hold no static (or global) non-constant data. Must not return the address to static (or global) non-constant data. Must work only on the data provided to it by the caller. Must not rely on locks to singleton resources. Must not modify its own code. 1 (unless executing in its own unique thread storage) Must not call non-reentrant computer programs or routines.
