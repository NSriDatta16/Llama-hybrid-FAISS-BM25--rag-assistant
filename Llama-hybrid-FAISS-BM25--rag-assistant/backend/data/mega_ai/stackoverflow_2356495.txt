Here's a code fragment that I use to build an entire project where, as usual, source files are in a deeply nested hierarchy and there are many .jar files that must go into the classpath (requires UNIX utilities): CLASSPATH= for x in $(find | grep jar$); do CLASSPATH="$CLASSPATH:$x"; done SRC=$(find | grep java$) javac -cp "$CLASSPATH" $SRC
