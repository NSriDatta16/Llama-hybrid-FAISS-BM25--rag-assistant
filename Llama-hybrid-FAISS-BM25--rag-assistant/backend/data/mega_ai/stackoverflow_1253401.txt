Firefox\/(\d+(?:\.\d+)+) Depending on your language (I’m assuming JS) it’ll be the second element in the array, i.e. const regex = /Firefox\/(\d+(?:\.\d+)+)/; const matches = useragent.match(regex); console.log(matches[1]); // "3.5.2"
