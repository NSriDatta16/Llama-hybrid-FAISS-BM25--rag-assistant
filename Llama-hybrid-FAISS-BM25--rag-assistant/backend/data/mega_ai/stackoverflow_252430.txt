Here are some more open source search engines Sphinx Xapian Zend_Search_Lucene Here are some search related tools Forage The Forage PHP5 library is an easy to use interface to multiple back end search libraries. It provides a common interface while supporting unique features in each library by allowing back-ends to support specific features or not. Marjory Marjory is a webservice for indexing and searching for documents, utilizing a full-text search engine.
