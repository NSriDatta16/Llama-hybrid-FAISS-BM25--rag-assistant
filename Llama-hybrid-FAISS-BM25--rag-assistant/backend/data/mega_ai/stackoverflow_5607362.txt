FYI, there is nothing about System.Windows.Forms timer that doesn't allow you to create in code (it's not just a "drag-and-drop" timer). Code: Constructor code: System.Windows.Forms.Timer timer = new System.Windows.Forms.Timer(); timer.Tick += OnTimerTick; timer.Interval = 10000; timer.Start(); Event Handler: private void OnTimerTick(object sender, EventArgs e) { // Modify GUI here. }
