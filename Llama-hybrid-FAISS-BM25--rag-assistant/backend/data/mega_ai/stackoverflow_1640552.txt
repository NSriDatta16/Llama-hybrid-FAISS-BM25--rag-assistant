Using JavaScript, it's impossible - JavaScript has no access to the client computer except cookies and the new HTML5 local storage. What you can do, however, is use a Java applet that will be controllable via JavaScript - hidden or not. You can find an example here . This requires Java runtime to be installed on the client computer.
