Web Parts allow building dynamic portal-like UI with drag/drop end-user customization. From ASP.NET Web Parts Overview : ASP.NET Web Parts is an integrated set of controls for creating Web sites that enable end users to modify the content, appearance, and behavior of Web pages directly from a browser. Examples of web portals are iGoogle , My Yahoo! , and Windows Live Personalized Experience .
