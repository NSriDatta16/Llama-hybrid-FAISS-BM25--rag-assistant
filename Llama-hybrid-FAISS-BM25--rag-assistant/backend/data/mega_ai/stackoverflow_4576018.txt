If you wan't to store information on the client you can use isolated storage. for example: // Create an instance of IsolatedStorageSettings. private IsolatedStorageSettings userSettings = IsolatedStorageSettings.ApplicationSettings; // Add a key and its value. userSettings.Add("userImage", "BlueHills.jpg"); // Remove the setting. userSettings.Remove("userImage"); Look here it could get you started: http://www.silverlight.net/learn/quickstarts/isolatedstorage/
