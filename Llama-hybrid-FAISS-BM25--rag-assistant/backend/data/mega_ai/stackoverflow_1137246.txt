You can use functors and pass it to your lookup method. That I mean is, define class which will overload bool operator( vectorElement element) and within this operator you will choose method how do you want to lookup the values. template class ILookUp { bool operator( vector elem) { if (elem == something) return true; false; } }; class VectorStorage { std::vector lookup( ILookUp lookup) { ..... if ( lookup(elem)) { //add element to vector or whatever. } ..... return result; } ..... }
