DatabaseMetaData.getSQLKeywords will return the databases keywords that are not in SQL-2003 (SQL-2003 keyword can be found in the standard). There are other methods within that interface for getting various types of function names and similar. As Chase Seibert mentions, don't think this is an effective way to block SQL injection attacks.
