Instead of having the setFP set the callback to lbDataBindingSource.AddNumber, create a private method in your code to handle the callback and then call lbDataBindingSource.AddNumber from that callback. void MyForm_Load(object sender, EventArgs e) { //... cdll.setFP(FPCallback); } private void FPCallback(int num) { lbDataBindingSoruce.AddNumber(num); }
