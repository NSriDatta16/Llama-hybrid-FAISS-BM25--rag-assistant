Important: Make sure you add the component to the IDE using the visual designer. First right click in the toolbox, click "choose items", click the COM tab, then check the TWS control. Now using the visual IDE drag this item to the surface of your form. It must be visible in your app. (Don't create it in the Form1_Load() ) Do something like this... (don't use an IP Address of 127.0.0.1, leave it blank): axTws1.connect("", 7496, 0); axTws1.reqMktData(0, "AMD", "STK", "", 0, "", "", "SMART", "ISLAND", "USD", "", 0); Last (if you're using 64-bit Windows) compile the project in 32-bit code. Go to the configuration manager and then create a new profile "x86" and set all the configurations to x86.
