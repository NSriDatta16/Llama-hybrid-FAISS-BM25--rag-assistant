You can use the Embed meta tag in plain AS3, too if you're targeting FP10+. For example, in a class add: [Embed(source="C:\WINDOWS\Fonts\myfontfile.ttf", fontFamily="myFont")] Then... TextField.embedFonts = true; myTextFormat.font = "myFont"; Check here for a good example: http://marumushi.com/news/embedding-fonts-in-as3 Any way to avoid library font linkage voodoo is a good choice. Oh, and make sure your font file is an accepted font format.
