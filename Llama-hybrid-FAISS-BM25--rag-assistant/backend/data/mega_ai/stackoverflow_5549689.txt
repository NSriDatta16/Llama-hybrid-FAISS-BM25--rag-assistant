You could use a javax.xml.transform.Transformer to convert a StAXSource wrapping the reader to a StAXResult wrapping the writer. TransformerFactory tf = TransformerFactory.newInstance(); Transformer t = tf.newTransformer(); StAXSource source = new StAXSource(xmlStreamReader); StAXResult result = new StAXResult(xmlStreamWriter); t.transform(source, result); Using the Event API you could also use the folloiwng: http://download.oracle.com/javase/6/docs/api/javax/xml/stream/XMLEventWriter.html#add(javax.xml.stream.XMLEventReader)
