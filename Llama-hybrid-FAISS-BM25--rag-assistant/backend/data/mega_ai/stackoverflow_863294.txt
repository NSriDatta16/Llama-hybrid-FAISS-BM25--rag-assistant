This command will leave only the desired files in their original directories: find test \( ! -path "test/mtndew/livewire/*" ! -path "test/icecream/cupcake/*" \) -delete No need for cpio. It works on Ubuntu, Debian 5, and Mac OS X. On Linux, it will report that it cannot delete non-empty directories, which is exactly the desired result. On Mac OS X, it will quietly do the right thing.
