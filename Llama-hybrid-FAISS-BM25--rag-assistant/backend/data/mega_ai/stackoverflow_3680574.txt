In order to establish a new lexical binding, use let or the with keyword of loop . In order to extend an existing list, you might want to use push ; if you need the original order, you can nreverse the new list finally . Another way would be to use the when and collect keywords of loop . Another hint: mapcan implicitly creates a new list.
