Try clearing cache.. on every request Response.Cache.SetCacheability(HttpCacheability.NoCache); Response.Cache.SetExpires(DateTime.Now.AddSeconds(-1)); If Still not work. Try this bunch of code in pag load event Response.AppendHeader("Pragma", "no-cache"); Response.AppendHeader("Cache-Control", "no-cache"); Response.CacheControl = "no-cache"; Response.Expires = -1; response.ExpiresAbsolute = new DateTime(1900, 1, 1); response.Cache.SetCacheability(HttpCacheability.NoCache);
