JavaScript objects are unordered. There is no guarantee about which order the keys should come out when you loop over them and JS engines are free to implement whatever storage and retrieval systems they like. If order matters, use an array: [] This can contain objects: [ { 'foo' : '1234', 'bar' : '5678' }, { 'foo' : 'abcd', 'bar' : 'qwer' }, { 'foo' : 'ldng', 'bar' : 'plma' } ]
