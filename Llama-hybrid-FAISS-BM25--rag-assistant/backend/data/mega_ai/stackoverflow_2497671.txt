To create a managed object and call methods on it you need to have the CLR running in your C++ process. Under windows you can host the CLR by referencing mscoree.dll and hosting the CLR in process. http://msdn.microsoft.com/en-us/magazine/cc163567.aspx http://msdn.microsoft.com/en-us/library/ms230997.aspx For Mono, you can embed the Mono runtime within your C++ application. http://www.mono-project.com/Embedding_Mono
