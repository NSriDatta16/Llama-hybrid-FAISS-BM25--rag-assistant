Statistical significance between explanatory variables

Let's suppose I have this data: $\mathbf{x}_1 = (0, 1, 0, 0, 0, 1)^{\mathbf{T}}$ $\mathbf{x}_2 = (0, 0, 1, 0, 1, 0)^{\mathbf{T}}$ $\mathbf{y} = (0, 1, 0, 1, 0, 0)^{\mathbf{T}}$ I am conducting a logistic regression to find $\beta_0$ , $\beta_1$ , and $\beta_2$ in $\mathbf{y} = \beta_0 + \beta_1\mathbf{x}_1 + \beta_2\mathbf{x}_2$ . Is there a simple way to test the hypothesis $H_0: \beta_1 = \beta_2$ vs. $H_A: \beta_1 > \beta_2$ ?
