From clang man page: % man 1 clang ... Driver Options -Xclang arg Pass arg to the clang compiler. The -Xclang flag is a LLVM driver option for “Pass argument to the clang compiler,” and the the argument here is -fobjc-nonfragile-abi2. So, if you put them in reversed order, the compiler will use -fobjc-nonfragile-abi2 as cflag, which isn’t, and get you the error in return. Looki Looki here: http://digdog.tumblr.com/post/833744044
