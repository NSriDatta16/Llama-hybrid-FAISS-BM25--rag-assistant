underscore ( _ ) stands for a private / protected function or variable. Don't know WHO actually came up with it, but I know it is "supported" by Zend ( and the Zend coding standards ). edit : http://framework.zend.com/manual/en/coding-standard.naming-conventions.html -> section B.3.4 -> paragraph 2
