I'd just make the HandlerRegistration object a field of the enclosing class, that way you won't be bothered by the compiler and it's more "elegant" than shuffling arrays and stuff: public class TestWidget extends Composite { //... HandlerRegistration handler; public TestWidget() { // ... handler = button.addClickHandler(new ClickHandler() { @Override public void onClick(ClickEvent event) { // ... handler.removeHandler(); } }); } }
