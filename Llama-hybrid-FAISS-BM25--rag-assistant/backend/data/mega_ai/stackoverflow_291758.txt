Per data scaling, this may also help you: http://code.google.com/apis/chart/formats.html#data_scaling Note the chds parameter option. You may also wish to consider leveraging a wrapper API that abstracts away some of these ugly details. They are listed here: http://groups.google.com/group/google-chart-api/web/useful-links-to-api-libraries I wrote charts4j which has functionality to help you deal with data scaling.
