This is kind of lame, but you can cast it to standard array and use array_splice on it: $ao = new ArrayObject(); $ao[] = 'element 1'; $ao[] = 'element 2'; $ao[] = 'element 3'; $ao[] = 'element 4'; var_dump($ao); $items = (array) $ao; array_splice($items, 1, 2); $ao = new ArrayObject($items); $ao[] = 'element 5'; var_dump($ao); This results in: object(ArrayObject)#1 (1) { ["storage":"ArrayObject":private]=> array(4) { [0]=> string(9) "element 1" [1]=> string(9) "element 2" [2]=> string(9) "element 3" [3]=> string(9) "element 4" } } object(ArrayObject)#2 (1) { ["storage":"ArrayObject":private]=> array(3) { [0]=> string(9) "element 1" [1]=> string(9) "element 4" [2]=> string(9) "element 5" } }
