Here's another idea. Again, not terribly elegant, but might be pretty fast. It's similar to Chacha102's second part, although it would be faster if you only have integer values in the sub arrays. // implode the sub arrays $tmpArray = array(); foreach ($arr as $key => $array) { $tmpArray[$key] = implode(',', $array); } // get only the unique values $tmpArray = array_unique($tmpArray); // explode the values $arr = array(); foreach ($tmpArray as $key => $string) { $arr[$key] = explode(',', $string); }
