What you seems to want is sometimes called fractional logistic regression , there are many Questions on this site, start with What is the difference between logistic regression and Fractional response regression? and search this site. But very short: You should NOT transform the scores, use them directly as the outcome variable. Logistic regression will transform implicitly their expectation, not the scores themselves. Since your data is not really binomial, the likelihood function is not strictly correct, so use quasi-likelihood With R you would use something like mod0
