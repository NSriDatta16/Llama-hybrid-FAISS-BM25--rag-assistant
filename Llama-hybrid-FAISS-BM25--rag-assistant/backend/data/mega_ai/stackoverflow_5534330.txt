A Popup doesn't give you an obvious way to move it; you can only place it relative to its placement target. However it does give you and additional offset relative to the placement target and that's enough to move it anywhere. Here's a working markup-only solution that demonstrates a draggable Popup using a Thumb using Markup Programming . There is a text box that has a popup that pops up when the text box receives the keyboard focus. The popup has some text and a thumb. @AssociatedObject.HorizontalOffset += @EventArgs.HorizontalChange; @AssociatedObject.VerticalOffset += @EventArgs.VerticalChange;
