Well all you do is your normal curl retrieval, but just set the proxy server option. curl_easy_setopt(handle, CURLOPT_PROXY, "127.0.0.1:96666"); EDIT /* create headers */ struct curl_slist *headers=NULL; headers = curl_slist_append(headers, "Content-Type: text/xml"); /* pass our list of custom made headers */ curl_easy_setopt(easyhandle, CURLOPT_HTTPHEADER, headers);
