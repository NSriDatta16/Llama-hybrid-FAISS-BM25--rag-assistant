I found an interesting link which could help: http://www.ejb3.org/jar_file_reverse/jar_file_reverse.html You can reverse your java project to show .java classes and then just drag and drop .class classes coming from the jar to your diagram. You can make a differentiation between them by adding the package name in the class name, or by changing the color of the class etc....
