Or in XML: android:divider="@drawable/list_item_divider" android:dividerHeight="1dp" You can use a color for the drawable (e.g. #ff112233), but be aware, that pre-cupcake releases have a bug in which the color cannot be set. Instead a 9-patch or a image must be used..
