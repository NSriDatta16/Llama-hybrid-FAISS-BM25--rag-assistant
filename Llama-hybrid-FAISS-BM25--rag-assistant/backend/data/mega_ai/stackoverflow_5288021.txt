I think it should be this: Bid.average(:amount, :conditions => ['item_id = ?', item_id]) item_id is getting put in there as a string, hence the quotes. Make sure to set item_id as a variable before such as: item_id = params[:item_id] #or whatever the params are Bid.average(:amount, :conditions => ['item_id = ?', item_id])
