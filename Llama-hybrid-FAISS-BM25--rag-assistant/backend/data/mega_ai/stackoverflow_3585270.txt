I've done this before with #pragma's. I don't have the actual code handy at the moment, but I think something similar to this might work with Visual Studio 2008 C++: //put this code in a dll that both processes link to #pragma section("shared",read,write,shared) __declspec(allocate("shared")) int i = 0; // in theory, both processes should be able to access and modify i
