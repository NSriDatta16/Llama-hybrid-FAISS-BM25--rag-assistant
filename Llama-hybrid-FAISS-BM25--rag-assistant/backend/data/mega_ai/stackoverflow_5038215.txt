Take Tools > Options > HTML Designer > CSS Styling > Check ' Change positioning to absolute for controls added using Toolbox, paste or drag and drop '. Then, in design page, select form, then in menu.. Format > Position > Positioning Style > Select ' Absolute '. Now you will be able to absolute positioning the components...
