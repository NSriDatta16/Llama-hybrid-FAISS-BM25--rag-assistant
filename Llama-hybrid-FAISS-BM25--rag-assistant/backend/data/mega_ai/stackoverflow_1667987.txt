Jboss Message Bean (EJB3) - configuration name

Is there a way of specifying the configuration name of a message bean through annotations? In the jboss.xml descriptor you can have different configurations and bind the beans to a particular config: PushUserAgentBean queue/PushUserAgent My Custom Message Driven Bean Then in standardjboss.xml you can specify a custom interceptor stack, etc. for the "My Custom Message Driven Bean" configuration: My Custom Message Driven Bean false custom-message-driven-bean org.jboss.ejb.plugins.ProxyFactoryFinderInterceptor org.jboss.ejb.plugins.LogInterceptor org.jboss.ejb.plugins.RunAsSecurityInterceptor Is there a simple way to do the same by using annotations? I don't mind using vendor-specific annotations (ie, non-standard jboss annotations).
