You can use the AliasToBean result transformer. ( Doc 1.2 ) It assigns every projection to a property of the same name. session.CreateCriteria(typeof(Item), "item") .CreateCriteria("Bids", "bid") .SetProjection(Projections.ProjectionList() .Add(Projections.Property("item.Id"), "Id" ) .Add(Projections.Property("item.Description"), "Description" ) .Add(Projections.Property("bid.Amount"), "Amount" )) .Add(Expression.Gt("bid.Amount", 100)) .SetResultTransformer(Transformers.AliasToBean(typeof(ItemRow))) .List();
