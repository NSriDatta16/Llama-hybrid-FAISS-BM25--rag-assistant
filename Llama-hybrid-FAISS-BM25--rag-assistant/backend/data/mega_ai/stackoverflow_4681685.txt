XML in Oracle is a nightmare. You can try something like this (XMLTABLE to the rescue): > select wmsys.wm_concat(''||column_value) text 2 from 3 xmltable( 4 XMLNAMESPACES('http://purl.org/dc/elements/1.1/' as "dc"), 5 '$xml/qualifieddc/dc:subject/text()' 6 passing xmltype(' ' || 7 ' info1 info2 info3 ' || 8 ' ') as "xml"); TEXT -------------------------------------------------------------------------------- info1,info2,info3 Note that I'm using wmsys.wm_concat for concatenation, which is not recommended. Use any other stragg implementation.
