Your problem I think boils down to your final selector. '$sel $eqNum' Isn't a valid jQuery selector. JavaScript doesn't do variable replacement in strings. Also, the use of $ on all of your variable names gets very confusing. I try to only use $ for variables that are jQuery objects. Rewriting your variable declarations a bit to reflect that. Near the bottom of your code block, you actually retrive a jQuery object for your select ( $sel ), you can use the .find() method on that object to look for your options based on eqnum . var index = $('.drop').index(this); var indexofmonth = index + 1; var indeximg = event.dragTarget.title; var indeximgnum = indeximg.replace(/[a-zA-Z]/g,""); var featurenumber = "Feature" + indexofmonth; var eqnum = 'option:eq('+indeximgnum+')'; // find the drop select var $sel = $("select[name="+featurenumber+"]"); // find the option using eqnum from before $sel.find(eqnum).attr('selected', 'selected' ); // or do it all in one brush stroke: $('select[name='+featurenumber+'] option:eq('+indeximgnum+')').attr('selected','selected'); The $ function returns a jQuery object from a selector string, you were passing '$sel $eqNum'
