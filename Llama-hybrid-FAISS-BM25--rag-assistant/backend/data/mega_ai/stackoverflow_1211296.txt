Add this variable at the top of your script: var timer; Implement this function: function setFlagAndImage(flag) { setAllowClick(flag); setImage(); } And then replace the dummy animation with: timer = window.setTimeout(function() { setFlagAndImage(true); }, 3000); If something else then happens and you want to stop the timer, you can just call: window.clearTimeout(timer);
