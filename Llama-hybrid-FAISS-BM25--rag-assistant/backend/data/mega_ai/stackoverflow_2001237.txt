You will need to download the jQuery and jQuery UI libraries onto your server. You can get them here: http://jquery.com/ (large button on right) http://jqueryui.com/download (select UI core and dialog - optionally get draggable and resizable if you need that functionality) According to your code ... ... you will need to put them into the /js folder of your site. If you are testing the HTML locally, remove the "/" from "/js", make a "js" folder alongside the HTML file, and put your javascript files in there.
