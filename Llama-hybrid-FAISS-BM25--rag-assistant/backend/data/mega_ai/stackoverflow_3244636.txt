Like Anders, I use beforeRender , but couple it with ACL. Here is a fragment of the code I use: function beforeRender() { if($this->layout=='admin') { $user = ($this->Auth->user())?$this->Auth->user():array('User'=>array( 'username' => 'anonymous', 'group_id' => null )); if(!$user['User']['group_id']) $this->redirect('/'); $usersIndexAllowed = $this->Acl->check($user,"users/index"); } $this->set(compact('user','usersIndexAllowed','configureAllowed','moduleTypesAllowed')); } You can then use conditionals in the view to hide or display blocks of HTML.
