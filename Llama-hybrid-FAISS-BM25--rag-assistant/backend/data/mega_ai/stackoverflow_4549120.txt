Have you tried JSON.parse ? var json = '{"dynamicvaluenumberone":3, "dynamicvaluenumbertwo":7}'; var obj = JSON.parse(json); For accessing dynamic keys, you can loop over the keys using for...in : for(var i in obj) { console.log(i + " (" + obj[i] + ")"); } Update: Depending on the browsers you need to support, the JSON library is also available here: http://www.json.org/js.html
