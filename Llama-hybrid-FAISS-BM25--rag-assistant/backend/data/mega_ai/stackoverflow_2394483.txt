And the css: div#floatedDiv {float: right; width: 50%; clear: left; height: 6em; padding: 0.5em; margin: 0 0 0.5em 1em; border: 1px solid #ccc; position: relative; } div#floatedDiv p {position: absolute; top: 50%; left: 1em; right: 1em; margin-top: -20%; } div#floatedDiv p {margin-top: -10%; } It's fragile because it won't automatically scale itself according to the dimensions, so it looks ever so slightly imperfect.
