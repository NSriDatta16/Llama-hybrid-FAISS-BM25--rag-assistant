In order to do this you have to create a tag library. There are two ways to define such: via a TagLib class and a taglib descriptor - tedious process, and an overkill for your case tag files - see here for how to do it. In short - you specify a code fragment (with the jsp syntax) that will be executed when the tag is found. The difference would be that you will have , and inside - the reference to the tag.
