In TortoiseSVN you have a function to move versioned items. If you drag the folder using the right mouse button to the location where you want to move it, you will get a menu when you drop it with several options using SVN methods of moving or exporting it. Then SVN will have control on what's happening. Edit: In your case, you might just want to rename the folder using Right click: TortoiseSVN->Rename...
