I don't know if this is a glitch in VS2008 or not, but I found a workaround. Switch to Source view, and drag a control from you Ajax Control Toolkit onto the edit screen. Then set the TargetControlId attribute to the name of the control you want to extend. Strangely enough, after doing this, the smart tag appears, although it only offers you the option of removing the extender. Important: You need to remove the ScriptControlManager (if you have added one) from the page and replace it with the ToolkitScriptManagerControl. I hope this hel
