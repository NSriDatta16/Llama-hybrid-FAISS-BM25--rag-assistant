Remove administer users from the role so they can't change passwords. Encourage users to do password resets. As for assigning user roles, you can use the Role Delegation module, and for node publishing, the Override Node Options module. http://drupal.org/project/role_delegation http://drupal.org/project/override_node_options The second module will let you remove the 'administer nodes' permission, removing a lot of 'overreaching power' from the role.
