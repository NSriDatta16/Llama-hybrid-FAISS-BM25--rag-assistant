Use FakeWeb to replay a prefetched HTTP request: tags_url = "http://website.com/tags/" request = `curl -is #{tags_url}` FakeWeb.register_uri(:get, tags_url, :response => request) agent = WWW::Mechanize.new page = agent.get(tags_url) page.links.each do |link| puts link.text.strip end Calling curl with the -i flag will include headers in the response.
