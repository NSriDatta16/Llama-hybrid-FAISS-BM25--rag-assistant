You can achieve the same effect by utilizing a dynamic proxy. An example is the Castle Dynamic Proxy . Such frameworks leverage the C# reflection facilities to construct 'proxy' or 'wrapper' classes. So, keep that in mind. There is a certain amount of overhead because of this. Alternatively you can use frameworks that can create classes statically via code generation .
