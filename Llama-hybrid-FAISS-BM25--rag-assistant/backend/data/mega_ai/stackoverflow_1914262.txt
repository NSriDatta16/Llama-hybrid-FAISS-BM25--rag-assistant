I found temporary solution for debug. Before executing I just re-export MAVEN_OPTS with some additional parameters so this will work only on currently open terminal window. My system MAVEN_OPTS are export MAVEN_OPTS="-Xms256m -Xmx512m -XX:PermSize=128m -XX:-UseGCOverheadLimit" and session changed to export MAVEN_OPTS="-Xms256m -Xmx1500m -XX:PermSize=128m -XX:-UseGCOverheadLimit -Xdebug -Xnoagent -Djava.compiler=NONE -Xrunjdwp:transport=dt_socket,address=8000,server=y,suspend=n" I will have to discuss "mvnDebug" alias set-up with my colleague when he gets back tomorrow...
