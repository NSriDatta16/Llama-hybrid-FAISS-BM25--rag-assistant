For dynamic content, use jQuery's .live() method. $('#ctl00_ContentPlaceHolder1_Button1').live('click', function (e) { alert('hello'); }); Also, I'd recommend using a class instead of the ID. Using the ID is fragile to any code changes which affect the ASP.NET container/control structure: $('.performsMyClickAction').live('click', function (e) { ... }
