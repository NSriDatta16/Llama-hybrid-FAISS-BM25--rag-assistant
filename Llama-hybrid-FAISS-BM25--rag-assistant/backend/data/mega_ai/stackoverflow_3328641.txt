This is how I got mine working finally (by embedding a ZoomControl in XML layout). mapView = (MapView) this.findViewById(R.id.mapView); ZoomControls zoomControls = (ZoomControls) findViewById(R.id.zoomcontrols); zoomControls.setOnZoomInClickListener(new OnClickListener() { public void onClick(View v) { mapView.getController().zoomIn(); } }); zoomControls.setOnZoomOutClickListener(new OnClickListener() { public void onClick(View v) { mapView.getController().zoomOut(); } }); See: Always show zoom controls on a MapView
