The part declares the generic type parameters of the method - basically what kind of element the sequence contains. You should definitely understand generics before you get too far into LINQ. Then, this IQueryable source indicates the first parameter of the method: this indicates that it's an extension method IQueryable indicates the type of the parameter source is the name of the parameter The fact that it's an extension method is probably what's confusing you. When you call query.Average(s => s.Length) that is converted by the compiler into Queryable.Average(query, s => s.Length)
