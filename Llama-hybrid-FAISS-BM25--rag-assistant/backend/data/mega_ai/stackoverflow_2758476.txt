Another elegant approach using pure HTML and CSS without JavaScript. HTML: here goes text before here come some details and after CSS: .details input, .details span { display: none; } .details input:checked~span { display: inline; border-bottom: dotted 1px gray; } .details em:after { content: "show..."; } .details input:checked~em:after { content: "...hide"; }
