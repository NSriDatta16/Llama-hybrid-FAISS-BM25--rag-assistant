Using LWP will follow the redirections for you. You can then interrogate the HTTP::Request object to find out the URI it requested. use LWP::UserAgent qw(); my $ua = LWP::UserAgent->new; my $response = $ua->get('http://www.ritzcarlton.com'); print $response->request->uri . "\n"; Output is: http://www.ritzcarlton.com/en/Default.htm
