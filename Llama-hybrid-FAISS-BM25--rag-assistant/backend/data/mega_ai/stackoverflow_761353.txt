You can try ActiveContent, which is Adobe's solution for embedding flash. It's the default for flex builder, and (I believe) CS4. You can find the source to this in the flex builder directory; $FLEX_BUILDER_LOCATION\sdks\3.3.0\templates\client-side-detection Generally, people prefer swfobject because of it's simplicity. But activecontent is a field-tested solution.
