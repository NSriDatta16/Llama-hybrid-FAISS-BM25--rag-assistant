You can reorganize the query a little to avoid the two identical sub-queries. SELECT t.[NAME] FROM ( SELECT [EventID], [Unregistered].[NAME] FROM [Unregistered] UNION ALL SELECT [EventID], [Registered].[NAME] FROM [Registered] ) t LEFT OUTER JOIN [Event] ON t.[EventID] = [Event].[ID] WHERE t.[EventID] IN ( SELECT [EventID] FROM [Registered] WHERE [NAME] Like '%blabla%' )
