It's as simple as: #!/usr/bin/perl -W use LWP::UserAgent; my $ua = LWP::UserAgent->new; $ua->agent("Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.1.7) Gecko/20070914 Firefox/2.0.0.7"); #pretending to be firefox on linux. for my $i (0..10000) { my $req = HTTP::Request->new(GET => sprintf("http://path/to/url?=%d", $i)); $req->header('Accept' => 'text/html'); my $res = $ua->request($req); if ($res->is_success) { $pagecontent = $res -> content; } # Do whatever with the $pagecontent } This is assuming you want to fetch all 10000 pages. If you only want to fetch particular ones then you should try throwing those numbers in an array, and then have for walk that array, rather than 1..10000
