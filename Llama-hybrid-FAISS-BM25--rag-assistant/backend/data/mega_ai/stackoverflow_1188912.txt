HUD.LastSyncDate = DateTime.Parse(mergeSubscription.SynchronizationAgent.LastUpdatedTime).ToString("MM/dd/yyyy") You can put any format string you want there. But it sounds like what you really want is something more like this: private static DateTime _lastSyncDate; public static DateTime LastSyncDate { get { return _lastSyncDate; } set { _lastSyncDate = value;} } public static string LastSyncDateString { get { return LastSyncDate.ToString("MM/dd/yyyy"); } } Keep it as a datetime in the background and just use the string property for display.
