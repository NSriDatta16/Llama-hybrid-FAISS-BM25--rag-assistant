You want to use the GetItemAt(Point point) function to translate X,Y location to the listview item. Here's quite good article about it: Drag and Drop Using C# . To make the item being dragged visible while dragging, you need to use COM ImageList, which is well described in the following article Custom Drag-Drop Images Using ImageLists .
