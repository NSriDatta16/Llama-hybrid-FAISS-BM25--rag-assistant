The following code works. If the download is stopped by the client, the database is not updated and the link can fire again. $file='pathto.zip'; header("Content-type: application/zip"); header("Content-Disposition: attachment; filename=$file"); header("Pragma: no-cache"); header("Expires: 0"); readfile("$file"); $sql = "UPDATE YOUR TABLE"; exit;
