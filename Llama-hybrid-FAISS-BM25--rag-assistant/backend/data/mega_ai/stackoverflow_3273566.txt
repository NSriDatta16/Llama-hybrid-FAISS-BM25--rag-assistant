While loop is specified, it is not implemented in any browser I am aware of Firefox [thanks Anurag for pointing this out] . Here is an alternate way of looping that should work in HTML5 capable browsers: var myAudio = new Audio('someSound.ogg'); myAudio.addEventListener('ended', function() { this.currentTime = 0; this.play(); }, false); myAudio.play();
