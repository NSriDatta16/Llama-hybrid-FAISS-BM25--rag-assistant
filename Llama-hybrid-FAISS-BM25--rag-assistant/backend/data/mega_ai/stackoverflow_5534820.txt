It's common to rely on rescue to capture the exception, then to return the default value: def compute_average(a, b, c, d, e) total = [a, b, c, d, e].sum.to_f average = [ a, 2*b, 3*c, 4*d, 5*e ].sum / total average.round(2) rescue ZeroDivisionError 0.0 end Also I'd write: average = numerator / denominator unless denominator == 0 then 0 as average = (denominator == 0) ? 0 : numerator / denominator
