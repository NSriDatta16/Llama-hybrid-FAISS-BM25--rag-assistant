Semantically, no, it is not correct. Your form inputs are not paragraphs in any sense of the word. If you're a CSS expert, try using and tags and restyling them to look how you like, since your form fields are an ordered list of items. See A List Apart's article on Prettier Accessible Forms for an example of the HTML and CSS necessary for this format.
