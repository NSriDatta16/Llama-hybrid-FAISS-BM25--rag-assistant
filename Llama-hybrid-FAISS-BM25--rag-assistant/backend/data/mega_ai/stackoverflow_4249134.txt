Have you taken a look at htmlcompressor ? In short it's a: Java HTML/XML Compressor is a very small, fast and easy to use library that minifies given HTML or XML source by removing extra whitespaces, comments and other unneeded characters without breaking the content structure. It's main function is so compress HTML and XML, but it also comes with JSP tags that can be used to compress inline JavaScript blocks by leveraging YUI Compressor. Check out the Google Code page, especially the Compressing selective content in JSP pages section.
