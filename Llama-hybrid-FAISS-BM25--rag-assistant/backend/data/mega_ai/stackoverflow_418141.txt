Ok, you need to be able to run a windows binary for this. But if you can run Total Commander , do this: Select all files with *, and hit ctrl-M In the Search field, paste "transform.php?dappName=Test&transformer=YAML&v_id=" (Leave Replace empty) Press Start It doesn't get much simpler than that. You can also rename using regular expressions via this dialog, and you see a realtime preview of how your files are going to be renamed.
