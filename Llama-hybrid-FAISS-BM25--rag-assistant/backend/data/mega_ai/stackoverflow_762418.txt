You could try something like this: $p = genKey($q,rand()); checkKey($p,$q) == true; function genkey($q,$seed) { if(array_search($_SESSION['seeds'],$seed) === FALSE) { $_SESSION['seeds'][] = $seed; } return hash("sha512", $q . $seed); } function checkKey($p,$q) { $returnVal = false; foreach($_SESSION['seeds'] AS $s) { if(hash("sha512", $q . $s) == $p) { $returnVal = true; break; } } return $returnVal; } Of course, I would recommend something else for storage of valid seeds, but this is just a proof of concept.
