Based on ucron's answer, it's possible to do this without activesupport as follows: module UpcasedName def self.included(base) base.send :alias_method, :name_without_feature, :name base.send :alias_method, :name, :name_with_upcase end def name_with_upcase @name.upcase end end class User attr_accessor :name include UpcasedName end u = User.new u.name = 'john' puts u.name
