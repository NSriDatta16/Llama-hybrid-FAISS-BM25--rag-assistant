------------------------------------------------------------------------------ |Ethernet | IPv4 |UDP | Data |Ethernet checksum| ------------------------------------------------------------------------------ 14 bytes 20 bytes 8 bytes x bytes 4 bytes \ (w/o options) / \___________________________________________/ | MTU If your MTU is 1500, you have 1500-20-8 = 1472 bytes for your data. If you exceed that, the packets will be fragmented ,i.e. split into more packets. There might be more layers involved, e.g. 4 byte a vlan header if you're on top of a vlan ethernet. Some routers inbetween you and the destination might add more layers.
