I will assume for the purpose of this answer that your HTML is like such: table.onetable { width: 100%; background-color: #F00; margin: 0; } Hello World The reason why your table isn't filling the body is because your browser's default style imposes a padding to your body tag (usually 8px on most user-agents). Use the following CSS to override that style: body { margin: 0; padding: 0; }
