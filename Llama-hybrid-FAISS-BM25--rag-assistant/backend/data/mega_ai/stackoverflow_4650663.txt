If you approach this from machine learning standpoint, Naive Bayes classifier probably has the greatest work/payoff ratio. A version of it is used in Winnow to categorize news articles. You will need a collection of pages, each tagged with it's proper category. Then you extract words or other relevant elements from each page and use them as features Dr.Dobbs has an article on implementing Naive Bayes
