1) Use the SQL Server Agent (open MS Management Studio) 2) New Job 3) Add Step 4) Choose Transact SQL 5) EXEC MyStroredProc 6) Choose database 7) Add schedule 8) Occurs every hour Add notification to know that process is doing well (or bad). In the properties of SQL Server Agent check that all Auto restart options are switch ON
