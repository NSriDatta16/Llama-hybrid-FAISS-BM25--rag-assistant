AVCaptureVideoPreviewLayer is exactly what you're looking for. The code fragment Apple uses to demonstrate how to use it is: AVCaptureSession *captureSession = ; AVCaptureVideoPreviewLayer *previewLayer = [AVCaptureVideoPreviewLayer layerWithSession:captureSession]; UIView *aView = ; previewLayer.frame = aView.bounds; // Assume you want the preview layer to fill the view. [aView.layer addSublayer:previewLayer];
