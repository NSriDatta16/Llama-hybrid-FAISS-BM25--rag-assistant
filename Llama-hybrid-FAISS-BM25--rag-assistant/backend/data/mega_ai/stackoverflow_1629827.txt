In case you still have this problem this is how i resolved it: listOrgs.SelectedIndex = -1; // Clears the SelectedIndex to avoid the exception listOrgs.DataSource = new Organization().DTListAll(SiteID); listOrgs.DataTextField = "OrganizationName"; listOrgs.DataValueField = "OrganizationID"; listOrgs.DataBind(); //Unless you have "listOrgs.AppendDataBoundItems = true" you don't need to clear the list
