The Pragmatic Programmers also came out with a book on using Ruby to do GUI testing. In particular, they give extensive examples using JRuby to test a swing app. The testing they are doing is mostly to test functionality, but I think it would not be hard to add in some performance measures. The benefit of doing it this way is that you get lots of flexibility, but it is not a packaged tool.
