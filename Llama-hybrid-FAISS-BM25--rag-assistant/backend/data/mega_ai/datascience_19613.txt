Training accuracy is not considered because it can give falsified evaluation value for over-fitting 100% in your case . Hence it is always computed against the batch not entertained for training. This corresponds to the generalized error . For more information see K-Fold cross validation. This might help: https://thebayesianobserver.files.wordpress.com/2012/02/debugging-ml.png
