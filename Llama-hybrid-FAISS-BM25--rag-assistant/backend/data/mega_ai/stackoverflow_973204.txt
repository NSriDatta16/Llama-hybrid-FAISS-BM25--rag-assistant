It's just an HTTP request. It could be that they're blocking the UserAgent curl sends with its requests. You could make it spoof a common browser (like firefox) instead. $useragent="Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.1) Gecko/20061204 Firefox/2.0.0.1"; $ch = curl_init(); // set user agent curl_setopt($ch, CURLOPT_USERAGENT, $useragent);
