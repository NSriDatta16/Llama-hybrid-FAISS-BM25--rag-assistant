Since Xcode or the Application Loader is the only way to submit apps now, renaming the file before using the web interface isn't an option. However, there is a workaround. From Xcode Organizer, choose Save to Disk, pick a file name without a space, and then use Application Loader (not the Xcode Organizer) to send the archive to Apple.
