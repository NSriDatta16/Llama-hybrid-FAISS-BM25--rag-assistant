% format - a url-encoded value for embedding into URLS, e.g. %20 is a space (ascii 20) &nbsp; - named character entity, a non-breaking space in this case U+0026 - a unicode character in hex notation, an & in this case &#...; - a numbered character entity in decimal (base10) &#38; = & &#x...; - a numbered character entity in hex (base 16): &#x26; = &
