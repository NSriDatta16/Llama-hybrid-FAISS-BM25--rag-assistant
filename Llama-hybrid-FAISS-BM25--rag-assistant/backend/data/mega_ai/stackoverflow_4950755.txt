convertto-html returns a list of strings, not a string. So I think $myreport ends up being an object array; e.g., try this: $Content = get-process | Select ProcessName,Id $headerString = " Foo. " $footerString = " " $MyReport = $Content | ConvertTo-Html -fragment -precontent $headerString -postcontent $footerString get-member -input $MyReport Instead force $myreport to be a string before passing it to send-SMTPMail: $MyReport = ( $Content | ConvertTo-Html -fragment -precontent $headerString -postcontent $footerString ) -join "`n";
