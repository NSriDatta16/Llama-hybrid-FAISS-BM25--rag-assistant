You need to explicitly specify self (or in your case, me ) when referring to a method of the current class: if me.has_no_changeset(ticket): You're using me instead of self - that's legal but strongly discouraged. The first parameter of member functions should be called self : def validate_ticket(self, req, ticket): # [...] if self.has_no_changeset(ticket):
