I tend to break an equation down into its root parts. def sum(array) array.inject(0) { |result, item| result + item } end def average(array) sum(array) / array.length end def sum_squared_error(array) avg = average(array) array.inject(0) { |result, item| result + (item - avg) ** 2 } end def variance(array) sum_squared_error(array) / (array.length - 1) end def standard_deviation(array) Math.sqrt(variance(array)) end
