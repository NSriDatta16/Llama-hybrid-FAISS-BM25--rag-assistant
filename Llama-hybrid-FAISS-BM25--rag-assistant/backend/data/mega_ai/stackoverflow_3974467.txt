Supposing you want to use regex to delete all tags that follow a paragraph with a class named myclass : Search for: ( \sclass\s =\s*["']? myclass ["']?[^> >. ? \s*) Replace with: $1 Note, you must ensure that all p tags in your HTML documents are properly closed.
