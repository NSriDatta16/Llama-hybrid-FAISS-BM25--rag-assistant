you can use a simple xml transform for this. for example: Source source = new StreamSource(new File("imput.xml")); Source style = new StreamSource(new FileInputStream("style-fo.xsl")); TransformerFactoryImpl saxon = new TransformerFactoryImpl(); Transformer transformer = saxon.newTransformer(); transformer.transform(source, new StreamResult(System.out)); transforms the in.xml document with the xsl-fo style sheet into xml-fo and prints in on the console.
