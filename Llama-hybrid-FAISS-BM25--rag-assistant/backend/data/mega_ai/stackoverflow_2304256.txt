_ Public Sub CacheOutput(ByVal Page As Page, ByVal Minutes As Integer) Page.Response.Cache.SetExpires(Date.Now.AddMinutes(Minutes)) Page.Response.Cache.SetCacheability(HttpCacheability.Server) Page.Response.Cache.SetValidUntilExpires(True) End Sub This works. Note that any version of the page that has a querystring or a POST parameter will not be cached.
