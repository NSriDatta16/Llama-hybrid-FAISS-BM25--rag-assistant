pTemp is on the Heap. You add into vector through value, so value is added into vector, vector on its own allocates storage on the heap. Case when contents would be on the stack is: Class Temp { std::vector m_dPtList; } double temp = 10.0; Temp* pTemp = new Temp(); pTemp->m_dPtList.Add (&temp);
