You need add a line: int Foo::bar; That would define you a storage. Definition of static in class is similar to "extern" -- it provides symbol but does not create it. ie foo.h class Foo { static int bar; int adder(); }; foo.cpp int Foo::bar=0; int Foo::adder() { ... }
