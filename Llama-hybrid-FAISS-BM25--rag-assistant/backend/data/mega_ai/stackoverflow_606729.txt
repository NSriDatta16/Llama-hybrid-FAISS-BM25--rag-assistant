You can see a clear difference in the hierarchy view of a given class (press F4 on the Class name). Then press the "Show all inherited members" button on the detail view. alt text http://help.eclipse.org/ganymede/topic/org.eclipse.jdt.doc.user/tips/images/hierarchy-lockview.png Methods/variables declared in the current type are shown in blue, inherited ones in black.
