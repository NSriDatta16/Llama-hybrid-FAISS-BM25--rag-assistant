If the function doesn't throw any errors, then lua_pcall will: Remove the parameters from the stack Push the result to the stack This means that, if your function doesn't throw any errors, you can use lua_setfield right away - lua_pcall will work just like lua_call : lua_pushstring (lua, "funcname"); lua_gettable (lua, LUA_GLOBALSINDEX); lua_pushstring(lua, "someparam"); lua_pcall (lua, 1, 1, 0); lua_setfield(L, LUA_GLOBALSINDEX, "a"); /* set global 'a' */ would be the equivalent of: a = funcname(someparam)
