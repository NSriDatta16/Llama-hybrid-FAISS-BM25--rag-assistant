You can browse all the jQuery plugins tagged with "cookie" here: http://plugins.jquery.com/plugin-tags/cookies Plenty of options there. Check out the one called jQuery Storage, which takes advantage of HTML5's localStorage. If localStorage isn't available, it defaults to cookies. However, it doesn't allow you to set expiration.
