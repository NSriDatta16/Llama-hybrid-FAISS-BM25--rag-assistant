Symfony supports template inheritance , it looks so much like django that the syntax looks compatible with django or other beautiful jinja python engines . Elegance, like using PHP as a template language: title; ?> comments as $comment): ?> body; ?> comments->count(); ?> Or even more elegant, jinja: {{ page.title }} {% for comment in page.comments %} {{ comment.body }} {% endfor %} {{ page.comments.count %}
