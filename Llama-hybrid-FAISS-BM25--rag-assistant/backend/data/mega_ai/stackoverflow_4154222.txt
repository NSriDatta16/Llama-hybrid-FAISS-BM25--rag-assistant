You need to use the jQuery UI dialog (http://jqueryui.com/demos/dialog/). It allows you to write things like: $(function () { $("#dialog").dialog({ bgiframe: true, draggable: true, resizable: true, height: 460, width: 800, modal: true, buttons: { Ok: function () { $(this).dialog('close'); } } }); }); Where dialog is the id of a div tag that holds the content you want to display as a dialog.
