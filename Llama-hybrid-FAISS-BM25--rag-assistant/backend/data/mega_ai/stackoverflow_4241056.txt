The src/ is a common place there c/c++ programmers put their sources within the project root. For example: doc/ it common to create subdirectories underneath src/ if you've got a lot of sources files but there are no limitations how to organize this substructure. Keep in mind that a directory structure in completely optional in c++. That is no connection between c++ namespaces and the directory structure.
