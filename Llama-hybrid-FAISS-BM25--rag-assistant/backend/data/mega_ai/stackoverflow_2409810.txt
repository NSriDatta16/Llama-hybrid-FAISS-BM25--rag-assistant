You will be able to detect what user agent is making the request. On that line, you will see something like this: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-GB; rv:1.9.2) Gecko/20100115 Firefox/3.6 (.NET CLR 3.5.30729) And there is your operating system.
