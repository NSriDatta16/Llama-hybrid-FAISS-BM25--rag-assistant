>>> def avg(items): ... return sum(items) / len(items) ... >>> hashes = [a, b] >>> dict([(k, avg([h.get(k) or 0 for h in hashes])) for k in set(sum((h.keys() for h in hashes), []))]) {'y': 0.25, 'x': 0.59999999999999998, 'z': 0.125, 'w': 0.25} Explanation: The set of keys in all of the hashes, no repeats. set(sum((h.keys() for h in hashes), [])) The average value for each key in the above set, using 0 if the value doesn't exist in a particular hash.
