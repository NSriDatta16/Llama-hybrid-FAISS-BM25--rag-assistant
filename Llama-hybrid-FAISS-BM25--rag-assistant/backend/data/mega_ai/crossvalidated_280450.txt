Tree based ensemble machine learning algorithms such as Random Forest and Gradient boost etc. give level of importances (impact) of individual predictors (attributes i.e. X) in determining the target (death by Diabetes i.e. y). You can implement these through python using Scikit-learn libraries. Note that one must also take into consideration the inherent drawbacks and biases of these method .
