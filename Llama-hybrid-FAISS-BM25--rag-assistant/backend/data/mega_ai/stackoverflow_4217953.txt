Silverlight doesn't allow local file system access by default. Your options are: Using IsolatedStorage . Run with elevated permissions . Embedding the file in the assembly, if you only need to read it, as suggested by Jon Skeet . If you need to store data in general, use IsolatedStorage if you can.
