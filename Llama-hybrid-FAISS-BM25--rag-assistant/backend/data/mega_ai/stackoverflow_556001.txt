The answers given while being roughly correct do not produce a correct solution to the question's example. Consider: CL-USER(4): (intern "foo" :keyword) :|foo| NIL CL-USER(5): (eq * :foo) NIL Usually you want to apply STRING-UPCASE to the string before interning it, thus: (defun make-keyword (name) (values (intern (string-upcase name) "KEYWORD")))
