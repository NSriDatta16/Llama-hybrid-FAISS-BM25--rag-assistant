How about using delegate? You can't use hover though, you need mousein and mouseout JQuery API -> Delegate() . The API docs specifically mention the hover event. // something more specific than body would probably be good, but some // common parent of these divs $('body').delegate('.block', 'mousein', function() { $(this).find(".drag").stop().fadeIn(1000); }); $('body').delegate('.block', 'mouseout', function() { $(this).find(".drag").stop().fadeOut(1000); });
