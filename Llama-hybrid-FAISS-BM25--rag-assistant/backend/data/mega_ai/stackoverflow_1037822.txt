Check Mozilla documentation at https://developer.mozilla.org/en/DOM/element.addEventListener . You need to add a third parameter to the function call. Just try adding the value false, like this: Zoom.addEventListener('click', function(){$(document).ready(function(){$("#draggable").draggable({containment: '#imgContainer', scroll: false});});},false);
