Configure an environment with a "standalone" MSTest A series of new test tools is being introduced with Visual Studio 2010. Team Agents provides a very small footprint and includes MSTest (it will be installed in a Visual Studio path). C:\Program Files\Microsoft Visual Studio 10.0\Common7\IDE\mstest.exe You can copy the contents of your test project(s) bin\ directory to the machine with Team Agents installed and execute MSTest against your test container. mstest /testcontainer:x:\test-project\automated-tests.dll"
