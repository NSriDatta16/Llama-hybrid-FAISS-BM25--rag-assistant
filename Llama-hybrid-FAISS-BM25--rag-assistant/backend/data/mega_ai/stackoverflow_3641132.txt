The ui parameter does not have an id attribute, as it is a reference to the element being drug. You need to get the id like ui.draggable.attr('id') , or whatever method you prefer to get the id of an element. $(".full-circle").droppable({ accept: ".unseated_guest", drop: function(event, ui) { //Stuff above var id_dropped = ui.draggable.attr('id'); alert(id_dropped); //Stuff below } });
