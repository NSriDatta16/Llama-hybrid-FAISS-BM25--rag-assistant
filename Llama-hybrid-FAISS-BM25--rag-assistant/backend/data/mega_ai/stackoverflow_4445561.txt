Depending on the organization of this code, you may want to use globals() , .__dict__ , or getattr(obj) here. My example uses globals() . def cmtoinch(x): return x / 2.54 def inchtocm(x): return x * 2.54 def convert(input): num, unit1, unit2 = input.split() func = unit1 + "to" + unit2 if func in globals(): return globals()[func](float(num)) else: raise NotImplementedException()
