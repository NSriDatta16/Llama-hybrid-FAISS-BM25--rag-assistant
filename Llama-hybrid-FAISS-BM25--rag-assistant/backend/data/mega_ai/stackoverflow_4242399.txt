Add the line below: -agentlib:jdwp=transport=dt_socket,address=1044,server=y,suspend=n to run eclipse in debug mode For eclipse, click on debug as icon > debug configurations > add a new remote java application > change the project to the project that you need and port to 1044 and click on apply and debug. Put your breakpoint and access the remote application
