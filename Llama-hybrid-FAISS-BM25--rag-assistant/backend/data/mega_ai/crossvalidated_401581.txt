To put it short, you can use aov or lm. You have to pay attention for categorical variables which must not be of class numeric but factors. You can use dummy or contrast coding. Using + adds the main effects while using * in the formula adds main effects and the interaction of the variables. Further, you can also use : if you want to include only the interaction
