It's not elegant, but it should work... SELECT * FROM SmallTable s INNER JOIN JoinTable j ON j.SmallTableID = s.ID INNER JOIN BigTable b ON b.ID = j.BigTableID WHERE (s.flag1 = 1 and s.flag2 = 1) OR ( (s.flag1=1 OR b.text1 NOT LIKE 'pattern1%') AND (s.flag2=1 OR b.text1 <> 'value1') )
