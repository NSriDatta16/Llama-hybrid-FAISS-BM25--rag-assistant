Yes you can, provided: You will reconnect if you lose the connection You can reset the connection state if something strange happens You will detect if the connection "goes quiet", for example if a firewall timeout occurs Basically it requires a good deal of attention to failure cases and correct recovery; connecting and disconnecting often is a lot easier.
