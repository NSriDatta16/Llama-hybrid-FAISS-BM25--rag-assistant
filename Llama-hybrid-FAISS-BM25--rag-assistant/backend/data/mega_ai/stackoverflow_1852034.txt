Putting the following in your .emacs file will help. You will either have to restart Emacs or evaluate the code. (define-key global-map [ns-drag-file] 'my-ns-open-files) (defun my-ns-open-files () "Open files in the list `ns-input-file'." (interactive) (mapc 'find-file ns-input-file) (setq ns-input-file nil))
