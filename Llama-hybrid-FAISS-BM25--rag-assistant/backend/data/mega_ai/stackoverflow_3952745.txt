This is actually returning an ISingleResult int currentRating = (int) db.sproc_GetAverageByPageId(pageId).ReturnValue; Change your sp to : ALTER procedure [dbo].[sproc_GetAverageByPageId]( @PageId int ) as return (select sum(score) from votes where pageId = @PageId)/(select count(*) from votes where pageId=@PageId)
