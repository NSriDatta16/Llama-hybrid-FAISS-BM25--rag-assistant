top two elements of an associative array in php

I've accomplished the end goal for this already, but maybe there is a more elegant way of accomplishing this. If I have an array like such: $data = array( 'wood' => 2, 'metal' => 5, 'plastic' => 3, ); I want to get the top 2 key/value pairs from $data (ie metal:5 & plastic:3). Here's what I came up with: arsort($data); //put values in order reset($data); //set pointer to first element $first = each($data); //assign first element to $first array_shift($data); //remove first element from array reset($data); //set pointer to the new first element $second = each($data); //assign the new first element to $second
