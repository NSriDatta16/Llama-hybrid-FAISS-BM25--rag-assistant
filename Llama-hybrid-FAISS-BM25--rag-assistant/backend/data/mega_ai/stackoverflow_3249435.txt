For performance and organizational reasons, I like to keep the markup in the HTML (instead of building big HTML strings in javascript), so I usually create hidden template nodes for this sort of thing. The jQuery function you want is replaceWith() . HTML: ORIGINAL TEXT ORIGINAL TEXT2 ORIGINAL TEXT SCRIPT: $(document).ready(function() { $('span.error').each(function(i,v) { var text = $(this).text(); var newBlock = $('#error-template').clone().removeAttr('id').show(); newBlock.find('strong:first').text(text); $(this).replaceWith(newBlock); }); }); See it in action at jsfiddle.
