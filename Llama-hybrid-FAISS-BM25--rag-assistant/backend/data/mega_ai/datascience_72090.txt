Automatic image annotation has its limitations. It's basically another already trained machine learning model that's doing the annotation for you. Keep in mind that automatic annotation also requires another step in between, which is selecting elements with low prediction confidence and review them by hand. These are the most problematic elements when you use an automatic tool to label images and almost always require manual review.
