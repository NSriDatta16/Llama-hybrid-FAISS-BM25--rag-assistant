You can try binding a css change to the mousedown event using jQuery. And then use mouseup() to change the cursor back when the user has finished dragging. e.g.: $(document).ready(function() { $('#sideBar ul li .template').mousedown(function() { $(this).css('cursor', 'move'); }); $('#sideBar ul li .template').mouseup(function() { $(this).css('cursor', 'pointer'); }); });
