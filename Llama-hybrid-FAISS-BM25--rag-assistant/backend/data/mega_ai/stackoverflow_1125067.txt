If you're using LWP to fetch the image, you can look at the content-type header returned by the HTTP server. Both WWW::Mechanize and LWP::UserAgent will give you an HTTP::Response object for any GET request. So you can do something like: use strict; use warnings; use WWW::Mechanize; my $mech = WWW::Mechanize->new; $mech->get( "http://stackoverflow.com/content/img/so/logo.png" ); my $type = $mech->response->headers->header( 'Content-Type' );
