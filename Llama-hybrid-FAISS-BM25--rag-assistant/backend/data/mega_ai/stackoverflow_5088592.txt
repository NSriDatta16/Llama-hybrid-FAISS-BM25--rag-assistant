I had the same problem and fixed it like that: from cStringIO import StringIO def wrapped_svm_predict(*args): """Run :func:`svm_predict` with no *stdout* output.""" so, sys.stdout = sys.stdout, StringIO() ret = svm_predict(*args) sys.stdout = so return ret
