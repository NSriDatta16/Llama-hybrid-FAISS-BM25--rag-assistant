As an alternative you could add an in_stock -method to your StorageItem class. Something along these lines: def in_stock(self): if 'destroyed' and 'permanent_retrieval' not in self.itemstatushistory_set.latest().status.description: return True Then you could simplify your list comprehension: total_items_in_stock = [item for item in storage_items if item.in_stock()]
