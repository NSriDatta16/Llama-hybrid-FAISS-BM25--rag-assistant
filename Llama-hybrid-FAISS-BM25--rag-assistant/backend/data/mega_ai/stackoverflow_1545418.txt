I would define an implicit conversion to a Scala Double implicit def javaToScalaDouble(d: java.lang.Double) = d.doubleValue Now your code works! And similarly expressions like 1.2 + x work as well. And due to auto-boxing code like the following does compile without requiring implicit conversions. def foo(d: java.lang.Double) = println(d) val z: scala.Double = 1.2 foo(z) -- Flaviu Cipcigan
