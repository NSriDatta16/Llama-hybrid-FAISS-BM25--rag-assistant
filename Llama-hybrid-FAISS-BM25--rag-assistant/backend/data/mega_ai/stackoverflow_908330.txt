If 1 in 20,000,000 instructions causes a page fault then you have a page fault rate of: Page Fault Rate = (1/20000000) You can then calculate your average time per instruction: Average Time = (1 - Page Fault Rate) * 2 ns + (Page Fault Rate * 10 ms) Comes to 2.5 ns / instruction
