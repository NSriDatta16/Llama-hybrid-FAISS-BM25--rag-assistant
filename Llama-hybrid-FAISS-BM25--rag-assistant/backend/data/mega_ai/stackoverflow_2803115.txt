How about RewriteEngine On RewriteCond %{HTTP_USER_AGENT} ^.*mobile.*$ [NC] RewriteRule /path/index.php /path/iphone.php [QSA,L] EDIT: QSA is Query string append , so you don't need to handle that differently. If you want you can add an extra parameter too, like: RewriteRule /path/index.php /path/iphone.php?extra=param [QSA,L]
