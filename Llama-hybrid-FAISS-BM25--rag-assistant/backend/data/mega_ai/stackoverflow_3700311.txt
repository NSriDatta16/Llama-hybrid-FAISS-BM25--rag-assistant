You could use the autovivification pragma to deactivate the automatic creation of references: use strict; use warnings; no autovivification; my %foo; print "yes\n" if exists $foo{bar}{baz}{quux}; print join ', ', keys %foo; It's also lexical, meaning it'll only deactivate it inside the scope you specify it in.
