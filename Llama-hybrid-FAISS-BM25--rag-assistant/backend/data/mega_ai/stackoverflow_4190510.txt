When you say rotate(45) , you are rotating the image 45 degrees? (make sure it's not radians, I don't use the plugin) from the original rotation, so if you want to keep rotating you have to keep adding or subtracting degrees: $(function() { // doc ready var rotation = 0; // variable to do rotation with $(".drag-and-rotatable img").click(function() { rotation = (rotation + 45) % 360; // the mod 360 probably isn't needed $(this).rotate(rotation); }); });
