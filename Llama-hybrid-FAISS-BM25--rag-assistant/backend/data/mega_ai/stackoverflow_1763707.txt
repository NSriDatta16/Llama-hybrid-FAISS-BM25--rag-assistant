You should look up link_to_remote and using AJAX with rails. Here is a nice tutorial to get started: http://railsonedge.blogspot.com/2008/03/tutorial-beginning-ajax-with-rails-20.html The idea would be: The user clicks on "vote" It sends an asynchronous message to the controller to save the vote. (use link_to_remote ) the :complete callback does something (eg "thanks for voting!") the :update changes the voting zone (eg "voted. 6 votes on this") There are a lot of tutorials on the subject.
