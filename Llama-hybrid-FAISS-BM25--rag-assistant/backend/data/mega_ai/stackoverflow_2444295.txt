Edit your config file: [/] tools.sessions.on = True tools.sessions.storage_type = "file" # leave blank for in-memory tools.sessions.storage_path = "/home/site/sessions" tools.sessions.timeout = 60 Setting data on a session: cherrypy.session['fieldname'] = 'fieldvalue' Getting data: cherrypy.session.get('fieldname') Source: http://www.cherrypy.org/wiki/CherryPySessions
