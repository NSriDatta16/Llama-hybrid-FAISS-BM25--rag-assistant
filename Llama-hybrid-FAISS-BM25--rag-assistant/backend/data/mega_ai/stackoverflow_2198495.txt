String.prototype.replace() does not modify the source string, but returns a new, modified string. Try the following: $("#big_img").draggable({ revert: 'invalid', drag : function(e, ui) { $this = $(this); $this.attr("src", $this.attr("src").replace("/Large/","/Small/")); } });
