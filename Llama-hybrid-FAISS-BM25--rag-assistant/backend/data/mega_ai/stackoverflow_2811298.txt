You can do this by writing a subclass of either NSFormatter or NSValueTransformer, depending on your needs. If you want to use the values together with bindings you should go with NSValueTransformer. If the values are user-editable and need to perform validation, use NSFormatter. Here is a quick example of the guts of an NSFormatter: - (NSString *) stringForObjectValue:(id)anObject { if(!anObject || ![anObject respondsToSelector:@selector(doubleValue)]) return nil; NSTimeInterval interval = [anObject doubleValue]; double intervalInSeconds = fabs(interval); double intervalInMinutes = round(intervalInSeconds / SECONDS_PER_MINUTE); if(intervalInMinutes >= 0 && intervalInMinutes = 0 && intervalInSeconds = 5 && intervalInSeconds = 10 && intervalInSeconds = 20 && intervalInSeconds = 40 && intervalInSeconds = 2 && intervalInMinutes = 45 && intervalInMinutes = 90 && intervalInMinutes = 1440 && intervalInMinutes = 2880 && intervalInMinutes = 43200 && intervalInMinutes = 86400 && intervalInMinutes = 525600 && intervalInMinutes
