Other considerations: Pros: Flexible: doesn't enforce one, true workflow So fast that version control becomes a minor task Lightweight branches, easy merging and staging area allow personalized workflows More powerful Very compact repos Cons: No built-in access controls Weak tools for binary files. Doesn't compact changes to binary files in repo and storing binary files prevents auto-handling of line endings in Windows. May be harder to learn because it is more flexible and more powerful. Doesn't always follow CVS/SVN semantics and isn't organized around files.
