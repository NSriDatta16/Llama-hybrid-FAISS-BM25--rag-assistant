If you're using S3 as a storage mechanism, you can't use the geometry method above (it assumes a local file). Paperclip can convert from S3 file to local TempFile with the Paperclip::Geometry.from_file : Here is my updated code: def photo_geometry(style = :original) @geometry ||= {} @geometry[style] ||= Paperclip::Geometry.from_file(photo.to_file(style)) end
