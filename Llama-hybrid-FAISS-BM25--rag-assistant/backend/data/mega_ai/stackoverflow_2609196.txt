A couple of suggestions: Add a tooltip to the JTable using setToolTipText(String) . You'll need to add it to the surrounding JScrollPane in order for the tooltip to be displayed when the user hovers on the empty viewport. Add a titled Border ; e.g. scrollPane.setBorder(BorderFactory.createTitledBorder("Drag items here"));
