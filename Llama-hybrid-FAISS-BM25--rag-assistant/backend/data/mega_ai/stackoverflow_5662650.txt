Try to add the following to the response header and the IE will not cache again: response.addHeader("Pragma", "no-cache"); response.addHeader("Cache-Control", "no-cache"); response.addHeader("Cache-Control", "no-store"); response.addHeader("Cache-Control", "must-revalidate"); response.addHeader("Cache-Control", "Post-Check=0"); response.addHeader("Cache-Control", "Pre-Check=0"); response.addHeader("Expires", "Mon, 1 Jan 2006 05:00:00 GMT");//in the past NOTE: the code is java code
