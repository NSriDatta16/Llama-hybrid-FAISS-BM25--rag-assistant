You add them up, and divide by the number of values. The exclusion of NULLs from the denominator of the average operation is a little tricky: SELECT (IFNULL(x, 0) + IFNULL(y, 0) + IFNULL(z, 0)) / (IIF(ISNULL(x), 0, 1) + IIF(ISNULL(y), 0, 1) + IIF(ISNULL(z), 0, 1)) FROM YourTable
