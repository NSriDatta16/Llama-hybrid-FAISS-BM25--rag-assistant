Well, lxml.etree provides full XPath support, which allows you to address the text items: >>> import lxml.etree >>> fragment = ' text1 childtext1 text2 childtext2 text3 ' >>> div = lxml.etree.fromstring(fragment) >>> div.xpath('./text()') ['text1', 'text2', 'text3']
