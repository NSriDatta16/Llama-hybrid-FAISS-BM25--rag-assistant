According to W3C standards, the correct is window.localStorage because the localStorage attribute is part of window object. And it is not a function, it is an attribute. Source: http://dev.w3.org/html5/webstorage/#dom-localstorage Example: http://www.rajdeepd.com/articles/chrome/localstrg/LocalStorageSample.htm
