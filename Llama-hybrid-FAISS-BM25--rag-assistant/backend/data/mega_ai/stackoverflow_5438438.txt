There are several ways to set the line-height, the easiest way is echo ' '.$myVar.' '; A more elegant way would be to define a class or set for all your paragraph tags a line-height. I. css global line-height for all paragraph tags p {line-height: 12px} II. css file with class for line height p.foo {line-height: 12px} corresponding php code echo ' '.$myVar.' ';
