Make the changes as part of a an install or upgrade script in your module's "sql" directory. In your module's "config.xml" file increment the version number of every matching change and also remember to fill in the node. Because the script is run within the context of Magento you have access to all the normal functionality too, updates don't have to be in the form of SQL commands.
