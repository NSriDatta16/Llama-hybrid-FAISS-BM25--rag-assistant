Copy the data to the column A . Select the data. Data -> Text to Columns... Delimited ( Next > ) Semicolon Finish Now you have data in columns A-F . In G1 enter: =INDEX(A1:F1,1,MATCH("~",IF(ISNUMBER(FIND("@",A1:F1)),A1:F1),-1)) and press Ctrl+Shirt+Enter. Drag the formula down.
