They do plain and simple browser checking via the User-Agent header. If they don't like it, they send a 403 (forbidden). Simply add one generic and it works. The following line is the first thing I try whenever I encounter this problem and it works close to always: URLConnection urlc = url.openConnection(); urlc.addRequestProperty("User-Agent", "firefox");
