If Paolo Bergantino is correct, another viable alternative is to use Jquery's LiveQuery. Documentation can be found at http://docs.jquery.com/Plugins/livequery Your code would probably look something like this: $(document).ready(function(){ $("img#sign_up_close").livequery(function() { $(this).hover(function() { $("img#sign_up_close").attr("src","images/buttons/btn_type1_23_close_pp_icon_.gif"); }, function() { $("img#sign_up_close").attr("src","images/buttons/btn_type1_23_close_pp_icon.gif"); }); }); });
