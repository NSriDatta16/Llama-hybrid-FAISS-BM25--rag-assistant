Another possibility is to create a gant script. For example a HelloWorld.groovy script can be created with: grails create-script hello-world Within this script you have access to the spring context via the appCtx variable and the GrailsApplication instance via the grailsApp variable. You can use the latter to load the User domain classes with def artistClass = grailsApp.classLoader.loadClass("com.example.User") and you can then invoke GORM methods in the usual way: artistClass.list()
