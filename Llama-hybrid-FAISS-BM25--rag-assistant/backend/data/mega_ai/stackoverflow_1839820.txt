Firstly, you will need a Payment Gateway that supports recurring payments, such as protx. Once you have that set up, they will provide you with an API, usually based on a web portal, which you can use to set up and take recurring payments. In your code when you are processing payment, you simply pass the required variables through in the post to the web address protx provide you and they will handle the set up and retrieval of funds. Paypal also support recurring payments.
