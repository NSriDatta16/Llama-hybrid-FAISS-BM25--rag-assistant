The server could detect it if advertises itself in the form of a user-agent string, or if the server makes use of it's own browser plugin, that plugin could be used to detect the presence of other plugins. However, if "your friend" was using greasemonkey to cheat, then the game probably detects the fact that he was cheating (e.g. submitting an inhuman number of requests per minute, say) and guessed it was greasemonkey. The moral of the story is: don't cheat.
