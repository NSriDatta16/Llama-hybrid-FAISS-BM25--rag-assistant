No. In Interface Builder (IB) and Xcode, you create links between interface elements and controller code via outlets and actions . This isn't as trivially easy as working with VS, but it encourages you to keep your View code and your Controller code separate; it promotes good practices. The IB User Guide will give you everything you need to know to get started.
