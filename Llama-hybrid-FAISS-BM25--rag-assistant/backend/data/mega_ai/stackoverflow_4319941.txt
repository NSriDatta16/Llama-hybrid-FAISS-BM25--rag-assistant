Please refer to this Magento wiki page http://www.magentocommerce.com/wiki/doc/webservices-api/custom-api#creating_custom_adapter_for_api . Steps: You need to create a new API Server Adapter that should implement Mage_Api_Model_Server_Adapter_Interface . Create a controller that will run your api server adapter Implement Mage_Api_Model_Server_Adapter_Interface::run() method for process JSON request and return result in JSON. See Mage_Api_Model_Server_Handler_Abstract for understanding Magento API workflow.
