Since you have gigantic files, prepare for some kind of connection recovery. Whatever method you will find out to get something from the http (WebClient, TcpStream, ...) you should probably code with recovery in mind from the start. That should be focus here. For that, it would be imperative to check if Stream returned from GetResponseStream() supports Seek() .
