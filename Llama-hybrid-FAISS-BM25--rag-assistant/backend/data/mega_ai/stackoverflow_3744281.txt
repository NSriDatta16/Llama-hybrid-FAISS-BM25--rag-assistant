BOOLs are just chars, either 0 or 1. As such, you don't need to use a storage keyword in the property declaration, so it should be: @property (readwrite) BOOL namesVisible; Second, when logging a BOOL, use the int format string, %d, or pass in a string: NSLog(@"My Boolean: %d, or %@", object.namesVisible, object.namesVisible ? @"Yes" : @"No");
