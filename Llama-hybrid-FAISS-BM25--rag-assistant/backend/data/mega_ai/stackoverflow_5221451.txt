localStorage stores values as strings, so you need to JSON serialize your objects on the way in and deserialize them on the way out. For example: var data = {'A': 9}; localStorage['screenshots'] = JSON.stringify(data); // Later/elsewhere: var data = JSON.parse(localStorage['screenshots']); // 9 console.log(data.A);
