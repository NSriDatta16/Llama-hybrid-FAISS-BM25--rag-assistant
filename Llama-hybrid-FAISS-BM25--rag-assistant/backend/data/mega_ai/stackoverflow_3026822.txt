You need to add &type=user_agent to the request. You will get the AuthToken with a hash marker in the following format. http://yourredirecturi#code=[accesstoken] If you set the request to &type=web_server , you'll get the AuthToken as a query string parameter: http://yourredirecturi&code=[accesstoken] . Here is a full explanation of how to implement the Facebook's OAuth protocol. The code samples are in ASP.NET MVC, but it should translate well enough to any language: Facebook Platform's OAuth 2.0 Protocol and ASP.NET MVC
