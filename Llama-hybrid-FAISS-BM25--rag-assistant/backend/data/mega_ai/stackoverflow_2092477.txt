Servlets are part of Java EE. If you look for javax.servlet you'll find it in the javaee jar. Some of the technologies in Java EE (from http://java.sun.com/javaee/technologies/ ) are: Servlets Web Services Enterprise Java Beans Java Persistence Java Message Services Java Transactions Java API for XML Processing Java Mail Java Authentication
