Using your own storage is an option but this means its some more development work for you. How about this option. Modify your WebPart and add the option to the Web Part property. Create a new permission level and add the following permission Update Personal Web Parts - Update Web Parts to display personalized information. This will make sure that users with read only access can only update webparts. Step 2 will allow them to odify all webparts. You can modify each webpart and remove the following option for webparts which you dont want you users to edit. Allow Editing in Personal View [Advanced] This way you can lock all webparts except the one you want.
