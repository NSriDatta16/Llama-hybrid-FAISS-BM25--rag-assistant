Cookies are for web pages. If you want to display web pages, use a UIWebView. If you want to store persistent data, use NSUserDefaults. If you want to parse cookies sent from a server, use NSURLConnection and parse the headers. If you want to see what cookies have been set by a UIWebView running in your applicaiton, use NSHTTPCookieStorage. NSHTTPCookieStorage Class Reference These cookies are shared among all applications and are kept in sync cross-process. That is only true if your application runs as root on the iPhone.
