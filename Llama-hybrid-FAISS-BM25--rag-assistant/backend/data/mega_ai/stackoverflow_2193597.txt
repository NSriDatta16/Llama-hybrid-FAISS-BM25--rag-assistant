This should work and is a little more elegant: INSERT INTO TABLENAME(col1, col2) VALUES ('xxx', 'yyy') ON DUPLICATE KEY UPDATE col1 = VALUES(col1) Note that you don't need to update the primary key part of the row. We know that's the same because there was a collision.
