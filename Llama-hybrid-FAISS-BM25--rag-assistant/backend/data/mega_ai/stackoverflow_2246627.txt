Monitor the overall server load (say, load average or equivalent of vmstat) in python? >>> import psutil, subprocess >>> subp = subprocess.Popen('python', stdout=subprocess.PIPE, stderr=subprocess.PIPE) >>> proc = psutil.Process(subp.pid) >>> rss, vms = proc.get_memory_info() >>> print "Resident memory: %s KB" %(rss / 1024) Resident memory: 136 KB >>> print "Virtual memory: %s KB" %(vms / 1024) Virtual memory: 356 KB >>> print proc.get_memory_percent() 0.00324324118077 Monitor the cpu load of the processes I spawn? >>> proc.get_cpu_percent() 0.0 Kill processes I've spawned if they're taking too long or taking too much cpu? >>> proc.kill() >>>
