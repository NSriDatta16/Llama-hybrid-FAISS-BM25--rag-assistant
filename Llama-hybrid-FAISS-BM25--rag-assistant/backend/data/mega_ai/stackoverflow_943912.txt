The Java HotSpot JIT compiler can detect "hotspots" and adaptively modify the executing code so it delivers better perfomance. Read about it here . On the other hand, if you want to write code which is efficient to begin with, read a book such as " Hardcore Java " by Robert Simmons or " Java Concurrency in Practice " by Brian Goetz.
