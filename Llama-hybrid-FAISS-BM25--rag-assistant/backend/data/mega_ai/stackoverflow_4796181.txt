In Opera 11.00, Revision 1156, the history API supported are these >>> history. back, current, forward, go, length, navigationMode The full HTML5 history API is not yet covered by Opera 11.00. In general if you would like to discover, explore what is supported, you can easily use the console mode of dragonfly , the Web developer tool.
