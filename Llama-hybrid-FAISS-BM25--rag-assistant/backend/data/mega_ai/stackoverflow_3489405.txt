Qt - accessing the bundle path

The Qt documentation " Mac Differences " page provides the following code for accessing an application's bundle path: CFURLRef appUrlRef = CFBundleCopyBundleURL(CFBundleGetMainBundle()); CFStringRef macPath = CFURLCopyFileSystemPath(appUrlRef, kCFURLPOSIXPathStyle); const char *pathPtr = CFStringGetCStringPtr(macPath,CFStringGetSystemEncoding()); qDebug("Path = %s", pathPtr); CFRelease(appUrlRef); CFRelease(macPath); However, what is the advantage of that over something simpler, like the following: QDir dir = QDir(QCoreApplication::applicationDirPath()); dir.cdUp(); dir.cdUp(); return dir;
