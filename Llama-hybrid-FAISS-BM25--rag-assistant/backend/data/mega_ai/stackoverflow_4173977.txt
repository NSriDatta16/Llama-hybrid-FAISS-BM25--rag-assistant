/tmp is the temporary directory on unix systems. It's specified by FHS. It's cleaned out when the system is rebooted. If you need more persistent data storage than that there's /var/tmp which is not cleaned out after reboots. Also FHS. http://en.wikipedia.org/wiki/Filesystem_Hierarchy_Standard
