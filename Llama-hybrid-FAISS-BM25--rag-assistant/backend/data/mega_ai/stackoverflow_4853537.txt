Just throwing it out there, but have you tried replacing the where clause with an inner join? Something like this: update dashboardtasks set deptdate = s.deptdate, tour = s.tour, tasktype = s.tasktype, [desc] = s.[desc], duedate = s.duedate, compdate = s.compdate, comments = s.comments, agent = s.agent, compby = s.compby, graceperiod = s.graceperiod from staggingtasks as s inner join dashboardtasks on s.tour=dashboardtasks.tour and s.taskname=dashboardtasks.taskname and s.deptdate=dashboardtasks.deptdate
