I haven't tried myself, but according to MSDN, to view a transform file (MST) you need to open your MSI database and then use the ApplyTransform method with the msiTransformErrorViewTransform parameter. This will give you a temporary _TransformView table, which you can query to get the desired information. So, your code should look like this: Const msiOpenDatabaseModeReadOnly = 0 Const msiTransformErrorViewTransform = 256 Dim installer, database Set installer = CreateObject("WindowsInstaller.Installer") : CheckError Set database = installer.OpenDatabase(MSIFileName, msiOpenDatabaseModeReadOnly) : CheckError database.ApplyTransform MSTFileName, msiTransformErrorViewTransform : CheckError
