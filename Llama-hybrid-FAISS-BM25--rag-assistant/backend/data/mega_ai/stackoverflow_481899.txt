A nice and elegant (in my mind :-) using only built-ins is to put it into an array var='system-source-yyyymmdd.dat' parts=(${var//-/ }) Then, you can find the parts in the array... echo ${parts[0]} ==> system echo ${parts[1]} ==> source echo ${parts[2]} ==> yyyymmdd.dat Caveat: this will not work if the filename contains "strange" characters such as space, or, heaven forbids, quotes, backquotes...
