Use the following XPath expression : number( translate(tr[@class='chartTableRow']/td/span[@id='cheapest wine'], ',', '.' ) ) where the current node from which the XPath expression is evaluated is the parent of the XML fragment shown in your question. The XPath expression above evaluates to 9.7
