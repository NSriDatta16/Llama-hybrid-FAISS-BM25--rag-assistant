This doesn't answer your initial question, but maybe you can use the builtin progress bar instead: $ua->show_progress $ua->show_progress( $boolean ) Get/set a value indicating whether a progress bar should be displayed on on the terminal as requests are processed. The default is FALSE. (from LWP::UserAgent POD)
