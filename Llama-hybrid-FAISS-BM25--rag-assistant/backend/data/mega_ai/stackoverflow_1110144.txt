Use the "Storage" Property, with the name of the column: [Column( Name="TBL_REGISTRATION_PK", Storage="TBL_REGISTRATION_PK", IsPrimaryKey = true, IsDbGenerated = true, AutoSync = AutoSync.OnInsert )] public Guid RegistrationID { get { return _RegistrationID; } set { _RegistrationID = value; } } Because the storage name of the column is TBL_REGISTRATION_PK.
