After jQuery 1.7 the preferred methods are .on() and .off() Sean's answer shows an example. Now Deprecated: Use the jQuery functions .live() and .die() . Available in jQuery 1.3.x From the docs: To display each paragraph's text in an alert box whenever it is clicked: $("p").live("click", function(){ alert( $(this).text() ); }); Also, the livequery plugin does this and has support for more events.
