We do use code coverage, it is integrated in our nightly build. There are several tools to analyze the coverage data, commonly they report statement coverage branch coverage MC/DC coverage We expect to reach + 90% statement and branch coverage. MC/DC coverage on the other hand gives broader sense for test team. For the uncovered code, we expect justification records by the way.
