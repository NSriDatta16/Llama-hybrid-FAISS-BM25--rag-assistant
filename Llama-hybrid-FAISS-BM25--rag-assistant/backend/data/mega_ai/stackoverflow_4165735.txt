You could try adding some more headers which sometimes seem to fix IE issues: header("Pragma: public"); header("Expires: 0"); header("Cache-Control: must-revalidate, post-check=0, pre-check=0"); But as meagar says, IE6 doesn't like blank width and height attributes, and always requires a value in there.
