Try using this code: var range = window.getSelection().getRangeAt(0); var content = range.cloneContents(); After this code is executed, content will be a document fragment that contains a copy of the select DOM nodes. Note that event listeners will not be cloned. For more information go to https://developer.mozilla.org/en/DOM/range.cloneContents
