As generically as possible with nothing but a Map . public static K getKey(Map map,V val){ for(Map.Entry entry:map.entrySet()){ if(entry.getValue().equals(val)){ return entry.getKey(); } } return null; } This will only return the key of the first value that matches. If you want to return all the keys for a specific value, then you can modify this method to accumulate they keys into a List and return that instead. (Don't get your hopes up for a speedy retrieval...)
