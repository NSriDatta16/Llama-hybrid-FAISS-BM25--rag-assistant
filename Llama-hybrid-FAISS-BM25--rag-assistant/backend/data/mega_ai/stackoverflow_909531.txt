If you have DDL to create the tables, parse off the table name and prepend some generated table name for this. Depending on the SQL engine you are using, you have different ways to query the schema: Using SQLite: PRAGMA table_info(my_table) PRAGMA table_info(temporary_my_table) Or on MySQL: SELECT table_name, column_name, data_type FROM information_schema.columns WHERE table_schema = 'my_schema' AND (table_name LIKE '%my_table';
