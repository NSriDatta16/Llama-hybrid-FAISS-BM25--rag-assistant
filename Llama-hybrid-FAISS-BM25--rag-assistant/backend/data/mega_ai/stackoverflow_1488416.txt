A SQL Server Agent job can do this. In Management Studio, create a new job, and paste in your restore query, e.g.: RESTORE DATABASE [test] FROM DISK = N'C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Backup\test.bak' WITH FILE = 1, NOUNLOAD, STATS = 10 GO
