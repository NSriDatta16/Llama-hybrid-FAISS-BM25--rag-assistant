I've actually figured my own way to get this working leveraging the built-in Set functions. Given: $input = array ( 'Post.id' => 1, 'Post.title' => 'Some post title.', 'Post.Tag.0.id' => 4, 'Post.Tag.0.name' => 'cakephp', 'Post.Tag.1.id' => 7, 'Post.Tag.1.name' => 'mysql', ); This code will put that into a nested associative array. $output = array(); foreach ($input as $key => $value) { $output = Set::insert($output, $key, $value); } Here's the docs for Set::insert()
