Use the OfType () method, which will result in EF doing an INNER JOIN to the specified T if your using TPT, or a filter based on the discriminator if your using TPH. public TPerson Get (int PersonId) where TPerson : Person { return ctx.People .OfType () .SingleOrDefault(x => x.PersonId == PersonId); } And that will work just like you wanted: Agent a = repository.Get (5005);
