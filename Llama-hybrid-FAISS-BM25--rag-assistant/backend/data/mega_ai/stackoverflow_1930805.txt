To disable: #pragma warning disable 0162 To restore: #pragma warning restore 0162 For more on #pragma warning , see MSDN . Please note that the C# compiler is optimized enough to not emit unreachable code. This is called dead code elimination and it is one of the few optimizations that the C# compiler performs . And you shouldn't willy-nilly disable the warnings. The warnings are a symptom of a problem. Please see this answer .
