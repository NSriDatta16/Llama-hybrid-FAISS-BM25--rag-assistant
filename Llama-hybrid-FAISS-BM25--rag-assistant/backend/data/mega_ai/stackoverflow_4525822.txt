Another issue is that in a web based app/component you are missing out on platform specific technologies and frameworks like core data, GCD (or any other threading model - which in web you might have do with e.g. Ajax),uitableview with caching etc. Should you hit a wall when it comes to performance or storage,then you have many more options in the native framework.
