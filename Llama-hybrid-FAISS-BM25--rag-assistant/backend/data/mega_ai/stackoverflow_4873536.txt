Using the above answer from Honibis I ended up with this. load in the image and invalidate the picture to cause a refresh picCanvas.Image = image; picCanvas.Invalidate() then in the paint event private void picCanvas_Paint(object sender, PaintEventArgs e) { if (_drag) { using (Pen pen = new Pen(Color.White, 2)) { e.Graphics.DrawEllipse(pen, pos.X, pos.Y, 10, 10); } } }
