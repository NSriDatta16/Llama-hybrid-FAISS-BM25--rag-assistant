The WS-I organization ( http://www.ws-i.org/ ) publishes the BSP 1.0 Testing Tools ( http://www.ws-i.org/deliverables/workinggroup.aspx?wg=testingtools ) which validates WSDL against the WS-I Basic Profile. This is a web service interoperability standard. jMM
