I would put the suffixes in a hash, where the common suffix is the key and the official suffix is the value. Then you can look up the last word in the hash. SUFFIXES = { "ALLEE" => "ALY", "ALLEY" => "ALY" } addy = "183 main allee" last = addy.split.last.upcase addy = addy[0..-last.length-1] + SUFFIXES[last] if SUFFIXES[last] puts addy
