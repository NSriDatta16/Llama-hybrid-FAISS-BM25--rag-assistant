Here's a 'brute force' version. This will loop through every photo. You could make this more elegant by restricting to certain albums if you wanted. tell application "iPhoto" set thePhotos to get every photo repeat with aPhoto in thePhotos if aPhoto's comment contains "theString" then tell aPhoto to set it's comment to "newString" end if end repeat end tell
