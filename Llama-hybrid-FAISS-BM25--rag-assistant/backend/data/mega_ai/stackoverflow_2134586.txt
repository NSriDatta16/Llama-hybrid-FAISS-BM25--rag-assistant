Stylesheet embedding is possible for most browsers but IE. Find the reference to a description and example in the posting below. IE6/7/8 do not support embedded stylesheets by default. You may want to use the workaround enabling IE to process embedded stylesheets from here: http://www.biglist.com/lists/lists.mulberrytech.com/xsl-list/archives/201001/msg00390.html
