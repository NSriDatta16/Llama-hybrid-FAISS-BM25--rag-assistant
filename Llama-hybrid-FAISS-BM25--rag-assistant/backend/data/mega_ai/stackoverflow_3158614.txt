Here you go some hints, some of them silly hints, but just in case: Review your mobile connection, try to open web browser, surf the web, and make sure it actually works I don't know which is the web page your are trying to access but take into account that depending on your phone User Agent (UA), the rendered content might be different (web pages specially designed for mobile phones), or even no content rendered at all. Is it a web page on your own. Try to access that same web page from Firefox, changing the UA (Use the User Agent Switcher for Firefox), and review the code returned. That will be a good start point to figure out what's your problem Ger
