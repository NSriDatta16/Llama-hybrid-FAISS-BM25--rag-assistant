Ok, the anwser is just to set the delegate of my UIPinchGestureRecognizer to self and to override - (BOOL)gestureRecognizer:(UIGestureRecognizer *)recognizer shouldRecognizeSimultaneouslyWithGestureRecognizer:(UIGestureRecognizer*) otherRecognizer { return YES; } Now the pinch is recognized even if the scroll began in the UIWebView
