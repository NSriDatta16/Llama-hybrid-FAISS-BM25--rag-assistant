How to estimate the confidence of a regression prediction for neural network

For a classification task, the output neurons give a continuous output of [0 1], which itself can be treated as confidence, ie, a value closer to 1 means highly confident and closer to 0 means not confident. Similarly, how do we get the confidence of a regression task prediction?
