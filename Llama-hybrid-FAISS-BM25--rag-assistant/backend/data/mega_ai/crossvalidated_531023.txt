There are a few approaches here to derive an index. Repeat Sales Index - Comparing changes in a value over time. The advantage of this is that because it is a repeat transaction, you know it is comparable. It doesn't need to be a sale, just a value. Hedonic Price Index - Which looks at price changes accounting for various including locational and quality based. Random Forest - A machine learning approach. The hpiR packages has worked examples in its vingette.
