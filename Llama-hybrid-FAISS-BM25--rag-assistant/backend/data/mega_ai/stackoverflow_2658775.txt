you need to change function setDraggable(){ $("#topContent img").draggable({ opacity: 0.7, revert: 'invalid', helper: 'clone', }); } to function setDraggable(){ $("#topContent img").draggable({ opacity: 0.7, appendTo: 'body', revert: 'invalid', helper: 'clone', }); } so the element's parent container will be the body
