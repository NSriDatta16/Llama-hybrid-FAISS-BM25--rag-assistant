You can use system() or %x[] depending what kind of result you need. system() returning true if the command was found and ran successfully, false otherwise. >> s = system 'uptime' 10:56 up 3 days, 23:10, 2 users, load averages: 0.17 0.17 0.14 => true >> s.class => TrueClass >> $?.class => Process::Status %x[..] on the other hand saves the results of the command as a string: >> result = %x[uptime] => "13:16 up 4 days, 1:30, 2 users, load averages: 0.39 0.29 0.23\n" >> p result "13:16 up 4 days, 1:30, 2 users, load averages: 0.39 0.29 0.23\n" >> result.class => String Th blog post by Jay Fields explains in detail the differences between using system, exec and %x[..] .
