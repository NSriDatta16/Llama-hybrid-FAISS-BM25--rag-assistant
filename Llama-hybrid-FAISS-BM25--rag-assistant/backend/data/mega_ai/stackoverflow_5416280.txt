/([^\d]*\d){8}/ Perhaps not the most elegant / efficient way to do it, but it works. Basically it will match eight decimals (optionally, with non-decimals between them). If there are more than eight, it will match too. EDIT As @Tomalak has pointed out, [^\d] equals \D by definition: /(\D*\d){8}/
