any uppercase character that is not followed by an uppercase character: Replace(string, "([A-Z])(?![A-Z])", " $1") Edit: I just noticed that you're using this for enumerations. I really do not encourage using string representations of enumerations like this, and the problems at hand is a good reason why. Have a look at this instead: http://www.refactoring.com/catalog/replaceTypeCodeWithClass.html
