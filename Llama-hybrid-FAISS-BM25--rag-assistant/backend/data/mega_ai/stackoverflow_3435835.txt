I imagine they are just embedding fonts like this: @font-face { font-family: " your FontName "; src: url( /location/of/font/FontFileName.eot ); /* IE */ src: local(" real FontName "), url( /location/of/font/FontFileName.ttf ) format("truetype"); /* non-IE */ } /* THEN use like you would any other font */ .yourFontName { font-family:" your FontName ", verdana, helvetica, sans-serif; } http://randsco.com/index.php/2009/07/04/p680 I tested that and it works for IE8, FF, Chrome and Safari on Windows.
