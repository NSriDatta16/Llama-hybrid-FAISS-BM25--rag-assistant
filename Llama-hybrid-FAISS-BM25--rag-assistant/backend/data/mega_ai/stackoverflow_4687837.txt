I realized that both my question and Andy's solution (using node.parentNode ) can be generalized. Here's a full version that will work for any jQuery selection, even if it has at least one ancestor. /** * Returns true if this selection is part of the current DOM; * false if it's a fragment. * @return {Boolean} */ jQuery.fn.inDom = function() { // Get the first element in the jQuery selection var node = this[0]; while (node) { if (node.nodeType === Node.DOCUMENT_NODE) { return true; } node = node.parentNode; } return false; }
