Change your Boss class to Person then add a @ManyToOne Person superior (assuming every employee has only one superior above). @ManyToMany List superiorList=new ArrayList () if there are more than one superiors for each employee; If a person is a boos(the highest authority in the organization tree) , superior or superiorList will be empty. For roles Create a role class Role @OneToMany List taskList = new ArrayList (); role_code put many to one role in Person class if one person can assign to only one role @ManyToOne Role role or @ManyToMany List roleList if a person can be assigned many roles.
