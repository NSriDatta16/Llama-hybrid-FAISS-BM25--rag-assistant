Don't know any solution in Spring, but Guava 's MapMaker class lets you create a computing map: ConcurrentMap graphs = new MapMaker() .concurrencyLevel(4) .softKeys() .weakValues() .maximumSize(10000) .expireAfterWrite(10, TimeUnit.MINUTES) .makeComputingMap( new Function () { public Graph apply(Key key) { // this is where your values are created on demand return createExpensiveGraph(key); } }); Reference: MapMaker.makeComputingMap(Function ) BTW, Apache Commons / Collections has a similar feature: MapUtils.lazyMap(Map, Transformer)
