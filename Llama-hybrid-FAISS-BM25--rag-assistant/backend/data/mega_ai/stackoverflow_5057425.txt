First of all, your original XML code is not valid XML, that is one reason it could fail you. Second, Java by default does not coalescates the fragmented text content. I have written a blog entry about the details of this, and how you are able to merge the different text nodes by setting a property on the parsing factory: http://www.drillio.com/en/software-development/java/fragmented-xml-text-nodes/
