You have to use NSPropertyListSerialization to convert it. Basically, assuming the object that you're serializing is called myObject and the ultimate file is stored in myPath , you would serialize it to XML using something like the following code fragment: NSString *error = nil; NSData *mySerializedObject = [NSKeyedArchiver archivedDataWithRootObject:myObject]; NSData *xmlData = [NSPropertyListSerialization dataFromPropertyList:mySerializedObject format:NSPropertyListXMLFormat_v1_0 errorDescription:&error]: if( xmlData ) { [xmlData writeToFile:myPath atomically:YES]; } else { NSLog(error); [error release]; }
