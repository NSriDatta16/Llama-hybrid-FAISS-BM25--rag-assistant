The search pattern for a sub-folder needs to include "*.*" at the end to pattern match any file, which would make your code something like the following: var IOstore = IsolatedStorageFile.GetUserStoreForApplication(); string searchpath = System.IO.Path.Combine("product", ProductName); searchpath = string.Format("{0}\\*.*", searchpath); string filesInSubDirs[] = IOstore.GetFileNames(searchpath);
