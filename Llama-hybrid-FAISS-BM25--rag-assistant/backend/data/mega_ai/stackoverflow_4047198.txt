Probably the easiest way to do this is to call the svn command line program, passing it the relevant commands and arguments to do whatever it is you want to do. Although there are probably Delphi-compatible Subversion API bindings, calling svn is undoubtedly going to be more straightforward. Most svn command line commands have an --xml switch that writes the output in program-friendly XML format, instead of making you rely on fragile text parsing.
