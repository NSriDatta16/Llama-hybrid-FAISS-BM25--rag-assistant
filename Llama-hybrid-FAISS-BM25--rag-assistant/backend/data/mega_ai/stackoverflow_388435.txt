Zeus assembler , was the best. I'd add a couple of the Spectrum books in there if I could remember the names, still have them at home. One was The Complete Spectrum ROM Disassembly by Ian Logan and Frank O'Hara (ISBN 0 86161 116 0), which was commented and described as if it was the source, a fantastic piece of reverse engineering, including a suggested bug fix for the known ROM bugs. If only flash memory had been around in those days. I also memorised a tiny book called the Z80 Workshop Manual which was a great summary of the processor.
