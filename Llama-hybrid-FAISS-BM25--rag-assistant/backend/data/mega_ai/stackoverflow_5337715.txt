You need to do call event.preventDefault() when you want the default action to be skipped. It's up to you to figure out if the item was being dragged and so should not be navigated, -vs- a standard click. But assuming you store a flag in data, it might look something like this: $('.something').click(function(event) { if ($(this).data('dragging')) { event.preventDefault(); $(this).data('dragging',false); } }
