You can joint multiple regexes into one by using alternate symbol | and creating branches in regex. Pay attention to anchors like ^ , $ and other, because if they appear in one branch, they only work for that branch, not whole regex lyrics = lyrics.gsub(/\n|^\{\"similar\": \[|\]\}$|^\{|\}$/, '')
