I needed to figure this out while working on a visual studio plug-in. It looks like it depends on what you have installed. For example, I have some clear case CommandBars available to me because I work with the clear case integration plug-in. I'll echo that the more common ones are "Code Window", "Project", "MenuBar", "Solution", and "Item". To specifically answer your question, you would use the "Standard" and "Project" command bars. To generate a list of all CommandBars you can create a visual studio plug-in and paste the following code into OnStartupComplete in Connect.cs: CommandBars commandBars = (CommandBars)_applicationObject.CommandBars; StringBuilder sb = new StringBuilder(); foreach (CommandBar cbar in commandBars) { sb.AppendLine(cbar.Name); } Clipboard.SetText(sb.ToString()); Results
