Usage: If you're going to store information that you want to access on different web pages, you can use SessionState If you want to store information that you want to access from the same page, then you can use Viewstate Storage The Viewstate is stored within the page itself (in encrypted text), while the Sessionstate is stored in the server. The SessionState will clear in the following conditions Cleared by programmer Cleared by user Timeout
