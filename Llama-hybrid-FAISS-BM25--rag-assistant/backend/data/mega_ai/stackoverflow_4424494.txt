ECMA 335 spec part 2 14.6.2 has a paragraph about this: The calling convention of T and D shall match exactly, ignoring the distinction between static and instance methods. (i.e. the this parameter if any, is not treated specially). What this sounds to me like that for static methods will be allowed in two variations: Without this, in which case NULL should be passed With the extra first parameter, presuming the type matches what was fed into the newobj call.
