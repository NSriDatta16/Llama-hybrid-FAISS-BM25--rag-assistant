In Keras this can be obtained by using the Functional API to build a multiple-input model. In your case, the sequential (or pixel, if you're doing vision) data will be the main_input , processed by CNN; and the additional data will be the aux_input , to which you can apply fully connected layers. Source: Keras Functional API Documentation
