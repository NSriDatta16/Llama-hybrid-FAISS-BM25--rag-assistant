It's merely a matter of definition, but in this case, yes . CLRS define an ancestor of x as any node on the unique path from the root to x, which by definition includes x. The sentence fragment you quoted begins by mentioning exercise 12.2-6 on the next page, which specifies this: (Recall that every node is its own ancestor.) :-)
