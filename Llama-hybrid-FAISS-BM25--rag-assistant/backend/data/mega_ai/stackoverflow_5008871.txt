If you are not too worried about client side security then you can store the the information in the IsolatedStorage and pass it to WCF or use it in silverlight If you want the information accessible anywhere in your WCF service. Then i suggest using a MembershipProvider that authenticates the user against the Active directory store, then you can access it via the OperationContext
