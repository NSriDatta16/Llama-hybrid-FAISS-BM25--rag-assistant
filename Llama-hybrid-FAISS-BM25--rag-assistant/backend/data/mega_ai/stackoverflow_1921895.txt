In the UISCrollViewController, have you tried passing the touchesEnded on to super, assuming a scroll is not in progress? - (void)touchesEnded:(NSSet *)aTouches withEvent:(UIEvent *)anEvent { if(![self isDragging]) { [super touchesEnded:aTouches withEvent:anEvent]; } }
