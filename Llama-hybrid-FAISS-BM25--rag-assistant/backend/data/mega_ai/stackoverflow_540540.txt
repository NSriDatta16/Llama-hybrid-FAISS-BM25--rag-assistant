To try and help you get there (though not very elegantly, I admit): I think the start and end anchors (^ and $) aren't helping. You may also want to filter the asterisks?: irb(main):001:0> mds = "asfsfsdfsdfsf sfda **joe@example.com** sdfdsf" => "asfsfsdfsdfsf sfda **joe@example.com** sdfdsf" irb(main):003:0> mds.match(/^([^@\s]+)@((?:[-a-z0-9]+\.)+[a-z]{2,})$/i) => nil irb(main):004:0> mds.match(/([^@\s]+)@((?:[-a-z0-9]+\.)+[a-z]{2,})/i) => # irb(main):005:0> mds.match(/([^@\s*]+)@((?:[-a-z0-9]+\.)+[a-z]{2,})/i) => #
