If you are formatting HTML emails, then you need to use: StringBuilder message = new StringBuilder(); message.append("Name: " + model.getName() + " \n"); message.append("Organization: " + model.getOrganization() +" \n"); message.append("Comment: " + model.getComment() +" \n"); contactMessage.setMessage(message.toString()); You need to insert a html line break because the newlines are ignored.
