Use an interface. interface IStorage { void Save(string filename); } Then create your various implementations: class LocalStorage : IStorage { void Save(string filename) { -implementation code- } } And then, in your consumer classes IStorage storage = new LocalStorage(); // uses local storage storage.Save(); storage = new DatabaseStorage(); // now use database storage storage.Save();
