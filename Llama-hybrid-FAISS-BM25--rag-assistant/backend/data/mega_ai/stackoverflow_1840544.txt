Did you try using QtWebKit with PyQt, you can load a specific url and read the content from Python. You could then search for urls and use Webkit again to access it. I think all those can be done with some basic Django(assuming you are using Django on GAE) view testing which will test the response code. Here's a sample QtWebKit PyQt code to get your started if you want to do it the GUI way: import sys import time from PyQt4.QtCore import * from PyQt4.QtGui import * from PyQt4.QtWebKit import * app = QApplication(sys.argv) web = QWebView() settings = web.settings() settings.setAttribute(QWebSettings.PluginsEnabled, True) settings.setAttribute(QWebSettings.JavaEnabled, True) settings.setAttribute(QWebSettings.JavascriptCanOpenWindows, True) settings.setAttribute(QWebSettings.JavascriptCanAccessClipboard, True) settings.setAttribute(QWebSettings.DeveloperExtrasEnabled, True) settings.setAttribute(QWebSettings.ZoomTextOnly, True) settings.setOfflineStoragePath('.') settings.setIconDatabasePath (".") url = 'http://stackoverflow.com' web.load(QUrl(url)) web.show() sys.exit(app.exec_())
