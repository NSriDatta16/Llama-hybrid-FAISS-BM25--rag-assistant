Here is an example of quite a few tests that use both XMLUnit and a custom framework for processing XML. The framework uses StAX to map XML to Java POJOs. https://simple.svn.sourceforge.net/svnroot/simple/trunk/download/stream/src/test/java/org/simpleframework/xml/core/ There are no external dependencies, and test coverage is approx 90% http://simple.sourceforge.net/download/stream/report/cobertura/
