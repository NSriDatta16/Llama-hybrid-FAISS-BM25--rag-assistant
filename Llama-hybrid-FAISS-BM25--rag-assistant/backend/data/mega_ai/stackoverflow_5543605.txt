With Zend Framework 1.11 zend introduced the wurfl adapter which is based on Zend_Http_UserAgent . It allows you to detect mobile devices and bootstrap different layouts. There is a HOW TO on youtube that came with the zend newsletter december 2010: http://www.youtube.com/watch?v=_A8yg73tqOY You don't have to use different modules! Just different layout files!
