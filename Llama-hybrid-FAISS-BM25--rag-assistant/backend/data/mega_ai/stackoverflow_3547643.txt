The event parameter is the DOM Event object . The ui parameter is typically a hash; its properties depend on the event being raised. For the dialog open event, both arguments appear to be null. For other events, such as dragStart , drag and dragStop , ui will contain the current position and offset: $('.selector').dialog({ drag: function(event, ui) { console.log("Current position: " + ui.position); } });
