SQL: update songs set rating='L' where trackid IN (id_list) Code: self.cursor.execute("update songs set rating='L' where trackid IN (%s)"%','.join(['?']*len(id_list)), id_list) self.db.commit() There is also more elegant, but suboptimal solution: self.cursor.executemany("update songs set rating='L' where trackid=?", id_list) self.db.commit()
