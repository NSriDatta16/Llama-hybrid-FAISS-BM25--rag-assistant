window.getSelection().getRangeAt(0); It returns a document fragment. It contains the nodes where the selection begins and ends and some other juicy stuff. Inspect it with FireBug or another JavaScript console, &&|| for more info https://developer.mozilla.org/en/DOM/Selection https://developer.mozilla.org/en/DOM/range https://developer.mozilla.org/en/window.getSelection
