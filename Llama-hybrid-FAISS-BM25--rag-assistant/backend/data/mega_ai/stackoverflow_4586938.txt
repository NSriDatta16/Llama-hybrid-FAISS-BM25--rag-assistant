It means that no row was inserted by a transaction, by that it's more an information than an error. If an INSERT-statement wasn't successfull or an transaction wasn't an INSERT at all, the insertId-property of the resultSet contains this message instead of the ID of the last inserted row. http://docs.phonegap.com/phonegap_storage_storage.md.html#SQLResultSet
