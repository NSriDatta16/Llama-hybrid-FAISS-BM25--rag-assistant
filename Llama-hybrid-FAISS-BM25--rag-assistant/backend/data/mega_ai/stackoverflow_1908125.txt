Be aware that Trac won't even let you browse your repository if it happens to be on a remote server/share/location ! This is very limiting in many cases where you want your trac/subversion web server to be separate from the storage of your repository (on a NAS for example). Here is the Trac ticket related to the issue if you need to know more : Ticket 493 It's been open for some time now, and is the main reason why I'm going to move to Redmine.
