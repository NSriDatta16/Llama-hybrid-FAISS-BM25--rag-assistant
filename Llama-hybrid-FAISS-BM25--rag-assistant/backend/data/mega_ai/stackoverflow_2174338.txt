you could share the same session on all subdomains. That is the code we use to accomplish that :-) void MasterPage_Unload(object sender, EventArgs e) { ///ASP.NET uses one cookie per subdomain/domain, ///we need one cookie for _all_ subdomains. if (Context.Response.Cookies["ASP.NET_SessionId"] == null) return; var sessionCookie = new HttpCookie("ASP.NET_SessionId", Context.Session.SessionID); sessionCookie.Domain = ".yourdomain.com" ; Context.Response.SetCookie(sessionCookie); } inside the Page_Load method is: Unload += MasterPage_Unload; it works great :-) robert
