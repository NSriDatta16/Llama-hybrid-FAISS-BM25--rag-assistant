You never set an IBOutlet. Only IB sets IBOutlets. So in this case, what you meant to say is: cell.flag1.image = [UIImage imageNamed:[rowData objectForKey:@"Flag1"]]; You should be receiving warnings in your code, complaining about a type mismatch. Be sure to pay attention to these warnings.
