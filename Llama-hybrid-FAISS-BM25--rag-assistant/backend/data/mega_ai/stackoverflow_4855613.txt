I guess the problem is that there's no window in the true sense. There's just a div inside the main page acting as DOMWindow. So any links in that div will surely refresh the main doc. Keep everything the same and add this: $('.AjaxDOMWindow a').live("click", function(){ $(this).openDOMWindow({ anchoredClassName:'DOMWindow', draggable: 1, eventType:'click', height:500, loader:1, loaderHeight:16, loaderImagePath:'/js/jquery/DOMWindow/animationProcessing.gif', loaderWidth:17, positionLeft:0, positionTop:0, positionType:'centered', width:700, windowHTTPType:'get', windowSource:'ajax' }); });
