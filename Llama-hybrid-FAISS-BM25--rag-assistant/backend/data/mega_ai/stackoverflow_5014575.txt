Here's an example on how to achieve this: Handle Click Events in the code-behind. For MouseDown - App.Current.MainWindow.DragMove(); For Minimize Button - App.Current.MainWindow.WindowState = WindowState.Minimized; For DoubleClick and MaximizeClick if (App.Current.MainWindow.WindowState == WindowState.Maximized) { App.Current.MainWindow.WindowState = WindowState.Normal; } else if (App.Current.MainWindow.WindowState == WindowState.Normal) { App.Current.MainWindow.WindowState = WindowState.Maximized; }
