You did not mention which package you are using. If you using scikit-learn, sklearn.pipeline.FeatureUnion concatenates results of multiple transformer objects. Something like this: from sklearn.cluster import DBSCAN from sklearn.pipeline import FeatureUnion, Pipeline from skearnsklearn.preprocessing import StandardScaler pipeline = Pipeline([('feats', FeatureUnion([ ('doc2vec', d2v_model, ['sentence1', 'sentence2']), ('numeric', StandardScaler(), ['price']) ])), ('cluster', DBSCAN()) ])
