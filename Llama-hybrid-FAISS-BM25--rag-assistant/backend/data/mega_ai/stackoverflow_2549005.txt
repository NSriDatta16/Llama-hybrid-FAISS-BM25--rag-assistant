My workaround... You get the user Appdata path: String appdata = System.Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData); Add VirtualStore folder to path: String VirtualStoreRoot = appdata + "\\VirtualStore"; Create folder: String backuppath = "C:\\Program Files\\Progname\\backup"; Directory.CreateDirectory(backuppath); Check created in VirtualStore String newpath = VirtualStoreRoot + backuppath.Substring(2); if (Directory.Exists(newpath)){ backuppath = newpath; } I found another stuff: Isolated Storage with .NET
