You could run it through an "external Tools Configuration": it can call any script/command you like, even if that script launches itself your Java program. (source: abdevelopment.ca ) Then, once it is launched, you can initiate a remote debug session, provided your script did launch the Java program with the right options -Djava.compiler=NONE -Xnoagent -Xdebug -Xrunjdwp:transport=dt_socket,address=8000,server=y,suspend=n alt text http://www.nakov.com/blog/wp-content/uploads/2008/08/eclipse-remote-debug-configuration.png
