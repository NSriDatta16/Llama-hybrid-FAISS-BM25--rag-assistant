It seems like it was a caching issue, solved by doing something like this: protected override void OnLoad(EventArgs e) { Response.Cache.SetNoStore(); Response.Cache.SetCacheability(HttpCacheability.NoCache); Response.Cache.SetExpires(DateTime.Now); Response.Cache.SetLastModified(DateTime.Now); Response.Cache.SetAllowResponseInBrowserHistory(false); base.OnLoad(e); }
