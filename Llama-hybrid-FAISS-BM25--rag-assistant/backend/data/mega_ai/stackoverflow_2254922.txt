I like Hunter's solution. You do have access to the inserted link node, though, and you can set its id if you wish: http://ericmiraglia.com/yui/demos/cssid.php YUI().use("get", function(Y) { Y.Get.css("http://ericmiraglia.com/yui/demos/red.css", { onSuccess: function(o) { o.nodes[0].setAttribute("id", "myElementId"); alert(o.nodes[0].getAttribute("id")); } }); });
