You can also do this another way. On your startDrag you stop observing the object. Then you observe the click again which leads to an enddrag function which recreates the first eventhandler. function onDragStart() { Event.stopObserving(this.OBJECT,'click'); Event.observe(this.OBJECT,'click',this.onDragEnd.bindAsEventListener(this)); } function onDragEnd() { Event.stopObserving(this.OBJECT,'click'); Event.observe(this.OBJECT,'click',this.PREVIOUSFUNCTION.bindAsEventListener(this)); } This will catch the leftover click event which is still active after the enddrag to just recreate the original handler. I hope this helps somebody out there :)
