It seems like the page requires cookies. Take a look here to see how to handle cookies with WebClient. To debug your problem you can also install a header sniffing tool like Live HTTP Headers for Firefox or ieHTTPHeaders for Internet Explorer and try to send the same information via WebClient (same User Agent, Referrer etc.).
