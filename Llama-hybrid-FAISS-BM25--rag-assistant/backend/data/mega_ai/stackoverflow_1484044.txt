According to the comments there are two actions you want to do: Send emails - quite easy with the JavaMail API Retrieve emails - You can do it with the JavaMail API using either IMAP or POP3. Set A scheduler to trigger the retrieval. (either Quartz . cron job + servlet, or perhaps even the Java timer will do)
