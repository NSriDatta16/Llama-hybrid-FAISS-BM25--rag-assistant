Try in this order: Pragmatic Real-World Scala - This video shows off all kinds of things that would make a Java developer drool. Programming In Scala - This is simply a great general-purpose programming book. In addition to being a gentle, clear introduction to the language, it's also a fantastic introduction to functional programming concepts and language design. Even if you hate Scala, this book will make you a better programmer. Scala For Java Refugees - Very well-written mostly gentle introduction to major Scala concepts. Another tour of Scala - A Java-centric breakdown of fundamental Scala features.
