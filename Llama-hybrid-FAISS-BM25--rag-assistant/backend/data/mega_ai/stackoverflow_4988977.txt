If you are using Resharper 4x, 5x this is also interesting (FREE) option http://www.agentsmithplugin.com/ Agent Smith is C# code style validation plugin for ReSharper (Visual Studio plugin). Current version includes following features: * Naming convention validation. * XML comment validation. * XML comment, string literals, identifiers and resources (.resx files) spell checking * Smart paste. * XML comment reflowing.
