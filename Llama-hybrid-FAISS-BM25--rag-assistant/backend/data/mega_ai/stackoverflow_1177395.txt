Using magic_quotes_gpc is a bad idea anyway, it's officially deprecated as of 5.3 and gone as of PHP 5.5. More importantly (and immediately), it encourages weak security because it doesn't really do anything useful that couldn't be done by mysql_real_escape_string or prepared statements or stored procedures. If you can get rid of it, please do...
