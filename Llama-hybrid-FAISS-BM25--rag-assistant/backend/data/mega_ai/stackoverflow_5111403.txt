For example, creating an IP packet with the DF (Don't Fragment) flag set: >>> packet = IP(flags=2) # alternatively, IP(flags='DF') >>> packet Reading a packet's flags : >>> packet.flags 2 As for the flag bits, Wikipedia outlines this succinctly. It's a three-bit value with the following meaning: bit 0: Reserved; must be zero. bit 1: Don't Fragment (DF) bit 2: More Fragments (MF)
