I've used Nutch extensively, when I was building the open source project index for my Krugle startup. It's hard to customize, being a fairly monolithic design. There is a plug-in architecture, but the interaction between plug-ins and the system is tricky and fragile. As a result of that experience, and needing something with more flexibility, I started the Bixo project - a web mining toolkit. http://openbixo.org . Whether it's right for you depends on the weighting of factors such as: How much flexibility you need (+) How mature it should be (-) Whether you need the ability to scale (+) If you're comfortable with Java/Hadoop (+)
