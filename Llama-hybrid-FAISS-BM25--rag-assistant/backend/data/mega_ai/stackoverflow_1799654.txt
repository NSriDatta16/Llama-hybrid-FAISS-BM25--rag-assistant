For more than 10GB in data, a database is exactly what you need. It will handle indexing for rapidly locating data (your random retrieval), the functionality for adding, modifying, and deleting data, and the actual storage, as well as much more if you so choose. There are dozens of posts here related to which databases are available for use in Delphi, including built-ins and FOS ones like Firebird .
