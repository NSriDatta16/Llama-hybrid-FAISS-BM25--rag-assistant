You can set the UserAgentCallback this way, and Flying Saucer will use it to resolve the urls (tested, works with Release 8): ITextRenderer renderer = new ITextRenderer(); renderer.getSharedContext().setUserAgentCallback(new MyUAC()); MyUAC should extend the NaiveUserAgent , and override the resolveAndOpenStream method as the other page suggests.
