...have you a navigation controller? If not, you can use detailViewController.modalInPopover = YES; detailViewController.wantsFullScreenLayout = YES; detailViewController.modalPresentationStyle = UIModalPresentationFullScreen; [self presentModalViewController:detailViewController animated:YES]; and you show it in fullscreen, hiding tabbar. To dismiss it use: [self dismissModalViewControllerAnimated:YES]; alberto.
