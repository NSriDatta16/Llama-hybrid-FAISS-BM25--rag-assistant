Using JQuery or other JavaScript libraries, your application will have many more function points and have many more lines of code. Why? Because you will be doing things easily that you hadn't thought possible or feasible before you started using these libraries. This will encourage you to write more features, rather than punish you for trying, as programming it from scratch does. They're that good.
