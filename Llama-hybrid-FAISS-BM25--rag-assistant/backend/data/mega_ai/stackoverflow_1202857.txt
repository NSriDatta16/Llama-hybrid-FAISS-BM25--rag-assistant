For a table, you must insert tr's into tbody. When you write html abc IE, FF, Chrome, Safari at least (don't know about others directly) will modify this to be: abc Therefore, something like: var tmp = document.createElement('tr'); tmp.innerHTML = " def "; var new_w = tmp.firstChild; var parent = insertion_point.parentNode; parent.insertBefore(new_w, insertion_point.nextSibling); if insertion_point is a tr tag. But honestly, with Jquery there are more elegant ways of going about this as christina toma notes.
