Cause libsvm use cv only for parameter selection. From libsvm FAQ: Q: After doing cross validation, why there is no model file outputted ? Cross validation is used for selecting good parameters. After finding them, you want to re-train the whole data without the -v option. If you are going to use cv for estimating quality of classifier on your data you should implement external cross validation by splitting data, train on some part and test on other.
