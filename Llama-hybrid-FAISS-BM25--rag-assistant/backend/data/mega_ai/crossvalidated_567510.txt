A subsequence like this is sometimes referred to as a time series snippet . The linked resource describes how to identify such using the matrix profile . Another approach is to apply a generic time-series segmentation to get candidate segments, and then compare the segments to each-other using some distance measure. This could be Dynamic Time Warping to allow for some time variations in the segments.
