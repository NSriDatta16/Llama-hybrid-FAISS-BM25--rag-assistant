Solution i found BaseCanvas - canvas with fixed height and width EditCanvas - dynamic canvas which params depends on it's children position. Snapshot takes from EditCanvas. Here part of code private function SnapshotButtonHandler():void { var snapshot:ImageSnapshot = ImageSnapshot.captureImage(EditCanvas); var file:FileReference = new FileReference(); file.save(snapshot.data, "canvas.png"); } private function ResizeCanvas():void { for each (var child:* in AppCanvas.getChildren()) { if ((child.x + child.width) > AppCanvas.width) AppCanvas.width = child.x+child.width; if ((child.y + child.height) > AppCanvas.height) AppCanvas.height = child.y+child.height; } }
