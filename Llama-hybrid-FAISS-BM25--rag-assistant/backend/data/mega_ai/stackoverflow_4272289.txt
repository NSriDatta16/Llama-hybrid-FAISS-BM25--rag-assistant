The more jQuery way would probably be to use the jQuery methods but whether that's the best way for a particular situation depends on what is meant by best. Using the jQuery methods may be best for readability. String concatenation, pushing to an array and then joining or using document fragments may be a better way for performance. One point that should be adhered to depending on whichever way you choose to use, is to build the structure in memory and perform only one insert into the DOM.
