Any a priori answer would be pure speculation. To answer such a question, I recommend you prepare a test set with sentences and annotated keywords, and test both your NLP model and the simple list-based approach. The test must be representative of the kind of data that the system will face when actually used in production. Of course, this test set must not overlap with the NLP model training data.
