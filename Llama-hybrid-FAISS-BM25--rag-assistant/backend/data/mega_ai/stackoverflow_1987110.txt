You can check User Agent in the HTML header. Blackberry's browser identifies itself as something like: BlackBerry8700/4.1.0 Profile/MIDP-2.0 Configuration/CLDC-1.1 . Just check whether "BlackBerry" is a substring. There is a incomplete List of user agents for mobile phones .
