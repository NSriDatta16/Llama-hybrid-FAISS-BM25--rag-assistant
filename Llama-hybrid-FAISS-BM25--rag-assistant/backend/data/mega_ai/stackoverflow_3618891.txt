Save the batch file in OEM encoding (a decent editor should allow this) or change the code page prior to running it with chcp 1252 You can also save it as UTF-8 without signature (BOM) and use chcp 65001 but down that path lies peril and dragons await to eat you (in short: It's usually painful and has a few weird side-effects).
