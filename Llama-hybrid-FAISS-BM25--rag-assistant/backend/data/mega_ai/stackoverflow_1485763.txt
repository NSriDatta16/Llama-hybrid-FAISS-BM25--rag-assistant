My preferred solution is always to save them to a directory on your file system and save the url / location in the database. I find this the easiest to maintain and it's easy to change databases, datastorage without affecting the files. This method, is not dependent on the database. I had a situation recently where a legacy system stored images and files as binary, which caused time and effort retrieving these to upgrade database software.
