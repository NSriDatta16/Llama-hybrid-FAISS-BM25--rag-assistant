This is a perfectly legitimate use of memcache. It is not only for database load reduction, it is for caching and object storage in general. :) Also note, PHP has two interfaces for memcached. Confusingly, they are named " memcache " and " memcached ". Read these to pick between the two: https://serverfault.com/questions/63383/memcache-vs-memcached http://code.google.com/p/memcached/wiki/PHPClientComparison
