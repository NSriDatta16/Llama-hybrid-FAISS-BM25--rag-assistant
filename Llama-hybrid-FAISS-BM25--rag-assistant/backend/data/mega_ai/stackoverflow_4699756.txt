You can use the diagnostics pragma to extend the diagnostics emitted by Perl: use diagnostics; # your_code This will provide you a verbose explanation: You've said "use strict" or "use strict vars", which indicates that all variables must either be lexically scoped (using "my"), declared beforehand using "our", or explicitly qualified to say which package the global variable is in (using "::").
