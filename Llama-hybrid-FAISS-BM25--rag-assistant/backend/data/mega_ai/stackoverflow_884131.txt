You could expand your Product class to have a method that uses a Select to find the Product by it's composite id, for example: public partial class Product{ public static Product FetchByCompositeId(int salesOrganisationId, int productId){ return DB.Select().From () .Where(Product.Columns.SalesOrganisationId).IsEqualTo(salesOrganisationId) .And(Product.Columns.ProductId).IsEqualTo(productId) .ExecuteSingle (); } }
