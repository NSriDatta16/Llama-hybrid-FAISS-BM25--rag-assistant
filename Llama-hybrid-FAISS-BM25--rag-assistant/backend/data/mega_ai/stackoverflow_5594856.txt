DevPartner Studio was created by NuMega Technologies, and the product has continued to be developed and maintained since it was acquired first by Compuware (1997) and then Micro Focus (2009). When installed, it extends the Microsoft Visual Studio, adding instrumented error detection, code coverage and performance analysis, plus a static code review package. July 2015: The latest revision, 11.3, supports Windows 10 and Visual Studio 2015.
