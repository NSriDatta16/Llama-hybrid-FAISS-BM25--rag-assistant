Using a little logic. (Humorous, similar to "gigantic switch") Ruby def validate num return false unless num.length == 4 return false unless num.to_i.between?(-1, 10000) num.each_char {|ch| return false unless '0123456789'.include? ch } true end puts validate '404' #false puts validate '9321' # true puts validate '-302' #false puts validate 'AAAA' # false
