This tutorial describes how to do it: Embedding a flash player control in a VB.NET winforms application The key here is to add the "Shockwave Flash Object" COM object to your project. You'll then have a Shockwave Flash Object control in the toolbox which you can drag onto the form. Set the Movie property to the .swf object.
