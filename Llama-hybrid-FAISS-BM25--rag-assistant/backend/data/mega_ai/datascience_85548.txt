In tf.keras.preprocessing.text ( docs ) you have utilities to process discrete token sequences, normally used to represent text. In tf.keras.preprocessing.sequence ( docs ) you have utilities to process both continuous value sequences (normally used to represent time series) like TimeSeriesGenerator , and discrete token sequences (i.e. text), like the skipgrams function .
