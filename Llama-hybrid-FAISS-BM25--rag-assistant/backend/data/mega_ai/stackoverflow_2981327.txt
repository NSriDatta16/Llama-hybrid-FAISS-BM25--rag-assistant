Use this regex instead. Also added code to echo the discarded text snippets. Pattern p = Pattern.compile( "( )" + // "([^ ]*>)", // Pattern.MULTILINE); Matcher m = p.matcher(content); // print all the matches that we find int start = 0; while (m.find()) { System.out.println("\t" + content.substring(start, m.end()).replaceAll("\n", "\n\t")); System.out.println(m.group()); start = m.end(); } System.out.println("\t" + content.substring(start).replaceAll("\n", "\n\t")); The output is Yada yada yada foo ddd yada yada ... more here bar bar etc etc more here again bar bar etc etc more here again bar bar etc etc and still more baz baz and now the end Where indented lines correspond to discarded fragments
