I have done some thing like that but in my case color was also coming from data also but it will help you. You have to override the Datagrid and override drawRowBackground method public class CustomDataGrid extends AdvancedDataGrid { protected override function drawRowBackground(s:Sprite, rowIndex:int, y:Number, height:Number, color:uint, dataIndex:int):void{ var XMLdata:XML=rowNumberToData(dataIndex) as XML; if(XMLdata!=null){ if(XMLdata.attribute(Constants.col) != undefined && XMLdata.attribute(Constants.col) != ""){ color=XMLdata.attribute(Constants.col); }else{ color=0xFFFFFF; } } super.drawRowBackground(s,rowIndex,y,height,color,dataIndex); } } By this you can get any data from the row and according to it give the color.
