You can't. The fragment identifier is only used client side, so it isn't sent to the server. You have to parse it out with JavaScript, and then run your Ajax routines. If you are loading entire pages (and just leaving some navigation and branding behind) then it almost certainly isn't worth using Ajax for this in the first place. Regular links work better.
