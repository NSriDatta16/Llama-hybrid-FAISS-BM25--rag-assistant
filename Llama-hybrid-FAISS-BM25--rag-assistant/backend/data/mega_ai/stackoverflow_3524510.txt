You can use Core Data or SQLite or any other storage (XML, JSON), since there are no cookies in iPhone. Core Data http://developer.apple.com/iphone/library/documentation/DataManagement/Conceptual/iPhoneCoreData01/Introduction/Introduction.html SQLite http://dblog.com.au/iphone-development-tutorials/iphone-sdk-tutorial-reading-data-from-a-sqlite-database/ Also remember to hash the password you should use MD5 at least.
