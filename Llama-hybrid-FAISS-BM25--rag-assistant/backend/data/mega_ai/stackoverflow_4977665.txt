You will need to use javascript for this. There is no server side event occurring when the user changes the selection of a radio button. So if you are using jquery you could subscribe for the .change() event and submit the form: m.Agents, "A", new { @checked = "checked", @class = "radio" }) %> m.Agents, "AG", new { @class = "radio" }) %> and then in a separate javascript file: $(function() { $('#myform .radio').change(function() { $('#myform').submit(); }); });
