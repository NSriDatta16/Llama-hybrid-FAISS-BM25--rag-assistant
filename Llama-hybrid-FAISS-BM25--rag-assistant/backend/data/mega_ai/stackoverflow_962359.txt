haven't used numpy, but in standard python you can filter out None using list comprehensions or the filter function >>> [i for i in [1, 2, None] if i != None] [1, 2] >>> filter(lambda x: x != None, [1, 2, None]) [1, 2] and then average the result to ignore the None
