just do the alter table and catch the exception. DECLARE allready_null EXCEPTION; PRAGMA EXCEPTION_INIT(allready_null, -1451); BEGIN execute immediate 'ALTER TABLE TAB MODIFY(COL NULL)'; EXCEPTION WHEN allready_null THEN null; -- handle the error END; / if you don't want to use PL/SQL set feedback off set echo off set feedback off set pages 0 set head off spool to_null.sql select 'alter table TAB modify (COL NULL);' from user_tab_columns where table_name = 'TAB' and column_name = 'COL' and nullable = 'N'; spool off set feedback on set echo on set termout on @@to_null.sql host rm -f to_null.sql or just do the alter table and ignore the error.
