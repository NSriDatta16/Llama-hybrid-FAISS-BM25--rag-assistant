.NET / C# : public static void DisallowBrowserCache( ) { HttpContext.Current.Response.Cache.SetCacheability(HttpCacheability.NoCache); HttpContext.Current.Response.Cache.SetAllowResponseInBrowserHistory(false); } You could make ASP.NET write a js file to the outputstream ( http://server.com/jsFile.aspx , set http headers), and control the caching behavior of the response with the above method.
