The first part is the choice of format: is it an internal/external file? is it just used as temporary storage, or does it need to be edited/processed? are there any speed or storage space concerns? Once you have the format, you need to think about things like versioning: is backward-only compatibility OK, or do you need forward compatibility? do you need this at all? And then you can think about libraries: various XML libraries Boost.Serialization s11n
