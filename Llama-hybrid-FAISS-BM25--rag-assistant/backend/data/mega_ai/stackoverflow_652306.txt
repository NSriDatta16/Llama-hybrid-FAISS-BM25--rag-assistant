It stands for "methods". From the comp.lang.objective-C FAQ : The organisation of Objective-C source is typically similar to that of C or C++ source code, with declarations and object interfaces going into header files named with a .h extension, and definitions and object implementations going in files named with a .m (short for methods ) extension.
