It's not elegant, but... If you have the DomainController object, you can do: bool isWritable = true; try { using (Domain d = Domain.GetCurrentDomain()) var dc = d.FindDomainController(theDomainController.Name, LocatorOptions.WriteableRequired); } catch(ActiveDirectoryObjectNotFoundException) { isWritable = false; } This should determine whether a specific domain controller is writable.
