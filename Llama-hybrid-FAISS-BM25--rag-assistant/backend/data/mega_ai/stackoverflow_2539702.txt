If you are looking to store data on a per user basis between execution sessions, you should consider Isolated Storage. Doesn't clutter install directory Doesn't cause issues with AnitVirus software Part of the OS including .Net objects, don't need to install anything else Already works with the Windows security model Exists on a per user basis, so saved settings are separated for each user Can serialize/deserialize obects directly into it
