For a geometric distribution with probability $A$ of success, the probability that exactly $k$ failures occur before the first success is: $A(1-A)^{k-1} \hspace{10pt} \forall k \geq 1$ . This is written as a probability mass function $P(X=k)$ . The mean or expected value of distribution gives useful information about what average one would expect from a large number of repeated trials. $\mathbb{E} (X) = \sum_{k\geq1}kP(X=k) = \sum_{k\geq1}kA(1-A)^{k-1} = A\sum_{k\geq1}k(1-A)^{k-1} = \frac{A}{(1-A)^2} = \frac{1}{A}$ The variance of a distribution measures how "spread out" the data is. $\mathbb{V}(X) = \mathbb{E}(X^2) - \mathbb{E}^2(X)$ $\mathbb{E}(X^2) = \sum_{k\geq1} k^2P(X=k) = \sum_{k\geq1} k^2A(1-A)^{k-1} = \sum_{k\geq1} k(k+1)A(1-A)^{k-1} - \sum_{k\geq1} kA(1-A)^{k-1} = \frac{2A}{(1-A)^3)} - \frac{A}{(1-A)^2} = \frac{2}{A^2} - \frac{1}{A}$ $\mathbb{V}(X) = \frac{2}{A^2} - \frac{1}{A} - \frac{1}{A^2} = \frac{1-A}{A^2}$
