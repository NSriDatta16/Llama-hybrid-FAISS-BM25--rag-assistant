Try inspecting the html responses directly by using something like Fiddler or Firebug . Check to see if the responses from IIS/Apache (which should be returning exactly the same text) have: Different data Different headers Pay particular attention to the Content-Type header, which should say what character encoding (utf-8, ISO/IEC 8859-1, Latin-1, etc.) the returned text is in.
