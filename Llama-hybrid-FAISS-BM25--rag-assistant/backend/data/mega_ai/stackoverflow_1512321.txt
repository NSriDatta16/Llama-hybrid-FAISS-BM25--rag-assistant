Give a look to the GeckoFX Project, an open-source component for embedding Mozilla Gecko (Firefox) in .NET applications. Written in clean, fully commented C#, GeckoFX is the perfect replacement for the default Internet Explorer-based WebBrowser control. (source: googlecode.com )
