I'd use the ISNULL function - it will return the value given if the field is NULL: select contactid,Title,FirstName,MiddleName, case ISNULL(MiddleName, 'NULLVALUE') when 'R.' then 'Robert' when 'B.' then 'Bids' when 'J.' then 'John' when 'NULLVALUE' then 'New Name' else 'No Name' end, LastName from Person.Contact
