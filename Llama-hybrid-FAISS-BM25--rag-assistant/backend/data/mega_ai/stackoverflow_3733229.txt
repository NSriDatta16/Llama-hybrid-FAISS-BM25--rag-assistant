From http://www.serverwatch.com/tutorials/article.php/1128981/The-Perl-Basics-You-Need-To-Know.htm Making Variables Global With Strict Pragma On First you use: use strict; Then you use: use vars qw( %hash @array); This declares the named variables as package globals in the current package. They may be referred to within the same file and package with their unqualified names; and in different files/packages with their fully qualified names. That's all that I was needed!
