Use histc , which return an index for each entry, to which bin did it 'fall': [n, bin] = histc (a(:, 2), bins); Then the samples above the k'th bin is: a(bin==k, 1); Pay attention, you must specify the bins' boundaries yourself (unlike hist which uses the mid-value between boundaries).
