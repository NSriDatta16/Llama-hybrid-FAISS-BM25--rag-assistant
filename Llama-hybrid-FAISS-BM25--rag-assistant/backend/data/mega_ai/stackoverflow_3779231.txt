The push function takes an array, so you must deference it back into an array: push @{$TEST{TEST1}}, "some value"; Also, your style makes me think you are not using the strict pragma. A better way to write that code is: #!/usr/bin/perl use strict; use warnings; sub atest { my $test = shift; push @{$test->{TEST1}}, "some value"; } my %test; atest(\%test); use Data::Dumper; print Dumper \%test;
