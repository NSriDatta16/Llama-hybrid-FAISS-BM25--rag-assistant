I stumbled upon some problems with this. In the end, just rebuild and re-reference will work. I had preferred to inherit from UserControl. It made my life simpler ;) If for example you want to create a "rounded border" label, do something like this: using System; using System.ComponentModel; using System.Drawing; using System.Drawing.Drawing2D; using System.Windows.Forms; namespace YourUIControls { [DefaultProperty("TextString")] [DefaultEvent("TextClick")] public partial class RoundedLabel : UserControl { public RoundedLabel() { InitializeComponent(); } protected override void OnPaint(PaintEventArgs e) { //Draw your label hereâ€¦ } } } Compile and add a reference to the output. You'll be able to drag that to the Toolbox and later to the Designer.
