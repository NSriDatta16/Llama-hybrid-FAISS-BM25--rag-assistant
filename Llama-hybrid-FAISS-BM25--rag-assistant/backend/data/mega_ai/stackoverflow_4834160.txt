I believe GAE is doing the correct thing, actually. Multiple cookies are separated by commas, not semi-colons. The semi-colon is used to separate parameters of a single cookie. I don't have time to look up the RFC link, but you can see examples all over the internet: CODE import httplib c = httplib.HTTPConnection("www.facebook.com") myHeaders = { 'Content-Type': 'text/html', 'User-agent': "Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.6; en-US; rv:1.9.2.12) Gecko/20101026 Firefox/3.6.12", 'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8', 'Keep-Alive': '300', 'Connection': 'keep-alive', 'Accept-Language': 'en-us,en;q=0.5', 'Accept-Charset': 'ISO-8859-1,utf-8;q=0.7,*;q=0.7', } c.request("GET", "/login.php", body="", headers=myHeaders) r = c.getresponse() print r.getheaders() RESULTS [ ... other headers snipped ...
