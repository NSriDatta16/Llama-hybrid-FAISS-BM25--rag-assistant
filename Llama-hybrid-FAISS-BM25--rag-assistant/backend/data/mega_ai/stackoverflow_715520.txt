After some digging, this is what I did. NOTE: I had the third party app write the xml to a StringWriter instead of an output stream named 'writer'. 'outputStream' is the name of the stream the resulting XML will be written to. ByteArrayInputStream inputStream = new ByteArrayInputStream(writer.toString().getBytes()); Document myXML = DocumentBuilderFactory.newInstance().newDocumentBuilder().parse(inputStream); myXML.getDocumentElement().setAttribute("xml:base", namespace); Transformer transformer = TransformerFactory.newInstance().newTransformer(); StreamResult result = new StreamResult(outputStream); DOMSource source = new DOMSource(myXML); transformer.transform(source, result); I really thought this would be easier.
