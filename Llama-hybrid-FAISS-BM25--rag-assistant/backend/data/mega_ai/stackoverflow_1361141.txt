This is done by embedding an ActiveX control inside the browser, so I think it only works in IE. But why would you do this? Instead of embedding a Windows Forms control you could embed a silverlight or flash application and get a solution that works on most browsers and has better support. Embedding Windows Forms controls has never been very popular and is more of a solution looking for a problem than an actually useful technology. :-) EDIT: If you really want to do this look here for a very simple example
