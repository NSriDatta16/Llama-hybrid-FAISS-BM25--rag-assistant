Check out http://joubert.posterous.com/crisp-html-5-canvas-text-on-mobile-phones-and . Looks like if you multiple the dimensions by the devicePixelRatio then scale by that ratio as well it should work. Here's some pseudo-code that worked for me. var ctx = myCanvasElem.getContext("2d"); ctx.attr("width", width * window.devicePixelRatio); ctx.attr("height", height * window.devicePixelRatio); ctx.scale(window.devicePixelRatio, window.devicePixelRatio); ctx.drawImage(img, x, y, width, height); Let me know if that solves it for you!
