Hi I am using magento 1.4.1.1 and this worked to sort the child categories: use $cats = Mage::getModel('catalog/category')->load(319)->getChildrenCategories(); to get the children categories of category with id 319 and edit the file at code/core/Mage/catalog/Model/Resource/Eav/Mysql4/category.php at line 582 under the function getChildrenCategories() to change ->setOrder('position','ASC'); to ->setOrder('name','ASC); hope this works for you too.
