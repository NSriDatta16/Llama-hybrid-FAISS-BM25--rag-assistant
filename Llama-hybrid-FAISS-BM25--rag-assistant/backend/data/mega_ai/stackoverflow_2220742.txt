It is not possible . The TCP seqment header does not describe its length. The length of the TCP payload is derived from the length of the IP packet(s) minus the length of the IP and TCP headers. So only one TCP segment per IP packet. Conversely, however, a single TCP segment can be fragmented over several IP packets by IP fragmentation.
