Microsoft's Azure Machine Learning Studio has an "always free" tier, subject to certain limitations , including 100 modules per experiment (a "module" in Azure-speak is any discrete operation such as "load data" or "train model", so you can do quite a bit with 100 of them) 10Gb of storage 1 hour per experiment No parallel execution on multiple nodes Training will continue to run while you are not connected, to answer your second question. You can set up your experiments via the web interface or on the command line .
