You can retrieve density and densityDpi via DisplayMetrics . Depending the results you can determine by yourself if you have hdpi, mdpi or ldpi. Clarification: If the density returns a value below 1, you have a LDPI screen. Exactly 1 you have a MDPI screen and more than 1 you have a HDPI screen (read the documentation of density )
