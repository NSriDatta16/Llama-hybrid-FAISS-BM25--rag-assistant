Is it a UserControl or a custom server control inheriting from CompositeControl , WebControl , etc.? Generally with user controls, you load them via the ascx location, such as: //load the control Control rc= LoadControl("~/UserControls/MyControl.ascx"); //set up values as needed rc.Username = this.Username; //add the control where needed this.pnlPB.Controls.Add(rc);
