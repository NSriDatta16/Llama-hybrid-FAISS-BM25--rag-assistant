I added the answer from the comments here. Data needed centering as suggested by @amoeba. According to the documentation also ( https://github.com/scikit-learn/scikit-learn/blob/a5ab948/sklearn/decomposition/base.py#L101 ) import numpy as np from sklearn import decomposition np.random.seed(0) my_matrix = np.random.randn(100, 5) mdl = decomposition.PCA(n_components=5) mdl_FitTrans = mdl.fit_transform(my_matrix) pca_components = mdl.components_ my_matrix_centered = my_matrix - np.mean(my_matrix,axis=0) mdl_FitTrans_manual = np.dot(my_matrix_centered, pca_components.T) (mdl_FitTrans.all() == mdl_FitTrans_manual.all()) True EDIT: because I am not consistent with my variable naming
