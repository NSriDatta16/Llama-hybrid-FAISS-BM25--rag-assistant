Some code on that (based on Dictionary): foreach (KeyValuePair pair in hashtable2) { if (hashtable1.ContainsKey(pair.Key)) { hashtable1[pair.Key] = pair.Value; } else { hashtable1.Add(pair.Key, pair.Value); } } I'm sure there's a more elegant solution using LINQ (though, I code in 2.0 ;) ).
