Categorising the PCA Method

PCA (Principle Component Analysis) is often used in statistical settings, yet it has also been described as an unsupervised machine learning method. It's surprisingly difficult to find clarity online and in academic literature as to whether it is either a Statistical Method, or a Machine Learning Method, or both. If you were listing methods used in a project by type, where would PCA go?
