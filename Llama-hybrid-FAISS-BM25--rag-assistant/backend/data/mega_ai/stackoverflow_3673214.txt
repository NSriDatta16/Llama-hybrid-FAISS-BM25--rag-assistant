I may not understand the question, then, but this code goes through the document, looking for lines that are only headers and gets the page its on. Public Sub SeeHeadingPageNumber() On Error GoTo MyErrorHandler Dim sourceDocument As Document Set sourceDocument = ActiveDocument Dim myPara As Paragraph For Each myPara In sourceDocument.Paragraphs myPara.Range.Select 'For debug only If InStr(LCase$(myPara.Range.Style.NameLocal), LCase$("heading")) > 0 Then Debug.Print myPara.Range.Information(wdActiveEndAdjustedPageNumber) End If DoEvents Next Exit Sub MyErrorHandler: MsgBox "SeeHeadingPageNumber" & vbCrLf & vbCrLf & "Err = " & Err.Number & vbCrLf & "Description: " & Err.Description End Sub
