Zend Framework has pretty good IMAP support. Check out Zend_Mail_Storage_Imap: http://framework.zend.com/manual/en/zend.mail.read.html There are lots of other useful things in Zend_Mail. Remember, that you can use just the one class you need. You don't need to use the whole framework.
