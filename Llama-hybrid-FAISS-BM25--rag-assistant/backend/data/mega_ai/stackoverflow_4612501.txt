To get the value of the URL fragment (the part after the # ) call Window.Location.getHash() . This will return all of "ForumMessage=918". getParameter() returns query parameters, not the URL fragment. See here for more information about the parts of a URL . The ?gwt.codesvr= part is needed to run in Development Mode .
