Manny tools are available that will serve the purpose like stylesnooper and Show Me The Template but if you have Microsoft Expression Blend you can extract the default control template by Draggint the control onto the design surface Right click the control and choose Edit Template -> Edit Copy When you do this, Blend will extract the base template from the control and explicitly declare it within document/application as a resource which you can then edit to your liking. Check this for more http://www.shafqatahmed.com/2009/01/wpf-kid-stuff-extracting-a-control-template.html
