You are relying on a deprecated feature of PHP, called register_globals . This feature caused many security issues, and shouldn't be used any longer. The PHP manual discusses this in-depth . For an excellent description of the problem, see this Google Groups post. Edit : If you aren't relying on register_globals , then you should pay attention to Gumbo and stereointeractive's answers.
