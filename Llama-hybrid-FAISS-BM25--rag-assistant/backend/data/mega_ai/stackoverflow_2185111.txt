I found a simpler answer here . That example extends a DataGrid control, but you can do the same with a List control. In my case, I use an image source instead of Class: public class CustomDragList extends List { [Bindable] public var dragProxyImageSource:Object; override protected function get dragImage():IUIComponent { var image:Image = new Image(); image.width = 50; image.height = 50; image.source = dragProxyImageSource; image.owner = this; return image; } } Then use that custom list like this: Where 'someImageSource' can be anything you'd normally use for an image source (embedded, linked, etc.)
