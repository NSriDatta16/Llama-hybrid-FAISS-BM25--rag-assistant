No no no...they are NOT in any way like the blink tag. There are many use cases. Notice iGoogle home page. The user interface is quite easy to use and modify because of draggable divs. Selecting multiple items from a list is another use case. The blogspot blog page editor uses draggable divs to help you rearrange your page. Like with everything else the features on your user interface must make sense to the user. If something can be moved by the user on the page then a draggable div is a good idea. Without it, a trip back to the server might be necessary.
