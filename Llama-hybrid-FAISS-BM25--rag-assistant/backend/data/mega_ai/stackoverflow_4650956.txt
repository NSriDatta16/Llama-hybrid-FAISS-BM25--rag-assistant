You can reach 12 bytes by using non-standard constructs such as Visual Studio #pragma pack : #pragma pack(1) struct TableEntry3 { unsigned __int64 lock; unsigned int move:3; int score:12; enum { VALID, UBOUND, LBOUND } flag:2; int height:7; }; Here, sizeof(TableEntry3) yields 12 for me.
