The non-standard embed tag is not supported in XHTML in favour of the standards based object element. See http://www.bernzilla.com/item.php?id=681 for more information on this, but in a nutshell: Keep in mind though that using object doesn't always work correctly in older browsers, so do a bit of testing using both formats first. A List Apart has a brilliant article on this called " Flash Satay: Embedding Flash While Supporting Standards " which shows you the cross browser way of implementing it as shown in the example above.
