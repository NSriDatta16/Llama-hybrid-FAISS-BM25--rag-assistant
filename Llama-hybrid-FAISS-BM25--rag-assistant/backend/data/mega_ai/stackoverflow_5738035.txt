You can: Use TMemo.LoadFromFile and load the entire file directly Memo1.LoadFromFile(YourFileNameHere); Use TStringList.LoadFromFile and add each line (including blank ones) from there var SL: TStringList; begin SL := TStringList.Create; try SL.LoadFromFile(YourFilenameHere); Memo1.Lines.Assign(SL); finally SL.Free; end; end; // or var SL: TStringList; s: String; // or i: Integer; begin SL := TStringList.Create; try SL.LoadFromFile(YourFilenameHere); for s in SL do Memo1.Lines.Add(s); // or // for i := 0 to SL.Count - 1 do // Memo1.Lines.Add(SL[i]); finally SL.Free; end; end; If you're looking to add a new paragraph, you can just use this: Memo1.Lines.Add(''); // Add a blank line to separate paragraphs and then use the code above to add new single lines to the memo. Finally, if you have text in the memo and you want to add an entire block of text as a new paragraph: var s: string; Stream: TFileStream; begin Stream := TFileStream.Create(YourFilenameHere, fmOpenRead); try SetLength(s, Stream.Size); Stream.Read(s[1], Stream.Size); // sLineBreak is in System.pas Memo1.Lines.Text := Memo1.Lines.Text + sLineBreak + s; finally Stream.Free; end; end; If none of these are what you want, you'll need to be more specific with your question.
