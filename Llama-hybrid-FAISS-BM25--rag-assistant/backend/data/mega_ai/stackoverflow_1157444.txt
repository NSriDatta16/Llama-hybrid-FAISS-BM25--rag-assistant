I think you can do something like this as well: Obviously do this in a base class that each page will use. So create like a PageBase which inherits System.Web.UI.Page and have all your code behinds inherit from PageBase. or do it in the Master Page. protected void Page_Init(object sender, EventArgs e) { Response.Cache.SetExpires(DateTime.UtcNow.AddMinutes(-1)); Response.Cache.SetCacheability(HttpCacheability.NoCache); Response.Cache.SetNoStore(); }
