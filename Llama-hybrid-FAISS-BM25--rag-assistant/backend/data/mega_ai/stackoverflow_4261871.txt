IP address and port may be passed as the the first argument (as Robert answered) --enable_sendmail works as-is The datastore parameters must be separated by a space instead of an equals Working version: /usr/bin/python2.5 ./manage.py runserver \ 192.168.1.8:9009 \ --enable_sendmail \ --blobstore_path /foo/django-nonrel/blobs \ --datastore_path /foo/data \ --history_path /foo/history
