For Apps that looks good in any android phone, you must create your apps that support multiple screen resolution.for that you can use: res/layout/main.xml // layout for normal screen size res/layout-small/main.xml // layout for small screen size res/layout-large/main.xml // layout for large screen size and your images for different screen should be :: res/drawable-ldpi/ .png // image for low density res/drawable-mdpi/dpi/ .png // for medium density res/drawable-hdpi/*.png // image for high density for more details visit:-->http://developer.android.com/guide/practices/screens_support.html
