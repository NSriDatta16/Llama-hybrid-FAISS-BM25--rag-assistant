I would use DOM parsing for that: // SimpleHtmlDom example // Create DOM from URL or file $html = file_get_html('http://localhost/blah.php'); // Find all paragraphs foreach($html->find('p') as $element) echo $element->innerText() . ' '; It would allow you to more reliably replace some of the markup: $html->find('p', 0)->innertext() = 'foo';
