Take some of the tips for emulating a browser with python+mechanize from here . Adding addheaders and set_handle_robots appears to be the minimum required. With the code below, I get output: from mechanize import Browser, _http from BeautifulSoup import BeautifulSoup br = Browser() br.set_handle_robots(False) br.addheaders = [('User-agent', 'Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.9.0.1) Gecko/2008071615 Fedora/3.0.1-1.fc9 Firefox/3.0.1')] url = "http://www.delicious.com/varunsrin" page = br.open(url) html = page.read() soup = BeautifulSoup(html) print soup.prettify()
