You could try using category to catch all the relevant method calls (template below). However beware dragons and make sure that you consider using the system proxy route before doing this. UIWebView+PrivateProxy.h @interface UIWebView (PrivateProxy) - (void)loadRequest:(NSURLRequest *)request; @end UIWebView+PrivateProxy.m @implementation UIWebView (PrivateProxy) - (void)loadRequest:(NSURLRequest *)request { if(request.something ....) { // handle yourself ... } else { [super loadRequest:request]; // use stanadrd implementation } } @end Off course you'll probably have to override more methods e.g. reload, stopLoading, etc.
