Have you considered using SQLite? It'll result in a small .s3db file. SQLite is used by all kinds of desktop applications for local storage. There's a SQLite .NET library that'll allow you to use ADO.NET to CRUD your data. Check out Mike Duncan's article on how to get started with SQLite in .NET .
