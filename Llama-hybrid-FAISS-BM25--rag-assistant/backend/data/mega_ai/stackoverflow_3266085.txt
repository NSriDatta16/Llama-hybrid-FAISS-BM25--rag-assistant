Assuming your database supports this kind of date math, you can say: CASE WHEN EntryDate - date(EntryDate) >= INTERVAL '9 hours' AND EntryDate - date(EntryDate) (That's the PostgreSQL interval syntax, btw... might be nonstandard.) But there are probably more elegant ways of doing it.
