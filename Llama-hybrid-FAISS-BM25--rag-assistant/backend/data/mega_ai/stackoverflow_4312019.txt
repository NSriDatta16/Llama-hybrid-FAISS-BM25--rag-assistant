MySQL: Average interval between records

Assume this table: id date ---------------- 1 2010-12-12 2 2010-12-13 3 2010-12-18 4 2010-12-22 5 2010-12-23 How do I find the average intervals between these dates, using MySQL queries only? For instance, the calculation on this table will be ( ( 2010-12-13 - 2010-12-12 ) + ( 2010-12-18 - 2010-12-13 ) + ( 2010-12-22 - 2010-12-18 ) + ( 2010-12-23 - 2010-12-22 ) ) / 4 ---------------------------------- = ( 1 DAY + 5 DAY + 4 DAY + 1 DAY ) / 4 = 2.75 DAY
