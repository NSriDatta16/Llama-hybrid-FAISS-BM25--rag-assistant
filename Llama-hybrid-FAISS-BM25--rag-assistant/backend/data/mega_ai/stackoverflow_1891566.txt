I figured out the answer myself. Here are the sample code: JpegBitmapDecoder decoder = new JpegBitmapDecoder(pixelStream, BitmapCreateOptions.None, BitmapCacheOption.None); BitmapFrame frame = decoder.Frames[0]; frame.CopyPixels(pixelBuffer, stride, 0); pixelBuffer is a preallocated byte array. And then I can use it to construct a Bitmap use in OnPaint().
