If you mean the alphabetically, simply use FogleBird's algorithm but reverse the parameters and the result! >>> print average('cat'[::-1], 'doggie'[::-1])[::-1] cumdec or rewriting average like so >>> def average(a, b): ... a = debase(a[::-1]) ... b = debase(b[::-1]) ... return enbase((a + b) / 2)[::-1] ... >>> print average('cat', 'doggie') cumdec >>> print average('google', 'microsoft') jlvymlupj >>> print average('microsoft', 'google') jlvymlupj
