We do this exact thing using a plethora of static converters in a utility class. It isn't elegant but it sure is easy and effective. class Util { public static Long StringToLong(String s) { ... } public static Integer StringToInt(String s) { ... } public static Date StringToDate(String s) { ... } public static Date StringToDateYYMMDD(String s) { ... } public static BigDecimal StringToBigDecimal(String s) { ... } // etc ad naseum }
