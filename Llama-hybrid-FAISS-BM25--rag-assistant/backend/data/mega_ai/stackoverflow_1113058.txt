I think this May Help you! By giving the same "Class" to all dragable elements you can achieve you goal. Js: $(function() { var $gallery = $('#something'), $trash = $('#divAccept'); $('.drop',$gallery).draggable({ revert: 'invalid', containment: $('#divAccept').length ? '#divAccept' : 'document', helper: 'clone', cursor: 'move', }); $trash.droppable({ accept: '#something > .drop', drop: function(ev, ui) { deleteImage(ui.draggable); } }); // image deletion function function deleteImage($item) { $item.fadeOut(function() { $item.appendTo($trash).fadeIn(); }); } }); Html: random text JQuery UI is can give you better guidance.
