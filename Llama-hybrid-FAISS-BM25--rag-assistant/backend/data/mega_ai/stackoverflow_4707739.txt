Instead of having arr and arrsize variables in your struct, you can define your struct like this: struct HeaderTest { int someparam; int more; std::vector data; } No arr, no arrsize. Just use std::vector , and std::vector::size() . That is elegant! And if you want to read binary data from a file, then you can write like this: struct HeaderTest { int someparam; int more; int size; char *data; } Otherwise, go with the first struct!
