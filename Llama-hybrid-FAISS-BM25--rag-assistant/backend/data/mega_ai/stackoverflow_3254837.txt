Drag-and-drop. Copy-and-paste. Accessibility input methods. There are many reasons why this approach is not appropriate. Instead restrict the contents through the Document . Set a DocumentFilter through AbstractDocument.setDocumentFilter so that you do not need to subclass or implement the document. A pop up is not great for user experience. Be more subtle. Not allowing any more character will do (please don't beep!). Possibly add a countdown as twitter and stackoverflow do.
