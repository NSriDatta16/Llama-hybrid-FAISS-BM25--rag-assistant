Line based retrieval may be what you want, but token based can also be useful. You can see in documentation of Scanner public boolean hasNext() Returns true if this Scanner has another token in its input. This method may block while waiting for input to scan. The Scanner does not advance past any input. Specified by: hasNext in interface Iterator Returns: true if and only if this Scanner has another token Throws: IllegalStateException - if this Scanner is closed See Also: Iterator
