I'm unable to comment, therefore I'll just post a new answer. Create an interface for value: public interface Value { public T getValue(); } And one for averagable: public interface Averagable { public T average(T value); } Then a number value would be something like: public class NumberValue implements Averagable , Value { public Number average(Number value) { // do your stuff } public Number getValue() { // do your stuff } } There is no need to let Averagable extend from Value.
