You want full text search of a model field. This is best accomplished not by regex scans, but by a specialized index for full text retrieval. Rather than roll your own, I'd recommend using one of the following: acts_as_indexed Sphinx Ferret Xapian Lucene/Solr Here's some links with some more detail on the options: http://locomotivation.squeejee.com/post/109284085/mulling-over-our-ruby-on-rails-full-text-search-options Full Text Searching with Rails
