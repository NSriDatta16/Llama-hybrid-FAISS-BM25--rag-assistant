Use SQL Server Database Mail. http://technet.microsoft.com/en-us/library/ms175887(SQL.90).aspx You can then create the process logic that calls the stored procedure sp_send_dbmail in order to send mail. You can then schedule your process/stored procedure using a SQL Server Agent job.
