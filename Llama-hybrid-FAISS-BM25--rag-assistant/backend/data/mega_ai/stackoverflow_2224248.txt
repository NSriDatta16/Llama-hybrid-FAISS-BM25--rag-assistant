Another option is TDM MinGW: http://www.tdragon.net/recentgcc/ It's basically an unofficial fork of MinGW with the latest GCC compiler available for both 32- and 64bit windows machines. Even some MinGW developers themselves use it. Equation Gcc is even better: http://www.equation.com/servlet/equation.cmd?fa=fortran
