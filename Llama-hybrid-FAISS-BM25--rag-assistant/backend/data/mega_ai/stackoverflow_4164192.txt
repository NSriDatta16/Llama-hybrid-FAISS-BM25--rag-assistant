To check if a column exists you should check if the column name exists in the DataTable.Columns colleciton like this: // Given a datatable named table: bool columnExists = table.Columns.Contains("columnname"); Alternatively, if you are on .Net 4, you could use the support for dynamic types to return the value of a column if a column exists and a default value in any other case. I blogged about something like that a while ago: http://blog.rag.no/post/Update-The-DataTable-meets-dynamic-improved.aspx and http://blog.rag.no/post/Creating-a-monster-The-DataTable-meets-dynamic.aspx
