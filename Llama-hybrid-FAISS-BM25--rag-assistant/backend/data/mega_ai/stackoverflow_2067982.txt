The average color is easy, just resize the bitmapData by draw() ing it in to a 1x1 bitmap, flash will average the color in the copy process. var m: Matrix = new Matrix(); m.scale( 1 / bmd.width, 1 / bmd.height); var averageColorBmd:BitmapData = new BitmapData(1, 1); averageColorBmd.draw(bmd, m); var averageColor:uint = averageColorBmd.getPixel(0,0); Something like that!
