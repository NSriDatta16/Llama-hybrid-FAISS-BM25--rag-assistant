Get major version of Chrome as an integer: function getChromeVersion () { var raw = navigator.userAgent.match(/Chrom(e|ium)\/([0-9]+)\./); return raw ? parseInt(raw[2], 10) : false; } I've updated the original answer, so that it does not throw an exception in other browsers, and does not use deprecated features. You can also set minimum_chrome_version in the manifest to not let users with older versions install it.
