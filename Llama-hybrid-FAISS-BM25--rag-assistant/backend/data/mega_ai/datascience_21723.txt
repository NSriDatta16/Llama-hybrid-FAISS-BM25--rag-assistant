You can never get 100% coverage for real-world grammar extraction. Grammar is complex and undefined for real-world data. In addition, it is an "open world" data problem because novel grammar phrases can always be created. You might get 100% coverage for a small data set through overfitting. Statistical learning theory provides a framework for predicting the limits of machine learning.
