In addition to Rob Sobers answer also look for triple . 's/ These can be used as wildcards depending on the mode you are running in, much the same as [ and ]. EDIT for the comment: I haven't got an instance in front of me to test this in, but I believe you can change it by explicitly setting the ECLTYPE to 'P' in the 'UOLOGIN' paragraph in the VOC file. UOLOGIN is like LOGIN. It is called when a session is first created. The difference being that UOLOGIN is called for UniObjects sessions
