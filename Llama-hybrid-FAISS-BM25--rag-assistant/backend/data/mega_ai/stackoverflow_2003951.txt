I recently had the same issue. I solved it with the following: $('#existing').droppable({ drop: function(e, ui) { // ui.draggable is the node that gets created in #existing // ui.draggable[0].originDragger is the node that was // dragged from #toolbox } }).sortable({ revert: true });
