You should use Q objects. The rough example is: from django.db.models import Q from itertools import combinations # this -- the unicorn to be matched with attr = ['horn_length', 'skin_color', 'average_speed', 'magical', 'affinity'] q = None for c in combinations(attrs, 3): q_ = Q(**{c[0]: getattr(this, c[0])}) & Q(**{c[1]: getattr(this, c[1])}) & Q(**{c[2]: getattr(this, c[2])}) if q is None: q = q_ else: q = q | q_ Unicorn.objects.get(q) not tested, though
