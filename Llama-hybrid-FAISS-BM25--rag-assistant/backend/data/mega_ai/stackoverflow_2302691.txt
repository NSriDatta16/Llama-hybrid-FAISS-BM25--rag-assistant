The Kindle Development Kit (KDK) is Java-based. From the FAQ : What APIs are available to me in the KDK? The KDK is comprised of two sets of APIs: Java version 1.4 Personal Basis Profile (PBP) APIs for mobile devices. PBP JavaDocs can be found at http://java.sun.com/javame/reference/apis/jsr217/ . Kindle custom APIs which complement the PBP APIs and provide UI components, JSON and XML parsers, HTTP and HTTPS networking, secure storage, and other features. Other APIs like audio and dictionary access will be available in a future release of the KDK. KDK JavaDocs can be found at http://kdk-javadocs.s3.amazonaws.com/index.html .
