Unfortunately, this is not supported by apache. See the paragraph titled "URL Rewriting" at http://httpd.apache.org/docs/2.0/env.html#using . An alternative solution could be setting an auto-prepend file in which you then set the save_path. In .htaccess: php_value auto_prepend_file /path/to/prepend.php In prepend.php: See: http://php.net/manual/en/ini.core.php#ini.auto-prepend-file http://www.php.net/manual/en/function.session-save-path.php
