You may want to try setting the User-Agent and Referer HTTP headers to something like what a normal web browser would send. You can pick a User-Agent string from this list: Seehowitruns: User-agent strings . In addition, if the page you are requesting is an internal page, it might also depend on cookies which were generated in previous page.
