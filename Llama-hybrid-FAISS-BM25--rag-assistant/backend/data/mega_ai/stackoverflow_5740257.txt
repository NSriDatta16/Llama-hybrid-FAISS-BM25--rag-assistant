Your y is not a constant in the understanding of C but a const qualified variable . Compile time constant expressions as you want to use them may contain: Literals enum constants sizeof expressions address expressions concerning functions address expressions concerning objects with static storage duration and may combine these mostly freely with the usual arithmetic, as long as the resulting type is compatible with the type of the left hand side.
