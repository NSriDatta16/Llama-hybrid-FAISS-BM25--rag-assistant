Which version of NetAdvantage are you using? There is no RegexPattern property of the UltraGridCell class in NetAdvantage 2008. However, if you are refering to the RegexPattern property of the UltraGridColumn class it will do what you want. The value of the cell must give exactly one match or else the validation will fail.
