In order to add the same file in two or more projects you can Right click on the project and choose Add >> Existing Item Choose the file in the filepicker Click the little triangle on the right side of the Add button in the file picker Choose Add As Link This will add the file to the project without making a copy of it. I don't know all the details of your situation, but you're solution sounds awkward. Two versions of the same dll is likely to lead to configuration problems on client computers eventurally. I encourage you to consider Kevin's idea of a single dll with all capabilities, and calling the appropriate code as needed.
