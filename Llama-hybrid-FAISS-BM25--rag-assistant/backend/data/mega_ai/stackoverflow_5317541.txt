CFLib is your friend. Mean - Returns the mean (average) value for a set of numeric values. These values can be passed as a list or one dimensional array. Mode - Returns a structure that contains 2 keys: Mode and Frequency. Mode is the value with the greatest frequency in the set. StdDevPop - Returns the standard deviation calculated using the divisor n method. This method is used when you have all the values for an entire population. Median - Returns the median (middle) value for a set of numeric values. The values can come from a list or a one dimensional array.
