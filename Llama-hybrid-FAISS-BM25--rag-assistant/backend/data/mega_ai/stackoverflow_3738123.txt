To have the com.test.Outerclass.InnerEnum identifier to work, the InnerEnum should be a nested enum, not an inner enum. I.e. add the modifier static to it. package com.test; public final class Outerclass { public static enum InnerEnum { // ommited for clarity } } Nested classes/enums can be constructed/used outside the outer class. Inner classes/enums can't, at least not directly without adding an extra layer of reflection logic. The average JSP EL implementation don't have that.
