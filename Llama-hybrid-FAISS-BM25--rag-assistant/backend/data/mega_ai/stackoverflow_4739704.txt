Using mayavi , you can create such a plot with import enthought.mayavi.mlab as mylab import numpy as np x, y, z, value = np.random.random((4, 40)) mylab.points3d(x, y, z, value) mylab.show() The GUI allows rotation via clicking-and-dragging, and zooming in/out via right-clicking-and-dragging.
