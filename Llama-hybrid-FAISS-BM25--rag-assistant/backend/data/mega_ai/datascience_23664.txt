I'd use sklearn's Hierarchical clustering from sklearn.feature_extraction.text import CountVectorizer, TfidfTransformer from scipy.cluster import hierarchy #Vectorizing X = CountVectorizer().fit_transform(docs) X = TfidfTransformer().fit_transform(X) #Clustering X = X.todense() threshold = 0.1 Z = hierarchy.linkage(X,"average", metric="cosine") C = hierarchy.fcluster(Z, threshold, criterion="distance") C is your clustering of the documents docs . You can use other metrics instead of cosine , and use a different threshold than 0.1
