Not sure about a regex, although I don’t think you need user agent sniffing — the element is designed to contain references to the video in several formats in such a way that user agents will play the one they support. See Dive into HTML5’s video chapter for a great example . Dive into HTML5 also has a good list of browser support of HTML5 video .
