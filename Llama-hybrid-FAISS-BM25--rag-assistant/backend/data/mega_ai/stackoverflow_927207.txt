Try in WndProc of your own ToolStripContainer where you have testing for WM_NCHITTEST returning HTTRANSPARENT (-1) for the area where you want drag to happen. This will cause the message to go up in chain to your form where you handle it and return HTCAPTION so drag happens. Hope this helps.
