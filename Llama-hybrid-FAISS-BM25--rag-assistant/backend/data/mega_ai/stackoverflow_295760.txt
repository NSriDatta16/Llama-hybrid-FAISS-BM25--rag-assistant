The c# partial class has been already explained here so I'll just cover the python part. You can use multiple inheritance to elegantly distribute the definition of a class. class A_part1: def m1(self): print "m1" class A_part2: def m2(self): print "m2" class A(A_part1, A_part2): pass a = A() a.m1() a.m2()
