Depends on what your email server is. In the elder days, you used to do this with a procmail mail filter (or better yet use procmail as MDA - mail delivery agent - on your Unix host), and then have the procmail rule which would run each matching email through a Perl script which re-formats them (easy to do in Perl) and forwards them on (you can probably use PHP if you know that but not Perl, but Perl excels at text crunching, obviously).
