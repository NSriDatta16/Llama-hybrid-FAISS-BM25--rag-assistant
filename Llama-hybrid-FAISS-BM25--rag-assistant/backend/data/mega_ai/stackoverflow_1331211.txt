Try this - appologies to Degan for stealing your example and changing it. SELECT t.ID, t.Class_ID, t.Pupil_ID, c.Class_ID AS Expr1, c.Class_Name, c.Class_Day, c.Class_OrderView FROM Classes AS c LEFT OUTER JOIN timetable as t ON (c.Class_ID = t.Class_ID) AND (t.Pupil_ID = MyPupil)
