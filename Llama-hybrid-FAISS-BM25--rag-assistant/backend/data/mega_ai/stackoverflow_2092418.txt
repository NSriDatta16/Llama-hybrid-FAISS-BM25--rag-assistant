You need to use a LEFT JOIN (see MySQL JOIN syntax ). This will return NULL for rows that have no matching row in the joined table, so I use COALESCE to replace them by 0 . SELECT s.id AS id, g.id AS group_id, -- [...] COALESCE( x.average, 0 ) AS average COALESCE( x.counter, 0 ) AS counter FROM stacje_main s JOIN stacje_grups g ON (s.grupa=g.id) JOIN wojewodztwa w ON (s.wojewodztwo=w.id) JOIN kraje k ON (w.kraj=k.id) LEFT JOIN ( SELECT rr.station_id AVG(rr.vote) as avarge, COUNT(rr.station_id) counter FROM stacje_ratings rr GROUP BY rr.station_id ) x ON ( x.station_id = s.id ) WHERE s.data > 0;
