Don't know if this is the kind of thing you are looking for, but there is an automated unit test generator that was created by Microsoft research called PEX . Pex automatically generates test suites with high code coverage. Right from the Visual Studio code editor, Pex finds interesting input-output values of your methods, which you can save as a small test suite with high code coverage. Microsoft Pex is a Visual Studio add-in for testing .NET Framework applications.
