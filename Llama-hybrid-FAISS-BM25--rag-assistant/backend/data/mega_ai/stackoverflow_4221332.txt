Don’t use the use encoding pragma: it’s broken. Either specify the encoding here: use open ":encoding(Latin1)"; or put it in the open itself: open(FH, " or binmode it after opening: binmode(FH, ":encoding(Latin1)") || die "can't binmode to encoding Latin1"; If you’re using , then use open is probably easiest. Don’t forget to set the encoding on your output streams, too.
