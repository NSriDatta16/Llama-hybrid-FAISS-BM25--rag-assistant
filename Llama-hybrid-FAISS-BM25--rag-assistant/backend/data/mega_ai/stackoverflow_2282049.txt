Also a custom comparer here if you want -1 to appear first, but the rest to be descending , but somehow I find it more elegant :) Note it's made for ints class Comparer : IComparer { public int Compare(int x, int y) { if (x == -1 || y == -1) return x - y; return y - x; } }
