When you write array++ != '\0' you check if the memory address array is '\0'. Try this instead: int numberOfCharsInArray(char* array){ int numberOfChars = 0; while (*array != '\0'){ numberOfChars++; array++; } return numberOfChars; } Edit: Oops, codaddict was faster and his code more elegant.
