IE6 postback in asp.net

I have a page that contains a user control that is just a personalized dropdown list . I assign to each item the attribute onClick=__doPostBack('actrl',0) . when I click the page postback fine and I got the expected results. However in IE6 my page doesn't change to the new values loaded from the server. The weird thing is that when I shift + click on the link The page reload fine with all changes. This problem happens only in IE6.
