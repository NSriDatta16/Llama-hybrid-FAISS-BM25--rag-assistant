As already mentioned, jQueryUI is your friend. Given the following HTML: Drag1 Drag2 Drag3 Drop Zone You can use the following jQuery and jQueryUI to get something close to what you are doing. $(document).ready(function() { $('.draggables p').draggable(); $('#droparea0').droppable({ drop: function(event, ui) { ui.draggable.detach(); // detach the dragged element from the DOM $(this).css({'background-color': '#999999'}) // start colour for drop area .animate({'background-color': '#f3f0ca'}) // animate to final colour .empty() // clear the contents of the dropzone .append(ui.draggable); // append the dragged element ui.draggable.css({top: 0, left: 0}); // reset top/left since it was changed during dragging } }); }); Working jsFiddle here: http://jsfiddle.net/2F8YE/
