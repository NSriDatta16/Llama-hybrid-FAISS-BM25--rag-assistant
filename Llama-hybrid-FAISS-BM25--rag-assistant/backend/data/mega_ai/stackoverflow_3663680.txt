There are "similar" events (event is not the best word for this) available in the Servlet API. For application start you should use a Context listener http://download.oracle.com/javaee/5/api/javax/servlet/ServletContextListener.html and for requests: http://download.oracle.com/javaee/5/api/javax/servlet/ServletRequestListener.html
