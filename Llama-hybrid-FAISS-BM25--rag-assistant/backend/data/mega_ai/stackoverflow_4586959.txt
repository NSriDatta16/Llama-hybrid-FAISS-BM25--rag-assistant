The exceptions are going to slow you down more than most average lines of code. Instead of casting and then catching the exception, do a check instead. For example BAD myType foo = (myType)obj; foo.ExecuteOperation(); GOOD myType foo = obj as myType; if (foo != null) { foo.ExecuteOperation(); }
