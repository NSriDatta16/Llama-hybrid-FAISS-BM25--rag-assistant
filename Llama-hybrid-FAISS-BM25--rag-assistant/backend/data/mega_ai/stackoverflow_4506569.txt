They can be changed inside of main, like in your example, but you have to explicitely allocate storage for them in global scope, like here: class CLS { public: static int X; }; int CLS::X = 100; // alocating storage, usually done in CLS.cpp file. int main(int argc, char* argv[]) { CLS::X=100; return 0; }
