You're not paying attention to the XML namespace in play: You need to take that into account when querying - do something like this: ;WITH XMLNAMESPACES('http://TempNameSpace.com/AnswerData.xsd' AS ns) SELECT ref.value('(ns:*)[1]', 'nvarchar(1000)') FROM Mytable CROSS APPLY xmldata.nodes('/ns:Answer') R(ref) You need to reference everything inside with the ns: XML namespace prefix.
