This would be done when you display your data: $pCategory = ''; $itemDisplay = ''; foreach ($items as $item) { if ($pCategory != $item['category']) { $itemDisplay .= ' ' . $item['category'] . ' '; $pCategory = $item['category']; } $itemDisplay .= ' ' . $item['title'] . ' by ' . $item['author'] . ' '; } echo $itemDisplay; This is of course a rough example. You will have to tailor it to coincide with your code and database structure etc.
