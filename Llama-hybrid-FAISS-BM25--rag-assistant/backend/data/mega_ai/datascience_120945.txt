How it can answer with multilingual language? The training set is multilingual, and if a text starts in some language, it'll most likely continue with the same language. See {1} for an evaluation of ChatGPT in the multilingual setting. References: {1} Lai, Viet Dac, Nghia Trung Ngo, Amir Pouran Ben Veyseh, Hieu Man, Franck Dernoncourt, Trung Bui, and Thien Huu Nguyen. " ChatGPT Beyond English: Towards a Comprehensive Evaluation of Large Language Models in Multilingual Learning. " arXiv preprint arXiv:2304.05613 (2023).
