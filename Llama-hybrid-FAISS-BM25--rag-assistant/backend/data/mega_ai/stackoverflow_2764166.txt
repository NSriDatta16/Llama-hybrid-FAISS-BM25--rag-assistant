Adding the matching LIB file to your project settings should do the trick. Open the project settings -> linker -> input -> additional dependencies and add the Shlwapi.lib to the list. As an alternative you can also put the following directive into your cpp file: #pragma comment(lib, "Shlwapi.lib");
