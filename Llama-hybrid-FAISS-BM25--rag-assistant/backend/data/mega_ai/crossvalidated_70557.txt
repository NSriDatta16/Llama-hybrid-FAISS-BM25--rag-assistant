Normally, you could take the marginal effect at the means, however this doesn't exactly fly dichotomous explanatory variables. Rather, recognize that a logistic regression's dependent variable can be rewritten as the log of the odds ratio. Read this article for a concrete interpretation example using STATA output: http://www.ats.ucla.edu/stat/mult_pkg/faq/general/odds_ratio.htm Read the first part on odds ratios, then read the section under the header "Logistic regression with a single dichotomous predictor variables"
