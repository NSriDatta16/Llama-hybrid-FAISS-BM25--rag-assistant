This is the way: SFTP from within PHP use cURL from PHP to query the remove SFTP server, for both listing and file retrieval. Manual for php is here . You don't need any SSH scripts (which by the way is usually strictly forbidden on most shared hosting environment), if your sFTP server is publicly available. Curl will do everything for you.
