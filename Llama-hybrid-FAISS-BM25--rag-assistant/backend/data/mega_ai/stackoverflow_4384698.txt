You could apply a transformer to Hashtable: string query = "select a.CustomerName as CustomerName, b.ProductName as ProductName from Customer as a join a.Product as b" IQuery query = applicationSession.CreateQuery(query); var listResult = query.SetResultTransformer(NHibernate.Transform.Transformers.AliasToEntityMap).List (); string customerName = (string)listResult[0]["CustomerName"]; string productName = (string)listResult[0]["ProductName"]; Please note that I added aliases to your original query to make the AliasToEntityMap working.
