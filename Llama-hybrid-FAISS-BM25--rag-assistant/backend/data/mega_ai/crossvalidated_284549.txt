Combining Multivariate Normal Variables

Consider a case where I have $n$ datasets and from each data set I am estimating a multivariate normal distribution denoted as $$\textbf{y}_i=[y_{i1},y_{i2},y_{i3}]^t \sim N \Big( [a_{i1},a_{i2},a_{i3}]^t, \Sigma_i \Big) $$ Note that $\textbf{y}_i$ independent of $\textbf{y}_j$ Is it correct to average the predictions as follows $$\textbf{y}=\frac{1}{n}\sum\limits_{i=1}^n \textbf{y}_i \sim N \Big( \Big[\frac{1}{n}\sum\limits_{i=1}^na_{i1},\frac{1}{n}\sum\limits_{i=1}^na_{i2},\frac{1}{n}\sum\limits_{i=1}^na_{i3}\Big]^t, \frac{1}{n^2} \sum\limits_{i=1}^n \Sigma_i \Big)$$ where the summation for the covariance is element wise Also is there a better way to average the predictions ?
