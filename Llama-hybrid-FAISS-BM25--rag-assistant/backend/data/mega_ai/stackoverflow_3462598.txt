"What kind of configuration changes are we talking about here?" Adding new hard drives, reconfiguring the SAN, replacing DAS with NAS or SAN. The point of ASM is to shield the database from the physical directory structure. Tablespaces are logical structures which provide one layer of indirection, but they are still tied to actual OS files. ASM uses another logical structure, the disk group, which operates at a lower level. It manages the allocation of actual OS storage resource to the disk groups.
