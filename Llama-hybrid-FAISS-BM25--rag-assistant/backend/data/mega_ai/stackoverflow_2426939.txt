Yes, you can use the vsinstr.exe and vsperfmon.exe command lines tools to generate coverage data. See Code Coverage with Visual Studio . Once you have the code coverage files, you need to get TeamCity to report the data. This article should get you started, in particular, look under the "Configure TeamCity to display statistics" header.
