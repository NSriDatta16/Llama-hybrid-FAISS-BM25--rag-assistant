Create a MavenFilteredUserAgent module that sets user.agent from various profiles in the pom.xml. MavenFilteredUserAgent.gwt.xml ... ... pom.xml ... ie6,ie8,gecko,gecko1_8,safari,opera gwt-firefox gecko1_8 .... src/main/java-filtered true ${project.build.directory}/filtered-sources/java ${project.basedir}/src/main/resources ... org.codehaus.mojo build-helper-maven-plugin 1.5 add-source generate-sources add-source ${project.build.directory}/filtered-sources/java ... ... Have all of your modules inherit the MavenFilteredUserAgent module. Then you can build for just firefox like so. mvn install -Pgwt-firefox http://9mmedia.com/blog/?p=854 has more details.
