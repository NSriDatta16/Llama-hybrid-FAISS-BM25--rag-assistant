jQuery's load method has a special syntax that allows you to select what content you want to load from the URL you provide. You simply specify a CSS selector after your URL. (Read the documentation , under "Loading Page Fragments") For example: $("#thecontent").load("about-us.html #thecontent"); Hosted demo: http://jsbin.com/uvera3 (Editable via http://jsbin.com/uvera3/edit )
