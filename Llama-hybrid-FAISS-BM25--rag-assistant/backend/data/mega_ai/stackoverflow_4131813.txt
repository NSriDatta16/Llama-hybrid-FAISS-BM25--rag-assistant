You could instantiate one delegate object per every NSURLRequest that you fire, and that delegate object could have a reference to the core data item being processed. After that you could just delete the corresponding core data object from the store in the callback, and everything should be dandy. (In response to your last paragraph I believe you are correct in that assumption)
