You cannot iterate backwards with the jQuery each function, but you can still leverage jQuery syntax. Try the following: //get an array of the matching DOM elements var liItems = $("ul#myUL li").get(); //iterate through this array in reverse order for(var i = liItems.length - 1; i >= 0; --i) { //do Something }
