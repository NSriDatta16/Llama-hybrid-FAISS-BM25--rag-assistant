By default VS2010 comes with just Help Library Agent that is capable of showing the help documentation in the default browser. Help Viewer power tool has to be installed separately to have a stand alone documentation program. http://www.hanselman.com/blog/VisualStudio2010HelpViewerPowerToolBETAHelpIndexAndStandaloneHelp.aspx
