In that class the default param is set. protected HttpParams createParams() { HttpClientParams params = new HttpClientParams(null); params.setParameter(HttpMethodParams.USER_AGENT, "Jakarta Commons-HttpClient/3.1"); . . . . . . } If a particular parameter value has not been set, this value will be drawn here.
