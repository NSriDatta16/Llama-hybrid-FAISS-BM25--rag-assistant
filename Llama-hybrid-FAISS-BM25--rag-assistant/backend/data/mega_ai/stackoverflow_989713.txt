Denny Cherry, a SQL Server MVP, is writing a replacement for SQL Server Agent. Denny's blog about it His Standalone SQL Agent project on Codeplex Using this, you would be able to automatically initiate code on a scheduled basis. But it's either this, or writing your own .NET app to kick off jobs. SQL Server Express Edition doesn't include any kind of automated job scheduling.
