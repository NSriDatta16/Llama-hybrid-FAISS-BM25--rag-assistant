Try the "sortable" plugin for jQuery UI: http://jqueryui.com/demos/sortable/ . Write some custom stylesheets to make some rectangular s, and make your events s within them. Then call jQuery('.sortable').sortable({connectWith: '.sortable'}) and the li elements will be draggable between ul's. Add a little ajax to update on the sortchange() event ( http://jqueryui.com/demos/sortable/#event-change ) and you're all set.
