What is the OS you are using for your client? If your client machine is *nix variant, you could write a simple bash script, which when paired with SSH Keys, will allow you your one command downloader. For a windows variant, you could do similar with a batch script, schedule tasks and pscp w/ pagent saving a private key in memory. Of course none of this deals with starting up or tearing down the vpn.
