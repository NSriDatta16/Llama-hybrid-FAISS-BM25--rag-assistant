YUI has lots of widgets for this sorta thing with lots of examples. Drag & Drop: Examples Especially this example Drag & Drop: Using Interaction Groups All you would have to do register a listener on the drop event to send an ajax request to the server and save the xy co-ordinates. ALSO, if you want to do resizing as well Resize Utility: Examples They have a few really neat examples, including this image cropper ImageCropper Control: Real Time Crop Feedback
