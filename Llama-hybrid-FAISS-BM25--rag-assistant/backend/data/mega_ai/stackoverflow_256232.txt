Robert Gamble and Steve separately have most of the pieces. Assembled into a whole: void print_substring(const char *str, int skip, int tail) { int len = strlen(str); assert(skip >= 0); assert(tail >= 0 && tail skip + tail); printf("%.*s", len - skip - tail, str + skip); } Invocation for the example: print_substring("0123456789", 1, 3);
