You could use a mixed-effects logistic regression, with R synatx such a model could be lme4::glmer( correct ~ x + (1 | Subject), family=binomial(link="log"), data=your_data_frame, ... ) with a random intercept for each Subject , or if you want also random slopes lme4::glmer( correct ~ x + (1 + x | Subject), family=binomial(link="log"), data=your_data_frame, ... )
