The Comparators answer is correct but incomplete. StringAndDateComparator implements Comparator { public int compare(MyObject first, MyObject second) { int result = first.getString().compareTo(second.getString()); if (result != 0) { return result; } else { return first.getDate().compareTo(second.getDate()); } } GlazedLists has a nice utility method to chain together different comparators to save you from writing this boilerplate. See the chainComparators method for more information.
