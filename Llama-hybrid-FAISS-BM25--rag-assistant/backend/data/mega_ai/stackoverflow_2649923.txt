You can use sed to achieve what you want: sed -s -n -e '/^Agent Name: agent10479475/,/^==/ p' run*.txt In this command, sed starts printing lines when it finds a line that matches the first pattern, and stops when it finds a line that matches the second pattern, treating files separately.
