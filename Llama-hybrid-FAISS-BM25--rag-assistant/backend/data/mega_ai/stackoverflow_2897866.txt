Instead of removing the droppable class, you can enable/disable droppables by setting the 'disabled' option. success : function() { $(ui.draggable).parent().droppable( "option", "disabled", false ); $(ui.draggable).appendTo(droppable); $(parent).droppable( "option", "disabled", true ); } http://docs.jquery.com/UI/Droppable#option-disabled
