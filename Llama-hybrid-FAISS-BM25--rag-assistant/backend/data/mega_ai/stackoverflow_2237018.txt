I think you'll need to replace the text in the paragraph, not use wrap. $("p").live("mouseup",function() { selection = getSelectedText(); if(selection.length >= 3) { var spn = " " + selection + " "); var html = $(this).html().replace(selection,spn); $(this).html(html); } }); Note that this is only going to work reliably if the paragraph contains only text and no markup.
