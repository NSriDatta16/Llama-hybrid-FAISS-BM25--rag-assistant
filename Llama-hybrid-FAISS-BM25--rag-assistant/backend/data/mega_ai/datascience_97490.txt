The correct way of scaling both the features and the target in Python with Scikit-Learn for a regression problem would be wit pipelines as follow: from sklearn.linear_model import LinearRegression from sklearn.compose import TransformedTargetRegressor from sklearn.pipeline import Pipeline from sklearn.preprocessing import StandardScaler tt = TransformedTargetRegressor(regressor=LinearRegression(), transformer = StandardScaler) model = Pipeline([("scaler",StandardScaler()),("regressor",tt)])
