Outlook will update based on a server-based .ics file. Store the file on a web server somewhere (I have used a Wiki page successfully) and then use a webcal:// address instead of the http:// address. Outlook should pick this up automatically. This will generally refresh as part of the "Send/Receive" but can also be configured through "Account Settings...Internet Calendars". I have done exactly what you are looking to do, same library and all. The update worked a treat within a large organisation with many people subscribing to the calendar files. This might help as well: http://blogs.msdn.com/b/michael_affronti/archive/2006/05/10/594986.aspx
