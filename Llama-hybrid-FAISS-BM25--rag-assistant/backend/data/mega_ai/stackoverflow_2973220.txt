Take a look here: http://publib.boulder.ibm.com/infocenter/wasinfo/v6r0/topic/com.ibm.websphere.express.doc/info/exp/ae/twbs_cookie.html and set the following HTTP headers: Expires with the value a hard coded GMT date in the past Last-Modified with the value the current date in GMT formatted "EEE, d MMM yyyy HH:mm:ss" Cache-Control with the following value 'no-store, no-cache, must-revalidate' Cache-Control with the following value 'post-check=0, pre-check=0' Pragma with the following value 'no-cache'
