$max = (array_sum($array) / sizeof($array)) + max($array) Examples: function getMax($array){ return (array_sum($array) / sizeof($array)) + max($array); } $array = Array(1, 2, 3, 4, 5); echo getMax($array) . ' '; $array = Array(100, 200, 300, 400, 500); echo getMax($array) . ' '; $array = Array(1123212, 2143212, 8323212, 7223212, 1023212); echo getMax($array) . ' '; Output: 8 800 12290424 So $max would never be more than array_max($array) + the average of the array values.
