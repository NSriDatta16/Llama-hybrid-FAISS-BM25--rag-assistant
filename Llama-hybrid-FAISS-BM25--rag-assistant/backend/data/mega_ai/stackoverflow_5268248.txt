in the beginning of every header file, #ifndef __MYHEADER_H__ #define __MYHEADER_H__ in the end, #endif /* __MYHEADER_H__ */ can avoid contain a header file repeatedly. if you use visual stuido, you can just put #pragma once in the beginning of header file. By the way, you can use some static code check tool to find these kinds of issues, such as lint.
