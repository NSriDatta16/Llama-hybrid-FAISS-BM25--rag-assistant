You can either do something like #ifdef _MSC_VER #pragma pack(push, 16) #endif /* your struct defs */ #ifdef _MSC_VER #pragma pack(pop) #endif to give a compiler directive forcing alignment Or go into the project options and change the default struct alignment [under Code Generation]
