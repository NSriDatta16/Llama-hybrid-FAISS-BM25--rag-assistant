I found this to be a good solution $("#myDialogID").dialog({ closeOnEscape: false, beforeClose: function (event, ui) { return false; }, dialogClass: "noclose" }); Not altering the existing styles, instead adding a new bit: .noclose .ui-dialog-titlebar-close { display:none; } Adding the class ended up being quite an elegant method, as i'm "classing" the dialog as one that cannot be closed.
