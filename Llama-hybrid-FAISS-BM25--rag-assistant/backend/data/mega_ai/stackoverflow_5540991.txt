In the file app/code/core/Customer/etc/config.xml , Magento defines the formats for addresses. If you want to modify all html-formatted addresses to contain company info, you can override this file (specifically, the path default/customer/address_templates/html ): {{depend company}}{{var company}} {{/depend}} {{if street1}}{{var street1}} {{/if}} {{depend street2}}{{var street2}} {{/depend}} {{depend street3}}{{var street3}} {{/depend}} {{depend street4}}{{var street4}} {{/depend}} {{if city}}{{var city}}, {{/if}}{{if region}}{{var region}}, {{/if}}{{if postcode}}{{var postcode}}{{/if}} {{var country}} {{depend telephone}}T: {{var telephone}}{{/depend}} {{depend fax}} F: {{var fax}}{{/depend}}]]> Otherwise, you could create your own module and add a new address format in your XML config file. Then, you would need to modify your email templates to use the new format: {{var order.getBillingAddress().format('yourformat')}}
