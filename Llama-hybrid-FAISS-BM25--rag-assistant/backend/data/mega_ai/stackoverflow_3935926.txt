Ok, I found a way: you have to add "gzip" to the User-Agent. NSMutableURLRequest* request = [NSMutableURLRequest requestWithURL:[NSURL URLWithString:myURL]]; [request addValue:@"gzip" forHTTPHeaderField:@"User-Agent"]; If you do it this way, you will lost default values set by the framework (appName/Version CFNetwork/blah blah blah), but GAE returns a gzipped response.
