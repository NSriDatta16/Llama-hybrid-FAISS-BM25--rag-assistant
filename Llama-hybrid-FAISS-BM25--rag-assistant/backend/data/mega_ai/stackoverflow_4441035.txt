You may have tested everything except what the documentation says at http://www.mono-project.com/Embedding_Mono : The params array contains the arguments to the method with the same convention: MonoObject* pointers for object instances and pointers to the value type otherwise. void* args[1]; int val = 10; MonoObject *result; args [0] = &val; result = mono_runtime_invoke(mono_method, NULL, args, NULL); /* result will be a boxed IntPtr since pointer types can't be boxed themselves */
