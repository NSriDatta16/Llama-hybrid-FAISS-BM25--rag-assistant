Yes. Fact about kernels: $K$ is a kernel iff for some $\phi : \mathbb{R}^n \to \mathbb{H}$, ($\mathbb{H}$ is an appropriate Hilbert space) $K(x,y) = \langle\phi(x), \phi(y) \rangle$ From this fact, for $\alpha \geq 0$, $\alpha K(x,y) = \alpha \langle\phi(x), \phi(y) \rangle = \langle \sqrt{\alpha} \phi(x), \sqrt{\alpha} \phi(y) \rangle$ It follows that $K'(x,y) = \alpha K(x,y)$ is a kernel for $\phi'(x) = \sqrt{\alpha}\phi(x)$ A kernel doesn't have to satisfy $K(x,x) = 1$. The simplest example for which $K(x,x) \neq 1$ is linear kernel, $K(x,y) = \langle x,y \rangle$ for $\|x\| \neq 1$ In fact, from the definition of kernel, it follows that $K(x,x) = 1$ if and only if $\|x\|=1$.
