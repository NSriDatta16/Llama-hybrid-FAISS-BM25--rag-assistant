I figured out how it is done. I made two drop-areas to which the blocks can be dragged. To make sure that after placing back a block, the CSS positioning doesn't make the draggable 'disappear', I remove the positioning code: $( "#source" ).droppable({ drop: function(event, ui) { var $draggable = $(ui.draggable).detach(); $draggable.css({'left':'','top':'','right':'','bottom':''}) ; $draggable.prependTo($(this)); } }); $( "#destination" ).droppable({ drop: function(event, ui) { var $draggable = $(ui.draggable).detach(); $draggable.prependTo($(this)); } });
