The documentation says: varchar [ ( n | max ) ] Variable-length, non-Unicode character data. n can be a value from 1 through 8,000. max indicates that the maximum storage size is 2^31-1 bytes. So -1 isn't technically valid. It might work, but I wouldn't use it if I were you.
