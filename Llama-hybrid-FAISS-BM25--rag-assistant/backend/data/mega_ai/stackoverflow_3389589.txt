The simplest answer would be that when you start a program in Visual Studio (2003 and onward) it creates namespaces by default. Other (older) IDEs do not, I suppose. .NET has encouraged it from the start. Namespaces weren't always part of the original implementation in C++. It has gained traction with .Net.
