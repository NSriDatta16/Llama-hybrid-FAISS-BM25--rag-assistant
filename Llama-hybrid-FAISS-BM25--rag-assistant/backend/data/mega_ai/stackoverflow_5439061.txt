Not requiring session affinity and thus using session state providers that rely on Azure storage or SQL Azure storage is the best choice. For some legacy applications you may still need session affinity. For those cases, ARR between Azure load balancer and the Web farm is an option. More details at http://go.archims.fr/hW54Xz
