If you server is *nix, the simplest solution is to leverage Ruby's OS integration and call GNU tar using backquotes. To create an archive: `tar cvzf #{archive_file_name}.tar.gz #{dir_to_be_archived}` To unpack it: `tar zxvf #{archive_file_name}.tar.gz`
