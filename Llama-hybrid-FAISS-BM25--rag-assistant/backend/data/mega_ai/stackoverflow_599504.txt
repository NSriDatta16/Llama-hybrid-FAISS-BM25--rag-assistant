You can try Yahoo's Term extraction service Update 02-May-2012 Yahoo has release a new Content Analysis Web service which will be replacing the Term extraction service. It can be invoked by syntactically sweet YQL as shown below select * from contentanalysis.analyze where url='http://www.cnn.com/2011/11/11/world/europe/greece-main/index.html'; There is also AlchemyAPI which offers a weservice which may serve your purpose.
