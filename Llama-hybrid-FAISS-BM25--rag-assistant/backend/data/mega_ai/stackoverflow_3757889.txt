Lets be pragmatic/objetive: Do you write to your data on runtime? Yes: Db, No: File Do you update your data more than once per week? Yes: Db, No: File It's a pain to release an updated data file? Yes: Db, No: File, Do you read that data often? Yes: File/Cache, No: Db It is a pain to update that data file and you need extra tools? Yes: db, No: File For sure I've forgotten other points, but I guess the basics are there.
