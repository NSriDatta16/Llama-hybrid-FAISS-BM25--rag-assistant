In your case, it will produce an error. :-) Set assigns an object reference. For all other assignments the (implicit, optional, and little-used) Let statement is correct: Set object = New SomeObject Set object = FunctionReturningAnObjectRef(SomeArgument) Let i = 0 Let i = FunctionReturningAValue(SomeArgument) ' or, more commonly ' i = 0 i = FunctionReturningAValue(SomeArgument)
