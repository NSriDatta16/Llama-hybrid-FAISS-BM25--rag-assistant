Simply apply draggable to the newly created DIV. If the DIV is exactly the same, except for position, you could also not recreate it, but just reposition it. This would retain all of it's handlers. // remove, reposition, reapply handlers $('#jobN').remove().appendTo('#dayViewN').draggable(); or // reposition, retain handlers $('#jobN').appendTo('#dayViewN');
