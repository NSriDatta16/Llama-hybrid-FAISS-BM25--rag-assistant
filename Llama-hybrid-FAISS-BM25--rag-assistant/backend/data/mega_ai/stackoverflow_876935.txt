You could use what Apple calls a SourceList. You see them in action in iTunes and in Mac OS X's Finder. It is an elegant solution to the problem you describe. A cross-platform, open source Java Swing component for doing this is here: http://explodingpixels.wordpress.com/2008/09/08/mac-widgets-for-java/
