The focus of code coverage is only if a line is executed or not. Nothing more. To get counters you would have to look at profiler software and not code coverage. There are plugins for both Eclipse and Netbeans. I am not sure if they show counters per line, but I think that they show counters per method, which might be what you want... (source: free.fr )
