You can do it if you have a single containing element. $(" ").append(data).hide().appendTo("#posts").slideDown(); Let me give you an example: var content = ' four five six '; $(function() { $("#add").click(function() { $(" ").append(content).hide().appendTo("#container").slideDown(1000, function() { $(this).contents().unwrap(); }); }); }); div, p { padding: 0; margin: 0; border: 0 none; } #container { background: yellow; border: 1px solid black; } one two three Here the content is not one block, it's three (paragraphs). If it was one block you wouldn't need to wrap it like this.
