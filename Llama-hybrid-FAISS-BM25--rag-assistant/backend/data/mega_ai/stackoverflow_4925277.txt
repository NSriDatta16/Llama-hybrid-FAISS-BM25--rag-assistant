Have you tried using CSS yet to stop this? textarea { resize: none; } That should disable the drag-to-expand feature. Update You could also set this programmatically by using jQuery: // CSS textarea.no-expand { resize: none; } // jQuery $('#my-text-area').addClass('no-expand'); Hope this helps!
