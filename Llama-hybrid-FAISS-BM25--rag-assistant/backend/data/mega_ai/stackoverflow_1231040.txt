Go to the magento admin site, choose System->Configuration->Advanced->Developer and turn on template tag hints. (Make sure your store name is selected in the Current Configuration Scope dropdown) You can then go back to your store homepage and you will be able to see which template is used for the left column. Modify the .phtml template file and add your image there.
