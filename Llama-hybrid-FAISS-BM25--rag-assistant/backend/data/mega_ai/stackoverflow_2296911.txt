Its easier to Dim a string and put your entire expression there and then debug it to see if it contains the valid report name. Like this: Dim strReportName as String strReportName = [Client Organisations].Code + "-" + Clients.Code + "-" + Invoices_Code + "-" + Format([Invoice Date],"yyyy") + ".pdf" //then try to print strReportName before you use DoCmd.OutputTo.
