If you are looking for server performance only, LoadRunner will simulate different browser engines. If you need the client side performance as well, that would require a functional testing tool. LoadRunner > Run-Time Settings > Browser Emulation > User-Agent Set this string to the desired string and as far as the server is concerned, this is the browser making the requests. BTW, BrowserMob is a great option!
