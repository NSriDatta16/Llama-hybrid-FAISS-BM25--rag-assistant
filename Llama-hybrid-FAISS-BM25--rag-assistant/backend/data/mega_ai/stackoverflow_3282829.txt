Just let read do the parsing stuff: while read page section rest do echo "Page: $page" echo "Section: $section" done For handling the optional argument elegantly, use an array: while read -a fields do wget --quiet --no-proxy "www.cs.sun.ac.za/hons/${fields[0]}" -O html.tmp unset "fields[0]" ./DokuWikiHtml2Latex.py html.tmp "${fields[@]}" done Always quote your variables!
