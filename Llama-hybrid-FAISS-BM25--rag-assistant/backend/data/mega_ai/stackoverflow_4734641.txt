Is ItemForm a ModelForm (see Django Model Forms )? If so form.save() will return a model instance. Then you can edit its fields if you need to. For example. my_obj = form.save() my_obj.current_item_status_date = datetime.date.today() my_obj.save() If not, then simply create a new instance of your model and save the field value. my_obj = StorageItem(current_item_status_date=datetime.date.today()) my_obj.save()
