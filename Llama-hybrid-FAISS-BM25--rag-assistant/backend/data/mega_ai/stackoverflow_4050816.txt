You can do this in the modern runtime (64-bit/iOS) with clang (“LLVM Compiler 1.5”) in Xcode 3.2.3 or 3.2.4, by adding -Xclang -fobjc-nonfragile-abi2 to the Other C Flags build setting. (Note that this is actually one option, not two.) Another effect of this flag is to cause properties to be synthesized by default.
