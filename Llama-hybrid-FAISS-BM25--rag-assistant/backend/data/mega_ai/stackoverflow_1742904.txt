Perhaps you could reduce your result set and use the > kind of paging controls. Nerd Dinner demo introduced an elegant solution using LINQ: // // GET: /Dinners/ // /Dinners/Page/2 public ActionResult Index(int? page) { const int pageSize = 20; var upcomingDinners = dinnerRepository.FindUpcomingDinners(); var paginatedDinners = upcomingDinners.Skip((page ?? 0) * pageSize) .Take(pageSize) .ToList(); return View(paginatedDinners); }
