hmm you could use a httpclient to get the complete content of the site and then send this to the user agent using httpcomponents httpclient or the jersey-client package as in: public void doGet(HttpServletRequest req,HttpServletResponse resp){ HttpClient client=new HttpClient() // dont instantiate like this it's a heavyweight ;) GetMethod get=new GetMethod("http://stackoverflow.com/"); int status=client.executeMethod(get); if (status=200){ resp.getWriter().write(get.getResponseBodyAsString().getBytes("UTF-8")); resp.getWriter().flush(); }else{ // handle error } get.releaseConnection(); }
