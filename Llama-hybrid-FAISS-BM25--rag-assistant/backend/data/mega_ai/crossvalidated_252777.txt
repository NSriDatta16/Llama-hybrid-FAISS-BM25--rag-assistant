An Elman network leaves the choice of the activation function to the user, since it only specifies these equations for the recurrence: \begin{align} h_t &= \sigma_h(W_{h} x_t + U_{h} h_{t-1} + b_h) \\ y_t &= \sigma_y(W_{y} h_t + b_y) \end{align} Variables and functions: $x_t$: input vector $h_t$: hidden layer vector $y_t$: output vector $W$, $U$ and $b$: parameter matrices and vector $\sigma_h$ and $\sigma_y$: Activation functions {1}, which is one of the most cited references defining an Elman network, doesn't seem to indicate that an Elman network should use some specific activation functions either.
