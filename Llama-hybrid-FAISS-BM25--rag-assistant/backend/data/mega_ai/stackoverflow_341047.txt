Depending on what tools you are looking at, just give him SQL Server Management Studio. He can then execute the stored procedure and save the results in a CSV format. That is your most elegant way to be able to do it in a dynamic fashion. Otherwise, you might look at creating an SSIS package if the values don't change often.
