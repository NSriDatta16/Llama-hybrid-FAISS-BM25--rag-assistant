Your question is slightly confusing because you state that the error didn't occur when using Haml for the view, but in the second paragraph you write that you get the error using Haml if you uncomment the fields line. Rails is looking for a view helper method named render_sequence_nav . This should be located in app/helpers/application_helper.rb or whichever helper corresponds to the controller for this view.
