You may use this : ls -R | grep ":$" | sed -e 's/:$//' -e 's/[^-][^\/]*\//--/g' -e 's/^/ /' -e 's/-/|/' Pay attention, this command will recursively list all files in the current directory, so don't run this command from the root directory ;)
