If you're going to be adding similar elements over and over, I would avoid using lengthy strings in your code. They are a pain to maintain, and create difficult-to-read code. I would instead encourage you to use a templating tool like Handlebars : {{filename}} Remove Now we build a compiled template from the above: var template = Handlebars.compile( $("#tmpl").html() ); All that is left now is to call the template function with our data object, append the resulting markup to our container (or body), and fade it in. var data = { filename: "foo.png", safeUrl: encodeURIComponent("foo image") }; $("body").append(template(data)).fadeIn(); The result is cleaner, more legible, code that will make sense and be easy to maintain.
