PRAGMA INDEX_LIST('table_name'); Returns a table with 3 columns: seq Unique numeric ID of index name Name of the index unique Uniqueness flag (nonzero if UNIQUE index.) Edit Since SQLite 3.16.0 you can also use table-valued pragma functions which have the advantage that you can JOIN them to search for a specific table and column. See @mike-scotty's answer .
