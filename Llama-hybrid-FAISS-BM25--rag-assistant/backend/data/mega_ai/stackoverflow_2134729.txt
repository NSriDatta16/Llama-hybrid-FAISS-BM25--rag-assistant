Another solution is to use associative storage. UIActionSheet might already have it. You can test it by [myActionSheet setValue:@"test value" forKey:@"testKey]; NSLog(@"%@",[myActionSheet valueForKey:@"testKey]; Associative storage is rather nifty if not overused.
