Criteria is probably not the best API for this... HQL is easier: var fooDTOs = session.CreateQuery(@" select foo as Foo, count(elements(foo.Bars)) as BarCount from Foo foo group by foo.id, foo.ImportantNumber ") .SetResultTransformer(Transformers.AliasToBean ()) .List ();
