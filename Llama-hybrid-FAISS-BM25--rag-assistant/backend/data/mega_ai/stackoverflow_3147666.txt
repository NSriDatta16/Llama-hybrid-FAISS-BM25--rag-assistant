There is a Raw API discussed in a previous StackOverflow question that you can leverage. Depending on your scenario, be careful about restricting accessibility - for example screen-input devices such as tablet touchscreens, hand-writing recognition or software designed to support those with disabilities may still be legitimate input sources for your application even if the input isn't from a traditional physical keyboard.
