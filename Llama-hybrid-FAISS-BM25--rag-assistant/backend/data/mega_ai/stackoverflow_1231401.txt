I solved problem with UrlMkSetSessionOption(), it changed IE settings for current session only: const char ua[] = "My user agent string"; HRESULT rez = UrlMkSetSessionOption(URLMON_OPTION_USERAGENT, (LPVOID)ua, sizeof(ua), 0); Just one but, it change settings for once. Another calls return no error and no changes. Some references: Changing the user agent of the WebBrowser control
