I sounds like your need to consider Language Understanding services instead. There are a couple of providers of this type of service. The one I have used before is called LUIS from Microsoft: https://www.luis.ai/home/index It is not the algorithm you are looking for per se, but to solve your problem by implementing an algorithm, you need to develop a language understanding system using models based on deep learning neural nets, which is not a trivial exercise.
