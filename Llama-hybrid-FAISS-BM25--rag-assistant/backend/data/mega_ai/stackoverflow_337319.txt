You can assign an onload event handler to an element within your SVG document and have it call a javascript function in the html page. onload maps to SVGLoad. http://www.w3.org/TR/SVG11/interact.html#LoadEvent The event is triggered at the point at which the user agent has fully parsed the element and its descendants and is ready to act appropriately upon that element
