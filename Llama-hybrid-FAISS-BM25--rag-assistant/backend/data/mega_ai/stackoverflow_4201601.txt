Ha, standard! No, certainly nothing like that. Pick your favorite, whichever looks nice and is organized well and is on everyone's computer already, and mimic it. My opinions follow: I don't see any value in alphabetical. Order it logically, arranged into categories and put useful stuff first. Make sure the parsing engine is solid and "standard", preferably using someone else's. Boost's Program Options or Python's optparse are good if they're in the right language for you. Most other languages have some too. Make sure to include many examples that span the gamut of use.
