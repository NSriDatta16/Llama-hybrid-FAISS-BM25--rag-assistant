urllib2 respects robots.txt. Many sites block the default User-Agent . Try adding a new User-Agent , by creating Request objects & using them as arguments for urlopen : import urllib2 request = urllib2.Request('http://www.example.com/') request.add_header('User-agent', 'Mozilla/5.0 (Linux i686)') response = urllib2.urlopen(request) Several detailed walk-throughs are available, such as http://www.doughellmann.com/PyMOTW/urllib2/
