You could use Array.Sort : int[] array = new[] { 1, 3, 2 }; Array.Sort(array, (x, y) => y.CompareTo(x)); As far as complexity is concerned: On average, this method is an O(n log n) operation, where n is the Length of array; in the worst case it is an O(n ^ 2) operation
