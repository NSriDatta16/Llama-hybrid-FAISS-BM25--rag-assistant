Yes, it is possible most easiest way is to return json_encoded string. so your code would look like this php: header("Expires: Mon, 26 Jul 1997 05:00:00 GMT"); header("Last-Modified: " . gmdate("D, d M Y H:i:s") . "GMT"); header("Cache-Control: no-cache, must-revalidate"); header("Pragma: no-cache"); header("Content-type: text/x-json"); echo json_encode($array); exit(0); and JavaScript would look like this: $.get('/forms/requestProcessor.php', function(data) { .... }, 'json');
