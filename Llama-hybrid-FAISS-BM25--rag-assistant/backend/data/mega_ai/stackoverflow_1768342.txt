Relatively simple conversion by looping: $newArray = array() foreach ($arr as $a) { foreach ($a as $key => $value) { $newArray[] = $value; } } Or, perhaps more elegantly: function flatten($concatenation, $subArray) { return array_merge($concatenation, array_values($subArray)); } $newArray = array_reduce($arr, "flatten", array()); John's solution is also nice.
