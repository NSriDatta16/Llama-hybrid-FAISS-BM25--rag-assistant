Google Translate API v2 allows automated queries but it requires the use of an API key that you can freely get at Google APIs console . To detect whether text is English you could use detect_language_v2() function (that uses that API) from my answer to the question Python - can I detect unicode string language code? : if all(lang == 'en' for lang in detect_language_v2(['some text', 'more text'])): # all text fragments are in English
