One way to frame the problem as approximate nearest neighbors retrieval. Given a set of features for a posting, what are the nearby candidates? Or given a set of features for posting, what are nearby candidates? In order to find the nearest neighbors, all entities need to be in the same feature space. That features space can be a learned embedding. One implementation of learned embedding is StarSpace .
