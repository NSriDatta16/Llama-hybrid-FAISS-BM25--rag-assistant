One way to find out if your method is called is to use a code coverage tool. Visual studio Ultimate contains such tools to help determine code coverage. You first instrument your code, and then run your tests, or exercise your app normally, and then check your results. Or you can just use a text editor to search for the method name in your source code. (That is by far the easiest).
