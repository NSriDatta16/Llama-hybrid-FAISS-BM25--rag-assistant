You can utilize modulo and a hash table in order to achieve some code style elegance. Pseudo code: int getLevel(int num) { hash_table = (30=>1, 49=>2, 79=>3); foreach threshold (keys hash_table) { if (num modulo (threshold + 1) == num) { return hash_table[threshold]; } } }
