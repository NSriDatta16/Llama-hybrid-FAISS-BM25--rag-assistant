You place the code in app/code/local/Packagename/Modulename Where Packagename/Modulename applied to your specific module (if you have the code it should already be in this structure) Then, in app/etc/modules Add an XML file named Packagename_Modulename.xml with the following contents true local > Again, replacing Packagename_Modulename with the specific packagename and module name of the module you're installing. You'll need to clear your Magento cache to see the changes take place.
