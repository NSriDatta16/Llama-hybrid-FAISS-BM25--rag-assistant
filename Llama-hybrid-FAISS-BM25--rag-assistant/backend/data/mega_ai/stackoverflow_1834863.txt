To create version number that is same across all the JSP pages. Have your build process create a file that contains the version number. execute something like this in your build: svnversion -c trunk > version.txt Package version.txt with your web application (inside .war file). Create a function to read version information from the version.txt file Call function from JSP pages and display version information. (Preferably from some footer/header fragment that is included in every page).
