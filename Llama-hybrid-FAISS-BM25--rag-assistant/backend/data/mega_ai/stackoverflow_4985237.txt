Do this: When user logs in, set User object for that user in HttpSession. This way, httpRequest.getSession().setAttribute("LOGGED_USER", userObject) Now, every time you hit the filter/security filter. The first thing you do is check for this attribute. If the attribute is not there, redirect/forward the request to login servlet. The pseudo code would look like this: //in your login servlet, on successful login request.getSession().setAttribute("LOGGED_USER", userObject); //in your security filter if(request.getSession().getAttribute("LOGGED_USER") == null){ //optionally, you may like to check if that attribute has a valid userId as well RequestDispatcher rd = request.getRequestDispatcher("relative/path/to/login/servlet") rd.forward(request, response); return; } Edit 1: see this http://download.oracle.com/javaee/5/tutorial/doc/bncbx.html
