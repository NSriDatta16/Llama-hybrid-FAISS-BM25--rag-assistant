Include user agent to your WebClient Headers Collection. I was able to get the html with the following. Dim WebRequest As WebClient = New WebClient WebRequest.Headers(HttpRequestHeader.UserAgent) = "Mozilla/4.0 (compatible; MSIE 7.0; Windows NT 5.1; .NET CLR 2.0.50727)" Dim URL as string = "http://www.professionalorganizervannuys.com" Dim WebPage As String = WebRequest.DownloadString(URL)
