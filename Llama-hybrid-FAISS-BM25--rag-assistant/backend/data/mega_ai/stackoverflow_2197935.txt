As @quickanalysis pointed out, you've to be aware of the separation of client-/server-side components. For getting the user agent string on server-side, the following code snippet does the job: ApplicationContext context = this.getContext(); if (context instanceof WebApplicationContext) { String userAgent = ((WebApplicationContext)this.getContext()). getBrowser().getBrowserApplication(); }
