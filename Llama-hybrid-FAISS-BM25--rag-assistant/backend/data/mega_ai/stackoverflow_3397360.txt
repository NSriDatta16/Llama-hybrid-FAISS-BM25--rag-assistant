I would definitely use blocktrans. Sometimes its not possible to split i18n html text into different fragments. Blocktrans has some powerfull features: {% url path.to.view arg arg2 as the_url %} {% blocktrans with object.title as title and author|title as author_t %} {{author}}: Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec quam sem, sodales in fringilla nec, lacinia a lorem. {{title}} molestie ante. {% endblocktrans %} Have a look at: url template tag blocktrans template-tag
