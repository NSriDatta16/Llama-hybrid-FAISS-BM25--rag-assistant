Apparently a bug in jQuery UI that was fixed in jQuery UI 1.8.6. You are using 1.7.2. It wasn't stopping selection.. Reference posts: http://forum.jquery.com/topic/chrome-text-select-cursor-on-drag http://bugs.jqueryui.com/ticket/4163 One solution: $(".ui-draggable").each(function() { this.onselectstart = function() { return false; }; });
