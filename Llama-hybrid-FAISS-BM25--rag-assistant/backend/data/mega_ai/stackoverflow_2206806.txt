I think I actually came up with a better solution... seems to work so far... I attached a droppable to a sortable to declare a global var of the dragged item. A bit like so: $(".sortlist").droppable({ drop: function(e, ui) { draggedItem = ui.draggable; } }).sortable({ receive: function(event, ui) { //do something with the 'draggedItem' here... var droppedElemTxt = draggedItem.text(); var droppedElemId = draggedItem.attr('id'); } });
