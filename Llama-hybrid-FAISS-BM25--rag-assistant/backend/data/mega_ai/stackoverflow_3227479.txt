A cookie will fit. Just at each page view you check if it is set. If so, don't count it. Otherwise, count it and set the cookie. You will have problems with people not having cookies enabled or cleaning them out after each visit. You should pair this approach with PHP sessions: instead of doing the process on each page view, do it only once, on session start. This will at least avoid counting visits within a single session. All this is you want to store everything client-side. If you were server-side, you could use IP addresses, or pair them with the user agent string. Or, just use google analytics...
