The fastest, but probably not the most elegant way is: write a script that mimicks user behavior when selecting and adding text plugin from a dropdown; override PageAdmin to include our script. It goes like this: # anywhere in your project, for example, site/admin.py from cms.models import Page from cms.admin.pageadmin import PageAdmin class ModPageAdmin(PageAdmin): class Media: js = ('js/cms.page.js',) admin.site.unregister(Page) admin.site.register(Page, ModPageAdmin) # in MEDIA_URL/js/cms.page.js $(document).ready(function(){ ph = $("div.form-row.main") // replace "main" with your placeholder name, lower-case $("select", ph).val('TextPlugin') window.setTimeout(function(){ $("span.add-plugin", ph).click() }, 500) })
