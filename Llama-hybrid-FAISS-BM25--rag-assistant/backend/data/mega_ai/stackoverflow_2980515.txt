It is not sufficient to use the normalize-space() function as it is leaving mid-spaces! Here is a simple and complete solution: http://latex.codecogs.com/gif.latex? When this transformation is applied on the provided XML document : \text{average} = \alpha \times \text{data} + (1-\alpha) \times \text{average} the wanted, correct result is produced : Do note : The use of the XPath 1.0 translate() function to strip off all unwanted characters. There is no need to use the replace() function here -- it may not be possible to use it, because it is only available in XPath 2.0.
