Visual Assist: link . Install, load solution, press Shift + Alt + O , search for files in solution by substring. Try also Shift + Alt + S , for the equivalent for symbols. This addin has a bunch of completion popup and syntax colouring stuff in it that aren't to all tastes, but the code browsing features are done well and seem uncontroversial. Judging by comments on the forums, compatibility with Resharper is something they pay attention to. For free, try also Nifty Solution: link . I haven't used this myself, but I use the author's Nifty Perforce plugin, and that is pretty tidy.
