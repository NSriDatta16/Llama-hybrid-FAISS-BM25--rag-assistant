Here's how you can do it in two simple steps: Use a find and replace maybe with a regular expression to change all the instances Flowers.NAME to just NAME . Then do a project wide 'Organize imports' like so: Select the project in the package explorer and press Ctrl + Shift + O (same keystroke as the single class version). Should work for packages, etc..
