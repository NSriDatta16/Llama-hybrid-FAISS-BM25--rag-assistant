Open vcproj file on a text editor and locate entries starting with Scc (SccProjectName, SccAuxPath, SccLocalPath, SccProvider...). These entries define the SCC binding of the project. As an example, a project controlled by Source Safe looks like this: SccProjectName="&quot;$/Services&quot;. TPCAAAAA" SccLocalPath=".." SccProvider="MSSCCI:Microsoft Visual SourceSafe" A project controlled by SVN looks line this: SccProjectName="Svn" SccAuxPath="Svn" SccLocalPath="Svn" SccProvider="SubversionScc" sln files have similar entries that you can check
