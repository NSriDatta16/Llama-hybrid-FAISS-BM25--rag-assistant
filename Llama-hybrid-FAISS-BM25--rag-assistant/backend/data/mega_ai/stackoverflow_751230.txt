Check out the jQuery table drag'n'drop plugin . Then just tie the resulting javascript calls into your back-end using Ajax (eg. the a4j:jsFunction from Richfaces). Get the back-end Bean to handle the shuffling of the subsequent jobs. This will definitely look and behave better than any out-of-the-box component that you'll currently find in a JSF library.
