You should call resignFirstResponder for the UITextView . Let's say you have an IBOutlet for the text view: @property (nonatomic, retain) IBOutlet UITextView *comment; Then [comment resignFirstResponder]; can be called from a touchesBegan or the like. See e.g. How to Dismiss the Keyboard when using a UITextView .
