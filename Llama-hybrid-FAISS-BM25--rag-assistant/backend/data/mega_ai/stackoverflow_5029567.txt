You could decorate your view model property with the [DisplayFormat] attribute: [DisplayFormat(DataFormatString = "{0:D}", NullDisplayText = "pas disponible")] public DateTime? PublicationDate { get; set; } and then your view simply becomes: @Html.DisplayFor(x => x.PublicationDate) So now it is reduced to a single and elegant line.
