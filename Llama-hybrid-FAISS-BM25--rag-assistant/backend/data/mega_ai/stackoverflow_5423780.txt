Yeah, you just do this: category="3&7&8" http://expressionengine.com/user_guide/modules/channel/parameters.html#par_category I personally hate EEs categories though. The way I'd do it is to use Playa. This plugin is well worth it. So instead of using EE categories, you create a channel of all your categories (cities). Then you create a playa field in your Articles channel, which allows you to select multiple cities. Then in your template, you just write an if statement: {if cities="Atlanta && Sheboygan"}//DO SOMETHING{/if}
