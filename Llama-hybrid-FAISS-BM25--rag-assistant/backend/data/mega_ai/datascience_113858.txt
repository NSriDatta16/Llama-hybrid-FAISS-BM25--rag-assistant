Hadoop is a really good platform to solve Big Data which is usually the task of a data engineer , since he needs to provide the data that is needed. The tasks of a data scientist are: Identification and investigation of data sources within an organization Selecting the appropriate information for a use case Finding patterns in the data from which added value can be generated Using the patterns found to make predictions for the future that are as accurate as possible If you don't have the support of a data engineer in your company, it is probably a good idea to learn Hadoop as a data scientist
