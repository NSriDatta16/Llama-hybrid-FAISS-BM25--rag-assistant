The R package "Ranger" should do. https://cran.r-project.org/web/packages/ranger/ranger.pdf A fast implementation of Random Forests, particularly suited for high dimensional data. Compared with randomForest, this package is probably the fastest RF implementation I have seen. It treats categorical variables in a native way.
