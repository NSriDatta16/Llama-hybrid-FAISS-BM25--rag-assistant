I think the easiest way to achieve this is to transform your data by dividing every value by 10. So, instead of ranging from 1-10, it will range from 10% to 100%. After this transformation, you could use a Logit Regression model. In Python, you would use the following function: sklearn.linear_model.Logistic Regression.
