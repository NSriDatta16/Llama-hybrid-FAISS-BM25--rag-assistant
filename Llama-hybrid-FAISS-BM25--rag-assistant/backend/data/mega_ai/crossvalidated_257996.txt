What is maxnorm constraint? How is it useful in Convolutional Neural Networks?

Here is a keras code sample that uses it: from keras.constraints import max_norm model.add(Convolution2D(32, 3, 3, input_shape=(3, 32, 32), border_mode='same', activation='relu', kernel_constraint=max_norm(3)))
