I understand the Unit tests not running on the target. But you might want test coverage collected in the target anyway. See SD C++ Test Coverage for a tool that operates with minimal footprint in a practical way inside most targets. You have to customize a provided small data collection procedure for this to work; normally an afternoons' straightforward exercise.
