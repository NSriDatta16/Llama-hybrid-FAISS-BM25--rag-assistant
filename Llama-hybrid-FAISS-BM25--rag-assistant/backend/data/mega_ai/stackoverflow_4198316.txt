This should do the trick for you: $(function() { $('.expandableContent').click(function() { $('.services').hide(); $(this).nextAll('.services:first').show(); }); }); First, you'll hide all .services divs, then select the next one after the paragraph that was clicked, and expand it. Here's a demo of that in action: http://jsfiddle.net/Ender/5nYpT/
