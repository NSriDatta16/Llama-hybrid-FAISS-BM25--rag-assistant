I think a more elegant approach would be to use HQL JOIN FETCH clause, since it minimizes the number of SQL queries issued, as well as makes your code more clear: public List getTripListFullyInitliazed() { return getHibernateTemplate().find( "from Address a left join fetch a.children"); } See also: 16.3.
