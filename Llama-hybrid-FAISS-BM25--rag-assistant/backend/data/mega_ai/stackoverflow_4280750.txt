Not an elegant solution. However, you could make use of eval() : $all = array( 'lastsum' , 'avatar', 'angels', 'city', 'square' ); foreach ( $all as $var ) { $code = "\$${var}->dosomething();"; eval($code); } Otherwise, store the objects in an array: $all = array( $lastsum , $avatar, $angels, $city, $square ); foreach ( $all as $obj ) { $obj->dosomething(); }
