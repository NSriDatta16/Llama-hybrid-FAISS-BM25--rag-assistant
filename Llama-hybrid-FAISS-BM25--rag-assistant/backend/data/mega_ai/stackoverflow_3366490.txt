How to add paragraph and then a line?

All of the examples I have seen so far using ITextSharp start from scratch and create a new document, add something to it and close it. What if I need to do multiple things to a PDF for example I want to add a paragraph and then add a line. If I put the highlighted code after the pdfDoc.Close() I get an error saying "The Document is not opened" How do I switch from adding text to adding a line?
