You can always start at the top index and iterate downward towards 0: for (int i = _collection.Count - 1; i >= 0; i--) { User user = _collection[i]; if (!user.IsApproved()) { _collection.RemoveAt(i); } } Mehrdad's answer looks pretty darn elegant, though.
