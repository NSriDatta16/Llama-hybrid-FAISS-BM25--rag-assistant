Alternatively, do it in one Regex pass: error_messages = /I'm an error message!|Another error message/ if agent.page.body =~ error_messages ... end You'll need to ensure that you escape any error messages that contain special regex characters. To make it maintainable: if agent.page.body =~ Regexp.union("foo", "bar", "jim.bob", "jam|jam") ... end You should only use this if you have tested and found the speed of Nakilon's answer is not enough, however. :)
