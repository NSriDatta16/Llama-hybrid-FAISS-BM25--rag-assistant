Short answer: you shouldn't. Provinding the user with the same experience in any browser he uses is a major design principle (if there are no resolution contraints, like in mobile phones) If you still want to do it, detect it with: jQuery(document).ready(function($){ var deviceAgent = navigator.userAgent.toLowerCase(); var agentID = deviceAgent.match(/(ipad)/); if (agentID) { // do something special } }); reference: http://snipplr.com/view/31607/iphone-ipad-ipod-detect/
