See an expanded version of this answer here: Xcode 4 Archive Version Unspecified I've found many issues with Xcode 4 when it comes to complex project structures. Create a group in your project called "Indexing" Drag the header files to this group When asked to select a target uncheck ALL targets This has solved most of my Xcode 4 issues.
