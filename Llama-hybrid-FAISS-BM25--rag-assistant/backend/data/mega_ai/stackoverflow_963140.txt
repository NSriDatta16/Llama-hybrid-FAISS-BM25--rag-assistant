GWT has made considerable leaps and bounds over the past 7 months. The mutiple GWT widget libraries have received a good deal of attention, and Google's work on the library has been impressive. Going GWT limits you to: Using GWT widgets Possibly becoming a GWT widget developer The benefits include: Coding in Java Debugging in Java Skipping the parts of JavaScript that make you pull your hair out GWT is the step past AJAX. It buries the HTML / JavaScript so that you don't need to deal with it, but also provides access in case you do want to deal with it. Overall, you will spend less time trying to get small things working, and more time adding awesome to your application.
