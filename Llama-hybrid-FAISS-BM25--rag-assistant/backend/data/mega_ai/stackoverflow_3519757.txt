You can raise any event on any UIElement using the RaiseEvent() method. Just create the appropriate event args for the handler and pass to RaiseEvent() . var args = new DataObjectPastingEventArgs(dataObject, isDragDrop, formatToApply) { Source = this, RoutedEvent = DataObject.PastingEvent //set the event here }; element.RaiseEvent(args);
