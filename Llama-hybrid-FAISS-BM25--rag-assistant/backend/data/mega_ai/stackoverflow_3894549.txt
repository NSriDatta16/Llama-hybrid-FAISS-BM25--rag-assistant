disabled is a boolean attribute. disabled="disabled" is the correct form; disabled alone is shorthand allowed in HTML. From On SGML and HTML: Boolean attributes may legally take a single value: the name of the attribute itself (e.g., selected="selected"). In HTML, boolean attributes may appear in minimized form -- the attribute's value appears alone in the element's start tag. Thus, selected may be set by writing: instead of: Authors should be aware that many user agents only recognize the minimized form of boolean attributes and not the full form.
