If you want the smallest possible size for the struct (or to align it to a 4-byte boundary, or whatever), you can use the packed or aligned attributes. In Visual C++, you can use #pragma pack , and in GCC you can use __attribute__((packed)) and __attribute__((aligned(num-bytes)).
