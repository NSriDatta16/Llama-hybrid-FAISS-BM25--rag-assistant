I don't think you need to know ruby as such since both projects are domain specific languages for writing html and css. I think knowing a bit of Ruby will help you in using both projects since you can leverage Ruby's capabilities in abstracting common things away in your code. Regardless of whether you want to learn Ruby or not, you will have to install Ruby and the dependencies for Haml and Sass in order to be able to use them on Windows. Finally, Ruby Gems is a package manager for Ruby, that is used to install and manage Ruby libraries on your system. You can get Ruby from here and Ruby Gems here .
