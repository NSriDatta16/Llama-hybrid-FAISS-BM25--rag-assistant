How could that even be possible? More than possible, that's becoming more and more common as machine learning problems require unstructured features such as Natural Language Process (NLP), pictures, audios and videos. NLP example: If a feature generation process takes one ore more text features and generate n-grams for each word found in each observation, the bigger the sample size, the much bigger will be the n-gram feature vector, although inherently an sparse matrix.
