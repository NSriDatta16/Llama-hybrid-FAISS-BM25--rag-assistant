Use the Application_Error method in your Global.asax file. Inside your Application_Error method implementation call Server.GetLastError(), log the details of the exception returned by Server.GetLastError() however you wish. e.g. void Application_Error(object sender, EventArgs e) { // Code that runs when an unhandled error occurs log4net.ILog log = log4net.LogManager.GetLogger(typeof(object)); using (log4net.NDC.Push(this.User.Identity.Name)) { log.Fatal("Unhandled Exception", Server.GetLastError()); } } Don't pay too much attention to the log4net stuff, Server.GetLastError() is the most useful bit, log the details however you prefer.
