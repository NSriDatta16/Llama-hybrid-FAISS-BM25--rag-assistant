See the documentation for the Create Strong Name Key Dialog Box . The password information is stored in your computer's cryptographic storage database. Other information from CLR Inside Out: Using Strong Name Signatures . You can install the private key in a key container using restrictive access control lists (ACLs) to prevent unauthorized access to the key container. Alternatively, you can store the private key on a smart card or other separate hardware device. The sn.exe tool allows you to use a different cryptographic service provider (CSP) for signing.
