I would strongly suggest to use the Xcode organizer to upload your apps. Because it's impossible that such things (upload iphone simulator build) will happen. Select Device as Target in the Build settings pop-down. Select Build and Archive in the Build menu of xcode Select the app in Xcode organizer, and use the Validate... button. Log in to itunes connect when prompted. Submit... app. Be rich
