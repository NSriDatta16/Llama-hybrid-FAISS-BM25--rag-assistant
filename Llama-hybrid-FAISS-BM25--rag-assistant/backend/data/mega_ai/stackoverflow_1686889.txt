To drop non-ASCII characters use line.decode(your_file_encoding).encode('ascii', 'ignore') . But probably you'd better use PLM escape sequences for them: import re def escape_unicode(m): return '\\U%04x' % ord(m.group()) non_ascii = re.compile(u'[\x80-\uFFFF]', re.U) line = u'\\B1a\\B \\tintense, disordered and often destructive rage\u2020.\u2020.\u2020.\\t' print non_ascii.sub(escape_unicode, line) This outputs \B1a\B \tintense, disordered and often destructive rage\U2020.\U2020.\U2020.\t . Dropping non-ASCII and control characters with regular expression is easy too (this can be safely used after escaping): regexp = re.compile('[^\x09\x0A\x0D\x20-\x7F]') regexp.sub('', line)
