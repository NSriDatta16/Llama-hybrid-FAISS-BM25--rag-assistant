I'm not sure exactly what you're asking? Your button is like any other JButton: buttonForPicture.addMouseMotionListener(new MouseMotionListener() { @Override public void mouseMoved(MouseEvent e) { } @Override public void mouseDragged(MouseEvent e) { } }); That catches movement events for the whole button, not just the image.
