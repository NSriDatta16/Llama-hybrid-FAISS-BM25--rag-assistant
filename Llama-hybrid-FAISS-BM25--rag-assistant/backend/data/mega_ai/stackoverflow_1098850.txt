Assuming the associated organs of an image are stored in the collection Image.Organs, the following query will do the trick. It depends on your LINQ provider if some modification are required because of unsupported functions. public static IEnumerable WithOrgans( this IEnumerable qry, IEnumerable organs) { return qry.Where(image => organs.All(organ => image.Organs.Contains(organ))); }
