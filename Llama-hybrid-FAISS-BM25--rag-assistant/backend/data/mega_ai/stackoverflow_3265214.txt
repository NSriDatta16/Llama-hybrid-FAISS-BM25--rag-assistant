Works for me: var remQueries = new[]{"Maine", "Maryland", "Massachusetts"}; var suggestions = new[]{"Maine", "Maryland", "Massachusetts", "Michigan", "Minnesota", "Mississippi", "Missouri", "Montana"}; var unqiueSugs = from sug in suggestions where ! remQueries.Any(q => q.Equals(sug, StringComparison.OrdinalIgnoreCase)) select sug; unqiueSugs; { "Michigan", "Minnesota", "Mississippi", "Missouri", "Montana" } Your problem is elsewhere. Also note that if your real lists are bigger, you will probably find Except (with a custom IEqualityComparer ) faster.
