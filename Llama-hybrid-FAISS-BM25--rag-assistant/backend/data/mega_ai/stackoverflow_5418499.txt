For marking up drop area's I always use the start option of the draggable for adding a class, for deleting the class I use the stop option. Something like this: $(".draggable").draggable({ start: function() { $('.droppable').addClass('droppableArea'); }, stop: function() { $('.droppable').removeClass('droppableArea'); }, }); When dragging a file, a hover class is always nice on the droppable: $('.droppable').droppable({ over: function() { $(this).addClass('droppableHover'); }, out: function() { $(this).removeClass('droppableHover'); } }); And your css: .droppableArea { border: 1px solid black; } .droppableHover { border: 1px solid red; } You can check it out in action here
