There are no built-in limitations - from SQL Server 2005 Workgroup Edition : Workgroup Edition is the data management solution for small organizations that need a database with no limits on size or number of users . Workgroup Edition can serve as a front-end Web server or for departmental or branch office operations. It includes the core database features of the SQL Server product line and is easy to upgrade to Standard or Enterprise Edition.
