You could extract the creation of the value in its own method: final Object myObject = getObjectOrNull(); public Object getObjectOrNull() { try{ return somethingThatCanFail(); } catch (Fail f) { return null; } } It's longer, but depending on your definition of "elegant" it might be more elegant.
