I'd do it in several stages: RegExp: Convert all br-tags to line-breaks. RegExp: Strip out all the white-space. RegExp: Convert the multiple line-breaks to single ones. Use Array.split('\n') on the result. That should give an array with all the 'real' paragraphs (in theory.) Then you can just iterate through it and wrap each line in p-tags.
