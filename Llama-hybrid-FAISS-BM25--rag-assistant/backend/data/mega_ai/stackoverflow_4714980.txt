To prevent caching of dynamic content, all I use is this (and I haven't noticed any caching issues yet): header('Cache-Control: no-cache, no-store, must-revalidate'); // HTTP 1.1 header('Pragma: no-cache'); // HTTP 1.0 header('Expires: 0'); // Proxies This is (hopefully) the PHP equivalent of what my Java apps use - apologies for any translation errors.
