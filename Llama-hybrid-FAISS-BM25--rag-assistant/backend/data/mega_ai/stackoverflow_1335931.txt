When you use LEFT JOIN , the LEFT table is always leading in MySQL . In your initial query, MySQL could choose which table to make leading, and it chose clg . Now it cannot choose, and this condition: date_add('1970-01-01 02:00:00', interval clg.uniqueid second) is not sargable. There is no index on date_add('1970-01-01 02:00:00', interval clg.uniqueid second) which MySQL could use to locate the value of val.event_time . Rewrite your query as this: SELECT val.PersonNo, val.event_time, clg.number_dialed FROM vicidial_agent_log val LEFT OUTER JOIN call_log clg ON clg.uniqueid = UNIX_TIMESTAMP(val.event_time) - 7200 ORDER BY val.event_time desc LIMIT 100
