It is the property of CNNs that they use shared weights and biases(same weights and bias for all the hidden neurons in a layer) in order to detect the same feature. This leads to a more deep learning as compared to simple neural networks. You can read this out as a reference : http://deeplearning.net/tutorial/lenet.html http://neuralnetworksanddeeplearning.com/chap6.html#introducing_convolutional_networks
