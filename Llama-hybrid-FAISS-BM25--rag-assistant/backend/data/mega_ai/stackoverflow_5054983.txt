I've had the same problem. This line did'nt work: set :ssh_options, {:forward_agent => true} Then I executed mentioned on Dreamhost wiki [local ~]$ eval `ssh-agent` [local ~]$ ssh-add ~/.ssh/yourpublickey # omit path if using default keyname And now I can deploy without password.
