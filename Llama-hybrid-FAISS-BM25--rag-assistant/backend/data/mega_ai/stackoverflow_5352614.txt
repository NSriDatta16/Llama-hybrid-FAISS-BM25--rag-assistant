As already said interface methods ARE public and abstract . It is now discouraged to specify them as public abstract: For compatibility with older versions of the Java platform, it is permitted but discouraged, as a matter of style, to redundantly specify the abstract modifier for methods declared in interfaces. It is permitted, but strongly discouraged as a matter of style, to redundantly specify the public modifier for interface methods. And for completeness interface fields ARE public static and final ...
