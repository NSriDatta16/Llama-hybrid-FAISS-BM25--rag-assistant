"Packages are nothing more than the way we organize files into different directories according to their functionality, usability as well as category they should belong to." from: http://www.jarticles.com/package/package_eng.html Basically, in your example, you are saying class masnun is package of package masnun. So essentially in other classes you could reference this class by import masnun; Which imports the class masnun which resides in masnun package.
