At the heart of a chat bot there is a natural language processor (NLP), the engine implements algorithms that would break a sentence entered by a human (e.g plain English) to a series of token the computer can process. while I am not familiar with a chat bot framework there are several open source NLP engines you can utilize to implement a chat bot: for example Open NLP googling for "natural language processor" will point you to other sources
