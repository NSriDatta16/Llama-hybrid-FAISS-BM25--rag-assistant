The project Browser Launcher ( http://browserlaunch2.sourceforge.net/ ) can be the solution for your problem. You can launch a browser from your code like this: String url = "http://...."; BrowserLauncher() launcher = new edu.stanford.ejalbert.BrowserLauncher(); launcher.openURLinBrowser(url);
