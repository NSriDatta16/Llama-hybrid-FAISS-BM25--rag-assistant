Improving Don's answer per OP comments, you can add a column to know what table the row was in. SELECT 'events' As Tbl, datetime, location, organizer, NULL as notes FROM events UNION ALL SELECT 'user_notes' As Tbl, datetime, NULL, NULL, notes FROM user_notes ORDER BY datetime DESC
