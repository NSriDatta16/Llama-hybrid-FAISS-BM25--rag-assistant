There may be a more elegant way to do this, but this is what I'd do: #ifdef DEBUG //Load debug plist #else #ifdef STAGING //Load staging plist #else //Load release plist #endif //staging #endif //debug Add 'DEBUG=1' to the preprocessor symbol definitions in the Debug configuration and 'STAGING=1' to the Staging configuration
