The jQuery animate enhanced plugin uses CSS transitions without having to write specific code for transition capable browsers The alternative is not very encouraging: you could add a feature detection library such as Modernizr and then write specific code for every case, such as... if (Modernizr.csstransitions) { $("#yourdiv").css({ "-webkit-transform" : "translate(0, 10)", "-o-transform" : "translate(0, 10)", "-moz-transform" : "translate(0, 10)", "-ms-transform" : "translate(0, 10)", "transform" : "translate(0, 10)" }); } else { //do jquery animate stuff }
