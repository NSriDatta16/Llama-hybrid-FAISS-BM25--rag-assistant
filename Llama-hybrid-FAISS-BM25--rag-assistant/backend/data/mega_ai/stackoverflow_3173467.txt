You can wrap plugins in an anonymous function and call them with the specific version of jquery. Example: $jq13 = jQuery.noConflict(true); $jq14 = jQuery.noConflict(true); (function($){ // All references to $ in this block will refer to $jq14 // Put plugins inline here })($jq14); Well written plugins will already be written this way, so you can just change the last line to set which jquery object it should use. Or you could change window.jquery before calling each plugin. Like this: $jq13 = jQuery.noConflict(true); $jq14 = jQuery.noConflict(true); window.jQuery = $jq14; window.jQuery = $jq13; A little less elegant, but it should work.
