This behavior actually contradicts to the Eclipse UI metaphor where editors are shared between perspectives. So you have to implement your own persistent storage for perpective:editors map. Also look at http://help.eclipse.org/help33/topic/org.eclipse.platform.doc.isv/reference/api/org/eclipse/debug/ui/sourcelookup/CommonSourceNotFoundEditorInput.html#getPersistable ()
