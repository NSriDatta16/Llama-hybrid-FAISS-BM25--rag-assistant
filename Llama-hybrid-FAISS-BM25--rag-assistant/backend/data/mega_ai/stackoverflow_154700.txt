You can write directly to the response stream: Response.Cache.SetCacheability(HttpCacheability.NoCache); XmlWriter XML_Writer = XmlWriter.Create(HttpContext.Current.Response.Output); To add settings to the writer you are better off using the newer XmlWriterSettings class. Give it as a parameter to the XmlWriter.Create function.
