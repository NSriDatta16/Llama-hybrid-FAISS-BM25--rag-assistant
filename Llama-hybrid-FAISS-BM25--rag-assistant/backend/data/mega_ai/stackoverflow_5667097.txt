If you just give them all the same name , the user-agent will automatically allow selection of only one radio button in the 'group'; selecting one will deselect any other that might have been selected: To do something on selection of a radio button you could hook into the change event: $('#select-container input[type="radio"]').change(function() { //check the id of this and trigger the appropriate action });
