^ip=(\d+(?:\.\d+){3})[\r\n]+userAgent=(.+)$ Apply in global + multiline mode. Group 1 will contain the IP, group 2 will contain the user agent string. Edit: The above expression can be simplified a bit, we can remove the IP address format checking - assuming that there will be nothing but real IP addresses in the log file: ^ip=(\d+\.?)+[\r\n]+userAgent=(.+)$
