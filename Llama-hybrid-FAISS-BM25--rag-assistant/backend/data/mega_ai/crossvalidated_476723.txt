As mentioned before NMS is used to remove false positives. Since you are using faster R-CNN, NMS is automatically set up at 0.7 in terms of the threshold. Therefore, You can use OpenCV function cv2.dnn.NMSBoxes(boxes, confidences, confid, thresh) You can dive into it deeper for any further information.
