Don't use execCommand, use range.insertNode instead. var img = document.createElement('img'); img.src = 'http://www.carsyouwilldrive.com/wp-content/uploads/2009/06/futurecar1.jpg'; var selection = document.defaultView.getSelection(); var range = selection.getRangeAt(0); range.insertNode(img); DEMO: http://jsbin.com/ogane4/5/edit
