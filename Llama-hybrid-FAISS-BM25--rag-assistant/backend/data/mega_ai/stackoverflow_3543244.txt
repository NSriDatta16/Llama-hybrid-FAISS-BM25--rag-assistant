It sounds like the "different profiles for different terminal environments" idea is probably your best bet. You can easily create a desktop shortcut for each profile. Another (less elegant) alternative would be to create a different "env" script for each directory, then source it (e.g. ". ./env") before you start working in that session. IMHO .. PSM
