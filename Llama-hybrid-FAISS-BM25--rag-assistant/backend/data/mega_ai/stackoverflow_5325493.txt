These two storage solutions are not quite comparable as they work differently: Although in both cases the data is store on the client side, a Cookie is sent along with every request while Web Storage in only for local use. But besides that: HTTP cookies were first drafted in 1994 ( IETF specification dated 1997); specification of Web Storage started in 2009 and is still in progress. Just think of how the Web evolved in those 15 years.
