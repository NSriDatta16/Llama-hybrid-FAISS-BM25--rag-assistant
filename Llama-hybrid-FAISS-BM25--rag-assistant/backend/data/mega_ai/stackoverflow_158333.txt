With PowerBuilder version 9 and up using the Oracle native driver and connecting to Oracle 9i and above databases, you can tell Oracle to maintain connections in a pool using the CnnPool='Yes' database parameter: Additional info from the PB 11.1 docs: http://infocenter.sybase.com/help/index.jsp?topic=/com.sybase.dc33820_1110/html/dbparm/BJEBJADI.htm
