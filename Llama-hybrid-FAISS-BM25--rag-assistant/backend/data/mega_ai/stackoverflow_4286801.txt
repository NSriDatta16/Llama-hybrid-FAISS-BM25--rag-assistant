Order of Maven+Surefire+Testng - BeforeClass, BeforeTest, Test, Test, AfterTest, AfterClass

I expected the following output: Running TestSuite [DEBUG] beforeClass [DEBUG] beforeTest [DEBUG] test [DEBUG] afterTest [DEBUG] beforeTest [DEBUG] test [DEBUG] afterTest [DEBUG] afterClass But instead, this actually happens. Notice 2 problems: BeforeClass runs after BeforeTest. Second, Before/AfterTest runs once .
