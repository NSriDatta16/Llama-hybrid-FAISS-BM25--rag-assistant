Export the table from phpmyadmin as CSV for excel - drag it into excel - add the desired column - Save - export the table again from phpmyadmin as .sql - copy the table creation(Not the data) - delete the table from phpmyadmin- Run the table creation query so you now have a blank table - Import the excel csv. IMPORTANT: backup your database
