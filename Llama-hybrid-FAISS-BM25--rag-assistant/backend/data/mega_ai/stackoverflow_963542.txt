It's not simply. Create field for uniquely key and this key saving onSave (or other event), but you must protect this number to be unique. You can create agent, which checking number on domino server and if agent find conflict then notify application administrator or other responsibility person to resolve this. Or each replica generate own number and after replicate on domino, agent assign number in right format.
