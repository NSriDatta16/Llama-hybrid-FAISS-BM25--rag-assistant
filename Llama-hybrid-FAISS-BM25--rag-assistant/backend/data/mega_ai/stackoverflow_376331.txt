It's not an XML document - there needs to be at least one and at most one root element. (the XML declaration is optional if the encoding is UTF-8 and the version is 1.0 - production 22 of the XML recommendation). It is a valid XML fragment - either as an empty text node, or an empty range. You'll have to define what 'piece of XML' means in your context.
