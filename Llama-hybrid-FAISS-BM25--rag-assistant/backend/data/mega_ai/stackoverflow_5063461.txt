Here's a method I found online that seems to work well for splitting long strings with : def split_str(str, len = 10) fragment = /.{#{len}}/ str.split(/(\s+)/).map! { |word| (/\s/ === word) ? word : word.gsub(fragment, '\0 ') }.join end
