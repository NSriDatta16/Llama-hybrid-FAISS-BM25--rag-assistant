A prior is a property of the data and not the algorithm used on the data. "Maximum conditional independence: if the hypothesis can be cast in a Bayesian framework, try to maximize conditional independence . This is the [ inductive ] bias used in the Naive Bayes classifier." - Wikipedia Inductive biases can be thought of as the assumptions about the data encoded in the learning algorithm.
