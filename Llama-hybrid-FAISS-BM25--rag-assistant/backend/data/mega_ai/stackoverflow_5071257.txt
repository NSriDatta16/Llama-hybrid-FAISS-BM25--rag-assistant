A paragraph will be in a new line, try to use Phrase like this: Phrase main1 = new Phrase("Form M.T.R. 17 \nPAY-BILL OF GAZETTED OFFICER"); main1.Font.SetStyle(Font.BOLD); Phrase main1a = new Paragraph(" DDO Code: 703"); pdfDocument.Add(main1); pdfDocument.Add(main1a);
