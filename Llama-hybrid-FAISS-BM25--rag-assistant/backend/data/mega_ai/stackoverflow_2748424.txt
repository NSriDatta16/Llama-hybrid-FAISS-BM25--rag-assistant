By default, HTTP protocol limits the user agent to 2 concurrent connections per HTTP/1.1 server.That is the limit you are hitting. Increase the limit by setting ServicePointManager.DefaultConnectionLimit. You can also set it per servicepoint, by setting ServicePointManager.GetServicePoint(url).ConnectionLimit
