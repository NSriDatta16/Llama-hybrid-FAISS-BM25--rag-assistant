lm implements OLS, which estimates the coefficient vector $\beta$ that minimizes the square error: $$ \mathcal L_\text{ols} = \left\|X\beta - y \right\|^2 $$ whereas the glm function with family=binomial estimates a logistic regression model. This model minimizes the binomial cross-entropy loss: $$ \mathcal L_\text{binomial} = \sum_i -y_i\log\left(\frac{1}{1 + \exp(-x_i\beta)}\right)-(1-y_i)\log\left(1-\frac{1}{1 + \exp(-x_i\beta)}\right) $$ There is no reason that the resulting $\beta$ values should be the same in general.
