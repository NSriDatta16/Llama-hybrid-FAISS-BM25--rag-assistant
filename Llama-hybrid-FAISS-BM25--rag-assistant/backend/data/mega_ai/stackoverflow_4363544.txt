Have you looked at Mechanize::FileSaver ? It looks like it can do what you require. Here is an example that saves all the PDF files it encounters: require 'rubygems' require 'mechanize' agent = Mechanize.new agent.pluggable_parser.pdf = Mechanize::FileSaver agent.get('http://example.com/foo.pdf')
