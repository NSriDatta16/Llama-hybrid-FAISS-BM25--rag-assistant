/^ *Fred *, *Jhon *, *Tree *,$/ Simple explanation: / */ is a regex that matches space zero or more times. Fred matches Fred, John John, etc pp use / +/ instead if you wish at least one space ^ matches the start and $ the end. note: using /\s*/ fragments instead of / */ fragments matches all whitespace - including tabs, newlines, etc
