The html helpers read the value from the ModelState. And there's no elegant way to override this behaviour. But if you add this line after SaveToDB(ID, SomeText) , it should work : ModelState["SomeText"].Value = new ValueProviderResult("", "", CultureInfo.CurrentCulture);
