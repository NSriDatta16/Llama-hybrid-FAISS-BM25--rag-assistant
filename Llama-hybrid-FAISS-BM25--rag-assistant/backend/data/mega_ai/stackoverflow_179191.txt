I suggest using XmlDocument.CreateDocumentFragment if you have the data in free form strings. You'll still have to use AppendChild to add the fragment to a node, but you have the freedom of building the XML in your StringBuilder. XmlDocument xdoc = new XmlDocument(); xdoc.LoadXml(@" "); XmlDocumentFragment xfrag = xdoc.CreateDocumentFragment(); xfrag.InnerXml = @" "; xdoc.DocumentElement.FirstChild.AppendChild(xfrag);
