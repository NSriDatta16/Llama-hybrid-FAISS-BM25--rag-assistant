I tend to do this to get GCC do the right thing with type-punning , which is explicitly allowed with unions: /*! Multi-family socket end-point address. */ typedef union address { struct sockaddr sa; struct sockaddr_in sa_in; struct sockaddr_in6 sa_in6; struct sockaddr_storage sa_stor; } address_t;
