How do I retrieve data from SQLite to VB6?

I am using SQLite3 ODBC Driver as my connection string, Dim conn As ADODB.Connection Set conn = New ADODB.Connection Dim rs As New ADODB.Recordset Set conn = New ADODB.Connection conn.ConnectionString = "DRIVER=SQLite3 ODBC Driver;Database=test.db;LongNames=0;Timeout=1000;NoTXN=0;SyncPragma=NORMAL;StepAPI=0;" conn.Open rs.Open "select * from Artists", conn, adOpenDynamic, adLockOptimistic MsgBox rs.Fields(0)
