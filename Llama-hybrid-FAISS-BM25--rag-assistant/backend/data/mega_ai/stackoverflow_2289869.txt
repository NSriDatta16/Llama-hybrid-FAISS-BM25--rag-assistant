To use the jQuery UI theme use this code: $('#'+gridName+' tr:nth-child(even)').removeClass("ui-priority-secondary"); $('#'+gridName+' tr:nth-child(odd)').addClass("ui-priority-secondary"); I use this code when I perform manual sorting (drag-n-drop)
