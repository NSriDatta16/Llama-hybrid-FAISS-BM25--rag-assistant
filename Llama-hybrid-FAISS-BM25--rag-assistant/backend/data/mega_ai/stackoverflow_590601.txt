Ask if they like server-round-trips. (To make them understand that client-side-stuff is necessary.) Ask if they want to write code that works in 3 different places at the same time? (To make them understand the need for a library that almost erases browser differences.) Show them a list of who's using jQuery . If they fancy elegant code, they will get what you're saying. Otherwise I think they'll be really hard to convince.
