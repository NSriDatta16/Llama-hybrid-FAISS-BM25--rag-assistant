One way to do this is to export the different layout settings into .vssettings files. You can then import the setting of your choice and get back the desired layout. To export the setting Tools -> Import / Export Settings Select "Export" and hit next Ensure "General Settings -> Window Layouts" is checked Complete the wizard and export the settings I'm not 100% sure this will work for multi-monitor settings. If it doesn't I would encourage you to file a bug on connect. http://connect.microsoft.com
