It seems like your score is simply number of correct out of 30 questions. Then I would model that using a logistic regression, which have parameters that are log odds directly. Your predictor variables would be a factor variable for person, a factor coding science/arts, and a factor for before/after, and interactions. Maybe the person factor should be a random effect (if using R you could use the package lme4). Then I would look out for possdible problems with overdispersion, which is always something to look out for with logistic regression.
