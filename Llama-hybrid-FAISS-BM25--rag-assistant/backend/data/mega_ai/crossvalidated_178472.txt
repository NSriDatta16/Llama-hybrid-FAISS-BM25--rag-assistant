preProcess does not return values, it simply sets up the whole imputation model based on the provided data. So, you need to run predict (requiring also the RANN package), but even if you do so with your artificial data you'll get an error: Error in FUN(newX[, i], ...) : cannot impute when all predictors are missing in the new data point as the imputation can not work in rows where both your predictors are NA's.
