The integration of MS SQL and Visual Studio is very good, better than any other database, so for that matter you are right to use MS SQL. However the solution to store binary files in a database is a subject which has advantages and disadvantages, you could read: http://www.childofthemachine.com/blog/2008/02/image-storage-database-blob-vs-file-system/ Or search the forums, there is a lot written about this subject. Also: SQL Server can also provide another option using FILESTREAM: http://technet.microsoft.com/en-us/library/bb933993.aspx
