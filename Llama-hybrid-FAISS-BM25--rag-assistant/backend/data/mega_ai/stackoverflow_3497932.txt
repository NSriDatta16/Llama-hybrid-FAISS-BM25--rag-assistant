you should set two more option to draggable to make it work element.draggable("option", "helper", 'clone'); element.draggable("option", "revert", 'invalid'); see documentation for connectToSortable and sortable draggable demo the complete function should look like this function onBtnClick2() { var element = $(" Hi there "); element.draggable({ connectToSortable: '.sortable', helper: 'clone', revert: 'invalid', appendTo: '#container' }); }
