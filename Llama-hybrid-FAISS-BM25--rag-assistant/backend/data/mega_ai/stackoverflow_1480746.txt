I guess if you really want to treat all these cases completely differently, your solution is okay, as it is very explicit. A compact solution might look more elegant, but will probably be harder to maintain. It really depends on what happens inside the if-blocks. As soon as there is a common handling of, say, the corners, one might prefer to catch those cases with one clever if-statement.
