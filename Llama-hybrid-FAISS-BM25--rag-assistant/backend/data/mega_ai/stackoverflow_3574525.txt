What you're trying to do in your method with the above code is to call GetType() on null , which is not possible. Why not use a generic list List in this case? You can elegantly skip the part of using GetType() to determine the type with this approach: static void func(List data) { if(data.Count > 0) data[0] = "Jimmy"; } static void Main(string[] args) { List lst = new List (1); string str = ""; lst.Add(str); func(lst); System.Console.WriteLine(lst[0]); //Prints out 'Jimmy' }
