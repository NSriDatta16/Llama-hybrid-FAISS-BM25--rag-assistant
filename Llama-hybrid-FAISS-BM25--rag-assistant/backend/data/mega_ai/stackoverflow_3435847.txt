You should probably look at moz-headless-screenshot You should be able to embed the functionality you need. http://blog.mozilla.com/ted/2010/07/29/moz-headless-screenshot/ he also provided a sample embedding client application called moz-headless-screenshot. This is a simple command line tool that takes a URL, image size, and output filename and generates a PNG screenshot of the webpage.
