VB6's UDTs align doubles at addresses that are multiples of 4. Here's how to contend with that in C: #pragma pack(push,4) typedef struct _UserDefinedType { signed int Integer; unsigned char Byte; float Float; double Double; } UserDefinedType; #pragma pack(pop)
