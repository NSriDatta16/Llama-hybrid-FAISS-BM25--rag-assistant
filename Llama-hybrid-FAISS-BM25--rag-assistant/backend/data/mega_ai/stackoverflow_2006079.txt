The auto-generated LINQ classes defined the NVarchar fields in the following way: [Column(Storage="_Text", DbType="NChar(20) NOT NULL", CanBeNull=false)] public string Text { get { return this._Text; } set { if ((this._Text != value)) { this.OnTextChanging(value); this.SendPropertyChanging(); this._Text = value; this.SendPropertyChanged("Text"); this.OnTextChanged(); } } } Where the Column settings should be: [Column(Storage="_Text", DbType="NVarChar(20) NOT NULL", CanBeNull=false)] The problem occurred because i updated the type in the SQL database without re-generating the LINQ classes.
