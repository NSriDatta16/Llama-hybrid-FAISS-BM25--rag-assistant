NSString has a number of methods that allow you to manipulate paths. It's not super elegant, but it's very useful. NSString *textFile = @"readme.txt" NSString *markdownFile = [[textFile stringByDeletingPathExtension] stringByAppendingPathExtension:@"markdown"]; // markdownFile is now "readme.markdown"
