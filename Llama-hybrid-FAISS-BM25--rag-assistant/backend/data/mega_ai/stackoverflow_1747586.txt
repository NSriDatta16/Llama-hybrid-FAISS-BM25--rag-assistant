The PostConstruct hook is right for that. Find more info on about PostConstruct here: in the javadoc lifecycle of EJBs in the JavaEE 5 tutorial Let's finish with a quick example: @Stateless public class TestEJB implements MyEJBInterface{ @PostConstruct public void doThatAfterInitialization() { //put your code here to be executed after creation and initialization of your bean } }
