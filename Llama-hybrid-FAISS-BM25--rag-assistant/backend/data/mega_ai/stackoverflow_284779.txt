I found a solution to my problem, however it may not be the best one. Using this : public var overImage:Boolean = false; public function checkAllow(evt:DragEvent):void { if(overImage == false) { evt.preventDefault() } } public function isOverImage():void { overImage = true; } public function isOutImage():void { overImage = false; } I call those functions like this : On my image component mouseOver="outerDocument.isOverImage()" mouseOut="outerDocument.isOutImage()" And for my tilelist I did this Tiles.addEventListener(DragEvent.DRAG_START, checkAllow); Hope it helps some people.
