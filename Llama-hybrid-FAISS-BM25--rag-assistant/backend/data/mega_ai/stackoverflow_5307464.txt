To display rating as you slide in your slider initialization you need to change the text of that div (Assuming you have a div (not an input box) with an id "ratingResult"). To update the value when user finishes dragging you need to add the change event to the initialization code. $("#ratingSlider").slider({ range: "min", value: 3, min: 1, max: 5, //this gets a live reading of the value and prints it on the page slide: function(event, ui) { $("#ratingResult").text(ui.value); }, //this updates the value of your hidden field when user stops dragging change: function(event, ui) { $('#rateToPost').attr('value', ui.value); } });
