One way of solving this problem is: var registeredRow = 0; function checkRow(){ var row = GetThreadTree().treeBoxObject.getFirstVisibleRow(); if (registeredRow != row) { registeredRow = row; updateCanvas(); } window.setTimeout(checkRow, 1); } And before checkRow is called: registeredRow = GetThreadTree().treeBoxObject.getFirstVisibleRow(); checkRow(); But it's not the most elegant solution but it works ;)
