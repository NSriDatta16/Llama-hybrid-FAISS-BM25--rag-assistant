I assume you mean a user-agent string in an HTTP request? This is just an HTTP header that gets sent along with your request. using Python's urllib2: import urllib2 url = 'http://foo.com/' # add a header to define a custon User-Agent headers = { 'User-Agent' : 'Mozilla/4.0 (compatible; MSIE 5.5; Windows NT)' } req = urllib2.Request(url, '', headers) response = urllib2.urlopen(req).read()
