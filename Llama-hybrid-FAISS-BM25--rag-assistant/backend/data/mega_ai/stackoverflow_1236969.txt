One way could be to strip html in three steps: Regex.Replace(htmlStr, " ]*>", "") // don't strip Regex.Replace(htmlStr, " ", "\r\n") // all paragraph ends are replaced w/ new line Regex.Replace(htmlStr, " ]*>", "") // replace remaining
