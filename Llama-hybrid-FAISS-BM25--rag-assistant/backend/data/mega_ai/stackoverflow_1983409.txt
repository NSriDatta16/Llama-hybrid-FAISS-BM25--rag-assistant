If your server is configured to support http:// file paths, you could use file_get_contents . If that doesn't work, the second simplest way is by using curl for which you will certainly find full-fledged download scripts. Some servers you pull images from may require a User-agent that shows that you are a regular browser. There is a ready-made class in the User Contributed Notes to curl that handles that, and provides a simple DownloadFile() function.
