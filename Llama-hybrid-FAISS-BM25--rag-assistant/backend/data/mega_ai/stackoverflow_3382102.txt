It selects all p elements that are directly after a h2 element in the DOM. Thus, the following p element would be styled: A heading This paragraph will be styled. But this wouldn't: A heading This paragraph will NOT be styled. And neither would this: This paragraph will NOT be styled. A heading ...or this: A heading This paragraph will NOT be styled.
