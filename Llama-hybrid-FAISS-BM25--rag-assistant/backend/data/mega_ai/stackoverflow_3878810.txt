You're not updating the pointer, you're just changing the local value of the argument. You need to use a pointer pointer (e.g. char** ) and change the pointed-to value instead. void compareStrings(char* firstFrag, char* secondFrag, int* longestCommonLength, char** suffix, char** prefix, int* relative_pos) { *suffix = firstFrag; *prefix = secondFrag; } Then you need to dereference it appropriately in the caller. Have fun!
