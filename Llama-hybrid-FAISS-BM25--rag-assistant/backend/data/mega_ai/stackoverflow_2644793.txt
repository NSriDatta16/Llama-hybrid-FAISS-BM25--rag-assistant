Font embedding is one of my favourite enemies of all time when creating flash. But I found a way to reduce the amount of holes in my walls (and bumps on my head). Embed the font into the Library Link the font symbol (in the Library) to a custom class extending the Font -class (lets call it MyFont for now) Create a TextFormat object (let's call it frmt ) Set frmt.font = new MyFont().fontName Create a TextField (let's call it tf ) Assign tf.embedFonts = true and tf.defaultTextFormat = frmt Hope this helps.
