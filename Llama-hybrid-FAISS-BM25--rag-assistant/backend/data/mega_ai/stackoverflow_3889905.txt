Not what you asked for, but I do this because of a focus bug in WebKit with spinboxes: // temporary fix for focus bug with webkit input type=number ui if (navigator.userAgent.indexOf("AppleWebKit") > -1 && navigator.userAgent.indexOf("Mobile") == -1) { var els = document.querySelectorAll("input[type=number]"); for (var el in els) el.type = "text"; } It might give you an idea to help with what you need.
