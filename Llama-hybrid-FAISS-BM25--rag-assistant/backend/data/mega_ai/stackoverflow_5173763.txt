Something like this will do what you want. Once the column is in date format you can represent it in any format you want. It's not pretty though. SET DATEFORMAT DMY SELECT CASE WHEN ISDATE(yourColumn) = 1 THEN CONVERT(DATETIME, yourColumn) ELSE CONVERT(DATETIME, SUBSTRING(yourColumn, CHARINDEX('/', yourColumn, 0)+1, CHARINDEX('/', yourColumn, CHARINDEX('/', yourColumn, 0)+1) - CHARINDEX('/', yourColumn, 0)) + STUFF(yourColumn, CHARINDEX('/', yourColumn, 0), CHARINDEX('/', yourColumn, CHARINDEX('/', yourColumn, 0)+1) - CHARINDEX('/', yourColumn, 0), '') ) END FROM yourTable This is highly fragile code though. If you can not be sure that a date is in dmy or mdy format, it is very likely you will produce incorrect results.
