[Flags] public enum MyColor : byte // sets the underlying type. { Red = 0; Green = 1; Blue = 2; White = 4; Magenta = 8; ... (etc) } It's not possible to directly set the endianness. You can use some well-crafted numbers that simulate big-endian bytes on a little endian system. However, I'd always use explicit APIs for converting byte orders.
