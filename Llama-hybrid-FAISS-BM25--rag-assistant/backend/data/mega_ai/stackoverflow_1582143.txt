In cases like this I usually find myself using an init function that does the binding/unbinding, then call that on ajaxComplete or the like. In this case it would probably look like this: function init() { $('.myfav').unbind('click'); $('.myfav').autocomplete('disable'); $('.myfav').datepicker('disable'); $('.myfav').autocomplete(); $('.myfav').datepicker(); $('.myfav').click(somefunction); } Of course, depending on the plugin the unbind might look different, I belive jQueryUI uses $('.myfav').draggable('destroy') for instance.
