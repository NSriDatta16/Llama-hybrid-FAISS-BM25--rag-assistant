The easiest solution in your case - is using System.Windows.Forms.Timer class, but in general case you may use following solution to access you GUI-stuff from non-GUI thread (this solution applicable for .net 2.0 but it more elegant for .net 3.5): public static class ControlExtentions { public static void InvokeIfNeeded(this Control control, Action doit) { if (control.InvokeRequired) control.Invoke(doit); else doit(); } } And you may use it like this no mater from what thread, from UI or from another one: this.InvokeIfNeeded(()=> { listBox1.Items.Add(text); listBox1.Update(); });
