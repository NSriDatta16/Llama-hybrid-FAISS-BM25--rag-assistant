A priority_queue is the closest thing in C++ with STL. You could wrap it in another class to create your own implementation that trims the size automatically. Language-agnostically (although maybe not memory-fragmentation-safely): Insert data Sort Delete everything after the nth element std::priority_queue does step 2 for you.
