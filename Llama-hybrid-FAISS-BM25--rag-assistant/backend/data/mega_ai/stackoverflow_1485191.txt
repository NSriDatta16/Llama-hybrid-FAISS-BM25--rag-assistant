Check out lxml , a really nice python library for dealing with xml. You can use drop_tag to accomplish what you are looking for. from lxml import html h = html.fragment_fromstring(' Hello World! ') h.find('*').drop_tag() print(html.tostring(h, encoding=unicode)) Hello World!
