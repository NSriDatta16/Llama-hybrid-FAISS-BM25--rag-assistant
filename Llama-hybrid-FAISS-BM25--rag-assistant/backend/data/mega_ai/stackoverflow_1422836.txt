Thanks for the suggestion about querystrings - This prompted me to go for the following solution: jQuery(document).ready(function($) { var tabshown = location.hash; var index; switch(tabshown) { case "#panel1": index = 0 break; case "#panel2": index = 1 break; default: index = 4 } $("#mastheadhome").tabs({ selected:index, fx: {opacity: 'toggle'} }); }); It's not hugely elegant, but hey, it works! If anyone can think of a better way, I'd love to hear about it!
