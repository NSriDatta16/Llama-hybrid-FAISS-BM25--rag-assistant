We also have a database system that is used exclusively within the organisation and use Window NT logins. This function returns the current users login name: CREATE FUNCTION dbo.UserName() RETURNS varchar(50) AS BEGIN RETURN (SELECT nt_username FROM master.dbo.sysprocesses WHERE spid = @@SPID) END You can use this function in your triggers.
