The SQL standard does not have a BOOLEAN datatype and Oracle does not support one. That is why VSIZE() hurls the exception. In PL/SQL boolean is implemented as an enumeration, which is interesting because PL/SQL doesn't support enumerations. However, ADA - the language which forms the basis for PL/SQL - does. Pete Finnegan wrote more about this; check it out .
