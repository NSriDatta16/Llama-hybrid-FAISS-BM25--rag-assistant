If you won't store times as a string but as a timestamp (in seconds) it would be easy to just select the data: Select StartDateTime,Activity from AgentLog where Activity='LogOn' AND time>timeofday ORDER by time LIMIT 1 Where timeofday is the timestamp of the current day at midnight
