The browser tells the server what kind of browser it is in the User-Agent string, which it includes with each HTTP request. You can access the User-Agent directly and parse it yourself, or you can use ASP.NET's built-in browser capabilities feature, which relies on several *.browser files, regular expressions, etc. User-Agent: ID: Browser: Type:
