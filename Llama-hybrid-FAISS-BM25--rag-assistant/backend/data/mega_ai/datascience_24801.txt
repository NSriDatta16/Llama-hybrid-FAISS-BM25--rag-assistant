About your first question, you are looking for Boolean indexing in Python Pandas. build_times = df[['Date','site','project','Duration','Status']] build_times.loc[build_times['Status'] == 1] About the second question, you can filter again the dataframe and draw multiple lines on the same graph like this: import seaborn as sns import matplotlib.pyplot as plt sns.kdeplot(build_group.vals[(build_group.site == "A") && (build_group.project == "example")],label='Project example on site A'); sns.kdeplot(build_group.vals[(build_group.site == "B") && (build_group.project == "example")],label='Project example on site B'); plt.xlabel('Date') plt.ylabel('Average build time') plt.show()
