For sharing complex persistent user-defined objects, the following approaches are recommended: Application Preferences Files contentProviders SQLite DB If the shared data needs to be retained across points where the application process can be killed, then place that data in persistent storage like Application Preferences, SQLite DB, Files or ContentProviders. Please refer to the Data Storage for further details on how to use these components. You need to go with SQLite database.
