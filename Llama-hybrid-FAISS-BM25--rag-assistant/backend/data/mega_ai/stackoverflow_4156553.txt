One possibility is that you can use regex. import re robj = re.compile(r'^(.*) [\"\'](.*)[\"\'].*[\"\'](.*)[\"\']') str1 = "Who will win the game 'Michigan' 'Ohio State'" r1 = robj.match(str1) print r1.groups() str2 = 'What color is the sky today? "blue" or "grey"' r2 = robj.match(str2) r2.groups() Output: ('Who will win the game', 'Michigan', 'Ohio State') ('What color is the sky today?', 'blue', 'grey')
