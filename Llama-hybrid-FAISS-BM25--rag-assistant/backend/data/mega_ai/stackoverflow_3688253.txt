What I've done in the past (using straight jQuery UI, without zend) is to set up an onclick event that calls a function to open the dialog: clickable item Initialize the dialog in the document ready: var $demoDialog; $(document).ready(function() { $demoDialog= $("#demoDialog").dialog({ autoOpen: false, draggable: true, resizable: true title: 'Demo Dialog' }); }); And add the function to open it: function showDialog() { if (!$demoDialog.dialog('isOpen')) { $demoDialog.dialog('open'); } }
