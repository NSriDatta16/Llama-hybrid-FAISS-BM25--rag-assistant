I geuss you missed part of how jQuery works, $('p') returns all paragraphs and the html( val ) function: Set the html contents of every matched element. This property is not available on XML documents (although it will work for XHTML documents). http://docs.jquery.com/Attributes/html#val So if you just want to set the contents for the first p use $("P").eq(0).html( 'something' ); or to get the html: $("P").eq(0).html(); http://docs.jquery.com/Core/eq#position more on jQuery selectors here: http://docs.jquery.com/Selectors
