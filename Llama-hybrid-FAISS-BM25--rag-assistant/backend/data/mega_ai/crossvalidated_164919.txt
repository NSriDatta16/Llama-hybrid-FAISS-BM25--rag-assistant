Is there any way to increase the sample size with bootstrapping? No. The bootstrap is basically a technique to approximate standard errors of complex estimators. It is not a data generating technique. Are there any methods for bootstrapping multivariate data? Yes. Usually, the data lines are resampled together as a whole (see code below). This strategy may fail e.g. if the order of the lines is relevant to the problem (like in time series analysis). How to e.g. bootstrap lines of a multivariate data set $X$: getBootSample
