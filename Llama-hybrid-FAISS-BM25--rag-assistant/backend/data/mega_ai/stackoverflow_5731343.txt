Thanks to @BertNase for suggestion. The following import command has fixed the issue: keytool -import -keystore myStore.jks -alias aliasA -file aaa.co.uk.cer keytool -import -keystore myStore.jks -alias aliasB -file bbb.co.uk.cer Now, this command imports both certificates in the same keystore .
