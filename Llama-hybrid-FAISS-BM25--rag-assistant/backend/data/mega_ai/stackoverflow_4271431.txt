Your curl_close calls are broken. Pass in the curl resource. E.g.: curl_close($ch); This is why you should enable and pay attention to warnings. php > curl_close(); PHP Warning: curl_close() expects exactly 1 parameter, 0 given in php shell code on line 1 This in turn means your cookie jar won't work properly.
