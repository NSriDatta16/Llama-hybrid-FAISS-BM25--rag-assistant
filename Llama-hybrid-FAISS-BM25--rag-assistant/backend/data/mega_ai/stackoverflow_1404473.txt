It looks like you are probably running into this bit of the standard: In C, a const-qualified object at file scope without an explicit storage class specifier has external linkage. In C++, it has internal linkage. Make this change to 2.cpp: #include extern const std::string QWERTY("qwerty"); There is some more detail on what "linkage" means in this question - What is external linkage and internal linkage in C++ .
