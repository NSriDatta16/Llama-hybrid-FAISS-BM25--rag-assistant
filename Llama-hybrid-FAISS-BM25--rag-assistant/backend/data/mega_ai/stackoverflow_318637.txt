I figured out that changing the value of EstimatedSize under HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\{my-guid-value} does not have any direct effect. This value is cached in the following key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\App Management\ARPCache\{my-guid-value}\SlowInfoCache Only after I remove (rename) this SlowInfoCache value, the updated size appears under Add or remove programs .
