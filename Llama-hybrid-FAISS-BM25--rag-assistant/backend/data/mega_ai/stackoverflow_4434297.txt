You can't have multiple tabs for the same file without using split mode (vertical or horizontal). If you want a copy of the file, use Refactor | Copy. To quickly split the view and make the same file visible in 2 splits hold Ctrl and drag the tab into the editor area. UPDATE : feature request added, please vote .
