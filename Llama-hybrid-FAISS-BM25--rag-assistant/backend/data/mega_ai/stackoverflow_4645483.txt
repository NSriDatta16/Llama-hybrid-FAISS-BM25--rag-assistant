As of JDK 7.0, the JNLP API is being provided by the javaws.jar file in your JRE's lib directory, i.e., ${java.home}/lib/javaws.jar . It is possible to use the maven dependency scope system . ... javax.jnlp jnlp-api 7.0 system ${java.home}/lib/javaws.jar ...
