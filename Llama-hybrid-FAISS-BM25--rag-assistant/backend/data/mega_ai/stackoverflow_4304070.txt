Add these headers: header("Pragma: public"); //This one may work by itself. header("Expires: 0"); header("Cache-Control: must-revalidate, post-check=0, pre-check=0"); header("200 HTTP/1.0 OK"); //HTTP 1.0 Compatible This will force Internet Explorer to download the file from the server.
