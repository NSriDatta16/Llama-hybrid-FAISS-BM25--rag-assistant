That is commonly called "support", the is the number of occurrences of the class in the specified dataset. There are several ways to use support: Filter data - Only allow instances with a specific number of occurrences. Weighted loss - Many loss functions give more weight to examples that occur more often. Also, it is often possible to increase the weightings. Bayesian frameworks - Bayesian machine learning gives explicit uncertainty predictions, support is one of the primary drivers of uncertainty.
