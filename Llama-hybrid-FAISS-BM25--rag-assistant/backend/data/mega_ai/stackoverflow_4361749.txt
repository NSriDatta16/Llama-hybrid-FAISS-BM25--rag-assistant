As far as I am aware there are two main ways that this effect is achieved: Using the anchor portion of the url (#gallery) Using a hidden iframe There are pre-built solutions that you can use to leverage this kind of functionality without having to deal with writing the code. For example if you are working with asp.net then you can use the Ajax History Control: http://www.asp.net/ajax/videos/how-do-i-use-the-aspnet-ajax-history-control
