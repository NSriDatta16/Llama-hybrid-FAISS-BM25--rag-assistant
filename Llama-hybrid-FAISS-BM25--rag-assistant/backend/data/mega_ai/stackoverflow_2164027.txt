What you need is a recursive function that traverse OrganizationalUnit relation tree and gets number of related Checkouts for each OrganizationalUnit. So your code will look like this: def count_checkouts(ou): checkout_count = ou.checkouts.count() for kid in ou.children.all(): checkout_count += count_checkouts(kid) return checkout_count Also note, that to get a number of related checkouts I use: checkout_count = ou.checkouts.count() insted of: count = ou.checkouts.all().count() My variant is more efficient (see http://docs.djangoproject.com/en/1.1/ref/models/querysets/#count ).
