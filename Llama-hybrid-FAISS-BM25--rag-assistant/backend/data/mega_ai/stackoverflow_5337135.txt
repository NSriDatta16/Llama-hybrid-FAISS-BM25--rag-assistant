Here is an example, albeit inelegant. This prints out 999 with 2 leading zeros. You'll need to duplicate the code for more digits. #Push number to print on stack 999 # macro to print a zero [[0]P]sa # Print a zero if less than 5 digits dZ5>a # Print a zero if less than 4 digits dZ4>a # Print a zero if less than 3 digits dZ3>a # Print a zero if less than 2 digits dZ2>a # Print out number p
