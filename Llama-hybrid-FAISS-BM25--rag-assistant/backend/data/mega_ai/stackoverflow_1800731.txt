Elegance To put it simply it just looks nicer AND is easier for a programmer to parse, especially for more complicated data structures: var a = [ ['a list', 'of', 'strings'], [1, 2, 3, 4], { hello: "This is an object property." } ]; Versus the clumsier OOP-oriented method: var a = new Array(); var a2 = new Array(); a2.push('a list'); a2.push('of'); a2.push('strings'); var a3 = new Array(); a3.push(1); a3.push(2); a3.push(3); a3.push(4); var o = new Object(); o.hello = "This is an object property"; a.push(a2, a3, o);
