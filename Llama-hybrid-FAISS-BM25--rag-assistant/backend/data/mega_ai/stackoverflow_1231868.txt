How to activate JMX on remote Glassfish server for access with jconsole?

I would like to monitor remote glassfish server. I have enabled JMX Connection in domain.xml: But this didn't help. I still can't connect to server with JConsole. Then I've found solution - I need to specify JVM properties in domain.xml to open 8686 port for remote connection. So I added this lines into java-config section: -Dcom.sun.management.jmxremote -Dcom.sun.management.jmxremote.port=8686 -Dcom.sun.management.jmxremote.local.only=false
