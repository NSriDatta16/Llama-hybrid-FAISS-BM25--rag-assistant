Yes. A Web framework should be used for all applications which are more complex than "Hello, World!". There's no need to reinvent the wheel. Web frameworks contain many useful tested components which you will almost certainly need on your site. They also help organize your code, often into an MVC or similar paradigm, which will make your code more maintainable. Some Java Web Frameworks (in no particular order): JavaServer Faces Grails Struts Tapestry Spring MVC Apache Wicket Play Framework Stripes
