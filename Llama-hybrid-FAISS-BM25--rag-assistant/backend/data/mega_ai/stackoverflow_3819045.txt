You strongly-typed your view to a Product type but your EF query returns an anonymous type that includes other fields. Create a class for the product view model: class ProductViewModel { public string Name { get; set; } public string Description{ get; set; } public string PCategory{ get; set; } public string ProductPicture{ get; set; } public string Price { get; set; } public string Picture { get; set; } } Then in the select of your EF query, populate this type rather than your anonymous type. Also, you should independently verify that your EF query is returning the expected results.
