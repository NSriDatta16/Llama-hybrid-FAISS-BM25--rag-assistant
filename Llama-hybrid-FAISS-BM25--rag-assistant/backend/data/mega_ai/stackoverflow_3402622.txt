The limit on a UDP datagram payload in IPv4 is 65535-28=65507 bytes, and the practical limit is the MTU of the path which is more like 1460 bytes if you're lucky. When UDP is fragmented, it loses the datagram if a fragment is lost, because there is no retransmission. Use TCP.
