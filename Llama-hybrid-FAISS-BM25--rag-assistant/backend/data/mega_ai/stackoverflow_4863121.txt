You can loop through the textboxes in your form in a fairly simple manner: Func > SearchTextBoxes = null; SearchTextBoxes = coll => { List textBoxes = new List (); foreach (Control c in coll) { TextBox box = c as TextBox; if (box != null) textBoxes.Add(box); if (c.Controls.Count > 0) textBoxes.AddRange(SearchTextBoxes(c.Controls)); } return textBoxes; }; var tbs = SearchTextBoxes(this.Controls).OrderBy(tb => tb.Name); Edit: Changed according to new requirements. Not nearly as elegant as the LINQ-solution, of course :)
