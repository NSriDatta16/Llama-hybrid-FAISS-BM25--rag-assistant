Staying in the realm of the bloomin' obvious, the Oracle documentation set is always the first - and best - place to start. Check it out. Edit The SS64 site is clear but very out-of-date. Its latest syntax is the 9i set, two major releases off the pace. It is also not complete. For instance, the SELECT page doesn't mention the WITH clause, which was a very useful feature introduced in 9i. If you want a redaction of the more recent documentation then Dan Morgan's Library site is exhaustive (and occasionally exhausting). Here is his page on Oracle datatypes .
