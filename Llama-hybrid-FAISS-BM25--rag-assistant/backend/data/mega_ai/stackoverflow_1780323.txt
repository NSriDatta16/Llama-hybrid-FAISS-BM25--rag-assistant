Eclipse does have a setting which (should) pick this up, see e.g. the desciption in the Eclipse 3.1 release notes : Preferences → Java → Compiler → Errors/Warnings → Deprecated And Restricted API → Forbidden/Discouraged Reference (and the project properties page to allow project-specific overrides of this setting)
