One thing you could do is, define a function that will grasp all events assigned to each type of elements and show them somewhere on the page(preferably at the bottom). To trigger that, function you may add a button which should be clicked. Of course, all this must be done in debugging state and you would certainly want to remove the button and the display panel when releasing the App. We used to do this for our project, may not be elegant but it does serve the purpose.
