if you just want the default CFRetain / CFRelease behaviour, this should work: void MONDictionaryReleaseCallback(CFAllocatorRef allocator, const void* value) { #pragma unused(allocator) assert(value); if (0 != value) { CFRelease(value); } } the retain callback should be easy to implement from there.
