Try oobash. It is an OO-style string library for Bash 4. It has support for German umlauts. It is written in Bash. Many functions are available: base64Decode base64Encode capitalize center charAt concat contains count endsWith equals equalsIgnoreCase reverse hashCode indexOf isAlnum isAlpha isAscii isDigit isEmpty isHexDigit isLowerCase isSpace isPrintable isUpperCase isVisible lastIndexOf length matches replaceAll replaceFirst startsWith substring swapCase toLowerCase toString toUpperCase trim zfill Look at the contains example: [Desktop]$ String a testXccc [Desktop]$ a.contains tX true [Desktop]$ a.contains XtX false oobash is available at Sourceforge.net .
