Try this path /Users/ /Library/Application Support/iPhone Simulator/ . In available dirs there look for /Applications/ /Documents . UPDATE (27th Jan, 2013): If you're OK with Terminal, install awesome ruby gem by Paul Samuels : http://paul-samuels.com/blog/2012/08/19/ios-simulator-directory-command-line-tool/ . If you're not OK with Terminal, Paul has also created an excellent workflow for Alfred 2.app: http://paul-samuels.com/blog/2013/01/24/sidir-alfred-workflow/ . If you're having hard time installing Paul's gem, follow this tutorial: http://pragmaticstudio.com/blog/2010/9/23/install-rails-ruby-mac .
