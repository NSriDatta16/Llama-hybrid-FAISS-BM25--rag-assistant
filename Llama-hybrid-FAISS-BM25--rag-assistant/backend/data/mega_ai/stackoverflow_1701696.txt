Java applets can do it, but they're a bit legacy nowadays. In Java terms, you nowadays also have the choice of Java Web Start (the better Google keyword is JNLP) and JavaFX . If you don't shy away from learning a new language and wanted to have a real RIA , then go for JavaFX. Otherwise, Java Web Start A.K.A. JNLP is a fine choice. With Java you can just do everything as if it is a client application, as long as it's all within the borders of the security restrictions. Here are some tutorials of use: JavaFX tutorial and Web Start tutorial .
