If you had markup like this: Toggle Div 5 Content for Div 5 You can do this in jQuery: $("a.toggler").click(function() { $(this.hash).slideToggle(); }); Or use rel or something with a div you click on, like this: Toggle Div 5 Content for Div 5 And adjust your jQuery to this: $("div.toggler").click(function() { var hash = $(this).attr("rel"); $(hash).slideToggle(); window.location.hash = hash; }); My recommendation would be to use display:block; on an anchor to do what you want, and leverage the default browser behavior here, it takes care of the hash on click. No matter which approach above, you can show the one on page load like this: $(function() { if(location.hash != "") { $(location.hash + ":hidden").slideDown(); } });
