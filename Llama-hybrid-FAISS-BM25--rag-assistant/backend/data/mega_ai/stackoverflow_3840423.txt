In SQL Server Management Studio, expand the SQL Server Agent, right click on "Jobs" and select "New Job..." in "Steps", create a "New..." one and enter this: DELETE YourTable WHERE YourDate or without regards to time use: DELETE YourTable WHERE YourDate in "Schedule", you can make it run every Sunday or whatever you need.
