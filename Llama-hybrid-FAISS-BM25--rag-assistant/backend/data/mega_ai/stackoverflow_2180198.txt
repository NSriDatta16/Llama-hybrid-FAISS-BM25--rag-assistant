Yes, that folder has the "read only" attribute set. This would work: var dir = new DirectoryInfo(@"c:\temp\test"); dir.Attributes = dir.Attributes & ~FileAttributes.ReadOnly; dir.Delete(); You should always pay attention to the file attributes when you delete stuff. Be sure to stay clear from anything that is System or ReparsePoint. And careful with ReadOnly and Hidden.
