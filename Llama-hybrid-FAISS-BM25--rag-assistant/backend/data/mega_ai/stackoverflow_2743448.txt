Here's the wiki page on embedding cython Assuming you installed python to C:\Python31 and you want to use Microsoft Compiler. smalltest1.py - is the file you want to compile. test.exe - name of the executable. You need to set the environmental variables for cl . C:\Python31\python.exe C:\Python31\Scripts\cython.py smalltest1.py --embed cl.exe /nologo /Ox /MD /W3 /GS- /DNDEBUG -Ic:\Python31\include -Ic:\Python31\PC /Tcsmalltest1.c /link /OUT:"test.exe" /SUBSYSTEM:CONSOLE /MACHINE:X86 /LIBPATH:c:\Python31\libs /LIBPATH:c:\Python31\PCbuild
