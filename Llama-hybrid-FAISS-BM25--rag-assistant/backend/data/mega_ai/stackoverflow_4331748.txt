Consider org.jfree.data.time.DynamicTimeSeriesCollection , which "adds the ability to append new data and discard the oldest." A javax.swing.Timer is convenient for the periodic updates, as it's action event handler executes on the event-dispatching thread. Addendum: As it's a time series, the domain axis can be formatted to show the age. An XYItemLabelGenerator is an alternative; here is an example using NumberFormat . The number of items in the queue can be displayed in the corresponding series' legend.
