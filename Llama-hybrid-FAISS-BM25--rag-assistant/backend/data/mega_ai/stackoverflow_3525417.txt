Going with rake or a gem is fine. If you want to centralize your rake tasks (assuming you are on OSX or Some Linux/*nix variant) you can create them in your home directory: ~/.rake/*.rake Rake will look there for tasks to run if in a directory w/ no Rakefile. Also, consider (again, if you are on some sort of *nix platform) just creating shell aliases to your commands.
