In regards to the number of rows Some storage engines, such as MyISAM, store the exact count. For other storage engines, such as InnoDB, this value is an approximation, and may vary from the actual value by as much as 40 to 50%. Taken from MySQL :: MySQL 5.1 Reference Manual - SHOW TABLE STATUS Syntax
