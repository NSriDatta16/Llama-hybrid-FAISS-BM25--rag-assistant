You could build a valid Xml-Document in memory containing your xml fragment. Then read this document with the XmlReader (e.g. by using MemoryStream ) and let the XmlWriter write indented xml. A faster approach wold be indenting the xml yourself by manipulating the string. Search for , increase the nesting level and add the indention spaces. If you find or a self closing tag decrease the nesting level and append a \n I don't think there is a fast and nice solution to your problem, but i might be wrong...
