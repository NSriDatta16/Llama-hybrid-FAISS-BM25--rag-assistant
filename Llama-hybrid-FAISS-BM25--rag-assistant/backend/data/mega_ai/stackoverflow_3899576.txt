You have to subclass UIScrollView to allow interception of the touch events. For example, you could create a "special" UISCrollView like this: @interface TouchableScrollView : UIScrollView { } @end .m file: @implementation TouchableScrollView - (void) touchesEnded: (NSSet *) touches withEvent: (UIEvent *) event { if (!self.dragging) { [self.nextResponder touchesEnded: touches withEvent:event]; } [super touchesEnded: touches withEvent: event]; } @end This example only overrides touchesEnded, but you can do the same for touchesBegan
