You would have to create a wrapper around pthread_create and pass in thread specific storage to pthread_create that contains an array of arguments, for example. The wrapper would use form such as: void *mythreadfunction(void *arg) {...} pthread_create_wrapper(context ctx, ...) { Array *arr; pthread_t mythread; arr = new Array(); // push arguments to array here blah blah ... // create thread and pass in argument list as thread data pointer. pthread_create(&mythread, NULL, mythreadfunction, (void *)arr); }
