Make two tables: DAILY_STOCK_DATA DATETIME Date CHAR(5) Symbol #or whatever the max symbol length can be INTEGER Open INTEGER High INTEGER Low INTEGER Close INTEGER Volume INTEGER Dividend Presuming that weekly data is averaged over 7 days? : WEEKLY_STOCK_DATA DATETIME Date #date when the week started CHAR(5) Symbol #or whatever the max symbol length can be INTEGER AvgOpen INTEGER AvgHigh INTEGER AvgLow INTEGER AvgClose INTEGER AvgVolume INTEGER AvgDividend Note: All the Open, Close etc. are integers. That's because they are meant to store the values in the lowest currency coinage such as cents, instead of storing float data.
