If you mean Control.Visible property, not attribute as you mentioned, then yes, it won't be rendered. on the link above: Control.Visible Property gets or sets a value that indicates whether a server control is rendered as UI on the page. If this property is false, the server control is not rendered. You should take this into account when organizing the layout of your page.
