When I've done this, I've typically embedded the CSS in the body of the email: span.orange { color: #FF6D06; ... } ... Avoid referencing external entities (like stylesheets, images, etc.) in HTML email. Those references may not be available, depending on the user's mail agent settings. Also, learn about the HTML support in mail agents. It may not be as rich as you expect. Here's some information on Outlook , for instance.
