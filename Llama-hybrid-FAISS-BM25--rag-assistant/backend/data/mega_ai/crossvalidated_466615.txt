You can probably fit this with a model that looks like this: $$E[y \vert x]= \alpha+\beta \cdot I(x>0) + \gamma \cdot x \cdot I(x>0)+ \delta \cdot x^2 \cdot I(x>0), $$ where $I()=1$ if $x>0$ and $0$ otherwise. There is probably a more elegant way to do this in R, but this should do it: library(dplyr) library(ggplot2) df % mutate(y = exp(x / 1000) + rnorm(100, 0, 1)) %>% mutate(y = ifelse(x == 0, runif(100, 0, 10), y)) df $pos_x x >0 , 1, 0) ols % ggplot() + geom_point(aes(x, y)) + geom_point(aes(x,predict(ols), color='red')) The predictions from the model are shown in red:
