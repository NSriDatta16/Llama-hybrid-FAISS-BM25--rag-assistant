I think Redmine (opensource, GPL) is a nice alternative. Below are some of the main features of Redmine (from the site). Multiple projects support Flexible role based access control Flexible issue tracking system Gantt chart and calendar News, documents & files management Feeds & email notifications Per project wiki Per project forums Time tracking Custom fields for issues, time-entries, projects and users SCM integration (SVN, CVS, Git, Mercurial, Bazaar and Darcs) Issue creation via email Multiple LDAP authentication support User self-registration support Multilanguage support Multiple databases support I'd also note: There are many plugins for Redmine on internet Issue reports generation in PDF Intuitive interface
