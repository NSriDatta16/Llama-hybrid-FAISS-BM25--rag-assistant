When we use BiLSTM or LSTM or any RNN as an encoder-decoder model then the input layer in the encoder model should have input_shape = (len(vocabulary),) and the output layer of the decoder model should have the same number of neurons i.e. model.add(Dense(len(vocab)), activation = 'x')
