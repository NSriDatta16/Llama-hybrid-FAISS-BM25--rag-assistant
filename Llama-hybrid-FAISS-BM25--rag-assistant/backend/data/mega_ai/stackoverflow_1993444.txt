I haven't kept track of this research area for a while, but I can point you towards the CiteSeerX search engine if you don't know it already. ( http://citeseerx.ist.psu.edu/ ) Searching for papers which cite Shenoy & Shafer's An axiomatic framework for Bayesian and belief function propagation (1990) will give you a list of other researchers who have tried to apply the algorithm.
