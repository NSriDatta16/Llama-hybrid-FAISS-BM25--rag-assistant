Or you could just go: SELECT AVG(grade_points) FROM grades JOIN articles_grades ON grades.id = articles_grades.grade_id WHERE articles_grades.users_articles_id = '$page' One call to mysql_fetch_array will return the only row returned by this statement, an array with one element, there's your average.
