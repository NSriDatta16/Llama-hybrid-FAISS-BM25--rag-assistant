C:\Temp is NOT a good choice. If you are using .Net use code like this: string baseFolder = System.Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData); string cfgFolder = Path.Combine(baseFolder, "MyAppName"); try { if (!Directory.Exists(cfgFolder)) { Directory.CreateDirectory(cfgFolder); } } catch { } // If no access, not much we can do. to get a place for medium-term storage of app data, or Path.GetTempPath() for transient storage of data.
