Although the W3C XSLT Spec supports embedding an XSLT stylesheet into an XML document, it seems that IE and Firefox do not support this. UPDATE : As per the comment by Robert Niestroj, years later, in Oct. 2014, this works in FireFox 33. However, there is a good alternative: embed the XML document into the XSLT stylesheet . Below is an example. An XSLT stylesheet containing an embedded XML document : Hello I am foo When tis file is opened in IE, the wanted result is displayed by the browser: Hello I am foo Do note , that it is necessary to include templates that ignore most of the XSLT instructions (in this case we are ignoring any by simply having no template body.
