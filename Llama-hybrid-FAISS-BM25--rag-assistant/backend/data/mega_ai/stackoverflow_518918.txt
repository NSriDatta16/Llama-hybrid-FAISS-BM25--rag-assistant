Since HTTP is stateless, your class' static variable ($_auth) won't 'survive' between pageloads, so if you're trying to make the variable stick, you'll need to store it as a Session variable . However, I would strongly encourage you to not write your own auth class unless you are really serious about it. There are dozens of excellent PHP auth scripts out there to pick from, that have already addressed all the intricacies of web authentication .
