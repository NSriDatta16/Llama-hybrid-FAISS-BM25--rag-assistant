If this is secure data, you should be serving it up HTTPS which will not cache it anyway. You could also try response.Headers.Add("pragma", "no-cache"); response.Headers.Add("Cache-control", "no-cache, no-store, must-revalidate"); response.Headers.Add("Expires", "01 Apr 1995 01:10:10 GMT"); to get the page to expire immediately.
