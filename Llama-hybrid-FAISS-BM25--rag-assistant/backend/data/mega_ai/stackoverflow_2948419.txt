You can do this with setTimeout . (Complete example below.) In jQuery's ready handler, set a function to be called in six seconds via setTimeout , and in your xforms ready handler, cancel that via clearTimeout if it hasn't happened yet. Edit Complete example (rather than my earlier fragmented code snippets), assumes it's okay if your xforms ready handler is within your jQuery ready handler: jQuery.ready(function() { var xformsReadyTimer; xformsReadyTimer = setTimeout(function() { // Too long, show the warning xformsReadyTimer = undefined; alert("XForms is taking too long!"); }, 6000); new EventListener(document.documentElement, "xforms-ready", "default", function() { if (xformsReadyTimer) { // Cancel the warning clearTimeout(xformsReadyTimer); xformsReadyTimer = undefined; } } ); }); (You might consider making those named functions rather than anonymous ones , but I've used anonymous ones above for simplicity.)
