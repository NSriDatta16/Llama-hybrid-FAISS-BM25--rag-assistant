this article solved the issue for me :) http://techliberty.blogspot.com/2009/12/dealing-with-rake-aborted-uninitialized.html Prior to Leopard sudo gem install mysql -- --with-mysql-dir=/usr/local/mysql Leopard on a PPC machine: sudo env ARCHFLAGS="-arch ppc" gem install mysql -- --with-mysql-config=/usr/local/mysql/bin/mysql_config Leopard on an Intel machine: sudo env ARCHFLAGS="-arch i386" gem install mysql -- --with-mysql-config=/usr/local/mysql/bin/mysql_config Snow Leopard (only on Intel): sudo env ARCHFLAGS="-arch x86_64" gem install mysql -- --with-mysql-config=/usr/local/mysql/bin/mysql_config
