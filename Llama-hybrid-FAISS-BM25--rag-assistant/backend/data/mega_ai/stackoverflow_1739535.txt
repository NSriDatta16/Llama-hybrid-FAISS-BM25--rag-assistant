When calling the /shorten action of the REST API, you have to add a &history=1 parameter to tell bit.ly to add the URL to the web history for the authenticated user. For example: http://api.bit.ly/shorten?version=2.0.1&longUrl=http://cnn.com&login=bitlyapidemo&apiKey=R_0da49e0a9118ff35f52f629d2d71bf07&history=1 Clickable version of above link
