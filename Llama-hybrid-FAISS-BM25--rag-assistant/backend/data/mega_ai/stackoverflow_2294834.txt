If by word you mean these letter things, seperated from the others by punktuation, try this: private String longestCommonWord(String s1, String s2) { String[] seperators = new String[] { " ", ",", ".", "!", "?", ";" }; var result = from w1 in s1.Split(seperators, StringSplitOptions.RemoveEmptyEntries) where (from w2 in s2.Split(seperators, StringSplitOptions.RemoveEmptyEntries) where w2 == w1 select w2).Count() > 0 orderby w1.Length descending select w1; if (result.Count() > 0) { return result.First(); } else { return null; } } This probably is not the most elegant way to do it, but it works for me. =)
