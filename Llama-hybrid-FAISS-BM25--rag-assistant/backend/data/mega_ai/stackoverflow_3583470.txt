It seems you are only returning $var if the user agent is iPhone, otherwise you are trying to return an undefined variable (within your userAgent function) try: function userAgent(){ $browser = strpos($_SERVER['HTTP_USER_AGENT'],"iPhone"); $var = 0; if ($browser !== false) { $var = 1; } return $var; }
