The problem is in the second parameter ( USERAGENT ). A QByteArray can be constructed from a char array, but not from a QString (see QByteArray 's documentation ). You need either to use QString::toAscii() , QString::toLatin1() or something similar, or to make USERAGENT a char* : const char* USERAGENT = "Mozilla/Firefox 3.6.12";
