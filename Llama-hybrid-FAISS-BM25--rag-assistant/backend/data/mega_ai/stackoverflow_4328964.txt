You might want to consider adding this as a computed column . That way the column is virtual, so you're not using extra storage space. Also, any changes to name or last_name are automatically reflected in the new column without any intervention. EDIT : Modified code based on change in example. alter table example add full_name as coalesce(name+' ','') + last_name
