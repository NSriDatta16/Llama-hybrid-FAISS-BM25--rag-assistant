If you have an array array , then the average difference is (array(end) - array(1))/(length(array)-1) because diff(array) , where array = [a b c d] , is [b-a c-b d-c] . The average of that is (b-a+c-b+d-c)/3 , which simplifies to (d-a)/3 . In your example array = [1 5 6]; (array(end)-array(1))/2 ans = 2.5
