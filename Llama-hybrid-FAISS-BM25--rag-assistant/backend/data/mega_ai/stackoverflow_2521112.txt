You could do $sql = substr($sql, 0, -5); But perhaps the more elegant solution is $array = array('id' => '3', 'name' => 'roger'); $clauses = array(); foreach ($array as $k => $v) $clauses[] = $k . ' = ' . $v; $sql = "SELECT * FROM table WHERE " . implode(' AND ', $clauses);
