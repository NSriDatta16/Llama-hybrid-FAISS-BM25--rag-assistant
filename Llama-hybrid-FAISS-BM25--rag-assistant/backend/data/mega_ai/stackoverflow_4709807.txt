Here's a "folded" one-liner: @(foreach ($unc in $list){ test-connection $unc.split("\")[2] | measure responsetime -average | % {$_.average.tostring() + " $unc"}}) | sort |% {$_.split()[1]} If you want to save and display the ResponseTimes, replace that last line with: sort | select @{l="Share";e={$_.split()[1]}},@{l="ResponseTime";e={"{0:F2}" -f $_.split()[0]}}
