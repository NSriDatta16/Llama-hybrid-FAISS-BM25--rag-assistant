Many of the Unix books that describe the main system calls also implement a shell to illustrate how and why you might use the various calls. Stevens and Rochkind are two such books: W Richard Stevens, Stephen A Rago Advanced Programming in the Unix Environment, 3rd Edn Marc J Rochkind Advanced Unix Programming, 2nd Edn
