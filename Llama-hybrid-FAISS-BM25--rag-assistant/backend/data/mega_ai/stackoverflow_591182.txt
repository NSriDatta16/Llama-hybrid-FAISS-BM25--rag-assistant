I have made some helper classes to create html controls, so my code would look like this: foreach (SearchResult image in Search.GetImages(componentId)) { ContainerMarkup.Controls.Add( Tag.Div.CssClass("captionedImage") .AddChild(Tag.Image(image.Resolutions[0].Uri).Width(150).Alt(Image.Caption)) .AddChild(Tag.Paragraph.Text(Image.Caption))); } At first it might not look simpler, but it's easy to work with, creates controls that does proper html encoding of the values, and there is no string escaping issues.
