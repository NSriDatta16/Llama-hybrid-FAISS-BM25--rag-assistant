Is not required to zip JavaScript file, instead, you should use mod_deflate Apache will handle the compression and detect clients browser support gzip header. If the clients browser support the gzip header, a compressed copy of JavaScript will be sent. You can also explicitly set an expiration header for static files to better leverage on browser cache.
