A closure is a first class function with bound variables. Roughly that means that: You can pass the closure as a parameter to other functions The closure stores the value of some variables from the lexical scope that existed at the time that is was created Java initially didn't have syntactic support for closures (these were introduced in Java 8), although it was fairly common practice to simulate them using anonymous inner classes.
