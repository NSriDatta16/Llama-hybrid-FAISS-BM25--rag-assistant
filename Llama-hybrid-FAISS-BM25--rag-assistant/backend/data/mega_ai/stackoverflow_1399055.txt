if you are using axis or Jax Rpc, then you can this to get the IP Address and Browser, This has to be done on the server side stub. HttpServletRequest httpReq = (HttpServletRequest) MessageContext.getCurrentContext().getProperty(HTTPConstants.MC_HTTP_SERVLETREQUEST); String ip = httpReq.getRemoteAddr(); String browser = httpReq.getHeader("User-Agent");
