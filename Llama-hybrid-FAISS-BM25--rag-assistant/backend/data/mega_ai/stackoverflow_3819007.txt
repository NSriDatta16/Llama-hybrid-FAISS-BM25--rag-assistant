I've always disliked the inline-block method, so I'm using floats for this instead. See if this is what you want (I've taken the liberty of making the anchors expand out to the size of the list items). http://jsfiddle.net/gSMVA/ The code: #menu { background: #5a5a5a; height: 100px; left: 0px; position: fixed; top: 50px; width: 100%; } #menu ul { list-style: none; margin: 0px; } #menu li { border-right: 1px solid #c6c6c6; height: 100px; float: left; } #menu ul li a { color:#fff; padding: 0 25px; height: 100px; display: block; text-decoration:none; /* Height * 2 - font size */ line-height: 188px; font-size: 12px; } #menu ul li a:hover { background: #c6c6c6; color: #fff; }
