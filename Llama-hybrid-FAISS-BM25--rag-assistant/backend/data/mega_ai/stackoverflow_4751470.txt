If you write: (defun negate (n) (if ( > n 0) (- 0 n) n)) then you're limiting your code to real numbers. If instead you use the primitive negate function provided by Common Lisp, it will work on any number: (mapcar (function -) '(1 2/3 #C(4 5))) --> (-1 -2/3 #C(-4 -5))
