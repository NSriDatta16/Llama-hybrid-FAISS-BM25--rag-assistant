Take a look at the Disconnected Service Agent Application Block from the Smart Client Software Factory: http://msdn.microsoft.com/en-us/library/cc540752.aspx It provides a store-and-forward mechanism for remote web service calls -- you could combine this with caching to acheive something similar.
