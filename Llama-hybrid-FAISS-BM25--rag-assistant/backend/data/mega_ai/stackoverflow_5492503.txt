the all trigger solves all the dependencies(access the triggers mentioned): svm-train svm-predict svm-scale Now, each of that trigger runs that like of script, meaning: compile svm.o , using the compiler declared in CXX with the flags -shared -dynamiclib , outputing the file named libsvm.so.$(SHVER) (where $(SHVER) will be replaced with 2 in your case)
