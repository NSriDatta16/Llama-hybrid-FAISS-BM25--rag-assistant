you could also check out pdo, zend_db, and mdb2. good thing is they have named parameters and drivers that can properly create prepared statements (or emulate prepared statements) on many storage engines. for example, zend_db will do some basic sql parsing to handle edge cases, like say when a regex embedded in a query is a character class with a colon...
