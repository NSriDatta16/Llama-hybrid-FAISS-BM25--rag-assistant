"Identify the most influential precursor conditions (comorbidities) for a medical condition like chronic kidney disease" I'm not sure that it's possible to ID the most influential conditions; I think it will depend on what model you're using. Just yesterday I fit a random forest and a boosted regression tree to the same data, and the order and relative importance each model gave for the variables were quite different.
