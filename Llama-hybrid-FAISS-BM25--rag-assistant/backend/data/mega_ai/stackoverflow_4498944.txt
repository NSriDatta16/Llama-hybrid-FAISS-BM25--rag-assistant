It's a common issue for the images that are binded directly to ImageSource of the Image control. You should create the ImageSource through BitmapImage and set BitmapImage.CacheOption property to BitmapCacheOption.OnLoad: BitmapImage bi = new BitmapImage(); // Begin initialization. bi.BeginInit(); // Set properties. bi.CacheOption = BitmapCacheOption.OnLoad; // bi.EndInit(); More Details about BitmapImage.CacheOption on MSDN
