You've tagged this as a vba question, but the result can be easily achieved with a formula. If you are using Excel 2007 or later, use the AVERAGEIFS function =AverageIfS(B:B,A:A,">="& ,A:A," +10/1440)) Replace with a reference to a cell containing the beginning of your 10min period.
