This can be done by intersection of two datatables using System.Data; public static class DataTableExtensions { public static IEnumerable Intersect(this DataTable table, DataTable other) { return table.AsEnumerable().Intersect(other.AsEnumerable()); } public static IEnumerable Intersect(this DataTable table, DataTable other, IEqualityComparer comparer) { return table.AsEnumerable().Intersect(other.AsEnumerable(), comparer); } } This is a shameless port of an elegant solution :) Suppose you have two datasets set1 and set2 which you wanna compare. Do this var newtable = set1.Tables[0].Intersect(set2.Tables[0]).CopyToDataTable(); Post more details, if this is not what you want.
