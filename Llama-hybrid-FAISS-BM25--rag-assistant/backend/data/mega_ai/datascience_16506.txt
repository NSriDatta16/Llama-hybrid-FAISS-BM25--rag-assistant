Have you looked at the Stanford Car Dataset ? It has about 16k images of 200 cars. While it does not have the number of images you are looking for, it does seem sufficient for building a classifier (see references below) This blog post by Justin Chien provides a good overview of the approach for building the classifier on this data using CNN. And this paper also provides an overview of a few different approaches.
