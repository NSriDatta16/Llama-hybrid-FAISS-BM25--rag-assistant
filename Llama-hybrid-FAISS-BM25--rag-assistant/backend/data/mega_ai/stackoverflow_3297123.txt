Instead of a huge, confusing regexp, why not write a few lines using various string functions? Example: $start = strpos($txt, 'árvíztűrő') + 9; // position of first char after 'árvíztűrő' $end = strpos($txt, 'tükörfúrógép', $start); $inner = substr($txt, $start, $end - $start); $words = preg_split("/[\s,]+/", $inner); $num = count($words); Of course, this will eat up memory if you have some gigantic input string...
