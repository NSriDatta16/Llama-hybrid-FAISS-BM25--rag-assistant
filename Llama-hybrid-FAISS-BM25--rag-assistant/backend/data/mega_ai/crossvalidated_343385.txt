Wikipedia and WolframAlpha define running, rolling, and moving averages as the same thing. Searching the Wolfram Documentation returns the same answer. The website DeskBright , which offers lessons in Excel, has the same opinion . 'Dr. Math' at The Math Forum wrote: The term "rolling average" is used in the financial community. In statistics and mathematics, the term "moving average" is usually used.
