Scanner can also do it (since Java 1.5). The advantages over String#split is that you get some sort of automatic type conversion using regular expressions. Example from javadoc String input = "1 fish 2 fish red fish blue fish"; Scanner s = new Scanner(input).useDelimiter("\\s*fish\\s*"); System.out.println(s.nextInt()); System.out.println(s.nextInt()); System.out.println(s.next()); System.out.println(s.next()); s.close(); Also, if your aim is to recover the numbers by their name, use some kind of hash table to store it for faster retrieval.
