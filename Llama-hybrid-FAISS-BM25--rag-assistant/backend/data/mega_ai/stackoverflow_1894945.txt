linebreaks Replaces line breaks in plain text with appropriate HTML; a single newline becomes an HTML line break ( ) and a new line followed by a blank line becomes a paragraph break ( ). For example: {{ value|linebreaks }} If value is Joel\nis a slug , the output will be Joel is a slug .
