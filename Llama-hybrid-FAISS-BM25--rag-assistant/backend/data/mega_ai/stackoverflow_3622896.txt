var textNode = ' This will be dynamically generated. '; myDiv.appendChild(textNode); You need to create the element and the text inside of it as distinct nodes. This should be: var paragraph = document.createElement('p'); var textNode = document.createTextNode('This will be dynamically generated.'); paragraph.appendChild(textNode); myDiv .appendChild(paragraph); That's assuming that you really want the ... element. If you can do without it then simply create the text node and append that to myDiv .
