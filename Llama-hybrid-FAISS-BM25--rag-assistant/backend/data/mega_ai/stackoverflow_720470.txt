MongoDB is another option which is very similar to CouchDB, but using query language very similar to SQL instead of map/reduce in JavaScript. It also supports indexes, query profiling, replication and storage of binary data. It has huge amount of documentation which might be overwhelming at fist, so I would suggest to start with Developer's tour
