When you get request from browser, this request may be GET /filename.ext HTTP/1.0 Host: ... User-agent: ... ... You should search for your filename.ext If .ext is .htm or .html Function sendMessage should send "HTTP/1.0 200 Ok\nConnection: close\nContent-type: text/html\nContent-length: 999999\n\n" replace 999999 with the actual length of your file send the content of your file If .ext is .gif Function sendMessage should send "HTTP/1.0 200 Ok\nConnection: close\nContent-type: image/gif\nContent-length: 999999\n\n" replace 999999 with the actual length of your file send the content of your file And so on
