You can pass a function as the replacement parameter to re.sub to generate the replacement string from the match object, e.g.: import re s = "Idaho Ave N,,Crystal,Mn,55427-1463,US,,610839124763,Expedited" def upcase(match): return match.group().upper() print re.sub("[A-Z][a-z],", upcase, s) (This is ignoring the concern of whether you're genuinely finding state initials with this method.) The appropriate documentation for re.sub is here .
