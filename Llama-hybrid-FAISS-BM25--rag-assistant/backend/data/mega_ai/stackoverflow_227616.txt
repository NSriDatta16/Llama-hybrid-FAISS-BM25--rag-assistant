Credit Card Payment Solutions for Desktop Applications

Perhaps to put this into a little context, the requirements: Should be customisable Should be available in Australia at least, if not world wide (less critical). No storage of credit card details Recurrent micro payments of varying amounts each time 1-click payment. It's ok for the user to be registered with the payment processor provider.
