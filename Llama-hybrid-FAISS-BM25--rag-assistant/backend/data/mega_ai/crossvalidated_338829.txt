If $Y_i \sim \text{logistic}(\mu_i, s)$ then $P(Z_i) = P(Y_i > S) = \text{expit} ((S - \mu_i)/s)$. So the logistic regression model for $\text{logit} ( P(Z_i = 1 | X_i = x) )= \alpha + \beta x$ is a maximum likelihood estimator for $s = 1/\beta$ and $S=-\alpha/\beta$.
