For an honest-to-God rules engine, you could look into using Clips . Clips is an OpenSource expert system that you can embed into your own applications. If you want true scripting, there are tons of tools out there to add in scripting to applications with languages like Lua, TCL, REXX, etc. No matter what you do, you should check out the licensing closely. Embedding someone else's code into yours often has repercussions on how you can license the resulting program.
