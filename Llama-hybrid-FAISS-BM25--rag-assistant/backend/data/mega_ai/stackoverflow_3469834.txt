In order to access extension's localStorage you need to send a request to your background page: In script.js: chrome.extension.sendRequest("getStorageData", function(response) { console.log("response:", response); } In background.html: chrome.extension.onRequest.addListener(function(request, sender, sendResponse) { if(request === "getStorageData") { sendResponse(localStorage["data"]); } }); Cookies API is described in details here .
