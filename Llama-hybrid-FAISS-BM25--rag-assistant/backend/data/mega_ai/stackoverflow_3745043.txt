You probably need to add a reference to the namespace http://schemas.microsoft.com/developer/msbuild/2003 . I had a similar problem, I wrote about it here . Do something like this: XmlDocument xdDoc = new XmlDocument(); xdDoc.Load("blah/blah.csproj"); XmlNamespaceManager xnManager = new XmlNamespaceManager(xdDoc.NameTable); xnManager.AddNamespace("tu", "http://schemas.microsoft.com/developer/msbuild/2003"); XmlNode xnRoot = xdDoc.DocumentElement; XmlNodeList xnlPages = xnRoot.SelectNodes("//tu:ItemGroup", xnManager);
