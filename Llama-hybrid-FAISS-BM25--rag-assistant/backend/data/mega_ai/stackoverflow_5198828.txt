If you have a bytearray you can transform it into a String. Pay attention to the encoding, in the example I use utf-8. I think this is a simple way to do that: String newString = new String(byteArray, "utf-8"); newString = newString.replace("com.foo", "org.bar"); return newString.getBytes("utf-8");
