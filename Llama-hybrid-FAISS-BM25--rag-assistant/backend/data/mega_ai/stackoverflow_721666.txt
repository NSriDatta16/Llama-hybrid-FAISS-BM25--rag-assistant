Alternatively, you can make your XMLHttpRequest specify your user-agent as IE (browser sniffing is stupid): var req = new XMLHttpRequest(); req.setRequestHeader('User-Agent','MSIE 7.0'); req.open("GET", sURL, true); req.onreadystatechange=function() { if (req.readyState==4) { alert(req.responseText); } } req.send(null); Or better yet, send a strongly worded e-mail to the site admin about their not accepting requests from non-IE browsers. In this day and age, building an IE-only site is unacceptable. It undermines the accessibility & usefulness of the web in so many ways.
