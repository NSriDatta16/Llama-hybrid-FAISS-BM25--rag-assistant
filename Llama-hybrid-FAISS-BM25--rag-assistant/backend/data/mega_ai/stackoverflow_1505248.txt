I don't think there is a very elegant solution. However, you could try: If Not IsError(Application.Match(x, Array("Me", "You", "Dog", "Boo"), False)) Then or you could write your own function: Function ISIN(x, StringSetElementsAsArray) ISIN = InStr(1, Join(StringSetElementsAsArray, Chr(0)), _ x, vbTextCompare) > 0 End Function Sub testIt() Dim x As String x = "Dog" MsgBox ISIN(x, Array("Me", "You", "Dog", "Boo")) End Sub
