Looking at the HTML you’ve got , you could use the following JavaScript (I think): $("#midden-offertes form .veld .radio :radio").change(function() { var $subvragen = $(this).parents('.veld').siblings('.subvragen'); if (this.value === 'Ja') { $subvragen.show(); } else { $subvragen.hide(); } }); Alternatively, if you could add id attributes to your subvragen s and radios, you could use them — it’d make the JavaScript less reliant on the HTML nesting. E.g. if your HTML were like this: Then your JavaScript would go something like this: $("#midden-offertes form .subvragen").hide(); $("#midden-offertes form .veld .radio :radio").change(function() { if (this.value === 'Ja') { $(".subvragen#subvragen_" + this.id).show() } else { $(".subvragen#subvragen_" + this.id).hide(); } });
