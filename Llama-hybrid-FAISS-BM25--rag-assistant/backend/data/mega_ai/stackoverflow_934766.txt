Yes, you should use a transaction. From Ayende's blog: "NHibernate assume that all access to the database is done under a transaction, and strongly discourage any use of the session without a transaction." For more details, here's a link to his blog posting: http://ayende.com/Blog/archive/2008/12/28/nh-prof-alerts-use-of-implicit-transactions-is-discouraged.aspx
