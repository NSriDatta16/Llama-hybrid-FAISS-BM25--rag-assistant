You should just be able to use: navigateToURL(new URLRequest('http://www.cnnibn.com'), '_blank'); (make sure you import the relevant packages: "import flash.net.*") This approach may have issues with some popup blockers, if so then you could have a look at the class here: http://www.zorked.com/flash/flash-and-navigatetourl-popup-blocking/
