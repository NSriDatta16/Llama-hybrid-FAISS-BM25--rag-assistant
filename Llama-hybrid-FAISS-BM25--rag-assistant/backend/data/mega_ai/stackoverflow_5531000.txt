You can use nonzero? , as in: def compute_average(a,b,c,d,e) total = [a,b,c,d,e].sum.to_f average = [a, 2*b, 3*c, 4*d, 5*e].sum / (total.nonzero? || 1) end More people would be more familiar with using the ternary operator (total == 0 ? 1 : total) , so that's another possibility.
