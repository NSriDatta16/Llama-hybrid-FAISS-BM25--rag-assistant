Using Wireshark and wget, the differences were in the headers only. The remote server requires User Agent and an Accept headers. eg: WebRequest request = WebRequest.Create("http://actueel.nl.pwc.com/site/syndicate.jsp"); ((HttpWebRequest)request).UserAgent = "stackoverflow.com/q/4233673/111013"; ((HttpWebRequest) request).Accept = "*/*";
