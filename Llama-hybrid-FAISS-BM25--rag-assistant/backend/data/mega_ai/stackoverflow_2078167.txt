I was following Microsoft example and didn't notice that the declarations in GUI.Designer.cs (automatic, by IDE) and in GUI.cs (manual, from example) are redundant. === GUI.cs === public GUI() { InitializeComponent(); this.lvwFiles.DragDrop += new System.Windows.Forms.DragEventHandler(this.lvwFiles_DragDrop); this.lvwFiles.DragEnter += new System.Windows.Forms.DragEventHandler(this.lvwFiles_DragEnter); } === GUI.Designer.cs === // // lvwFiles // ... this.lvwFiles.DragDrop += new System.Windows.Forms.DragEventHandler(this.lvwFiles_DragDrop); this.lvwFiles.DragEnter += new System.Windows.Forms.DragEventHandler(this.lvwFiles_DragEnter);
