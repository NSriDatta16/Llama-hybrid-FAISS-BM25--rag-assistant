Its actually pretty common to write server-side code with branches based on a incoming http user agent, such as the case when you want to output different data for the Googlebot spider vs everyone else. Given a truly pathological example, its wholly possible for someone to write server-side code which, for some reason or another, throws an exception when users access the page with an unexpected user agent string.
