Write udev rule. Something like this: KERNEL=="sd*", SUBSYSTEMS=="scsi", ATTRS{model}=="USB 2.0 Storage Device", RUN+="/somewhere/copy.sh" You can find more information on udev rules here: http://www.reactivated.net/writing_udev_rules.html#example-usbhdd Edit: on ubuntu you need to put this rule to /etc/udev/rules.d/80-usbcopy.rules
