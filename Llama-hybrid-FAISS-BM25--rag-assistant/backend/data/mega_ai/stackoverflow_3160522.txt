Each div is probably going to need its own click event to grab its inner text and store it in a hidden form field. This can be more elegantly accomplished with jQuery outside of the HTML so as to separate it from the content. jQuery click event: http://api.jquery.com/click/ jQuery method for getting element's text: http://api.jquery.com/text/
