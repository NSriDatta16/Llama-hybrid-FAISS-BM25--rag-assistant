I can recommend the open source project javaEventing. It's available at http://code.google.com/p/javaeventing , and makes it easy to define, register for and trigger custom events, much like in C#. An example: Class MyEvent extends EventManager.EventObject {} EventManager.registerEventListener(new EventManager.GenericEventListener(){ public void eventTriggered(Object sender, Event event) { // } }, new MyEvent()); EventManager.triggerEvent(this, new MyEvent()); // bob
