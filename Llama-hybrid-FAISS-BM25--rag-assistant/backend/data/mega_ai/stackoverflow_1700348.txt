Generally, this isn't possible. However, if the client is using Internet Explorer, and has InfoPath installed (which is part of Office), you can check the user agent for InfoPath.2 . Another option is to check for MS-RTC LM , if they have the Office Live Meeting installed. This is very limiting, but it just might work on a local intranet.
