Know I'm a bit late, but to get probabilities from xgboost you should specify multi:softmax objective like this: xgboost(param, data = x_mat, label = y_mat,nround = 3000, objective='multi:softprob') From the ?xgb.train : multi:softprob same as softmax, but output a vector of ndata * nclass, which can be further reshaped to ndata, nclass matrix. The result contains predicted probabilities of each data point belonging to each class.
