Is there any way to edit values in a table, like how you can use phpMyAdmin to manipulate values in a MySQL database? Xcode has a means of creating a quick and dirty interface for a data model . You just drag the data model file into a window in interface builder and it autogenerates an interface for you. This lets you view the data without having to have your entire app up and running.
