Using Dns.GetHostAddresses(host); . A fragment of the actual code is as follows: public void Connect(string host, int port){ // Checking parameters etc, removed. IPAddress[] addresses = Dns.GetHostAddresses(host); Connect(addresses,port); } So, the constructor with the hostname simply calls the constructor with the IP addresses. See http://msdn.microsoft.com/en-us/library/system.net.dns.gethostaddresses.aspx for documentation about Dns.GetHostAddresses() .
