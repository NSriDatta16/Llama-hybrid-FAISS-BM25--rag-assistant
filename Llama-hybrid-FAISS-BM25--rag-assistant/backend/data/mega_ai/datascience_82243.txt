You can use tf.keras.utils.plot_model . For instance this example given in the tensorflow doc: import tensorflow as tf input = tf.keras.Input(shape=(100,), dtype='int32', name='input') x = tf.keras.layers.Embedding( output_dim=512, input_dim=10000, input_length=100)(input) x = tf.keras.layers.LSTM(32)(x) x = tf.keras.layers.Dense(64, activation='relu')(x) x = tf.keras.layers.Dense(64, activation='relu')(x) x = tf.keras.layers.Dense(64, activation='relu')(x) output = tf.keras.layers.Dense(1, activation='sigmoid', name='output')(x) model = tf.keras.Model(inputs=[input], outputs=[output]) dot_img_file = 'model_1.png' tf.keras.utils.plot_model(model, to_file=dot_img_file, show_shapes=True) you get the following: For this function to work you need to install the dependencies: pydot (pip3 install pydot) an graphviz (install will depend on your OS) have a look at this link
