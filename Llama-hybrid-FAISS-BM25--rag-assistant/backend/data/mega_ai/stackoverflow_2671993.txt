You can rearrange your code a bit and use .delegate() , like this: return this.each(function() { var id = o["id"]; jQuery(' ', { 'class': o['className'], html: o.msg }) .append(' X ') .delegate('a','click', function() { $(this).parent().remove(); dismiss(id); }) .appendTo(this); }); This uses the new jQuery(html,props) added in jQuery 1.4 to make the creation a bit cleaner (and faster! document fragment caching!). What it's doing is instead of attaching an onclick to the X, it's listening for a click from any in the div and when it bubbles, it executes the same code as it used to only on the X anchor.
