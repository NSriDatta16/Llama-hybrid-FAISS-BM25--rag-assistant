Most web browsers have their own storage. Mozilla uses SQLite for some things. Whenever your app/browser needs to retrieve a remote resource (URL of any kind), simply log it to a database table. Perhaps use SQLite yourself for this. A decent start would be to create a history table like this: URL --varchar(max) LastAccessed --datetime TotalRequests --int
