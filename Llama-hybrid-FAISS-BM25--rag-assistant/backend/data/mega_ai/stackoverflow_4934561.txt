If you also want to match for classname fragments (e.g. to show "apple" when typing "ple"), consider using the attribute contains selector '[class*="'+filter+'"]' . Combined with the answers above: function filter() { var filter = $("#filter").val(); var photos = $('.photo'); photos.show(); if (filter != '') photos.not('[class*="'+filter+'"]').hide(); }
