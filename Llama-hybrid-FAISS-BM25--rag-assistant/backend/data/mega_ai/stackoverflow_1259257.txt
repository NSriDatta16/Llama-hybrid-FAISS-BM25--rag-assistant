If you have clicked the X button in the top-right corner of the web part, it is simply closed. You can get it back as follows: Click on Actions , Edit Page Click Add a web part Click Advanced Web Part Gallery in the top bottom-right corner of the dialog Click on Closed Web Parts in the tool pane on the right Drag the closed web part from the tool pane back onto the page If you have actually used the drop-down menu on the web part and clicked Delete , it really is deleted. If you have check-in/out available to the page you can go back to a previous version to restore it. Otherwise you will need to restore from backup if you have one.
