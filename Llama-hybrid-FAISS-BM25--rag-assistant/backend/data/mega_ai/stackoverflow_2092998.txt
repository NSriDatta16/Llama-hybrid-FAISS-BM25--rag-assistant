Ok - managed to sort this in the end, although it's not the most elegant solution. Steps to take: Use Eclipse's create plug-in from JARs wizard Import jaxrs-api-1.2.1.GA.jar resteasy-jaxrs-1.2.1.GA.jar resteasy-jaxb-provider-1.2.1.GA.jar jaxb-api-2.1.jar jaxb-impl-2.1.12.jar slf4j-api-1.5.8.jar slf4j-simple-1.5.8.jar Export the following packages javax.ws.rs javax.xml.bind.annotation org.jboss.resteasy.client (and all subpackages) org.jboss.resteasy.plugins.providers org.jboss.resteasy.plugins.providers.jaxb org.jboss.resteasy.spi Add that plug-in as a dependency, and obtain a reference to the client in the usual way
