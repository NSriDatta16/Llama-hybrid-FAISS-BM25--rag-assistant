Looks like there's a similar question asked here: Google App Engine self.redirect post The answer to that one recommends using the urlfetch.fetch() to do the post. import urllib form_fields = { "first_name": "Albert", "last_name": "Johnson", "email_address": "Albert.Johnson@example.com" } form_data = urllib.urlencode(form_fields) headers = {'Content-Type': 'application/x-www-form-urlencoded'} result = urlfetch.fetch(url=url, payload=form_data, method=urlfetch.POST, headers=headers)
