you need to use a custom itemRenderer for your datagrid field. There are some great tutorials out there... http://www.adobe.com/devnet/flex/quickstart/using_item_renderers/ http://www.adobe.com/devnet/flex/articles/itemrenderers_pt1.html (is a series of And if you know what you are looking for then I'm sure that you can find the right itemRenderer for your datagrid. Then using your custom component you can move it around, zoom, etc. If you are talking about the container that contains the datagrid checkout the flexlib mdi container. This container can have other windows inside of it then you can tile, cascade, fill, etc. This also supports dragging. example: http://www.returnundefined.com/flexmdi/explorer/ webpage: http://code.google.com/p/flexmdi/
