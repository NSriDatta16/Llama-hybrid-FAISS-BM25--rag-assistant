Tools, Options, HTML Designer, Start Pages, Source View. Also, you mean Shift+F7. EDIT : If you want to view the code-behind and the HTML source simultaneously, you'll need to drag the designer tab to the middle of the screen and create a second horizontal (or vertical) pane. EDIT : Once you're in Design view, pressing Shift + F7 again will switch to HTML source view.
