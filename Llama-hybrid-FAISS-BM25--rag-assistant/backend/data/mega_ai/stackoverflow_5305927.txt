You should use key based authentification , possibly coupled with ssh-agent to remember key passphrase. You can invoke sh -c as the command, and pass it a string containing the list of command to execute. ssh invoke a shell on the remote machine, so you can pass a list of command as a string. For example: $ ssh user@ip "echo 'Hello world'; whoami; cd / ; ls"
