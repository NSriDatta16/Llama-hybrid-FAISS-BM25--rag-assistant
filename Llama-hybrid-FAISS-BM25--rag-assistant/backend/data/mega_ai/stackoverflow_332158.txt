You can do this without embedding the font by using the blendMode, with a minor sacrifice in text appearance (I believe). import flash.display.BlendMode; import flash.text.TextField; // later... var tf:TextField = new TextField(); tf.blendMode = BlendMode.LAYER; tf.alpha = 0.5;
