Probably the most elegant solution is using a Custom Attribute, you can later access Model metadata using: ViewData.ModelMetadata. e.g: [Foreign(Type="DropDown", TableName="Countries")] public int IdCountry { get; set; } where ForeignAttribute is a class you must declare, and later use it to build your editor template.
