The table must be InnoDB. From SHOW STATUS Doc Rows The number of rows. Some storage engines, such as MyISAM, store the exact count. For other storage engines, such as InnoDB, this value is an approximation, and may vary from the actual value by as much as 40 to 50% . In such cases, use SELECT COUNT(*) to obtain an accurate count.
