Your JSON syntax is wrong. You can't have the same key twice in an object. Instead, you need an array: var homes = { "Agents" : [ { "name" : "Bob Barker" }, { "name" : "Mona Mayflower" } ], ... } Then you can access the agents like so: homes.Agents[1] // => { "name": "Mona Mayflower" } // or homes.Agents[1].name // => "Mona Mayflower"
