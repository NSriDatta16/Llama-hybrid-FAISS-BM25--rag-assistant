I think that you should include the header server side instead of embedding it using an iframe. How you do this depends on what platform you're using. If you are using plain html , the simplest option is to just copy the header. If you are using IIS you could extract the header to its own file and include it using SSI : If you're using some kind of templating system , perhaps you could extract the header so that it exists in its own template file and then include it where needed.
