Plug-ins will not fire for operations directly against the SQL database. You must use mechanisms which work against the web services for your plug-ins to fire. CRM has a couple of import functions: The Import functions through their web-ui and the Data Migration Manager (download for the SDK). Here is SDK coverage . I'm actually not 100% sure whether the DMM goes against the web services or not.
