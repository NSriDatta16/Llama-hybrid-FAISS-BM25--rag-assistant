You can use the balanced accuracy metric for such cases. It is defined as the average of recall obtained on each class. from sklearn.metrics import balanced_accuracy_score y_true = [0, 1, 0, 0, 1, 0] y_pred = [0, 1, 0, 0, 0, 1] balanced_accuracy_score(y_true, y_pred)
