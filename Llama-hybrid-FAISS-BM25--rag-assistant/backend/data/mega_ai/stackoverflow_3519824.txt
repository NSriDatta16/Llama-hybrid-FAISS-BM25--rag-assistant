The ucdetector is strictly limited to eclipse and can't be used in a different environment. So if you wanted to use it, you'd have to install a basic eclipse somewhere with projects pointing to your source directories and use it for unused code detection only. Another way to identify unused code is instrumenting the (live) application, run it for a while and analyse the logs. Usual tools are Emma, Cobertura, and Clover.
