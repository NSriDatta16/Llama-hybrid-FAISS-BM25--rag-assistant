Instead of using a timer, leverage the thread pool instead: bool fired = false; ThreadPool.RegisterWaitForSingleObject(new ManualResetEvent(false), (state, triggered) => { fired = true; }, 0, 9000, true); GC.Collect(); Thread.Sleep(10000); Assert.IsTrue(fired); This survives garbage collection since you don't have to retain a reference to anything.
