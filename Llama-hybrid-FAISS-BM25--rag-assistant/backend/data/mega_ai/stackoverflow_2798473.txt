First, make sure your device is not mounted. Either Android or the host PC can access the SD card, but not both simultaneously. Second, it is unclear why you are using a FileInputStream and getFD() . Just pass the path to the file on the SD card to the MediaPlayer (e.g., new File(Environment.getExternalStorageDirectory(), "yourfile.mp3") ) and let the player open the file.
