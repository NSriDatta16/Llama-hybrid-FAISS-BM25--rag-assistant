You may want to consider creating a gem for your library. This would make it easy to use the library both stand-alone and with Rails, as well as make deployment/updates simpler. Further, because gems normally follow a specific directory structure, it also solves your issue of not knowing how to organize the library. There is plenty of documentation available for creating gems. Here 's a bit more info about file structure, as well as other useful tips.
