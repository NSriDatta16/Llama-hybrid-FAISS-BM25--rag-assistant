We are also in the same situation wherein we want to control when the .XAP file gets downloaded to the browser. An approach that you might want to take a look at is to use the Silverlight Isolated Storage as a "cache" to store your .XAP files. Check out this blog: IsolatedStorage as a Silverlight object cache
