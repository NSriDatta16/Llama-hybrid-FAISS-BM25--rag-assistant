We use the following solution to set foreign keys without retreiving the corresponding entity from the context. Blog blog = new Blog { Name = blogName, Slogan = slogan, UsersReference.EntityKey = new EntityKey("EntityModel.Users", "UserId", userId) } Entities.AddToBlogs(blog); Entities.SaveChanges();
