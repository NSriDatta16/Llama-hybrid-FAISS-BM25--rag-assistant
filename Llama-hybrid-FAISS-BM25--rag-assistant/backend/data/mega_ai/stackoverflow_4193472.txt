First, you should put your Javascript code outside the element itself. This is fairly simple to do. It makes your HTML and Javascript much more easily comprehensible and ultimately allows much more organised code. First, give your a element an id : Link Then make your call in a script tag: $(document).ready(function() { // when the whole DOM has loaded $('#loadPage').click(function(){ // bind a handler to clicks on #loadPage $('#target') .hide() // make sure #target starts hidden .load('page.html', function() { $(this).fadeIn(1000); // when page.html has loaded, fade #target in }); }); }); Edit To comment, yes you can use a URL in the a tag and then use this.href . Link $(document).ready(function() { $('#loadPage').click(function(e){ e.preventDefault(); $('#target') .hide() .load(this.href, function() { $(this).fadeIn(1000); }); }); });
