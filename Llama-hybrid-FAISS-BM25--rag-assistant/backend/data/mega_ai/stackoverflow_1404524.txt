If you dump the System.properties you'll find a property named user.name . This should more or less be the OS user that started the JDK process. Here's a code fragment to help Properties props = System.getProperties(); Enumeration e = props.propertyNames(); while( e.hasMoreElements() ){ Object key = e.nextElement(); Object val = props.getProperty( key.toString() ); System.out.println( "key:"+ key +" val:"+ val ); }
