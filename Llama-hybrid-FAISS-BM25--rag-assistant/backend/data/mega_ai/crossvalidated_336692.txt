It is used extensively, but it is still a tool in a giant set of tools. If you have prior knowledge of the parameters or coherence is a key element of the work you are doing, then it should clearly be used. If, on the other hand, you find a need for a guaranteed minimum rate against false positives or controlling power is important, then you would use a Pearson and Neyman method. Likewise, if your goal is epistemological and you need to rule out a wide class of alternatives, Fisher's method may be preferable. And, finally, if you do not know the functional form, you may consider neural networks.
