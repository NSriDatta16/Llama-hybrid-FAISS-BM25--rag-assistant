By default Jersey will use JAXB to process the JSON messages by leveraging the Jettison library. So you can do the following: @PUT @Consumes({MediaType.APPLICATION_XML, MediaType.APPLICATION_JSON}) public String putTodo(JAXBElement r) { Route route = r.getValue(); route.toString(); System.out.println("Received PUT XML/JSON Request"); return "ok"; } For More Information on Using Jettison with JAXB: http://bdoughan.blogspot.com/2011/04/jaxb-and-json-via-jettison.html http://bdoughan.blogspot.com/2011/04/jaxb-and-json-via-jettison-namespace.html
