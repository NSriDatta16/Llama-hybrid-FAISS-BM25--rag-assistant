Try this code : StatFs stat = new StatFs(Environment.getExternalStorageDirectory().getPath()); long bytesAvailable = (long)stat.getBlockSize() *(long)stat.getBlockCount(); long megAvailable = bytesAvailable / 1048576; System.out.println("Megs: " + megAvailable); Explanation getBlockCount() - return size of SD card; getAvailableBlocks() - return the number of blocks that are still accessible to normal programs (thanks Joe)
