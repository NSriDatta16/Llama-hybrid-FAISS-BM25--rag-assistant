ImageButton inherits from View so you can always use: imageButton.setVisibility(View.INVISIBLE); In order to have the view disappear after x amount of time you can use a handler Handler handler = new Handler(); handler.postDelayed( new Runnable() { public void run(){ imageButton.setVisibility(View.INVISIBLE); } }, 5000);//delayed 5 secs Ensure to call this after you are done with everything of the view and after setContentView or onViewCreated (for fragments) is called
