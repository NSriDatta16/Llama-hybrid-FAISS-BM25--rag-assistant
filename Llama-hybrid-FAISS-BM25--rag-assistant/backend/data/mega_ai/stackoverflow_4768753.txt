Try this (JSFiddle): $('#Add').click(function(){ var id = "newtextarea"+Math.floor(Math.random()*1000); $('#pane').append($(' ')); $('textarea:last').focus(); bindAgain(id); }); //inital resize resizeTextArea($('#tst')); //'live' event $('textarea.new').bind('keyup scroll', function() { resizeTextArea($(this)); }); function bindAgain(id) { $('#'+id).bind('keyup scroll', function() { resizeTextArea($(this)); }); } Basically, it rebinds the event using a dynamically created ID. Not as elegant as karim79's solution, but it works.
