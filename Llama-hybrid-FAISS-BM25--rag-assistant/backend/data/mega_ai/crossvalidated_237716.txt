Correlation: $$Y_t = \alpha + \beta X + \epsilon$$ Auto-correlation: $$Y_t = \alpha + \beta Y_{t-1} + \epsilon$$ Auto-correlation is usually an issue in time series data. While correlation appears in every data set with at least two variables.
