The fragment identifer, e.g. #some-anchor, is controlled by the browser, not the server. JavaScript would be needed to redirect and remove this, although why you would want to do so I am not sure. [SNIPPED after clarification] To rewrite only when the query string is empty: RewriteCond %{QUERY_STRING} ^$ RewriteRule ^/?([a-zA-Z/]+)/?$ /index.php [NC,L]
