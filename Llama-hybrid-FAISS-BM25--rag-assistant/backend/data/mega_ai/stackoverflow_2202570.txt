What I'd do is to set up "#selectables1" with an event handler for a made-up event called "dragSetup". That would look something like this: $('#selectables1').bind('dragSetup', function() { $(this).find('span:not(.dragReady)') .draggable({ ... }) .addClass('dragReady'); }); Then whenever you add a new you can just call: $('#selectables1').trigger('dragSetup');
