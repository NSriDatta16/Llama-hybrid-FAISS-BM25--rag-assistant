Create a custom attribute with a single string property, tag your classes with it, then GetCustomAttributes to test a particular class is tagged with the attribute, and if it is you can read the property to pull the name you set. public class ActivityTypeAttribute : Attribute { public Name { get; set; } } [ActivityType(Name="MyClass")] public class MyClass { } ... { ActivityTypeAttribute att = (ActivityTypeAttribute)Attribute.GetCustomAttribute(typeof(MyClass), typeof(ActivityTypeAttribute)); Debug.Assert( att.Name == "MyClass" ); } ...
