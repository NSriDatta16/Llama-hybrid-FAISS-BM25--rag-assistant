How do I stop a form submitting with submit() when I return false within a nested each() function?

I need to check a form before submitting but need to loop through some inputs. If an input is invalid, I need to return false and stop the submit, however because the test is within the nested each() function it's not returning false in the outer submit() function: $("#my_form").submit(function() { $("#my_form .my_text_inputs").each(function() { if ($(this).val() == "bad value") { alert("Input was invalid"); return false; } }); return true; }); My solution is: $("#my_form").submit(function() { var _bSubmitTheForm = true; $("#my_form .my_text_inputs").each(function() { if ($(this).val() == "bad value") { alert("Input was invalid"); _bSubmitTheForm = false; return false; } }); return _bSubmitTheForm; }); ...is there a more elegant way to do this? Thanks
