Prevent stroke border from overflowing path

strokeRect (0, 0, canvasWidth, canvasHeight); draws a full rectangle but the line width is cut in half, here is an example: var canvas = document.getElementsByTagName ("canvas"); var lineWidth = 10; var canvasHeight = canvas[0].height; var canvasWidth = canvas[0].width; var ctx = canvas[0].getContext ("2d"); ctx.lineWidth = lineWidth; ctx.strokeRect (0, 0, canvasWidth, canvasHeight); I can fix that but isn't elegant: var canvas = document.getElementsByTagName ("canvas"); var lineWidth = 10; var canvasHeight = canvas[0].height - lineWidth; var canvasWidth = canvas[0].width - lineWidth; var ctx = canvas[0].getContext ("2d"); ctx.lineWidth = lineWidth; lineWidth /= 2; ctx.strokeRect (lineWidth, lineWidth, canvasWidth, canvasHeight); Is there a native method to do that ? Something like the "box-sizing" css3 attribute: canvas { box-sizing: border-box; } Thanks.
