Try this: If you are doing something like $('#thelist').sortable(); , do the following too: $('#thelist img').mousedown(function(event) { event.preventDefault(); }); This will prevent normal behaviour on all the images inside the list (which is, at least on Chrome and Firefox, to start dragging the image).
