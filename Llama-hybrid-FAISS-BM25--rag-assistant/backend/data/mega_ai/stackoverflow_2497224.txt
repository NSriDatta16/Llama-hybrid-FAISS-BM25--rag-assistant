NSURLDownload not on iPhone see the note: iPhone OS Note: The NSURLDownload class is not available in iPhone OS as downloading directly to the file system is discouraged. Use the NSURLConnection class instead. See “Using NSURLConnection” for more information. Have a look at Apple's documentation about URL loading system and NSURLDownload .
