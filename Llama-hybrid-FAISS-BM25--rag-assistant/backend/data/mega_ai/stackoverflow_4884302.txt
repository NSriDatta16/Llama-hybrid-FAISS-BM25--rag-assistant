For the DropDown property you need a ContextMenuStrip . The easiest way to find out how to fill it up is to drag&drop one from the toolbox onto your form, fill it up, select it in the DropDown property and afterwards take a look into the Designer.cs file to see how all the stuff is glued together. The drawback of using the DropDownItems property is that you can't alter some properties like ShowImageMargin .
