From https://source.android.com/source/downloading#installing-repo : $ curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo $ chmod a+x ~/bin/repo $ mkdir working-directory-name $ cd working-directory-name $ repo init -u git://android.git.kernel.org/platform/manifest.git Authoritative instructions
