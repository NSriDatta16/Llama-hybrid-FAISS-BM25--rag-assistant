To dispatch an event, use the Mage::dispatchEvent function ( %magento%/app/Mage.php around line 425 ). Calls look like this: $data = array( 'somedata' => 'foo', 'layout' => $this->getLayout()); Mage::dispatchEvent('my_event_name', $data); In order to observe an event, specify the observer in the config.xml file of your extension. singleton myextension/observer someMethodName Create a corresponding class and method (Observer.php) and you're set to go: class Myextension_Model_Observer { public function someMethodName($event) { $layout = $event->getLayout(); $someData = $event->getSomedata(); } }
