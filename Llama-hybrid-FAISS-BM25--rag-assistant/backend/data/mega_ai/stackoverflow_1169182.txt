See How many Python classes should I put in one file? Sketch your overall set of class definitions. Partition these class definitions into "modules". Implement and test the modules separately from each other. Knit the modules together to create your final application. Note . It's almost impossible to decompose a working application that evolved organically. So don't do that. Decompose your design early and often. Build separate modules. Integrate to build an application.
