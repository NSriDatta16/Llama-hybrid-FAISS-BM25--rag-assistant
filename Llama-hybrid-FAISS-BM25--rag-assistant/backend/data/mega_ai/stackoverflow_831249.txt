I just found the answer to this in another question on SO: How can I enable live preview for FCKeditor in an ASP.Net site? Also, when I use a div element instead of a paragraph element, it works. Here's my final working code for anyone it might help: FCKeditor - Sample function FCKeditor_OnComplete( oFCKeditor ) { oFCKeditor.Events.AttachEvent( 'OnSelectionChange', function() { document.getElementById("postText").innerHTML = oFCKeditor.GetHTML(true); }) ; } var oFCKeditor = new FCKeditor('fckEdtr'); oFCKeditor.BasePath = "./fckeditor/"; oFCKeditor.ToolbarSet = 'Custom' ; oFCKeditor.Create();
