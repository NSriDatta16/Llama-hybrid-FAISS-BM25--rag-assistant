I think the problem is here: File sdCard = Environment.getExternalStorageDirectory(); File dir = new File (sdCard.getAbsolutePath() + "/SDCARD/Carburant"); dir.mkdirs(); copyfile(context,"/data/data/carburant.android.com/files/", "/SDCARD/Carburant/storeddata.dat"); You create the directories with the absolute path of the SD card prepended to "/SDCARD/Carburant/" , but when you call your copyfile() routine, you forgot the absolute path of the SD card. Try this: copyfile(context,"/data/data/carburant.android.com/files/", sdCard.getAbsolutePath() + "/SDCARD/Carburant/storeddata.dat");
