Add this to your module Public Declare Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As Long) Or, for 64-bit systems use: Public Declare PtrSafe Sub Sleep Lib "kernel32" (ByVal dwMilliseconds As LongPtr) Call it in your macro like so: Sub Macro1() ' ' Macro1 Macro ' Do Calculate Sleep 1000 ' delay 1 second Loop End Sub
