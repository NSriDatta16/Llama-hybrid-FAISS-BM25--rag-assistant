You can apply an XSLT stylesheet to extract the desired content. This stylesheet should fit your example: Apply it with something like the following (exception handling not included): String xml = " Content Content "; Source xsl = new StreamSource(new FileReader("/path/to/file.xsl"); TransformerFactory transformerFactory = TransformerFactory.newInstance(); Transformer transformer = transformerFactory.newTransformer(xsl); transformer.setOutputProperty(OutputKeys.OMIT_XML_DECLARATION, "yes"); StringWriter out = new StringWriter(); transformer.transform(new StreamSource(new StringReader(xml)), new StreamResult(out)); System.out.println(out.toString()); This should work with any version of Java starting with 1.4.
