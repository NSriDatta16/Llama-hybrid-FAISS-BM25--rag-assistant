Let jQuery handle the escaping and encoding of the parameters: $.ajax({ type: "POST", url: "Article.asmx/postComment", data: { article_id: article_id, comment_body: comment_body }, contentType: "application/json; charset=utf-8", dataType: "json", success: function(msg) { page_num = 1; getComments(); clearComment() } }); Pay attention to the data property.
