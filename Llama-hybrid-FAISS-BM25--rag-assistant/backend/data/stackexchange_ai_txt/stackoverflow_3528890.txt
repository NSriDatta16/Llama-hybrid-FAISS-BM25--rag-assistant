First of all, you need to find a suitable SMS gateway provider. Now if you were able to find one that suits you (price, network coverage, etc), they usually provide various examples how to use their messaging APIs and sometimes even provide helpful code snippets and tutorials on their own website. There are also bunch of high-level open source projects available in different languages and providers, but as there's no language currently specified, I'm not going to list these here...
