You can try to set different weights on each samples. For example, you can set 0.05 weight on those who bought other products, and 0.95 weight on those who bought product X. In sklearn , there's a sample_weight parameter which provides this method. In R, checkout this guide . If you're using XGBoost , here is a guide which shows how to deal with imbalanced class in XGBoost.
