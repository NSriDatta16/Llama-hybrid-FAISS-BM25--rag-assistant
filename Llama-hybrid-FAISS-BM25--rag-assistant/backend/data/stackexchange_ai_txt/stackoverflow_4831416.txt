There's no way to get every touch point when the user drags their finger. You'll only get the touch points of where the screen is being touched every time the input loop is cycled. You'll have to "predict" where the user touched for the points that you don't receive. Adding on to this, I ran into the same problem a while ago. UIResponder Delays
