You mean remote debug JBoss from Eclipse ? From Configuring Eclipse for Remote Debugging : Set the JAVA_OPTS variable as follows: set JAVA_OPTS= -Xdebug -Xnoagent -Xrunjdwp:transport=dt_socket,address=8787,server=y,suspend=n %JAVA_OPTS% or: JAVA_OPTS="-Xdebug -Xnoagent -Xrunjdwp:transport=dt_socket,address=8787,server=y,suspend=n $JAVA_OPTS" In the Debug frame, select the Remote Java Application node. In the Connection Properties, specify localhost as the Host and specify the Port as the port that was specified in the run batch script of the JBoss server, 8787 .
