I've only used LHS with Pandoc but Pandoc does specify that a bird-track delimited block quote is a paragraph level element, and paragraphs in Pandoc markdown (and regular markdown) do need a blank line before them. So the following should work... # Hello World # > main = putStrLn "hello, world"
