You have to cast it to an SPCalculatedField: SPFieldCalculated cf = (SPFieldCalculated)myItem.Fields["CIDandTitle"]; string value = cf.GetFieldValueForEdit(myItem["CIDandTitle"]); or string value = cf.GetFieldValueAsText(myItem["CIDandTitle"]);
