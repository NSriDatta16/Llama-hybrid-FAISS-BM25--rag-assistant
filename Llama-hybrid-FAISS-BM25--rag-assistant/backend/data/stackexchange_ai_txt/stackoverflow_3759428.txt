If you need to concat one after the other, as PatrickS mentionned bb.concat will do the job. If you need to insert values in the middle you'll need something like for each (var i : int in aa.reverse()) { bb.splice(1,0,i); } or, more elegantly, this, if you cannot change aa var insertingIndex : int = 2; for each ( var i : int in aa ) { bb.splice( insertingIndex, 0, i ); insertingIndex++; }
