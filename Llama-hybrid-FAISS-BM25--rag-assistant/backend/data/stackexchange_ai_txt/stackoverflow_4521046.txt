I use AudioQueues for this purpose. In your callback, get the input level like so: AudioQueueLevelMeterState meter[NUM_INPUT_CHANNELS]; UInt32 dataSize = sizeof(meter); AudioQueueGetProperty(aqInput, kAudioQueueProperty_CurrentLevelMeterDB, meter, &dataSize); // input 'level' is in meter.mAveragePower And simply don't write the audio into a file.
