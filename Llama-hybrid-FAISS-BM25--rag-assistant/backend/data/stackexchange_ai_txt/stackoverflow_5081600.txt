If you're trying to do it immediately, it may be because the image isn't fully loaded yet. Add a single load handler to the image using .one() . $("#img1").attr("src", "pic1.png").one('load',function() { alert($(this).width()); }); or in case the image was cached, you can try this: $("#img1").attr("src", "pic1.png").each(function() { if( this.complete ) { alert($(this).width()); } else { $(this).one('load',function() { alert($(this).width()); }); } }); As noted by @Humberto , you were using scr instead of the proper src .
