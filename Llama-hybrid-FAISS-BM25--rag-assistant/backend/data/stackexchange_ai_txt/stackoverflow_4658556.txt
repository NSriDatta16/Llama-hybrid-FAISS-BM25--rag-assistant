You can try to detect runtime which version of SDK using your application and depending on that prepare different branches. Like: @Override public boolean onKeyDown(int keyCode, KeyEvent event) { if(Build.VERSION.SDK_INT==Build.VERSION_CODES.CUPCAKE) //if it's 1.5 { if (keyCode == KeyEvent.KEYCODE_BACK && event.getRepeatCount() == 0) { // do something on back. return true; } } return super.onKeyDown(keyCode, event); }
