Ok So I found an elegant way through a gud soul on the python google group. It works like charm. Here's the code import logging import getopt import sys import logging.config def stop(m,handl): consoleHand=[h for h in m.handlers if h.__class__ is handl] print consoleHand if consoleHand: h1=consoleHand[0] h1.filter=lambda x: False logging.config.fileConfig('log.conf') my_log = logging.getLogger('netmap') args=sys.argv[1:] opt,arg = getopt.gnu_getopt(args,'v') l='' for o,a in opt: if o=='-v': l='verbose' stop(my_log,logging.handlers.RotatingFileHandler) if not l: stop(my_log,logging.StreamHandler) my_log.debug('Starting.....') my_log.warning('Unstable') my_log.error('FIles Missing') my_log.critical('BlowOut!!') The config file is still the same. I can now easily activate or deactivate handlers.
