You could probably catch them changing it in the SizeChanged event and check the WindowState , if its been set to Minimized then set it back Normal . Not the most elegant solution but should work. eg. private void myForm_SizeChanged(object sender, System.EventArgs e) { if (myForm.WindowState == FormWindowState.Minimized) { myForm.WindowState = FormWindowState.Normal; } }
