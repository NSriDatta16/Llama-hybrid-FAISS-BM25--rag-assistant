Since you have a list of links, you should be marking them up as a list (and not as paragraphs). Listamatic has a bunch of examples of how you can style lists of links , including a number that are vertical lists with each link being centred (which is what you appear to be after). It also has a tutorial which explains the principles . That part of the styling essentially boils down to "Set text-align: center on an element that is displaying as a block which contains the link text" (that could be the anchor itself (if you make it display as a block) or the list item containing it.
