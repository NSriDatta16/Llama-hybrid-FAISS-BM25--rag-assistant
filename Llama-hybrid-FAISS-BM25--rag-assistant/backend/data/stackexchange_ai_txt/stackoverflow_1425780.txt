I'm not a C# guy, but I would attack this with a Decorator in Java, I think. Assuming you have an interface for Product, then you can create a Decorator which manages the part number issue. class Product implements IProduct { public String getProductCode(); // etc } class ProductChannelDecorator implements IProduct { // constructor, like this in C#? public ProductChannelDecorator(IProduct product, Channel channel) { this.product = product; this.channel = channel; } public String getProductCode() { switch (this.channel) { case Channel.RETAIL: return this.decorated.getProductCode(); case Channel.TRADE: return retailToTradeTransformer(this.product.getProductCode()); // etc } } // etc }
