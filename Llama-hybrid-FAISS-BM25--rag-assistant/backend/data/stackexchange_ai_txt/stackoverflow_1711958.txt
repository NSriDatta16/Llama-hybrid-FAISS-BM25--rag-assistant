The command line is slightly incorrect... you need to specify the storage "my" after the '-ss' argument: makecert -sky "privatekey" -sk "MyCompany" -n "CN=MyCompany" -ss "my" -pe CertFile.cert After that change I was then able to locate the certificate using the MMC Certificate snap-in.
