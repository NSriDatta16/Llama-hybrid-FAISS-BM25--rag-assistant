To get the overlay div to not disturb the elements around it use absolute positioning. Change your CSS to: #draggable { border: 3px solid red; padding: 0.5em; width: 120px; position: absolute !important; left: 113px; top: 69px; } . Adjust left and top to taste. These 2 values will be ignored once dragging starts. Note that the absolute !important; is needed because the UI Draggable plugin automatically adds position:relative; and the docs don't indicate an easier way to override it.
