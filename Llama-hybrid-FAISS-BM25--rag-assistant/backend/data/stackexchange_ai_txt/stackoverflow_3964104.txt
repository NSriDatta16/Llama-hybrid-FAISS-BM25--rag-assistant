Doh! I think I can answer my own question by using SelectMany to condense the Collection of Collections into one collection like so: orgNames = context.Users.Where(u => u.LoweredUserName == userName.ToLower()) .Take(1) .SelectMany(u => u.Organisations) .Select(o => o.Name);
