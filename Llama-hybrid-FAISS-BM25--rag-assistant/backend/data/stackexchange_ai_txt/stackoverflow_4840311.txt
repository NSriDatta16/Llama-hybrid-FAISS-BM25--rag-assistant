Not sure using find is most elegant or idiomatic, but it is concise and wont throw an IndexOutOfBoundsException. def foo foo = ['bar', 'baz'] assert "bar" == foo?.find { true } foo = [] assert null == foo?.find { true } foo = null assert null == foo?.find { true } -- Update Groovy 1.8.1 you can simply use foo?.find() without the closure. It will return the first Groovy Truth element in the list or null if foo is null or the list is empty.
