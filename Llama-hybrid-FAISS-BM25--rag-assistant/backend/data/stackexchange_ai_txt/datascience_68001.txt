Machine learning is not needed to solve that problem. This deterministic code tests if numbers in an array are increasing: def strictly_increasing(nums): "Deterministically check for only increasing values" return all(x2 > x1 for x1, x2 in zip(nums, nums[1:])) assert strictly_increasing([1, 2, 3]) assert strictly_increasing([2, 4.2, 5.6]) assert not strictly_increasing([1, 1, 1]) assert not strictly_increasing([3, 2, 1])
