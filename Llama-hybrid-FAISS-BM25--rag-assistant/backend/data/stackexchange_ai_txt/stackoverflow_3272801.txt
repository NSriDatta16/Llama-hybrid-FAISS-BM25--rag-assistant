Get the jQuery plugin from: http://github.com/localhost/jquery-fieldselection Then in Your script tag play with it: $('input[@type="text"], textarea, p').keyup(update).mousedown(update).mousemove(update).mouseup(update); function update() { var range = $(this).getSelection(); $('#output').html(range.start + range.end + range.length); } This links might be helpfoul: snipplr.com/view/775/getselection/ - Support: IE6, Fx2, Opera9, Safari Cross browser getSelection(). stackoverflow.com/questions/717224/how-to-get-selected-text-in-textarea/717226#717226 - How to get selected text in textarea? stackoverflow.com/questions/2236982/jquery-select-text-and-add-span-to-it-in-an-paragraph
