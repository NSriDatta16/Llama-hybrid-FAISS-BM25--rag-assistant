Autocomplete data is stored in the registry in two places: HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\IntelliForms\Storage1 HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\IntelliForms\Storage2 Direct manipulation is not supported. Technically, the IE7+ API to do this is: rundll32.exe inetcpl.cpl,ClearMyTracksByProcess 48 But I'm not sure this is formally supported for public callers.
