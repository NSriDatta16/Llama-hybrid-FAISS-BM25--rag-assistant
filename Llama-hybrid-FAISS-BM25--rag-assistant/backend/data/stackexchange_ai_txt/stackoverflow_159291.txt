Management Studio Express cannot manage the following: SQL Server Analysis Services Integration Services Notification Services Reporting Services SQL Server Agent SQL Server 2005 Mobile Edition ( from this page. Look at the Note in the Overview section. ) These are all features that are not supported by SQL Server Express. Also note that the full version of SQL Management Studio is included with SQL Server 2005.
