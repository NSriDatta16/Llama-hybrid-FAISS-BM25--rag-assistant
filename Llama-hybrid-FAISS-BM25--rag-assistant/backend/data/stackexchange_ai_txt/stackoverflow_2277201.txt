I finally managed this via the following snippet: finalHandler = new StreamResult(new OutputStreamWriter(System.out)); // customHandler extends DefaultHandler CustomTransformerHandler customHandler = new CustomTransformerHandler( finalHandler); customHandler.startDocumentExplicitly(); InputStream is = null; while ((is = customHandler.createNextInputStream()) != null) { // multiple inputStream parsing XMLReader myReader = XMLReaderFactory.createXMLReader(); myReader.setContentHandler(customHandler); myReader.parse(new InputSource(is)); } customHandler.endDocumentExplicitly(); The important part was to leave the startDocument and endDocument methods empty. All other methods (characters, startElement, endElement) will be redirected to the finalHandler. The customHandler.createNextInputStream method returns null if all inputstreams are read.
