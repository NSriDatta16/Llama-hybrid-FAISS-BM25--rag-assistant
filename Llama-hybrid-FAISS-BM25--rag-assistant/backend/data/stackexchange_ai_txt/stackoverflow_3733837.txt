Removing duplicates from a QList

For years I have been using the following pattern to remove duplicates from an object of the C++ std::vector type: std::vector cont; std::sort(cont.begin(), cont.end()); cont.erase(std::unique(cont.begin(), cont.end()), cont.end()); Now I am wondering if the same paradigm is the one to be used with the Qt QList<> class, or if there is a more elegant way to do it.
