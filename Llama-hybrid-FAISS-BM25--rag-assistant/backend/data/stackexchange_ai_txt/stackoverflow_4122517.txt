Displaying list in Java as elegant as in Python

In Python it is pretty easy to display an iterable as comma separated list: >>> iterable = ["a", "b", "c"] >>> ", ".join(iterable) 'a, b, c' Is there a Java way that comes close to this conciseness? (Notice that there is no "," at the end.)
