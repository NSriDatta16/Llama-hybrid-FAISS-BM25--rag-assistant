There is plugin in Jquery UI called draggable through this you can change any element into draggable object Here is the link for the demo http://jqueryui.com/demos/draggable/ It is as simple as this $(function() { $("#draggable").draggable(); }); It is also possible to make it scrollable inside a div. Check the demo page
