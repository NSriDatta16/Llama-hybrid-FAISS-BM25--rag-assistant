Here is something that a co-worker figured out just using string::find with boost::filesystem::slash . I like this that it doesn't require iterate over the entire path breaking it up, but also using the path's OS-independent definition of the path separation character. Thanks Bodgan! boost::filesystem::path pop_front_directory(const boost::filesystem::path& path) { string::size_type pos = path.string().find(boost::filesystem::slash ::value); if (pos == string::npos) { return path; } else { return boost::filesystem::path(path.string().substr(pos+1)); } }
