To center the paragraph text in the middle of the page, you need to add the following CSS (in a style element or an external .css file): first example p { text-align: center; } This is probably not what you're after because it wont push the text away from the edges. If you would like the text to have a fluid width with some padding, use this CSS instead: second example p { padding: 0 100px; } You can of course center the text using both methods to make the text centered and stay away from the edges. If you would like a static width that stays centered, use this CSS instead of the padding: third example p { margin: 0 auto; width: 300px; }
