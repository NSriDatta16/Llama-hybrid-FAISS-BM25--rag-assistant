In the R rms package the lrm function outputs a large number of indexes including Nagelkerke (but not Cox-Snell). If you have $\LaTeX$ installed you can get the output automatically nicely typeset. require(rms) f The various statistics are defined in the binary logistic regression chapter in my online Regression Modeling Strategies course notes where you can also see lrm output printed.
