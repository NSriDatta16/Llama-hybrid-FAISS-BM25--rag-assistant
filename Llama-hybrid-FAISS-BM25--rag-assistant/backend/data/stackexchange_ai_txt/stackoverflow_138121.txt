It doesn't directly answer your question (so I'll probably be downgraded), but you may be interested in Jython . Jython is an implementation of the high-level, dynamic, object-oriented language Python written in 100% Pure Java, and seamlessly integrated with the Java platform. It thus allows you to run Python on any Java platform. Since you are coming from Java, Jython may help you leverage Python while still allowing you to use your Java knowledge.
