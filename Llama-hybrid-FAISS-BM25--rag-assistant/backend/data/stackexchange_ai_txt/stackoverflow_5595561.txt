I experienced a similar problem when using mkdirs(), however because running the command: mkdir one/two fails on Linux, then the method http://download.oracle.com/javase/1.4.2/docs/api/java/io/File.html#mkdirs() subsequently fails too. I guess this means there is no way to use mkdirs on Android? My (probably rather hacky) work-around was to create each necessary directory separately: String extStorageDirectory = Environment.getExternalStorageDirectory().toString(); new File(extStorageDirectory + "/one/").mkdirs(); new File(extStorageDirectory + "/one/two/).mkdirs();
