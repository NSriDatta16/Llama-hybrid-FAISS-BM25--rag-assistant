If you restructure your data a bit it is relatively straight forward: import matplotlib.pyplot as plt data = { "LR": [0.6, 0.7, 0.8, 0.7], "SVM": [0.7, 0.6, 0.8, 0.5], "Linear SVC": [0.8, 0.5, 0.7, 0.6], "ADABOOST": [0.7, 0.8, 0.6, 0.7], "DT": [0.6, 0.8, 0.5, 0.7] } subsets = [3, 5, 10, 20] for model in data: plt.plot(subsets, data[model]) You can add a legend using plt.legend , axis titles can be set using plt.xlabel / plt.ylabel .
