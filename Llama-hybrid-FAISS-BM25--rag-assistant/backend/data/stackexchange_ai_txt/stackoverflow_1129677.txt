You need to use javascript framework such as JQuery or ExtJS or just manual XmlHttpRequest to invoke a web service to obtain the value from the server side. And finally change the input control using Dhtml. I am assuming that there is no code behind in classic ASP. This article might help you : http://www.andrewrowland.com/article/display/consume-dot-net-web-service-with-jquery http://elegantcode.com/2008/12/02/calling-remote-aspnet-web-services-from-jquery/
