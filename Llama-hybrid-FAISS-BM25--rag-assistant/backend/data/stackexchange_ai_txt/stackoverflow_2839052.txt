Actually localStorage 'should' be capable of storing integers, strings and arrays. I have been working on a notepad app using localStorage and save all my data using object literals: var someData = { withvars: "and values", init: function() { // not sure if this works but it should. }, var1: { subvar1: "data", subvar2: "data2" } }; Then store it using JSON.stringify(): localStorage.setItem(varName, JSON.stringify(someData)); Then access it later with JSON.parse(): var dataBack = JSON.parse(localStorage.getItem("varName")); If you always use object literals then you will have less trouble keeping track of how to store and how to retrieve data from localStorage.
