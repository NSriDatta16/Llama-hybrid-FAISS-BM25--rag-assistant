There's no easy answer for this becasue we have no idea what the data is, the level of complexity of the data, the PC-side infrastructure, or really much of anything. You have several general options though. Here's a list of offhand ways, though there could probably be more: Have the user transfer the file manually via a storage card Have the user manually transfer the file via ActiveSync Have the device application create the file if it's not there Have a PC application transfer the file via RAPI Have the PC application transfer the file via a proprietary protocol Have the device retrieve the file from a server (web server, ftp, Azure cloud, etc) Write an ActiveSync data provider and let ActiveSync synchronize the entities
