The problem here is that attr_accessor creates a User.name method that override the UpcasedName.name method, so one solution would be using attr_writer : module UpcasedName def name @name.upcase end end class User attr_writer :name include UpcasedName end u = User.new u.name = 'john' puts u.name # outputs 'JOHN'
