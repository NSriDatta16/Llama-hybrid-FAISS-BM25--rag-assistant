A great resource for this is: Bayesian Model Averaging with BMS by Stefan Zeugner (2012) It is using the R-package BMS , more info can be found here: http://bms.zeugner.eu/ Two hands-on tutorials for reproducing real-world examples with the package can be found here: http://bms.zeugner.eu/tutorials/fls/ http://bms.zeugner.eu/tutorials/fat/ A more general motivational and current introduction to Bayesian methods is the following paper: The Time Has Come: Bayesian Methods for Data Analysis in the Organizational Sciences by John K. Kruschke, Herman Aguinis and Harry Joo
