return Session.CreateCriteria () .CreateAlias("BackupManagers", "bm", LeftOuterJoin) .Add(Restrictions.Disjunction() .Add(Restrictions.Eq("Manager", manager)) .Add(Restrictions.Eq("bm.Id", manager.Id))) .SetProjection(Projections.Distinct(Projections.ProjectionList() .Add(Projections.Property("Name"), "Name") .Add(Projections.Property("Id"), "Guid"))) .SetResultTransformer(Transformers.AliasToBean(typeof(NameGuidDto))) .List (); I threw a distinct in there because I'm not sure if it is possible to have backup and primary be the same person.
