this will dispense the need of specifying the onClick event for multiple controls. If this is your only goal, I suggest using jQuery to elegantly specify event handlers on multiple elements. This one of examples from jQuery tutorial that does exactly this: $(document).ready(function() { $("a[href*=/content/gallery]").click(function() { // do something with all links that point somewhere to /content/gallery }); })
