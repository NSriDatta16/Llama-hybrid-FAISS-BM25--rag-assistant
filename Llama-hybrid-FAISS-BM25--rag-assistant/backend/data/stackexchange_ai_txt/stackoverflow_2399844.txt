If an application could do that it would make a great denial-of-service attack on the machine. In particular Ctrl+Alt+Delete is the Secure Attention Sequence . Microsoft goes to great lengths to insure that when the user hits those keys, they switch to a secure desktop that they can be confident that the logon box is the real Windows logon and not a counterfeit. What you need to look at isn't functions that your application can call, but System Administration options that allow an Administrator to configure a machine for limited use. These exist, but it's more a question for Super User than for Stack Overflow. This should get you started http://msdn.microsoft.com/en-us/library/aa372139(VS.85).aspx
