Visual Studio 2010 has something that does almost everything you need View -> Other Windows -> Bookmark Window You can create virtual folders to organize your bookmarks. Click on anywhere in a code file, then click on "Toggle a bookmark on the current line", or CTRL-B,T if you have default mappings. You can then just mark the top of the file for each file. The bookmark manager will let you open all your files easily. Update A gentleman is working on updating his "Favorites Add-in" for Visual Studio 2010 -> Favorites Menu
