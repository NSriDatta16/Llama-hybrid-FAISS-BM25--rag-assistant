When you're using PHP 5.3 the solution can be quite elegant: $b = array_map(function($item) { return $item['id']; }, $arrayA); On PHP function mapCallback($item) { return $item['id']; } $b = array_map('mapCallback', $arrayA); or use create_function() to create a dynamic callback function (I won't show this, because it actually hurts my eyes ;-))
