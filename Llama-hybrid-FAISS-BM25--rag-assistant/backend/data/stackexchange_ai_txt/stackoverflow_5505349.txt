Rather than using archaic terminal codes, use tput instead which makes the code much easier to read and a lot harder to mess up: BLACK=$(tput setaf 0) RED=$(tput setaf 1) GREEN=$(tput setaf 2) YELLOW=$(tput setaf 3) LIME_YELLOW=$(tput setaf 190) POWDER_BLUE=$(tput setaf 153) BLUE=$(tput setaf 4) MAGENTA=$(tput setaf 5) CYAN=$(tput setaf 6) WHITE=$(tput setaf 7) BRIGHT=$(tput bold) NORMAL=$(tput sgr0) BLINK=$(tput blink) REVERSE=$(tput smso) UNDERLINE=$(tput smul) # Set Titlebar and Prompt TITLEBAR='\e]0;\h: ${PWD/$HOME/~}\a' PS1="${TITLEBAR}${WHITE}[${POWDER_BLUE}\u@\h${WHITE}]${NORMAL}$ " Setting the titlebar is optional. Just be sure to use ${NORMAL} at the end to turn off the color change.
