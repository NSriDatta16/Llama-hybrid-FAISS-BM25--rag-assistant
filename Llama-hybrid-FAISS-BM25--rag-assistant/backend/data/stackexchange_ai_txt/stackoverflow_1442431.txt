It is easily possible. First install the ODBC driver from http://www.ch-werner.de/sqliteodbc/ Then you can connect with e.g. DRIVER=SQLite3 ODBC Driver;Database=mydb.sqlite;LongNames=0;Timeout=1000;NoTXN=0;SyncPragma=NORMAL;StepAPI=0; I would suggest looking at ajaxed library which offers you a data abstraction for any database. Just configure the connection string and start working ...
