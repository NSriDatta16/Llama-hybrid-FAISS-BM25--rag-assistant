Quick, off-the-cuff solution: $regex_output = preg_split('/ff#/', $stuff); $output = Array(); foreach ($regex_output as $string) { while (strlen($string) > 12) { $output[] = substr($string, 0, 12); $string = substr($string, 12); } $output[] = $string; } I'm sure someone will come up with something much more elegant.
