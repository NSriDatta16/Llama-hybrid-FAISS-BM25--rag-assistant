I would first attempt to find all paragraphs via a global match: my @matches = ($string =~ m! (.*?) !sig); Then I would loop through, and replace any that contain your identifier: foreach(@matches) { #keep a copy for substitution below my $before = $_; #if the identifier is found, replace it if($_ =~ s!MY_IDENTIFIER!replacement text!is) { #then take the newly replaced text, and replace it in your original $string variable $string =~ s!$before!$_!is; } }
