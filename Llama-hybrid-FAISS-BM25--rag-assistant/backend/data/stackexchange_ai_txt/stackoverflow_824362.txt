Any changes of the loction (either window.location or document.location ) will cause a request on that new URL, if you’re not just changing the URL fragment. If you change the URL, you change the URL. Use server-side URL rewrite techniques like Apache’s mod_rewrite if you don’t like the URLs you are currently using.
