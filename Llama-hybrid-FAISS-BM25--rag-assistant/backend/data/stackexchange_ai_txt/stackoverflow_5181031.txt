File newxmlfile = new File("C:/new.xml"); You're trying to create your file on drive C. Android is a linux-based system, so there are no drives there. The storage devices can be mounted to root ("/") or any other folder. For your application /data/ folder is available. Try to write there. Also, you can try to write to external storage, but your progrma will need a permission to do that: This should be mentioned within your manifest file. Read more here .
