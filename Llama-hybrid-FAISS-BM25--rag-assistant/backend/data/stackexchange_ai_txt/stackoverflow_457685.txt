You could also use isolated storage . Really, you need to know if the application is ever shared across user profiles on the same machine. If not, you can get away with storing it in a config file. If it is, then you can use the registry (maintainable), or isolated storage (secure). EDIT: After your last update, you can save typed values in the app config. For example, If I want to save a color, I can save the value as a type System.Color, and the runtime will parse it for me. Check this article for more info and examples.
