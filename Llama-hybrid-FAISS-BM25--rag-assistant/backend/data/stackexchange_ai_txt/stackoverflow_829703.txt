You have to shrink & backup the log a several times to get the log file to reduce in size, this is because the the log file pages cannot be re-organized as data files pages can be, only truncated. For a more detailed explanation check this out. WARNING : Detaching the db & deleting the log file is dangerous! don't do this unless you'd like data loss
