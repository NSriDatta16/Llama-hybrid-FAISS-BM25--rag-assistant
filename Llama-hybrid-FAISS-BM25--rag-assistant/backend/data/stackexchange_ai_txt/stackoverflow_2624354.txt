The following code was the correct way of calling the above webservice. I've just only modified the $ser_params array. Now it has a sub array $url = 'http://www.webservicex.net/globalweather.asmx?WSDL'; $conn_params = array( 'proxy_host' => '10.2.0.1', 'proxy_port' => '6588' ); $ser_params = array ( 'GetWeather' => array ( "CityName" => "San Juan", "CountryName" => "Argentina" ) ); $client = new SoapClient($url, $conn_params); $result = $client->__soapCall('GetWeather', $ser_params); print_r ($result);
