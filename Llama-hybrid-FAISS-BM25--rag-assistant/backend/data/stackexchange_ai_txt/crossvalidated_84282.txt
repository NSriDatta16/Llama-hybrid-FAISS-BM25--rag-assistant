kernel for a (semi-) metric space

Let's say I have a metric space $(\mathcal{X}, d)$. Is there any kernel function that I can use with SVM? If we change the RBF kernel a little bit, we have $k(x,y) = e^{-d(x,y)^2}$. Is this a valid kernel? Additionally, what if $d$ is semi-metric?
