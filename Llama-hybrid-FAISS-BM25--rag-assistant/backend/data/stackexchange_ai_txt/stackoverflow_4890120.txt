If you use EclipseLink JAXB (MOXy) then you can leverage the @XmlPath extension for this (I'm the MOXy tech lead): import javax.xml.bind.annotation.XmlAccessType; import javax.xml.bind.annotation.XmlAccessorType; import javax.xml.bind.annotation.XmlRootElement; import org.eclipse.persistence.oxm.annotations.XmlPath; @XmlRootElement @XmlAccessorType(XmlAccessType.FIELD) public class SlideText extends Slide { @XmlPath("layout/text/text()") private String text; } http://bdoughan.blogspot.com/2010/09/xpath-based-mapping-geocode-example.html Using standard JAXB you could leverage an XmlAdapter: http://bdoughan.blogspot.com/2010/07/xmladapter-jaxbs-secret-weapon.html
