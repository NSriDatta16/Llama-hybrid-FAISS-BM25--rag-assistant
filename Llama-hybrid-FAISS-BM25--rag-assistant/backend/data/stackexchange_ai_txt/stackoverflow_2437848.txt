The solution is to include the red div inside the blue div in the DOM hierarchy. And then set the droppable of the red div to greedy. Check the documentation on the droppable plugin If true, will prevent event propagation on nested droppables. $("#c").draggable(); $("#b").droppable({greedy: true}); $("#a").droppable({ drop: function(event,ui) { alert('ss'); } });
