If you already have some experience with similar programming - you could pick up a cookbook for the respective language you're trying to learn. A cookbook has to the point answers which will get you up and running in no time. I did the same thing when I wanted to pick up C#.Net after having done VB 6 previously. However, if you want to spend some more time picking up things and getting to the crux of it all, I'd recommend a dummies book. And yes, there is a clock control which you can drag and drop and program for the _tick event once you have enabled it and set its interval.
