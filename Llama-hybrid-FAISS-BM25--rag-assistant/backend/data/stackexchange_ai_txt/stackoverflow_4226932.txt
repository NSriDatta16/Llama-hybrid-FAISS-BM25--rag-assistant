You can have multiple XML representations for an object model by leveraging an extension in EclipseLink JAXB (MOXy) . This extension allows you to provide additional mappings via an XML representation. For more information see: http://wiki.eclipse.org/EclipseLink/Examples/MOXy/EclipseLink-OXM.XML
