Simple regex way: String input = "123 456 789"; String output = input.replaceAll("\\b[1-9]\\d*", "0$0"); System.out.println(output); //prints "0123 0456 0789" This replaces all "words" consisting of any digit but beginning with 1-9 (not 0) with the same word, prefixed by 0. I agree with the other assessments that you should just make your parsing less fragile, using something like String.split() instead of requiring fixed widths.
