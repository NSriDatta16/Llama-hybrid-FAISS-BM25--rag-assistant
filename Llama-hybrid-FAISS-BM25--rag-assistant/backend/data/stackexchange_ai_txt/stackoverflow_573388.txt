Collada is an open XML based format for 3d models owned by the Khronos group(OpenGL standards body) From the Collada.org FAQ: The COLLADA 1.4.x feature set includes: Mesh geometry Transform hierarchy (rotation, translation, shear, scale, matrix) Effects Shaders (Cg, GLSL, GLES) Materials Textures Lights Cameras Skinning Animation Physics (rigid bodies, constraints, rag dolls, collision, volumes) Instantiation Techniques Multirepresentations Assets User data
