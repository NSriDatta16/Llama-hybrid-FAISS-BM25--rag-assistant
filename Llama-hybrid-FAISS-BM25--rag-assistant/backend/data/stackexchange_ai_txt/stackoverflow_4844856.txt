Ajax requests only work on urls from the same host as the webpage it is called from. Local files don't work because it imposes a security risk, because any site could load a local file and send it to the host. A quick google search gave me this link: http://docs.phonegap.com/phonegap_file_file.md.html#File (Android & BB only) http://docs.phonegap.com/phonegap_storage_storage.md.html#Storage (Android, BB, iphone)
