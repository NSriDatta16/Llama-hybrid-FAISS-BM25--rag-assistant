Assuming datahorageracao is of type DateTime.Two different ways you can write LINQ code first using lambda expressions and second using query expression syntax either way works - //Lambda Expression way get distinct codigobanco... var codigobancolist =Context.headerarquivo.Where(c=>c.datahorageracao.Month ==12 && c.datahorageracao.Year==2010).Select(c=>c.codigobanco).Distinct(); //Query Expression way to get actual data... var data= from b in bancos where b.codigobanco.Contains(codigobancolist) select b;
