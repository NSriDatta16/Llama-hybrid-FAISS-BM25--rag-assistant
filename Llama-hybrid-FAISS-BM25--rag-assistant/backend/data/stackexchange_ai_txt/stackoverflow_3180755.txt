Change the line where you set the contents of the paragraph from priceDesc = "Enter price per month"; to priceDesc.innerHTML = "Enter price per month"; Currently, you are just changing the priceDesc variable to contain a string instead the paragraph node. Setting the innerHTML attribute of a node changes the html contained inside of it. :D
