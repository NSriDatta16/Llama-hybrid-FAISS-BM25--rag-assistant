This is incorrect. CSS rules work the same way, whether they are defined in external style sheets or in inline CSS. There are some mistakes one can make when embedding external style sheets, for example using a title attribute in the link tag, or using the wrong media attribute. Maybe one of these things causes your external style sheet not to work. In that case, show the code you are using to embed it.
