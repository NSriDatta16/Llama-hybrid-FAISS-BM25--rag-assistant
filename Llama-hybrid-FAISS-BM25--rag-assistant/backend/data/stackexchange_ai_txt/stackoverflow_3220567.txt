It's 2 steps: "the ungrouped maximum of the grouped averages" You can expand this as needed which shows how to apply an aggregate on top of an aggregate SELECT MAX(AveragePrice) AS MaxAveragePrice FROM ( select avg(UnitPrice) AS AveragePrice, ProductID from OrderDetails group by ProductID ) foo Or with CTE ;WITH AvgStuff AS ( select avg(UnitPrice) AS AveragePrice from OrderDetails group by ProductID ) SELECT MAX(AveragePrice) AS MaxAveragePrice FROM AvgStuff
