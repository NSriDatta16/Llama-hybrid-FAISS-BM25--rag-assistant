Make sure you set the CellLayout.BackgroundUse=true. If you don't the Background color is ignored. And since you use ASP.NET the cellLayouts will generate a css, so you must add the new cloned CellLayouts to the collection on CellLayouts: Gantt_ASP.Gantt.Grid.CellLayouts.Add(cl);
