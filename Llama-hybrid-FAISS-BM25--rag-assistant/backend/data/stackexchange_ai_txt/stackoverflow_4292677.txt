If you want to manipulate the type, you can use patter matching(among other things, but I find this the most elegant). It's essentially syntactical sugar for Match with. For example: let identify_formula=function And(l) -> 0 |Or(l) -> 1 |Literal(b) -> 2 | Variable(v) -> 3;; For a less trivial example of how to manipulate types: type l=Cons of int*l | Nil;; let rec sum_l=function Cons(hd,tl)->hd+ sum_l(tl) | Nil-> 0;; sum_l(Cons(1,Cons(2,Cons(3,Nil))));; (*r6*)
