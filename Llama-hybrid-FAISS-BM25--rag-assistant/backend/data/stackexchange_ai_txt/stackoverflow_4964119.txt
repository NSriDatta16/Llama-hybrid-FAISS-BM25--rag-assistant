For those who would face the same issue : Context on server A : ssh user_foo@serverB "ssh user_bar@serverC" To get it working in command line or in the shell, the ssh agent shall be forwarded, eg using the ssh option "A" (see ssh man page), like : ssh -A user_foo@serverB "ssh user_bar@serverC"
