Two simple debugging tips: Inspect the real SQL code you are sending to the server. Add this line to your PHP code: echo htmlspecialchars($sql); ... find the query in the browser and paste it in your MySQL client (HeidiSQL, PHPMyAdmin or whatever). Pay close attention to the possible error messages. Check error conditions in all functions: $con = mysql_connect(...); if( !$con ){ die( htmlspecialchars(mysql_error()); } ... $res = mysql_query(...); if( !$res ){ die( htmlspecialchars(mysql_error()); }
