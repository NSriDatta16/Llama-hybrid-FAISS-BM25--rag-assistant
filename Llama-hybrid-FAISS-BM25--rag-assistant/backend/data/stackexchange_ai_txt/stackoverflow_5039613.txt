This is not very elegant but it works nonetheless: the rowspan attribute has to switch from 3 to 1 to 3 etc., right? So just calculate it by subtracting the current rowspan value from the number 4: 4 - 1 = 3 4 - 3 = 1 Tested here: http://jsfiddle.net/xpABa/ $(document).ready(function(){ $(".toggle").click(function() { $(this).parent().nextUntil("tr.master").toggleClass("toggled"); $(this).attr("rowspan", 4 - $(this).attr("rowspan")); }); }); Good luck! Edit: I just noticed the second part of the table has only two rows. So instead of the number four, calculate the number of rows ($(this).somethingsomething.size()) and add 1.
