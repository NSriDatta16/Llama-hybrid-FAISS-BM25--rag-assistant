Shell folders usually implement the IStorage interface, so this is pretty simple. For example, the following creates a folder named "abcd" on the desktop: CComPtr pDesktop; HRESULT hr = SHGetDesktopFolder(&pDesktop); if (FAILED(hr)) return; CComQIPtr pStorage(pDesktop); if (!pStorage) return; CComPtr dummy; hr = pStorage->CreateStorage(L"abcd", STGM_FAILIFTHERE, 0, 0, &dummy);
