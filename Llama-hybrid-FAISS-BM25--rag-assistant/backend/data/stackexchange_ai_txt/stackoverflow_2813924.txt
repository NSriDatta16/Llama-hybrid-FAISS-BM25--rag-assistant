I'm not sure if there's a way to do it for arbitrary DOM elements like paragraphs, but for textarea elements, I believe you need to use the selectionStart and selectionEnd properties and specify where to start and end. var textarea = document.getElementsByTagName('textarea')[0]; textarea.selectionStart = 10; textarea.selectionEnd = 15; Hope this helps!
