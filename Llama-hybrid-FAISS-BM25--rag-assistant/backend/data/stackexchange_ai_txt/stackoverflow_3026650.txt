Well, you can enumerate all object properties using the for...in statement, for example: if (typeof Object.keys != 'function') { Object.keys = function (obj) { var result = []; for (var prop in obj) { if (Object.prototype.hasOwnProperty.call(obj, prop)) { result.push(prop); } } return result; }; } alert(Object.keys({foo: 1, bar: 2})); // "foo, bar"; But for debugging purposes I would highly encourage you to get a real debugger, like Firebug . With the Console API you can easily examine objects on the fly.
