The most important tool for Ruby work is RubyGems. Ideally users should be able to install gems themselves. RubyGems can be configured to install gems in the user's home directory instead of system wide, which could work well on multi-user machines. Of course a good text editor, revision control, and Firebug in Firefox or Dragonfly in Opera are also essential.
