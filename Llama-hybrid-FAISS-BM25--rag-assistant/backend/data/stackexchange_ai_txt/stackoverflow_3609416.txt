I've done this kind of project earlier. For the graphs I'd recommend Google Visualization API For storage I would recommend a Round Robin scheme. There is a Round Robin database engine called RRDtool , but you can also implement a Task (or table partition) in your MySQL database that would speed queries once the stored data gets too big to be queried fast enough, by deleting (or separating) old data.
