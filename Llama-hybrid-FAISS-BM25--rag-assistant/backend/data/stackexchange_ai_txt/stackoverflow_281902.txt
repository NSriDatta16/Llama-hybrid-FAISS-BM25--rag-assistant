Depending on the speed in which you wish to deploy your solution, one method is to just use the HTML table tag, store all your data in tables using style markup, and then use PHP header's option to force the browser to save is as a .xls file. For proof of concept, copy this code into notepad, save as .xls, and then open with Excel: Column 1 Column 2 Answer 1 Answer 2 Answer 3 with 2 columns Its not the most elegant solution, but it will absolutely suit your needs.
