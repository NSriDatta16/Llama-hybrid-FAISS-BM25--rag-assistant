The easiest way to do this would be to use the .css rule of the jQuery javascript library: http://api.jquery.com/css/ (See the example about a quarter down: To change the color of any paragraph to red on mouseover event. ) So using this you'd use this javascript on clicking the button/link: $('div#contact_form').css('display', 'none');
