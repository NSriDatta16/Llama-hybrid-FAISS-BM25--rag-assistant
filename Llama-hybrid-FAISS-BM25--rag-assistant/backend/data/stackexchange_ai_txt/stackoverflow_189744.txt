Unless I misunderstand the question, if text is an instance of the Infragistics WebTextEdit, you should just be able to do: text.setValue(box[select].value) Or if text is the underlying input control, but 'id' is the ID of it, var edit = igedit_getById(id) edit.setValue(box[select].value) See the WebTextEdit CSOM for more.
