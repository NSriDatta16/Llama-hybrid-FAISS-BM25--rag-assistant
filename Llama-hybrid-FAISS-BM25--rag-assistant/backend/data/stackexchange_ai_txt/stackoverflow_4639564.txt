I think you should use SWFObject. You can download the js file and the expressinstall file from here: http://code.google.com/p/swfobject/downloads/detail?name=swfobject_2_2.zip&can=2&q= You need the js file for embedding and the expressinstall.swf automaticly updates the users Flash Player if it is older then required. You can set the minimum player version, add custom variables to Flash and also you can set the parameters and attributes of your Flash movie. For more details see the swfobject docs. I modified your code, so if you put the swfobject.js file to the js folder and the expressinstall.swf file to your media folder and replace your code with the one below, it should work fine. I tested it on FF3 and Chrome, it works for me.
