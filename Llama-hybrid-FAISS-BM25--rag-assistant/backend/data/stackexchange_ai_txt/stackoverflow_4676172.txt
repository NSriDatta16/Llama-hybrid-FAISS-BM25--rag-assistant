Thanks to @Thechoyce for helping me. Simply renaming the send action to "List" fixed the problem. They need to both be the same to overload. [HttpGet] [Url("organizations/{organizationId?}/alerts/", Order = 1)] public JsonResult List(Guid? organizationId) { ... return Json(data, JsonRequestBehavior.AllowGet); } [HttpPost] [Url("organizations/{organizationId?}/alerts/", Order = 2)] public JsonResult List(Guid? organizationId, string message) { ... return Json(data, JsonRequestBehavior.AllowGet); }
