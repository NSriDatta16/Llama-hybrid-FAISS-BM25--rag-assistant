If you are including the annotated jQuery file in your source solely for intellisense, I recommend leveraging preprocessor directives to remove it from your view when you compile. Ala: Then later in your code you can really reference jQuery. This is handy when using the Google AJAX Libraries API , because you get all the benefits Google provides you, plus intellisense. Here is a sample of using the Libraries API: google.load("jquery", "1.3.2", { uncompressed: false });
