The Cobertura plugin for Maven can be downloaded from the Maven central repository , however you shouldn't need to download the plugin directly, it is sufficient to declare the plugin in your POM, Maven will download it automatically from the central repository (assuming you have an internet connection). The configuration would be something like this: ... org.codehaus.mojo cobertura-maven-plugin 2.3
