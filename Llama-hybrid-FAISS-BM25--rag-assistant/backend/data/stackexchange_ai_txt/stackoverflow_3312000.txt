If your maintenance plan has to appear below the "Maintenance Plans" node in SSMS, I can't help. Otherwise... Maintenance Plans in SQL Server 2008 are usual SSIS packages. You can use BIDS (Business Intelligence Developer Studio) and create your maintenance package. After that, you can store the package where ever you want and call it by SQL Server Agent. Hope this helps a bit, Flo
