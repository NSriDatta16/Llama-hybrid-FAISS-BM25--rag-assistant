You have a couple of options: Use the query string as you suggested. This makes sense from a web development perspective, but we aren't in a web development perspective. Use a static variable. This is probably the simplest. Just declare another class with a static property, and you can share data this way. The only concern with this approach is thread-safety. Use Isolated Storage.
