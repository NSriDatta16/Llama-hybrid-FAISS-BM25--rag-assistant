AFAIK, JEditorPane is a very primitive HTML component: it is confused by CSS and knows nothing about JS. I doubt you will find a "small + free" class doing better, HTML parsing and displaying isn't a simple business, even less today. Perhaps it is better to let the big names in the business to take care of this task, ie. using Internet Explorer or Mozilla components (depending on what is available, etc.): JDIC: Embedding a Web browser in Java .
