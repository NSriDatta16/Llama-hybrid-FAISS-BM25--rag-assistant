You can save the data for using in future: 1) you can save the simple data in sharedpreferences. 2) you can also save the data in database To save the last data you have to save the data in activity's onStop() method. Because before exiting from the application the onStop() method is called of the current activity. This link Data Storage in Android will help you a lot.
