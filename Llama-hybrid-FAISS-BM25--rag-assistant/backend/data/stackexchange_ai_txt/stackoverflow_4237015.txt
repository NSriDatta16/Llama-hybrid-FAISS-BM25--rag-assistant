Looks like simply referring to the image file name when trying to read it was not enough, and I had to call getFilesDir() to get the path of the file storage. Below is the code I used: String path = context.getFilesDir().toString(); String fileName = cursor.getString(cursor.getColumnIndex(DbAdapter.KEY_PRODUCT_ID)); if (fileName != null && !fileName.equals("")) { Bitmap bMap = BitmapFactory.decodeFile(path + "/" + fileName); if (bMap != null) { thumbnail.setImageBitmap(bMap); } }
