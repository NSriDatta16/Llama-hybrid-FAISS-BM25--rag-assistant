My guess is that part of the problem is that the web server (e.g. Tomcat) is using the wrong character set when translating the request parameters into UTF-16. If you are using Tomcat, read the Character Encoding FAQ , and check each of the issues listed. (Pay particular attention to the bit that talks about the effects of filters and valves ...)
