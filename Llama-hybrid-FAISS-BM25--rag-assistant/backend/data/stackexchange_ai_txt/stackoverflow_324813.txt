You could create your own Replace extension method that takes a IEnumberable parameter. Then loop through the iterator and use the replace method. Then you could call it like this .Replace(new string[] {"Matt", "Joanne", "Robert"}, "") I think this should do it public static string Replace(this string s, IEnumerable strings, string replacementstring) { foreach (var s1 in strings) { s = s.Replace(s1, replacementstring); } return s; }
