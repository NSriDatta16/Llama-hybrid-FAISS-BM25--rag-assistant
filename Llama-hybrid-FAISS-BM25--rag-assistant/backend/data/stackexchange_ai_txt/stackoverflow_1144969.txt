By default an anchor tag does not inherit attributes like color if an href attribute is present. Check out the difference between these two on a page: test test The following link is to the w3 c: http://www.w3.org/TR/html401/struct/links.html#h-12.2 User agents generally render links in such a way as to make them obvious to users (underlining, reverse video, etc.). The exact rendering depends on the user agent. Rendering may vary according to whether the user has already visited the link or not. ..... Usually, the contents of A are not rendered in any special way when A defines an anchor only.
