One way: "rxrx".gsub(/(x)/) { |s| s.upcase } The only reason I suggest this instead of just normal substitution ( gsub("x", "X") ) is because it'd work well if there were more than one character you wanted to make uppercase: "rxbrx".gsub(/([xb])/) { |s| s.upcase }
