To expand on Hallagan's answer, here are several methods that might help you: File.ReadAllLines(path) - reads a text file into a string array File.WriteAllLines(path, array) - writes a string array into a text file string.PadRight(totalLenght, paddingChar) - pads a string from the right using a specified char.
