It depends what you want to do with this XML information. If you're targeting the iOS Plattform, Apple encourages you to do the XML parsing inline, means that you should not keep complete dom trees in memory. If you would like to work with a DOM tree after your parsing you sure can use e.g. touchXML or we worked with KissXML (http://code.google.com/p/kissxml/) which became pretty handy for us.
