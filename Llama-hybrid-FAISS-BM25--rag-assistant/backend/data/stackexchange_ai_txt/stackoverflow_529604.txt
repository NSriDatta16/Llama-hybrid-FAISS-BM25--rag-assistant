Code page 0 is CP_ACP, current Windows ANSI code page. From Windows.pas: {$EXTERNALSYM CP_ACP} CP_ACP = 0; { default to ANSI code page } From MSDN : CP_ACP The current system Windows ANSI code page. This value can be different on different computers, even on the same network. It can be changed on the same computer, leading to stored data becoming irrecoverably corrupted. This value is only intended for temporary use and permanent storage should be done using UTF-16 or UTF-8 if possible.
