you drag the icon into your Applications folder. It is really a folder with a name ending in .app . The OS treats that as a single unit. Inside of the "folder" is the application distribution and metadata to tell the OS how to run it. Normally, systemwide components are installed on first-run. You sometimes find package ( .pkg ) bundles that work exactly the same was as a .msi .
