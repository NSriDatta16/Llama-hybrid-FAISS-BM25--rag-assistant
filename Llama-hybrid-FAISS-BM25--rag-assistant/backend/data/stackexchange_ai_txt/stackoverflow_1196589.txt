I've had sucess recently using reflection on output assembly files to generate code. Try using something like this in a console app you can call from your post-build process: Assembly assembly = Assembly.LoadFile("FileName"); Type myEnumType = assembly.GetType("EnumName"); foreach(MemberInfo mi in myEnumType.GetMembers().Where(m => m.MemberType == MemberTypes.Field)) Console.WriteLine(mi.Name);
