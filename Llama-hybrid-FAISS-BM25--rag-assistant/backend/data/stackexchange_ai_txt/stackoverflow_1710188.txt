Add a new Copy Files Build Task to your target. Right Click (Control Click) on the target, Add -> New Build Phase -> New Copy Files Build Phase. In the Dialog select the destination that you want the files to be copied. This will create the phase under the target. Drag the files you wish to copy to the phase.
