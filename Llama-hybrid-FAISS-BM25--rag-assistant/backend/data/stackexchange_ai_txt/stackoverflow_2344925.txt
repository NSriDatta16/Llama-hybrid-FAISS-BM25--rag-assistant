There are different designer experiences, and totally different framework classes depending on whether you select Framework 3.5 or 4.0 In order to create a sequential workflow in WF 4.0, use the Sequence activity (in the Control Flow) toolbox group, by dragging that into a new Activity. In order to consume a WCF web service within your sequence, the easiest way may be using add service reference (right-click project, add service reference, build).
