For the specific Mozilla string you mention, see the user-agent string documentation . The U stands for "strong (encryption) security"; rv/xyz is the version number and date of the Gecko rendering engine. If you're not doing this for the learning process, there are several good libraries out there for browser detection, including PHP's native get_browser() .
