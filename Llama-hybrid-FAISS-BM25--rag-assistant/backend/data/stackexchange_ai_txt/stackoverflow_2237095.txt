If the content is not a paragraph, then it is not semantic to mark it as such. If your end marker is designed to be read, then it is debatable whether or not it is actually a paragraph or not. If it's not designed to be read, then it shouldn't be marked up with reading-oriented tags like , but instead should be in a span or div with a stylesheet applied to make the font weight bold.
