Yes you can, but you have to apply the same attributes as the linq2sql generated property i.e. [Association(Name="Test_TestData", Storage="_TestDatas", ThisKey="SomeId", OtherKey="OtherId")] public System.Data.Linq.EntitySet MyTestDatas { get { return this.TestDatas; } } TestDatas being the original relation. Update: A sample query I ran: var context = new DataClasses1DataContext(); var tests = from d in context.Tests where d.MyTestDatas.Any(md=>md.MyId == 2) select new { SomeId = d.SomeId, SomeData = d.SomeData, Tests = d.MyTestDatas }; foreach (var test in tests) { var data = test.Tests.ToList(); }
