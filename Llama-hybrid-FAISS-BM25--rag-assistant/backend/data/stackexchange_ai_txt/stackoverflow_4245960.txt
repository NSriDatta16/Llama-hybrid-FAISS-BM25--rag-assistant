Adding multiple fonts to UIAppFonts overrides each other

I'm trying to add some custom fonts to an iPhone app through UIAppFonts in Info.plist . I can reach both fontnames by code, i.e. MyFont-Bold and MyFont-Medium . My problem is that the last font in UIAppFonts array overrides the other one, making both MyFont-Bold and MyFont-Medium render out MyFont-Bold if this is the last entry in the plist array. By dragging MyFont-Bold as the first entry makes both font-names render MyFont-Medium. The property list excerpt: UIAppFonts item 0 MyFont-Medium item 1 MyFont-Bold I'm calling the font with: UIFont* applicationFontBold = [UIFont fontWithName:@"MyFont-Bold" size:[UIFont buttonFontSize]]; UIFont* applicationFont = [UIFont fontWithName:@"MyFont-Medium" size:[UIFont buttonFontSize]]; Anyone tried adding multiple fonts to one font-family?
