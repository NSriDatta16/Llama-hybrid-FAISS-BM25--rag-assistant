If you would like to use you XML like a data container (database like) the SimpleStorage is probably what you are looking for. You can find it here: http://www.cromis.net/blog/downloads/simplestorage/ It uses OmniXML as an XML parser. With SimpleStorage it is very easy to query and manipulate data inside XML.
