Assuming you're using Doctrine, you need to override the findOneBy*** methods in your InfoTable class to join data on retrieval Class InfoTable extends Doctrine_Table { [...] public function findOneById($id) { $q = $this->createQuery('i') ->leftJoin('i.User u') ->where('i.id = ?', $id); return $q->fetchOne(); } } Doctrine will handle and hydrate the associated object, and save one query.
