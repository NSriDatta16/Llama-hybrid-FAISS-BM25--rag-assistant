The concatenate function can connect 2 neural networks. You just have to be careful with the dimensions between them. Here is a pseudo-code: from tensorflow.keras.layers import Concatenate, Dense, Input from tensorflow.keras.models import Model import numpy as np input_a = ... layer_a_1 = ... layer_a_n = ... output_a = Dense(1)(layer_a_n) input_b = Input(shape=(28,)) concat_2 = Concatenate()([output_a, input_b]) layer_b_1 = Dense(1)(concat_2) ... layer_b_n = ... output_b = Dense(1)(layer_b_n) model = Model(inputs=[input_a], outputs=[output_b]) model.compile(loss='mse', optimizer='sgd') hist = model.fit([input_a], output_b, epochs=500)
