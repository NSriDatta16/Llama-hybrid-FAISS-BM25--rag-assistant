What you really want to be doing is parsing the HTML properly. Use the BeautifulSoup library - it's wonderful at doing so. Sample code: import urllib2, BeautifulSoup opener = urllib2.build_opener() opener.addheaders[0] = ('User-agent', 'Mozilla/5.1') response = opener.open('http://www.whitepages.com/carrier_lookup?carrier=other&number_0=1112223333&response=1').read() bs = BeautifulSoup.BeautifulSoup(response) print bs.findAll('div', attrs={'class': 'carrier_result'})[0].next.strip()
