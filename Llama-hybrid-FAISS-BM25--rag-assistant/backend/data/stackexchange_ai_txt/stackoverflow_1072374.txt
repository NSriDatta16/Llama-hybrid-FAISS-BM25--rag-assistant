It means your site is being accessed through the JVM on someones machine. It could be a crawler or simply someone scraping data. You can replicate the user-agent string using the HttpURLConnection class. Here is a sample: import java.net.*; public class Request { public static void main(String[] args) { try { URL url=new URL("http://google.ca"); HttpURLConnection con=(HttpURLConnection)url.openConnection(); con.connect(); System.out.println(con.getResponseCode()); } catch (Exception e) { e.printStackTrace(); } } }
