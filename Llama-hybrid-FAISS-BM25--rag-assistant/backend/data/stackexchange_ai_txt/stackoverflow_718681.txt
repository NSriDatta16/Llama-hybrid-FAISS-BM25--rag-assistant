To name a few items: Large developer community (not saying that python doesn't) Convention over configuration rocks Can be used with JRuby/Warbler to make rails run on Java Application servers. Very important when trying to drag Enterprise into the 21st century (Django can be used with Jython to work with application servers, but is less mature). MVC framework helps keep your application structured. Scaffolding and generators to get your project moving. Rails doesn't try to be the only way to do web applications, that's not the intent. It fulfills a specific set of needs.
