You could use SharpNLP , which does phrase-structure parsing. Link Grammar doesn't have PHP bindings (or C# bindings). You'd have to write your own bindings. And Link Grammar is written in C, so if you can't put your own C code on a shared host, then you can't use Link Grammar.
