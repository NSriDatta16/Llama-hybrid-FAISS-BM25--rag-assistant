You should use joins to acheive this: @shops = Shop.published.search params[:keyword], { :conditions => conditions, :star => true, :group_by => :city_id, :group_function => :attr, :page => params[:page] }.merge(:order => :rating_average, :sort_mode => :desc) You should also add cities. or shops. before columns to specify which table's column.
