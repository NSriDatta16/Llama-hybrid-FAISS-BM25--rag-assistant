Java Web Start can suggest desktop icons & menu items for a application. I would recommend doing both, since from memory some OS' will disallow one or the other. Alternately, JWS can leave the choice to the user, with the assistance of the javax.jnlp.IntegrationService . JWS is available for platforms that run Java, and is supported by Oracle.
