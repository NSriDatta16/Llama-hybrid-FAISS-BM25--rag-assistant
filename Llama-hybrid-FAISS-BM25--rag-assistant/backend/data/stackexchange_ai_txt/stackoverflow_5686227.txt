A line of text in a paragraph will always stretch to \hsize (the value of \hsize at the end of that paragraph). Thus, use for example: \hbox to \hsize { \vbox to 3 true in { \hsize=3 true in \strut aaa aaaaa aaaaa aaa aaaaa aaaa aaa aaaa aaa. aaa aaaaa aaaaa aaa aaaaa aaaa aaa aaaa aaa aaa aaaaa aaaaa aaa aaaaa aaaa aaa aaaa aaa. aaa aaaaa aaaaa aaa aaaaa aaaa aaa aaaa aaa.\vfil } \hfil \vbox to 3 true in { \hsize=3 true in \strut bbb bbbbb bbbbb bbb bbbbb bbbb bbb bbbb bbb. bbb bbbbb bbbbb bbb bbbbb bbbb bbb bbbb bbb bbb bbbbb bbbbb bbb bbbbb bbbb bbb bbbb bbb. bbb bbbbb bbbbb bbb bbbbb bbbb bbb bbbb bbb.\vfil } }
