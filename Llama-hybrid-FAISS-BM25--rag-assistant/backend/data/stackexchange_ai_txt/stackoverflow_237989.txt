Here's one example from a real project. I have getter functions like this: bool getValue(wxString key, wxString& value); bool getValue(wxString key, int& value); bool getValue(wxString key, double& value); bool getValue(wxString key, bool& value); bool getValue(wxString key, StorageGranularity& value); bool getValue(wxString key, std::vector & value); And then a variant with the 'default' value. It returns the value for key if it exists, or default value if it doesn't. Template saved me from having to create 6 new functions myself. template T get(wxString key, const T& defaultValue) { T temp; if (getValue(key, temp)) return temp; else return defaultValue; }
