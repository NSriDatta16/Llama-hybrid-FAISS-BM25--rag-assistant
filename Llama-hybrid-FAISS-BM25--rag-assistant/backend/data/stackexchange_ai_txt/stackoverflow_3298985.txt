ABAddressBook is a C API. If you want to code against a nice Objective-C API, use Google Toolbox for Mac: http://code.google.com/p/google-toolbox-for-mac (it's actually for iPhone, too!) Using GTM, you access the attributes like this: GTMABPerson* person; [person setValue:@"Company Name" forProperty:kABPersonOrganizationProperty]; [person setValue:(NSNumber*)kABPersonKindOrganization forProperty:kABPersonKindProperty];
