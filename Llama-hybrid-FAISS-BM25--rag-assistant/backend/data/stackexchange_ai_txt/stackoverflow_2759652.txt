Probably more elegant ways to do this, but it works. currency_in = 'USD' currency_out = 'NOK' import urllib2 req = urllib2.urlopen('http://finance.yahoo.com/d/quotes.csv?e=.csv&f=sl1d1t1&s='+currency_in+currency_out+'=X') result = req.read() # result = "USDNOK=X",5.9423,"5/3/2010","12:39pm" Then you can split() the result for the modifier.
