You can check the HTTP_USER_AGENT server variable. The user agent transfered by IE contains MSIE if(strpos($_SERVER['HTTP_USER_AGENT'], 'MSIE') !== false) { ... } For specific versions you can extend your condition if(strpos($_SERVER['HTTP_USER_AGENT'], 'MSIE 6.') !== false) { ... } Also see this related question .
