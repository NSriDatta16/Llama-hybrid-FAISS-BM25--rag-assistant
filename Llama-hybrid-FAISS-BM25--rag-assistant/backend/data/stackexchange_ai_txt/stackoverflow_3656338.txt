I would go for this beast. Not the most elegant version, but will rename any files ending with (case-insensitive) JPG.jpg find . -type f | grep -i -E '(JPG)*\.*(JPG)+$' | while read file ; do mv $file `echo $file | perl -pe 's!(JPG)*\.*(JPG)+$!.jpg!gi'`; done
