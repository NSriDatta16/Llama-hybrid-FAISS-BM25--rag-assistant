You can use simulate(arima_model, xreg = new_x) with a the parameter xreg to simulate a time series with a different regressor vector or matrix. Below is an example ### generate data set.seed(1) t = c(1:1000) x = sin(t/1000*2*pi*3) noise = arima.sim(n = 1000, list(ar = c(0.2,-0.5), ma = c(1,0.5)), sd = 0.1) ### plot time series demand = x + noise plot(t,demand, main = "black: original model \n red: simulate(mod, xreg = new_x)") ### model time series and plot newly generated series arima_model = auto.arima(y, xreg = x) new_x = sin(t/1000*2*pi*3+pi) points(t,simulate(arima_model , xreg = new_x), col = 2) mod
