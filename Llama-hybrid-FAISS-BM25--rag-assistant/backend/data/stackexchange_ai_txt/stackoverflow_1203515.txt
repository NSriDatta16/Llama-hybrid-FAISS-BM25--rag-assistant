To "shorten" an array $randomList to $X elements in the way that you've described, you could use array_chunk() and array_map() together like this: $randomList = array_chunk($randomList, count($randomList) / $X); $randomList = array_map('array_average', $randomList); And define array_average() as: function array_average($array) { return array_sum($array) / count($array); }
