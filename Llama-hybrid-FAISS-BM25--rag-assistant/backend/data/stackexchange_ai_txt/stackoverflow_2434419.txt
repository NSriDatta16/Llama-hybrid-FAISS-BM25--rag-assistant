The Showcase sample supporting crawlable hyperlinks is in the following branch: http://code.google.com/p/google-web-toolkit/source/browse/branches/crawlability/samples/showcase/?r=7726 It defines a filter in the web.xml to redirect URLs with the _escaped_fragment_ token to the output of HTML Unit.
