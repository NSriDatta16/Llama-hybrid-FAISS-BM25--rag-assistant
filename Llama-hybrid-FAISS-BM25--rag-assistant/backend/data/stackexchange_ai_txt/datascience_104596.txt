It is possible to create a hybrid system between human selected and machine learned rules in a decision tree. Hybrid systems have fallen out of favor because they are more difficult to create, use, and maintain. Often times the human domain experts that are most capable of creating useful rules are not capable of formatting the rules so the machine learning system can use them.
