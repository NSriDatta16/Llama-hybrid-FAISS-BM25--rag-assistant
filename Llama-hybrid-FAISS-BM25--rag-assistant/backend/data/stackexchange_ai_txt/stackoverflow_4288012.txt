Firstly, if you use xhtml, you must use tag " ". Also draw attention on str.replace: it replaces only once. js> var x = "hello"; js> x.replace("l", "L"); heLlo Make your replace implementation: js> function myreplace(str, pattern, change_to) { var s = str; var s2 = s; do { s2 = s; s = s.replace(pattern, change_to); } while (s2 != s); return s; } js> myreplace("helllllo", "l", "L"); heLLLLLo
