are there PHP-like optional parameters in C#?

with PHP optional parameters , if you don't send a parameter it will be assigned to a default value: public function getCustomer(id, optionalMessage = "(no message)") { ... } in C# I generally solve this with C# method overloading , e.g.: public void GetCustomer(int id) { ... } public void GetCustomer(int id, string optionalMessage) { ... } but I miss the pragmatic PHP variant, does C# also have some sugary syntax to do optional parameters as well , as in the PHP example?
