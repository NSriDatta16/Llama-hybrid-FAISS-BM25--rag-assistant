Don't forget you can use C# in: MS .NET (Windows) .NET Compact Framework (PocketPC, XBox 360) Silverlight (Windows, Mac) Moonlight (Mono / Linux) MonoTouch (iPhone) MonoDevelop (various including android) Micro Framework (some watches etc) Not exactly bad coverage. The language has very few requirements on the runtime / OS.
