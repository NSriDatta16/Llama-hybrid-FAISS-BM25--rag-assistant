Yes, initial load time can usually be reduced by a lot. You probably want to look into Code Splitting . You'll have to make a decision, which parts you need initially, and which parts can be loaded later (and when). Then make absolutely sure, that you don't accidentally use those later parts from your initial code fragment - except from inside GWT.runAsync .
