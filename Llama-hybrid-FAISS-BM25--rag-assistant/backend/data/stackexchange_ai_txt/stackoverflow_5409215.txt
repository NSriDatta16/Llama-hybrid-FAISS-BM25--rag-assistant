It's not required, no. The part of the HTML Living Standard you're interested in is The link element , which states: A link element must have either a rel attribute or an itemprop attribute, but not both. The type attribute gives the MIME type of the linked resource. It is purely advisory. The value must be a valid MIME type string. For external resource links, the type attribute is used as a hint to user agents...
