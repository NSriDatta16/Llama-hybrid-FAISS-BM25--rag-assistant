I don't think theres an easy way to do it with bash/sed/tr/find. I'd make a Ruby/Perl script that does the renaming. #!/usr/bin/ruby # Upcase.rb ARGV.each{ |i| newname = i.gsub(/(^.|\s.)/{ |x| x.upcase } `svn mv "#{i}" "#{newname}" ` } Then just do ./Upcase.rb foo.txt test.txt test2.txt foo/bar/test.txt or if you want to do a whole dir find ./ -exec ./Upcase.rb {} +
