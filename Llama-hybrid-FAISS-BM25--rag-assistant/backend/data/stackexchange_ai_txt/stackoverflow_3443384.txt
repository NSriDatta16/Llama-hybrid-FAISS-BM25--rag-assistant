Here is an example that will retrieve a list of Team Projects for a given Team Project Collection (TPC) in TFS 2010: static void Main(string[] args) { var results = GetTfsProjects(new Uri("http://mytfsserver:8080/tfs/DefaultCollection")); } private static List GetTfsProjects(Uri tpcAddress) { var tpc = TfsTeamProjectCollectionFactory.GetTeamProjectCollection(tpcAddress); tpc.Authenticate(); var workItemStore = new WorkItemStore(tpc); var projectList = (from Project pr in workItemStore.Projects select pr.Name).ToList(); return projectList; } Hope this helps.
