CMS gave a good answer about the value of this in different contexts. But as a side note, here's a handy function that you can use to generalize the effect of Dragger.wrap (which is similar to dojo.hitch ) if you're not using a library or using a library that doesn't have such a tool: var lockContext = function(context, callback) { return function() { callback.apply(context, arguments); } };
