It will work for the first change, but after that it will just do the same change again. The events are applied once when the page is loaded, changing the class won't remove and add event handlers. Instead of using one event handler for hiding and one for showing, you can use one that does both: $(function() { $("a.hide-para, a.show-para").click(function(e) { if ($(this).hasClass('hide-para')) { $('p').hide(); $(this).html('Show Paragraphs').removeClass('hide-para').addClass('show-para'); } else { $('p').show(); $(this).html('Hide Paragraphs').removeClass('show-para').addClass('hide-para'); } e.preventDefault(); }); }); Also, calling preventDefault prevents the default action for the click, i.e. following the link.
