Here's a solution that worked for me, although it may not be very elegant: Sub Macro1() ' Macro1 Macro Dim temp As String For u = 1 To 28000 If Cells(u, 4) <> 0 Then Cells(u, 4).Select temp = Mid(ActiveCell.Text, 1, InStr(1, ActiveCell.Text, "'") - 1) Cells(u, 4) = temp End If Next End Sub
