Derived tables to create "variables" and reduce repeated code. Something like this but can be expanded upon. Obviously "Average Value" can be a much more complex calculation, and if you have several it helps clean up code. Select *, case when AverageValue > 50 then 'Pass' Else 'Fail' end From ( Select ColA, ColB, AverageValue = (ColA+ColB)/2 From InnerMostTable ) AverageValues Order By AverageValue Desc
