Just to illustrate the point, my personal table.copy also pays attention to metatables: function table.copy(t) local u = { } for k, v in pairs(t) do u[k] = v end return setmetatable(u, getmetatable(t)) end There is no copy function sufficiently widely agreed upon to be called "standard".
