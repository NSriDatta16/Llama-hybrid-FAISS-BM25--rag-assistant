I have this in my MasterPage to get this info to display certain things on-screen, might help with the info you need (might not be the most elegant solution) public string ControllerActionName { get { return String.Concat(ViewContext.RouteData.Values["Controller"], ViewContext.RouteData.Values["Action"]).ToLower(); } }
