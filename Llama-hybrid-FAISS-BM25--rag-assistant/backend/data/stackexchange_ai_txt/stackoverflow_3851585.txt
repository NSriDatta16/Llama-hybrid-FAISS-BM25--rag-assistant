Try putting it in your onCreate(). Also, getDefaultSharedPreferences() is easier: @Override public void onCreate(Bundle savedInstanceState) { super.onCreate(savedInstanceState); // Access the default SharedPreferences SharedPreferences preferences = PreferenceManager.getDefaultSharedPreferences(this); // The SharedPreferences editor - must use commit() to submit changes SharedPreferences.Editor editor = preferences.edit(); // Edit the saved preferences editor.putString("UserName", "JaneDoe"); editor.putInt("UserAge", 22); editor.commit(); }
