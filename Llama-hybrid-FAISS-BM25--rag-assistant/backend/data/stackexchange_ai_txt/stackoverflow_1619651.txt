To do this in SPARQL 1.0, you'd have to write: SELECT * WHERE { ?x rdf:type foaf:Agent OPTIONAL { ?y rdf:type foaf:Person . FILTER (?x = ?y) . } FILTER ( !BOUND(?y) ) } As Phil M says , SPARQL 1.1 will introduce new syntax to make this much more straightforward to write.
