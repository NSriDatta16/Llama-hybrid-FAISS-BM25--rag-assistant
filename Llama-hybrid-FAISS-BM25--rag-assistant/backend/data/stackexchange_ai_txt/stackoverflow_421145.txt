If you don't want to do Group by then DataTable class has a method called Compute that executes few SQL functions. The following functions are supported : COUNT, SUM, MIN, MAX, AVG, STDEV, VAR . string salary = empTable.Compute("SUM( Salary )", "").ToString(); string averageSalaryJan = empTable.Compute("AVG( Salary )", "Month = 1").ToString(); // Assuming you have month stored in Month column and Salary stored in Salary column Other alternatives you can explore are: You may use the Microsoft KB article HOW TO: Implement a DataSet GROUP BY Helper Class in Visual C# .NET If you are using .NET 3.5 you may use Linq ref : LINQ DataTable Query - Group Aggregation
