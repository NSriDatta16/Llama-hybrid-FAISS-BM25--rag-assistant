@PostConstruct is the way to go for JSF 1.2. If you are using JavaEE 6 then you can also use @Produces annotation on a method to create custom factory method. public class CustomBeanFactory { @Produces public Bean getBean() { Bean bean = new Bean(); System.out.println("Created new bean."); return bean; } }
