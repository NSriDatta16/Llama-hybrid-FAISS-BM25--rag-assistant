Yes, this is possible. You need to use a function pointer or a wrapper that encapsulates function pointers and other callable objects, like function , which can be found in various forms in C++0x, C++ TR1, and Boost . Depending on how you are using the callbacks, a signals library like Boost.Signals might be more elegant.
