Might be late but for anyone with the same question the answer (as almost everything with Scikit-learn) is the usage of Pipelines from sklearn.impute import SimpleImputer from sklearn.preprocessing import FunctionTransformer from sklearn.pipeline import Pipeline import pandas as pd df = pd.DataFrame(dict( x=[1, 2, np.nan], y=[2, np.nan, 0] )) imputer = Pipeline([("imputer", SimpleImputer()), ("pandarizer",FunctionTransformer(lambda x: pd.DataFrame(x, columns = ["x", "y"])))]) imputer.fit_transform(df)
