Red5 is an Open Source media server written in Java that may be used in-place of Adobe FMS or Wowza; it currently supports: Streaming Video Formats (FLV, F4V, MP4, 3GP) Streaming Audio Formats (MP3, F4A, M4A, AAC) Recording Client Streams (FLV and AVC+AAC in FLV container) Shared Objects Live Stream Publishing Remoting Protocols: RTMP, RTMPT, RTMPS, and RTMPE Codecs: Any flash supported codec (PCM, FLV1, ScreenVideo, VP6, NellyMoser, MP3, AAC, AVC/h.264) Includes plug-in support with several existing applications. NSV streaming RTP/RTSP streaming SIP MPEG/TS streaming (Live and VOD) Security Red5 installs and runs using Tomcat. Applications are typically compiled as .war files and placed in the $RED5_PATH/webapps folder where the server unpacks them and creates the appropriate folders.
