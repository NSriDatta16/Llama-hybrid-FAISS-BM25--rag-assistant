Well, depending on the nature of the response variable $Y$ there are many possible regression models using only one predictor $x$ (and an intercept, mostly.) Logistic regression when $Y$ is binary Other glm's (generalized linear models) If $Y$ is ordinal, ordinal regression, How to handle ordinal categorical variable as independent variable If $Y$ is categorical/nominal, multinomial regression ... You should read up on generalized linear models, which is a natural starting point. Look at Understanding of GLM , Is there a GLM bible? or search this site.
