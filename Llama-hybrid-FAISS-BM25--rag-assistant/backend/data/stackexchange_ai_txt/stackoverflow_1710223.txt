scanf() can take a limited amount of characters with the "%c" specifier. ATTENTION It will not add a terminating null byte. char cityname[21]; scanf("%20c%d%d %c%d%d %c", cityname, &lat1, &lat2, &lat_letter, &lon1, &lon2, &lon_letter); cityname[20] = 0; But you're better off using fgets() and parsing the string "manually". Otherwise you're going to have END-OF-LINE issues
