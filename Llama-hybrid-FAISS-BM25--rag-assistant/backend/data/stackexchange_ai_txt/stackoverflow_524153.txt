General answer: Use a regex to match both AAAA and CCCC, then substitute each match with CCCC and AAAA respectively. edit to clear up the confusion str1.gsub(/(AAAA|CCCC)/) { $1 == 'AAAA' ? 'CCCC' : 'AAAA' } edit i thought of a more elegant way too :) str1.gsub(/((AAAA)|(CCCC))/) { $2 ? 'CCCC' : 'AAAA' }
