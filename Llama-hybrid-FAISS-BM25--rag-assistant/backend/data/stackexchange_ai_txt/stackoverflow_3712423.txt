You can hook on the load event of the element. E.g. Jsfiddle demo . Update : Then create new Image() instead (the average browser is smart enough not to request the same image twice and multiple references will point to the same image request): var img = new Image(); img.src = 'http://upload.wikimedia.org/wikipedia/commons/3/31/Atlantic_hurricane_tracks.jpg'; img.onload = function() { alert('finished!'); } Another jsfiddle demo (don't forget to clear browser cache, the same image might be already cached :) ).
