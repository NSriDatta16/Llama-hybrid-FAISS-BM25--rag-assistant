For some components ( I don't know if this apply to your components while being dragged ) you can set the background and use a "transparent color" The Color class does implements Transparency. To use it you may specify the alpha value in the Color constructor. For instance this is a semi transparent black background: // 0: totally transparent // 255: totally opaque, // 192 semy transparent. this.setBackground(new Color( 0, 0, 0, 192 )); See the [constructor][1] Again, I'm not sure if this applies to you. Give it a try [1]: http://java.sun.com/javase/6/docs/api/java/awt/Color.html#Color(int , int, int, int)
