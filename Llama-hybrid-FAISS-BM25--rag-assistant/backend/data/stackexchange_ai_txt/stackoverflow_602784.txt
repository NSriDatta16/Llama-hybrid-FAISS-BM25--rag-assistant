A memory manager. That should get you thinking about: free store management pointers (of course!) inheritance (you will want your driver code to use this) templates (another way to pass the manager around -- driver #2) designing user defined data structures (a memory block) efficient use of standard container(s) algorithms (to move around, figure out empty blocks, defragment)
