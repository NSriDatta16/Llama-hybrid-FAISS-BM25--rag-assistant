Process the file in a different directory and then move it to a place where CUTE FTP will pick it up after the conversion is finished. [EDIT] Don't use copy , use move . Both directories must be on the same harddisk. When using the Windows Explorer, use "Cut" or just drag the file with the mouse. Make sure there is no little "[+]" when you drop it.
