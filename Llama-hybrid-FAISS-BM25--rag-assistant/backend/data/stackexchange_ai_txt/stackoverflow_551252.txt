The embedding process is fully documented : Embedding Python in Another Application . The documents suggests a few levels at which embedding is done, choose whatever best fits your requirements. A simple demo of embedding Python can be found in the directory Demo/embed/ of the source distribution. The demo is here , should be able to build from the distro.
