As Greg mentions above, testing the trimmed .text() will remove paragraphs w/ no text, but do have a self-contained element like the tag. To avoid, trim the .html() return. As text is considered a child element in the DOM, you'll be set. $("p").filter( function() { return $.trim($(this).html()) == ''; }).remove()
