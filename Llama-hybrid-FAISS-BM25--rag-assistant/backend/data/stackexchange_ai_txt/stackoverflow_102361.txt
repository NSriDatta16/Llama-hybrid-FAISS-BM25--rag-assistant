If you use Oracle you can create a PL/SQL function you can use in your query that accepts an organization_id as input, and returns the first name of all employees belonging to that org as a string. For example:- select o.org_id, o.org_address, o.org_otherdetails, org_employees( o.org_id ) as org_employees from organization o
