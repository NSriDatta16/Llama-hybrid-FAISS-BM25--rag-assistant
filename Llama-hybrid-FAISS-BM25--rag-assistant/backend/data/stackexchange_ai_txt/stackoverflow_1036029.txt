The following works perfectly for me (You'll have to modify the containment plots to match your draggable element dimensions though): jQuery $(document).ready(function(){ $("#draggable").draggable({ containment: [-150,-150,0,0], scroll: 'false' }); }); #container { width:200px; height:200px; border:1px solid #cccccc; overflow:hidden; } #draggable { width:359px; height:359px; background:#cc0000 url("http://www.sudoku.4thewww.com/Grids/grid.jpg"); }
