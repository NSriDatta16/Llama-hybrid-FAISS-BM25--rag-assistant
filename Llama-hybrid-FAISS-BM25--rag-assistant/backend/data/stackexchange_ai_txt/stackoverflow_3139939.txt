Since features are just a collection of plugins (aka, a "packaging" process), you should be able to do in a PDE headless build what you can do with the PDE GUI. The PDE build page does mention: Building plug-ins, fragments or features consists of the 3 stages: fetch, generate scripts and build that are described bellow. You could have some issue for modified plugin , but building features or a final product should work.
