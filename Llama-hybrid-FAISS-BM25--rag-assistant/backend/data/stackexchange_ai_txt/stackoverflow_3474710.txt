Perl's Proc::ProcessTable will do the trick: http://search.cpan.org/dist/Proc-ProcessTable/ You can install it in debian or ubuntu with sudo apt-get install libproc-processtable-perl Here is a one-liner: perl -MProc::ProcessTable -Mstrict -w -e 'my $anHourAgo = time-60*60; my $t = new Proc::ProcessTable;foreach my $p ( @{$t->table} ) { if ($p->start() pid, "\n" } }' Or, more formatted, put this in a file called process.pl: #!/usr/bin/perl -w use strict; use Proc::ProcessTable; my $anHourAgo = time-60*60; my $t = new Proc::ProcessTable; foreach my $p ( @{$t->table} ) { if ($p->start() pid, "\n"; } } then run perl process.pl This gives you more versatility and 1-second-resolution on start time.
