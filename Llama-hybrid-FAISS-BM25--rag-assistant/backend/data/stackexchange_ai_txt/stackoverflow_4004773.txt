How about putting them in resources and them copying them to storage when app is run for the first time? You can use this code to check that the app is run for the first time: SharedPreferences pref = PreferenceManager.getDefaultSharedPreferences(this); boolean firstRun = pref.getBoolean("pref_first_run", true); pref.edit().putBoolean("pref_first_run", false).commit();
