Sounds like you're using the Wufoo "Embed Form Code" option which adds an iframe to your page. You should use their "Full Page Form Code" option instead. -- No iframe. Instead, you're embedding the entire form within your web page. You can then use jquery to modify it. Remember that the form's submit url will be the Wufoo server. Docs
