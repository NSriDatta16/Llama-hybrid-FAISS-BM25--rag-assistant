The object returned from values need have up . class Mycompute def initialize(str) @str=str end def values @@result=@str def @@result.up self.upcase end @@result end def up @@result.upcase end end This works well. Mycompute.new("Abc").values #=> "Abc" Mycompute.new("Abc").up #=> "ABC" Mycompute.new("Abc").values.up #=> "ABC"
