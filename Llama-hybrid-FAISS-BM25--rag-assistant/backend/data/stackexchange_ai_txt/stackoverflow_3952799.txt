A variant of Nick Craver's answer: $(".load-fragment").each(function() { var fileName = this.href.split(/[/.]/).slice(-2,-1); //store fileName where needed }); However, this assumes the links always have the format you show. The regex may be safer since it also is validation. "/english/india/index.aspx".split(/[/.]/) // returns: ["", "english", "india", "index", "aspx"] ["", "english", "india", "index", "aspx"].slice(-2, -1) // returns "index"
