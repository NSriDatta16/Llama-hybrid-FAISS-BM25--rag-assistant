You can query remote registry without that module. PSRemoteRegistry module only makes it easy. However, the requirement is to have remote registry service enabled and running on the remote machine. For an example, without the PSremoteRegistry module, check my blog post: http://www.ravichaganti.com/blog/?p=1920 Also, try at an elevated PowerShell console. You need admin rights to query remote registry. This is what I found on my system.
