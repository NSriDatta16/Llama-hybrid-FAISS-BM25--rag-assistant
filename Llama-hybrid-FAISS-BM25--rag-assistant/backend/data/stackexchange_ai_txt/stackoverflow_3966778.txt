Serializin/deserializing XML with Linq is very easy, I strongly recommend you take a look at it if you don't know it yet. Serialization will look like this: // add reference to System.Xml.Linq var xml = new XDocument( new XDeclaration("1.0", "utf-8", "yes"), new XElement("users", from pair in dic select new XElement("user", new XAttribute("userId", pair.Key), new XAttribute("Groups", pair.Value.groups)))); // xml.Save(...) to save in the IsolatedStorage Deserialization: XDocument loaded = XDocument.Load(...your stream...); var results = from c in loaded.Descendants("users") select new { userIdAttribute = (string)c.Attribute("userId"), groupsAttribute = (string)c.Element("Groups") }; foreach (var user in results) { dic.Add(user.userIdAttribute, new User() { name = user.userIdAttribute, groups = user.groupsAttribute }); } You can find some good documentation here and some examples here .
