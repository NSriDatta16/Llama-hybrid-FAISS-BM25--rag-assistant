If you implement the UIResponder touchesBegan:withEvent: method.. - (void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event ...on your view, you'll be able to tell when the user has pressed outside of the buttons. See the UIResponder Class Reference for more information.
