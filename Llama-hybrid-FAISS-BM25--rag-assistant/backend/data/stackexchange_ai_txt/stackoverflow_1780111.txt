To change the storage engine of an existing table: ALTER TABLE my_table ENGINE = InnoDB; To set InnoDB as default: Normally, this is MyISAM, but you can change it by using the --default-storage-engine or --default-table-type server startup option, or by setting the default-storage-engine or default-table-type option in the my.cnf configuration file.
