This is how to set it up to do it how you want: var test = array( 'toyota' => array('type' => array('camry','siena')), 'mitsubishi' => array('type' => array('mirage','galant')) ))); ?>; alert(test.vendor['toyota'].type[1]); // siena alert(test.vendor['mitsubishi'].type[0]); // mirage I would recommend skipping the vendor and type part of it altogether unless you're holding other stuff in that object too, and doing something like this: var vendors = array('camry','siena'), 'mitsubishi' => array('mirage','galant') )); ?>; alert(vendors['toyota'][1]); // siena alert(vendors['mitsubishi'][0]); // mirage
