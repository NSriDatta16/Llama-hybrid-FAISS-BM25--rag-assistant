In $(".draggable_folder").draggable , change drag: function(event, ui) to start: function() . This will make .folder visible and the droppable will become active. The final code for draggable: $(".draggable_folder").draggable({ cursor: 'move', cursorAt: { top: 8, left: 0 }, start: function() { $(".folder").show(); }, helper: function(event) { var subject = $(this).find('.message_subject').html(); return $(' '+subject+' '); } });
