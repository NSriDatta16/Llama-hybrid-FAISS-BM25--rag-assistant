All task definitions are declared in ant.jar!org\apache\tools\ant\taskdefs\defaults.properties Here is the JUnit-related ones: junit=org.apache.tools.ant.taskdefs.optional.junit.JUnitTask junitreport=org.apache.tools.ant.taskdefs.optional.junit.XMLResultAggregator Here is corresponding Java code may look something like this: FileSet fs = new FileSet(); fs.setDir(new File("..")); fs.createInclude().setName("TEST-*.xml"); XMLResultAggregator aggregator = new XMLResultAggregator(); aggregator.addFileSet(fs); AggregateTransformer transformer = aggregator.createReport(); transformer.setFormat(Format.FRAMES); transformer.setTodir(new File("..");
