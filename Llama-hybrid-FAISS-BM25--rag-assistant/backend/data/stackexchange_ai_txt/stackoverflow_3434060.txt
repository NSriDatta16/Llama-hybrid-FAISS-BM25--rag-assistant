I think you have misunderstood the "package" concept. Packages is an organization of classes very similar to a directory structure, but with a convention that they should start with the authors DNS domain reversed. You may want to read up on the Java Tutorial. The appropriate page is http://download.oracle.com/javase/tutorial/java/package/packages.html
