To parse/validate against three expected formats, you can use something like below. Given the pattern, once you know it is valid you could just use string.Split to get the first part; if you need something more elegant you could use TryParseExact for each pattern in turn and extract the desired portion (or re-format it). string s1 = "12/August/2010", s2 = "August/12/2010", s3 = "2010/12/August"; string[] formats = { "dd/MMMM/yyyy", "MMMM/dd/yyyy", "yyyy/dd/MMMM" }; DateTime d1 = DateTime.ParseExact(s1, formats, CultureInfo.CurrentCulture, DateTimeStyles.None), d2 = DateTime.ParseExact(s2, formats, CultureInfo.CurrentCulture, DateTimeStyles.None), d3 = DateTime.ParseExact(s3, formats, CultureInfo.CurrentCulture, DateTimeStyles.None);
