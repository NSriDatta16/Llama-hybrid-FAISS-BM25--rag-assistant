HTTP_USER_AGENT contains this information, and will be passed in the environment variables your application uses. In mod_python, this is expressed as: def my_request_handler(req): req.add_common_vars() agent = req.subprocess_env.get("HTTP_USER_AGENT") # `agent` now contains the full user agent of the browser, or None It's a basic CGI thing, but this is how mod_python gives it to you.
