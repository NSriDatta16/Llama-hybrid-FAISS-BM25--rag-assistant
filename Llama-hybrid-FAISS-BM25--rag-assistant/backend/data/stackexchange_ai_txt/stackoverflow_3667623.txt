Firefox's CSS files are in /whereverfirefoxisinstalled/gre/ These two are probably of most interest, but there are some others in there as well which should not be ignored. forms.css html.css You can use the Firebug addon to see what is being applied to the page you're working on. When inspecting an element, select "show user agent CSS" from the style dropdown in the right hand panel.
