Use $n$ as regressor in logistic regression with binomial proportion target

In a logistic regression with binomial proportion target variable -- i.e. $Z_i \sim \text{Bin}(n_i,p_i)$ with target $Y_i := Z_i/n_i$ -- is there any reason for not allowing $n_i$ to be one of the regressors?
