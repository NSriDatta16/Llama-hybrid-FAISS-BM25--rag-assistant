You can use cgi... #!/usr/bin/env python import cgi def myMethod(some_parameter): // do stuff return something form = cgi.FieldStorage() my_passed_in_param = form.getvalue("var_passed_in") my_output = myMethod(my_passed_in_param) print "Content-Type: text/html\n" print my_output This is just a very simple example. Also, your content-type may be json or plain text... just wanted to show an example.
