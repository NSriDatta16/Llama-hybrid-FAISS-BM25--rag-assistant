You probably want to stripslashes even if magic_quotes_sybase is on: function esc($s) { if (get_magic_quotes_gpc()) { if (ini_get('magic_quotes_sybase')) $s = str_replace("''", "'", $s); $s = stripslashes($s); } //if return mysql_real_escape_string($s); } You might also want to take a look at PHP's get_magic_quotes_gpc function page, there are several user comments on the page with fairly elegant solutions for ensuring slashes are stripped.
