Try this: CSS div.square { cursor: pointer; width: 100px; height: 100px; border: 2px dashed purple; position: absolute; top: 30px; left: 30px; font-size: 50px; line-height: 100px; text-align: center; color: white; } jQuery + jQueryUI var count = 25; var colors = ['red','green','blue','orange','yellow']; while(count--) { $(' ',{className:'square', text:count}).draggable().css({position:'absolute','z-index':count, text:count, backgroundColor:colors[count % 5]}) .appendTo('body'); } EDIT: I just noticed that for some reason in IE and Safari .draggable() overrides the absolute positioning with relative , so you need to set it back to absolute after you made it draggable.
