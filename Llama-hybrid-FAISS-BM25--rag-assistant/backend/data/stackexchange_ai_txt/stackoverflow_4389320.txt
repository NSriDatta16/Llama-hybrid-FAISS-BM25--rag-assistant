a cleaner and more elegant (but not more efficient) way to convert a certain type into std::string is to use template Target lexical_cast(const Source& arg); from #include The type to be converted needs to provide the usual " example usage: std::string s = boost::lexical_cast ( 17 ); assert( s == "17" );
