What is the actual status code sent by the server, e.g. if you use another tool such as fiddler. A lot of pages out there send a 200 when they should send a 404. You can find a lot by doing a search for "page not found", though these days there's a fewer number of them and a larger number of useful pages about 404 pages (an improvement on the part of the search engines, not on the part of the average website).
