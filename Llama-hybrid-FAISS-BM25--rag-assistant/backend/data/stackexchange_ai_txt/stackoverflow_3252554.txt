I wasn't able to get elements to be both selectable and draggable , but it was easy enough to have a parent container div that's draggable, with a child unordered list ( ) that's selectable. Here's my code (tested): .sel { padding: 20px; border: solid 1px #000; list-style-type: none; background-color: #c0c0c0; width: 90%; margin-left: auto; margin-right: auto; } .sel .ui-selecting { background: #FECA40; } .sel .ui-selected { background: #F39814; color: #fff; } .sel li { margin: 3px; padding: 0.4em; font-size: 1.4em; height: 18px; } .drag { margin: 10px; border: solid 1px #f00; background-color: #333; color: #fff; } $(document).ready(function() { $(".sel").selectable(); $(".drag").draggable(); }); Drag me Item 1 Item 2 Item 3 Item 4
