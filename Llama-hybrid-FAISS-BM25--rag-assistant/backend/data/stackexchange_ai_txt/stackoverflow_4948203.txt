Here is a safe version of Peter Lawrey's answer, using BigDecimal : public static double safeAverage(final Collection coll){ BigDecimal dec = BigDecimal.ZERO; for(final Number n : coll){ dec = dec.add(BigDecimal.valueOf(n.doubleValue())); } return dec.divide(BigDecimal.valueOf(coll.size())).doubleValue(); } (Not quite safe, however, as BigInteger and BigDecimal also extend Number . Safe if used with any of the Number implementations in java.lang )
