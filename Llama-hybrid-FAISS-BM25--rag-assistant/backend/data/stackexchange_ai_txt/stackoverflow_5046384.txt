Answering my own question :) Found an easy and trivial(!) way to find the storage of the path:** if(selectedVideoPath.indexOf(Environment.getExternalStorageDirectory().getPath()) == 0) { getContentResolver().delete(MediaStore.Video.Media.EXTERNAL_CONTENT_URI, MediaStore.Video.Media._ID + "=" + videoId, null); } else { getContentResolver().delete(MediaStore.Video.Media.INTERNAL_CONTENT_URI, MediaStore.Video.Media._ID + "=" + videoId, null); }
