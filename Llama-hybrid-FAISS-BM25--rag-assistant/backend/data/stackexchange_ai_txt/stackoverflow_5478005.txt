Clearly this won't work the way you want it if the RTF on the clipboard contains a fragment with a /font directive. Which is very likely. Filtering the RTF fragment is only practical by pasting into a helper RichTextBox. Use the SelectionFont property then copy it back to the clipboard and Paste(). Or do it directly: int oldpos = richTextBox1.SelectionStart; richTextBox1.SelectionLength = 0; richTextBox1.Paste(); int newpos = richTextBox1.SelectionStart; richTextBox1.SelectionStart = oldpos; richTextBox1.SelectionLength = newpos - oldpos; richTextBox1.SelectionFont = richTextBox1.Font; richTextBox1.SelectionStart = newpos;
