PIL font's do have a getsize method which you can use to get width and height of font, did you try that? e.g. from PIL import Image, ImageDraw, ImageFont fontFile = "/usr/share/fonts/truetype/freefont/FreeSansBold.ttf" font = ImageFont.truetype(fontFile, 10) text="anurag" print font.getsize(text) assert font.getsize(text)[0]*2 == font.getsize(text*2)[0]
