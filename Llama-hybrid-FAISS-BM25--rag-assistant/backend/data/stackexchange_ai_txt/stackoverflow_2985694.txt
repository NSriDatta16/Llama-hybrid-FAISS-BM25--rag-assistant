I believe you forgot to allocate memory for mystat. Try this: typedef struct pcap_stat mystat; ... mystat actualStat; /* allocate memory for mystat on stack - you can also do it on the heap by malloc-ing */ mystat *mystatp = &actualStat; /* use allocated memory */ /* Put the interface in statistics mode */ if(pcap_stats(handle, mystatp) In Pcap.Net I use pcap_stats_ex() but it's probably only available on WinPcap and not on libpcap.
