If you only need to organize your code and don't need libraries you may use http://code.google.com/p/joose-js/ . Otherwise use the oop model of the library you are using. Simple example Module("Test", function (m) { Class("Point", { has: { x: { is: "rw", init: 0 }, y: { is: "rw", init: 0 } }, methods: { clear: function () { this.setX(0); this.setY(0); } } }) })
