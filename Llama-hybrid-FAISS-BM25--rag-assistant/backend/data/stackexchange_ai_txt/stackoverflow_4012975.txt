In the CS5 IDE : Select the textfield with the font you want embedded. Selected Embed... to open theFont Embedding dialogue. In the right-hand side panel select the ActionScript tab. At the very bottom under the heading Sharing check the box that says Import for runtime sharing Inside the textfield that says URL you enter a url pointing to the SWF file where you have embedded the font. Now the SWF with the embedded font that you specified in the url will automatically be loaded by the flash player.
