Re-organize the data to a single array for sorting: my @a = ([7, "seven", "VII"], [2, "two", "II"], ..); @a = sort { $a->[0] $b->[0] } @a; Then recreate the original arrays: my(@a1, @a2, @a3); for (@a) { push @a1, shift @$_; push @a2, shift @$_; push @a3, shift @$_; }
