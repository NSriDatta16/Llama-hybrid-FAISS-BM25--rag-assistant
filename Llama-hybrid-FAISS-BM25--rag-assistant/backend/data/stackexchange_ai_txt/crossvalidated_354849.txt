There are different approaches Recursive strategy one many-to-one model prediction(t+1) = model(obs(t-1), obs(t-2), ..., obs(t-n)) prediction(t+2) = model(prediction(t+1), obs(t-1), ..., obs(t-n)) Direct strategy multiple many-to-one models prediction(t+1) = model1(obs(t-1), obs(t-2), ..., obs(t-n)) prediction(t+2) = model2(obs(t-2), obs(t-3), ..., obs(t-n))` Multiple output strategy one many-to-many model prediction(t+1), prediction(t+2) = model(obs(t-1), obs(t-2), ..., obs(t-n))` Hybrid Strategies combine two or more above strategies Reference : Multi-Step Time Series Forecasting
