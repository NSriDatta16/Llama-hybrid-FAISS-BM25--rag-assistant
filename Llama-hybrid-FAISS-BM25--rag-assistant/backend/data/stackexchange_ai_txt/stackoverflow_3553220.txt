Yes. Just register each device you want in Apple's iPhone Dev site and generate a provisioning profile including all devices you would like to test on. After that, plug each device to your Mac, open Organizer in XCODE, and just run the application in device mode . Application will be installed on the iPhone, and you can freely test it for a couple of months. After that, provisioning profile will expire and you will have to generate a new one, install the application again, and start the process over.
