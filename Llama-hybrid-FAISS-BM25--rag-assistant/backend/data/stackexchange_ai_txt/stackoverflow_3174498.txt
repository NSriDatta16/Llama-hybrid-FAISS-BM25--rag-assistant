The limit is really determined by the user agent and client machine being used. HTML, in the same way as XML, is a tree format of data. Therefore the more elements, the further through the tree the client browser has to search to render the page. I had issues adding more than 100 tables to a div (as an old workaround to IE6 not being able to create table elements dynamically).
