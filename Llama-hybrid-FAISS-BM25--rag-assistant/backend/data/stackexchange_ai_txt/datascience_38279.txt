Logistic Regression is used for classification. Linear Regression is used for prediction. Given a set of features, logistic regression uses sigmoid function to find whether it belongs to class 0 or 1. Example: Success or failure of a event can be predicted using logistic regression. In linear regression, given a set of features, it is used to predict the value of output. The sales of a company can be found using linear regression. Even though there is the work regression in logistic regression, it is a classification algorithm.
