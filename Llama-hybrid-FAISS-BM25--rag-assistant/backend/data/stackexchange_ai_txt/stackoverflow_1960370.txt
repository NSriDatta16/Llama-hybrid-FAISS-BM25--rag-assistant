Visual Studio uses an in proc compiler to help with the UX of the build process. Becuase of this there can be some difference between VS and MSBuild. Try setting the property UseHostCompilerIfAvailable to false in your project file. This will force VS to use MSBuild for the build. The drawback is performance inside Visual Studio. You would use the follwing fragement in your .csproj (or .vbproj) file. false
