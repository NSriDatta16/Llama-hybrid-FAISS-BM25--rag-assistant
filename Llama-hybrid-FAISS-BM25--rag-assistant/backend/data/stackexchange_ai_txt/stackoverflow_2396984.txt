Use a space delimiter for events like this: $("body").delegate("input", "mouseup mouseover", function() { $(this).after(" Another paragraph! "); }); You probably meant to select elements, I changed this above, correct me if you meant otherwise. Alternatively, use .bind() like this: $("body input").bind("mouseup mouseover", function() { $(this).after(" Another paragraph! "); }); I find this easier to read, but whatever floats your boat.
