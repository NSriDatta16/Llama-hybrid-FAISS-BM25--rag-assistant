Also added in CSS2 was the direct adjacent combinator. For example, imagine you want to indent all paragraphs in a document except for ones immediately following a sub-heading ( ). The second rule below shows an h2 selector combined with a p selector using a plus sign as the combinator to indicate that the declaration applies only if p immediately follows h2 : p { text-indent: 0.5in; } h2 + p { text-indent: 0; } From - http://www.xml.com/pub/a/2003/06/18/css3-selectors.html
