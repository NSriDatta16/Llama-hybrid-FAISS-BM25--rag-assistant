You can't read the values without opening the Excel file at all. But you may read the values without having to open Excel. If the file is saved in the xml format it's going to be easier. If not, the easiest method is to still use Excel but use Office Automation to do it. The hard way is to create an excel file parser - quite hard on the non-open xml excel format (pre Office 2003) - hard but still possible. However, it is quite impossible to read from an excel spreadsheet without opening the file at all..
