You could use a querystring parameter... Page1 protected void Button1_Click(object sender, EventArgs e){ Response.Redirect("~/page2.aspx?showButton=true", true); } protected void Button2_Click(object sender, EventArgs e){ Response.Redirect("~/page2.aspx?showButton=false", true); } Page2 protected void Page_Load(object sender, EventArgs e){ someButton.Visible = (Request[@"showButton"] == @"true"); } The example isn't that elegant, but should get you started on what you want.
