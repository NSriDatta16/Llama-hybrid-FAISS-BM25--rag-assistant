It always happens that some subset or subgroup of data may not be predicted well using the ML Model. This may happen due to many reasons : The sub category may not have enough data to learn from Bias in how data was captured Features created are not able to capture the subcategory As you mentioned model retraining may not improve the model but you could try the following ; Row Wise Weighting : Increase the weights of these rows while training so that model gives a little more importance to them while learning. XGBoost provides this intrisically Feature Engineering : Spend a little bit more time on feature engineering to capture variables which may help you in this specific group Almost impossible, but see if you could collect more data
