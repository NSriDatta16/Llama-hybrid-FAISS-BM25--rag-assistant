User controls Easy to implement because you can visually drag & drop other controls on the markup part. Good designer support in Visual Studio Can only be reused in the same project You can also create templated user controls if you want (though not so commonly used) Custom server controls Harder to create but there's a whole variety of possible scenarios: Inherit from existing controls like Label, Button, ... Create a composite control Make templated, + databound, controls Great reusability in other projects Ideal to make frameworks that can be used company wide
