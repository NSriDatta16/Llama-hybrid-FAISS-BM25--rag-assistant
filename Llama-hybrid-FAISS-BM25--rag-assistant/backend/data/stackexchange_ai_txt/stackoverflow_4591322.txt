Just use this code for all tab activity without giving option menu for TabHost (Main Activity) /**...........Context Menu........cg.*/ public boolean onCreateOptionsMenu (Menu menu) { menu.add(0, 0, Menu.NONE, "Refresh"); menu.add(0, 1, Menu.NONE, "Exit"); return true; } /**---------- Handles item selections ------------cg */ @Override public boolean onOptionsItemSelected(MenuItem item) { switch (item.getItemId()) { case 0: System.out.println("Refresh"); return true; case 1: System.out.println("Exit"); return true; } return false; } I hope it will help you.
