Is there a way to compare two variables in Template Toolkit?

[% IF OrgType.id == Organization.org_type_id %]selected="selected"[% END %] Does not work even when they both evaluate to the same number. [% IF OrgType.id == 3 %]selected="selected"[% END %] (i.e. hard-coding in a number for testing purposes) does work. [% OrgType.id %] and [% Organization.org_type_id %] both print "3" on the page.
