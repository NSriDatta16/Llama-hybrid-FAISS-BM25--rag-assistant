Server side OS information JSP which is java can find the OS of the computer where it is executing. we can use the getProperty() method to find the System properties of the system. It returns a String with the name of OS. It supports various OSs in java. Example: Example for Printing the OS name Client side information String agent = request.getHeader ("user-agent"); StringTokenizer st = new StringTokenizer (agent ,";"); st.nextToken (); // Get the user's browser name String userBrowser = st.nextToken (); // Get the user's operating system name String userOs = st.nextToken ();
