Named Entity Recognition is technique which can be used here. Location is one of the 3 most studied classes (with Person and Organization). Stanford NLP has an open source Java implementation that is extremely powerful. For Example let say sentence is " i will be going to Sweden from Boston. " Now here you can use regular expression to detect these LOCATION tags.
