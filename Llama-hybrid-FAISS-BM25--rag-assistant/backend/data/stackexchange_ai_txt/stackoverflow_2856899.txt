Average because this is a Generic Method . The method can be run on a query or enumeration over any type (as long as a suitable selector for that type is provided). this IQueryable source because this is an Extension Method . Extension methods allow you to add additional methods to existing types without having to alter that type's definition. In this case Linq adds the Average method to the IQueryable interface without altering that interface .
