If you have a second profile (like 'sidekick'), the following will launch a new Firefox process: firefox.exe -no-remote -p sidekick However, if that profile is already used by a current Firefox session, that will not work. To create a new profile launch Firefox from the command line with the -P flag and create it. firefox.exe -P The OP ykaganovich adds in the comments (in Oct. 2020, three years later): recent versions of Firefox (as of 2020) also have profile management UI available at about:profiles
