before you make the method call 'request' you have to add the proxy information to your UA object. Start here: http://search.cpan.org/~gaas/libwww-perl-5.836/lib/LWP/UserAgent.pm#Proxy_attributes And is 'POST' a valid bareword operator or namespace? If not, you'll want to look into how to create a HTTP::request object first.
