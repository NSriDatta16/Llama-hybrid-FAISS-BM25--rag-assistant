Use the array version in the view. CakePHP does reverse routing to determine the string link to use, i.e. '/my-route', from the array of controller / action / params in the array. $html->link('My Route Name', array('controller' => 'my_controller', 'action' => 'index')); Also check out this from Mark Gandolfo
