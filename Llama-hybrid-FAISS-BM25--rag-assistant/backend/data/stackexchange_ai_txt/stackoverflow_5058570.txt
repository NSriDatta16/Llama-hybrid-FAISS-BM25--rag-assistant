Referenced from the OpenCV wiki page: http://opencv.willowgarage.com/wiki/CompileOpenCVUsingMacOSX Set the environment variables like this: export PKG_CONFIG_PATH=/opt/local/lib/pkgconfig export LD_LIBRARY_PATH=/opt/local/lib and compile opencv code like this: g++ -bind_at_load `pkg-config --cflags opencv` morphology.c -o morphology `pkg-config --libs opencv` and then run the code like this: ./morphology
