Some of the parameters, like CURLOPT_USERAGENT are send in the HTTP headers and can be retrieved using special globals like $_SERVER['HTTP_USER_AGENT'] (see http://www.php.net/manual/de/reserved.variables.server.php ). Others, like CURLOPT_SSL_VERIFYPEER are only local to CURL and don't get send to the server.
