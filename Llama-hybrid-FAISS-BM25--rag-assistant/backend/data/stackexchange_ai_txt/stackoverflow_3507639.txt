First of all, consider redesigning your classes. Circular dependencies are bad practice, and chances are that you could avoid this altogether with a more elegant class design. That said, you can get around the problem using forward references (at which point, you need to use pointers or references) - class B; class A { B *pPtr; }; class B { A typeA; };
