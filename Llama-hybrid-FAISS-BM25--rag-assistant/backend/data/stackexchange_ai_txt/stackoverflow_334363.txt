If you search on http://community.installshield.com you'll find an article by Robert Dickau entitled "Exiting an MSI Installation from a Custom Action". It's at the following link: http://community.installshield.com/showthread.php?p=418197 Here's the snippet of code he uses as his example: Function ExitSetupFromVBS( ) Const IDABORT = 3 ' ...do some work... ' abort the installation ExitSetupFromVBS = IDABORT End Function Good luck.
