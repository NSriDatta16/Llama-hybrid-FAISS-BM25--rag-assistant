Get THEM to write it! AFAIK, cucumber can be used with .NET Cucumber lets the client (with minimal training) write just how they'd like the app to behave. I did an informational interview with a shop that got the clients to write all of the BL in cucumber so there was little misunderstanding, and as a bonus, it encourages non techs to start to understand how we need to think about problems.-
