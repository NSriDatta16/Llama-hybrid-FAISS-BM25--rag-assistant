"Best" is subjective, especially since you didn't specify any particular features you may be looking for. With that said, here's a couple to get you started: Simple Java Mail - built on top of JavaMail, simplifies API to produce a correct emails Commons Email - built on top of JavaMail, simplifies certain commonly used scenarios. Spring Email integration (only if you're using Spring already) javaee.github curates a comprehensive list with many more: https://javaee.github.io/javamail/ThirdPartyProducts Oracle has a curated list as well, although the lists are suspiciously similar: http://www.oracle.com/technetwork/java/javamail/third-party-136965.html
