One can find sequence labelling libraries by searching for the term conditional random fields , the state of the art method. Probably one could also find libraries and tutorial by searching the term Named Entity Recognition, which is certainly the most standard NLP application of sequence labelling. Here are a few libraries that I know of: CRF++ crfsuite (there is a python wrapper ) Wapiti is a particularly efficient library (also with python wrapper ) See also this question .
