You can do this by cloning the helper element and then removing the original element from the DOM. Note, you have to clear the style attribute as it has position:absolute set for dragging and this prevents the clone from positioning correctly in the ul . Of course you could be more granular about this and add/remove classes etc. $('.container').droppable( { accept: '.sortable li', drop: function(event, ui) { ui.draggable.clone().attr("style", "").appendTo( $(this).find("ul") ); ui.draggable.remove(); } }); Here's an updated JSBin example.
