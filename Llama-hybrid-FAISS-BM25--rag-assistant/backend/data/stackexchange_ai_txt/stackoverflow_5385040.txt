All you need to do is set AllowDrop to True and handle the Drop event. The ListBox definition: The event handler: private void ListBox_Drop(object sender, DragEventArgs e) { if (e.Data.GetDataPresent("FileName")) { string folderPath = e.Data.GetData("FileName"); //do whatever you need to do with the folder path } }
