Use Element instead of DocumentFragment : NodeList folderNodes = document.getElementsByTagName("Folder"); Element node = (Element)folderNodes.item(0); NodeList subNodes = node.getElementsByTagName("OtherNodes"); // and so on... The Element interface supports getElementsByTagName , whereas DocumentFragment is minimal and really doesn't do much.
