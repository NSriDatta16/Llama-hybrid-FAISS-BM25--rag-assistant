You can't have multiple elements with the same ID, it's invalid HTML, IDs have to be unique. Your menu elements needs to be a class instead of an ID, like this: edit add delete Content_2 [drag] And jQuery like this: $(function(){ $('div[id^="content_"]').hover( function () { $(this).find('.menu, .dragable').show(); }, function () { $(this).find('.menu, .dragable').hide(); } ); });
