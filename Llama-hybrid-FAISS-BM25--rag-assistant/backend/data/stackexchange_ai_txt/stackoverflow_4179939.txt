The data of your app can not be accessed by other apps. Simply use a private storage. You have several options: Sava timetracking data to database. App database is only visible to the app that created it. Other apps will not be able to see this data. When sending an email just query the database and create a text attachment. Create a private file via Context.openFileOutput() .
