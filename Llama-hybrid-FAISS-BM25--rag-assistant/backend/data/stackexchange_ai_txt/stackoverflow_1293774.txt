Store the {string, size, offset} tuples in sorted order (by string) and use a binary search for the string. You might also store, at the start of the file, offsets for each first letter of strings. For example if strings starting with 'a' began at position 120 and those starting with 'b' began at position 2000 in the file you could start the file with something like 120, 2000, ...
