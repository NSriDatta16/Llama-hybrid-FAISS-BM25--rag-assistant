You need to pay attention to the error, if any, reported by mysql_error() . This should give you some information you can use to diagnose the connection problem. See this MySQL Forge wiki page about connection failure: http://forge.mysql.com/wiki/Error2003-CantConnectToMySQLServer Or this page from the MySQL Manual: http://dev.mysql.com/doc/refman/5.1/en/can-not-connect-to-server.html
