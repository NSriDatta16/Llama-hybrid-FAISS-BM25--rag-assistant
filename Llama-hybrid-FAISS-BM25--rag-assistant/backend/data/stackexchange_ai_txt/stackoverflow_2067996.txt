SAS has the datalines construct which is used for embedding an external data file inside the source program, e.g. in the following program, there are 5 datalines (the terminator is the semi-colon on a line by itself) data output; input name $ age; datalines; Jim 14 Sarah 11 Hannah 9 Ben 9 Timothy 4 ; run;
