It sounds like you're trying to add an unbound column. In this case you can add the following in the InitializeLayout delegate of the grid: private void myUltraGrid_InitializeLayout(object sender, InitializeLayoutEventArgs e) { e.Layout.Bands[0].Columns.Add("New Column Name"); }
