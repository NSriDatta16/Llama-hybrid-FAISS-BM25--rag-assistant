The Stopwatch class, available since .NET 2.0, is the best way to go for this. It is a very high performance counter accurate to fractions of a millisecond. Take a look at the MSDN documentation , which is pretty clear. EDIT: As previously suggested, it is also advisable to run your code a number of times in order to get a reasonable average time.
