You could set the columnDefinition attribute. From the Javadocs: columnDefinition (Optional) The SQL fragment that is used when generating the DDL for the column. @Entity private class Person { private Integer id; @Id @Column( columnDefinition="number(8,0)", nullable=false) public Integer getId() { } }
