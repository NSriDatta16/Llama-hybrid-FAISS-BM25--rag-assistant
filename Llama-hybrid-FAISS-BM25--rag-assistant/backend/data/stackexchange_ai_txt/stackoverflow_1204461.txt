You should be able to use deMorgan's theorem to rewrite the query so it is evaluated more efficiently. You can also be factor out common subexpressions into outer if statements. However, you will likely be sacrificing clarity of intent when you do so. Nowadays it's probably better form to write a clear, if long if statement rather than a short incomprehensible one.
