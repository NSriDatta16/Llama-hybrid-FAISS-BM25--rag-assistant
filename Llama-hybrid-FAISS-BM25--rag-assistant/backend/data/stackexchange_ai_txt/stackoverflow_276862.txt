You should be using CWnd::SetRedraw around your adds, to prevent the control updating all its internal state after each add. If you're not already doing it, then do this: combo.SetRedraw(FALSE); ... All the adds combo.SetRedraw(TRUE); combo.Invalidate(); You should also consider using the CComboBox::InitStorage function, which preallocates memory for the combo-box.
