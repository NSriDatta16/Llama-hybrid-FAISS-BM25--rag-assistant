Yes, you'll need to do an build with an adhoc provisioning profile. Your client will need to drag the provisioning file to iTunes. I'm assuming you'll be emailing a ZIP file of the application package to the client. They should unzip it and drag the resulting folder on Windows or package on OS X to iTunes and sync their phone.
