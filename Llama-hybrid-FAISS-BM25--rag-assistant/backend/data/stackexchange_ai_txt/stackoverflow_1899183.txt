Yes, Hardik it is very simple if you want to open a local xls file. Add a local xls file in to your project Drag and drop an UIWebView control on the view. Connect Files owner and UIWebView Object Add this: - (void)viewDidLoad { [super viewDidLoad]; //Get the path where your local xls file is located NSString *FilePath = [[NSBundle mainBundle] pathForResource:@"sample" ofType:@"xls"]; //Now To use a local resource file in a web view you need to use NSURL fileURLWithPath: NSURL *url=[NSURL fileURLWithPath:FilePath]; //URL Requst Object NSURLRequest *requestObj=[NSURLRequest requestWithURL:url]; //Load the request in the UIWebView. [m_webView loadRequest:requestObj]; } Now just Build and Go
