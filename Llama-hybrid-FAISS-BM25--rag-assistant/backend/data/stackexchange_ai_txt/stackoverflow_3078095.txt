Look into WPF, which leverages XAML, which is probably going to be a not-too high a learning curve for you. If you're using Ubuntu, you might want to take a look at Mono framework, which is C# on Linux systems. A form of XAML exists, called Moonlight, which might be useful for you. At least an option you can examine!
