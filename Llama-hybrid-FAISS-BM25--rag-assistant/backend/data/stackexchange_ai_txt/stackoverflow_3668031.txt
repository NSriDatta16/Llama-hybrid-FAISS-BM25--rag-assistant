Simone Gianni's Answer I think is the most elegant and clean. and you can even simplify it to become even more easy to use: jQuery.fn.loadExtended = function(url,completeCallback){ return this.load(url,function(responseText, textStatus, XMLHttpRequest) { if (completeCallback !== undefined && completeCallback !== null) { completeCallback(responseText, textStatus, XMLHttpRequest); } $(this).trigger("ready"); }); }; So, now instead of using: $(".container").load(url,function(responseText, textStatus, XMLHttpRequest) { $(this).trigger("ready"); }); you can just use: $(".container").loadExtended("tag_cloud.html"); or: $(".container").loadExtended("tag_cloud.html",function(){ alert('callback function') }); This has the advantage of only applying the trigger on the div that's being updated.
