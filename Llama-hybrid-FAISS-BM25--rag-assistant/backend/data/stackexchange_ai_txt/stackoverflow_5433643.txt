With a table view, the drag type for data from the view itself is, weirdly, an arbitrary string that you set, like this: #define arbitraryDataTypeForInterRowDragging @"arbitraryDataTypeForInterRowDragging". Then you use it in the acceptDrop delegate method like this: NSData *rowData = [pboard dataForType:arbitraryDataTypeForInterRowDragging]; NSCollectionView delegate methods appear to follow the same pattern.
