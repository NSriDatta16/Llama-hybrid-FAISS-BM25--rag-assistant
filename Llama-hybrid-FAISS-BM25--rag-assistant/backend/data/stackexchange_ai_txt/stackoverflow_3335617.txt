Consider storing it in the document via jQuery's data() method - global variables are always annoying to maintain. $(document).data("myVar", 12345); $(document).data("myVar"); // returns 12345 Edited: More concrete example: // soft_left = $(".ui-draggable").css('left'); $(document).data("soft_left", $(".ui-draggable").css('left')); // get it somewhere else $(document).data("soft_left") And yes, you could also use some hidden text-field - you only lose the type. Just use $('#textfield-soft-left').val($(".ui-draggable").css('left')); $('#textfield-soft-left').val(); // returns a String
