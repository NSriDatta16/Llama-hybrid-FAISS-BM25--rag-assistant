Did you try this: btnButtonTool.DataBindings.Add("Enabled", bizObj, "BoolPropertyInBizObj"); EDIT Ok, since the Enabled property is not exposed directly on ButtonTool, you could expose it yourself by creating your own ButtonTool that inherits the Infragistics one, and then implement property like this: public class MyButtonTool : ButtonTool { public bool Enabled { get { return Shared.Enabled; } set { OnPropertyChanging("Enabled"); Shared.Enabled = value; OnPropertyChanged("Enabled"); } } } I'm not sure now what ButtonTool implements (INotifyPropertyChanged/ing) but anyway you get the idea...
