Note that an OOM doesn't mean you're out of physical RAM, but that you can't fit a new memory block of the desired size into the address space. There are a number of possible causes for that Heap fragmentation the 2GB limit of a 32 bit process a maximum memory size set in the config And then there are fake OOM exceptions. For example GDI+ returns out-of-memory errors for a wide class of problems including invalid parameters. These are turned into an OOM exception too.
