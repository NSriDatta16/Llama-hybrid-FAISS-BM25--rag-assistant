You could create your own class that uses an NSMutableArray for storage. When adding an item, first check whether there are already 20 objects in the array. If there are, remove the last object from the array. Then add the new object at the front. - (void)addObject:(id)anObject { if ([dataArray count] == 20) { [dataArray removeLastObject]; } [dataArray insertObject:anObject atIndex:0]; } The above will always add the newest object at the front of the array. I guess that is what you want.
