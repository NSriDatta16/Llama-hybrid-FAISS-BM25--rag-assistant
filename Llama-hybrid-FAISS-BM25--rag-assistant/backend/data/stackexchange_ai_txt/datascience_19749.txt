It would be a good idea to create some visualizations of your data before choosing any particular method. Start with this and you might see some useful trends! For starters, I would probably make a binary classifier from logistic regression. Feed a fraction of the data you have for training and see how it performs on the rest. This is one of the simplest approaches and could work very well.
