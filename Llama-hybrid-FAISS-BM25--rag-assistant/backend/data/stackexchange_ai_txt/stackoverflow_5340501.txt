In SQL2008 you can use the elegant MERGE statement: MERGE T1 AS target USING T2 AS source ON (target.KeyC = source.KeyC) WHEN MATCHED THEN UPDATE SET c1 = source.c1, c2 = source.c2 WHEN NOT MATCHED THEN INSERT (c1, c2) VALUES (source.c1, source.c2);
