You normally use mod_rewrite to hide query strings! You can use file names like these in your pages: /dynamic-images/pie-chart/red/average-profits-in-scuba-diving-industry-19.png And have mod_rewrite translate them to: /dynamic-images.php?type=pie-chart&color=red&datasrc=19
