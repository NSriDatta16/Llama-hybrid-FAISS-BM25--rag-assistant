I'm going to throw my 2 cents in, then see if anybody comes up with anything more elegant. In the general case: List cons(E e, List list) { List res = Lists.newArrayListWithCapacity(list.size() + 1); res.add(e); res.addAll(list); return res; } With an ImmutableList (don't know how efficient this is): ImmutableList cons(E e, ImmutableList list) { return ImmutableList. builder() .add(e) .addAll(list) .build(); }
