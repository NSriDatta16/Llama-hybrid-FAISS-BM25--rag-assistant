Generally for every type of key-value pairs storage you will need separate module. For example: Memcache via Cache::Memcached CouchDB via DB::CouchDB BerkeleyDB via BerkeleyDB Amazon S3 via Amazon::S3 I'm not sure if there is module for Google AppEngine Datastore - generally, search on MetaCPAN for something that suits you.
