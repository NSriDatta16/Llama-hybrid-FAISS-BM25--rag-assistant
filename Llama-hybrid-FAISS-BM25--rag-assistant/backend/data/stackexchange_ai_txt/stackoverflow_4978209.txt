DataTable is overkill for single value retrieval, besides your not even accessing the value correctly, better way is to use execute scalar: var query = "SELECT SUM (Price) FROM Bill"; using (var cmd = new OleDbCommand(query, DBcon)) { TotalValueLabel.Text = cmd.ExecuteScalar().ToString(); }
