Have now worked it out. I iterate through a collection of data that I want the value to be stored against and use the following two LINQ queries: foreach (var data in compoundData) { var controlValue = from d in GeometricAverage where d.study == data.study where d.groupNumber == "1" select d.GeoAverage; var treatmentValue = from l in GeometricAverage where l.study == data.study where l.groupNumber == data.groupNumber select l.GeoAverage; data.TGI = CalculateTGI(controlValue, treatmentValue); }
