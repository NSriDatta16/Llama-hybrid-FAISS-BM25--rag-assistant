To detect whether a window is in incognito mode, check the incognito property of the relevant Tab or Window object. For example: var bgPage = chrome.extension.getBackgroundPage(); function saveTabData(tab, data) { if (tab.incognito) { bgPage[tab.url] = data; // Persist data ONLY in memory } else { localStorage[tab.url] = data; // OK to store data } http://code.google.com/chrome/extensions/overview.html
