Here's an applescript that will get all of the posix paths from the clipboard, not just the first one... set theFiles to paragraphs of (get the clipboard) set posixPaths to {} repeat with aFile in theFiles try tell application "Finder" to set thePath to item aFile as text set end of posixPaths to (POSIX path of thePath) end try end repeat return posixPaths
