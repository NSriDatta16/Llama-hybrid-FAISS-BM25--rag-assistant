Concurrency will bite you. As will complexity and decreased performance as the file gets larger. As will the increased risk of corruption of something happens during writing of the file like power loss of system crashes. A database will generally shield you from these issues and leave you to concentrate on the logic You can implement hierarchical storage in a database using self joins e.g ItemID Data ParentID -------------------------- Where ParentID is a pointer to ItemID of a different row
