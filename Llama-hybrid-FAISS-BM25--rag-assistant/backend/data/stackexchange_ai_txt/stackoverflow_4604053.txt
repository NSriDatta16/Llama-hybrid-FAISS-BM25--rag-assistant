You can't. Server receives only name and value of the cookie. You can set additional cookie parameters only during cookie creation. After that, user agent is responsible for managing cookies. Setting cookie without a path will implicitly use current url path as cookie path. If it's not an issue, you can set constant, common path (i.e., / ) for your cookies, and then use it to delete them.
