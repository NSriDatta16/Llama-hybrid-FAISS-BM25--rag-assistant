An elegant way to deal with trailing separation characters is to use Class Separator StringBuilder buf = new StringBuilder(); Separator sep = new Separator("\t"); for (String each: list) buf.append(sep).append(each); String s = buf.toString(); The toString method of Class Separator returns the separater, except for the first call. Thus we print the list without trailing (or in this case) leading separators.
