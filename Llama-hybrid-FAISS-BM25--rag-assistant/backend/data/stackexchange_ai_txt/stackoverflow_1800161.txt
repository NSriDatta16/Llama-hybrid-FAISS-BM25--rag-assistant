As a shameful self-advertising, I encourage to use java-apns library. Your code will look like: ApnsService service = APNS.newService() .withCert("/etc/Certificates.p12", "password") .withSandboxDestination() // or .withProductionDestination() .build(); String payload = APNS.newPayload() .alertBody("My alert message") .badge(45) .sound("default") .build(); String deviceToken = "f4201f5d8278fe39545349d0868a24a3b60ed732"; log.warn("Sending push notification..."); service.push(deviceToken, payload);
