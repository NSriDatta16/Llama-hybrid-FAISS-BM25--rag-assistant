Check if you have correct permissions : How to set read permission on the private key file of X.509 certificate from .NET You can also try adding storage flags to the certificate constructor: X509Certificate2 certificate = new X509Certificate2("certFileName", "password", X509KeyStorageFlags.PersistKeySet | X509KeyStorageFlags.MachineKeySet); Setting this permmission might help: https://serverfault.com/questions/48124/disabling-strong-private-key-encryption-on-a-personal-certificate
