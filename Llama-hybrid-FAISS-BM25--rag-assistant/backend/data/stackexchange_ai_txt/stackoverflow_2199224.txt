The IIF did the trick. I adopted the Query with the average scores to add the minimum A, B and C-scores Table(avg,stdev,Ascore,Bscore,Cscore) as averages The final query looked like SELECT user.Id, user.avgScore, IIf(avgScore>averages.Ascore,"A", IIf(avgScore>averages.Bscore,"B", IIf(avgScore>averages.Cscore,"C","D"))) AS label FROM averages, users
