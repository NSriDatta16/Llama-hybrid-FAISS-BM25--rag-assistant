I imagine it has to do with how you are creating the dialog multiple times. You might want to try creating the dialog outside of the live. Then just change the dialog content as needed and reopen the dialog. What I normally do in this situation is put a loading message, open the dialog and then start the ajax call to replace the contents. Something like this. $(function () { $("#dialogDiv").dialog({ modal: true, bgiframe: true, autoOpen: false, height: 600, width: 450, overlay: { opacity: 0.7, background: "black" }, draggable: true, resizeable: true }); $('a.click').live('click', function() { var url = '/Tracker/Info?id=' + $(this).attr("id"); $("#dialogDiv") .html('Loading') .dialog("open") .load(url); }); });
