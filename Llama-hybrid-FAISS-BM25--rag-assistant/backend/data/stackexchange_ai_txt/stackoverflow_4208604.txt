You can't update UI objects from another thread. They have to be updated in the UI thread. Try adding this code to the MainGrafik_ProgressUpdate(double progress) void MainGragfik_ProgressUpdate(double progress) { if (InvokeRequired) { BeginInvoke((MethodIvoker)(() => { MainGragfik_ProgressUpdate(progress); })); return; } ProgressBar = progress; }
