Others have already offered good explanations, but I thought I should clarify the differences between the various extensions: Source Files for C: .c Header Files for C: .h Source Files for C++: .cpp Header Files for C++: .hpp Of course, as it has already been pointed out, these are just conventions. The compiler doesn't actually pay any attention to them - it's purely for the benefit of the coder.
