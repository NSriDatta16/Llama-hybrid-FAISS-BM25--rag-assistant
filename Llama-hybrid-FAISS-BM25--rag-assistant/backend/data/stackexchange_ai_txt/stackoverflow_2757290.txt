The problem is solved once you realize that neither class A nor class B should be tyring to manage the shared data storage, as it's not exclusive to either. Hence, you are right in assuming that the Manager should do so. For the best performance, don't pass a pointer to the vector to A and B. Pass a raw float* instead, as this saves a level of indirection. A compromise would be to pass a boost::array &
