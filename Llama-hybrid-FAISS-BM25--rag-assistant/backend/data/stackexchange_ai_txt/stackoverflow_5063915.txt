Silverlight does not have any straightforward direct db access feature. You'll have either: create a service (RIA service for instance) or use COM in elevated trust mode. Both of which can potentially give you access to any db, not access dbs only. RIA services can leverage linq to entity for which you'll find implementations for SQL server, Oracle...
