In your project file add new PropertyGroup section for lite version $(DefineConstants);lite Remove lite from all definitions of DefineConstants . MSBuild.exe myprojeect1.sln MSBuild.exe myprojeect1.sln /p:LiteVersion=true You can create additional configuration in VS to switch between versions. But it can lead to configuration mismatches when you forgot to add a flag to lite config. I can suggest more elegant solution. Create .bat file or change the link to run myproject1.sln: set LiteVersion=true devenv.exe myproject1.sln
