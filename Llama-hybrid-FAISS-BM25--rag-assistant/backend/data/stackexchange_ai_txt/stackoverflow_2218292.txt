The Electronic Frontier Foundation (EFF) has set up a demo web site showing how astoundingly easy it is to identify a browser even if cookies are disabled or you are connecting from a different IP/provider: Panopticlick: How unique - and trackable is your browser They use a combination of User agent string HTTP headers Installed browser plug-ins Time zone Screen size and color resolution System fonts Cookie settings However, the typical scenario (and probably the one used in your sample application) would be to store a cookie locally and identify the returning user via this cookie.
