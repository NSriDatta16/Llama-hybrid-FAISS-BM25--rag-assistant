The easiest way is to chain successive .animate calls using queue: true , which will animate each one after the preceding animation completes. Eg.: $(this).animate( { top: "-=200"" }, { queue: true, duration: 3000 }) .animate( { top: "-=200"" }, { queue: true, duration: 3000 }) .animate( { top: "-=200"" }, { queue: true, duration: 3000 }); // And so on. A more elegant solution would be to create your own $.fn method...
