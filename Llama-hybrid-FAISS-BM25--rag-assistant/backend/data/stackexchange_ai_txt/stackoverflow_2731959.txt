The alt attribute is for alternate text; it should be short and describe the function it has or the content it presents. Here's a good read on the subject from Roger Johansson on 456 Berea Street - Writing good alt text All in all, if you're writing a paragraph, then you're doing it wrong. Look at using the longdesc attribute (which is a link to a page describing the image that complements the alt text). HTH
