So the private accessor processing was angry at a type that had partially constructed generic class definitions. For example, it failed if the class was declared like this: private class LocalNameGrouping : IGrouping { } However, it was happy with this, which still satisfied the need (albeit a tad cheesy): private class LocalNameGrouping : IGrouping where K: StorageCategory, new() { }
