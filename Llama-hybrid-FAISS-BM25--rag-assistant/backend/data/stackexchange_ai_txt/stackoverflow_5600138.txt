To change this in a custom package/theme, copy the layout file checkout.xml from $MAGENTO/app/design/frontend/base/default/layout/checkout.xml to $MAGENTO/app/design/$PACKAGE/$THEME/layout/checkout.xml Then find the following lines: in that file. Then just comment those lines out (put at the beginning of each line and put --> at the end of each line). In terms of CSS selectors, this would be: layout > default > referance[name='top.links'] > block > action
