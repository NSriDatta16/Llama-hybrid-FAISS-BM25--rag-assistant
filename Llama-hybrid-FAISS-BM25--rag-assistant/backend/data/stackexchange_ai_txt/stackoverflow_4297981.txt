You can use mktime function putting zero into the hour, minute, second fields. Then compare the results like this: function compareDate($d1, $d2) { $d1 = date_parse ($d1); $d2 = date_parse ($d2); return mktime (0, 0, 0, $d2['month'], $d2['day'], $d2['year']) - mktime (0, 0, 0, $d1['month'], $d1['day'], $d1['year']) } Pay attention to the odd order of the mktime arguments. PHP 5.3 adds the datediff istruction.
