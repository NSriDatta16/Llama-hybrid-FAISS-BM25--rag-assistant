You can use the Mobile Safari user agent string to detect mobile safari server-side, see: How do I detect Mobile Safari server side using PHP? You can also do this in JavaScript: if(navigator.userAgent.match(/(iPhone|iPod|iPad)/i)) See iPhone & iPod Detection Using JavaScript for more information.
