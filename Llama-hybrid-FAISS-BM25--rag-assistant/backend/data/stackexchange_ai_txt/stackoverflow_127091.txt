If you happen to need more formatting options (beyond paragraphs), employ something like Text_Wiki or PHP Markdown . The advantages would be: no need to allow HTML and deal with all the filtering (that's good :-)) clear/familiar guide to formatting data a lot of flexibility when you generate the HTML (in the end for display) The disadvantages: no HTML (blessing and curse ;-)) people might not be familiar the syntax
