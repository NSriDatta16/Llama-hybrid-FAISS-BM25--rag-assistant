It is rather to browser cache than undestroyed session, so you have to send some headers to prevent caching the page while user is logged in: header('Expires: '.date('r',time()-86400) ); // Date from the past header('Cache-Control: no-store, no-cache, must-revalidate, post-check=0, pre-check=0'); header('Pragma: no-cache'); [+] And if you are using cookies as sessions identifiers in iframe, be aware that Internet Explorer neds additional header header('P3P: CP="IDC DSP COR ADM DEVi TAIi PSA PSD IVAi IVDi CONi HIS OUR IND CNT"'); to set cookie from inside iframe.
