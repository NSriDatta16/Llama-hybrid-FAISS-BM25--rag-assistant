This fragment will display your results in a themed table: $sq = 'SELECT COUNT(*) cnt, DATE(FROM_UNIXTIME(timestamp)) day ' . 'FROM {comments} c ' . 'GROUP BY 2 ' . 'ORDER BY 2 DESC'; $q = db_query($sq); $stats = array(); while ($o = db_fetch_object($q)) { $stats[$o->day] = array($o->day, $o->cnt); } return theme('table', NULL, $stats)); Using DATE(timestamp) doesn't work because comments.timestamp is in UNIX_TIMESTAMP format, whereas DATE() expects an ASCII date.
