Isolated storage is another alternative. Doesn't clutter install directory Doesn't cause issues with AnitVirus software Part of the OS including .Net objects, don't need to install anything else Already works with the Windows security model Exists on a per user basis, so saved settings are separated for each user Can serialize/deserialize obects directly into it
