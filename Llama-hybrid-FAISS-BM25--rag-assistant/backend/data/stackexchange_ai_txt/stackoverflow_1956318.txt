One easy option here would be to use a recursive CTE. Make sure you finish the previous statement with a semi-colon. with recCTE as ( /* Base case first */ SELECT 1 as theLevel, * FROM theTable WHERE AgentID = 0 /* Recurisve bit */ UNION ALL SELECT r.theLevel + 1, t.* FROM recCTE r JOIN theTable t ON r.ID = r.AgentID ) SELECT * FROM recCTE WHERE theLevel
