Check out PyIMSL Studio , which wraps the IMSL C numerical libraries in Python using ctypes . It is free-as-in-beer for academic and non-commercial use. For your question, two routines are documented in Chapter 8: Optimization of the Math Guide: quadProd - Solves a quadratic programming problem subject to linear equality or inequality constraints. constrainedNlp - Solves a general nonlinear programming problem using a sequential equality constrained quadratic programming method.
