Maybe there's an easier way, but the docs say: Draggable elements gets a class of ui-draggable so you could do something like: if(!$("#foo").hasClass("ui-draggable")) { ... } so to wrap that up (untested): $.fn.isDraggable = function() { return $(this).hasClass("ui-draggable"); } console.log($("#someElement").isDraggable());
