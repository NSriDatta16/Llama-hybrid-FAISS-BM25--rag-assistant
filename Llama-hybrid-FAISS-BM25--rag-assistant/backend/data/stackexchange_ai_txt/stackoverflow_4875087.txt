Don't know if this counts as elegant, but you could use a different settings file for testing... Something like: # test_settings.py from settings import * PACKAGE_STORAGE = FileSystemStorage(location='/test/files', base_url="/") Then run your test using the test settings, python manage.py test --settings=test_settings .
