The only tool I found so far is partimat from klaR package. It's not the same as plotting projections in PCA or LDA. But it can give you an idea about the separating surface. Example: library(klaR) partimat(Species ~ ., data = iris, method = "qda", plot.matrix = TRUE, col.correct='green', col.wrong='red')
