You want a module called cgi, and a function called FieldStorage. Do it like this: import cgi url_parms = cgi.FieldStorage() session = url_parms.getvalue("session","default") You can do this for any of the URL parameters. Replace "default" with whatever you want the default value to be (like an empty string).
