We compared a lot of different results and it was all different. I run the xgb_tree it showed the max_depth is 0, but it is supposed to be 4. When I run xgb_tree$results it says the max_depth = 4 . We also tried a lot of different save methods (.rda, .rds, .model) but none of them worked. Any suggestion would be welcome, thanks.
