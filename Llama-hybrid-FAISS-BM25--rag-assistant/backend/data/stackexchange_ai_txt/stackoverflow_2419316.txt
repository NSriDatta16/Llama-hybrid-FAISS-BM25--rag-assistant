Can't you simply put the import statement in the __init__ method of each class? Then it won't be run until you try to make an instance: class UnsatisfiedExample(object): def __init__(self): try: import flibbertigibbet except ImportError: raise RuntimeError("You need to install flibbertigibbet-py") super(UnsatisfiedExample, self).__init__()
