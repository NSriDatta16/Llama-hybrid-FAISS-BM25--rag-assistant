In ASP.NET when you click a button, you're posting the entire page's fields by default (as it's contained within a gigantic tag last time I checked. You can access these values after clicking the button like this: string MyPostedValue = Request.Form["MyFormField"]; *Edit as per your update in your question, change Response.Redirect() to Server.Transfer() like this: protected void btnSubmitForPost_Click(object sender, EventArgs e) { Server.Transfer("GetViaPost.aspx", true); } Then in your GetViaPost.aspx's page you can get any form/query string variable you passed from your sending page like this: string MyPostedValue = Request.Form["MyFormField"];
