Pasting PHP code into a plugin isn't a good idea. There are three approaches to embedding PHP scripts into a WordPress site: Placing the code directly into a theme's main index template (messy) Placing the code into a theme's functions.php file and hooking it into the site using the Plugin API . (cleaner) Fully encapsulating the code in a plugin . (cleanest)
