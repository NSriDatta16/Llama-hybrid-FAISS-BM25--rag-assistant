selectedChild expects the child itself, not the label. Instead - try this: public function removeContent():void { Viewstack(this.parent).selectedIndex = 0; this.parent.removeChild(this); } Note - it's generally reccomended to avoid using FlexGlobals.topLevelApplication , as it leads to a very tightly coupled & fragile application.
