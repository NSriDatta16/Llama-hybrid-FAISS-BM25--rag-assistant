First, $tmp data type is Result Tree Fragment. So, besides copying, in all allowed operations with $tmp , only counts its string value. XPath have many rules for implicit casting. In general, whenever an operator or function takes a number data type as argument, the expression will be cast to number with number() function. Bottom line: in mostly every case you don't need that explicit casting.
