I think what you're looking for is itertools.groupby : import itertools def average_duplicates(names, values): pairs = sorted(zip(names, values)) result = {} for key, group in itertools.groupby(pairs, key=lambda p: p[0]): group_values = [value for (_, value) in group] result[key] = sum(group_values) / len(group_values) return result See also zip and sorted .
