You are talking about variable or feature selection , of which the literature is vast. Various methods exist, which can be broadly characterized in three categories: Filter methods : these methods typically use univariate statistics to filter out or select features in function of your target variable. Wrapper methods : these methods incrementally add/remove features which are best/worst performing. Embedded methods : these methods instantly give you a score along with your prediction. As the answer above, Random Forests for example, will be able to give you along with your trained model an output which is called variable importances .
