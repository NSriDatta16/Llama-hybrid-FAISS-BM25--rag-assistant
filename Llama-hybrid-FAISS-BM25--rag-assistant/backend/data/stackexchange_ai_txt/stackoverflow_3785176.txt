If you mean Lucene search, you could take a look at Examine it's a non-enterprise Lucene search. If you need your search as a service then use Solr, you'll need to install a Solr server . For that you'll need to install Tomcat or another servlet container and create a Solr instance. Once you've done that you can post XML fragments from your code to create your index as you would normally perform HttpPost's. Then you use HttpGet requests to search the index .
