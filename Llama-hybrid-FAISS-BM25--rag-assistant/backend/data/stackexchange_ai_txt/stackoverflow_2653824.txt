UserAgent is under the Request property. By default, Indy's user agent string is "Mozilla/3.0 (compatible; Indy Library)". Change it to a current string used by FireFox, IE or other recognized browser. You can find examples on the UserAgentString site. There are more details in the Indy KB PDF .
