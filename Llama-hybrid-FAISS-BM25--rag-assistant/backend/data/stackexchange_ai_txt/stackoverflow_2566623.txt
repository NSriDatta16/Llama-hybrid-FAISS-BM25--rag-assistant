I found the answer here: http://wiki.eclipse.org/FAQ_How_do_I_add_Content_Assist_to_my_editor%3F In Ganymede it's necessary to override createActions to register the content assist one: Action action = new ContentAssistAction(resourceBundle, "ContentAssistProposal.", this); String id = ITextEditorActionDefinitionIds.CONTENT_ASSIST_PROPOSALS action.setActionDefinitionId(id); setAction("ContentAssistProposal", action); markAsStateDependentAction("ContentAssistProposal", true); That was not needed in Galileo.
