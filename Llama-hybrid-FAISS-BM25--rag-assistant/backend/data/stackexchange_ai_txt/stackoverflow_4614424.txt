you can add external jar file using properties file. You have to use grinder.jvm.classpath property. When you specify this property, it is added to class path every time grinder agent starts. grinder.jvm.classpath=/lib/jyson-1.0.1.jar:/home/lib/my-loadtest-0.0.1-SNAPSHOT.jar
