The DateTime module is well-suited to the needs of this problem: use strict; use warnings; use DateTime; my $oneHourAgo = DateTime->now()->subtract( hours => 1 ); my $threshold = join ' ', $oneHourAgo->ymd, $oneHourAgo->hms; # Time as string open my $logFile, ' ) { chomp $log; my ($time) = split /,/, $log; # Gets current log's time print $log if $time ge $threshold; # String-compares log's time to threshold } close $logFile;
