Use TranslateTransform to place the shapes. Use the RenderTransformOrigin to set the 'center'. EDIT RenderTransforms can be accelerated by the GPU. You wouldn't add a transformation each frame but add just one and change its properties over time. See BitmapCache and this is also quite interesting
