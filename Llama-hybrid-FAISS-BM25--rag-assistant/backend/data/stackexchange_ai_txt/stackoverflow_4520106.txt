Yes, it's OK. Every DOM member can have an ID. There is nothing fundamentally wrong with a tag using an ID. Now, for applying CSS may be the case that is better for you using a class attribute instead of an id in case because you might want to apply that style to several paragraphs. But it's just a matter of convenience.
