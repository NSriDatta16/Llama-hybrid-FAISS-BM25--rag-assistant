There are several ways and some depend on the type of device you have. There are several tools for installing driver packages. DpInst is a complete application which can show a wizard and be customized to install a driver package DifXApp builds a msi package which can be used to install drivers DifxApi is the API which DpInst and DifxApp use to install drivers. Directly using the SetupApi functions. Here the functions SetupCopyOEMInf and UpdateDriverForPlugAndPlayDevices provide the corresponding entry points for a driver setup. These are contained in the WinSDK. DpInst/DifxApp/DifxApi are part of the Windows Driver Kit (WDK).
