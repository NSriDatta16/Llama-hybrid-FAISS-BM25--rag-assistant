The order they're inserted isn't that important since you will likely be using the ORDER BY clause in your queries when pulling the data back out. Remember, MySQL is for storing the data, not necessarily presenting it. SELECT col1, col2 FROM table1 ORDER BY col2 ASC // fixes any ordering issues in the native-storage
