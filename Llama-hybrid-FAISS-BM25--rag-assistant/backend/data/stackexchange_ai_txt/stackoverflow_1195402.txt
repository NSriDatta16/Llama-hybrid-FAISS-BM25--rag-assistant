enum ShippingType : int { Normal = 0, Fragile = 1 } double ShippingPrice(Product p) { if (p.ShippingType == ShippingType.Fragile) // Fragile return GetFragileShippingPrice(p); else return GetNormalShippingPrice(p); } Then you would have the enum auto-generated (created by something like CodeSmith) from a table in your database.
