There are almost certainly more elegant solutions, but this command line hack gets the job done. $ perl -i~ -pe 's/ver-1.0.js/ver-2.0.js/g' *.html That will make the change in all of the html files in the current directory. And you'll have a set of back-up files with the extension .html~.
