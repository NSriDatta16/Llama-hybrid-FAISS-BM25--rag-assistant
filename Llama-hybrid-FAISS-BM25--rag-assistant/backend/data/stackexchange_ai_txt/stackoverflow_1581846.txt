Depending on what the code looks like that you want to include, and if you really have no other choice like the ones presented by Druid, you can use one of the following less-than-perfect embedding solutions: An IFRAME with frameborder="0" pointing to the PHP script (Downside: dynamic resizing according to output size is possible only with JavaScript An AJAX Snippet that loads the PHP file onLoad and injects the output into a DIV of your choice A tag pointing to the PHP file. The PHP file must then issue JavaScript commands like "document.write()". The IFRAME is the most fail-safe in the event the client has no JavaScript. The AJAX Snippet is the most elegant solution.
