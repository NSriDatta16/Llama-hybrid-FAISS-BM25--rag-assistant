if you want to get your hands dirty and really understand how the web works, go MVC if you want to drag and drop your way to a functional but overhead loaded website, use webforms. really, this question is pretty difficult to answer not knowing your background. if you're comfortable with html, css, javascript it may not be too difficult to pick up MVC. if you're new to the web entirely, it can be daunting to learn that many technologies all at once and webforms abstracts a lot of that kind of stuff for you.
