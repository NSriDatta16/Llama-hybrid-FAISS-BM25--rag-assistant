Python support for hierarchical time series is pretty poor. What you want to do is developed very nicely in R Hierarchical/Grouped time series However, there are active/dead developments of the same/similar in python that you can take a look. pyaf has an implementation of hierarchical time series scikit-hts is the python implementation of Hyndman's work. Hierarchical forecast Documentation is pretty poor and each of them will have specific bugs around them. Might not be reliable for commercial implementation.
