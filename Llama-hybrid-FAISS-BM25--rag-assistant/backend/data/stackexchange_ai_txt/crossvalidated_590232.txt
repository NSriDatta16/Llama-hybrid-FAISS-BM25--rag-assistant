The key question you have to ask yourself first is: what is the aim of the Reinforcement Learning (RL) agent? Phrased another way - how should it select questions? RL learns to maximize a reward signal, so you need to know what the aim of the agent is to set that reward. There are many RL methods you could choose from, here's a description from OpenAI of a lot of them. Alternatively, if you'd like to learn RL in more depth, here's a practical course I worked on which teaches many of these along with exercises in Python.
