The column you are using must be of string data type. First filter the dataset where it contain "-" and create another dataframe. On this new dataframe perform split string operation and create two different column and convert them to integer. Now create another column which has average or whatever you prefer with two column. Rename column and drop unnecessary columns. Merge it back to dataframe.
