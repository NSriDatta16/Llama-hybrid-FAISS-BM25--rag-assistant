Unfortunately, yes. You probably have to cache all the values of interest to your code, and update this cache yourself whenever you get a change trigger, or else set up multiple watchers, one on each of the individual data items of interest. As you noted the second solution gets unwieldy very quickly. If you can implement the required code in .Net you can get the same effect more elegantly via RegistryEvent and its subclasses .
