Replace $(this).append($(ui.draggable).clone().attr('alt', 'nat')); with $(this).append($(ui.draggable).clone(true).attr('alt', 'nat')); The true value is used to Clone matched DOM Elements, and all their event handlers, and select the clones. Read the doc clone( bool )
