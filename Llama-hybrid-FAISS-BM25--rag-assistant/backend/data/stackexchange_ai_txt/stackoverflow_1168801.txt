This is basically just going out of your way to avoid a list comprehension, but if you like functional style programming, then you'll like functools.partial . >>> from functools import partial >>> def counter(text, paragraph): return len(re.findall(text, paragraph)) >>> tester = partial(counter, paragraph="I eat bananas and a banana") >>> map(tester, ['banana', 'loganberry', 'passion fruit']) [2, 0, 0]
