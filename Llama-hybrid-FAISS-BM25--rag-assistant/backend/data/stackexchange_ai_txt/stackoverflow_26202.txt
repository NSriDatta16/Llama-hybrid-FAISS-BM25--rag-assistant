The problem with the System.Xml approach in my case was that it required too much code to build this simple XML fragment. I think I've found a compromise. XmlDocument doc = new XmlDocument(); doc.InnerXml = @" /Value> "; XmlNode valueNode = doc.SelectSingleNode("Where/Eq/Field/Value"); valueNode.InnerText = @"Text <>!$% value>";
