Doing large file transfers of HTTP is not usually fun -- but sometimes it's necessary. For large files, you'll definitely want to provide some kind of progress gauge for end-users. There are flash-based tools that do this ( swfUpload comes to mind). If you want to avoid flash and do it with pretty html/javascript/css, you can leverage PHP's APC extension, which for some reason provides support for getting upload status from the server, as explained here
