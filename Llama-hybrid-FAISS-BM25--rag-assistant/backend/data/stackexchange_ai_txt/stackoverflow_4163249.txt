If you want to avoid code behind or custom controls (preferable), this sort of functionality is available via behaviours. e.g. this one in the Expression blend Gallery converts any text box to which it is attached into uppercase. Then is just a matter of drag-drop in Blend or adding the trigger manually in VS2010.
