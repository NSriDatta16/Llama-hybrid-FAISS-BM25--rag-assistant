It sounds to me like you need to use jQuery UI Droppable as well as Draggable. The following code provides an example of how to use Droppable. It assumes the divs you want to allow items on have class name someClassName . When you drag an item onto one of the divs, it pops up an alert box with the ID of the div you dropped your item onto: $(document).ready(function() { $("div.someClassName").droppable({ drop: function(event, ui) { alert("You dropped an item onto a div with ID '" + event.target.id + "'"); } }); });
