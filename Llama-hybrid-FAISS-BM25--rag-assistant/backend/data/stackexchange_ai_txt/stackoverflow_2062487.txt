Command definitions can be nested; double the # for each level. \newcommand\newchar[2]{% \newcommand #1 [1] {% \raggedright #2: \\ \hspace{0.5in} ##1% }% } Update: Just a comment about the # doubling. This makes more sense when you're defining macros using the \def primitive; in this case, the general construction is something like \def\foo{% \def\bar##1{bar: ##1}% }
