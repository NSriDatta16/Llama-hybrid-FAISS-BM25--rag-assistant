Letting users to input the embedding HTML is BAD PRACTICE , and leaves a security hole in your web site. (Just imagine if some user enters some random HTML or an evil piece of JavaScript.) The real solution would be to ask the user for the URL of the video and then constructing the embed HTML yourself. If you examine the youtube embedding HTMLs, you'll see it is not difficult.
