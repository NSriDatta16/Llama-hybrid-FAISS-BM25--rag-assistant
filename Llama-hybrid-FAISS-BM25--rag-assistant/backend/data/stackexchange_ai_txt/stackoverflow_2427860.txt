For Visual C++ you need to use the #pragma warning directive. The warning you are getting is C4003 (C => Compiler), 4003 => warning number. #pragma warning (disable: 4003) #define B(X) A1##X int main() { int B(123), B(); return 0; } Not sure about GCC, but I suspect you can opt to not define this pragma for GCC and suppress the warning (if any some other way).
