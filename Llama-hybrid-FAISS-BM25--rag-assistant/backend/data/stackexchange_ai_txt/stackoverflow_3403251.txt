If you are using Cygwin on your local host, you can connect to Cygwin's sshd (SSH Daemon). 3 easy commands to install the sshd as a service: (always there when you startup the machine) $ ssh-host-config -y # enable sshd. "-y" answers yes to all queries. $ cygrunsrv --start sshd # start it as an autostart Windows service $ ssh localhost # try to recursively log in via ssh Taken from: http://nfnaaron.posterous.com/tag/puttycyg#Section2 (The second approach) => You get the entire ssh infrastructure: public key authentication, ssh-agent, etc...
