It sounds like you may be stuck embedding tooltip content in the label elements - but if you're free to move the content into the alt attribute of your input elements, you can do what you want with qTip , like this: and // content: false tells qtip to use the selected elements' alt text $('.tooltipped').qtip({ content: false, show: { when: { event: 'focus' } } });
