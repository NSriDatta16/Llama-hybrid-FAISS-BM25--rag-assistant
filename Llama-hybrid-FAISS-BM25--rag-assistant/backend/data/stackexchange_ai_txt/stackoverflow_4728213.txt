Before setting the background color to the new value, store it as a property on that element for later retrieval like so. var el = document.getElementsByName(formId)[0].title; el._originalBackgroundColor = el.style.backgroundColor; // Set the new color el.style.backgroundColor = "#7FB75E"; // Set it back to original el.style.backgroundColor = el._originalBackgroundColor;
