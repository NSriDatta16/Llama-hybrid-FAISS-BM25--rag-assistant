Here's a simple example:- Hibernate mapping file { call get_balance_sp :name } Code List list = sessionFactory.getCurrentSession() .getNamedQuery("mySp") .setParameter("name", name) .setResultTransformer(Transformers.aliasToBean(MyBean.class)) .list(); Bean class This bean holds the results from the stored procedure. The field names must match the column names from the Hibernate mapping file. public class MyBean { private Date date; private Long balance; private String name_of_person; // getters and setters }
