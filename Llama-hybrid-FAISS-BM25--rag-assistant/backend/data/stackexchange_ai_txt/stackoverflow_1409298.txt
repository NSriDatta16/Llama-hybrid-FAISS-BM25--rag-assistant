Try sending a random hash as parameter in the url, so each request gets "different" to the browser .. also, if possible, print some headers in the file your calling with metatags to prevent caching, like this (in case it's php) : header( "Expires: Mon, 20 Dec 1998 01:00:00 GMT" ); header( "Last-Modified: " . gmdate("D, d M Y H:i:s") . " GMT" ); header( "Cache-Control: no-cache, must-revalidate" ); header( "Pragma: no-cache" );
