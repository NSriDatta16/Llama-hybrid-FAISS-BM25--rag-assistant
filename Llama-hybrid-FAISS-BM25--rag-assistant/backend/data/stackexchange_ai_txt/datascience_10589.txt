Number of parameters in an RNN

I'm using a basic RNN as in the figure below (say for translation). The model has the following structure: \begin{aligned} s_t &= \tanh(Ux_t + Ws_{t-1}) \\ o_t &= \mathrm{softmax}(Vs_t) \end{aligned} Assume that the vocabulary size is $m$ and that of the hidden layer is $n$ . If $x_{t}=\{0,1\}^{m}$ and U is a $ n \times m$ matrix then W is a $ n \times n $ matrix. If $o_{t}$ is $\mathbb{R}^{k}$ and $s_{t}$ is $\mathbb{R}^{n}$ then V is a $ k \times n $ matrix. What are the # parameters for this RNN model?
