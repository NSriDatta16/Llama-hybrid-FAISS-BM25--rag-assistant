According to Testing Standards 's glossary , Structured Basic Testing is a technique to design test cases from the logic of the code aiming at maximizing the test coverage. It just seems to be another name for code/logic coverage testing. The same definition can be found on Independent Test glossary, with a reference to BCS SIGIST .
