You can make use of conditional compilation to determine if the client is using MSIE. var IE = /*@cc_on!@*/false; which can be used as if (IE) { // IE. } else { // Others. } Only in IE, the ! will be compiled and taken in the expression, resulting in a new expression !false , which is logically true . This works better than $.browser.msie because it can be fooled by the useragent and also better than document.all because it would affect certain Opera versions as well. That said, what is it you're trying to disable? You can on the other hand also make use of feature detection. Here's a discussion about this: Browser detection versus feature detection
