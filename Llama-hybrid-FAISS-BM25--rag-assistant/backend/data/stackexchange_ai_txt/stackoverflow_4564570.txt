The reason the color isn't being applied is because :first-line creates a "pseudo-element" around the first line of a paragraph. So, while the code looks like this: november 30 Rod Dibble The browser ends up interpreting it like this: november 30 Rod Dibble Even though you've declared the color property as important, the CSS selector it's declared under ( #main p.byline ) is not as specific to the first line pseudo-element as the one being declared for with the #main p:first-line selector. The W3 docs go into much greater detail: The :first-line pseudo-selector
