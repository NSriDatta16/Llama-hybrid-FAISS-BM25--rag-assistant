Opening a web page with the default web browser is easy: java.awt.Desktop.getDesktop().browse(theURI); Embedding a browser is not so easy. JEditorPane has some HTML ability (if I remember my limited Swing-knowledge correctly), but it's very limited and not suiteable for a general-purpose browser.
