Add this on the Drag enter event (this will change the cursor type when you are dragging a file) private void Form1_DragEnter(object sender, DragEventArgs e) { // If file is dragged, show cursor "Drop allowed" if (e.Data.GetDataPresent(DataFormats.FileDrop)) e.Effect = DragDropEffects.Copy; else e.Effect = DragDropEffects.None; } Then on the DragDrop event you need to handle what do ou want to do. And also set the AllowDrop property to true
