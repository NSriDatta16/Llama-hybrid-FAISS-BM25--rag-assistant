Why it is so large?? All styles for Infragistics controls (brushes, controls styles etc) are less than 10 Mb. You should to consider to use styles inheritance as frequently as possible. Also you should split this file to few smaller files. It is torture to work with so large file. Try to have all files smaller than for example 1000 lines.
