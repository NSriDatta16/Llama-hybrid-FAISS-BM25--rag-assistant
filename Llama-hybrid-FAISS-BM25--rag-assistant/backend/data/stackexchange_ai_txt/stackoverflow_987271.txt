In urllib , it's done like this: import urllib class AppURLopener(urllib.FancyURLopener): version = "MyStrangeUserAgent" urllib._urlopener = AppURLopener() and then just use urllib.urlopen normally. In urllib2 , use req = urllib2.Request(...) with a parameter of headers=somedict to set all the headers you want (including user agent) in the new request object req that you make, and urllib2.urlopen(req) . Other ways of sending HTTP requests have other ways of specifying headers, of course.
