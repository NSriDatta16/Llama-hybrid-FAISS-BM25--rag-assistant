An elegant way to specify the width would be theText.width = AdvancedDataGrid(handle.owner).columns[handle.columnIndex].width; where var handle:AdvancedDataGridListData = AdvancedDataGridListData(listData); So the item renderer can be reused, instead of tying to specific column with hard-code.
