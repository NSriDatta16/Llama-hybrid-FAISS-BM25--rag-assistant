Amazon S3 (Simple Storage Service) is an online strongly consistent storage web service offered by Amazon Web Services . Amazon S3 provides storage through a simple web services interface. It gives any developer access to the same highly scalable, reliable, secure, fast, inexpensive infrastructure that Amazon uses to run its own global network of web sites.
