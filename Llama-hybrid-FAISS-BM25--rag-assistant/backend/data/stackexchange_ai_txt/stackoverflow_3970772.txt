clean & minimal code (no fancy optimizations, max. a few thousand sloc) C/C++ open source The venerable Neural Networks at your Fingertips (or archived here ) is written very cleanly in C, it's very easy to play with and contains implementations for a number of 'popular' architectures (since you don't specify what neural network architecture(s) you're interested in). ample testing data You can find ample testing data (amongst other places) at the UC Irvine Machine Learning Repository .
