"Perhaps send JSON from the server but also include the HTML snippet as an attribute so the Javascript can do the filtering, sorting, etc. and then just insert the given fragment?" I recommend keeping the presentation layer on the client and simply downloading data as needed. For a rudimentary templating engine, you can extend Prototype's Template construct: http://www.prototypejs.org/api/template As your client scales and you need a rich and flexible MVC, try PureMVC. http://puremvc.org/content/view/102/181/
