I ended up using this transformer: @interface NullTransformer : NSValueTransformer {} @end @implementation NullTransformer + (BOOL) allowsReverseTransformation { return YES; } - (id) transformedValue: (id) value { return [value count] ? value : nil; } - (id) reverseTransformedValue: (id) value { return value ? value : [NSDictionary dictionary]; } @end In other words, I use an empty dictionary to represent unset values and convert between this special value and nil using a custom transformer so that my classes don’t have to care. It’s not perfect, but seems like a tolerable solution.
