Check out http://www.evanfosmark.com/2009/01/cross-platform-file-locking-support-in-python/ You can use a simple lock with his code, as written by Evan Fosmark in an older StackOverflow question: from filelock import FileLock with FileLock("myfile.txt"): # work with the file as it is now locked print("Lock acquired.") One of the more elegant libraries I've ever seen.
