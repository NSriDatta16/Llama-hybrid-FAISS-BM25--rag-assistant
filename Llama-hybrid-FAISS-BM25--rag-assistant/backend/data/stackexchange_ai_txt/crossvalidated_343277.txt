Moving average (MA) process with nonzero coefficients at lags 7, 14, 21, etc. is just what you need. I.e. $$ x_t=u_t+\theta_7 u_{t-7}+\theta_{14} u_{t-14}+\theta_{21} u_{t-21} $$ where $u_t$ is i.i.d. noise and the $\theta$s are scalars. Below is an example code for R: n=1e4 # sample size theta=rep(0,21); theta[7]=0.4; theta[14]=-0.6; theta[21]=0.8 # coefficients theta set.seed(1) # fix seed for random number generation u=rnorm(n) # generated i.i.d. standard normal noise x=u[1:(n-21)]+theta[7]*u[(1+7):(n-21+7)]+theta[14]*u[(1+14):(n-21+14)]+theta[21]*u[(1+21):n] # generate the MA process acf(x) # autocorrelation function (produces a graph)
