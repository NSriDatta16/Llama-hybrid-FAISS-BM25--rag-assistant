You would have to go the route of: Super s = new Super(); s.anotherMethod("Test"); ...but that will defeat the purpose of inheritance if you also need whatever Sub's got. You could hack it like below but this seems an unelegant way to do it. class Sub extends Super { public String anotherMethod( String s, boolean bSuper ) { if( bSuper ) return super.retValue(s); else return retValue(s); } public String retValue(String s) { return "Sub " + s; } }
