I have used REXML for tasks like this successfully in the past. The following script will extract the GUID values: require "rexml/document" doc = REXML::Document.new File.open('doc.xml') guids = doc.root.get_elements('//ShippingMethodId').map { |element| element.get_text } assuming a file name 'doc.xml' or you can just pass in the XML string instead of a file. You will have to wrap the xml fragment in a single root element to make it well-formed XML before REXML will parse it: Description Goes Here 16.98 GUID Express Overnight Description2 Goes Here 19.98 GUID2 More Express Overnight
