Do this: str == str.upcase E.g: str = "DOG" str == str.upcase # true str = "cat" str == str.upcase # false Hence the code for your scenario will be: # In the code below `upcase` is required after `str.pluralize` to transform # DOGs to DOGS str = str.pluralize.upcase if str == str.upcase
