You don't need ajax, using only on-page javascript would be enough. However, with ajax you can display the content from the page you're submitting the form to. Try the jQuery From plugin for an elegant ajax solution: $(document).ready(function() { $('#myForm').ajaxForm({ target: '#divToUpdate', url: 'comment.php', success: function() { alert('Thanks for your comment!'); } }); });
