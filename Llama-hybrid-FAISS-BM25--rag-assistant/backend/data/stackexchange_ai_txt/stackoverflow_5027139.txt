It's as simple that you can inherit properties from another javascript object like so: var a = function () {} a.prototype.sayHello = function () { alert('hello') } var b = function () {} b.prototype = new a(); var c = new b(); c.sayHello(); // Alerts "hello" John Resig of jQuery has an exellent guide about it http://ejohn.org/apps/learn/ about inheritance http://ejohn.org/apps/learn/#76 EDIT Updated code based on Roberts comment.
