For the Bayesian side of things there is a really outstanding book called Bayesian Core ( amazon page ) with an emphasis on practice . It begins with an introduction of grad level basics like normal distributions and quickly moves to more advanced topics like linear model, regression, mixture models etc with a lot of programming listings in R or Matlab that help grab the essential feeling of the course.
