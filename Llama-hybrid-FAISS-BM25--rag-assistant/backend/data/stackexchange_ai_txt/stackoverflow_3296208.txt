Check out the axis option: "Constrains dragging to either the horizontal (x) or vertical (y) axis. Possible values: 'x', 'y'." Examples: // Initialize a draggable with the axis option specified. $( ".selector" ).draggable({ axis: 'x' }); // Get the axis option, after initialization. var axis = $('.selector').draggable('option', 'axis'); // Set the axis option, after initialization. $('.selector').draggable('option', 'axis', 'x');
