uniq(1) SYNOPSIS uniq [OPTION]... [INPUT [OUTPUT]] DESCRIPTION Discard all but one of successive identical lines from INPUT (or standard input), writing to OUTPUT (or standard output). Or, if you want to remove non-adjacent duplicate lines as well, this fragment of perl will do it: while(<>) { print $_ if (!$seen{$_}); $seen{$_}=1; }
