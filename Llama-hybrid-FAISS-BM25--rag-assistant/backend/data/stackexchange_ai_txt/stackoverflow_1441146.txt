No. The closest is __defineGetter__ available in Firefox, which defines a function callback to invoke whenever the specified property is read: navigator.__defineGetter__('userAgent', function(){ return 'foo' // customized user agent }); navigator.userAgent; // 'foo' It differs from __getattr__ in that it is called for a known property, rather than as a generic lookup for an unknown property.
