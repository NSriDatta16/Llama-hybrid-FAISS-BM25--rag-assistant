You can use server side detection, but as you've not specified what language you are using, I can only suggest a frontend (javascript) method. jQuery has a utility function jQuery.browser() which will return one of a number of "flags" based on the useragent string. Take a look at the docs at the link below for more information. jQuery.browser() @ jQuery Docs
