Page A is probably cached in your browser. You get this cached version when clicking back. Try forcing the browser not to cache the page in the the code behind (of page A) with: Response.Cache.SetCacheability(HttpCacheability.NoCache); Response.Cache.SetExpires(DateTime.Now - new TimeSpan(1, 0, 0)); Response.Cache.SetLastModified(DateTime.Now); Response.Cache.SetAllowResponseInBrowserHistory(false);
