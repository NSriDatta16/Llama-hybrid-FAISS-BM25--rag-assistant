See an example of the following here. You can find the index() of the [box] paragraph and then wrapAll() the after using :gt() to get all the parapgraphs following it: var boxAt; $('p').each(function(){ var $t = $(this); if ($t.html() === '[box]') { boxAt = $t.index(); $t.remove(); } }); $('p:gt(' + (boxAt - 1) + ')').wrapAll(' ');
