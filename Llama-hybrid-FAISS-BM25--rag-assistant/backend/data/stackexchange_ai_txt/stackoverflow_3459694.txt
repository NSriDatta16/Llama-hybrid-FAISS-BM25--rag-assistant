This is normally done when embedding Quicktime movies. It can be done directly with the video if you are able to set its poster frame. It is also commonly done by placing an image on the screen in the same place as the video and switching between them. The code could look something like this. HTML ... ... ... JS function switchToVideo(){ document.getElementById('video').style.display = "block"; document.getElementById('poster').style.display = "none"; document.getElementById('video').Play(); }
