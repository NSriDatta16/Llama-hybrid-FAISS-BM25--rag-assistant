Applying transformation of `GatherBy` to a different list

I have listA and listB of the same size. I'm doing GatherBy on listA , which rearranges that list. What is an elegant way to apply identical rearrangement to listB ? For example listA = {1, 2, 3}; listB = {a, b, c}; listA1 = GatherBy[{1, 2, 3}, OddQ]; listB1 should become {{a, c}, {b}} Update Thanks for interesting ideas, I eventually ended up doing something similar to belisarius. This reminds me of Python's "decorate-sort-undecorate" pattern decorated = Thread[{listA, listB}]; grouped = GatherBy[decorated, OddQ[First[#]] &]; listB1 = Map[Last, grouped, {2}]
