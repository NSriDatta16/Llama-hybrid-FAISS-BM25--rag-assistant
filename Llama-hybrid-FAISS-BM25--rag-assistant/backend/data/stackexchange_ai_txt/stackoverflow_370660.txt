You will need to know Bytes sent from the beginning (To calculate average rate of transfer) Bytes sent since last second (To calculate current transfer speed.) This can be easily done if you know bytes sent from the beginning. (New bytes sent from the beginning - Bytes sent from the beginning 1 second ago) Total file size (To calculate percentage of progress.) (Bytes transfered / Total number of bytes)
