Ok, I figured out a moving average solution which is fast without changing the data structures: def updateCentroid(self, label): cluster = self.clusters[label] n = len(cluster.points) cluster.centroid.x = ((n-1)*cluster.centroid.x + cluster.points[n-1].x)/n cluster.centroid.y = ((n-1)*cluster.centroid.y + cluster.points[n-1].y)/n This lowered computation time (for the whole k means algorithm) to 13% of original. =) Thank you all for some great insight!
