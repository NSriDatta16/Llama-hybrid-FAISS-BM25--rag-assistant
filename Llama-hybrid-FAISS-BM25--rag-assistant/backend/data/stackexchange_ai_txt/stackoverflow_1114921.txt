This page might be of interest to you: " Demonstration of Different Ways to Play a Sound from a Web Page " It covers this: Normal Hyperlink to a Sound File Embedding a Sound File Controlling an Embedded Sound using JavaScript (New) Using a Background Sound and JavaScript Using Dynamic HTML Using a Java Applet Using a Java Applet called from JavaScript Increasing compatibility with Real Player It doesn't cover Flash , and you don't seem to want to use it. However, nowadays flash has a large install base. It's not unreasonable to ask visitors to have flash installed if you're going to do advanced audio stuff on your page. Flash works properly on almost every OS and browser that you can think of (unlike silverlight), so I'd definitely consider using it.
