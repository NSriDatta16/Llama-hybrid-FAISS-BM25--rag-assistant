You need two statements to prevent caching: Response.Cache.SetCacheability(System.Web.HttpCacheability.NoCache); // For IE Response.Cache.SetNoStore(); // For Firefox For more detail see: http://blog.httpwatch.com/2008/10/15/two-important-differences-between-firefox-and-ie-caching/ Also, make sure that you always follow a POST with a redirect so that the back button works correctly: http://blog.httpwatch.com/2007/10/03/60-of-web-users-can%E2%80%99t-be-wrong-%E2%80%93-don%E2%80%99t-break-the-back-button/
