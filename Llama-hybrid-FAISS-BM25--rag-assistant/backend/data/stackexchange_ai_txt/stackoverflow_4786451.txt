This will depend on what dr is. The first invokes the ToString method that each object could override on the indexer of the dr["RAGStatusCID"] . The second casts the dr["Description"] variable to a string. If we use our telepathic skills and suppose that dr is a SqlDataReader then the two expression do pretty much the same thing except that the cast might throw an exception at runtime if the Description column is not a string.
