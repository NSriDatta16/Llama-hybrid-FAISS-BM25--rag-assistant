The background-position style is a composite style, so when you read it, it might not give the result that you expect. Also, the result may differ between browers. Try using a variable to keep track of the position instead of reading it from the style. Set the position before you start the fade in: $(function(){ var $slogan = $('#header h2 span'); var offset = 0; window.setInterval(function(){ $slogan.css('background-position', '0 -'+offset+'px') .fadeIn('slow') .fadeOut('slow'); offset = (offset + 41) % 123; }, 2000); });
