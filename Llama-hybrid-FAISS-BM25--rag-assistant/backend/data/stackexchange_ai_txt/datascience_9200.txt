I guess classification is a better term here. One idea would be to calculate histogram of an image. Make 32 bins from the histogram. So if an image has Red, Blue, Green channels, each having 256 values, you will get a vector of 32*3=96 dimensions. You can run SVM or Naive Bayes on it.
