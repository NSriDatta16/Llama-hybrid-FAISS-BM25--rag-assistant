Looks like you're working on a machine that has some security policies in place that restrict you from installing software yourself. This is very common in most organizations nowadays. Do the following: Uninstall tomcat. Find a folder where you can read/write into (eg. D:\neetu) Install tomcat in the above folder. Run tomcat. It should work now.
