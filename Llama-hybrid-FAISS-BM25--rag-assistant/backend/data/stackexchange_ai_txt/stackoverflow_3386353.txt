It works correctly. The facebook window closes mean your user login successfully. You should handle some delegate: #pragma mark FBSessionDelegate - (void)session:(FBSession*)session didLogin:(FBUID)uid { } - (void)sessionDidNotLogin:(FBSession*)session { } - (void)session:(FBSession*)session willLogout:(FBUID)uid { } - (void)sessionDidLogout:(FBSession*)session { // NCLog(@"FacebookHelper logged out"); } Then you will know what happens to the login process
