There is a difference in embedding with the Flex SDK you are using (embedAsCFF propertie). Flex 4.x SDK [Embed(source="../lib/CustomFont.ttf", embedAsCFF="false", fontName="CustomFont", mimeType="application/x-font")] public var CustomFont:Class; Flex 3.x SDK [Embed(source="../lib/CustomFont.ttf", fontName="CustomFont", mimeType="application/x-font")] public var CustomFont:Class; Note1: Don't forget to set textfield.embedFonts = true Note2: You cannot use a swf as font, it should be an .TTF or .OTF file format reference source: http://www.bit-101.com/blog/?p=2555
