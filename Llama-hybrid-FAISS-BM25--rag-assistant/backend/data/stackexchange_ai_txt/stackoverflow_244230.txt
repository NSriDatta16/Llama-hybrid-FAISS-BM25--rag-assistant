Putting a pragma -No Cache- will prevent the .xap from being stored on the machine, instead it will be streamed by the Silverlight plugin. Without the pragma the .xap file is stored in the temp internet files. Putting the application on a page on https will further protect the transmition of the .xap If possible require authentication to view the web page / .xap file (thanks Joel)
