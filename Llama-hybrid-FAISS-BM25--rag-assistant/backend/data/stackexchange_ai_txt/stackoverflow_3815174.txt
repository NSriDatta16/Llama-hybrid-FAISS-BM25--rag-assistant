Does your organization use namespaces? If so, then the syntax of calling the function changes slightly from: var myURL = sforce.apex.execute("MyWebServices", "myUrl", {}); to: var myURL = sforce.apex.execute("MyNamespace.MyWebServices", "myUrl", {});
