C# - Drag item out of listview into a trash can?

How do I drag a item out of a Winforms-listview control onto another control (picture of trash can)? UPDATE1: I think the basic flow is: for the ItemDrag event on the listview have a DoDragDrop Then have a DragEnter event on the picturebox that captures that drag? UPDATE2: The basic flow (based on answers): add 'ItemDrag' event to the listview. add a 'DoDragDrop' inside the 'ItemDrag' add 'DragEnter' event to the picturebox. add a 'GetDataPresent' check inside the 'DragEnter' to check on the data type add a 'DragDrop' event to the picturebox add a 'GetDataPresent' check inside the 'DragEnter' to check on the data type
