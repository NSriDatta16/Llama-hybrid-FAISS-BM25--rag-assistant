Environment definition: \newenvironment{VerbExample} {\example\semiverbatim} {\endsemiverbatim\endexample} Frame definition: \begin{frame}[fragile] \frametitle{Title} \begin{VerbExample} test test test $t$ $\\omega$ test test \end{VerbExample} \end{frame} Verbatim cannot go inside \newcommand . Semiverbatim is defined by Beamer and works well with it. The three characters \ { } must be escaped as \\ \{ \} . Source: Beamer user guide, pp. 119-120 http://www.ctan.org/tex-archive/macros/latex/contrib/beamer/doc/beameruserguide.pdf
