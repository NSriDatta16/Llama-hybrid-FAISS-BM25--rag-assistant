The problem with your code is you're trying to invoke jQuery calls before the DOM is full loaded. You need to put your handlers in a jQuery.ready() callback: codeBlock(' $(document).ready(function(){ $("#InvoiceClientId").change(function () { // stuff }); }); ', array('inline'=>false')); ?> (I also took the liberty of using Cake's Javascript helper to output your JS in the .)
