This is a long-shot but... If your server is accessed over SSL and you are using IE6 the problem could be that " Internet Explorer file downloads over SSL do not work with the cache control headers " If so adding the following header should fix it: Response.AddHeader "pragma", "no-store, no-cache, must-revalidate"
