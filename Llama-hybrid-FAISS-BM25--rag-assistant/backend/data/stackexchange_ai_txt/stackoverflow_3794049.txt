You don't need stored procedure. Just a simple request can make this : SELECT r.RDB$FIELD_NAME AS field_name FROM RDB$RELATION_FIELDS r WHERE r.RDB$RELATION_NAME='TABLE_NAME' --table name ORDER BY r.RDB$FIELD_POSITION; you can learn more here : http://www.alberton.info/firebird_sql_meta_info.html
