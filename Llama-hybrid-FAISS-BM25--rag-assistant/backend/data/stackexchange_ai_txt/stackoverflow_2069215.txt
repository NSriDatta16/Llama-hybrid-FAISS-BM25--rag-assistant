Pcap::Capture#each or Pcap::Capture#each_packet takes an optional count argument. If it is -1, it loops until EOF. You could try passing a number to each and see what happens. cap.each_packet(4) {|pkt|p pkt.src} If that hangs, there might be an issue in the extension code.
