You must subclass and UITableViewCell @interface OutlineCell : UITableViewCell Then, in the ViewController of your table - (UITableViewCell*) tableView: (UITableView*) tableView cellForRowAtIndexPath: (NSIndexPath*) indexPath { NSString* MyIdentifier = @"MyIdentifier"; OutlineCell* cell = (OutlineCell*) [tableView dequeueReusableCellWithIdentifier: MyIdentifier]; if (cell == nil) { cell = [[OutlineCell alloc] init]; } // Populate cell data } You can define the interface via IB, creating an Empty XIB and dragging an UITableViewCell, with your custom OutlineCell class as File Owner. Then add buttons, labels and so on. EDIT: if you want an inner table within a cell, try the same approach adding a subview to the UITableViewCell that you define, with a UITableView inside it.
