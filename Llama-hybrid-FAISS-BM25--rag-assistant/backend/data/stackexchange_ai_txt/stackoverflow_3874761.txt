What you have shouldn't be working, it should be .draggable() and .droppable() (lower-case). For the other piece, specify the revert option , like this: $('#drag').draggable({ revert: 'invalid' }); Also accept is a selector, so change it to .dropaccept like this: $('#drop1').droppable( { accept : '.dropaccept', activeClass: 'dropactive', hoverClass: 'drophover', tolerance: 'intersect' } ); Note the casing changes on the other properties as well, JavaScript is case sensitive ! You can test it out here .
