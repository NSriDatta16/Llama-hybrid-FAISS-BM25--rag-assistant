Under Eclipse, use the Clover Eclipse Plugin . Under Maven, use the Maven Clover Plugin . Add a Clover Report to the site generation: [...] com.atlassian.maven.plugins maven-clover2-plugin [...] pre-site instrument [...] [...] com.atlassian.maven.plugins maven-clover2-plugin [...] [...] Optionally, you can check for a test coverage percentage and fail the build in case of non-compliance: com.atlassian.maven.plugins maven-clover2-plugin 80% verify instrument check The maven build remains the master. Run it using your preferred method (command line or m2eclipse).
