Possibly the web server recognizes the Chrome user agent and allows the file to be downloaded by it, but not by other user agents. Set up your WebRequest to use the same user agent strings and settings as Google Chrome - by default it might be using the IE settings. Edit: Here's instruction from MSDN about how to set the WebRequest user agent string . To find out the user agent of any browser, enter this into its address bar: javascript:prompt('my user agent string is', navigator.userAgent);
