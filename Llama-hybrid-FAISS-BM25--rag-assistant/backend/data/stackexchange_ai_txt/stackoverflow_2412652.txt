If you are using C# (linq) and the number of factors in the cartesian product is static, then it can also be quite elgantly expressed static void Main(string[] args) { IEnumerable firstFactor = Enumerable.Range(0, 2); IEnumerable secondFactor = Enumerable.Range(0, 3); var cartesianProd = from first in firstFactor from second in secondFactor select new { First = first, Second = second }; foreach (var tuple in cartesianProd) { Console.WriteLine("({0}, {1})", tuple.First, tuple.Second); } Console.ReadLine(); }
