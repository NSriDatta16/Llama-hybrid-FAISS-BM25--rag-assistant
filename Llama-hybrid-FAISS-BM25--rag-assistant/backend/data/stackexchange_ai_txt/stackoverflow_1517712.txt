Our biggest speed increase came from placing ./magento/var/cache onto a tmpfs filesystem. We also moved the contents of the .htaccess files into the Apache configuration. That eliminated checking for the file each page hit. We enabled gzip compression on the server too. It seemed to help a bit. You may also want to check out your MySQL query cache config. There are a few guides to tuning your Magento install online. I liked this one: http://www.neurosoftware.ro/programming-blog/blogposter/web-resources/magento-performance-hosting/
