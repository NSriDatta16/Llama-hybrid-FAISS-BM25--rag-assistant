On Windows you can use the built-in esent database engine. There is an API you can use from C++ http://blogs.msdn.com/windowssdk/archive/2008/10/23/esent-extensible-storage-engine-api-in-the-windows-sdk.aspx There is also a managed interop layer that you can use from C# code: http://www.codeplex.com/ManagedEsent
