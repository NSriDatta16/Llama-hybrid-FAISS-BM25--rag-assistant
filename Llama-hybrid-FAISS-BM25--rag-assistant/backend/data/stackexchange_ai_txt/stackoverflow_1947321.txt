Try following these steps Right Click on the database and select Tasks , then Generate Scripts... Select a the database again then click Next (Note: You could just use the tick box at the bottom here and script all objects in the DB and finish - save some time!) Drag the bar down until you get to the Table/View Options Make sure Script Indexes is set to True, then click Next Select the required objects to script, then click Next Select the items to script Finally choose if you want to script to file/clipboard/query window
