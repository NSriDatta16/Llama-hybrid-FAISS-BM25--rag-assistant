Installing Sharepoint on Windows Server 2008 is not a regular install. Make sure you have the latest Cumulative update installed + SP2. Better yet, slipstream the installation. Take note, especially when installing Language Packs attention is need when performing the install, there is a specific order in which to install the product / updates / packs.
