If that version include the SQL Agent you can schedule a backup with a SQL script like that: Edit : corrected code to use dynamic SQL: declare @sql as nvarchar(1000) set @sql='BACKUP DATABASE YourDB TO DISK = ''D:\Backups\YourDB-'+convert(char(8),getdate(),112)+'.bak''' exec sp_executesql @sql
