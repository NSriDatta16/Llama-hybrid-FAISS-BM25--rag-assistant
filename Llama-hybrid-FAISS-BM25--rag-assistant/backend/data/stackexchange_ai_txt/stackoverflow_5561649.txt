In addition to the other solutions I needed to edit the code signing on the main project and the Target file to get the app building to the device again after an expired provisioning profile. ::Delete the old expired profiles ::Add the new profile with the Organizer ::Clean All Targets ::Get Info -> Code Signing on both the main project and the Target ::Build and Run
