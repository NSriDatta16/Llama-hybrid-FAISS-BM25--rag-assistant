You seem to rebind the pseudo event handler toggle over and over again. Try this: $('.dragbox .dragbox-content').toggle( function () { $(this).css({"display":"none"}); }, function () { $(this).css({"display":"block"}); }); $('#hideRows').live('click', function() { $('.dragbox .dragbox-content').click(); });
