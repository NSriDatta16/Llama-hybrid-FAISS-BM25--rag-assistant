Let's use Northwind database as our test data. These are the steps: Configure datasource . Drag a DataGridView to the form. Drag Products onto the DatagridView. Right click on DatagridView - Edit Columns. Select CategoryID, change column type to DataGridViewComboBoxColumn. Choose the datasource(Category table), display member(CategoryName) and value member(CategoryID) for the column. Click OK to apply changes. Build and run the project. Voila~
