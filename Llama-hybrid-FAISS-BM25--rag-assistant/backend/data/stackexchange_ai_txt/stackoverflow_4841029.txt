Robert's right, but here's how I would write it: Enumerable.Range(0, currentScene.Textures.Count).Select(i => new Task(() => { TextureObject texture = (currentScene.Textures[i]); MainWindow.Instance.Dispatcher.BeginInvoke(new Action(() => MainWindow.Instance.StatusBarText.Text = "Loading Texture " + i + " - " + texture.Name ), null); LoadTexture(texture); }); ).Run(x => x.Start()); No need to create a Task whose sole job is to sit and wait on other Tasks.
