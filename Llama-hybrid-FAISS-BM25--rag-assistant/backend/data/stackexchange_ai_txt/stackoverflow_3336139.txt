If you want to embed text you may want to look at this in particular , which comes from the PNG specification. It seems a little on the complicated side. The "easy" steganographic method looks a little more simple. What may actually be better suited for your purpose is to create a javascript object that contains the image data and the JSON data - then just pass that object around wherever you need it.
