Here is a quick and dirty example of how to lock the other handle in relation to the handle that is being dragged: var offset = 40; $('#slider').slider({ range: true, max: 600, values: [40, 80], slide: function(e,ui){ if (ui.value == ui.values[0]) { $(this).slider('values',1, ui.value+offset); } else { $(this).slider('values',0, ui.value-offset); } } }); Should help you figure out how to do what you are wanting to do. Working Example: http://jsfiddle.net/cBHdD/
