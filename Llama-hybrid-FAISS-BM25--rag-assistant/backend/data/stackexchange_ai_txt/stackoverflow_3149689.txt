Just to add to sepp2k's excellent answer to Q3, where he says It lives in the scala package because it is considered so basic that it belongs in the base package. This applies to Scala 2.7 In Scala 2.8, the collections classes have been reorganized, and now the :: class lives in scala.collection.immutable , and the name scala.:: is a type alias for scala.collection.immutable.:: .
