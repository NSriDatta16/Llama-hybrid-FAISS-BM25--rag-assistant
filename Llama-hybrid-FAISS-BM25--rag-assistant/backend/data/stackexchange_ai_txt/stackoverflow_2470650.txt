As stated in perldoc LWP::UserAgent , the default is to follow redirects for GET and HEAD requests: $ua = LWP::UserAgent->new( %options ) ... KEY DEFAULT ----------- -------------------- max_redirect 7 ... requests_redirectable ['GET', 'HEAD'] Here is an example: #!/usr/bin/perl use strict; use warnings; use LWP::UserAgent; my $ua = LWP::UserAgent->new(); $ua->show_progress(1); my $response = $ua->head('http://unur.com/'); if ( $response->is_success ) { print $response->request->uri->as_string, "\n"; } Output: ** HEAD http://unur.com/ ==> 301 Moved Permanently (1s) ** HEAD http://www.unur.com/ ==> 200 OK http://www.unur.com/
