Apply the Haversine formula to find the distance. See the C# code below to find the distance between 2 coordinates. Better still if you want to say find a list of stores within a certain radius, you could apply a WHERE clause in SQL or a LINQ filter in C# to it. The formula here is in kilometres, you will have to change the relevant numbers and it will work for miles. E.g: Convert 6371.392896 to miles.
