You can do this by getting the bytes via WebClient and then perform encoding via MediaStreamSource . You are using SHOUTcast, therefore the application/octet-stream can be obtained by requesting the following URL: http://208.53.158.48:8364/ ; Or you might need to fake the user agent to avoid standard HTTP requests.
