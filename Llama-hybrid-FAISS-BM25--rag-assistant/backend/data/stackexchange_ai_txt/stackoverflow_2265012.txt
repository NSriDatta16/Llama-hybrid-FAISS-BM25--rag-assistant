routes.MapRoute("WithTarget", "{controller}/{action}/{id}#{target}"); and RedirectToAction("Edit", "Customers", new { id = iNewId, target = "Tab1" }); or RedirectToRoute("WithTarget", new { controller = "Customers", action = "Edit", id = iNewId, target = "Tab1" }); But remember that the fragment is never sent from the client to the web server, so there's no way to read its value in a controller action.
