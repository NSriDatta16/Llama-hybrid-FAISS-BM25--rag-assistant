There is a documented API for getting info on file positions on disk since Windows 2000. Look for DeviceIoControl function with FSCTL_GET_RETRIEVAL_POINTERS control code on MSDN: http://msdn.microsoft.com/en-us/library/aa364572(VS.85).aspx The API has been provided for writing custom disk defragmenters and consists of several other control codes.
