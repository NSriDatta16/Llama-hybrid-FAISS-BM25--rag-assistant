You can implement the following method in a custom subclass of either UIView or UIViewController : - (void)touchesMoved:(NSSet *)touches withEvent:(UIEvent *)event Assuming there's one instance of UIViewController that loads the main view and its subviews, if you implement the method in your controller class it should get called for drag events in any of the views.
