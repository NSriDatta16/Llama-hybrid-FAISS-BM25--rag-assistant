One way is always modify something in the Car table. Car is aggragetion root for your wheels because wheels cannot exists without Car. If you modify wheel you modify whole aggragated object - car. So add some car update to model this behavior - common is maintaining some column like ModifiedDate. Pesimistic concurrency is supported by serialized transaction - default isolation level for TransactionScope .
