In addition to the Preferences API for storing user settings there are a few services that can be found in the javax.jnlp package . For your concrete requirement the PersistenceService would be particularly useful. Alternatively you can simply provide all data that your application requires as part of your .jar files, reference them in your .jnlp file and customize how and when they are downloaded by using the DownloadService .
