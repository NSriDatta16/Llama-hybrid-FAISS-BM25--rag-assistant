Yes, it's pretty easy to create an image from text: http://chiragrdarji.wordpress.com/2008/05/09/generate-image-from-text-using-c-or-convert-text-in-to-image-using-c/ Use a generic handler (.ashx) file to output an image. Then link to the handler from your page..and it will show up as an image.
