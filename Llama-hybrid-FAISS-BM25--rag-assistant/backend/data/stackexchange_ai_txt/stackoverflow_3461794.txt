You have to run the Agent Controller on the java application in server mode. Currently your command runs it 'headless' and writes it to a log file. The website you linked has a list of the various options for the server parameter. java '-agentlib:JPIBootLoader=JPIAgent:server=enabled;CGProf' MyApp
