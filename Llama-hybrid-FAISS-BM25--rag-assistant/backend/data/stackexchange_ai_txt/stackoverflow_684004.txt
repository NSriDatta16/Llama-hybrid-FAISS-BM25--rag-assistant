You need to be in Ruby mode, so that Emacs will understand that “ # This is a comment. ” is a comment. If you are in fundamental mode, it will just treat everything as a text paragraph, which makes it think that the text on the next line is part of the same paragraph. Here are some instructions for installing Ruby mode if you don't already have it.
