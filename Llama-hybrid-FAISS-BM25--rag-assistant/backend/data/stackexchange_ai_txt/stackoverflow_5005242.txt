For easier operation you might consider simply using the registry where those names are listed eg: ErrCode := RegOpenKeyEx( HKEY_LOCAL_MACHINE, 'HARDWARE\DEVICEMAP\SERIALCOMM', 0, KEY_READ, KeyHandle); (I've ommited the hand-waving stuff). You might also consider using WMI - see this example from Magenta Systems - you can get a pretty much everything hardware-related now.
