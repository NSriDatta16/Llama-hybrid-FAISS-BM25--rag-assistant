The confidence interval of an absolute value should be written as $$|0.05| \pm 0.12$$ because the confidence band (0.12) is absolute in itself, but is described (with $\pm$ ) as going both ways. The symbol for average standard deviation is bar-sigma: $\bar{\sigma}$ .
