Yes this is inappropriate because it uses the same data twice, leading to falsely overconfident results. This is known as 'double dipping'. For references, I would start with Carlin and Louis (2000). Although 'double dipping' has been one of the primary critiques of Empirical Bayes, Ch. 3, in particular section 3.5, of this book describes ways to estimate appropriate confidence intervals using the EB approach.
