Absolutely right, catch the Exception via begin/rescue/end, like this: def begin success_flag = open(the_url, "UserAgent" => "Ruby-OpenURI").read ... do something else... return success_flag rescue Exception => e # log(e) here, maybe? return false end end It will catch any exception thrown inside the begin/rescue block.
