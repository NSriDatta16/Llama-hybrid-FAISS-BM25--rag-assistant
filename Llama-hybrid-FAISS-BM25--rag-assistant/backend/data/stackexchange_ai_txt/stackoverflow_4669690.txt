Use navigator.userAgent to get the userAgent of the user and then change your code accordingly. I'd create an array from mobile userAgents and then check if the current userAgent is in the array, see this list to get you started: http://en.wikipedia.org/wiki/List_of_user_agents_for_mobile_phones
