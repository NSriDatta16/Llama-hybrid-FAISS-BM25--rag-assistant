There are a number of tools in GHC's runtime system explicitly to support memoization. Unfortunately, memoization isn't really a one-size fits all affair, so there are several different approaches that we need to support in order to cope with different user needs. You may find the original 1999 writeup useful as it includes several implementations as examples: Stretching the Storage Manager: Weak Pointers and Stable Names in Haskell by Simon Peyton Jones, Simon Marlow, and Conal Elliott
