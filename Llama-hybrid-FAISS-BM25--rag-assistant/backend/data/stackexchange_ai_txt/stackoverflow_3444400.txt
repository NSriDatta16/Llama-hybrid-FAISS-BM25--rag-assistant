Martijn, I'll bite. Numpty Physics, (released open-source, GPL, here: http://numptyphysics.garage.maemo.org/ ) is a free game similar to Crayon Physics. It uses the Box2D physics engine (box2d.org) and has a built-in level editor. In addition to the source code, you will also find binary packages for many popular operating systems. Here's a shortcut to the code if you are impatient... 8) https://garage.maemo.org/scm/?group_id=638 Hope that helps you out... -Alan
