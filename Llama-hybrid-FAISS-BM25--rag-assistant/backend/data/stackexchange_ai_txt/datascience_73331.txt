This seems like a problem that can be fixed with any regression problem. Here are models I would recommend to get started with: If you need your model to be explainable (i.e. show which factors contributed to the score most): Decision trees / random forests Linear regression If you only care about giving a final score Essentially any other model, which can include: Support vector machines Neural Networks K-Nearest Neighbours Gaussian Processes and more
