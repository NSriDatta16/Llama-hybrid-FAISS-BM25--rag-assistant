From the introduction paragraph to Chapter 3 of the Java Virtual Machine Specification : For example, the memory layout of run-time data areas, the garbage-collection algorithm used , and any internal optimization of the Java virtual machine instructions (for example, translating them into machine code) are left to the discretion of the implementor . [emphasis mine]
