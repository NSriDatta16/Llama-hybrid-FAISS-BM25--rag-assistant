This little snippet will do the trick. Ask me if you need some questions about the Regex public class test { public static void main(String[] args) { String htmlFragment =" "; String attributesToRemove = "id|style"; System.out.println(htmlFragment); System.out.println(cleanHtmlFragment(htmlFragment, attributesToRemove)); } private static String cleanHtmlFragment(String htmlFragment, String attributesToRemove) { return htmlFragment.replaceAll("\\s+(?:" + attributesToRemove + ")\\s*=\\s*\"[^\"]*\"",""); } }
