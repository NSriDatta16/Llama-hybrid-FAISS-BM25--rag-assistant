Have a look at numpy . It can read the three files into three arrays (using fromfile ), calculate the average and export it to a text file (using tofile ). import numpy as np a = np.fromfile('A.csv', dtype=np.int) b = np.fromfile('B.csv', dtype=np.int) c = np.fromfile('C.csv', dtype=np.int) d = (a + b + c) / 3.0 d.tofile('D.csv') Size of "some MB" should not be a problem.
