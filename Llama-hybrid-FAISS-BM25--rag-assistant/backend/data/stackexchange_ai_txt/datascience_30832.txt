A CNN automatically extracts features, so hand-crafting features has become unnecessary for most applications. It learns what features to extract via backpropagation. You can introduce handcrafted features in a few ways to boost the performance of a CNN: inject hand-crafted features into the fully connected layer feed the CNN output into another classifier alongside the hand-crafted features This paper explores such concepts.
