Try using a batch file (.bat). Type in whatever commands you want to execute in the proper order, in notepad, such as: python program.py Save the file as iHateTyping.bat Open the command prompt using Run . Go to the directory where you saved the file using cd . Type in: iHateTyping.bat & you're done. I encourage you to read more about batch files in the link highlighted above.
