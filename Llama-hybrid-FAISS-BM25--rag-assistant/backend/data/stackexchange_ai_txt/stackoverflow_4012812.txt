Call event.preventDefault() while handling DRAG_DROP event: private function tree_dragDropHandler(event:DragEvent):void { event.preventDefault(); tree.hideDropFeedback(event); DragManager.showFeedback(DragManager.NONE); } This handler should run before Tree 's own DRAG_DROP handler, this can be done via adding listener with priority: tree.addEventListener(DragEvent.DRAG_DROP, tree_dragDropHandler, false, 1);
