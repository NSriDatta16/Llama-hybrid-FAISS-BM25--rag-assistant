You shouldn't need two clones, one will do. Taking Paolo Bergantino answer we have: jQuery.fn.swapWith = function(to) { return this.each(function() { var copy_to = $(to).clone(true); $(to).replaceWith(this); $(this).replaceWith(copy_to); }); }; Should be quicker. Passing in the smaller of the two elements should also speed things up.
