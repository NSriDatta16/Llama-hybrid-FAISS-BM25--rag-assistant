NHibernate offers the Tuple class that could be used here: var hql = @"select t1.info1 as First, t2.info2 as Second from table1 t1 left outer join t1.table2 t2"; // instead of Tuple you can use other types of course var variable = session.CreateQuery(hql) .SetResultTransformer(new AliasToBeanResultTransformer( typeof(NHibernate.Linq.Tuple ))) .List >(); You would then access the items with variable[0].First variable[0].Second
