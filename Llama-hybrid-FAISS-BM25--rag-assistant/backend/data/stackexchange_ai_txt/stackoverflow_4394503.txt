If you're testing in Chrome, everything under file:// , even folders in the same directory are considered separate domains, so the same origin policy is kicking in, and the localStorage is unique to each. I don't have a test environment setup at the moment, but launching with the command line flag: --allow-file-access-from-files should be what you're after.
