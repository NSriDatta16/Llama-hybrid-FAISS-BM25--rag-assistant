http://robertnyman.com/2008/10/14/javascript-how-to-get-private-privileged-public-and-static-members-properties-and-methods/ explains it in excruciating detail. In your case... // Constructor function Vertex (data) { // Private var id = 1; // Privileged this.getID= function () { return id; }; // Public this.data = data; } // Public Vertex.prototype.getData = function () { return this.data; }; // Static property Vertex.static = "something";
