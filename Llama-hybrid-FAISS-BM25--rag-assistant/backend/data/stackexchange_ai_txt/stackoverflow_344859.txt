At design time? Just drag the stored procedure on to the LINQ designer surface. At runtime? You need sql like this: SELECT * FROM syscolumns WHERE id = ( SELECT id FROM sysobjects WHERE Name = @ProcName ) Which might be generated by LinqToSql like this: var params = db.sysobjects .Where(o => o.Name == ProcName) .SelectMany(o => db.syscolumns .Where(c => c.id == o.id) )
