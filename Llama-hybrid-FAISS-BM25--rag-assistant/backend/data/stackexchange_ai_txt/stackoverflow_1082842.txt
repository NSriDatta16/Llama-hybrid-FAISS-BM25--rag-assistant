You have to create a Media Player plugin and access the state using the Media Player API. Theoretically you could also do some external analysis of the Media Player, for example by enumerating its windows and handles and reading window texts, but that would be very "hacky" and most fragile. Although creating a plugin sounds like a lot of work, it'll be the better solution in the long run.
