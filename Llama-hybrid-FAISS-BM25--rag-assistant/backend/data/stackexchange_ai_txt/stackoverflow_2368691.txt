One alternative is to give up using a scroll-bar at all and do it using CSS and a jQuery slider control . This also gives you the option of making it look more like a time-line. you can set the scroller elements to whatever CSS you want. There are a few out there, but it's not too hard to roll-your-own using a jQuery draggable control and constraining one axis inside a long, narrow container DIV.
