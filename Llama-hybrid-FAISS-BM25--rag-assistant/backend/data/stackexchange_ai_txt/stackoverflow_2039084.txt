The easiest is most likely to learn how to write a simple servlet, how the HTML looks that you want to generate and then adapt your existing application to run inside the servlet code, and print html instead of just plain text. This is described in the Servlet part of the Java EE tutorial (which is not great, but a start) http://java.sun.com/javaee/5/docs/tutorial/doc/bnafe.html If you want a more accessible book, I can recommend the Head First series. http://oreilly.com/catalog/9780596005405 For a web container, Apache Tomcat is fine. http://tomcat.apache.org/
