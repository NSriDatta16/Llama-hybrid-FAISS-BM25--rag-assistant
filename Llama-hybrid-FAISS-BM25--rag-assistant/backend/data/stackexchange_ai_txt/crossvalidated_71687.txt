The short answer is that you seem to be on the right track. If you are on the wrong track in some way, it is because you seem to be searching for a precise definition for data mining, when in fact no such precise definition yet exists. In this respect, data mining is like several other modern data terms that continue to evolve: data science, machine learning, and exploratory data analysis.
