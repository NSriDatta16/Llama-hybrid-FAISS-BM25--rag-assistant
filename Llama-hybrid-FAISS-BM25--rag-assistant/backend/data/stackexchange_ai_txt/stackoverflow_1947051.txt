Select Avg(differ) from ( Select s1.r, s2.r, s2.time - s1.time as differ from ( Select * From (Select rownum as r, inn.time from table inn order by time) s1 Join (Select rownum as r, inn.time from table inn order by time) s2 On mod(s2.r, 3) = 2 and s2.r = s1.r + 1 Where mod(s1.r, 3) = 1) ); The parameters can be changed as the number of stages changes. This is currently set up to find the average between stages 1 and 2 from a 3 stage process. EDIT a couple typos
