There are plenty of imputation methods but the best is to drop data if you have enough data... Imputation methods -> http://www.stat.columbia.edu/~gelman/arm/missing.pdf Example of R package -> https://cran.r-project.org/web/packages/imputeR/imputeR.pdf Another idea is to use algorithms that can deal with missing values such as XGBoost for example... For your special case of time series, see -> How to fill in missing data in time series? And the R package : ImputeTS -> https://cran.r-project.org/web/packages/imputeTS/vignettes/imputeTS-Time-Series-Missing-Value-Imputation-in-R.pdf In a nutshell, if you can drop periodic data, you may adjust your interpretation either try imputation...
