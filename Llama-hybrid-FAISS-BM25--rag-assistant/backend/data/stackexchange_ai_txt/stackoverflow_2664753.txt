Lot's of folks have this problem when adding sample data. You can probably fix the problem by cleaning out the magento system logs Log in as administrator System -> Configuration -> System (under ‘Advanced’, bottom left) -> Log Cleaning If you get the error again, hit the back button a few times and then try to recreate the error.
