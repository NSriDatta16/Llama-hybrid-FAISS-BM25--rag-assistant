You are running the code when the DOM isn't ready yet. There are 2 solutions: Solution One: Add the Javascript after the elements it affects. Preferably as far down the page as possible. Doing this is not always possible, but it is suggested by YUI for speeding up your website . a paragraph // This is after .demo $(".demo").click(function() { alert("JavaScript Demo"); }); Solution Two: Wrap your script in a doc ready. In jQuery there are several forms . The quickest to type is $(function() { ... }); : // doc ready $(function() { $(".demo").click(function() { alert("JavaScript Demo"); }); }); a paragraph
