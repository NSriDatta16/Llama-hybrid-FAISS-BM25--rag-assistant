If you really don't want to check access rights on the target page: You could hash the PK with the UserID and then add the hash value to the query string. string hash = hashFunction(PK.toString() + UserID.toString()); Then you have to make sure the hash in the queryString equals the hash value calculated before loading the page. Assuming this is an internal organization Web application.
