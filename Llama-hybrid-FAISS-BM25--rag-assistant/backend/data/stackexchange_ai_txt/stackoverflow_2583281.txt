I know no way to do this with CSS but with JavaScript, you can check: var parent = element.parentElement; if (element.clientTop + element.clientHeight > parent.clientTop + parent.clientHeight) { ... element is outside parent ... } You can use a binary search to find the last completely visible paragraph.
