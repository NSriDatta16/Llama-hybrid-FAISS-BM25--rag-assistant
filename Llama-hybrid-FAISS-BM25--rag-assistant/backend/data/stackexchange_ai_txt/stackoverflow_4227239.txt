I would recommend you to do the same but on the client side. You can use Custom Formatter to construct the HTML fragment inside your custom formatter function: { name: 'CallID', index: 'CallID', width: 50, formatter:function (cellvalue, options, rowObject) { return ' " } }, The server part should return only data without any HTML. Moreover I would recommend you to send back as the values for the CallStartTime column the data formatted as ISO date ISO8601Long ("Y-m-d H:i:s") and use newformat setting of the formatoptions of the formatter:'date' .
