Dictionary class is not optimized for this case, but if you really wanted to do it (in C# 2.0), you can do: public List GetKeysFromValue (Dictionary dict, TVal val) { List ks = new List (); foreach(TKey k in dict.Keys) { if (dict[k] == val) { ks.Add(k); } } return ks; } I prefer the LINQ solution for elegance, but this is the 2.0 way.
