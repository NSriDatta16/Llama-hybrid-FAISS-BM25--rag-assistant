You should be careful not to store that pointer for further use. For example: void store(int* param) { // Store pointer in some global storage for further use global_storage.add_param(param); } void func() { int test_var = 5; store(&test_var); // Pass the pointer to the global storage } // At this point stack object test_var will be destroyed // Global storage contains a pointer, that points to some garbage
