Orange is an open source data mining software developed by the University of Ljubljana . It is used primarily for developing and testing machine learning models as well as conducting exploratory data analysis and visualization. It is unique in that users construct workflows using drag and drop widgets. Orange has many Add-ons made specifically for different types of data including Bioinformatics and Spectroscopy.
