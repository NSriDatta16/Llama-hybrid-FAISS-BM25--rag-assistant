I found a solution without PREG functions. Let us say we have a $str = " My Text"; I needed an aux variable to store an integer: $tmp = strlen($str) - strlen(ltrim($t3)); // If ltrim is not ok for you // You can use preg_replace() $tmp = str_repeat("\t", $tmp / 2 /* 2 spaces -> 1 tab */) . ltrim($t3); The PREG function is more elegant, but I'm not sure of how efficient it is
