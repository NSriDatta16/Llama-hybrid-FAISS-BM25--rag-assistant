You can load several views at once, or views inside other views. So in this case I recomend you to create one header view where you load all css and js files example: css/moorainbow.css" type="text/css" media="screen"/> And call it like: $this->load->view('header'); $this->load->view('view1'); $this->load->view('view2'); This way you can control the files (css+js+etc) you load in just one file.
