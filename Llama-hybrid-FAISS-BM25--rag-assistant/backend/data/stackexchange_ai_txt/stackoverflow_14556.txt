For basic data retrieval I would be going for Linq without hesitation. Since moving to Linq I've found the following advantages: Debugging my DAL has never been easier. Compile time safety when your schema changes is priceless. Deployment is easier because everything is compiled into DLL's. No more managing deployment scripts. Because Linq can support querying anything that implements the IQueryable interface, you will be able to use the same syntax to query XML, Objects and any other datasource without having to learn a new syntax
