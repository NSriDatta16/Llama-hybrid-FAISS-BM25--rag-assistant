I am also using symlinks to my custom code and I managed to overcome this issue by using mount --bind instead of creating symlinks, e.g. custom theme directory: /home/user/workspace/magento/app/design/frontend/default/mytheme cd /app/design/frontend/default mkdir mytheme sudo mount --bind /home/user/workspace/magento/app/design/frontend/default/mytheme mytheme This method will not work on OSX.
