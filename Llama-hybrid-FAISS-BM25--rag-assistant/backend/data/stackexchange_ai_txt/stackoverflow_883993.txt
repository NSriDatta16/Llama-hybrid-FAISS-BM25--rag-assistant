If you are deliberately displaying it for the end user, wrap the JSON text in and tags, e.g.: [ { color: "red", value: "#f00" }, { color: "green", value: "#0f0" }, { color: "blue", value: "#00f" }, { color: "cyan", value: "#0ff" }, { color: "magenta", value: "#f0f" }, { color: "yellow", value: "#ff0" }, { color: "black", value: "#000" } ] Otherwise I would use JSON Viewer .
