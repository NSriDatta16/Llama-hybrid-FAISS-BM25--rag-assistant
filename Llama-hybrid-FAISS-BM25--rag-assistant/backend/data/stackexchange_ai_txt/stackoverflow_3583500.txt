Do you expect the $var argument to be processed by the userAgent function? In that case, you need to specify it as an argument. function userAgent( $var ) { ... Otherwise, you may want to initialize the $var in the function body. It's a wise strategy to keep your if-then-else s symmetrical: if( $browse ) { $var = 1; } else { $var = 0; } And you can write this a bit more succinct like function userAgent() { $browser = ..... return $browser ? 1 : 0; }
