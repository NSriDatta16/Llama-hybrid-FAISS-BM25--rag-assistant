Can You please specify what are different results? From what I see, code is doing much more than verifying account. From what I see, You're appending to a single file from multiple threads, I'd say it's not thread-safe. Also, AFAIK Mechanize uses shared cookie storage for all requests, so they are probably interfering. Use separate mechanize.Browser() inside run() instead of mechanize.Request() .
