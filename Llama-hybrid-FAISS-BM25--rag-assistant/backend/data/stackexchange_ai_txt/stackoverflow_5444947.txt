First : From the pure Directory point of vue, you've got in your organizationalUnit (OU) an attribute called " distinguishedName " which looks like : OU=currentOU,OU=parentOU,...,DC=domain,DC=.. so you can easily compute the string of the parent OU. Second : On the programming point of view, you've got a property of DirectoryEntry class which is parent . Here is a sample code. String myADSPath = "LDAP://onecity/CN=user,CN=Users,DC=onecity,DC=corp,DC=fabrikam,DC=com"; DirectoryEntry myDirectoryEntry=new DirectoryEntry(myADSPath, UserName, SecurelyStoredPassword); Console.WriteLine("Parent is :"+myDirectoryEntry.Parent.Path); JP
