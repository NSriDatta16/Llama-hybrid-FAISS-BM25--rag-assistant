I think you'll have to manually set up the mapping to begin with, unless someone has already done it. With a little tinkering you could use a db or similar to do a join on the names and probably get 90% or more of them, then fill in the rest by hand. I would keep the mapping in an external file, database table, etc., so it's not hard-coded (not sure how likely the GeoID values are to be added or updated...) and load them into a hashtable/dictionary if you need to be able to look them up very quickly.
