You know that you can just select the components from jQuery UI want, right? You can download the individual components and put them on the page. You'd only need UI Core, Draggable, and Droppable ~ 34K minified. Another alternative is loading jQuery UI from the Google content network , which case the user may already have it loaded making the download size immaterial.
