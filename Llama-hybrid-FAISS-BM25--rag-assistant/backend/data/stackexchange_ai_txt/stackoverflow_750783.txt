BlueJ is a GUI Java IDE that allows students to go from UML diagrams to code. It's free and recently open source. It also supports extensions similar to popular Eclipse extensions: PMD, checkstyle, etc. It's specifically designed for teaching: The BlueJ environment was developed as part of a university research project about teaching object-orientation to beginners. The aim of BlueJ is to provide an easy-to-use teaching environment for the Java language that facilitates the teaching of Java to first year students. Special emphasis has been placed on visualisation and interaction techniques to create a highly interactive environment that encourages experimentation and exploration. It includes several tutorials .
