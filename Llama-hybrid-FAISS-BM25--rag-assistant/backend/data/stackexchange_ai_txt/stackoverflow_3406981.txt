I solved the my issues by adding `` around my variables. Here is the code I ended up with. DELIMITER // CREATE FUNCTION `modx`.getSTID (x VARCHAR(255)) RETURNS INT DETERMINISTIC BEGIN DECLARE `y` INT; SELECT id INTO `y` FROM `modx`.coverage_state WHERE `coverage_state`.name = `x`; RETURN `y`; END// DELIMITER ;
