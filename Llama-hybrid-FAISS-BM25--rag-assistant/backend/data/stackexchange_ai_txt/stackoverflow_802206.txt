Python and PHP both have a binding for GTK : PyGTK for Python; PHP GTK for PHP; You can use Glade as a RADD to draw a GTK app. GTK runs on Windows, Linux and Mac OS. If you are looking for something embedding everything, I can think of SPE for Python, but you really can run Glade as stand alone. Since any picture is worth 1000 words : I let the PHP lovers choosing their language equivalent.
