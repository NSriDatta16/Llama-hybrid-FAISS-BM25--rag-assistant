You can try django-tables app, which allows you to do the following, given model Book: # Define class BookTable(tables.ModelTable): id = tables.Column(sortable=False, visible=False) book_name = tables.Column(name='title') author = tables.Column(data='author__name') class Meta: model = Book # In your views initial_queryset = Book.objects.all() books = BookTable(initial_queryset) return render_to_response('table.html', {'table': books}) # In your template table.html {% for column in table.columns %} {{ column }} {% endfor %} {% for row in table.rows %} {% for value in row %} {{ value }} {% endfor %} {% endfor %} I think the above is much more elegant and self explanatory than just doing {% table book_list %}
