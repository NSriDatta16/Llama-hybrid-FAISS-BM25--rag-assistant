If you need help setting up Azure resources, ask those questions on Server Fault or Super User . Microsoft Azure (formerly Windows Azure) is Microsoft's application platform for the public cloud. Microsoft Azure can be used to build web applications or services that run and store data in Microsoft data centres. It can connect on-premises applications with each other or map between different sets of identity information. Azure offers several high-level services Cloud Services Storage / Data Management SQL Database (formerly SQL Azure) Azure Media Services Virtual Machines Virtual Networks App Services: Web Apps Azure Functions Azure Mobile Services Azure HDInsight (Big Data) Scheduler Cache Service Cosmos DB Traffic Manager Stream Analytics Machine Learning Remote Applications - RemoteApp Data Lake Analytics Application Insights Event Grid Batch Azure offers several services currently in preview Mobile Engagement SQL Data Warehouse Azure Data Factory Intro to Azure Introducing Microsoft Azure Each of these services offers the ability to scale up or down based on demand using predetermined virtual machine sizes to run the services.
