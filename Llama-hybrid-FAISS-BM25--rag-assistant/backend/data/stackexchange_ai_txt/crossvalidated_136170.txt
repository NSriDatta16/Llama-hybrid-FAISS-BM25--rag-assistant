The best choice is to use an R package like mi (mice, or amelia should also work). It will scan your data and propose the appropriate format for each variable (which you can also manually change if needed) In the case of categorical (ordered or not) variables in the MI package - a series of chained, ordinal regressions are run with the variables targeted for imputation as the DVs. This is done until some convergence criteria are met (which can be adjusted in the package You could think of it as a markov chain of regressions using all included data to predict missing data in all of the variables you included in the analysis
