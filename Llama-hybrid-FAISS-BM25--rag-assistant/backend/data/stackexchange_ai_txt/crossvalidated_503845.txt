If you have information about the other named features for the next two months, so you can actually use that information to predict the sales. Regression and Time-series problems are not completely independent. In fact, most of the methods are based on learning the relationships between the relevant predictors and predicting a future response. ( Autoregressive Dynamics ). You will be using some regression methods in time series forecasting too. You can start by understanding what types of data would be available for the next few months while making a prediction, and then only use those features to train models. You can start with autoregressive models, and proceed to more complicated ones.
