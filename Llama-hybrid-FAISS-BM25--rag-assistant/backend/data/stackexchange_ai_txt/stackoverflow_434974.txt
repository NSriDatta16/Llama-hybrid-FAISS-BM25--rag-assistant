You can't pass the second param directly - so add it to myImage: myImage = new Image(); myImage.id = "oneCent"; myImage.num = 1; myImage.addEventListener(MouseEvent.MOUSE_MOVE, dragIt); Then in the dragit function: function dragIt(event:MouseEvent):void { trace("PARAM =", event.target.num, event.target.id); } Where event.target automatically becomes a reference to the image
