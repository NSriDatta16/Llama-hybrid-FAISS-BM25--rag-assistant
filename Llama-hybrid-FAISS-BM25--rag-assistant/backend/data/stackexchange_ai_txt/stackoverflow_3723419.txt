Try eXtplorer component . It is "Editors Pick" extension. eXtplorer is a web-based File Management Component for all your needs. It has a desktop-application-like interface with drag&drop, grid and a directory tree and makes heavy use of the ExtJS Javascript Library. You can use it to access and modify the files and directories on your server via FTP or direct file access.
