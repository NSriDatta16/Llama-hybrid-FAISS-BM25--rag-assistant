First of all. 9 categories sound like a lot. How big is you sample? Start by assessing the independent variables. You might want to remove the ones which have a score on 3 standard deviations above them mean. This should reduce you range. Next you want to assess if the independent variables follows a normal distribution. If not, you want to apply some transformation to make them normal distributed. It is not a necessity but it is good common practice. The way you describe it, logistic regression is the way to go with the methods you are mentioning. It all comes out on the characteristics of your dependent variable Cluster analysis is an explorative/undirected technique so that can not be used for prediction. K-means clustering can however be used for prediction...
