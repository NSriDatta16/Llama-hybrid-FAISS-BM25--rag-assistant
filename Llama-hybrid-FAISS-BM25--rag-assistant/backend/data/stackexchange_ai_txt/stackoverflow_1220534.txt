There are a few places to hide session information other than cookies. a session key in the URL ( http://example.com/app/234348738790/main ) a session key as a GET variable ( ?sess=257892345 ) a session key as a POST variable (input type='hidden') store it in local storage in the browser use javascript with any of the above to communicate the session info back to the server.
