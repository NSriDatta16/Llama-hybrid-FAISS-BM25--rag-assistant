That's because your code get executed before the DOM is created. You can do something like this window.onload = function() { document.forms['FormName'].elements['txtName'].value = "Robert"; } or use JQuery $(function() { document.forms['FormName'].elements['txtName'].value = "Robert"; // for a shorter syntax use id $('#txtNameId').val("Robert"); }
