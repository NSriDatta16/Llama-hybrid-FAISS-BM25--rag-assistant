Project Properties->Linker->Input->Additional Dependencies You can also use #pragma comment( lib, "mapi32" ) in one of your source files. As noted MSDN here is a similar library addition using the pragma technique MSDN - Creating a Basic Winsock Application #include #include #include #pragma comment(lib, "Ws2_32.lib") int main() { return 0; }
