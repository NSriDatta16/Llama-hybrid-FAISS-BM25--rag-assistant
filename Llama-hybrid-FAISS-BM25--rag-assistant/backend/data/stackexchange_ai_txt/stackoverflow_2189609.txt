So, you need UPnP or NAT-PMP. Your system may have libraries for doing those protocols, or you can use muniupnp , which has both a server to run on a Linux or BSD box to test against, and a client library. There is no guarantee that will work, so you may need to go further and use STUN, TURN or ICE. There's a library for doing those things here . And always remember, sometimes it's just impossible. Implement IPv6 as well, sometimes that works when IPv4 doesn't. Encourage users to install v6 and Teredo.
