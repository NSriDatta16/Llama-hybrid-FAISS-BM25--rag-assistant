Using mechanize-1.0.0 the following works: agent = Mechanize.new page = agent.get('http://127.0.0.1/some.aspx') form = page.form("aspnetForm") form.add_field!('__EVENTARGUMENT', 'Page$2') form.add_field!('__EVENTTARGET', 'ctl00$ContentPlaceHolder1$gvwSomeList') page = agent.submit(form) # this gets page 2
