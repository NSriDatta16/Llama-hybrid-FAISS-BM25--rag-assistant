If you set the "checkForMouseListeners" property to false on your Graphic objects, then the default map click/drag behavior will be preserved. graphic.addEventListener(MouseEvent.DOUBLE_CLICK, function(event:MouseEvent):void { var graphic:Graphic = event.target as Graphic; if(graphic) { ... } }); //preserve the default click/drag behavior on the map graphic.checkForMouseListeners = false; http://resources.esri.com/help/9.3/ArcGISServer/apis/Flex/apiref/com/esri/ags/Graphic.html#checkForMouseListeners
