You should use location.replace instead of window.location Example: if( (navigator.userAgent.match(/iPhone/i)) || (navigator.userAgent.match(/Android/i)) || (navigator.userAgent.match(/iPod/i)) ) { location.replace("http://sitename.com/m/"); } I used this code and it works on iphone/itouch and android phones/devices.
