You can't insert fragments, as they need to be complete DOM elements. Instead you should use .wrap() in some spots here; it should look like this instead: $(document).ready(function() { $('#template').wrap(' ') .wrap(' ').parent() .before(' Top ') .after(' Bottom '); }); You can test it out here .
