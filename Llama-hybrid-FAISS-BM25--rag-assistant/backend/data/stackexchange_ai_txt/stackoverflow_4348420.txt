Great hack by @strager. Implement it into jQuery like this: jQuery.fn.wait = function (MiliSeconds) { $(this).animate({ opacity: '+=0' }, MiliSeconds); return this; } And then use it as: $('.notice').fadeIn().wait(2000).fadeOut('slow');
