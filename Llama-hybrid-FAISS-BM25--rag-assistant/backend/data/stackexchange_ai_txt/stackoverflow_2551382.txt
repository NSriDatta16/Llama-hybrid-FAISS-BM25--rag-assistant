If you want an elegant solution, define a new selector. Unfortunately, :empty is not sufficient as anything with text node children isn't empty. $.extend($.expr[':'], { leaf: function(elem, i, match) { return $(elem).children().length == 0; } }); And then you can do: $("div:leaf:contains('test')").hide();
