In the Tools... menu, select External Tools... and add references to the batch files. Then right-click on a toolbar, select Customize... , go to the Toolbars tab, click on New... , name your new toolbar, click on OK , go to the Commands tab, select the Tools category and drag-drop the appropriate External Command Command onto your custom tool bar. If you need to run batch files that always run right before or after a build, you're probably better off making use of build events .
