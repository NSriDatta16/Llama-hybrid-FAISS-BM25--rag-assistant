In the case of IE7, it appears that the Youtube's object doesn't "unload" itself when it's hidden compared to IE8 or FF 3.6. For example, in IE8 if the object is hidden the function youtubeobject.stopVideo is undefined, but in IE7 it's still exists. I think a more elegant solution is to check to see if the "stopVideo" function exists in the Youtube object, and then call it. function toggle(id){ var element = document.getElementById("id"); if(element.style.display != 'block'){ element.style.display = 'none'; var ytobject = document.getElementById("youtubeplayerid"); if(ytobject.stopVideo) { ytobject.stopVideo(); } } else { element.style.display = 'block'; var element = document.getElementById("youtubeplayerid"); if(ytobject.playVideo) { ytobject.playVideo(); } } }
