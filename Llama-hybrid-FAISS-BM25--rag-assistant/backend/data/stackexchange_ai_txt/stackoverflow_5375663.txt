If your application uses Windows Forms, any class deriving from Control can set the AllowDrop property to true and handle D&D (almost) automatically. See here for something similar: Drag and Drop files from Windows Explorer to Windows Form Otherwise, you can still reuse Windows forms implementation at a lower level as IDropTarget is defined here: IDropTaget
