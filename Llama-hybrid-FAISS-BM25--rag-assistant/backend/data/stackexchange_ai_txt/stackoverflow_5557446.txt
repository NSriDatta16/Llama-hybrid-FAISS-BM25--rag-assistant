Do this: In finder, go to your hard drive, Go to Developer => Library and drag the cocos2d folder into this folder, Next, open terminal, type cd , and drag the cocos2d folder into the terminal window. Now do sudo ./install-templates. That should work. After, restart xcode. Hope that helps.
