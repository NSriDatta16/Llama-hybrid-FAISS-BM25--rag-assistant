To remove an element you need to copy all but the element to be removed, in PSH this can be done with Where-Object : $newArray = $oldArray | Where-Object {some-condition-on-$_} Similarly Select-Object with -First and -Skip parameters can be used to select elements before or after (resp3ectively) an index. NB System.Array does implement System.Collections.ILst but the explicit implementation of IList.Remove just throws a NotImplementedException .
