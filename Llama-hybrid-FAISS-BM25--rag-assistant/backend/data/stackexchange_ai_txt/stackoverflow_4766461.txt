If you plan to write extension to Microsoft SNMP agent, it may not support C# assemblies, CodeProject article on this If in fact you try to extend another SNMP agent, of course you should go to its vendor for assistance. Only the MIB to C# convertor provided by this vendor can be used to develop extensions for that agent.
