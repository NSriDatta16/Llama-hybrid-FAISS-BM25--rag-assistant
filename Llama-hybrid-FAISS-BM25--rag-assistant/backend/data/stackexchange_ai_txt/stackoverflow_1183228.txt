Well, it's not python and the method is not very elegant either but it'll work. After a brief inspection of the ctrl+alt+del archive site I found that all the comics are stored in /comics/ as jpg files. With that assumption, running the following command in a shell on a *nix machine will download all comics. The filenames are obviously dates in the format YYYYMMDD. $ for filename in $(seq 20020101 20090726); do wget http://www.ctrlaltdel-online.com/comics/"$filename".jpg; done Oh, and it may take a while : ).
