If you want the two NSTextView to have difference NSTextStorage , then you have to create two parallel text systems. First, start with the following architecture (this the one describe in the Text System Overview ): A common NSTextStorage A first branch with NSLayoutManager/NSTextContainer/NSTextView A second branch with NSLayoutManager/NSTextContainer/NSTextView Then, call the replaceTextStorage: method on each NSLayoutManager to set different NSTextStorage.
