A really neat way of doing this is to create your own Custom Agent by implementing UMOAgent. ... Agents have a start and Stop method which gets called by Mule, your implementing class can then add in any code you want. See link for more info http://www.mulesoft.org/documentation/display/MULE2USER/Mule+Agents
