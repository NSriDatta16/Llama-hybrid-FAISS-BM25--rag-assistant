First of all, as stated by Markos Fragkakis , use a basic instead of an ajaxified , as in your case, you have no interest of using an Ajax action here. Second, remove the immediate="true" attribute on this button. Using this attribute means that the default ActionListener should be executed immediately (i.e. during Apply Request Values phase of the request processing lifecycle), rather than waiting until the Invoke Application phase.
