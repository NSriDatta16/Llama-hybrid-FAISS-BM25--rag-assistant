/ does not return a zero division error if either the number to be divided or the denominator is a float. def compute_average(a,b,c,d,e) total = [a,b,c,d,e].sum.to_f average = [a, 2*b, 3*c, 4*d, 5*e].sum / total average.finite? ? average.round(2) : 0.0 end More generally, under ruby1.9, def compute_average *args average = args.to_enum.with_index.map{|x, w| x * w}.sum / args.sum.to_f average.finite? ? average.round(2) : 0.0 end
