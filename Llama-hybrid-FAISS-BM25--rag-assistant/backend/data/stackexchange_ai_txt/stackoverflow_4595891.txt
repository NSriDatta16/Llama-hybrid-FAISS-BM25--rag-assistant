A few things to try if you're using IE; apparently it infers less about the data from the content type than other browsers: Specify a public Pragma header, and a must-revalidate CacheControl header. Try explicitly specifying the content transfer encoding: Response.AddHeader("Content-Transfer-Encoding","binary"); .
