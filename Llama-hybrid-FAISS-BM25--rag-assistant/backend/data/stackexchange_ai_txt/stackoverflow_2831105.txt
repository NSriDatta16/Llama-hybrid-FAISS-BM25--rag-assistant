You can do it using .find(selector) and .end() like this var data = ' ... ... ... '; ​$(data).find("#remember").button().end().dialog();​​​​ You can see a quick demo here For a breakdown of how this works: $(data) - Creates a document fragment out of your string .find("#remember") - Locates the inside .button() - Creates the jQuery UI button effect on that element .end() - Returns the previous set selector, effectively $(data) .dialog() - Creates a dialog on the whole data element, since that's where .end() puts the chain.
