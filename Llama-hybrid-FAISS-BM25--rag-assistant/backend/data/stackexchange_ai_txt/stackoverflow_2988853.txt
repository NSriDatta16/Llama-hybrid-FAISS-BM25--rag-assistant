There are many organizations who have started using it. Like BBC Programmes and BBC Music, Crunchbase, OpenCalias are few examples. Many Biological data repositories store data using these technologies as well. Commercial software to store and query RDF like virtuoso are also available. Search engines have also started using and displaying such data (For example see Google snippets)
