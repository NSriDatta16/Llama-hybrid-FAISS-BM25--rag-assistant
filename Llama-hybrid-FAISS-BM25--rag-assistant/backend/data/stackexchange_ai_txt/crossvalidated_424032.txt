Optimal proposal for the Metropolis-Hastings algorithm using Tierney's theorem

Let $(E,\mathcal E,\lambda)$ be a measure space $p:E\to[0,\infty)$ be $\mathcal E$ -measurable with $$c:=\lambda p\in(0,\infty)$$ and $$\mu:=\underbrace{\frac1cp}_{=:\:\tilde p}\lambda$$ $q:E^2\to[0,\infty)$ be $\mathcal E^{\otimes2}$ -measurable with $$\lambda q(x,\;\cdot\;)=1\;\;\;\text{for all }x\in E$$ and $$Q(x,\;\cdot\;):=q(x,\;\cdot\;)\lambda\;\;\;\text{for }x\in E$$ $$\alpha(x,y):=\left.\begin{cases}\displaystyle1\wedge\frac{p(y)q(y,x)}{p(x)q(x,y)}&\text{, if }p(x)q(x,y)\ne0\\1&\text{, otherwise}\end{cases}\right\}\;\;\;\text{for }x,y\in E$$ and $$k(x,y):=\alpha(x,y)q(x,y)=\left.\begin{cases}\displaystyle q(x,y)\wedge\frac{p(y)q(y,x)}{p(x)}&\text{, if }p(x)\ne0\\q(x,y)&\text{, otherwise}\end{cases}\right\}\;\;\;\text{for all }x,y\in E$$ $f\in L^2(\mu)$ If $(X_n)_{n\in\mathbb N_0}$ is the Markov chain generated by the Metropolis-Hastings algorithm with proposal $Q$ and target $\mu$ and we want to minimize the asymptotic variance of $$A_n:=\frac1n\sum_{i=0}^{n-1}f(X_i)\;\;\;\text{for }n\in\mathbb N,$$ Tierney's theorem (a generalization of Peskun's theorem) tells us that we can equivalently maximize the quantity $$\int\mu({\rm d}x)\int\lambda({\rm d}y)k(x,y)|f(x)-f(y)|^2\tag1.$$ How is one usually solving this optimization problem (analytically)? Maybe using Lagrange multipliers ? Note that $$p(x)k(x,y)=p(x)q(x,y)\wedge p(y)q(y,x)=p(y)k(y,x)\;\;\;\text{for all }x,y\in E\tag2.$$ Moreover, using $2(a\wedge b)=a+b-|a-b|$ for all $a,b\in\mathbb R$ , we may write $$p(x)k(x,y)=\frac{p(x)q(x,y)+p(y)q(y,x)-|p(x)q(x,y)-p(y)q(y,x)|}2\tag3$$ With $(3)$ we can rewrite $(1)$ as \begin{equation} \begin{split} &\int\mu({\rm d}x)\int Q(x,{\rm d}y)|f(x)-f(y)|^2\\&\;\;\;\;-\frac1{2c}\lambda({\rm d}x)\int\lambda({\rm d}y)|p(x)q(x,y)-p(y)q(y,x)||f(x)-f(y)|^2. \end{split}\tag4 \end{equation} While I don't know how useful this is, let me finally note that $$\frac1{2c}\lambda({\rm d}x)\int\lambda({\rm d}y)|p(x)q(x,y)-p(y)q(y,x)|=\left\|\mu\otimes Q-\lambda\otimes\overleftarrow Q_\mu\right\|\tag5,$$ where $\otimes$ denotes the kernel product and $\left\|\;\cdot\;\right\|$ the total variation distance and $$\overleftarrow Q_\mu(y,\;\cdot\;):=q(\;\cdot\;,y)\mu.$$ Remark : The particular instance of the problem I'm trying to solve is this one: Variance reduction of an estimator arising from the marginal destribution of a Metropolis-Hastings chain .
