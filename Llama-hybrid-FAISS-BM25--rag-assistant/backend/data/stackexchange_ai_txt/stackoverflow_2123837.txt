I had the same problem. Dragging the library or headers to XIB Browser didn't work. Read Class Files didn't work. So I called: [MyLibraryClass version]; // Substitute your class name for "MyLibraryClass". This worked. version is a class method of NSObject, so all subclasses of NSObject inherit it.
