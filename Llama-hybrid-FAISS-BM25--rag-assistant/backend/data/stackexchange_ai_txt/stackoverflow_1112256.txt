OK, few things: Delegates are extremely simple in javascript since functions are first class members. Function.apply lets you call a function with an array of arguments. So you can write it this way function Field_Changed(delegate, args) { if (isBlocking()) return false; if (isDirty(args[0])) { //args[0] is el delegate.apply(this, args); waitForCallback(); } } And call it as: Field_Changed(Agent().Field2_Changed, [el, getApplicationState(), whatever...]);
