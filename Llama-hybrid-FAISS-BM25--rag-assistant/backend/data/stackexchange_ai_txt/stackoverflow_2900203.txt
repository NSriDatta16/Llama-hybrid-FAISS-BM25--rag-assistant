Since SQLite 3.16.0 we can use PRAGMA functions sqlite> create table my_table (a int, b TEXT); sqlite> .headers ON sqlite> .mode columns sqlite> pragma table_info(my_table); cid name type notnull dflt_value pk ---------- ---------- ---------- ---------- ---------- ---------- 0 a int 0 0 1 b TEXT 0 0 sqlite> select name from pragma_table_info('my_table'); name ---------- a b
