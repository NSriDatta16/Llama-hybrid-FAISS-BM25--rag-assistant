jQuery should not be used to parse XML . Instead, you should specify the dataType of the XmlHttpRequest to tell the browser to parse the XML, by adding , 'xml' after the callback. response will then be an XML DOM tree, which you can traverse using jQuery. For example: $.get(this.href, function(response){ var cName = $(response).find("fragment cName"); //... }, "xml");
