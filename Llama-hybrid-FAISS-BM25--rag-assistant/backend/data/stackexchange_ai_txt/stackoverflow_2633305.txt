You could make wrapper function like this: function liveDraggable(selector, options){ jQuery(selector).live("mouseover",function(){ if (!jQuery(this).data("init")) { jQuery(this).data("init", true); jQuery(this).draggable(options); } }); } (I use prototype with jQuery - that's why i placed jQuery() instead of $()) And now instead of $(selector).draggable({opts}) use liveDraggable(selector, {opts})
