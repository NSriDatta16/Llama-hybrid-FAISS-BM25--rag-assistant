It seems like your trying to create a file at the location of c:/ which is not a valid path identifier on a android system. Android comes from a linux environment. http://developer.android.com/guide/topics/data/data-storage.html follow this link to learn more about data storage on android If you want to create a file on your local pc you first need to create it on your Android device and then pull it from the device (either the emulator or your real phone)
