You can get a quick array of the names a bit simpler using .map() , like this: $(document).ready(function(){ var arrayOfClassNames = $("#myParagraph").find("[class]").map(function() { return this.className; }).get(); }); You can test it out here . If you want a string you can just do arrayOfClassNames.join('') ...or however you want to use it really.
