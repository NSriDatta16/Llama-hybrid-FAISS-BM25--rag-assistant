I think this can be done by any Javascript framework (JQuery, Scriptaculous, MooTools, Dojo) as vrinek says. You could also cook up your own Javascript solution, but stable dragging and dropping routines are at the core of most frameworks. I think whether an element is a block or inline doesn't really matter as you can set the properties for this as well in CSS.
