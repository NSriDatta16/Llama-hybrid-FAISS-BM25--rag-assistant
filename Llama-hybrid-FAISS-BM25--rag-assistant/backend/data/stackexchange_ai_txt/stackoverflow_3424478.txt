You can use an element in the main web.xml deployment descriptor to list only the jar files in WEB-INF/lib, which you want to be automatically analyzed for annotations and web-fragment.xml descriptors during deployment. If you do so, the content of web-fragments or annotated classes in other JAR files are not deployed automatically.
