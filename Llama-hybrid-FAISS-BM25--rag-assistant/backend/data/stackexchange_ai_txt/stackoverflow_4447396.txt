This has to do with this setting in /app/config/core.php: /** * When set to false, HTTP_USER_AGENT will not be checked * in the session. You might want to set the value to false, when dealing with * older versions of IE, Chrome Frame or certain web-browsing devices and AJAX */ Configure::write('Session.checkAgent', true); It's simply a hash of the user agent used as extra security measure to ensure the session cookie is unique to one machine/browser.
