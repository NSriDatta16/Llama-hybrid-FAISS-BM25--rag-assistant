Use jQuery's each() function to iterate over every one of them: $('p').each(function() { if ($.trim($(this).html()) == '&nbsp;') { $(this).remove(); } } You can also use the ":contains" selector to only get the paragraphs that contain a non-breaking space: $('p:contains("&nbsp;")')
