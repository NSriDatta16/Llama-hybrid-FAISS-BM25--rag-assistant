http://openquery.com/graph/doc Available as plugin for MySQL From the site: The Open Query GRAPH computation engine, or OQGRAPH as the engine itself is called, allows you to handle hierarchies (tree structures) but also complex graphs (nodes having many connections in several directions). [...] In the MySQL server infrastructure, OQGRAPH is a storage engine, just like MyISAM, InnoDB, MEMORY and NDBCluster, to name a few.
