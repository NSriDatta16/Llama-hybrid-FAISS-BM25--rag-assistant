Make sure you're configuring the params correctly in both places config is needed. In COMPONENTNAME.xml, you need the block you've got above (although I think only 'name' and 'default' are used here). Also, in admin/config.xml, you'll need something like: You'll then need to make sure there's a way to get to these config options, with this in your 'toolbar.COMPONENTNAME.html.php': JToolBarHelper::preferences('com_magentocatalogue'); Then, a 'config' button should appear in the toolbar for your component. Only once you save some changes will these parameters appear in the #__components.params field.
