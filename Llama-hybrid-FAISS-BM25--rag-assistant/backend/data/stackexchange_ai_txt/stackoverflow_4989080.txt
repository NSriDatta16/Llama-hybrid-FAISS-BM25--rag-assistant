You could use some of RPC frameworks, for example, Apache Thrift , which supports C++, Java, Python, PHP, Ruby, Erlang, Perl, Haskell, C#, Cocoa, Smalltalk, and OCaml. There's also BERT-RPC client and server library for Haskell , but I'm not sure a Java port exists.
