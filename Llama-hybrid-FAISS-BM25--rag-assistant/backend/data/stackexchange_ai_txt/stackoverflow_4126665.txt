Member function and "free" function pointers are not interchangeable in C++. Your pcap_callback function appears to be an instance (non-static) member of class Sniff . If it is a static member, then you have to qualify it with the class name. pcap_loop(loc,-1,Sniff::pcap_callback,NULL);
