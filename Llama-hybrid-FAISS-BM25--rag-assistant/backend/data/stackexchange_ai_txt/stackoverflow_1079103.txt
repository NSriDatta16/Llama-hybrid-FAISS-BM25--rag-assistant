It was MS_POSITIONING="GridLayout" in the body tag. But this is removed since Visual Studio 2005. You can imitate this result by doing the following: Tools - Options - HTML Designer - CSS Positioning - Positioning Options, and setting it to absolute positioned. UPDATE: Visual Studio 2008: Tools - Options - Html Designer - CSS Styling - Change positioning to absolute for controls added using Toolbox, paste or drag and drop.
