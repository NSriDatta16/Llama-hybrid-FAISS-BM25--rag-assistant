You can't do it with only the base libraries, but you can get all you want and more with the SIGAR API . Some examples of things you can get: System memory, swap, cpu, load average, uptime, logins Per-process memory, cpu, credential info, state, arguments, environment, open files File system detection and metrics Network interface detection, configuration info and metrics TCP and UDP connection tables Network routing tables
