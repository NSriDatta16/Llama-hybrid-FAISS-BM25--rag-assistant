Assigning a byte to an int works: int myInt = myByte; But maybe you're getting an exception inside IDataRecord.GetByte , in which case you should check that the index you're using to access the data record really points to a tinyint column. You can check the type returned from GetValue . It should be a byte for a tinyint column. Trace.Assert(rdr.GetValue(j).GetType() == typeof(byte)); Another option is to forego the fragile numeric index altogether: int myInt = rdr.GetByte(rdr.GetOrdinal(TheNameOfTheTinyintColumn))
