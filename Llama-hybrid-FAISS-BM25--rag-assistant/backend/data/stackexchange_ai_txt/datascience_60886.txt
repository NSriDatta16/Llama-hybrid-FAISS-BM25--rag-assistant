Use make_column_transformer and set remainder to 'passthrough' so all remaining columns that were not specified in transformers will be automatically passed through. from sklearn.compose import make_column_transformer ... df = ... transformer = make_column_transformer((TfidfVectorizer(), ['text_column']), (OneHotEncoder(), ['categorical_column']), remainder='passthrough') X = transformer.fit_transform(df)
