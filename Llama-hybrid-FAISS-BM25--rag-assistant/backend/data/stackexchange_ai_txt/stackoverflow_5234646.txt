If you're worried about that level of micro-optimisation, you should be hand-rolling assembly. Write clear and elegant code first and foremost. If speed is an issue, do some profiling and optimise the hot-spots. Given how simple the definition of Monad ((->) r) is, you really should trust GHC to inline the definitions for you, at which point the two versions become identical up to alpha-renaming.
