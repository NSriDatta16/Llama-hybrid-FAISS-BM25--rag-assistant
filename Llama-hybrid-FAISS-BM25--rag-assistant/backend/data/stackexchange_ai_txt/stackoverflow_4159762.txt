GCC always supported variable length arrays AFAIK. Setting -std to C89 doesn't turn off GCC extensions ... Edit: In fact if you check here: http://gcc.gnu.org/onlinedocs/gcc/C-Dialect-Options.html#C-Dialect-Options Under -std= you will find the following: ISO C90 programs ( certain GNU extensions that conflict with ISO C90 are disabled). Same as -ansi for C code. Pay close attention to the word "certain".
