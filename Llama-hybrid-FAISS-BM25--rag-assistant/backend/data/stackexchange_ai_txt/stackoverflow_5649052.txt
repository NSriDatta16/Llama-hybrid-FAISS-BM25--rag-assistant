Use application/octet-stream instead of image/jpg : If [the Content-Disposition] header is used in a response with the application/octet-stream content-type, the implied suggestion is that the user agent should not display the response, but directly enter a `save response as...' dialog. — RFC 2616 – 19.5.1 Content-Disposition
