If you require to know the IE version into a web application you can get the User-Agent or use javascript: You got here a Microsoft sample of how to get the internet Explorer version http://msdn.microsoft.com/en-us/library/ms537509(VS.85).aspx If you require to detect the IE Version into a Desktop program with X language you need to read the Windows registry This registry key HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer contains the attribute Version with the IE version
