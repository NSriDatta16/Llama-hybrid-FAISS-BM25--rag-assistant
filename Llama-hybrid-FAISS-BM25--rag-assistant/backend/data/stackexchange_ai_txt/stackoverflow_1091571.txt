You may want to look into a full-text search engine such as Lucene . Lucene is a separate software programming which indexes whatever you tell it to index (e.g., some of the objects in the database). When performing a search, you first use the Lucene API to search the full-text index and retrieve a set of object IDs. Then you perform an easy database retrieval to get the objects with those IDs.
