Has your Action class already implemented the SessionAware interface? EDIT: Try this(It's a struts2 style solution): public class anAction implements SessionAware{ private Map session; public Map getSession() { return session; } public void setSession(Map session) { this.session = session; } public String getLoginStatus(){ session.put("userName", "test"); //Hard code here for testing. return SUCCESS; } } Then use your jsp code to get the userName on page. I've tested this approach on my machine and it works. EDIT2: BTW, such login check can be done easily and elegant with "Interceptor" provided by Struts2 Framework.
