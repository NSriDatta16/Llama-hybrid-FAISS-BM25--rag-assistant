The variable marker is not in the global scope and only available in the local scope for placeMarker() . Try moving the var declaration to the top of you script: var initialLocation; var marker = null; var siberia = new google.maps.LatLng(60, 105); //... function placeMarker(location) { marker = new google.maps.Marker({ position: location, map: map, draggable: true }); map.setCenter(location); } Then you should be able to get the values with this: var markerPosition = marker.getPosition(); document.getElementById("t1").value=markerPosition.lat() document.getElementById("t2").value=markerPosition.lng();
