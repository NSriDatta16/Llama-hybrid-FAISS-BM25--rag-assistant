You're setting the variables but the formula itself is not returning anything. If you want to return the value of totMdp, just add it after the last line: WhilePrintingRecords; numberVar rt; numberVar layMdp; numberVar totMdp; rt=Round(({Command.GENGNPIAMT}/{Command.TOTALGNP})*100,2); layMdp:={Command.GENPREMMDP}; totMdp:=(layMdp)*Truncate((rt/100),4); totMdp
