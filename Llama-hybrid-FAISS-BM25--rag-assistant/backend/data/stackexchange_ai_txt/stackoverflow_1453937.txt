I solved my above problem using below logic I changed my SQL insert Procedure and added new Parameter @OrgID int OUTPUT and after insert command I used SET @OrgID = SCOPE_IDENTITY() RETURN Further in my Application after reconfiguring my ObjectDataSource, I got below parameter in my objectdatasource insert Parameter I write below code in my ObjectDataSource inserted event. Protected Sub RAOOrganisationDataSource_Inserted(ByVal sender As Object, ByVal e As System.Web.UI.WebControls.ObjectDataSourceStatusEventArgs) Handles RAOOrganisationDataSource.Inserted Dim OrgID As Integer = e.OutputParameters("OrgID") Session("OrgID") = OrgID End Sub And thus I got my latest inserted OrgID in Session. Cheers! Please do let me know if there is any errors while using above concept
