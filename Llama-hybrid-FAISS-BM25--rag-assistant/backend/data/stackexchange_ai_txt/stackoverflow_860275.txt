You can drag the stored procedure in the DBML editor in Visual Studio. When you go to call your stored procedure you can just call it directly. You don't need to add any special LINQ query to do this: using (MyDataContext db = MyContextFactory.CreateContext()) { db.spStoredProcedure(parameters); }
