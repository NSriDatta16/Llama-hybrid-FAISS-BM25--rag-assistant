The rails active_support gem has string extensions that can handle this. For example: # $ sudo gem install activesupport require 'active_support/core_ext/string' 'Laurent, où sont les tests ?'.mb_chars.upcase.to_s # outputs => "LAURENT, OÙ SONT LES TESTS ?"
