Your problem is probably caused because of insufficient rights to save the file. Like Hans Passant suggested, you should take a look at Isolated storage: http://msdn.microsoft.com/en-us/library/system.io.isolatedstorage.isolatedstoragefile.aspx The purpose of isolated storage is to allow applications with limited rights to save data. The MSDN page contains a code sample that should get you up-and-running.
