Unlimited could theoretically fill the disk up. However, setting a maxsize less than disk capacity would mean simply the database stops working before the disk fills up. In practice, it makes no difference. GROWTH and initial size is more important because it can affect physical file fragmentation. I've never set MAXSIZE, personally.
