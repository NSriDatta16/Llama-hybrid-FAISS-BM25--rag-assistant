#pragma warning(default:4265) It might seem like that would set the warning to it's default setting(which would be disabled), but that's not the case. It turns it on. http://msdn.microsoft.com/en-us/library/2c8f766e%28VS.80%29.aspx You can also do this: #pragma warning(X:4265) // where X is the warning level(1,2,3 or 4) that you want this warning to be generated at
