There's no API for copy/duplicate so yes, your answer is the best way. If you want to avoid the half a second delay then you'll need to do that via your application design - e.g. writing new data to a new file, or perhaps using smaller files. If you're interested in the details of IsolatedStorage performance, then this blog has done a superb analysis: http://appangles.com/blogs/mickn/wp7/?p=6
