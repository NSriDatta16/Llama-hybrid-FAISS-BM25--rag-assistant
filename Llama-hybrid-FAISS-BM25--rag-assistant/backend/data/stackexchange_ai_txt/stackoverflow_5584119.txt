Finally I could solve this.... after two hard days. I don't know if it's a bug or not but the error is how to send OUTPUT variables. The types are the same that Function Import in Model Browser properties Dim varTotalRegistros As Objects.ObjectParameter = New Objects.ObjectParameter("varKontador", GetType(Global.System.Int64)) Dim varError As Objects.ObjectParameter = New Objects.ObjectParameter("varError", GetType(Global.System.String)) Dim varRespuesta As Integer = varEntidades.spCargaArchivos(parIDPoliza, Now.Date, varError, varTotalRegistros) Good luck!!
