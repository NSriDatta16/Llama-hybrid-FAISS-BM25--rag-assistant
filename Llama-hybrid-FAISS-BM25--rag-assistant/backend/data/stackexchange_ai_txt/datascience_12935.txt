They should be uncorrelated. Just think of a simple example, where you have two variables $x_1$ and $x_2$, which are 100% correlated. If you run PCA on those, what would be the outcome? Well, you'd get the first principal component $\tilde x_1$, which would be some linear combination of your original variables $x_1$ and $x_2$ absorbing all the fluctuations and a second component which is always zero.
