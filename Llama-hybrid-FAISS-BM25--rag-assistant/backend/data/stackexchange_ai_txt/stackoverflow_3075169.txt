You can insert footer into UITableView (simple UILabel with transparent background). In Interface Builder just drag UILabel control on UITable bottom. Or you can make it in code: - (NSString *)tableView:(UITableView *)tableView titleForFooterInSection:(NSInteger)section { return @"My footer text"; }
