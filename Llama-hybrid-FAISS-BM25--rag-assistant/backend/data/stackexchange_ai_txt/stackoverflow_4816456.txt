Servlet Spec 3.0 (which is what Tomcat 7 supports) allows this by calling setDomain on SessionCookieConfig. Details here: http://download.oracle.com/javaee/6/api/javax/servlet/SessionCookieConfig.html You get SessionCookieConfig progammatically at webapp init time with a ServletContextListner - or you should be able to set it the value in web.xml.
