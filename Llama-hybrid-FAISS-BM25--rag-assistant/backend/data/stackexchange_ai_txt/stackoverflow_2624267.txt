Have a look at the containment option: Constrains dragging to within the bounds of the specified element or region. Possible string values: 'parent', 'document', 'window', [x1, y1, x2, y2]. Example: $( ".selector" ).draggable( { containment: 'parent' } ); Update: Read about the drop event. It is triggered whenever you drop a draggable onto a droppable. Say you have your droppable #droppable , then you can do it this way: $('#droppable').droppable({ drop: function(event, ui) { // update the draggable to be only moveable inside the droppable ui.draggable.draggable("option", "containment", this); } }); ui.draggable contains the element you just dropped.
