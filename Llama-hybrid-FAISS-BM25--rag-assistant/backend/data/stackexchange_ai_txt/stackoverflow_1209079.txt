I have never used ZipFile class, that being said when i send files i use Response.BinaryWrite() //Adds document content type context.Response.ContentType = currentDocument.MimeType; context.Response.Cache.SetCacheability(HttpCacheability.NoCache); context.Response.AddHeader("content-disposition", "attachment;filename=\"" + currentDocument.Name + "\""); //currentDocument.Document is the byte[] of the file context.Response.BinaryWrite(currentDocument.Document); context.Response.End();
