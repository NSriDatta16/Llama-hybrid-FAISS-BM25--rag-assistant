It is called distributed lag model. The model from of your example can be estimated with simple linear regression. You can use dynlm package for easier lag handling: > library(dynlm) > n data("USDistLag", package = "lmtest") > dynlm(consumption~L(gnp,0:4)) Time series regression with "ts" data: Start = 1967, End = 1982 Call: dynlm(formula = consumption ~ L(gnp, 0:4), data = USDistLag) Coefficients: (Intercept) L(gnp, 0:4)Series 1 L(gnp, 0:4)Series 2 L(gnp, 0:4)Series 3 -30.54113 0.46253 0.09632 -0.06309 L(gnp, 0:4)Series 4 L(gnp, 0:4)Series 5 0.13511 0.03176
