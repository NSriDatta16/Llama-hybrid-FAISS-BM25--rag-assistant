Used to be that bz2 wasn't an option for tarballs. Then someone added an option to GNU Tar to create and read bz2 archives, and pretty soon the format began to spread. So the answer is: If you believe in LZMA then submit patches to the Free Software Foundation (with all appropriate paperwork) and you'll make the world that much of a better place.
