The easiest way is to filter the person list, like so: List personList = PersonDAO.getAll(); listBox.ItemsSource = personList; listBox.DisplayMemberPath = "Name"; personListView = (CollectionView) CollectionViewSource.GetDefaultView(personList); personListView.Filter += (item) => { someCriteriaForInclusion }; When a drag occurs call personListView.Refresh() to reapply the filter.
