In Indesign create a sample of what you would like to automate. Using character and paragraph styles. Select the sample, and Go to File | Export . Open this file in a text editor. This will give you the basic structure you are trying to create. You can use whatever tech you like to fill it in.. Then once you have saved the snippet. You can drag the file onto an Indesign page to create the new item.
