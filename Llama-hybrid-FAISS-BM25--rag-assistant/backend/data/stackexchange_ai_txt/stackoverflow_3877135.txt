The pyaudio module may fit your needs, it is a wrapper aroung portaudio. On MacOsX, this hack installs the portaudio libray: svn co https://www.portaudio.com/repos/portaudio/trunk portaudio cd portaudio/ ./configure make sudo make install sudo /usr/bin/install -c -m 644 -m 644 ./include/pa_mac_core.h /usr/local/include/pa_mac_core.h sudo easy_install pyaudio Check their website: http://people.csail.mit.edu/hubert/pyaudio/
