The type of a function in Scala is denoted (Types,To,Pass) => ReturnType (you can leave off the parens if there is only a single type to pass), and the way to convert a method into a function to pass to another method is myObject.myMethod _ So, putting these together--and paying attention to the types of the Java classes: scala> def addMySize(adder: Int => Boolean, sizer: () => Int) = adder(sizer()) addMySize: ((Int) => Boolean,() => Int)Boolean scala> val a = new java.util.ArrayList[Int]() a: java.util.ArrayList[Int] = [] scala> addMySize(a.add _, a.size _) res0: Boolean = true scala> addMySize(a.add _, a.size _) res1: Boolean = true scala> println(a) [0, 1] (Note that ArrayList has an add that takes an object and returns a boolean--not one that returns void.)
