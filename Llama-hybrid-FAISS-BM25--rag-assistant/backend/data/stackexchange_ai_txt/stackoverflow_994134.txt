No, it's not possible to do this using anything provided by the iPhone SDK. The only way to do something similar would be to store the media in your iPhone application's local storage, write an OS X application that discovered your iPhone via Bonjour, synced the content via WIFI, and then added it to iTunes. Take a look at "Things" (an iPhone todo list application) if you'd like to see an example with sync-to-desktop functionality.
