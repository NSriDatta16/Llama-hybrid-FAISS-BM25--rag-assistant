It definitely could cause conflicts. If you want to use this in a multi-user environment then the configuration of Resharper needs to format your code to a set of standards which are enforced in your organization regardless of whether users make use of Resharper or not. That way you are using the tool to ensure your own code meets the standards, not blanket applying your preferences to the whole codebase.
