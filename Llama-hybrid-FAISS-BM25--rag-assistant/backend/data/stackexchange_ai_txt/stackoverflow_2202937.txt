Unfortunately, dojo.query() will only return native DOM nodes. I think you want to get back the rendered Dijit Widget. To do that, you'll need to assign your inputs IDs and use dijit.byId() . Also, unlike native HTML event names, Dojo event names are case-sensitive. So, onkeyup refers to the native HTML and is different from the Dojo event name onKeyUp . I think you may also have an extra 't' in contstraints . An example usage: Test dojo.require("dijit.form.NumberTextBox"); dojo.addOnLoad( function() { dojo.connect(dijit.byId("input1"), 'onKeyUp', function(e) { console.log('oh yeah'); }); } );
