The reference .delegate() | jQuery says about second parameter: A string containing one or more space-separated JavaScript event types , such as "click" or "keydown," or custom event names. Therefore, in your case: $("body").delegate("input", "mouseup mouseover", function() { $(this).after(" Another paragraph! "); });
