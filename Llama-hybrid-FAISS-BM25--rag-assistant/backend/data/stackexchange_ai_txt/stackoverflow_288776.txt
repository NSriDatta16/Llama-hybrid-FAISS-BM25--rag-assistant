First of all, the use of Struts and Hibernate has no impact on the interactivity of the UI, because both of those are server-side technologies. Unless you can use a technology like GWT, OpenLaszlo or Flex, the only way you`ll be able to achieve a rich user interface is by writing it yourself in raw JavaScript. This would be a lot easier if you can persuade your organisation to allow you to use a JavaScript library such as JQuery or Prototype.
