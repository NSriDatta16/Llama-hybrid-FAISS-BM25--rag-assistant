You might want to check out CMake for maintaining this large number of builds. Scripting can certainly get the job done, but once the project grows to a significant number of files, or if you find yourself changing the number of builds, changing the script could get hairy. CMake will let you organize those scripts, so it's at least a bit easier to do the modifications in the future.
