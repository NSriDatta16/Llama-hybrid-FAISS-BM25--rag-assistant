Make an Ajax call in the draggable callback function. This is because from the callback you can get the current co-ordinates, and you will be able to save them to the database. Here's an example : $("#draggable").draggable({ connectToSortable: '#sortable', helper: 'clone', revert: 'invalid', stop : function (event, ui) { //callback function $.post("savepos.php", { //parameters to send to your ajax page top: ui.position.top, left : ui.position.left } ); } });
