Yes, no problem. C++/CLI supports inheritance just as well as any other managed language. What is missing is the point-and-click IDE support, you'll have to type. Add the base form to your project, change the derived Form declaration. For example: #pragma once #include "BaseForm.h" ... public ref class Form1 : public BaseForm //
