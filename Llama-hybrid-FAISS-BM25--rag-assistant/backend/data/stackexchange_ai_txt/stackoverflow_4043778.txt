That's the onbeforeunload event ( MSDN docs , Mozilla docs ). They've added a handler to fire that prompt when you try to unload the page. In particular, if you register a function that returns a string as the event handler, the user agent will pop up an "OK/Cancel" dialog box containing the string as a message.
