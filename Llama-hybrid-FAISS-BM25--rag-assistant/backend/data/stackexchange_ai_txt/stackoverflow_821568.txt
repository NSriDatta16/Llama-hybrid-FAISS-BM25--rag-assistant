The problem is when you use :parameter_name in an sql fragment and then call query. setParamter("parameter_name", values) it is expected that values should be some sort of collection . Try this. List values = new ArrayList (); values.add(1); values.add(2); values.add(3); query.setParameter("colValues", values);
