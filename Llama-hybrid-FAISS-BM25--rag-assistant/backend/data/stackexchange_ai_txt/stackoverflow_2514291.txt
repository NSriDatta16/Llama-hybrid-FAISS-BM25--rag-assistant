Why would you do that? Tables are to be used to present tabular data. In this case there is no need to use any other markup. This is just fine and respects the HTML specification. However, you should not use tables to markup your document's layout. In this case you should use a combination of headers, divs, paragraphs, list elements, and so on... Styled with some CSS. A great book on this subject is "Bulletproof Web Design" by Dan Cederholm: http://simplebits.com/work/
