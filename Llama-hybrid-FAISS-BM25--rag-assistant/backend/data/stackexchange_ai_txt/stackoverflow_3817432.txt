In your header file, paste this: - (IBAction)backgroundClicked:(id)sender; in your implementation file, paste this: (IBAction) backgroundClicked:(id)sender { [nameField resignFirstResponder]; [numberField resignFirstResponder]; } In InterfaceBuilder, create a button that covers the entire view. with the button selected, click Layout > Send to Back Control drag from the button to File's Owner and select the outlet called: backgroundClicked.
