I believe you need to set the X509KeyStorageFlags.Exportable flag when you import the certificate. You don't show that code, but there is an overload of the Import method with this signature: public override void Import(string fileName, string password, X509KeyStorageFlags keyStorageFlags); or this one: public override void Import(byte[] rawData, string password, X509KeyStorageFlags keyStorageFlags); Which will let you set it before you import. Otherwise, everything looks good! Richard
