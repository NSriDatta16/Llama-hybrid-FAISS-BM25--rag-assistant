A more pragmatic approach: rudimentary namespacing $GLOBALS['mydomain.com']['db'] = ...; You can make a wrapper class for accessing and mutating your globals Class Globals { private $domain = 'mydomain.com'; function get($key) { return $GLOBALS[$this->domain][$key]; } function set($key,$value) { $GLOBALS[$this->domain][$key] = $value; } }
