There's nothing quite like COM in Mac OS X (I assume you're talking about OS X, not classic Mac OS). If you want to write reusable components or Cocoa applications, it really is just about as simple as: Write your classes Create a Shared Library project Build the library and distribute it For the iter-process communication and embedding parts of COM, there isn't anything standardized, really.
