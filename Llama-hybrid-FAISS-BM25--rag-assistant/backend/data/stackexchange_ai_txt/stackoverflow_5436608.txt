All you need is figure out URL to an extension, then you can just download it in any browser and drag the file over Chrome window to install it. Extension URL has the following format: https://clients2.google.com/service/update2/crx?response=redirect&x=id%3D %26uc%26lang%3Den-US&prod=chrome&prodversion=11.0.696.16 If you go to extension gallery in any browser you can get extension id from the url: /detail/
