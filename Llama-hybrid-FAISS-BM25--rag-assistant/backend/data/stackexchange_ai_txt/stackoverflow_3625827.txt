This will give you proper results. Add your database name in the query - select spid, status, loginame, hostname, blocked, db_name(dbid) as databasename, cmd from master..sysprocesses where db_name(dbid) like '% %' and spid > 50 This will include logins from SQL Agent. Note that the same user can be using multiple connections from the same application, and thus be counted more than once.
