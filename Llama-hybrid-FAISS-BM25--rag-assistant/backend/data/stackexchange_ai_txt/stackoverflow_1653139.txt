You might want to see pr(1) , which may do what you want out-of-the-box. To roll your own, expand this posix shell script fragment: ls -1 | while read f; do cat "$f"; echo This is a spacer line; done > /tmp/outputfile This might more readably be written as: ls -1 | while read f; do cat "$f" echo This is a spacer line done > /tmp/outputfile You don't really need the -1 for ls .
