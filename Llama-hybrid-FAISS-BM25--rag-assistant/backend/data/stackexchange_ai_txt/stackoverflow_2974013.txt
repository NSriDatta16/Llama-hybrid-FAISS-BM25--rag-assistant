Start a new project from the Windows Forms Application template. Project + Add New Item, select Cursor File. Draw something. Project + Properties, Resources tab. Drag and drop the cursor onto the resource window so a resource is added. Make your form constructor look like this: public Form1() { InitializeComponent(); this.Cursor = new Cursor(new System.IO.MemoryStream(Properties.Resources.Cursor1)); } Press F5 and have a look-see at your new mouse cursor.
