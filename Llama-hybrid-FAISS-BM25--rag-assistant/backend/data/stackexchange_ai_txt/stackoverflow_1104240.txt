You're looking for the Name property off of FieldType, which is available via FieldInfo. You'll need to cast MemberInfo to a FieldInfo first: foreach (MemberInfo mi in theMemberInfoArray) { if (mi.MemberType == MemberTypes.Field) { FieldInfo fi = mi as FieldInfo; Console.WriteLine(fi.FieldType.Name); } } Output: Char[] UInt16 Char Byte UInt32 UInt64
