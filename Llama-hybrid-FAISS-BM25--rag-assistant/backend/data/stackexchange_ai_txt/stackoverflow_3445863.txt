You cannot do this using pure Java. On a UNIX / Linux machine, you would configure your operating system to enable process accounting, then read the information logged in the "acct" file. According to the acct(5) manual entry I read, this logs "average" memory usage rather than maximum memory usage. A Windows system is bound to be different.
