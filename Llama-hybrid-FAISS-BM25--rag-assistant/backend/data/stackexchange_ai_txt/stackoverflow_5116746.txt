Try to use this old but still good library: FileHelpers Library It's very easy to use: char delimiter = ','; var dt = FileHelpers.CsvEngine.CsvToDataTable(fileName,delimiter); then just do: var rowStats = dt.AsEnumerable() .Select(x => x.ItemArray.Select(y => Convert.ToInt32(y))) .Select(x => new { avg = x.Average(), sum = x.Sum() }); foreach (var rowStat in rowStats) { Console.WriteLine("Sum: {0}, Avg: {1}", rowStat.sum, rowStat.avg); }
