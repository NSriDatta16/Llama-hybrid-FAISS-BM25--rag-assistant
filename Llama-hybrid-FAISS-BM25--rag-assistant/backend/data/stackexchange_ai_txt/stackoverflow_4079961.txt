No, it won't. at any point before executing the query, you would still be able to cast it back to ObjectQuery and invoke methods like Include on it: var query = context.AuditLogs.AsQueryable(); query = query.Where(o => o.WebsiteId == 1); var auditLog = ((ObjectQuery )query).Include("yourNavPropertyName") .ToList(); If your intention is to build up a criteria incrementally, then the other option would be to leverage EntitySQL with QueryBuilder methods: var query = context.AuditLogs.Where("it.WebsiteId = 1"); query = query.Where("..."); var auditLog = query.Include("yourNavPropertyName") .ToList();
