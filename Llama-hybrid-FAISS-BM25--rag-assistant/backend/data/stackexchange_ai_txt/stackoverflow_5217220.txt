You probably want to read paragraphs, which are marked by two newlines in a row. Hence: use strict; use warnings; my(@a); { # Limit the scope in which you reset the $/ variable local($/) = "\n\n"; while (<>) { s/\n+//gm; # Remove all newlines push @a, $_; } } # Now your array contains three items with no newlines - process away...
