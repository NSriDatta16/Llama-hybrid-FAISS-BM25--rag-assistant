Filestream doesn't work with SQL Authentication. From FILESTREAM Storage in SQL Server 2008 : Security There are two security requirements for using the FILESTREAM feature. Firstly, SQL Server must be configured for integrated security . Secondly, if remote access will be used, then the SMB port (445) must be enabled through any firewall systems. See also How It Works: File Streams Requires Integrated Security (Windows Authentication) .
