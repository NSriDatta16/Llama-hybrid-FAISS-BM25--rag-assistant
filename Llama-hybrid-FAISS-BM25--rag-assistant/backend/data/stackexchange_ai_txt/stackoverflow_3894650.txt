Use a DOM parser such as PHP Simple HTML DOM Parser to traverse the HTML (yes it will work on HTML fragments such as the one in your example). As you traverse, store the innerText in an array and replace that inner text with a place holder such as (or whatever you choose, so long as it's something that won't be anywhere else in the DOM and has an integer that will match up with the array key in your innerTexts array). Dump the DOM back to a string then do a global replace of the quotes. Now loop through your innerTexts array, e.g., foreach ($innerTexts as $index => $innerText) replacing with $innerText
