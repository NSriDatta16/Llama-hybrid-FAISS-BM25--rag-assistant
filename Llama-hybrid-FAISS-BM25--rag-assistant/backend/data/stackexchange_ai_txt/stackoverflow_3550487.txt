If your input data comes in as an array of strings (@strings), this for my $s (@strings) { my $output = join ' ', map /^\s*(.+)\s*$/ ? $1 : (), unpack('A19 A15 x19 A*', $s); print "$output\n" } would extract and trim the information needed.
