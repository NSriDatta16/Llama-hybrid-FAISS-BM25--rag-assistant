It sounds to me like your cancel button isn't actually cancelling anything (i.e. not running undo or rollback , as Shaggy Frog states. As for the context being saved, did you start by creating an Xcode project with the "Use Core Data for storage" box checked? If so, the generated template includes a context save on application exit. Check in - (void)applicationWillTerminate:(UIApplication *)application in your AppDelegate.
