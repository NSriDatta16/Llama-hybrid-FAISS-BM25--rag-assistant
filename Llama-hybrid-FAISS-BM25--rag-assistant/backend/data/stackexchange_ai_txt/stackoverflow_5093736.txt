Assuming that your enum is already defined, you can create an array of pairs: std::pair pairs [] = { std::pair (Linux, "Linux"), std::pair (Windows, "Windows"), std::pair (Apple, "Apple"), }; Now, you can create a map: std::map stdmap(pairs, pairs + sizeof(pairs) / sizeof(pairs[0])); Now, you can use the map. If your enum is changed, you have to add/remove pair from array pairs[]. I thinkk that it is the most elegant way to obtain a string from enum in C++.
