Yes, it is possible. Run this example, click the button to launch an Ext.Window , then drag it partially out of view. But if you uncomment the constrain: true configuration, it will not be allowed to be dragged outside of the browser view. Ext.BLANK_IMAGE_URL = 'ext-3.1.1/resources/images/default/s.gif'; Ext.onReady(function(){ var p = new Ext.Panel({ renderTo: 'panel', html: 'panel', tbar: [{ text: 'Show a Modal Window', handler: function() { new Ext.Window({ //constrain: true, title: 'Title', html: 'Window', modal: true }).show(); } }] }); });
