I've figured this out. It turns out, the Soap server is only compatible with SOAP version 1.1. In Zend, you can specify the version like so: $client = new Zend_Soap_Client('wsdl url', array('soap_version' => SOAP_1_1)); This changed the envelope and the Soap server began responding as it should.
