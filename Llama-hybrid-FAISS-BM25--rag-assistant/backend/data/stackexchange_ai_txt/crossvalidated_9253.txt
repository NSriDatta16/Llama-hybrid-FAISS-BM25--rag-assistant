$\nu$-svm parameter selection

For the $\nu$-SVM (for both classification and regression cases) the $\nu \in (0;1)$ should be selected. The LIBSVM guide suggests to use grid search for identifying the optimal value of the $C$ parameter for $C$-SVM, it also recommends to try following values $C = 2^{-5}, 2^{-3}, \dots, 2^{15}$. So the question is, are there any recommendations for values of the $\nu$ parameter in case of $\nu$-SVMs?
