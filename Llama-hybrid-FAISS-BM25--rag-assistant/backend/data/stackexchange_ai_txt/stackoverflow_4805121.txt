Either set the background in a CSS rule with the same selector as the original rule, and include your new CSS file after the original one, or make sure your new rule has a selector which has a higher specificity: http://www.w3.org/TR/CSS2/cascade.html#specificity Finally you could give the background property the !important flag, however that is usually a last resort and the sign of a badly organized style sheet.
