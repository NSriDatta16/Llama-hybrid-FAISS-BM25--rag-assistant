To make an abstract class implementing the interface: public abstract class AbstractAnalysis implements Analysis { public abstract double getAverage(); } To make a concrete class implementing the interface: public class MyAnalysis implements Analysis { public double getAverage() { // TODO } } Or, to make a concrete class extending the abstract class and thereby implementing the interface: public class MyAnalysis extends AbstractAnalysis { public double getAverage() { // TODO } }
