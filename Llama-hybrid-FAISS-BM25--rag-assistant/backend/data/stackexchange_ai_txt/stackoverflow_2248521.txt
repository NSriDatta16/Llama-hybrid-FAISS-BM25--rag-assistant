There are two ways to do it: Use -setValue: forKeyPath: instead of -setValue: forKey: ; or Grab the second object from the first object as follows: id selection = [self selection]; id storage = [selection valueForKey:@"storage"]; [storage setValue:sum forKey:@"number"];
