I typically use range to swap out text when needed. Try using Application.Selection: Word.Selection curSel = Application.Selection; curSel.TypeText("Type Some Text"); In your case, try this: Word.Selection curSel; int endOfRange = previousRange.End + 1; curSel.SetRange(endOfRange, endOfRange); curSel.Font.Bold = 1; curSel.TypeText("Hello, world!"); curSel.Font.Bold = 0; curSel.TypeParagraph();
