If your goal is to show the alert message after the .hide("normal") function is finished animating then you need to pass a "callback" function as an argument to the .hide() function. See the documentation . See an example here: $('#submit_btn').click(function(){ $("input:not([type=radio],[type=checkbox])").each(function() { if ($(this).val() == ""){ $('#'+$(this).attr("name")+'_error').show(""); } else{ $('#'+$(this).attr("name")+'_error').hide("normal", function() { // This function will be executed when the "normal" animation is complete if(!($("div[id$=error]").is(":visible"))) alert("a"); }); } }); }); My 2 Cents...
