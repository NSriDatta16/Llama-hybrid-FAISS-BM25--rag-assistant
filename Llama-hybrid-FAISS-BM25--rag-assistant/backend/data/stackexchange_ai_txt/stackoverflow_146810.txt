Zoom Editor In If your window displays both the detail and editor view, you can zoom the editor in to expand the editor view to the full height of the window. (This is fairly easily found, but many seem to overlook it.) You can do this by using one of the following methods: Command ⌘ Shift ⇧ E View > Zoom Editor In Drag the splitter (between the editor window and the file list above it) upwards.
