Override onTouchEvent in your View: @Override public boolean onTouchEvent(MotionEvent event) { super.onTouchEvent(event); if (!mEnable) { return false; } int action = event.getAction(); int x = (int)event.getX(); int y = (int)event.getY(); switch (action) { case MotionEvent.ACTION_DOWN: onTouchActionDown(x, y); break; case MotionEvent.ACTION_MOVE: onTouchActionMove(x, y); break; case MotionEvent.ACTION_UP: onTouchActionUp(x, y); break; } return true; } Then in either onTouchActionUp/Down just use the co-ordinates and redraw your image resource, i.e. invalidate() your view. Override onDraw to draw the dragged graphic: @Override protected void onDraw(Canvas canvas) drawMyDragGfx(); }
