This is not unescaped XML, this is URL encoded text. Looks to me like you want to use the following on the URL strings. URLDecoder.decode(url); This will give you the correct text. The result of decoding the like you provided is this. http://cliveg.bu.edu/people/sganguly/player/ Rang De Basanti - Tu Bin Bataye.mp3 The %20 is an escaped space character. To get the above I used the URLDecoder object.
