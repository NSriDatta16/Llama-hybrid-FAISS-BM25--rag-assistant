Yes, this is known as "unary" data (or often "implicit" data if you're only using clicks or impressions). The most common matrix factorization technique used is probably alternating least squares outlined in this paper (PDF): Hu, Koren, and Volinsky .There are implementations in many common machine learning software packages such as Mahout, Myrrix, and GraphLab.
