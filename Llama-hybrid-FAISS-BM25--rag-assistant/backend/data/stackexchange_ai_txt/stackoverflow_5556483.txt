You already got an answer on how to define the binary operators, so just as little addendum here's how you can define the unary - (like for negative numbers). > class String .. def -@ .. self.swapcase .. end .. end #=> nil >> -"foo" #=> "FOO" >> -"FOO" #=> "foo"
