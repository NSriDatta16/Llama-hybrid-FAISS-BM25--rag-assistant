Well...I eventually figured it out. I believe this is the best way to do it while using entity framework. To set the user_version you need to do this... this.DatabaseConnection.ExecuteStoreCommand("PRAGMA user_version = 5"); To get the user_version you need to do this... long result = this.DatabaseConnection.ExecuteStoreQuery ("PRAGMA user_version").FirstOrDefault();
