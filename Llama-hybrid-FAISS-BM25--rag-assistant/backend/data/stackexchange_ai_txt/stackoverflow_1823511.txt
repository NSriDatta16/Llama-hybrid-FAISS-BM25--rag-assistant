In order to get an string[] at first you must select only one string property on your query, not an anonymous object: var assetsInStorage = from a in assets.Assets where a.Name.Contains(prefixText) orderby a.Name ascending select a.Manufacturer.Name; // or a.Name assetsInStorage at this moment is an IEnumerable , and then you should convert it to string[] by: return assetsInStorage.ToArray();
