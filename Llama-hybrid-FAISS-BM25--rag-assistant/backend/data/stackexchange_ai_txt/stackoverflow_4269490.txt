Nested class documentation says (last paragraph): To instantiate an inner class, you must first instantiate the outer class. Then, create the inner object within the outer object with this syntax: OuterClass.InnerClass innerObject = outerObject.new InnerClass(); So Android can not instantiate inner class automatically. Just move class B to top-level (into separate file B.java ).
