You need something that actually parses the xml data instead of treating it as "plain" text. DOMDocumentFragment and DOMDocumentFragment::appendXML() can do that. E.g. $doc = new DOMDocument; $doc->loadhtml(' ... '); // find the parent node of the new xml fragement $xpath = new DOMXPath($doc); $parent = $xpath->query('//div[@id="formcontrols"]'); // should test this first $parent = $parent->item(0); $fragment = $doc->createDocumentFragment(); $fragment->appendXML(' '); $parent->appendChild($fragment); echo $doc->savehtml();
