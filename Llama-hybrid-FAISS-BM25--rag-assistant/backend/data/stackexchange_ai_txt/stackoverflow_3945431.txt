I'm not sure which version that is, but you should be able to do something along these lines: Right-click the field, select "Format Field" Select the "Paragraph Formatting" or "Paragraph" tab. Find the "Text interpretation" dropdown, and instead of none , select HTML text . It might be slightly different depending on the version you are using, but that's the general idea.
