You can try using HTML 5's file upload API, which will require they drag and drop the file into the browser window. Read more here: http://www.appelsiini.net/2009/10/html5-drag-and-drop-multiple-file-upload Additionally, you can do the same thing with Google Gears (which requires an install for IE but is default on Chrome). It is possible to detect the presence of either and degrade gracefully.
