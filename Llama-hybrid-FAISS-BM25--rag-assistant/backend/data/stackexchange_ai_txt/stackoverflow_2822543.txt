You can create a page break within a Run element using the element. In raw OpenXML, it would look something like: If you're using the OpenXml SDK, you can use new Paragraph( new Run( new Break(){ Type = BreakValues.Page })); EDIT: If you just want to specify that a paragraph is the last thing on a page, you can try the tag. new Paragraph( new Run( new LastRenderedPageBreak(), new Text("Last text on the page")));
