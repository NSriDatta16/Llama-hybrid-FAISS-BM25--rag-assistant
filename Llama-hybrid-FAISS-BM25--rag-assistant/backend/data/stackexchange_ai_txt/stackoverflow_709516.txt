The @INC variable contains a list of directories to in which to look for modules. It starts with the first entry and then moves on to next if it doesn't find the request module. @INC has a default value that created when perl is compiled, but you can can change it with the PERL5LIB environment variable, the lib pragma , and directly manipulating the @INC array in a BEGIN block: #!/usr/bin/perl BEGIN { @INC = (); #no modules can be found } use strict; #error: Can't locate strict.pm in @INC (@INC contains:)
