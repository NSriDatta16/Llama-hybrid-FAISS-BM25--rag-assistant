Your problem is in line 662 of your style.css for the organic natural summer theme. Change the margins to 0px, going from this: #contenthome h1, #content h1 { color: #333333; font-size: 32px; font-family: Arial, Helvetica, sans-serif; font-weight: bold; text-transform: normal; letter-spacing: -1px; margin: 5px 0px 5px 0px; padding: 0px; line-height: 32px; } To this: #contenthome h1, #content h1 { color: #333333; font-size: 32px; font-family: Arial, Helvetica, sans-serif; font-weight: bold; text-transform: normal; letter-spacing: -1px; margin: 0px; padding: 0px; line-height: 32px; } Note: Tested using Developer Tools for Chrome.
