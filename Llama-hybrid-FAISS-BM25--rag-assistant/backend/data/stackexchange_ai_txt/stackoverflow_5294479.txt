You either misremembered the question or the interviewer screwed it up when he was asking you. The key part of this question is that the reversal must be done in place , that is, without using any storage other than that which is already holding the string. The canonical answer about reversing the whole text, then re-reversing each word works in-place, while using substrings requires extra memory.
