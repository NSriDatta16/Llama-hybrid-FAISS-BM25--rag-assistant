Here, you want some kind of inter-process communication mecanism. You cannot use a PHP variable for that : these are local to the script they're in . Which means you'll have to use some "external" tool to store your data, like, to only speak of a few : a file a database (SQLite, MySQL, ...) some shared-memory segment In each case, you'll have : One script that write to the data-storage space -- i.e. your first always running script One or many other scripts that will read from the data-store
