The following doesn't answer the question with respect to Gant, but it might help with the problem: Gradle is a Groovy-build tool. It is more sophisticated than Gant. I've blogged on this here . From the Gradle FAQ ( here ): Gradle can import any Ant build script. Gradle integrates deeply with an Ant build. Every Ant target is represented as a Gradle task. This task can be further enhanced in your Gradle build script.
