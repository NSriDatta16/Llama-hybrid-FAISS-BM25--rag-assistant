Here are a few: Effectively leverage Inlining (depending on your platform). Avoid using temporaries as much as possible (and know what they are) x = y + z; Would be better optimized if written as: x=y; x+=z; Also, avoid virtual functions and only create objects when you need to use them. If you're in the mood, check out Efficient C++ . I've got a copy at home from back when I was in school.
