(via google search) To read a file from internal storage: Call openFileInput() and pass it the name of the file to read. This returns a FileInputStream. Read bytes from the file with read() . Then close the stream with close(). Ref: http://developer.android.com/guide/topics/data/data-storage.html#filesInternal
