A bit more efficient and way is to make use of uniq keys in hashes, so check this: ["a", "A"].inject(Hash.new){ |hash,j| hash[j.upcase] = j; hash}.values will return the last element, in this case ["A"] whereas using ||= as assign operator: ["a", "A"].inject(Hash.new){ |hash,j| hash[j.upcase] ||= j; hash}.values will return first element, in this case ["a"] especially for big Arrays this should be faster as we don't search the array each time using include? cheers...
