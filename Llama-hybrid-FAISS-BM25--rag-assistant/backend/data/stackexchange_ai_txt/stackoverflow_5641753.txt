Basically your problem is this: int / int = int . You instead probably want one of these: int / float = float float / int = float Try this: nums = [30,34,40,36,36,28,29,32,34,44,36,35,28,33, 29,40,36,24,26,30,30,32,34,32,28,36,24,32] def average(n): return sum(n) / float(len(n)) print average(nums)
