No it is not possible. It is possible to change the User-Agent string using the Add-on User Agent Switcher , but it does not change the behavior of the Javascript engine. Having two Firefox versions installed at a time is possible. Just choose a different installation folder. When running Firefox, use different profiles and pass the --no-remote option to prevent a new window being opened with the active Firefox application. For example, you can create the profile "firefox36" and "firefox4" and run these using: "%ProgramFiles%\Firefox 4\firefox.exe" --no-remote -P firefox4 "%ProgramFiles%\Firefox 3.6\firefox.exe" --no-remote -P firefox36
