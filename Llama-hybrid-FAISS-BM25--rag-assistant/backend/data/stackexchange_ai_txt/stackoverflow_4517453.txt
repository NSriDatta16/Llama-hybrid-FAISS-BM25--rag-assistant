You can use this the System.Uri class like this: System.Uri uri = new Uri("http://stackoverflow.com/search?q=something"); string uriWithoutScheme = uri.Host + uri.PathAndQuery + uri.Fragment; This will give you stackoverflow.com/search?q=something Edit: this also works for about:blank :-)
