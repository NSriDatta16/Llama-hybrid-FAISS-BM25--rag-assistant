Does language-agnostic means you have to use pseudocode? If not, then in Ruby: "First_name-Middle_nameLast_name".gsub('-', ' - ').gsub(/\B[A-Z]+/, ' \&').split(' ').map { |s| s[0..0] }.join.upcase => "F-ML" If it turns out the lack of space in the third example is a typo, you can skip the second call to gsub (with the ugly regexp.)
