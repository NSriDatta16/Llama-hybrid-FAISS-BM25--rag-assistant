I think you can use RoR parameter substituion, than RoR will escape your dangerous strings for you. something like: AddressBook.find(:all, :conditions => { "organization_name => ?", "Claire's Inc" }) or AddressBook.find(:all, :conditions => { :organization_name => "Claire's Inc" })
