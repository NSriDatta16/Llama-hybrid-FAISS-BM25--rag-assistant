__block storage modifier may be helpful in such a situation. __block BOOL quitRender = NO; It makes to be able to modify the variable from blocks and blocks can see the modified variable. But be careful that accessing __block variable is not thread-safe. atomic functions are preferred to ensure accessing a variable as atomic. Also you can use ivar from blocks. If the variable is a property with atomic attribute, it is thread-safe to access. How do you use quitRender variable?
