Well, theoretically , yes you could. However, it would really be a pain and not at all foolproof as it would likely rely upon javascript for implementation which could easily be circumvented. Also, in practice, just about any user agent parameter can be spoofed, so you really couldn't rely upon user agent parameter detection to determine whether or not a user agent actually supports a specific tag.
