I'm not sure if your question means that you're trying to figure out any way to display SVG in IE9, or specifically only with the tag. If you just want a way to display SVG in IE9, I recommend embedding SVG directly in XHTML5: untitled body { background:#eee; margin:1em } svg { background:#fff; display:block; border:1px solid #ccc; width:100%; margin:1em -1px } If you want to test this technique, here are some examples (on my site) using SVG that work in IE9: http://phrogz.net/svg/rainbowgrid.xhtml http://phrogz.net/svg/3-point-circle2.xhtml http://phrogz.net/svg/stirling_numbers.xhtml http://phrogz.net/svg/soldiers.xhtml http://phrogz.net/svg/drag_under_transformation.xhtml http://phrogz.net/svg/complex_butterfly.xhtml If this is not what you want, please clarify what your needs are.
