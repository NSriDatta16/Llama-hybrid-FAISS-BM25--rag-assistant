Multiple files are good for organization, but one request to the server is definitely best. If you watch the performance videos from Google they suggest the least amount of HTTP requests possible. Each HTTP request has overhead in the handshake that you do not want to incur if you wish your site to be fast. Check out this great script which will take your multiple CSS/JS files and turn them into one file: http://code.google.com/p/minify/
