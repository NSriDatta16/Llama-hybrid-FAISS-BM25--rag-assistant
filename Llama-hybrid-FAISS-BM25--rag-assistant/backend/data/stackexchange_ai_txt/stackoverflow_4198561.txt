After a bit of playing around, this is what worked for me: Create a new page called "Human Resources Page". Link to parent item "Navigation Page". Enter a menu link title "Human Resources Page" (otherwise it will not appear in menu). Create a new menu called "Human Resources Menu" (human-resources-menu) Show block on specific pages: . Add menu item called "Organization Chart". Attach to parent menu "Human Resources Menu". In blocks, assign "Human Resources Menu" to "Right Sidebar" Set "Page specific visibility settings" to hr* .
