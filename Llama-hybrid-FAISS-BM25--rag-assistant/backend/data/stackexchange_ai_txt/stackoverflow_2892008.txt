You can do this with reflection, but it's easiest and clearest to write it out, e.g. Object defaultValue(Class cls) { Map defaults = new HashMap(); defaults.put(Integer.TYPE, Integer.valueOf(0)); defaults.put(Double.TYPE, Double.valueOf(0)); defaults.put(Boolean.TYPE, Boolean.FALSE); //... etc return defaults.get(cls); } Of course, you will probably want to move the map initialization out to a constructor or similar for once-only initialization. Reasonably concise - it is elegant?
