If the XML can be imported into DataSet using ReadXml(...) method, you may store it into a database in very elegant way. Schematically: // Create and fill dataset DataSet set = ...; set.LoadXml(...); // Open sql connection, make command and adapter SqlDataAdapter adapter = ...; adapter.InsertCommand = new SqlCommand("your sql command", connection); // Save the dataset adapter.Update(set, "TableName");
