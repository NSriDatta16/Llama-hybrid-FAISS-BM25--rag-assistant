The line Yada Yada wrote: does end with a linebreak, not with a space. So your regular expression should be: /.* wrote:\n.*/m Attention: Windows systems and some protocols like HTML can use different linebreak encodings. If you want to be sure to be compatible, convert your input to unix linebreak encoding first and then do the data extraction. You could use my linebreak gem for this.
