For SQL Server 2005+, use the STUFF & FOR XML PATH to create a comma separated list: SELECT DISTINCT t.name, t.dob, t.agentid, STUFF(ISNULL(SELECT ', ' + x.agentname FROM TABLE1 x WHERE x.agentid = t.agentid GROUP BY x.agentname FOR XML PATH ('')), ''), 1, 2, '') FROM TABLE1 t
