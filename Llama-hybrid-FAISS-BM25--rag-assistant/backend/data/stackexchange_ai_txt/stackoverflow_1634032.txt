If you liked the Do-It-Yourself-Framework tutorial mentioned before, but you want to manage these things in a config file, Paste Deploy would be the obvious answer. (It is mentioned in the tutorial, but only very briefly in the very last paragraph). This is what the Pylons framework uses, by the way (and Turbogears 2, which is built upon Pylons, also).
