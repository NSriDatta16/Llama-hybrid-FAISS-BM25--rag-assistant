I adopted the Project Naming guidelines from Blitz: Blitz Project Naming Conventions & Organizational Guidelines My DEV_Source is divided in database/flash/flex directories. The Flex directory is standard to the Adobe conventions, but the flash has a custom setup. bin -- Output of SWF, contains JS and index.html classes -- AS3 code doc -- AS Doc output of code libs -- 3rd party libraries and components src -- FLA files (set the Publish settings to compile in bin) test -- AS Unit test cases of classes
