Hook up the KeyPress event to a method like this: protected void myCombo_OnKeyPress(object sender, KeyPressEventArgs e) { if (e.KeyChar == 13) { MessageBox.Show("Enter pressed", "Attention"); } } I've tested this in a WinForms application with VS2008 and it works. If it isn't working for you, please post your code.
