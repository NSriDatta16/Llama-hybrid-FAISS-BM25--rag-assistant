I guess this is a question for Stack Overflow, but (at least in R) it's quite simple with the dplyr package: library(dplyr) df % mutate(date = as.Date(observationTimeUtcIso)) by_date % group_by(date) by_date %>% summarise( SumRain = sum(precip1Hour), AverageTemp = mean(temperature)) # A tibble: 2 x 3 date SumRain AverageTemp 1 2018-02-14 1.41 6.533333 2 2018-02-15 1.23 6.800000
