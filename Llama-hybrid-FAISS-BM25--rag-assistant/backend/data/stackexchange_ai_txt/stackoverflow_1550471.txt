What is wrong with "mounting" the SAN LUNs as block devices, formatting the device with a filesystem and accessing it using the normal file IO operations? By the way: A SAN normally cannot be accessed via a network share? A SAN is a block-level abstraction. But some NAS (Network Attached Storage) Appliances like a NetApp Filer provide in addition to the CIFS/NFS access a block level access.
