Right click your exe file listed in the File System Browser in Setup Project. Choose Create Shortcut. If you want to place this shortcut in desktop drag the shortcut file to User's Desktop folder. If you want to place the shortcut in a folder in program menu then add a folder under User's Program Menu folder and drag the shortcut to that folder.
