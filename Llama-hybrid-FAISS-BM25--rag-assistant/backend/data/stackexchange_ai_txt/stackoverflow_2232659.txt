Time series data from the ?ts example. gnp The seq function can work with date objects. The example above provides the starting date, specifies a monthly frequency and signifies how long of a date vector to create. Hopefully this is helpfully in your data preparation before using ggplot2 or something else. You can combine the example above into a data.frame like this: dat This can be plotted in ggplot like this: ggplot(data=dat) + geom_line(aes(date, gnp)) All the best, Jay
