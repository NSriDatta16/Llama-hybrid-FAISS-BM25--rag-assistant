Classes describe a model/concept/type and defines the possible behavior and possible states of that (in your example, a Person can have a name, address, etc. A data structure is some type that can be used to organize and group uh...data, in some way. For example, vectors and linked lists are data structures that can used for storing data in an ordered way. You can have a class that represents a data structure, like std::vector in C++, or java.util.ArrayList in Java.
