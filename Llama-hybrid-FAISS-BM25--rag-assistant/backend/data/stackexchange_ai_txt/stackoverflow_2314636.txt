You can actually use the CPP __LINE__ pragma in Haskell. {-# LANGUAGE CPP #-} main = do print "one" print __LINE__ $ runhaskell A.hs "one" 5 Also, the Control.Exception.assert function will emit a line number if its condition fails. import Control.Exception main = do print "one" assert False $ print "two" $ runhaskell A.hs "one" A.hs: A.hs:5:5-10: Assertion failed
