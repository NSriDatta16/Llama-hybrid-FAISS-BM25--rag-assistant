Apply the following style: a, a:link, a:visited, a:hover { text-decoration: none; } I've intentionally given you complete coverage of all the states that an tag can have, but you very well may be able to get away with the following as well: a { text-decoration: none; } Finally, if you only want this applied to the footer: #footer a, #footer a:link, #footer a:visited, #footer a:hover { text-decoration: none; }
