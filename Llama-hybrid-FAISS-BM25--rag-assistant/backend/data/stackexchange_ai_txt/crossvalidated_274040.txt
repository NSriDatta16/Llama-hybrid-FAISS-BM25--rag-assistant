Supervised Learning and Poisson Regression

I'm starting to studying something about Machine Learning and I've a doubt. Given $Y$ the ouput, $X \in \mathcal{X}$ the input, let $L(Y,f(X))=(Y-f(X))^2$ the loss funtion. Then $f$ should be the function that minimizes $$ \int (y-f(x))^2 dF_{(X,Y)}(x,y) $$ where $F_{(X,Y)}(x,y)$ is the joint distribution of $(X,Y)$. So $\hat{f}(x)$ approximates $\mathbb{E}[Y|X=x]$. But often I read that $f$ is a function such that $Y=f(X)$ or $Y=f(X)+\epsilon$, where $\epsilon$ is a noise. The problem I have is the following: $i)$ If $Y\sim \text{Poisson}(\lambda)$ then if $f(x)$ is an approximation of $\mathbb{E}[Y|X=x]$, so $f(x)$ can assume all positive values. $ii)$ While if I assume $Y=f(X)$, $f$ must map an element of the feature space to a natural number , hence $\hat{f}:\mathcal{X} \rightarrow \mathbb{N}$. The correct assumption is the first one, $\hat{f}(X) \approx \mathbb{E}[Y|X]$?
