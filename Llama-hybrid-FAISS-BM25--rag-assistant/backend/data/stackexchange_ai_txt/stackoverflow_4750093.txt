Not sure you need a generic list here. You can just use a PowerShell array e.g.: $items = @(Get-ChildItem '\\server\C$\Program Files (x86)\Data1\' -r) $items += @(Get-ChildItem '\\server\C$\Web\DataStorage\' -r) PowerShell arrays can be concatenated using += .
