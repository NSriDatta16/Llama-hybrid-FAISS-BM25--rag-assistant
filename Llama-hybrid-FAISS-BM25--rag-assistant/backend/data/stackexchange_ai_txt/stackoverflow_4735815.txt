Got the answer from another post here: Check the file app/design/adminhtml/default/default/layout/sales.xml for the section - which matches the requested URL. It has a block of type "adminhtml/report_sales_sales" which is a grid container, grid containers automatically create their child grid block. In case anyone is having similar issue, here is the link with details: Configuring Magento Layout for New Admin Module
