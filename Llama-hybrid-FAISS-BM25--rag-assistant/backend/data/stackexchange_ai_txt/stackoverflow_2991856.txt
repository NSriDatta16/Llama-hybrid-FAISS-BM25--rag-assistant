The documentation at http://docs.dojocampus.org/dojox/grid/DataGrid#required-css says you need this css: @import "/moin_static163/js/dojo/trunk/dojox/grid/resources/Grid.css"; @import "/moin_static163/js/dojo/trunk/dojox/grid/resources/tundraGrid.css"; .dojoxGrid table { margin: 0; } html, body { width: 100%; height: 100%; margin: 0; } I used this which worked too: .dojoxGrid table { margin: 0; } html, body { width: 100%; height: 100%; margin: 0; } In my application I also had a conflict with my global css which I fixed using height: 999px instead of 100%.
