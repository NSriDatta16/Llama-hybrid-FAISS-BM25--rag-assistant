SQLite does not enforce text storage limits. If you declare a column VARCHAR(1) or TEXT(1) you still can store some very long blob of 500 Megabytes inside it. Even though that is not adviseable. ~$ sqlite SQLite version 3.7.2 Enter ".help" for instructions Enter SQL statements terminated with a ";" sqlite> create table foo ( bar text(3)); sqlite> insert into foo values ('aeiou'); sqlite> select * from foo; aeiou
