JDOM XSLTTransformer is a convenience wrapper around javax.xml.transform.Transformer for JDOM input and output. A JDOM input is easily transformed to text output. Transformer transformer = TransformerFactory.newInstance().newTransformer(new StreamSource(stylesheet)); JDOMSource in = new JDOMSource(doc); StringWriter writer = new StringWriter(); StreamResult out = new StreamResult(writer); transformer.transform(in, out); return writer.toString();
