While researching this topic, I found this proof-of-concept code (via a comment at http://jlebar.com/2010/2/1/Replacing_Bash.html ) that lets you "write shell-like pipelines in Python using a terse syntax, and leveraging existing system tools where they make sense": for line in sh("cat /tmp/junk2") | cut(d=',',f=1) | 'sort' | uniq: sys.stdout.write(line)
