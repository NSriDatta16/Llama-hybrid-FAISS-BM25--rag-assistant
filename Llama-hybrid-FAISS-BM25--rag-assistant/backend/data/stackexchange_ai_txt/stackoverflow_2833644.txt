Error is there because as per th rule when your using aggregate function with group by clause you have to include the columns which are in your select list following may help you to achieve functionality you want (select count(distinct requirementid),requirementid, from RequirementsDetailsforOrganization group by requirementid) d inner join ( SELECT req.requirementid, org.organizationid, req.locationofposting,org.registereddate FROM OrganizationRegisteredDetails AS org, RequirementsDetailsforOrganization AS req WHERE org.organizationid =req.requirementid order by org.RegisteredDate desc) d1 on d.requirementid= d1.requirementid
