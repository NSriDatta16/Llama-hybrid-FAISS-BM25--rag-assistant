The function would be something like this (untested): public int GetAverageAge (IEnumerable list, Func ageFunc) { int accum = 0; int counter = 0; foreach(T item in list) { accum += ageFunc(item); counter++; } return accum/counter; } You could use the LINQ Average method instead: public int GetAverageAge (IEnumerable list, Func ageFunc) { return (int)list.Average(ageFunc); }
