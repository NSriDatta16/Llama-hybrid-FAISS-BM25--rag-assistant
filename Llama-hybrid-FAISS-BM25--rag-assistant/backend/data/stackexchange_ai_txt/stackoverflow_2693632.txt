Try something like this. It gives offsets for the draggable element and the droppable element. And with a little math, you can get the position of the draggable element relative to the top left corner of the droppable. $('#droppable').droppable({ drop: function(event, ui) { var thisOffset = $(this).offset() var x = ui.offset.left - thisOffset.left; var y = ui.offset.top - thisOffset.top; alert(x+','+y); } });
