Your ContetPages will have different images and you want to trigger the SetupFeaturedProperty-function from MasterPage? If so, try following: MasterPage.master ... function mp_onload() { if(window.body_onload != null) window.body_onload(); } ... If you have a content page that may require an onload event, then you just need to create a function called body_onload in Headers content area of each page that requires it. it'll get called from MasterPage automatically.
