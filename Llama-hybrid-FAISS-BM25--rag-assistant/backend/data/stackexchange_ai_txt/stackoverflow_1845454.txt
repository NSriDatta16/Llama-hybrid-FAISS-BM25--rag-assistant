You want to determine the sum of all the nodes in the tree? Tree walking lends itself to an elegant recursive solution: public int SumTree (TreeNode n) { if(n.isLeafNode) return n.value; return SumTree(n.left) + SumTree(n.right); } Assuming a binary tree.
