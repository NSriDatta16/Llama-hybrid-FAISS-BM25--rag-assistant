Here is what I have done in that case: Since you have several properties you want to bind to on each button, including a command, I would make a separate, small ViewModel ( MyButtonViewModel ) for each button, which includes the properties and command. Then, in the primary ViewModel, I would create a collection ( ObservableCollection ) of your button view models, and populate it using some simple looping (or LINQ) code. Then, in the View, I would declare an ItemsControl to use an ItemPanel of UniformGrid . Then, For the ItemTemplate , I'd define the button to bind to the properties in MyButtonViewModel . Simple, small, and elegant IMO. Let me know if you need any clarification, B
