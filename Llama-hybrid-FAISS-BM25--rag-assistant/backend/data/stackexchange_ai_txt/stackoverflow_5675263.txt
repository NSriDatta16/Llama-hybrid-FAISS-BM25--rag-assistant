One way is to use a function and DOM methods: var elementData = { tagName: 'div', properties: { id: 'div0', onclick: function(){alert(this.id);}, className: 'someClass' }, content: ' New Div ' } function elementBuilder(obj) { var el = document.createElement(obj.tagName); var props = obj.properties; for (var p in props) { if (props.hasOwnProperty(p)) { el[p] = props[p]; } } if (typeof obj.content == 'string') { el.innerHTML = obj.content; } return el; } Insert element The above could be expanded to create more than one element per call, I'll leave that you. Another method is to take an HTML string, convert it to an element (or elements) and return it in a document fragment: var htmlString = ' paragraph ' + ' Another p '; function byInnerHTML(s) { var d = document.createElement('div'); d.innerHTML = s; var frag = document.createDocumentFragment(); while (d.firstChild) { frag.appendChild(d.firstChild); } return frag; } Insert element The second one returns a document fragment that can be inserted into the page. Of course if you want to generate parts of a table, the second method isn't quite so good.
