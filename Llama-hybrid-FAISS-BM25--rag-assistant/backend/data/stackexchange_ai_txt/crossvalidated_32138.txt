The best way would be to explicitly supply the tuneGrid dataframe. For instance, random forest has only one tuning parameter, 'mtry', which controls the number of features selected for each tree. To set mtry at a specific value, you might choose the randomForest default (?randomForest) do this: model where M is the one value of the tuning parameter you wish to use. for multiple tuning parameters do this: tuneGrid = data.frame(.par1 = P1, .par2 = P2, .par3 = P3)
