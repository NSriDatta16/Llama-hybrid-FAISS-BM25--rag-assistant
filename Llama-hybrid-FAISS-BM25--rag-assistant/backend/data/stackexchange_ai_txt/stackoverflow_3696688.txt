UniData is an extended relational database and integrated development environment. It is part of the MultiValue Database family. It features its own query language called UniQuery and programming language called UniBasic. UniData was originally developed by the Unidata Corporation for embedding in vertical software applications, but after several acquisitions, including by IBM, it is now owned and developed by Rocket Software.
