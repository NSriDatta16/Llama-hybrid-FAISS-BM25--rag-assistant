Browser Mode refers to the user-agent header sent by the browser. Since the UA has already been sent before the HTTP response comes back containing your EmulateIE7 directive, it's too late to change the browser mode; only the document mode is changed. See http://msdn.microsoft.com/en-us/library/dd565628(VS.85).aspx#bdmodes
