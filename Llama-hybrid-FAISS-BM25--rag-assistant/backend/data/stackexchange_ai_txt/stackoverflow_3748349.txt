You need to use the WebRequest or WebClient classes to manually download the content; they allow you to set headers. EDIT : For example: var request = (HttpWebRequest)WebRequest.Create(url); request.UserAgent = "..."; using (var response = request.GetResponse()) using (var responseStream = response.GetResponseStream()) using (var reader = XmlReader.Create(responseStream)) { ... }
