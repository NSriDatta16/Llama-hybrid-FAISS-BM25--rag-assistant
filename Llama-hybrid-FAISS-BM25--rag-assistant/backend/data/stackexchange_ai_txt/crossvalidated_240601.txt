In your case, you can use logistic regression. If you are using SAS, here is the code to use logistic regression in SAS. proc logistic data = a; class a(ref='no') b(ref='no') b(ref='no') b(ref='no')/param=ref; freq count; model outputv(event='yes')= a b c d; output out = out pred=pred; run; data aout; set out; if pred = 0.40 then yhat=1; run; proc freq data= aout; table outputv*yhat; weight count; run;
