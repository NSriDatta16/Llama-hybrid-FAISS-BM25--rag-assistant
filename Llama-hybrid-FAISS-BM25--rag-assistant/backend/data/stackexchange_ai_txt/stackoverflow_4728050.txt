Why do you need a style for a TreeViewItem? It can be done using DataTemplates only, and don't specify a key if you want them to apply automatically. tree.ItemsSource = new object[]{ new DiskSpec{Title = "Disc 1"}, new DiskSpec{Title = "Disc 2"}, new DiskSet{Title = "Set 1", Disks = new List {new DiskSpec{Title="Disc 1.1"}}}, new DiskSet{Title = "Set 2", Disks = new List {new DiskSpec{Title="Disc 2.1"}, new DiskSpec{Title="Disc 2.2"}}}}; After that you can enable drag in the Disk template and drop in the Set template.
