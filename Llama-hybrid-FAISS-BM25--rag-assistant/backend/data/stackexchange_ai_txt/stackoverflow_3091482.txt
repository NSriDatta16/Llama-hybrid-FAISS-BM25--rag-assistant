I think you can create UDF to perform required strings manipulation on DB level. http://msdn.microsoft.com/en-us/library/bb386973.aspx http://weblogs.asp.net/scottgu/archive/2007/08/16/linq-to-sql-part-6-retrieving-data-using-stored-procedures.aspx Create UDF. Use LTRIM, RTRIM, LOWER, REPLACE functions. Something like this. CREATE FUNCTION dbo.ToUrlString(@string VARCHAR(MAX)) RETURNS VARCHAR(MAX) BEGIN RETURN REPLACE(REPLACE(LOWER(LTRIM(RTRIM(@string))), ' ','-'), ':', '') END Add it as a method to your DataContext. Just drag'n'drop it from Server Explorer onto L2S designer. I hope it helps.
