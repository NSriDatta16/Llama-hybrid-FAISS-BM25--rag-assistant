The general strategy to accomplish this within the context of decision trees is to create dummy variables representing the various forms of variable interaction . However, you can achieve the same result (mostly) by the various techniques in random forest implementations. If you're asking whether a decision tree can create new data to decide on based on the prior node's result, you will have to write your own algorithm to add a transformation step prior to each split.
