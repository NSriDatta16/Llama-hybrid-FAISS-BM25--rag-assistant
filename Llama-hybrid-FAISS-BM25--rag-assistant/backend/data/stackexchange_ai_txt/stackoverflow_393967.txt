When you run load tests on multi-core computers, load generation is limited as follows: If the computer is running Visual Studio Team System Test Edition or Visual Studio Team System, the load generation is limited to one core. If the computer is running Visual Studio Team System Test Load Agent, the load generation is not limited; it runs on all cores and processors.
