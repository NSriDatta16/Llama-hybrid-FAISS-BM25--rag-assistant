Solution: The problem definitely lies with the elseif ($_GET['mode']='edit') { line; the = operator there sets $_GET['mode'] to 'edit' (which always evaluates to true ). A good but lexically-confusing practice to get into is writing conditionals like so: if (5 == $some_var) Which will immediately give an error if the second = was not included.
