you should use more than one css file rather using one big file. It helps you while maintaining your site also use different definitions (classe or id names) in different css otherwise it will take the one which declared later. But for performance reasone you can use one large file because, One large CSS file leads to fewer HTTP requests, which can improve performance. Several smaller files leads to easier organization which will make development and maintenance cheaper and easier.
