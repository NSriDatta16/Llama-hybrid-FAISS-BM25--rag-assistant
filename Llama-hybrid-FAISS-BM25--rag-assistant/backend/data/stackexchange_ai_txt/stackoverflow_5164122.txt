You're not likely to find anything that specific, but there are NLP libraries which have NLP capabilities which you can design your app around. See NLP programming tools using PHP? which accepted Solr as the recommended solution. You'll likely have more luck if you relax the language constraints: much more NLP of that nature is written in Java (e.g. Stanford JavaNLP ), C/C++ , or even Lisp .
