Use PostMessage, not SendMessage: SendMessage can take a while, and this may cause delays in other applications. When initializing the hook, give your application's HWND in a to the hook DLL using a function you define. This should be stored in a shared data segment in the DLL, like this: #pragma data_seg(".shared") HWND hWndApp = NULL; #pragma data_seg() #pragma comment("linker, /section:.shared,rws") void setAppWnd(HWND hWnd) { hWndApp = hWnd; } You can see a full example for doing this here .
