A proper JSON request should look like this if you have a single cat: { 'id': '123', 'Data': [ { 'Name': 'Felix', 'Age': 6 } ] } and for multiple cats: { 'id': '123', 'Data': [ { 'Name': 'Felix', 'Age': 6 }, { 'Name': 'Morgan', 'Age': 2 } ] }
