It seems that you have to use a SqlMethods.DateDiff in linq2sql statement (DATEDIFF function if SQL): public int getAverageTime() { return (from EventTime in db.EventTimes where EventTime.EventTimesID > 0 select (SqlMethods.DateDiffMinute(EventTime.TimeStart, EventTime.TimeEnd)).Average(); }
