You can do it like this: $("#cart").droppable({ activeClass: "ui-state-default", hoverClass: "ui-state-hover", accept: ":not(.ui-sortable-helper)", drop: function(event, ui) { $(this).find(".placeholder").remove(); $(" ").append(ui.draggable.contents().clone()).appendTo(this); } }); This actually clones the elements by using .contents() , if you need any event handlers and data, use .clone(true) instead.
