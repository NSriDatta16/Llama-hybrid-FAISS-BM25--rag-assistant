You would redirect based on the User Agent. Doing this in JavaScript is possible, however, it would mean the iPhone visitor would have to load the full size page before getting redirected. Another approach would be to handle the redirecting in Apache with mod_rewrite. Here are a few links that would allow you to accomplish what you're looking for.
