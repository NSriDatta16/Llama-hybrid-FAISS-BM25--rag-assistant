There are two ways to fix the problem: You have to wrap the parameter passed to getAndSaveValue in quotation marks (otherwise you pass the variable price_productID (which is undefined ) to the function and not the string): onclick="getAndSaveValue('price_productID')" // ---^ ---^ Pass the element directly: onclick="getAndSaveValue(this)" then you have to change the function to: function getAndSaveValue(element) { var thisValue = element.value; } Side note : At least for element retrieval and traversing, I would use jQuery consistently. In solution #2 it is not necessary, but in #1, you could change your code to: onclick="getAndSaveValue('#price_productID')" // ---^ and var thisValue = $(elementId).val();
