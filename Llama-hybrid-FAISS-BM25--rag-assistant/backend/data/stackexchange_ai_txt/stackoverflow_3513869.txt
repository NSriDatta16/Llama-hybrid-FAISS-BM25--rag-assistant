There was a Python PEP to add this notation to Python, but it was rejected. Robert Bradshaw decided to implement it anyways, but for the Sage preparser. He implemented the following: (a..b) -- like xrange, so an iterator [a..b] -- list, including endpoints [a,b,..,c] -- arithmetic progression
