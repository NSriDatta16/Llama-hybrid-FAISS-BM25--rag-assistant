If you can find a pattern in UserAgent string, you may use DISALLOWED_USER_AGENT setting. Mine is: DISALLOWED_USER_AGENTS = ( re.compile(r'Java'), re.compile(r'gigamega'), re.compile(r'litefinder'), ) See the description in Django docs .
