One approach is to leverage an open-source solution for this. Log4Net is one that many of my peers have used over and over again. It has the functionality to handle what you are talking about. You add the reference, add a config file to set your logging options (how much to keep, etc), and call the library to log. http://logging.apache.org/log4net/
