Best practice is use shared storage, but if can't use it, than you can use "proxy_store" option from nginx. Example from nginx doc: location /images/ { root /data/www; error_page 404 = @fetch; } location @fetch { internal; proxy_pass http://backend; proxy_store on; proxy_store_access user:rw group:rw all:r; proxy_temp_path /data/temp; root /data/www; }
