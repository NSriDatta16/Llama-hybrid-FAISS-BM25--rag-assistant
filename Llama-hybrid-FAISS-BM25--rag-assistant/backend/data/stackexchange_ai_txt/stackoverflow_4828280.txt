If you are uploading file using HttpWebRequest, you must specify content headers correct. Content must be specified as Content-Type: multipart/form-data sample snippet string DataBoundary = "-----------------------------" + DateTime.Now.Ticks.ToString("x"); string contentType = "multipart/form-data; boundary=" + DataBoundary ; req.Method = "POST"; req.ContentType = contentType ; req.UserAgent = userAgent; req.CookieContainer = new CookieContainer(); req.ContentLength = formData.Length; Check out this and this posts for more detail explanation
