You have to use OLE Drag&Drop , but then you have to be able to process the data you receive, because each application can store data in its own format. You can find a nice Delphi implementation of OLE Drag&Drop here Outlook Express and Thunderbird should transfer you data in RFC2822 format or something alike, Outlook will probably transfer data in its own message format, it shuould have been documented as part of the Microsoft Open Specification program
