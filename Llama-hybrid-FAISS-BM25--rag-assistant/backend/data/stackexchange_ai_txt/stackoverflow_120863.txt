You can import the eclipse plugins into your workspace by using import -> Plugins and Fragments from the package explorer. Then use the following options: Select from all plug-ins and fragments found at the specified location import from the target platform with source folders (last option) Import the org.eclipse.pde.ui plugin The code you seek is in org.eclipse.pde.internal.ui.editor.plugin More specifically, the ExtensionPointsPage class
