I assume that you used netsnmp snmpget. Well, it hides too many details from you, as it loads MIB documents in background and nicely translate OIDs and SNMP values to all kinds of user friendly formats. So next time pay attention to what decoration it performs and simulate that in your own code to achieve the same effects.
