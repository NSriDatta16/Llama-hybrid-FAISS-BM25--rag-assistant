The pragma pack and pop might be helpful. The example below will align each member on one byte boundaries. In VS this can also be set on a project level by changing the project setting C++/CodeGeneration/StructMemberAlignment. #pragma pack(push, 1) struct test { int a; char b[3]; double c; }; #pragma pack(pop)
