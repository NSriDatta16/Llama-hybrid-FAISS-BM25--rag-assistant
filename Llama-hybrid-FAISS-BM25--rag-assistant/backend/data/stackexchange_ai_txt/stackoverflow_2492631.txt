iTextSharp supports several barcode standards (as per the tutorial at http://itextsharp.sourceforge.net/tutorial/ch09.html ) * Barcode39: code 39 and code 39 extended * Barcode128: code 128 and the special encoding UCC/EAN-128 * BarcodeEAN: EAN13, EAN8, UPCA, UPCE, and EAN with supplemental 5, EAN with supplemental 5, EAN with supplemental 2 * BarcodeInter25: interleaved 2 of 5 * BarcodePostnet: postnet and planet
