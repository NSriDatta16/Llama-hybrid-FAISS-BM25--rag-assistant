You cannot cast, you must convert. You can use the built-in compiler intrinsic _bstr_t (from comutil.h ) to help you do this easily. Sample: #include #include #pragma comment( lib, "comsuppwd.lib") int main() { LPTSTR p = "Hello, String"; _bstr_t bt = p; BSTR bstr = bt; bstr; }
