Instead of using a double pointer, would it not be more elegant to use an NSMutableString instead? - (void)createAString:(NSMutableString *)str { [str setString:@"Hi all!"]; } .... NSMutableString *createStr = [[NSMutableString alloc] init]; [self createAString: createStr]; NSLog(@"%@", createStr); [createStr release]; Or, even better, just have the createAString method return an NSString . - (NSString *)createAString { return @"Hi all!"; // this is autoreleased automatically } I wouldn't want to presume that your needs are this simple, though. =)
