A subscription is a SQL-Server Agent job, and is given a GUID as a title. Check the SQL-Server Agent job history, and if necessary stop the job. If you have more than one subscription, check this link for a useful View on which jobs correspond to which subscription. http://www.sqlservercentral.com/Forums/Topic254010-150-1.aspx
