TSQL is a language specification, not an actual database. If you're talking about MS SQL Server, it is possible to bundle and install an instance of SQL Server Express, if you want to distribute a database with your app. There's some documentation on it over on MSDN , specifically, Embedding SQL Server Express into Custom Applications.
