Do you have to use a regular expression? Tools like COCO/R could make this job pretty easy as well. In addition it might just prove to be faster than generating code at runtime using a regex. COMPILER YourParaProcessor // your code goes here TOKENS newLine= '\r'|'\n'. paraLetter = ANY - '\n' - '\r' . YourParaProcessor = {Paragraph} . Paragraph = {paraLetter} '\r\n' .
