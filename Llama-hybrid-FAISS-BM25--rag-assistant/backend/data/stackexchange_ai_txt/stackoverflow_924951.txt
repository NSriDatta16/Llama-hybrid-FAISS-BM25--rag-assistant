As Johannes has already suggested , a background image is probably the best way to achieve this in CSS alone. A JavaScript solution would also have to affect "dragstart" to be effective across all popular browsers. JavaScript: your text jQuery: var _preventDefault = function(evt) { evt.preventDefault(); }; $("div").bind("dragstart", _preventDefault).bind("selectstart", _preventDefault);
