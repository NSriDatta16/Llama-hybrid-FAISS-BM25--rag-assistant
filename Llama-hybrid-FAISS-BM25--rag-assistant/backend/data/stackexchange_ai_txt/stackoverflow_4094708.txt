You can use the Ping.Send(IPAddress) method. Edit: Or one of the overloads that takes the hostname directly. Ping.Send(string) Example from MSDN: Ping pingSender = new Ping (); PingReply reply = pingSender.Send ("www.contoso.com"); if (reply.Status == IPStatus.Success) { Console.WriteLine ("Address: {0}", reply.Address.ToString ()); Console.WriteLine ("RoundTrip time: {0}", reply.RoundtripTime); Console.WriteLine ("Time to live: {0}", reply.Options.Ttl); Console.WriteLine ("Don't fragment: {0}", reply.Options.DontFragment); Console.WriteLine ("Buffer size: {0}", reply.Buffer.Length); } else { Console.WriteLine (reply.Status); }
