It depends whether you are coding a server or a client application, whether you can leverage Outlook on the client and which Exchange server you are talking to. Client with Outlook: You can use the Outlook object model. Exchange 2007/2010: Use the Exchange Managed API ( http://msdn.microsoft.com/en-us/library/dd633710(EXCHG.80).aspx ). Exchange 2000/2003: Use WebDAV. I have some articles on how to do this on my website. Have a look at my website (http://www.infinitec.de), I have several articles on accessing Exchange programmatically.
