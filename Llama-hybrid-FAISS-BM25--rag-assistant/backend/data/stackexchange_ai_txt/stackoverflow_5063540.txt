If you want to retrieve the average grade's together with the main query: SELECT * FROM ES_TOOL INNER JOIN ES_HARDWARE ON ES_HARDWARE.eshw_ID = ES_TOOL.ESTOOL_HARDWARE INNER JOIN ES_PAYMENT on ES_payment.espay_id = es_TOOL.estool_payment LEFT JOIN ( SELECT b.estch_tool, avg(a.esmrk_value) AvgValue FROM ES_MARK a Inner Join ES_TOOL_CHAPTER b ON a.esmark_tool_chapter = b.estch_id GROUP BY b.estch_tool ) g ON ES_TOOL.ToolColumn = g.estch_tool Replace ToolColumn with the correct column name. Note: SELECT * is not recommended, please read This question .
