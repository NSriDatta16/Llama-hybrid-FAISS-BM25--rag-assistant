It depends on the RDBMS to some extent. In SQL Server (prior to version 2008), the advice (gained from benchmarking), if less than 256K put in database, if greater than 1MB put in file system (with grey area in between). Ref: To BLOB or Not To BLOB:Large Object Storage in a Database or a Filesystem?
