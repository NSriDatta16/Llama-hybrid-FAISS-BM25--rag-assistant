Perform the check in the DB, using the exists? method. This ensures all the calculations are done at the DB rather than the client code. validates_presence_of :email, :unless => Proc.new { organizations.exists?(:contact_24 => false)} If you still insist on performing this at the client side then: validates_presence_of :email, :unless => Proc.new { organizations.any?{|o| o.contact_24 != true}}
