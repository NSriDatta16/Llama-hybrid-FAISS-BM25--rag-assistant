I think the boolean flag is quite an elegant solution, and you can keep it "contained" by using a property of the handler, like so: $(someElement).click(myHandler); function myHandler() { if (!myHandler.inProgress) { myHandler.inProgress = true; // Do stuff // Set it back to false later } }
