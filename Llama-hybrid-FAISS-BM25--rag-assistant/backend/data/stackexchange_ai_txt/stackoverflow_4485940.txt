It really depends on what you want to do with the information. If you plan to use the data for very detailed statistics, use separate columns so you can run detailed queries like "show all screen resolutions for users with Internet Explorer 6" and such. If you don't expect to need such detailed statistics, storing the user agent string in one field will usually work fine.
