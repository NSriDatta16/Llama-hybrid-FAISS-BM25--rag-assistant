Here's a small batch file we use to create our interop dll so it's strongly named: sn.exe -i MichiganLTAP.pfx MagicContainerName tlbimp.exe ourVb6.dll /out:Our.Strongnamed.Interop.dll /asmversion:7.1.0.0 /keycontainer:MagicContainerName /machine:X86 /namespace:Our.Strongnamed /verbose /sysarray sn.exe -d MagicContainerName The important switch from your perspective is: /asmversion:7.1.0.0 You should be able to set that to whatever you want. Check out the MSDN Library page for more info on the switches available.
