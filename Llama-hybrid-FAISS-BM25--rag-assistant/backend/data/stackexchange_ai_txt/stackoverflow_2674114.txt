Add the activeClass option to your .droppable() call, setting it to whatever class you want to be applied when dragging is active: $(".searchPage").droppable({ accept:".drag_check", activeClass: "ui-state-hover", hoverClass: "dropHover", // .. See the jQuery UI docs for a demo of this functionality: http://jqueryui.com/demos/droppable/#visual-feedback
