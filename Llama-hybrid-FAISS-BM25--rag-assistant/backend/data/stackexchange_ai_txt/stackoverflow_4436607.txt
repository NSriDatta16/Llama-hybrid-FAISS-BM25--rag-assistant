Whether or not someone is visiting your webpage on an iPad can be done like so via JavaScript: var userHasiPad = navigator.userAgent.match(/iPad/i) != null; PHP: $userHasiPad = (bool) strpos($_SERVER['HTTP_USER_AGENT'],'iPad'); Both of these are essentially checking the User Agent string for the existence of the string "iPad".
