You can retrieve the current draggable item through the draggable property of the ui argument. To complete your example: over: function(event, ui) { ui.draggable.addClass('red'); }, out: function(event, ui) { ui.draggable.removeClass('red'); } Or, if you want the ID: ui.draggable.attr("id");
