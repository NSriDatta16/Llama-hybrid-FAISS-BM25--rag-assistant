You need to create the Deck element first: XElement deck = new XElement("Deck"); Then add both the treasure (which i've taken the liberty of renaming from root to treasure) and the door to it: XElement treasure = new XElement("Treasure") ... deck.Add(treasure) ... XElement door = new XElement("Door") ... deck.Add(door)
