You're trying to modify a string literal - a string allocated in static storage. That's undefiend behaviour (usually crashes the program). You should allocate memory and copy the string literal there prior to reversing, for example: char *someString = "Hi there, I'm bad at this."; char* stringCopy = new char[strlen( someString ) + 1]; strcpy( stringCopy, someString ); strReverse( stringCopy ); delete[] stringCopy;//deallocate the copy when no longer needed
