you could use php's strip_tags($string, $tagsToRemove) $justText = strip_tags(' This is a test Another test '); And then you'd have the text, so you could use str_replace("new text", $justText); You might have to break it up using the second parameter of strip_tags() to keep the tags seperate, though. $html = ' This is a test Another test '; $anchorText = strip_tags($html, ' '); $paraText = strip_tags($html, ' '); $html = str_replace("new anchor text", $anchorText); $html = str_replace("new paragraph text", $paraText);
