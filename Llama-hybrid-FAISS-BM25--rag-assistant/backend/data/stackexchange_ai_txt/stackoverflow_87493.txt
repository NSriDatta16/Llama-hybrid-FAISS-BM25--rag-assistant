Some additional info, that might be of interest: Using the ip address, one can resolve the hostname, net provider / organization the IP belongs to, and rough geographic location. Using the referer, the list of queries a specified client makes, and a reliable cookie mechanism, one can resolve the path the visitor makes (even clickthroughs to other sides, with AJAX and/or a forwarder page) Using flash, with a combination of AJAX, the mouse location within the browsing window can be captured The User Agent might contain information regarding operation system, installed .NET frameworks, and other curiosities
