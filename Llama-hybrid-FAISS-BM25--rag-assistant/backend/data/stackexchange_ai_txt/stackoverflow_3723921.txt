To enable debugging through eclipse : I pass following to Tomcat startup: -Xnoagent -Xrunjdwp:transport=dt_socket,server=y,address=9999,suspend=n Then through Eclipse do Remote Debug. Goto Debug Menu > Debug Configuration Scroll Down to Select Remote Java Application rt click to create new configuration Select connection type as Standard (Socket Attach) and add hostname and port. To start debugging simple open it when server is running.---
