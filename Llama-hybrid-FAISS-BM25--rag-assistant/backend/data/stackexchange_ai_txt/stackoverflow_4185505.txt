From the PHP docs for HTTP_REFERER : The address of the page (if any) which referred the user agent to the current page. This is set by the user agent. Not all user agents will set this, and some provide the ability to modify HTTP_REFERER as a feature. In short, it cannot really be trusted. So keep in mind this parameter may not even be set by the browser...
