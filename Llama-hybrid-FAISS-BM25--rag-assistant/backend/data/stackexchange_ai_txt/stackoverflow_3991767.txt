Write the data as an HTML document and serve it up as a .XLS file rather than .HTML. All versions of Excel 2003 or higher will accept an HTML file, just as if it was .XLS binary or OpenOfficeXML. The users won't see the difference. Demo: Copy this into Notepad name age Albert 21 Brian 47 Charles 32 Save As "simple.XLS". Open that with Excel.
