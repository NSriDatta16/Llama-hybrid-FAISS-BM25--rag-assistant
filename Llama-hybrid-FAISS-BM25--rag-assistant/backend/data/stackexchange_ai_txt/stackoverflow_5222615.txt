If it's an SQL Server it's likely the internal storage format of the date that you are getting, which is the number of days since 1900-01-01. You can use the DateTime.AddDays method to convert it. Example: Dim n as Integer = 40683 Dim DepartDate As DateTime = New DateTime(1900, 1, 1).AddDays(n)
