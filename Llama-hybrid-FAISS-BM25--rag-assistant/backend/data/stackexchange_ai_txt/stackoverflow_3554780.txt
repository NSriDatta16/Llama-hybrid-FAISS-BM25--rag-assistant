You need to use the ondrop event, which will only fire if the ondragenter and ondragover events are cancelled. Turns out it's a bit trickier than that because the behavior is different in Firefox than IE, Safari and Chrome. (function () { var inp = document.getElementById("test"), chg = false; inp.ondragover = inp.ondragenter = function () { chg = inp.value == "Drop here"; return false; } inp.ondrop = function (evt) { evt = evt || event; if (chg) { this.value = evt.dataTransfer.getData("text") || evt.dataTransfer.getData("text/plain"); return false; } } })(); Example - Firefox 3+, IE5+, Chrome and Safari. Near as I can tell, Opera doesn't support the event. At least you can get it working for 95% of your visitors though. Drag Operations - MDC
