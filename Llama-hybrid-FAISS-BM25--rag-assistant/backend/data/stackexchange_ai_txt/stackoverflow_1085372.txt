It's a simple question i believe. You'll need to implement "virtual assignment operator" which will require upcast imho: class CBase { ... virtual assign(const CBase & from) = 0; ... }; class CSub { ... void operator=(const CSub & from) { assign(from); } virtual assign(const CBase & from) { if (dynamic_cast (from)) { ... implement simple copy here } else { ... you'll have to decide what to do with other types } } };
