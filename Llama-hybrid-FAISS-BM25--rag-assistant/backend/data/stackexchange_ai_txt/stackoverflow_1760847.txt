The GetList method in the Lists.asmx web-service will return a field called DefaultViewUrl. It'll look like this : DefaultViewUrl="/Site_Name/Lists/List_Name/AllItems.aspx" The following code would then give you the static name : String pattern = ".*/(? .+)/[^\\.]+\\.aspx"; Regex regex = new Regex(pattern); MatchCollection matches = regex.Matches(DefaultViewUrl); String listStaticName = matches[0].Groups["listStaticName"].ToString(); Far from an elegant solution, but it will work. EDIT : Actually, the SPList.RootFolder.Name should give you the same result for a document library.
