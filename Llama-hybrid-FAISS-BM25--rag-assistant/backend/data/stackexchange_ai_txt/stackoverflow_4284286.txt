If you need only the columns from the House table : SELECT * from HOUSE where ID in (SELECT ID from FEATURES where FEATURE IN('GARDEN','GARAGE')); If you need the entire details: SELECT * FROM HOUSE H JOIN FEATURES F ON H.HOUSE_ID = F.HOUSE_ID WHERE FEATURE IN ('GARDEN','GARAGE');
