You could use a modifier key, for instance normal behavior would be horizontal & press down the shift key to move vertically. function mouseMoveEventHandler(evt:MouseEvent):void { if(!evt.shiftKey) dragSprite.x = evt.stageX; else dragSprite.y = evt.stageY; }
