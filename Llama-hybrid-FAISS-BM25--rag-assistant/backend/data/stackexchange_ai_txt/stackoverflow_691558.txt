TeamCity supports svn checkout on agent. It is able to call svn update automatically in build checkout directory. Please select checkout on agent under build configuration settings/2.Version Control. Please have a look to documentation page at http://www.jetbrains.net/confluence/display/TCD4/Build+Checkout+Directory
