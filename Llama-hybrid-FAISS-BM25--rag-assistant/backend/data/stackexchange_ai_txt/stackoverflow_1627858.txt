A short jQuery Plugin does it for me IE6TableHover: function() { if(jQuery.browser.msie && /6.0/.test(navigator.userAgent)) { $(this).hover( function(){ $(this).addClass('jshover'); }, function(){ $(this).removeClass('jshover'); }); } } required jQuery Version here 1.2.6 for 1.3.x you have to change the browser check.
