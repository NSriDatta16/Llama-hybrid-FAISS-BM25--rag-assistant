Pesto's 99% of the way there, however Nokogiri supports creating a document fragment that doesn't wrap the text in the declaration: text = Nokogiri::HTML::DocumentFragment.parse(' this is a text string').children.select {|n| n.text? and n.content } puts text # >> this # >> is # >> a # >> text # >> string
