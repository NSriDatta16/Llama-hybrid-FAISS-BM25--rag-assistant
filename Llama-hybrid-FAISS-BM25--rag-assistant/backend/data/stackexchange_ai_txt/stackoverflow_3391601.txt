My choice is Project ------- ProjectID (PK) Date ProjectLoc ---------- ProjectID (FK) Lang Name ShortDesc FullDesc Then you can run a simple query like SELECT Project.ProjectID, Date, Name, ShortDesc, FullDesc FROM Project LEFT JOIN ProjectLoc ON Project.ProjectID = ProjectLoc.ProjectID WHERE ProjectLoc.Lang = %CurrentLang% Pros: Elegant and simple Cons: Large number of tables
