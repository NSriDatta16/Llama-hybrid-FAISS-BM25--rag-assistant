You could use list and then use classes to format them in CSS, even though that would also not be very semantic: Level 1 Level 2 .level1 { font-size: 1.2em; } .level2 { font-size: 0.8em; } Paragraph tags may be more semantic: Level 1 Level 2 .level1 { font-size: 1.2em; } .level2 { font-size: 0.8em; }
