You may be interrested by MVP (Model View Presenter), and rich application frameworks that rely on it, like CAB (Composite UI Application Block, from MS Pattern & Practice team) Using MVP, you can have a presentation model and leverage winform databinding. Jeremy D. Miller blog is a good source of information regarding this design, and all other UI patterns.
