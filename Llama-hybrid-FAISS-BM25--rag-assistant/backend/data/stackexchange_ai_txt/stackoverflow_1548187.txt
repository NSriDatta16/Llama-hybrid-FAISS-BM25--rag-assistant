Ok, Here is the route I have gone down... When the page is loaded, I add a container div around the draggable list. I set it's height to twice that of the list, then position it up the page a little: $(document).ready(function() { CreateContainerDiv(); $('.dragPanel').addClass("hover"); $('.myList').draggable({ axis: 'y', containment: 'parent' }); }); function CreateContainerDiv() { var dragPanel = $('.dragPanel'); var dragTarget = $('.myList'); // add the container panel var newPanelHeight = dragTarget.outerHeight() * 2 - dragPanel.outerHeight(); dragTarget.wrap(document.createElement("div")); // set its height and put it in the right place dragTarget.parent().css("height", newPanelHeight); dragTarget.parent().css("position", "relative"); var newPanelPosition = ((dragTarget.outerHeight() * -1) / 2); dragTarget.parent().css("top", newPanelPosition); } The list is then contained to this new element. This appears to do the job, but the positioning is all a little shakey if the list has any padding / margin applied. Any thoughts on that would be appreicated! ------Edit --------- Ok, I think I have solved the placement issue.
