There is. See the MSDN page on how to suppress compiler warnings. From Visual Studio, go to your project properties, select the build tab, and enter the warning number in the Suppress Warnings field. From code, to disable specific warnings, you can use the #pragma directive: public class MyClass { #pragma warning disable 0168 // code // optionally, restore warnings again #pragma warning restore 0168 // more code }
