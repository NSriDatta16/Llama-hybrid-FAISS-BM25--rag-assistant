We can use Isolated Storage as a virtual file system to store data in a hidden folder on our machine in silverlight application. Silverlight application is allocated the storage of 1 MB per application but we can increase the amount of storage. This is the function to get the data in the isolated storage Private Function LoadData(ByVal fileName As String) As String Dim data As String = String.Empty Dim isfstream As New IsolatedStorageFileStream(fileName, FileMode.Open, IsolateStorageFileObj) Dim sr As New StreamReader(isfstream) data = sr.ReadLine() Return data End Function
