Personally, I'd use Javascript for this one. if(navigator.userAgent.match(/Chrome/i)) { alert("You're using Chrome!"); } else { alert("You're using something other than Chrome!"); } ... but if you really wanted to, you could accomplish the same thing in PHP: if (preg_match("/Chrome/i", $_SERVER['HTTP_USER_AGENT']) == 0) { // zero matches echo " alert('You're not using Chrome!') "; } else { echo " alert('You're using Chrome!') "; }
