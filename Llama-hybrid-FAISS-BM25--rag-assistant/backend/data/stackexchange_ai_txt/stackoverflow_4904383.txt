I think it's not elegant to first generate a sequence of numbers only to throw it away. These methods deliver random numbers on the spot (you have to pay with verbosity though): IEnumerable GetRandomSequence(int maxNumber) { var random = new Random(); while (true) yield return random.Next(maxNumber); } IEnumerable GetRandomSequence(int maxNumber, int maxCount) { return GetRandomSequence(maxNumber).Take(maxCount); } var random = new Random(); var list = GetRandomSequence(100, 10).ToList (); And I love how I can write the same in F#: let random = new Random (); let list = Seq.init (random.Next 10) (fun _ -> random.Next 100) |> List.ofSeq;
