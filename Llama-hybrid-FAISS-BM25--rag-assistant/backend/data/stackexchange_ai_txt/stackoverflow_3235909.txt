You might want to have a look at the jQuery Calculation Plug-in . The syntax for your example would be something like this: var MinVal=$("input[name^='min_field_']").min(); // Get minimum value var MaxVal=$("input[name^='min_field_']").max(); // Get maximum value var AvgVal=$("input[name^='min_field_']").avg(); // Get average value This would be faster version: var MinField=$("input[name^='min_field_']"); var MinVal=MinField.min(); // Get minimum value var MaxVal=MinField.max(); // Get maximum value var AvgVal=MinField.avg(); // Get average value
