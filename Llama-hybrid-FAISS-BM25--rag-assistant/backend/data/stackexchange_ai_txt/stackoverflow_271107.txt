The C++ specification itself (old version but good enough for this) leaves this open. There are four signed integer types: ' signed char ', ' short int ', ' int ', and ' long int '. In this list, each type provides at least as much storage as those preceding it in the list. Plain ints have the natural size suggested by the architecture of the execution environment* ; [Footnote: that is, large enough to contain any value in the range of INT_MIN and INT_MAX, as defined in the header . --- end foonote]
