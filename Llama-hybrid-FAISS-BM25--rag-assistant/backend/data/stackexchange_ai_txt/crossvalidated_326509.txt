Forward and backward selection are not recommended. See the discussions here. What are disadvantages of using the lasso for variable selection for regression? Using PCA to do feature selection is also not recommended. Since, it only check for the variance of the independent variable, but not the "correlation" with response variable. See discussion here. How to decide between PCA and logistic regression? If you are focusing on getting an accurate model, you can use regularized logistic regression. An example can be found here.
