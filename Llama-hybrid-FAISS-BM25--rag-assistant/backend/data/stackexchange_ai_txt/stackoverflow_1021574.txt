No, to my knowledge, there are no events that'll fire as soon as you make the drop. Here's a possible solution: var input = document.getElementById("input"); var lastVal = input.value; setInterval(function() { if (input.value !== lastVal) { if (input.value) { if ( /\.(jpe?g|gif|bmp|png)(\?.*)?$/.test(input.value) ){ alert('It\'s an image URL!!!'); } else { alert('Not an image URL...'); } } lastVal = input.value; } }, 100); Demo here: http://jsbin.com/izake Note: It appears that some browsers (IE) don't let you drop items into fields like other browsers.
