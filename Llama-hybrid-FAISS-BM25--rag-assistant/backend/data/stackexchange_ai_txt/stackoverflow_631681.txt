You don't actually specify the lines that gives the error but I imagine they might look a bit like struct genericarray ga; float fValue; fValue = ga.storage[idx]; What you could do is add a cast fValue = ((float*)ga.storage)[idx]; I'd recommend creating some macros or functions for setting and getting values #define GET_STORAGE(_type, _src, _idx) \ ((_type*)_src.storage)[_idx] #define SET_STORAGE(_type, dst, _idx, _src) \ ((_type*)_dst.storage)[_idx] = _src fValue = GET_STORAGE(float, ga, 3); SET_STORAGE( float, ga, 3, sin(ang) ); I see 'Chris Young' using switch statements in his answers and that is a good thing for is these access functions.
