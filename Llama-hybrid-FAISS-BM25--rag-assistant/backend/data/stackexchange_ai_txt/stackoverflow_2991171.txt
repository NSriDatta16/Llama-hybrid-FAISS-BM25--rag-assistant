Default values are defined by each browser individually. In many cases they coincide, but sometimes they don't. This is why it is recommended to use "CSS reset" stylesheets to default styles to the same values. You can find some (unofficial) reference to the default browser styles here: CSS2.1 User Agent Style Sheet Defaults Official Mozilla default style sheet
