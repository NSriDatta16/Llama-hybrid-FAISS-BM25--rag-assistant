This is the script I use and it seems to work well enough: // Returns 0 if the browser is anything but IE function getIEVersion() { var ua = window.navigator.userAgent; var ie = ua.indexOf("MSIE "); return ((ie > 0) ? parseInt(ua.substring(ie+5, ua.indexOf(".", ie))) : 0); } Hope that helps someone...
