To append parameters to the query string (instead of replacing them) use the QSA flag: RewriteRule ^folder1/(.*) $1&organisation=folder1 [NC,QSA] "This flag forces the rewrite engine to append a query string part of the substitution string to the existing string, instead of replacing it. Use this when you want to add more data to the query string via a rewrite rule." See Apache documentation for mod_rewrite .
