i think you need to specify the ajax method you are using.It might be a $.ajax, $.get or $.getJson. but i use $.ajax and here is a snippet $.ajax({ url:"event/service_ajax_handler.php", type: "GET", data: {action:"getTime"}, dataType : "json", success: function(data) { $("#cmbTimeRange").html(" Please select time range "); $.each(data, function(){ $("#cmbTimeRange").append(" " + this.hours +" ") }); }, error: function(){ alert("error"); } }); pay attention to the data parameter. see also getJSON
