First off, as a general rule, you want to stay away from embedding JavaScript in your HTML as tag attributes. Secondly, Simile is likely overwriting the onload (using it the same way you are). So, you'll want to add your onload event after you've included Simile. Using jQuery: $(document).ready(function() { initComments('myID'); }); Using no library (found here ): function addOnloadEvent(fnc){ if ( window.addEventListener ) { window.addEventListener( "load", fnc, false ); } else if ( window.attachEvent ) { window.attachEvent( "onload", fnc ); } else { var oldOnload = window.onload || function() {}; window.onload = function(e) { oldOnload(e); window[fnc](e); }; } } addOnloadEvent(function() { initComments('myID'); });
