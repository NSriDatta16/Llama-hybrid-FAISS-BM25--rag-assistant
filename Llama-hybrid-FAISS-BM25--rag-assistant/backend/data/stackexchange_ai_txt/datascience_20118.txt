Advantages of pandas dataframe to regular relational database

In Data Science, many seem to be using pandas dataframes as the datastore. What are the features of pandas that make it a superior datastore compared to regular relational databases like MySQL , which are used to store data in many other fields of programming? While pandas does provide some useful functions for data exploration, you can't use SQL and you lose features like query optimization or access restriction.
