Taking Oskar's solution and changing it slightly: private void DealCards(Func > handProjection) { for (int i = 0; i cards = handProjection(player.Hand); if (cards.Count 0) { cards.Add(Deck.TakeTopCard()); } } } } } public void DealStartingCards() { DealCards(hand => hand.FaceDownCards); DealCards(hand => hand.FaceUpCards); DealCards(hand => hand.InHandCards); } (That's assuming the type of player.Hand is Hand - adjust accordingly otherwise, of course.)
