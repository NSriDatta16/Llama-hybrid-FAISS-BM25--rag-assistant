In-browser applets support JNLP deployment since 1.5 (or so). There's an example on how to do it here: http://java.sun.com/javase/6/docs/technotes/guides/jweb/deployment_advice.html You're still using the Java plugin then, of course. It may or may not work to set the heap size.
