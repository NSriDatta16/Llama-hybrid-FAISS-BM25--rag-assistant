You can schedule a SQL Agent job to download the file locally and use BULK INSERT : CREATE TABLE StagingCSV ( col1 VARCHAR(60), col2 VARCHAR(60), col3 VARCHAR(60), col4 VARCHAR(60), -- ... ) GO (error rows will be ignored) BULK INSERT StagingCSV FROM 'c:\mycsvfile.txt' WITH ( FIELDTERMINATOR = ',', ROWTERMINATOR = '\n' ) GO Other methods: About Bulk Import and Bulk Export Operations Importing Bulk Data by Using BULK INSERT or OPENROWSET You can use Powershell to download a file: $clnt = new-object System.Net.WebClient $url = "http://www.supplier.com/products.csv " $file = "c:\temp\Mycsv.txt" $clnt.DownloadFile($url, $file)
