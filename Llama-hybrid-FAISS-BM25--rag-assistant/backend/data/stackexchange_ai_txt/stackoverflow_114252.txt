Data for all users should be stored in %ALLUSERSPROFILE%, or call SHGetFolderPath() with the parameter CSIDL_COMMON_APPDATA to get the all users storage area. See http://www.deez.info/sengelha/2006/02/28/windows-vista-changes/ for more details.
