To enable chaching add this: ExpiresDefault "access plus 1 month" To prevent caching do this: According to http://www.askapache.com/htaccess/using-http-headers-with-htaccess.html#prevent-caching-with-htaccess , add this FileETag None Header unset ETag Header set Cache-Control "max-age=0, no-cache, no-store, must-revalidate" Header set Pragma "no-cache" Header set Expires "Wed, 11 Jan 1984 05:00:00 GMT"
