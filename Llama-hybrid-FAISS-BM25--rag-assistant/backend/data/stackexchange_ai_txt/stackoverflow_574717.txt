Using http://www.mathias-bank.de/2007/04/21/jquery-plugin-geturlparam-version-2 : $(document).ready(function(){ var param = $(document).getUrlParam('selectedTab'); $('#menu').tabs('select', param); }); From documentation : #select Signature: .tabs( 'select' , [index] ) Select a tab, as if it were clicked. The second argument is the zero-based index of the tab to be selected or the id selector of the panel the tab is associated with (the tab's href fragment identifier, e.g. hash, points to the panel's id).
