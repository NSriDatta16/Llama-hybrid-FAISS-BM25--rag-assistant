If you know the storeId, set in setStoreId : /** * call the Magento catalog/product model * set the current store ID * load the product */ $product = Mage::getModel('catalog/product') ->setStoreId($storeId) ->load($key); Display in a block : echo $product->getName(); We can also use print_r to see the values : print_r($product->getData()); The following code will show current store ID : $storeId = Mage::app()->getStore()->getId(); To get all product ID's with each store view : $product = Mage::getModel('catalog/product'); $products = $product->getCollection()->addStoreFilter($storeId)->getData(); If you change the $storeId will show different product.
