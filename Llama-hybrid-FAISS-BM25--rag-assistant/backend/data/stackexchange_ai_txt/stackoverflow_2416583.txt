In C# you can make an indexer, but not a named indexer: public T this[string name] { get { return (T)HttpContext.Current.Session[name]; } set { HttpContext.Current.Session[name] = Value; } } You can make a collection that uses the Session state as storage, and declare a property that returns the collection: public class DatosCollection { public T this[string name] { get { return (T)HttpContext.Current.Session[name]; } set { HttpContext.Current.Session[name] = value; } } } private _collection = new DatosCollection(); public DatosCollection Datos { get { return _collection; } }
