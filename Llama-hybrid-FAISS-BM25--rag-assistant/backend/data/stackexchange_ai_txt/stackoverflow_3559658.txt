Your drag does not set a drag cue. SImplest fix is to add this code in the middle of your OnDragQuery. You will then get a nice matching rectangle dragging around on the cursor: source.Visibility = Visibility.Collapsed; /* add these 5 lines - copies properties into a drag cue */ var cue = new Rectangle(); cue.Height = source.Height; cue.Width = source.Width; cue.Fill = source.Fill; e.Options.DragCue = cue; e.Options.Payload = source; I turned on the grid so you could see the effect clearly in this screen-shot as the 2x2 rectangle is dragged:
