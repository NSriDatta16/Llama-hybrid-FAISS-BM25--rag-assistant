If a src attribute is provided, user agents are required to ignore the content of the element , if you need to access it from the external script, then you are probably doing something wrong. Update: I see you've added a comment to the effect that you want to cache the script and use it later. To what end? Assuming your HTTP is cache friendly , then your caching needs are likely taken care of by the browser already.
