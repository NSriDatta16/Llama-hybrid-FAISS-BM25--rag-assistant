What would be the exact function for a ridge logistic regression with multiple variables?

I am looking for the correct equation for a ridge logistic regression for multiple variables. I thought it simply was: $$y=\frac1{1+e^{-(\beta_0+\beta_1X_1+\beta_2X_2+\cdots+\beta_nX_n)}}$$ with an additional penalty parameter added, in this case $$\sum_i^n (y_i-\widehat{y_i})^2 + \lambda \sum_j^p \beta_j^2$$ Eventually $$y=\frac1{1+e^{-(\beta_0+\beta_1X_1+\beta_2X_2+\cdots+\beta_nX_n)}}+\sum_i^n (y_i-\widehat{y_i})^2 + \lambda \sum_j^p \beta_j^2$$ Is this true for a ridge regression?
