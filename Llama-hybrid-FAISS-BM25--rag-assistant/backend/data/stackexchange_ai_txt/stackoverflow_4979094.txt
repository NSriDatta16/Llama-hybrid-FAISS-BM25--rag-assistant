If you have only the Express Edition, then you have a problem. Best approach if you don't have money for tools would be to download another IDE with a designer for e.g. wxWindows and configure it to run the MS compiler. With VS2010 Professional or higher, there's a native designer, called the dialog editor, which provides drag-and-drop layout much like the WinForms designer does.
