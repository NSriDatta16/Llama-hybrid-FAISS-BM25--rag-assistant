How to mark POSTing URLs?

Search engines and pre-fetching browser plugins can cause quite some trouble with elements where the destination page changes the state of the server. In a , I'd mark it as modifying with method="POST" . Is there a similar way to mark regular links as modifying? rel="nofollow" does not solve the problem. From the specification : By adding rel="nofollow" to a hyperlink, a page indicates that the destination of that hyperlink should not be afforded any additional weight or ranking by user agents which perform link analysis upon web pages (e.g. search engines)
