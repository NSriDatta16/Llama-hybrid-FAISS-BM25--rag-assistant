You'll want to have CategoryAttribute set to "File" for BOTH properties: [Category("File")] public string FilePath { get; set;} [Category("File")] public int FileSize { get; set;} I recommend reading " Getting the most out of the .NET Property Grid Control " for other ideas you can use for organizing your properties, including adding descriptions.
