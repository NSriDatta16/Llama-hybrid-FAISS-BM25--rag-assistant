Note that fragment rules only "live" inside the lexer and cease to exist in the parser. For example: grammar Measurement; options { output=AST; } parse : (m=MEASUREMENT { String contents = $m.text; boolean isMeasurementType = $m.getType() == MeasurementParser.MEASUREMENT; System.out.println("contents="+contents+", isMeasurementType="+isMeasurementType); })+ EOF ; MEASUREMENT : MILLIMETRE ; fragment MILLIMETRE : 'millimetre' | 'millimetres' | 'millimeter' | 'millimeters' | 'mm' ; SPACE : (' ' | '\t' | '\r' | '\n'){$channel=HIDDEN;} ; with input text: "millimeters mm" will print: contents=millimeters, isMeasurementType=true contents=mm, isMeasurementType=true in other words: the type MILLIMETRE does not exist, they're all of type MEASUREMENT .
