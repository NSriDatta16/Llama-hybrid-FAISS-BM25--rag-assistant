Write appropriate subclasses of javax.swing.TransferHandler for both trees. Use TransferHandler.MOVE as the source action. Override exportDone() and see if the action includes the MOVE flag. If so, delete the object represented by the Transferable . Also see this question: Adding drag and drop support to Jtree
