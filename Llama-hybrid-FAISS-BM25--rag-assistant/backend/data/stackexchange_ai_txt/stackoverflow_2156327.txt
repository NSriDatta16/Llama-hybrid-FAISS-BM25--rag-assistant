There are some free services out there that let you make country and ip-based geolocalization from the client-side. I've used the wipmania free JSONP service, it's really simple to use: // plain JavaScript example function jsonpCallback(data) { alert('Latitude: ' + data.latitude + '\nLongitude: ' + data.longitude + '\nCountry: ' + data.address.country); } Or if you use a framework that supports JSONP, like jQuery you can: // jQuery example $.getJSON('http://api.wipmania.com/jsonp?callback=?', function (data) { alert('Latitude: ' + data.latitude + '\nLongitude: ' + data.longitude + '\nCountry: ' + data.address.country); }); Check the above snippet running here .
