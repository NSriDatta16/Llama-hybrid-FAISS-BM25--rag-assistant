Not explicitly, no. However, you can set up a TRIGGER on the INSERT for the database to delete any rows older than thirty days (so you only delete rows as you add more, not just for the sake of it). Or, you can use SQL Server Agent and set up a Stored Procedure to do a delete from table where insertdate
