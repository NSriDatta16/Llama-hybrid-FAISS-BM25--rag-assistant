One of the first things you should do when compiling for the first time is to try one piece of code at a time. Don't throw a bunch of code together and hope that it compiles. Instead, take it one fragment at a time. So here, you might comment out your includes and the code that you expect to use the stuff in those files. At first glance, it looks like void readImage(vector &filenames) is missing a semicolon at the end of the line, since you're declaring it.
