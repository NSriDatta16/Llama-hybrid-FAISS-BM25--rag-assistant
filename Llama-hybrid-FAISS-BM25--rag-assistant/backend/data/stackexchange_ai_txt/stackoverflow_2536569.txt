If this code is part of your project then the control should automatically appear in the toolbox after you compiled the code. But it is an option that might have been turned off. Tools + Options, Windows Forms Designer, General, Toolbox, AutoToolboxPopulate must be True. If it is a separate assembly then Drag+Drop won't work. Right click the toolbox, Choose Items, use the Browse tab.
