Well, deleting the last paragraph can be done like: $(document).ready(function () { $('p:last').remove(); }); If an RSS newsitem has its own class, like class="RssRow" , you could use each() to remove the last child for each row: $('.RssRow').each(function() { $(this).find('p:last').remove(); });
