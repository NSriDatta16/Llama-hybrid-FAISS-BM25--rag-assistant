Depends on your language and it's properties. For example, if you can only expose C-style functions, then the default WinAPI supports this style. If you can do the whole C++ classes malarky, then you could do MFC, GDI+, WIC, etc. Ultimately, if you want something easy to embed, it's simpler to write good embedding code than worry about what you're going to embed.
