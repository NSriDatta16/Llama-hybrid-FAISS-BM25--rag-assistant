The User Agent string for a 64-bit IE browser will indicate that it's 'x64' or 'Win64' if it's a 64-bit browser. http://blogs.msdn.com/b/ie/archive/2009/01/09/the-internet-explorer-8-user-agent-string-updated-edition.aspx A 64-bit IE8 user agent string: Mozilla/4.0 (compatible; MSIE 8.0; Windows NT 6.0; Win64; x64; Trident/4.0) vs. a 32-bit Mozilla/4.0 (compatible; MSIE 8.0; Windows NT 6.0; WOW64; Trident/4.0)
