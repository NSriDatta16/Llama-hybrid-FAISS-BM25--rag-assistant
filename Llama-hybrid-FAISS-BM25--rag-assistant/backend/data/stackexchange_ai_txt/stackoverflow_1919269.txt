Java has PrintFormat class which does very similar thing as sprintf in C, you might want to leverage it to format your query. Following code snippet is taken from http://java.sun.com/developer/technicalArticles/Programming/sprintf/ System.out.println( new PrintfFormat("Pre %s Post%%").sprintf("target") );
