The reason your code doesn't work is because there is JavaScript on the page that is actually writing out the tag, so if you're requesting the page from a User Agent (or via AJAX) that doesn't execute JavaScript then you won't find the element. I was able to get my own name using the following selector: string name = doc.DocumentNode.SelectSingleNode("//a[@id='navAccountName']").InnerText;
