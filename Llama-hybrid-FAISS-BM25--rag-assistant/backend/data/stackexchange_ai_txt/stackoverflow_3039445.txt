If you'd like to use JavaScript to grab the letter from the URL query string, run a regular expression on window.location.search : var letterParam = window.location.search.match(/letter=([a-z])/i), letter; if (letterParam) { letter = letterParam[1]; } To match paragraphs starting with that letter, use the charAt() method in JavaScript strings: if (letter) { $('p').each(function() { if ($(this).text().charAt(0).toUpperCase() == 'B') { // Apply the CSS class or change the style... } }); }
