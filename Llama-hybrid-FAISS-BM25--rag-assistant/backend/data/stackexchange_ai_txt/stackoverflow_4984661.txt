Assuming that Article is defined something like: case class Article(title: String, content: String) and articleList as: val articleList = List( Article("title a","content a"), Article("title b", "content b")) You could produce the XML fragment with: val xml: scala.xml.NodeSeq = articleList map { article => {article.title} {article.content} }
