Well, if you do decide to run Magento with PHP 5.3 anyway, here's how to do it (tested with version 1.3.2.3): 1: in index.php, around line 35, replace error_reporting(E_ALL | E_STRICT); with error_reporting((E_ALL | E_STRICT) & !E_DEPRECATED) (reason: Magento often uses the split function, which is deprecated in PHP 5.3) 2: in /lib/Varien/Object.php, around line 484 change public function ___toString(array $arrAttributes = array(), $valueSeparator=',') to public function __invoke(array $arrAttributes = array(), $valueSeparator=',') (reason: I don't really know, but it works!)
