Try this: string name = GetPropertyName(() => default(SomeClass).SomeInstanceProperty); You may get a compiler warning about "always causing a System.NullReferenceException", but that's not actually happening, since you don't execute the expression, which means that you can safely discard this warning. If you want to get rid of it, either disable it via pragma or just move the default() call into a function like this: public static T Dummy () { return default(T); } string name = GetPropertyName(() => Dummy ().SomeInstanceProperty);
