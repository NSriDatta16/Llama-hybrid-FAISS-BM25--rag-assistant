If you want to check for the existence of title before output, you could use: // generates the title string from all found titles $str = ''; foreach ($array AS $k => $v) { if (isset($v->title)) { $str .= $v->title; } } echo $str; If you wanted these in an array it's just a quick switch of storage methods: // generates the title string from all found titles $arr = array(); foreach ($array AS $k => $v) { if (isset($v->title)) { $arr[] = $v->title; } } echo implode(', ', $arr); stdClass requires you to use the pointer notation -> for referencing whereas arrays require you to reference them by index, i.e. [4] . You can reference these like: $array[0] $array[1]->aid $array[1]->atitle $array[2]->aid $array[2]->atitle // etc, etc.
