Alloc an NSMutableArray , then use [NSMutableArray addObject:dictionary to add the dictionaries. At what point in time you write the Array with the dictionaries to persistent storage (file) is your choice, but you don't need to write an empty file first. Just ensure you update the file before entering background and/or before exiting the App
