Because tool A and B might result in recommendations with different numbers of items, using ratio is more suitable than the actual score. calculate hit ratio for each one eg. hit ratio = 1 * ( Purchase hit count / total recommended count ) + 0.8 * ( Wishlist hit count / total recommended count ) use an average of hit ratio to evaluate recommendation performance
