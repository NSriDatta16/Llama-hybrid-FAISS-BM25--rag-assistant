Default value of zIndex parameter of info_dialog is 1000. The function info_dialog from grid.common.js part of jqGrid will be called from grid.celledit.js without usage a 4-th parameter which can change the option. So the best pragmatical way which I could recomend you is to decrease zIndex value of your div with the video so that it will be less then 1000.
