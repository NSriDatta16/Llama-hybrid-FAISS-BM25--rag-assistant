Here's a simple example in R using the bfi dataset: bfi is a dataset of 25 personality test items organised around 5 factors. library(psych) data(bfi) x A hiearchical cluster analysis using the euclidan distance between variables based on the absolute correlation between variables can be obtained like so: plot(hclust(dist(abs(cor(na.omit(x)))))) The dendrogram shows how items generally cluster with other items according to theorised groupings (e.g., N (Neuroticism) items group together). It also shows how some items within clusters are more similar (e.g., C5 and C1 might be more similar than C5 with C3). It also suggests that the N cluster is less similar to other clusters.
