The problem seems to be related to the Code Coverage you activated. Disabling code coverage solves the issue. You could put the following in your AssemblyInfo.cs: [assembly: SecurityRules(SecurityRuleSet.Level1, SkipVerificationInFullTrust = true)] and reactivate code coverage.
