Relatively simple solution with Boost.Lambda: #include using namespace boost::lambda; template void dual(const T& functor, U& a1, V& a2) { functor(a1); functor(a2); } struct Foo { int a; }; struct Bar { char a; }; int main() { Foo a1; Bar a2; dual(_1 = 5, a1.a, a2.a); } Extending dual() with variadic templates / Boost.Preprocessor shenanigans is left as an exercise to the reader.
