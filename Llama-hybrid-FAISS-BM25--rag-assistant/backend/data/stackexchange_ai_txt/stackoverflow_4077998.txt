The GlobalAlloc function just allocates and returns a block of memory and the GPTR option zeroes the bytes of the allocated memory so you can just use: if (strlen(hwndTitleValve) == 0) assuming an ANSI string. Note that this would be better tagged as "C" and "Windows" rather than C++.
