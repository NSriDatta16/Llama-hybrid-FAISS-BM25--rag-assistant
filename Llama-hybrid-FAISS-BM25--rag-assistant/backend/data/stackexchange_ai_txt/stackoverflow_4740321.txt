You can either inline it in a case statement select COUNT(id), case status when 1 then 'initial order' when 2 then 'cancelled' # without an else, the rest go to NULL end status from tbl group by status # yes, just on status Or I would strongly encourage you to create a reference table for this Tbl Status contains 2 columns ID and Description select COUNT(tbl.id), status.description from tbl LEFT join status on status.id = tbl.status group by status.description
