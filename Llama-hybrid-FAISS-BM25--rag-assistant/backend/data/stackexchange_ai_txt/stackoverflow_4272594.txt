You can pass arguments to your java virtual machine that gives you GC diagnostics such as -verbose:gc This flag turns on the logging of GC information. Available in all JVMs. - XX:+PrintGCTimeStamps Prints the times at which the GCs happen relative to the start of the application. If you capture that output in a file, in your application you can periodcly read that file and parse it to know when the GC has happened. So you can work out the average time between every GC
