You need quotes on attributes, like this: $('# ') .append(' ' + exampleName + ' '); Also, depending on the browser, it will show the original page's source when you do "View Source" As an aside: there is a faster approach if adding many elements, since jQuery will cache the fragment: $('# ') .append($(" ").attr("value",exampleValue).text(exampleName));
