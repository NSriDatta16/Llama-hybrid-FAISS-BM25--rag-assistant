Here'e one possibility: TimeSpan.Parse(s.Remove(2, 1).Remove(5, 1)); And if you want to make it more elegant in your code, use an extension method: public static TimeSpan ToTimeSpan(this string s) { TimeSpan t = TimeSpan.Parse(s.Remove(2, 1).Remove(5, 1)); return t; } Then you can do "05h:30m".ToTimeSpan();
