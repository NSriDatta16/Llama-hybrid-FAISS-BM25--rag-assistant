Yes, that's a problem when you mixing sortables draggables droppables etc. Make sortable container as droppable as well: $("#sortable").sortable({ revert: true, accept: '.control', receive: function(event, ui) { // here i need to get draggable element id } }); $('#sortable').droppable({ accept: '.control', drop: function(e, ui){ var your_draggable_element = $(ui.draggable); var your_helper = $(ui.helper); } }); that should work
