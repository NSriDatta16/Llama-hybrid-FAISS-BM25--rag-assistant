This is classic C++ behavior. Trying to learn C++/CLI without first learning basics of standard C++ is going to be very difficult. The general pattern to make this work is: Forward declare types Define types Define type member functions in that order. For example: ref class ResizeLabel; ref class ResizePanel; public ref class ResizeLabel : public System::Windows::Forms::Label { protected: virtual void OnTextChanged(System::EventArgs^ e) override; ... }; public ref class ResizePanel : public System::Windows::Forms::Panel { ... }; void ResizeLabel::OnTextChanged(System::EventArgs^ e) { __super::OnTextChanged(e); // Not elegant I know, // but this is just to force the panel to process the size change dynamic_cast (this->Parent)->CurrentWidth = 0; }
