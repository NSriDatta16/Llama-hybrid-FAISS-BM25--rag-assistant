div is block element and span is inline element. There are two things that deserve attention. Visible behaviour - block formating concepts VS Inline formatting context (X)HTML validity relations - basicaly you never should sink block element inside inline, so if you have to create some block behaving elements structure inside a link element (which is inline) use spans and set their display:block via CSS
