This control is intended to be used exactly as the original ASP.NET UpdatePanel, not as A usercontrol. Paste the sourcecode into a *.cs file. Add the file to your current ASP.NET project. Change the namespace to your own namespace to aviod tinkering with the web.config. Then it should popup in the toolbar in vs2005 drag that into the page as you would any other control.
