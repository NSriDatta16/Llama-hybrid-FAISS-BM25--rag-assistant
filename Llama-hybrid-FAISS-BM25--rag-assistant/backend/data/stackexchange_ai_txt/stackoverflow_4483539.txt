drag'n drop of views in a linear layout works... provided the layout is not empty and its heigth/width are not set to wrap_content !!! example : create a linearLayout. Set the layout width / height to fill parent or some fixed value (e.g. 64dip). You can now drag'n drop, let's say, a textView in it.
