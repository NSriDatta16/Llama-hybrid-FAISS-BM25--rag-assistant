This is a shortcut the we have used before for forms that have a lot of process to do on create. Drop a TTimer on the form and set it to false. OnCreate of the form enable it. Then put all the code you had in OnCreate into the OnTimer Event. Setting the Interval to 250 to 500 is enough. This is not an elegant solutions, but its simple.
