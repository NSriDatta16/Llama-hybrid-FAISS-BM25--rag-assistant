The STAN bundle and project dependency views are free add-ons. Here's a sample dependency graph: http://stan4j.com/images/stories/misc/plugin-deps.png Quick feature list: horizontal, vertical, narrow layout options node markers +/- indicate outgoing/incoming dependencies currently not shown double-clicking a node adds outgoing dependencies double-clicking a node while holding the control key adds incoming dependencies not shown edge labels indicate "requires-bundle" and/or "imports-packages" supports drag'n drop from the project/package explorer Visit http://stan4j.com for further information and download/install instructions.
