You are at liberty to create an extension method on IEnumerable that is type safe: public static void Each(this IEnumerable @this, Action action) { if (@this == null) throw new ArgumentNullException("this"); if (action == null) throw new ArgumentNullException("action"); foreach (TElement element in @this) { action(element); } } It is subtly different to the foreach loop and will occur additional overhead, but will ensure type safety.
