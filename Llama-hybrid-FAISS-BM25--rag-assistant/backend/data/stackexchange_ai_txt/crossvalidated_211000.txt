Inverse gamma posterior

I am working on bayesian analysis and I have a normal likelihood function and an inverse gamma (IG) prior for the parameter $\lambda$. I have the following posterior: $$ \propto \frac{1}{\lambda^{\tilde \alpha +1}} \cdot \text{exp}\Biggl( -\frac{1}{2} \Biggl[\sum_{t=1}^T \frac{(\psi^2 + \Omega) \Delta t}{\Omega (2\gamma)^{-2} \psi^2} \lambda^2 - 2 \sum_{t=1}^T \frac{(\psi^2 + \Omega)\left(R_t - \left(r - \frac{1}{2} h_{t-1} \right) \Delta t \right)}{\Omega \sqrt{h_{t-1}} (2\gamma)^{-2} \psi^2 } \lambda + \frac{2 \psi \left(h_t - h_{t-1} - \left(\omega + (\beta + \alpha(1+ \gamma^2) -1)h_{t-1} \right)\Delta t\right)}{\Omega \alpha h_{t-1}(2\gamma)^{-1} \psi } \lambda\Biggr] - \frac{\tilde \beta} {\lambda}\Biggr) $$ I can reduce it to: $$\propto \frac{1}{\lambda^{\tilde \alpha +1 }} \cdot\text{exp} \left(-\frac{1}{2} \left(a_{\lambda} \lambda^2 - 2 b_{\lambda} \lambda\right)- \frac{\tilde \beta}{\lambda} \right)$$ Where $a_{\lambda}$ and $b_{\lambda}$ are defined as: $$ a_{\lambda} \equiv \sum_{t=1}^T \frac{(\psi^2 + \Omega) \Delta t}{\Omega (2\gamma)^{-2} \psi^2} $$ $$ b_{\lambda} \equiv \sum_{t=1}^T \frac{(\psi^2 + \Omega)\left(R_t - \left(r - \frac{1}{2} h_{t-1} \right) \Delta t \right)}{\Omega \sqrt{h_{t-1}} (2\gamma)^{-2} \psi^2 } + \frac{2 \psi \left(h_t - h_{t-1} - \left(\omega + (\beta + \alpha(1+ \gamma^2) -1)h_{t-1} \right)\Delta t\right)}{\Omega \alpha h_{t-1}(2\gamma)^{-1} \psi }$$ I know this looks like an Inverse gamma density, but I dont know how to find the parameters of this distribution? If it was a normal distribution, we could do complete square, but that does not seem to work here.
