http://download.oracle.com/javaee/6/api/javax/ejb/TransactionAttribute.html The TransactionAttribute annotation specifies whether the container is to invoke a business method within a transaction context. The TransactionAttribute annotation can be used for session beans and message driven beans. It can only be specified if container managed transaction demarcation is used.
