While perhaps not the most elegant solution, I used static data members in my ActivityA class which I can set from ActivityB. //ActivityA.class public static boolean refreshData = false; public void onResume() { if (refreshData) { mRefreshData(); refreshData = false; } } //ActivityB.class ActivityA.refreshData = true; Intent myIntent = new Intent(v.getContext(), ActivityT.class); startActivityForResult(myIntent, 0); I'm interested to know if there's a better way to do this kind of thing!
