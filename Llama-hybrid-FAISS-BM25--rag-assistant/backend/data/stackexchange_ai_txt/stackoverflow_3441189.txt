Ruby 1.9's numeric parsing is thinking in ascii only. I don't think there's any convenient elegant parsing methods that properly handle fullwidth unicode numeric codepoints. A quick filthy hack function: def parse_utf(utf_integer_string) ascii_numeric_chars = "0123456789" utf_numeric_chars = "\uff10\uff11\uff12\uff13\uff14\uff15\uff16\uff17\uff18\uff19" utf_integer_string.tr(utf_numeric_chars, ascii_numeric_chars).to_i end Pass in a string of fullwidth numeric characters and get out an integer.
