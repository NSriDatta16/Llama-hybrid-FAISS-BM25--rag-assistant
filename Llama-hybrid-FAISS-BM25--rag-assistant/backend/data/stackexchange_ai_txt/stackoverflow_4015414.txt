From the C standard (n1256) : 6.7 Declarations ... 5 A declaration specifies the interpretation and attributes of a set of identifiers. A definition of an identifier is a declaration for that identifier that: — for an object, causes storage to be reserved for that object; — for a function, includes the function body; 101) — for an enumeration constant or typedef name, is the (only) declaration of the identifier. In this case, int x; is a definition (or a defining declaration).
