If you specify @"" for a section title, the section header won't be shown, which will give you the effect you desire. To use table indexing (the right hand side tiny list), you should still organise your table in sections so it knows where to jump when a index is touched, you just have to hide the section titles as I described. To add index items, see UITableViewDelegate's method: - (NSArray *)sectionIndexTitlesForTableView:(UITableView *)tableView
