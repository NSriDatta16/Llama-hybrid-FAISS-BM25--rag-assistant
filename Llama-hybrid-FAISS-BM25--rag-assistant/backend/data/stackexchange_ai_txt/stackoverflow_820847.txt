Three problems: You're using the process ID when you should be using the thread ID. Your HHOOK needs to go into shared memory: #pragma data_seg(".shared") HHOOK hhk = NULL; #pragma data_seg() #pragma comment(linker, "/SECTION:.shared,RWS") You need to pass your HHOOK to CallNextHookEx : return CallNextHookEx( hhk, code, wParam, lParam);
