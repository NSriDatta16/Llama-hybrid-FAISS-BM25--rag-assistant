I think you are looking for an ADODB.Command. Try something like this: Dim cnn As New ADODB.Connection Dim cmd As ADODB.Command cnn.Open ConnectionString Set cmd = New ADODB.Command With cmd .ActiveConnection = cnn .CommandText = "EXEC spNameHere param1" .CommandType = adCmdText .Execute End With
