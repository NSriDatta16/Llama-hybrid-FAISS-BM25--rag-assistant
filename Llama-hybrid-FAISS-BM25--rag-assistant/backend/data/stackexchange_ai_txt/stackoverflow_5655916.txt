It seems that the “search” TextBox is placed onto INamingContainer, so it is ClientID property cannot be evaluated. Use the approach, described in the infragistics get clientID of dropdown in Rowedit template thread: protected void search_Init(object sender, EventArgs e) { TextBox txt = (TextBox)sender; string script = string.Format("var _{0} = document.getElementById('{1}');", txt.ID, txt.ClientID); Page.ClientScript.RegisterStartupScript(Page.GetType(), "ANY_KEY", script, true); } function voidsearch() { alert(_search); alert(_search.value); }
