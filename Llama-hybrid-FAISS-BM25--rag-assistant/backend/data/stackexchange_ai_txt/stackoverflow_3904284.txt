Got to love Flex, man. Where the obvious stuff takes a ton of time. So this is what I ended up doing: mygrid.addEventListener( DragEvent.DRAG_OVER, handleDragOver); public function handlerDragOver(event:DragEvent):void{ var dropIndex:int = mygrid.calculateDropIndex(event); var rowsDisplayed:Number = mygrid.rowCount; var topvisibleIndex:int = mygrid.verticalScrollPosition; var botvisibleIndex:int = topvisibleIndex + rowsDisplayed; if ( dropIndex = botvisibleIndex - 1 ){ mygrid.verticalScrollPosition += 1; } }
