I would say so - to a greater or lesser extent. I'd say almost everyone follows most of Sun/Oracle's Code Conventions for the Java Programming Language document - but many places tweak it. For example, I've very rarely seen much Java code which doesn't follow its naming conventions for methods, but often organisations will have their own whitespace conventions.
