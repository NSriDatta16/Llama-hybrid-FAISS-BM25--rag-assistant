Embedding fonts in IE

I have a site that uses custom fonts. I've got the font form the designer in oft format. I've used a converter to convert them to eot format. and added the following code to the css @font-face { font-family: 'AdumaFOTRegular'; src: url('fonts/AdumaFOT-Regular.eot'); src: local('AdumaFOTRegular'), local('AdumaFOTRegular'), url('fonts/AdumaFOT-Regular.otf') format('opentype'); } This works great in FF and chrom but doesn't work in IE. The sample site is here: http://test.hannitcohen.co.il/eshkol/
