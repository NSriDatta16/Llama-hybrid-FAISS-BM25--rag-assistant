You can replace this line: float2 fragment = float2(stepsize * floor(uv.x * num), stepsize * floor(uv.y * num)); with these 3 lines: int tmp1 = uv.x * num; int tmp2 = uv.y * num; float2 fragment = float2(stepsize * tmp1, stepsize * tmp2); The conversion to int is an implicit floor().
