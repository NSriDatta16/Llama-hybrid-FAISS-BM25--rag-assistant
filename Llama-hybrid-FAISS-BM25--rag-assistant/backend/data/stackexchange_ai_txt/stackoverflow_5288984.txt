Why don't you try sorl-thumbnail . It has the exact same interface as the default ImageField django provides and it seems like it would be a lot nicer to work with than the roll-your-own support. Storage support Pluggable Engine support (PIL, pgmagick) Pluggable Key Value Store support (redis, cached db) Pluggable Backend support Admin integration with possibility to delete Dummy generation Flexible, simple syntax, generates no html ImageField for model that deletes thumbnails CSS style cropping options Margin calculation for vertical positioning
