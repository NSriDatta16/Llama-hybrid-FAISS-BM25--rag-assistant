I would suggest to use clone() from a model to add lines to your html like that: #line_model { display: none; } $(document).ready(function () { $('#add_line').click(function () { $('#line_model').clone().prependTo('#editor_lines').show(); }); }); That will allow you to define the behavior of your line only once. For update to your db, you can use the update() function by predrag.music
