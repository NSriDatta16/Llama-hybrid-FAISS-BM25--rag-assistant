See http://www.rgagnon.com/javadetails/java-0426.html Create a class based on the given implementation. Compile the class and put it in a directory where iReport can read the file. Update the CLASSPATH in iReport to point to the directory containing the class (be aware of directory relationships to package namespaces). Restart iReport. Change the text field expression to: EnglishNumberToWords.convert( $F{field_name} ) You will have to change field_name and the data type of the convert method according to your implementation details.
