You can use the javax.xml.transformer.Transformer API for this. Here's a basic kickoff example: Source xmlInput = new StreamSource(new File("c:/path/to/input.xml")); Source xsl = new StreamSource(new File("c:/path/to/file.xsl")); Result xmlOutput = new StreamResult(new File("c:/path/to/output.xml")); try { Transformer transformer = TransformerFactory.newInstance().newTransformer(xsl); transformer.transform(xmlInput, xmlOutput); } catch (TransformerException e) { // Handle. }
