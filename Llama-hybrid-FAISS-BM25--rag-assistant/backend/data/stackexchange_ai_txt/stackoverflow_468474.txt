If you are looking for a simple solution, there is none, except perhaps faking user input. But there is another way, but it is really hard: Use the DoDragDrop function to programmatically carry out a drag and drop operation. The hard parts are to implement the IDataSource interface and to create a IDataObject which the explorer can understand.
