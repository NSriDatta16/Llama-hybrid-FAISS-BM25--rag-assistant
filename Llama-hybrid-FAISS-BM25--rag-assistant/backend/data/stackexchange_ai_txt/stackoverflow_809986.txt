Okay, then try this: #!/usr/bin/perl use strict; use warnings; use LWP::UserAgent; my $ua = LWP::UserAgent->new; my $response = $ua->get("http://ru.wikipedia.org/wiki/Perl"); die $response->status_line unless $response->is_success; my $content = $response->decoded_content; my @russian = $content =~ /\s([\x{0400}-\x{052F}]+)\s/g; print map { "$_\n" } @russian; I believe that the Cyrillic character set starts at 0x0400 and the Cyrillic supplement character set ends at 0x052F , so this should get many of the words.
