If each algorithm works on each element on dataset , map() would be an elegant option: dataset=workflow[0] for algorithm in workflow[1:]: dataset=map(algorithm, dataset) e.g. for the square roots of odd numbers only, use, >>> algo1=lambda x:0 if x%2==0 else x >>> algo2=lambda x:x*x >>> dataset=range(10) >>> workflow=(dataset, algo1, algo2) >>> for algo in workflow[1:]: dataset=map(algo, dataset) >>> dataset [0, 1, 0, 9, 0, 25, 0, 49, 0, 81]
