If you want only one include, namespaces have nothing to do with this. You can create a file that only contains #include statements. Something like this: //classes file #include "classA" #include "classB" #include "classC" And the include all of them with only one include #include "classes" A real example can be found in the STL. Take vector for instance: #ifndef _GLIBCXX_VECTOR #define _GLIBCXX_VECTOR 1 #pragma GCC system_header #include #include #include #include #include #include #ifndef _GLIBCXX_EXPORT_TEMPLATE # include #endif #ifdef _GLIBCXX_DEBUG # include #endif #endif /* _GLIBCXX_VECTOR */ You get all of this by just doing #include
