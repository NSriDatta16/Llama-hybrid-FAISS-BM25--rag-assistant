You could do this... probably not the most graceful but it would work... Just swap out @Table with the table you're querying... SELECT DISTINCT T.AGENT, (SELECT COUNT(AGENT) FROM @TABLE WHERE AGENT = T.AGENT AND Status LIKE 'Fax%') AS Fax_Count, (SELECT COUNT(AGENT) FROM @TABLE WHERE AGENT = T.AGENT AND Status LIKE 'Mail%') AS Email_Count FROM @TABLE T
