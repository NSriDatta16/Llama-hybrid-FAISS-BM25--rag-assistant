Put your XmlWriter in a using statement; using (XmlWriter writer = XmlWriter.Create(stream)) { writer.WriteStartElement("logentry"); writer.WriteAttributeString("t", DateTime.Now.ToString()); writer.WriteString("Something to log."); writer.WriteEndElement(); } You'll probably have to do some XmlDocumentFragment tricks also.
