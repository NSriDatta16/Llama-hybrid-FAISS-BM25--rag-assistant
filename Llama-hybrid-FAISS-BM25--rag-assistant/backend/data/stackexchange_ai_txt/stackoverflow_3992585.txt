If you want to have the icon appear correctly in all screen densities, then you should create your own drawable-mdpi, drawable-hdpi, etc folders and copy the icons into each of those. That way the correct size icon will be used for each density. For more details on how this works, see supporting multiple screen sizes , particularly the section on resource qualifiers.
