Not particularly elegant, but simple and direct: class X(object): """This class has a method foo().""" def foo(): pass class Y(X): __doc__ = X.__doc__ + ' Also bar().' def bar(): pass Now: >>> print Y.__doc__ This class has a method foo(). Also bar().
