Instead of using hard-coded myFile.txt , simply buid a file name using DateTime.Today . For example: string fileName = DateTime.Today.ToString("yyyyMMdd", DateTimeFormatInfo.InvariantInfo); Then you can create your StreamWriter : StreamWriter sw = new StreamWriter(new IsolatedStorageFileStream("Data\\" + fileName, FileMode.Create, isf));
