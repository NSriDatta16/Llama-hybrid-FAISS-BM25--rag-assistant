Read the file in main (or anywhere else in the normal control flow of your program). Put the data into an instance variable accessible to the JPanel . In paint() , read the instance variable and draw from the data stored therein. For example: public class Foo extends JPanel { private SomeDataStorageClass myData; public static void main(String[] args) { Foo f = new Foo(); f.myData = readSomeData(); } public void paint(Graphics g) { doSomeDrawing(myData); } }
