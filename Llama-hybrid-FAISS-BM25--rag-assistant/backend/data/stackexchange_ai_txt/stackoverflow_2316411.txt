You could use a Hashtable but its use is discouraged in favour of Map and HashMap : public static void main (String[] argv) { Map h = new HashMap (); // Insert a string and a key. h.put("Ali", "Anorexic Ali"); h.put("Bill", "Bulimic Bill"); h.put("Chen", "Cadaverous Chen"); h.put("Dave", "Dyspeptic Dave"); String d = h.get("Dave"); System.out.println (d); // Prints "Dyspeptic Dave" } You could replace the declaration with: Map h = new Hashtable (); In general you want to use interfaces for your variable declarations, parameter declarations and return types over concrete classes if that's an option.
