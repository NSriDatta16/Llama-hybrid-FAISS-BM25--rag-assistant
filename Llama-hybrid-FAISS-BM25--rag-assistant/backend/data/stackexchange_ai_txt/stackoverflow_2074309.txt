Make the basic voting actions mini-forms, then use javascript to disable their posting action. To replace these with links for javascript-enabled users: document.getElementById("voteupbutton").innerHTML=" Vote up "; I haven't tested the above. If you're using jQuery or some other framework, there will be more elegant ways of doing all of this.
