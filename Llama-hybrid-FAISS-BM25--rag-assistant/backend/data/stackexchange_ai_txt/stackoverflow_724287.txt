jQuery seems to the market leader at the moment. It is lightweight (19kb) and has a large number of plugins available from the community. ( Please note - the file size is only when when minified and GZipped. Adding UI will increase the file size ) One of the most used plugins is that which relates to improving the User Interface is called UI and includes draggable and droppable as functionality. The home page of the jQuery UI is at http://docs.jquery.com/UI A list of other plugins whih relate to 'drag and drop' is located at http://plugins.jquery.com/project/Plugins/category/45
