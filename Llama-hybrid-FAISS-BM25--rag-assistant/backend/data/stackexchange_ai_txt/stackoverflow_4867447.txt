How about using ADODB? I assumed SQL is placed in A1 and below: Const MY_CONNECTION_STRING = "Driver={SQL Server};Server=X;Database=Y;Trusted_Connection=Yes" Sub RunSQL() Dim cnn, c, rs Set cnn = CreateObject("ADODB.Connection") cnn.Open MY_CONNECTION_STRING For Each c In Range("A1:A" & UsedRange.Rows.Count) Set rs = cnn.Execute(c.Value) c.Next.Value = rs.collect(0) rs.Close Next c cnn.Close End Sub
