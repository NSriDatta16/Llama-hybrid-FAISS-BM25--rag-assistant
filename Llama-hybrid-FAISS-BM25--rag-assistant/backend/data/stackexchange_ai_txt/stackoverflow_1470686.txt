You probably want ~/Library/LaunchAgents if your app will interact with the user. I'm not aware of a Cocoa method that will return this path for you. There is a detailed tech note on agents and daemons that you should probably check out: http://developer.apple.com/mac/library/technotes/tn2005/tn2083.html
