Yes, there's a WebBrowser control in Microsoft.Phone.Controls If you then save your (offline) files to IsolatedStorage you can then view the file via a call like this from code: webBrowser1.Navigate(new Uri("offline-file-name.html", UriKind.Relative)); Things to note: You can use directories within isolated storage. Just specify the whole path in the Uri. If navigating between offline pages, all paths must be relative.
