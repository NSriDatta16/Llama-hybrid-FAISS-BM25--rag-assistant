You could match those patterns pretty easily as suggested by BalusC. As a side note, StringTokenizer has been deprecated. From JavaDoc : StringTokenizer is a legacy class that is retained for compatibility reasons although its use is discouraged in new code. It is recommended that anyone seeking this functionality use the split method of String or the java.util.regex package instead. An easier way to split your string into the appropriate segments would be: String phoneParts[] = phoneNumber.split("-");
