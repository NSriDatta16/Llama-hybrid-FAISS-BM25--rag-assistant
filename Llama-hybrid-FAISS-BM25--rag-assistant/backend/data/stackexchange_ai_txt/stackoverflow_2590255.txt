You can check out: bow - a veteran C library for text classification; I know it stores the matrix, it may require some hacking to get it. Weka - a Java machine learning framework that can handle text and build the matrix Sujit Pal's blog post on building the term-document matrix from scratch If you insist on using Lucene, you should create an index using term vectors, and use something like a loop over getTermFreqVector() to get the matrix.
