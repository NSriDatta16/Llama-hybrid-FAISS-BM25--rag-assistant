Not sure if this is causing the problems you're running into, but a better way to do a JSON response would be to use the existing "Magento/Zend way" of doing it. Instead of: header('Content-Type: application/json',true); ob_end_clean(); echo json_encode($result); exit(); Use: $this->getResponse()->setHeader('Content-Type', 'application/json', true)->setBody(json_encode($result));
