What does your $sPostData look like? I use the following to encode arrays for CURL before POSTing. $post contains an associative array of POST fields. foreach($post as $key=>$value) { $postString .= urlencode($key) . '=' . urlencode($value) . '&'; } rtrim($postString,'&'); Then: curl_setopt($ch, CURLOPT_POST, count($post)); curl_setopt($ch, CURLOPT_POSTFIELDS, $postString); The idea is that CURLOPT_POSTFIELDS should look like a GET string and URL encoded. You may also need to set CURLOPT_USERAGENT, CURLOPT_ENCODING, CURLOPT_HTTPHEADER. For debugging you can enable CURLOPT_HEADER, CURLOPT_HEADER and CURLINFO_HEADER_OUT.
