What you're looking for is a ORM layer. There are a number of tools out there, probably the quickest solution is to add a linq2sql data context. Just right click on the model folder and select LINQ to SQL Classes then drag your tables from the server explorer onto the canvas. It will generate all your data classes for you. Other options are such thigs as subsonic and NHibernate A more complete list can be found on the source of all human knowledge, wikipedia: http://en.wikipedia.org/wiki/List_of_object-relational_mapping_software#.NET
