- (id)init { if (self = super init]) { [NSApp setDelegate:self]; } return self; } You can also do this in Interface Builder; from "File's Owner" in MainMenu.xib, just drag the "delegate" outlet to your object. You may want to consider using -awakeFromNib instead though.
