You're looking for the #pragma warning disable directive http://msdn.microsoft.com/en-us/library/441722ys(VS.80).aspx Essentially you add the following command above the call site in the .cs file. #pragma warning disable 612 SomeMethodCall 612 is the error message ID for calling obsolete methods
