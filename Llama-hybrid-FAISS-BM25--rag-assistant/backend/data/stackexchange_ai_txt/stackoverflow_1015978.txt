Generically, you can do something like this: SELECT ( (COALESCE(val0, 0) + COALESCE(val1, 0) + ...... COALESCE(valn, 0)) / (SIGN(ABS(COALESCE(val0, 0))) + SIGN(ABS(COALESCE(val1, 0))) + .... ) ) AS MyAverage The top line will return the sum of values (omitting NULL values) whereas the bottom line will return the number of non-null values. FYI - it's SQL Server syntax, but COALESCE is just like ISNULL for the most part. SIGN just returns -1 for a negative number, 0 for zero, and 1 for a positive number. ABS is "absolute value".
