This how I change the input size in Keras model. I have two CNN models, one with input size $[None, None, 3]$ while the other has input size $[512,512,3]$ . Both models have the same weights. By using set_weights(model.get_weights()) , the weights of model 1 can be transferred to model 2. inputs = Input((None, None, 3)) ..... model = Model(inputs=[inputs], outputs=[outputs]) model.compile(optimizer='adam', loss='mean_squared_error') model.load_weights('my_model_name.h5') inputs2 = Input((512, 512, 3)) .... model2 = Model(inputs=[inputs2], outputs=[outputs]) model2.compile(optimizer='adam', loss='mean_squared_error') model2.set_weights(model.get_weights())
