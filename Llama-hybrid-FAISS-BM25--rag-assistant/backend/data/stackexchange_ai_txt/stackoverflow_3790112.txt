1) Install GhostScript 2) Install ImageMagick 3) Create "Convert-to-TIFF.bat" (Windows XP, Vista, 7) and use the following line: for %%f in (%*) DO "C:\Program Files\ImageMagick-6.6.4-Q16\convert.exe" -density 300 -compress lzw %%f %%f.tiff Dragging any number of single-page PDF files onto this file will convert them to compressed TIFFs, at 300 DPI.
