Parrot and LLVM are two different things made by two different organizations. Parrot is a VM, and LLVM is a VM that can also compile statically. Edit Assuming the comments to my answer are correct: LLVM allows you to call other LLVM code and also external C like libraries. Parrot allows you to call other Parrot code, but not external C like libraries. http://en.wikipedia.org/wiki/Comparison_of_application_virtual_machines
