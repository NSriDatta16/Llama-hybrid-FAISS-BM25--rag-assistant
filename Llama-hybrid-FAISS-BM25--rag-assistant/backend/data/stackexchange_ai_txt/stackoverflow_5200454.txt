Function CalcSubjectAverage(ByVal studentArray(,) As Integer, ByVal subject As Integer) As Single Dim total As Single For i As Integer = 0 To studentArray.Length total += studentArray(i, 0) Next Return total / studentArray.Length End Function You need to add the subjects up first, then you can calculate the average :)
