Here's a little snippet to not only get the average but also the standard deviation. class User attr_accessor :score def initialize(score) @score = score end end @users=[User.new(10), User.new(20), User.new(30), User.new(40)] mean=@users.inject(0){|acc, user| acc + user.score} / @users.length.to_f stddev = Math.sqrt(@users.inject(0) { |sum, u| sum + (u.score - mean) ** 2 } / @users.length.to_f )
