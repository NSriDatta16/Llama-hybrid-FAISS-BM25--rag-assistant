Assume that in the parent window this is the text box that needs to be updated. In parent window create a function to update the textbod with the new text .... .... .... function updateTextBox(theTextString) { $("#txtBoxDisplay").val(theTextString); //assuming you are using jquery document.getElementById('txtBoxDisplay').value = theTextString; // if not using jquery } In the child page/modal window use the following code to access the parent jsp function window.opener.updateTextBox("Display this text in the parent text box"); You can also use parent.updateTextBox("Display this text in the parent text box"); For more details see the following links http://chiragrdarji.wordpress.com/2007/03/10/call-parent-windows-javascript-function-from-child-window-or-passing-data-from-child-window-to-parent-window-in-javascript/ http://www.rgagnon.com/jsdetails/js-0066.html
