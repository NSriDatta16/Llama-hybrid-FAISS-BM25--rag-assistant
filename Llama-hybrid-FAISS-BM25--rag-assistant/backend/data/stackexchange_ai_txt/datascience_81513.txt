For Python, Python: The Complete Reference and Head First Python .These two should be good enough. For Machine learning, it really depends whether you want to get into all the math or not. If not, I think there are enough resources online. But if you do, I would recommend Pattern Recognition and Machine Learning by Christopher Bishop. It really dives deep into the math for ML.
