checkout sass... it includes the ability to convert css to sass. http://haml.hamptoncatlin.com/docs/rdoc/classes/Sass.html A sass file is a yaml file that can be parsed down into a css file. It allows you to use variables and alternate organization... sass example: !main_color = #00ff00 #main :color = !main_color :p :background-color = !main_color :color #000000 css output: #main { color: #00ff00; } #main p { background-color: #00ff00; color: #000000; }
