There isn't much to it. You need to loop over the array, maintaining variables with the indexes of the high/low elements so far for each attribute. After the loop, access those elements and add your extra data. $lowAverage = $highAverage = array('index' => null, 'value' => 0); $lowFast = $highFast = array('index' => null, 'value' => 0); foreach($dogs as $index => $data) { if($lowAverage['index'] == null || $data['av_calc_7'] $index, 'value' => $data['av_calc_7']); } if($highAverage['index'] == null || $data['av_calc_7'] > $highAverage['value']) { $highAverage = array('index' => $index, 'value' => $data['av_calc_7']); } // same for $lowFast, $highFast } if($lowAverage['index'] != null) { $dogs[$lowAverage['index']]['av_calc_7_lowest'] = true; } // same for the other 3 values
