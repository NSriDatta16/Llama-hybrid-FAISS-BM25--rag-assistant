There is no perfect solution to your question if you don't know the number of columns. You can try: table { table-layout:fixed; } This will force the table to use fixed, equal cells. The problem is if your content is too large for the cell size it'll cause issues with content overlap, etc. Here's a basic sample using your markup: http://jsfiddle.net/9HC8b/ . Notice when the screen width is wide enough everything looks great. If you start dragging (shrinking) the window you'll see the content blows up.
