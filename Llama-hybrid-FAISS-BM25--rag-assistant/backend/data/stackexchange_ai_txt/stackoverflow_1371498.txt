The P tag isn't a vanilla block level element. Its default state in most user agents specifies some top and bottom margins. The TABLE tag doesn't. So the P tag is pushing the inline DIVs farther apart. margins on P tag http://homepage.mac.com/estranged/images/css01.png margins on TABLE tag http://homepage.mac.com/estranged/images/css02.png
