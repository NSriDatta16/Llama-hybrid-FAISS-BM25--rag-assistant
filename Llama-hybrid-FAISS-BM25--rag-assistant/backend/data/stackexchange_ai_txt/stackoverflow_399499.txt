A really nice system done entirely in Python is Plone if your more comfortable with Python vs PHP. I'd imagine that you hope to get up and running quickly with a design that suits the organization, so it really does make sense to use Drupal or Wordpress because so many free themes and designs are available for either. I do quite a bit of work for several non-profits, many users have found Joomla a little intimidating on the back end.
