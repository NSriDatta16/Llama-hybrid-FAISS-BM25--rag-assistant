There can be two types of difference between versions of Flash Player: From end user point of view. From application developer point of view. From the end user point of view differences are not so big. Performance, player size, security fixes. But every new version of Flash Player brings new APIs, language and runtime environment changes. So with new versions of Flash Player end users can use web applications which uses these new advantages. For example, the latest public preview of Flash Player codenamed Molehill offer new level of 3D support. So developers can develop applications which leverages 3D APIs and end users can play realistic 3D games with that version of Player. To see the particular differences follow this link .
