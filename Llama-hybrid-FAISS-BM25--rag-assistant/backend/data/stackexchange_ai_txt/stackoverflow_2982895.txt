$('#selector').click(function(event) { event.preventDefault(); }); The event object is passed to your click handler by default - you have to have something there to receive it. Once you have the event, you can use jQuery's .preventDefault() method to cancel the link's behavior. Edit: Here's the fragment of your code, corrected: $('a', ui.panel).click(function(event) { $(ui.panel).load(this.href); event.preventDefault(); return false; }); Notice the addition of the word 'event' when creating the anon function (or you could use just e, or anything else - the name is unimportant, the fact there's a var there is.
