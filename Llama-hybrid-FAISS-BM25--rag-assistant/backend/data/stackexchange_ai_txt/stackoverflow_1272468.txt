You can do this without floats, if you set the form and every element inside it to be display inline then they will sit next to each other. The reason they are not side by side is because forms, just like divs and paragraphs are block level elements, setting them to be display inline will fix this. For Example .button-container form, .button-container form div { display: inline; } .button-container button { display: inline; vertical-align: middle; } With the following HTML Confirm Cancel
