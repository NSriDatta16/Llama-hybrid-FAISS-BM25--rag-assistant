Decorators are a good way to add metadata to functions. Add one that takes a list of types to append to a .params property or something: def takes(*args): def _takes(fcn): fcn.params = args return fcn return _takes @takes("time", "temp", "time") def do_stuff(start_time, average_temp, stop_time): pass
