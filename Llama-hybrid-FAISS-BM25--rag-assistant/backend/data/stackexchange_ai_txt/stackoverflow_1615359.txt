It will return a list whose elements are object[] ; each element of array is the value of projected property. You can map the results to a class instead by using result transformer: criteria.SetResultTransformer(Transformers.AliasToBean(typeof (MyResult))); The above assumes you have a MyResult class whose property names match the names (or aliases) of properties you've specified in projection.
