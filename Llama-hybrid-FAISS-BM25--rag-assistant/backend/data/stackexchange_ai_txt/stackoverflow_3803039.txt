There is command line tool that name is VsPerfMon. Whit this tool you can run your code coverage via command line. You can check this link for more information, http://msdn.microsoft.com/en-us/library/ms182404(VS.80).aspx So if you can run code coverage via command line, you can invoke this tool via TFS 2008 Build Scripts.
