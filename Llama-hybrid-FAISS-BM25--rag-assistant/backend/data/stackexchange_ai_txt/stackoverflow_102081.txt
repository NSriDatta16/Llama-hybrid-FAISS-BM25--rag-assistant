Launchd is a the preferred way in OS X. If you want it to run on your login put it in ~/Library/LaunchAgents Start launchd item launchctl load /Library/LaunchDaemons/com.bob.plist Stop item launchctl unload /Library/LaunchDaemons/com.bob.plist Example com.bob.plist Label com.bob RunAtLoad ProgramArguments /usr/bin/java -jar /Users/user/program.jar
