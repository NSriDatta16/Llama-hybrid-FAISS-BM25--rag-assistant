To create an anchor using l(): $path = isset($_GET['q']) ? $_GET['q'] : ' '; l(t('link text'), $path, array('attributes' => array('name' => 'name-of-anchor'))); This will output: link text Then, to link to this using l(): $path = isset($_GET['q']) ? $_GET['q'] : ' '; l(t('link to anchor'), $path, array('fragment' => 'name-of-anchor')); This will output: link to anchor
