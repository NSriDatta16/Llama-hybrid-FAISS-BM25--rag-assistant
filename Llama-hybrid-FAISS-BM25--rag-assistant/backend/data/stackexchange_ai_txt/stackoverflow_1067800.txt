There is no built-in function for this in the iPhone SDK. You should file a bug that you want the functionality. In the normal Mac OS X SDK you can either load the fragment into an NSAttributedString as HTML and ask it to hand back a plain string, or use CFXMLCreateStringByUnescapingEntities() . @interface NSString (LGAdditions) - (NSString *) stringByUnescapingEntities; @end @implementation NSString (LGAdditions) - (NSString *) stringByUnescapingEntities { CFStringRef retvalCF = CFXMLCreateStringByUnescapingEntities(kCFAllocatorDefault, (CFStringRef)self, NULL); return [NSMakeCollectable(retvalCF) autorelease]; } @end
