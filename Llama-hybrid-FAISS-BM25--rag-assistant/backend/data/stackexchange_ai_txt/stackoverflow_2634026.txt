I'm putting my answer in the form of a (passing) test, as an example of how you might research this yourself. Not to discourage you from using SO - it's great - just to try to promote characterization tests . import java.util.ArrayList; import junit.framework.TestCase; public class ContainsTest extends TestCase { public void testContains() throws Exception { ArrayList list = new ArrayList (); assertFalse(list.contains(new Integer(17))); list.add(new Integer(17)); assertTrue(list.contains(new Integer(17))); } }
