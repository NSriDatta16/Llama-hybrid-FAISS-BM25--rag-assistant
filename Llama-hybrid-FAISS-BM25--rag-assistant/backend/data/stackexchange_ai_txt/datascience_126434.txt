You are using the GPT-2 tokenizer instead of the model's own tokenizer. Try getting the tokenizer from the model instead: from transformers import AutoTokenizer import transformers tokenizer = AutoTokenizer.from_pretrained(model) pipeline = transformers.pipeline( "text-generation", model=model, tokenizer=tokenizer, )
