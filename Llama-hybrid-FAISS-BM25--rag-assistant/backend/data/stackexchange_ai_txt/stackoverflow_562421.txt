As long as you have a unique id for each user (which is not their name) you can have a table that maps name variations to a unique id, and then associate each post with that unique ID. (Table mapping names to UIDs) Name UID Robert S 123456 Bob S 123456 Bert S 123456 Darren 987654 (Table with post information, including author's UID) Title Author ... Post 1 123456 Post 2 123456 Post 3 987654 (Table with author information) UID Preferred Name Webpage ... 123456 Robert Smith http://www.robert.com 987654 Darren Jones http://www.jones.com
