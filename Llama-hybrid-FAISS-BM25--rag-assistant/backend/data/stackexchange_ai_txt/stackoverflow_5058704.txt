User Authentication makes sure that if a user tries to access a page which application denies free access, it redirects the user to logging page and after a successful login brings back to the requested page. One such implementation of cake's default Auth is explained in following wrt pitfalls, approach and ways. http://enbake.com/cakephp-user-authentication-auth-component The framework doesnt make you restricted. But rather grants you speed of dev with existing modules and more organized code. Can show you comparison b/w frameworks if interested.
