This kind of problem is called record linkage (or sometimes entity matching or other variants). The task consists in finding among a list of strings representing entities (persons or organizations) those which represent the same actual entity. There are two main approaches (which can be combined): String similarity matching methods. See for example this question or this one . Note that in case the list of companies is large, there can also be an efficiency issue: see this question . Databases or third-party resources. See for example this related question .
