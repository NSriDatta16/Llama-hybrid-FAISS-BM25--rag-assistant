To keep your code decoupled and your controllers easily testable you need to stick with dependency injection so either: public DinnersController() : this(new DinnerRepository(), new CategoryRepository()) { } or the less elegant public DinnersController() : this(new DinnerRepository(new CategoryRepository())) { }
