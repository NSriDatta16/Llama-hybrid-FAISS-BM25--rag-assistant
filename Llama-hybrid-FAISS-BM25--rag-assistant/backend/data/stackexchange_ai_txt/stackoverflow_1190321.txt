I'm not sure what the original problem was, but you definitely should avoid using XslTransform altogether (it's deprecated), and use XslCompiledTransform instead. Here's how: transformer = New XslCompiledTransform() transformer.Load("firelist.xslt") ''# transform XML data output = New StringWriter() transformer.Transform(navigator, Nothing, output) Try this and see if you get a different behavior.
