For the first Strict standards, you just have to modify function __call in AlfrescoWebService.php like this public function __call($function_name, $arguments) $arguments is not optionnal For the second, public function __soapCall($function_name, $arguments=array(), $options=array(), $input_headers= array(), $output_headers=array()) This must be modify like this public function __soapCall($function_name, $arguments, $options=array(), $input_headers= array(), &$output_headers=array()) $arguments still not optionnal $output_headers passed by reference.
