The old html presentational attributes are not a useful thing to use nowadays. I would discourage you from using any attributes on any elements in your body with the exceptions of class, id, data-* and things on input, audio, video or img. To get the same effect, use table { border-collapse: collapse; border-style:hidden; } table td { border-width: 1px; border-style: inset; border-color:black; } As demoed here: http://jsfiddle.net/HknDE/35/
