Access the underlying LWP::UserAgent object and set the proxy. LWP::Simple exports the $ua variable so you can do that: use LWP::Simple qw( $ua get ); $ua->proxy( 'http', 'http://myproxy.example.com' ); my $content = get( 'http://www.example.com/' );
