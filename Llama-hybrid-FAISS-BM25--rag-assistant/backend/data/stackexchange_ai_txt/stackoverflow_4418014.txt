You can use simple storage built into the browser. You start with this commands: var pref = Components.classes["@mozilla.org/preferences-service;1"].getService(Components.interfaces.nsIPrefBranch); Then you can save data into like this: pref.setCharPref("freelayer.mydata", mydataold + mydatanew); Then you can read that data with: var mydata = pref.getCharPref("freelayer.mydata").split(' ');
