If it is too difficult to scale the black line you can scale the red one and the horizontal line instead. In the following code you get overlap: n = 10^7 tibble(cauchy_real = rcauchy(n)) %>% filter(cauchy_real >= -10, cauchy_real % ggplot() + geom_density(aes(x = cauchy_real)) + geom_function(fun = function(x){dcauchy(x) / (pcauchy(10) - pcauchy(-10))}, colour = 'red') + geom_hline(yintercept = 1/pi / (pcauchy(10) - pcauchy(-10)))
