I think "gsub" is your friend here: class UrlParser attr_accessor :text, :url_counter, :urls def initialize(text) @text = parse(text) end private def parse(text) @counter = 0 @urls = [] text.gsub(%r{(\A|\s+)(http://[^\s]+)}) do @urls If you really need extra fast parsing of long strings, you should build a ruby C extension with ragel .
