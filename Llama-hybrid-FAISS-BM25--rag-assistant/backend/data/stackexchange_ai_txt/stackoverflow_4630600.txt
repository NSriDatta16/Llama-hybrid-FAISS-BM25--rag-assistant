session_cache_limiter('nocache'); That should automatically generate all the required headers. Note that some browsers, such as Opera, still ignore these when using the back button. EDIT The relevant headers are as follows, in case you want to generate them some other way: Expires: Thu, 19 Nov 1981 08:52:00 GMT Cache-Control: no-store, no-cache, must-revalidate, post-check=0, pre-check=0 Pragma: no-cache
