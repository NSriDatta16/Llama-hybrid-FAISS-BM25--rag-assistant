Nokogiri is a great tool for parsing HTML and XML in Ruby. Using it frees you from dealing with all sorts of HTML inconsistencies due to malformed markup, or changing structure. This would wrap the contents of all tags throughout a HTML document: require 'nokogiri' html = ' x.com ' doc = Nokogiri::HTML::DocumentFragment.parse(html) doc.search('a').each do |_node| _node.inner_html = " #{_node.content} " end puts doc # >> x.com
