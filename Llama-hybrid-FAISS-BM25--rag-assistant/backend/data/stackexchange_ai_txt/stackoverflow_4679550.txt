In fact ORI gem can bring RI to your IRB console and give some more neat class exploration functions. Watch intro screencast . Setup ( ~/.irbrc ) require "rubygems" require "ori" Request RI on a Class Array.ri String.ri [].ri "".ri 5.ri Request RI on a Method String.ri :upcase "".ri :upcase [].ri :sort Hash.ri :[] Request Interactive Method List String.ri // "".ri // "".ri /case/ "".ri /^to_/ User.ri /^validates_/ etc.
