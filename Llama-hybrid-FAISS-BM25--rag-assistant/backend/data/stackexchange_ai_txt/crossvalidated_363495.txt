Is it possible to calculate the volume of a point cloud using principal eigenvalues from PCA?

Assume I have a point cloud in $\mathbf{R}^N$ and we take the $N$ principal component eigenvalues: $\lambda_1 , ... , \lambda_N$. Does $\prod_i \lambda_i$ represent the effective volume surrounding the point cloud? For example consider this random 3D data 200 point cloud: %matplotlib inline import numpy as np from sklearn.decomposition import PCA from matplotlib import pyplot from mpl_toolkits.mplot3d import Axes3D from mpl_toolkits import mplot3d from scipy.spatial import ConvexHull X=np.random.normal(0, 1, (200, 3)) pca = PCA(n_components=3) pca.fit(X) lambdas=pca.explained_variance_ fig = pyplot.figure() ax = Axes3D(fig) ax.scatter(X[:, 0], X[:, 1], X[:,2]) hull = ConvexHull(X) print(np.prod(lambdas), lambdas) print(hull.volume) Which gives the following results: $\sqrt {\det{XX'}}$: 0.959663480711 volume of convex hull: 67.221497558671 Why is the volume of the convex hull almost 75 bigger than $\prod_{i=1}^3 \lambda_i$? How can I calculate the minimal ellipsoid that contains all/most of the points? When I say most I mean give some percentage $\alpha$ (e.g. 90%) find the volume of an ellipsoid that contains 90% of the points.
