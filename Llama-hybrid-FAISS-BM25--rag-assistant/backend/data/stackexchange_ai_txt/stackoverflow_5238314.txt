I can only think of a solution using reflection. You can use reflection to invoke the private constructor that uses a shared character array: char[] chars = {'a', 'b', 'c'}; Constructor con = String.class.getDeclaredConstructor(int.class, int.class, char[].class); con.setAccessible(true); String password = con.newInstance(0, chars.length, chars); System.out.println(password); //erase it Arrays.fill(chars, '\0'); System.out.println(password); Edit For anyone thinking this is a failproof or even useful precaution, I encourage you to read jtahlborn's answer for at least one caveat.
