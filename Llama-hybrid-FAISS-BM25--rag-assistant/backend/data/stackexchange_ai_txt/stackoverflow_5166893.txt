I have resolved this issue. I am using the code to solve the issue. Here 'Path' is a local file System path that can be got using: Environment.getExternalStorageDirectory().getAbsolutePath() public static String loadImageFromWebOperations(String url, String path) { try { InputStream is = (InputStream) new URL(url).getContent(); System.out.println(path); File f = new File(path); f.createNewFile(); FileOutputStream fos = new FileOutputStream(f); try { byte[] b = new byte[100]; int l = 0; while ((l = is.read(b)) != -1) fos.write(b, 0, l); } catch (Exception e) { } return f.getAbsolutePath(); } catch (Exception e) { System.out.println("Exc=" + e); return null; } } Hope this helps :).
