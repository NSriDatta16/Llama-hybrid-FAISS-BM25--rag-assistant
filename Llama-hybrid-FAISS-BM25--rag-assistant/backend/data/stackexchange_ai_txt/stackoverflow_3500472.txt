You have several options, but the recommended way* is to use JPA (Java Persistence API) for storage and retrieval of your data. To learn JPA, you can read this chapter of Java EE tutorial: Introduction to the Java Persistence API *: For some projects it makes sense to consider other options as well, but JPA is the standard persistence technology of Java EE 5 and 6.
