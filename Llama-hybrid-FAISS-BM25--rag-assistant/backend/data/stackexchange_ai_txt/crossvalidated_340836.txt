I would recommend using the AUC metric, since you are using xgboost. Also, you can try F1 metric, however it's a custom metric: def xg_f1(y,t): true = t.get_label() y_bin = [1 if y_cont > 0.5 else 0 for y_cont in y] return 'f1',f1_score(true,y_bin)
