Assuming you like to live dangerously, you can run multiple "threads" from one script using the pragma AUTONOMOUS_TRANSACTION. For example: DECLARE PROCEDURE foo(i IN PLS_INTEGER) AS PRAGMA AUTONOMOUS_TRANSACTION; BEGIN INSERT INTO qux SELECT * FROM bar WHERE baz = i; COMMIT; EXCEPTION WHEN OTHERS THEN ROLLBACK; END; BEGIN foo(1); foo(2); foo(3); END;
