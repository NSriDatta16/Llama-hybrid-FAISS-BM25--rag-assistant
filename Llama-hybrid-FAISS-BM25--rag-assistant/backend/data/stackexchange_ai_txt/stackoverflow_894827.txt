I found it. The following code will return the path to the user.config file. You need to add a reference to System.Configuration.dll Configuration config = ConfigurationManager.OpenExeConfiguration(ConfigurationUserLevel.PerUserRoamingAndLocal); string path = config.FilePath; Now I can use a FileSystemWatcher to get notified when the file changes.
