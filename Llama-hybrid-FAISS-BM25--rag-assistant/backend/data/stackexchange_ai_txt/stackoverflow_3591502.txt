If you're going to code your own, you might want to do it programmatically. Hard-coding all the menu options in there isn't going to be very workable. I was interested (and bored), so I coded this up. class Menu def menu_options self.class.instance_methods(false) - ['title'] end def query puts title puts '=' * title.length menu_options.each_with_index do|meth,idx| puts " %3s: %s" % [ idx + 1, meth.capitalize.gsub(/_(\w)/){ ' '+$1.upcase } ] end print '? ' choice = gets.chomp.to_i - 1 if choice >= menu_options.length or choice
