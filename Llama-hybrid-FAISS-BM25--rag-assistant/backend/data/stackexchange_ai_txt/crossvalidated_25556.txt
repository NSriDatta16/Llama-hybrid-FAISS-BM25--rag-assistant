There's no single R package which is as comprehensive as NLTK, but there are a number of packages which provide similar functionality. The Natural Language Task View at CRAN is a good place to start, especially if you have a particular goal in mind. For general purpose NLP tasks, the most useful packages are probably openNLP , which provides an interface to the Java package OpenNLP, and tm , which provides basic quantitative tools for text mining.
