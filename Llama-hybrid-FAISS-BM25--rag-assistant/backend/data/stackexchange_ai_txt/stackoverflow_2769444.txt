The JavaEE distro of Eclipse will do that, but only in those cases where the JSP contains type information. Ironically, this only really happens with , which is the old, pre-JSTL way of declaring java beans in the JSP. New-style JSPs are pretty devoid of java type info, and Eclipse has little or nothing to get a handle on.
