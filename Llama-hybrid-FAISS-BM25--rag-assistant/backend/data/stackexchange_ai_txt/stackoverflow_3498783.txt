You might be setting cache mode to BitmapCache a little too aggressively. Using BitmapCache can hurt performance in some cases. You can find some basic guidance on Silverlight performance issues at http://msdn.microsoft.com/en-us/library/cc189071(VS.95).aspx (which includes some tips on using BitmapCache).
