In ASP.NET 1.0 - 1.1 the style attribute was automatically added but in ASP.NET 2.0 onwards this is not a default behavior. However here is a workaround To set the style properties for each control you add [VS2010]You can set Tools->Options->HTML Designer->CSS Styling Check -> Change positioning to absolute for controls added using Toolbox, paste or drag and drop Choose -> Auto Style Application
