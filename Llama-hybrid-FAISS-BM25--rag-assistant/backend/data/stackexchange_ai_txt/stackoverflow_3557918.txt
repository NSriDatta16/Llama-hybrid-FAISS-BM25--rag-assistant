The recommended base class to use for creating your own collection implementations is System.Collections.ObjectModel.Collection (from MSDN) This base class is provided to make it easier for implementers to create a custom collection. Implementers are encouraged to extend this base class instead of creating their own. public class CustomerCollection : Collection { }
