With the sigmoid function in logistic regression, these two loss functions are totally same, the main difference is that $y_i\in\{-1,1\}$ is used in first loss function; $y_i\in\{0,1\}$ is used in the second loss function. Two loss functions can be derived by maximizing likelihood function.
