Create a SQL Server agent job that runs daily at your selected time. Have that job run a stored procedure. In that stored procedure you'll need to loop over what ever you need to send an e-mail for. One iteration per distinct e-mail, calling EXEC msdb.dbo.sp_send_dbmail ...,...,.. to send the actual e-mail. Beyond that, you need to ask specific questions. You'll need to build up a list of recipient e-mail addresses into a local variable, as well as construct a message body, but without any details in the question how can I explain what to do?
