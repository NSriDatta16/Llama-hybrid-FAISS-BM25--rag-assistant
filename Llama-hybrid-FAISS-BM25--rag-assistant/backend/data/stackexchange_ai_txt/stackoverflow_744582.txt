Might be overkill for something like this, but there are a number of really good utilities for these types of problems in the Apache Commons Collections library. Map map = new HashMap (); map.put("key1", "a "); map.put("key2", " b "); map.put("key3", " c"); TransformedMap.decorateTransform(map, TransformerUtils.nopTransformer(), TransformerUtils.invokerTransformer("trim")); I highly recommend the Jakarta Commons Cookbook from O'Reilly.
