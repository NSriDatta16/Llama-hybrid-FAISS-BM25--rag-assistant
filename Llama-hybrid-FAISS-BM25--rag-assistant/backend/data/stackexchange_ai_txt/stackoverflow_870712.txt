A shorter form can be used: return (accountNumber == DBNull.Value) ? string.Empty : accountNumber.ToString() EDIT: Haven't paid attention to ExecuteScalar . It does really return null if the field is absent in the return result. So use instead: return (accountNumber == null) ? string.Empty : accountNumber.ToString()
