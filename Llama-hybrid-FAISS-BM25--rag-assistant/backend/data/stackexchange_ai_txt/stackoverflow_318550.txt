Add the function to the struct: struct DeviceInfo { std::wstring DevAgentVersion; std::wstring SerialNumber; WhatEverReturnType GetDeviceInfo() { // here, to your calculation. DevAgentVersion and SerialNumber are visible. } }; DeviceInfo d; WhatEverReturnType e = d.GetDeviceInfo(); Note the typedef struct { ... } name; pattern is not needed in C++. If you for some reason have to use a free function for this, use reference: WhatEverReturnType GetDeviceInfo(DeviceInfo &info) { // do your calculation. info.DevAgentVersion and info.SerialNumber are visible. } DeviceInfo d; WhatEverReturnType e = GetDeviceInfo(d);
