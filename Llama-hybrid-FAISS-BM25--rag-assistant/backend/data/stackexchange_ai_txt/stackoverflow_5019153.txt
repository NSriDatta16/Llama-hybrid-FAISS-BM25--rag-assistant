Your error codes seem to be in CSV format. So: On first use of application parse CSV file line by line using one of CSV parsing libs: http://opencsv.sourceforge.net/ Save every line of data as a row in database: http://developer.android.com/guide/topics/data/data-storage.html#db Create a simple GUI where you enter error code and it returns all data for this error code.
