I'm genuinely embarrassed by how easy this one was. $("*:visible").live('dragenter dragover',function(event){lightdz(event)}); $("#page").live('dragleave dragexit',function(event) { if(event.pageX == "0") dimdz(event); }); $("*:visible").live('drop',function(event){dimdz(event)}); #page is a page-wide container. If the dragleave event takes the dragged object outside of the browser window, event.pageX will have a value of 0. If it happens anywhere else, it'll have a non-zero value.
