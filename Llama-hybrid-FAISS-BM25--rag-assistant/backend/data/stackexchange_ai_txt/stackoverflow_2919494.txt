The containsAll() would not help if target has more elements than dest : target: [a,b,c,d] dest: [a,b,c] target.containsAll(dest) is true, so dest is [a,b,c] but should be [a,b,c,d]. I think the following code is more elegant: Set uniqueSet = new LinkedHashSet (target.size()); uniqueSet.addAll(target); if(uniqueSet.contains("")) uniqueSet.remove(""); dest.addAll(uniqueSet);
