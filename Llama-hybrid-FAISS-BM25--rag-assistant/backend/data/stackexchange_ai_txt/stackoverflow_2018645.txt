Depending on how much control you have over the server, you might find it useful to install mod_security (Apache module). mod_security acts as a firewall for Apache, allowing you to block requests that match (or do not match) a set of criteria (including user agent, referring site, etc.). Here is a blog post that has information on using mod_security to deal with referral spam: http://atomicplayboy.net/blog/2005/01/30/an-introduction-to-mod-security/
