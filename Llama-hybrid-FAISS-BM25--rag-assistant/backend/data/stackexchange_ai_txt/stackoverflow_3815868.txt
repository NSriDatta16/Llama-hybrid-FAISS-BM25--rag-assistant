How often does the report data change? If for example you were using SQL Server, you could have a stored procedure to create the report data to a table and have the stored procedure run as a SQL Server Agent job. Set the job to run a frequently as you want the data updated. This should speed up your pages considerably. Sounds like you may have both requests on the same thread?
