For my own project, I wrote a fabric python files that manage all the unit testing. Since I am using django, I have django incorporate with nose and coverage. Here are my part of my scripts: local('%(local_virtual_env)s/Scripts/activate.bat' ' & cd %(local_project_path)s/configs/common' ' & python manage.py test test1 ' '--settings=myprofile.configs.local.settings_local ' '--with-coverage --with-xunit --with-xcoverage ' '--cover-tests --cover-erase --cover-inclusive ' '--cover-package=myprofile.apps' % env, capture=False)
