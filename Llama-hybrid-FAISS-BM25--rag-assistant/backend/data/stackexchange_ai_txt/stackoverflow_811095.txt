appendTo Element, SelectorDefault:'parent' The element passed to or selected by the appendTo option will be used as the draggable helper's container during dragging. By default, the helper is appended to the same container as the draggable. Code examples Initialize a draggable with the appendTo option specified. $('.selector').draggable({ appendTo: 'body' });
