Quick solution Datatables should be partical classes as far as I remember Create a 2nd file for your datatable MyDataTable.custom.cs Add a string property in the partial datatable class called "DisplayProperty" In that property return a string.format("{0} {1} {2}", Auftragsnummer, Kunde, Beschreibung); Bind your Datamember to your DisplayProperty
