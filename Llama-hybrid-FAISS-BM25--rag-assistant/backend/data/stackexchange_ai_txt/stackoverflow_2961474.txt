The best way is to pass everything in an associative array. class Template { public function render($_page, $_data) { extract($_data); include($_page); } } To build the template: $data = array('title' => 'My Page', 'text' => 'My Paragraph'); $Template = new Template(); $Template->render('/path/to/file.php', $data); Your template page could be something like this: Extract is a really nifty function that can unpack an associative array into the local namespace so that you can just do stuff like echo $title; . Edit: Added underscores to prevent name conflicts just in case you extract something containing a variable '$page' or '$data'.
