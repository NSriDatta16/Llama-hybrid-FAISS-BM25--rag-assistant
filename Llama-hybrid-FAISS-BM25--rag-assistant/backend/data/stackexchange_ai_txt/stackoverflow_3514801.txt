A database would be a good fit. SQLite is good as others have mentioned. You could also use a local instance of SQL Server Express to take advantage of improved integration with other pieces of the Microsoft development stack (since you mention C#). A third option is a document database like Raven which may fit from the sounds of your data. edit A fourth option would be to try Lightswitch when the beta comes out in a few days. (8-23-2010) /edit There is always going to be a limitation on data storage (the empty space of the hard disk). According to wikipedia, SQL Express is limited to 10 GB for SQL Server Express 2008 R2
