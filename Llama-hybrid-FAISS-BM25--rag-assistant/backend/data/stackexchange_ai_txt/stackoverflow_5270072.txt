To be sure that all migrations are always done using INNODB, you should set the storage engine as INNODB in the database definition directly, like this : DATABASES = { 'default': { ... 'OPTIONS' : { 'init_command' : 'SET storage_engine=INNODB', }, } If you are using MySQL 5.7.x and above, DATABASES = { 'default': { ... 'OPTIONS' : { 'init_command' : 'SET default_storage_engine=INNODB', }, } But you should know that it can have a performance hit . So you may want to set this option only when running migrations.
