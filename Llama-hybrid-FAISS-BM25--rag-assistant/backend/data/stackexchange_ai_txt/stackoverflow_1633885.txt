You can't do it with css, because div#container element didn't get any class triggered from draggables. Best way is to do it with callback: $('.draggable').draggable({ start: function(event, ui) { $('#container').addClass('expand'); }, stop: function(event, ui) { $('#container').removeClass('expand'); } });
