As your category is data derived from existing data, the use of a new column in the database for it should be discouraged. Instead, you could just add a case statement to your database query to work out the category as required or put it into a view in the database. The query would look something like this: select ... case when Age 65 then 'over65' else 'normal' end AS Age_Category from ...
