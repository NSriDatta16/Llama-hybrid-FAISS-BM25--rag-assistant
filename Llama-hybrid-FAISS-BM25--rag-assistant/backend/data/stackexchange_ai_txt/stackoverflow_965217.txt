$("div:has(p)").addClass("has-paragraph"); will add a class to all divs that contain p children. If the paragraphs aren't direct children, this won't pick them up. Note: doing this: $("div p").addClass("has-paragraph"); will add the class to the paragraph not the div. You can fix that by doing this: $("div p").parents("div").addClass("has-paragraph"); but this will catch multiple div ancestors. You could do this: $("div p").parents("div:first").addClass("has-paragraph"); to fix that. Or more programmatically: $("div").each(function() { if ($(this).find("p").length > 0) { // do stuff } });
