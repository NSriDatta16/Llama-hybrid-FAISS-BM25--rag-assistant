Check out this wikipedia article . The reason why JSON is 'safe' is because you have to pass it through a callback. The remote site will run return JSON and your javascript library will not just run it blindly but try to pass it to a function you specify, like jsonpCallback( response ) . Since you aren't running the remote code directly much more is under your control and all is mostly well in the world.
