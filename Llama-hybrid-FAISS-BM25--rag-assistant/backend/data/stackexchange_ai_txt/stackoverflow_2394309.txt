If you change facebook/__init__.py line 1292+ from this: if params.get('expires'): self.session_key_expires = int(params['expires']) To this: if params.get('expires'): if params['expires'] == 'None': params['expires'] = 0 self.session_key_expires = int(params['expires']) It will work, but it is a hack and maybe it could be done more elegantly, but it works.
