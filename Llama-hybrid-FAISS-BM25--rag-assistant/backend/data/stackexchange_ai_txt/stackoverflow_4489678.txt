I wouldn't take that test as a benchmark. If you try the same test on different browsers, it might give you a different result. (I got better fps on flex then on silverlight using firefox, also when going to the back button of the browser and waiting a bit would slow down silverlight.) As Kelsey Rider said, do some tests for your particular app and try use the average setup of your target group.
