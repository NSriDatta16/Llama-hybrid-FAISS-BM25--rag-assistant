I was able to solve my problem the following way: instead of using the stanfordparser ruby wrapper (which implements java ruby bridge to connect the java stanford parser to pure ruby), I use jruby to just implement the java from the stanford parser. the code that ended up working: include Java require 'C:\\Stanford-Parser\\Current\\stanford-parser.jar' require 'rubygems' include_class 'edu.stanford.nlp.parser.lexparser.LexicalizedParser' lp = LexicalizedParser.new(args) #args is the arguments, not copied here
