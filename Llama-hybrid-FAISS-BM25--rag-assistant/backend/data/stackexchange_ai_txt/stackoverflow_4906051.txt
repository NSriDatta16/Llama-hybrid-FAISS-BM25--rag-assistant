I also offer an alterntaive soultion. This is not an exact answer to your question though, but may help you to solve your problem in an elegant way. Subclass TabWidget and define how the row with tabs is drawn. Create a layout file for TabHost with your TabWidget in it. Subclass TabActivity and set your own layout you've created. Read the saved settings and use setDefaultTab method to set the preferred activity. Voila! You got your own look without losing any functionality of TabActivity . You can read Tab Layout Tutorial for more information on how to do it properly.
