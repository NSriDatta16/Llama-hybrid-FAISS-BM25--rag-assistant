You can do it with a suquery. But I don't know about performance. UPDATE `houses` AS h SET `avg_price` = ( SELECT AVG(`price`) FROM `houses` AS h2 WHERE h2.area == h.area ) Or you could create a temporary table in which to put the average prices and then update.
