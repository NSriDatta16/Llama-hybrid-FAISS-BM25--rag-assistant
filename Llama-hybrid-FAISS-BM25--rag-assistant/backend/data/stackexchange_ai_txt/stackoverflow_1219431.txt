I normally do this: // In header file: #include namespace mine { struct agent { std::string name; }; } // In source file: using std::string; namespace mine { // Code } That way you don't have to write out std:: over and over again in the implementation, but you avoid the problem of people who use your headers inadvertently importing std:: symbols.
