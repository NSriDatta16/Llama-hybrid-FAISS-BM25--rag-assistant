If you are using AS3, ExternalInterface.call is what you want. In the script tag in the html page: function theJSMethod() { $("div.boxclose").addClass("boxopen").show("slow"); } In flash: ExternalInterface.call("theJSMethod"); If you are using AS2, you can use fscommand to achieve this. Make sure allowScriptDomain in the embedding html code is set appropriately.
