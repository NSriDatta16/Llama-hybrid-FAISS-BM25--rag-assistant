There is no way. It can't be encrypted because the browser has to execute it and he doesn't know how to decrypt. Best is to minify/obfuscate (see anax) in order to let users don't bother trying to read the code. If someone wants it badly, he eventually get it. The time to do so is high, and hopefully this will dis-encourage users.
