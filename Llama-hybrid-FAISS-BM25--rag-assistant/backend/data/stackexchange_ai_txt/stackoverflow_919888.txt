I'm getting this to work by a combination of the suggestions above: Set meta tag cache-control/pragma http-equiv attributes to 'No-Cache' Use an ASP.NET page to host the silverlight control (as opposed to an html page) Set the Source property of the ASP.NET Silverlight control in the code behind, appending a time stamp to the .xap url e.g. Silverlight1.Source = "ClientBin/MyApplication.xap?" + DateTime.Now.ToString("dd-MM-yy-HH:mm:ss");
