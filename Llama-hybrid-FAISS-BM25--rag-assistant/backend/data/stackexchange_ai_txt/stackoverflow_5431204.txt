The GlobalMemoryStatus function provides information about the computer's random access memory (or, RAM): This function gets information on the physical and virtual memory of the system. The GetDiskFreeSpaceEx function provides information about the computer's hard disk drive , or other fixed storage medium: This function obtains the following information about the amount of space available on a disk volume: the total amount of space, the total amount of free space, and the amount of free space available to the user associated with the calling thread. Ask more questions about the difference between those two things on Super User .
