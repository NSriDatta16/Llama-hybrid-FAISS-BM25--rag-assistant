In C++, if a template argument has limited visibility, this restriction is implicitly propagated to the template instantiation. #include class __attribute__((visibility ("hidden"))) A {}; void doSomething() { std::map m; } should do the job. -- edit -- One more thing, `#pragma GCC visibility' affects only namespace-scope declarations. Class members and template specializations are not affected ( Visibility pragmas )
