If you are using Visual C++, you can use #pragma message : #include #pragma message("BOOST_VERSION=" BOOST_PP_STRINGIZE(BOOST_VERSION)) Edit: Thanks to LB for link Apparently, the GCC equivalent is (not tested): #pragma message "BOOST_VERSION=" BOOST_PP_STRINGIZE(BOOST_VERSION)
