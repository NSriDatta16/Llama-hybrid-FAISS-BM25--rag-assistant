Looking at the demo on the nettuts site: Demo ui.item is already a jQuery object so you can reference it as such. 0-based index position of the column : ui.item.parent().parent().children().index(ui.item.parent()) 0-based index position of the item : ui.item.parent().children().index(ui.item); so if you were to drag an item underneath the "Welcome to iNettuts" in the first column the above code would return 0 for column (so the first column) and 1 for the item position (so the second item in the column)
