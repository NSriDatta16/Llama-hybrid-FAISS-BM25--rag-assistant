Paraphrased from Unix and Color Highlighting : To be able to use syntax highlighting on a Unix/Linux box you need either a terminal that supports color. Some systems (AIX for me) require a TERMINFO file generated by tic on a xterm-color.tic file. Here's a snip: setenv TERMINFO /home/cfl/lib/terminfo tic xterm-color setenv TERM xterm-color Again, read the Unix and Color Highlighting article for all the details.
