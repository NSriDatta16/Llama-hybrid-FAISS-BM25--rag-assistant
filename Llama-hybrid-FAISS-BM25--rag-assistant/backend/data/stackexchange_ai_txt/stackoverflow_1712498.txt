In most cases, you'd roll your own. Take a look at PHP's $_REQUEST documentation and write this data into a SQL table. You can do this portion very easily in 20 lines or less, and if you're clever - probably 5 or so. Now, displaying that data can be done in any method you so choose, since you've got all the data in an SQL table. Sort, filter, and organize using any method you please.
