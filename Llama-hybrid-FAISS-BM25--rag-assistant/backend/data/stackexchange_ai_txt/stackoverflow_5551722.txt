Actually you can't. Codemirror2 is written in the way that all implementation is hidden in closures. Public methods which can be used are described in documentation http://codemirror.net/manual.html The only available options are to use anothe syntax highlighters or dive into the code of CodeMirror2 to strip necessary parts out. If you will chose last option, please give attention to function refreshDisplay(from, to) method it loops through lines and highlights them.
