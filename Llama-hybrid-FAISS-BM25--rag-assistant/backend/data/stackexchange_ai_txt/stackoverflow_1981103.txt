Do you really need them running on separate threads? I'm thinking since you are using Workflow already it should be easiest to solve the problem by using workflow to 'organize your work'. { var ArgsToProcess = new List { "arg_one", "arg_two", "arg_three" }; var delegateArg = new DelegateInArgument { Name = "s" }; Activity toRun = new ParallelForEach { Body = new ActivityAction { Argument = delegateArg, Handler = new Workflow1() //Plug your workflow here { Arg = delegateArg } } }; WorkflowInvoker.Invoke(toRun, new Dictionary { {"Values", ArgsToProcess} }); }
