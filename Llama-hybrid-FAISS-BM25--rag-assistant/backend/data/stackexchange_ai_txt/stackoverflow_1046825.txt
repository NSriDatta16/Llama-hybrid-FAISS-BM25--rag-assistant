There are several ways to add things to the PYTHONPATH . Read http://docs.python.org/library/site.html Set the PYTHONPATH environment variable prior to running your script. You can do this python -m agent to run agent.py from your PYTHONPATH . Create .pth files in your lib/site-packages directory. Install your modules in lib/site-packages .
