In data science - Data singular is a single value of any variable. Data plural is all the values of a variable. for example:- A 5, 6, 3, ... B 4, 3, 9, ... Here Column A and Column B are the variables of data set. 5 is the single value of variable A (data singular) All the values of variable A (data plural)
