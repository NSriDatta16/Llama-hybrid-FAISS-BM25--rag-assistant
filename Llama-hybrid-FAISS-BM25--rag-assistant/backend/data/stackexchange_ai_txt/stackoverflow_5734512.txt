Perhaps the elegance or speed of this could be improved, but it's the straightforward solution: def months(start_month, start_year, end_month, end_year): month, year = start_month, start_year while True: yield month, year if (month, year) == (end_month, end_year): return month += 1 if (month > 12): month = 1 year += 1 EDIT: And here's a less straightforward one that avoids even needing to use yield by using a generator comprehension: def months2(start_month, start_year, end_month, end_year): return (((m_y % 12) + 1, m_y / 12) for m_y in range(12 * start_year + start_month - 1, 12 * end_year + end_month))
