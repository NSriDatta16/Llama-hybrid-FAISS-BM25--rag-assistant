One is an average while the other is a sum. Otherwise, they're the same, just using different symbols. Because they're the same (aside from the scalar $\frac{1}{m}$ ), there's no mathematical distinction between one and the other. Doing arithmetic on a computer would require special care, because floating point calculations can be fragile.
