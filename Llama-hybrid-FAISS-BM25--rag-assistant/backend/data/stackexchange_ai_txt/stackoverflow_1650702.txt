The number after binary() is the number of bytes, see MSDN : binary [ ( n ) ] Fixed-length binary data of n bytes. n must be a value from 1 through 8,000. Storage size is n+4 bytes. Whether 30kb is 30000 or 30720 bytes depends on which binary prefix system your file system is using.
