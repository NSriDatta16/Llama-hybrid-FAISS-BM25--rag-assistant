C# and .NET is a great choice, especially if you want a language that is a lot like Java. For starters, download Visual Studio Express (free), and you'll be able to create Windows Apps with C# and .NET. GUIs are as easy to create as Dragging controls on to a form. You can also use C# to assist in programming on the Linux platform with an open-source port of .NET, called Mono .
