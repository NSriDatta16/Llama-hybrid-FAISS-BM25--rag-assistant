Webstorage is perfect for chunks of data with that dimension and it' really fast retrieving data from the browser. You don't need any cache, because the data are on the browser, you can rather use those data with offline applications as well. I would suggest using pure JS, as it doesn't add any overhead or any dependencies and the WebStorage API are quite straightforward. If you wanna know more about WebStorage, try to read my article
