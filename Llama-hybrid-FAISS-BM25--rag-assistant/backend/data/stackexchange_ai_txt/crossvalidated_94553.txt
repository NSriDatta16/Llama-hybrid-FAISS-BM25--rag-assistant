In scikit-learn all classifiers implement the ClassifierMixin interface. In order to so they must provide the fit() method. The Logistic Regression classifier provides two methods to get the probability for each sample: predict_log_proba() and predict_proba() (see documentation on the Logistic Classifier for more details). There are a few examples using the Logistic regression model in the examples page
