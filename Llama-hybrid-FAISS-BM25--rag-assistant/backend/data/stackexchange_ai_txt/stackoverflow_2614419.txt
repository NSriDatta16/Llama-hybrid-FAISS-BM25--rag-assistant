It's somewhat unusual to pass HTML through an AJAX request; normally you pass a JSON string that the client can evaluate directly, and work with that That being said, I don't think there's a way to parse HTML in javascript the way you want that's cross-browser, but here's a way to do it in Mozilla derivatives: var r = document.createRange(); r.selectNode(document.body); var domNode = r.createContextualFragment(HTTPRequest.innerText);
