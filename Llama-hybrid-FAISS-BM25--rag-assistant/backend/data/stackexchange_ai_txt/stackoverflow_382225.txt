Some of my techniques include Documentation. This gets you thinking about your project. If it's not easy to describe, it's probably not elegant enough yet Static Code Analysis - FxCop , Lint , Cyclomatic Complexity , Security Analyzers. Now is a good time to step back from your code and check on best practices Unit Tests. This again gets you thinking about the code and how to improve it.
