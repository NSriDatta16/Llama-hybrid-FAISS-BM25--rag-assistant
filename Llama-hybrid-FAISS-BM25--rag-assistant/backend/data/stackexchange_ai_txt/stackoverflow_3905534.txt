If you don't want to call setHeader() for every request you create you can set the http client parameter CoreProtocolPNames.USER_AGENT . After doing this HTTP client will automatically add this header parameter to each request. Something like: client.getParams().setParameter(CoreProtocolPNames.USER_AGENT, "Custom user agent"); when you create your HttpClient .
