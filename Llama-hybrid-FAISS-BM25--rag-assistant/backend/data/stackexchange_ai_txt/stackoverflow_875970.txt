If you want to do this is ruby, The Mechanize gem would be perfect for this ` require 'mechanize' agent = WWW::MECHANIZE.new page = agent.get('localhost/path/to/form') login_form = page.forms.first #assuming the first form is the one we want login_form.username = 'myusername' login_form.password = 'mypassword' page = agent.submit(login_form) puts page.body # just to see the results `
