Following is the link for the Python wrapper for CMU tweet tagger, https://github.com/ianozsvald/ark-tweet-nlp-python Its a Simple Python wrapper around runTagger.sh of ark-tweet-nlp. It passes a list of tweets to runTagger.sh and parses the result into a list of lists of tuples, each tuple represents the (token, type, confidence). Download the ark-tweet-nlp jar to use it.
