Create your own subclass of NSApplication, and implement something like this: - (int)requestUserAttention:(NSRequestUserAttentionType)requestType { if (dontDoThatBouncyThing) { return 0; } return [super requestUserAttention:requestType]; } Don't forget to change "NSPrincipalClass" in your Info.plist from NSApplication to your own NSApplication subclass.
