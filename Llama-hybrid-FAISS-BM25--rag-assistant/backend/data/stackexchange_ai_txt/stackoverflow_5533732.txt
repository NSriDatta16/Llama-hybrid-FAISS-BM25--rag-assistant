Since both the ViewData and ViewBag use the same storage pattern, you should be able to use ViewData[yourKey] in your tests. So your test will look like this: [TestCase] public void About() { var controller = new AboutController(); var ar = controller.About() as ViewResult; Assert.AreEqual("About Us", ar.ViewData["Title"]); }
