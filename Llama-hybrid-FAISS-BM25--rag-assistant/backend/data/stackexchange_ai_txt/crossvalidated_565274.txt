The use of linear returns (percentage change) and log returns are both used in financial applications. Two arguments for using log returns for time series modelling are The distribution of log returns can unlike linear returns easily be project to any horizon Log returns typically have a symmetric distribution which makes modelling easier (stock prices are often assumed to be log normally distributed - log-returns follow a normal distribution)
