Did you look at http://sds.sourceforge.net/src/antlr/doc/lexer.html ? A possible solution is to define the following: fragment INT : DIGIT+ ; fragment RANGE : INT DOTDOTSEP INT ; fragment FLOAT : INT (('.' INT (('e'|'E') INT)? 'f'?) | (('e'|'E') INT)? ('f')) ; NUMBER : (INT '.') => FLOAT { $type=FLOAT; } | (INT DOTDOTSEP) => RANGE { $type=RANGE; } | INT { $type=INT; } ;
