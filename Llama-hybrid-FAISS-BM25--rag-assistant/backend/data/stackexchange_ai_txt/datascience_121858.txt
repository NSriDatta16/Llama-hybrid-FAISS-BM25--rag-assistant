You have mentioned X_train shape is (1400, 64, 35), So we can create a LSTM model whose input size will be (64,35) And you can take the number of units in LSTM as per your choice. from tensorflow.keras.models import Sequential from tensorflow.keras.layers import LSTM, GlobalAveragePooling1D, Dense # Define the model model = Sequential() # Add the LSTM layer model.add(LSTM(units=64, input_shape=(64, 35), return_sequences=True)) # Add the GlobalAveragePooling layer model.add(GlobalAveragePooling1D()) # Add the Dense layer model.add(Dense(units=10, activation='softmax')) # Compile the model model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy']) # Print the model summary model.
