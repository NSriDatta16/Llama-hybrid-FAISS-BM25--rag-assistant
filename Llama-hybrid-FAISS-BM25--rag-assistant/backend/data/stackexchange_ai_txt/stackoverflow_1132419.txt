To do a local upload (rather than a web upload via the browser) you just have to call the putContents method passing the files data. Example: $nirvanix = new Zend_Service_Nirvanix(array('appKey' => $key, 'username' => $user, 'password' => pass)); $NSImfs = $nirvanix->getService('IMFS'); $response = $NSImfs->putContents($destination_file_and_path, file_get_contents($source_file)); if($response->ResponseCode != 0) { echo 'Fail!'; } You would only call GetStorageNode if you want to generate a token to pass a browser the upload token.
