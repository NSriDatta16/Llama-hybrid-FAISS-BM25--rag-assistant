Use the fragment of the URL to do this. That way you can modify the URL in javascript without reloading the page. Then, when you hit refresh, that fragment sticks with the URL. So, each time the dropdown value changes, set the value in the url like this: window.location.hash = '#somevalue' Then, when the page loads: $(document).ready(function() { // access hash using window.location.hash and show/hide based on its value }
