It's saying that it can't find that path. So, does it exist on your machine? Note that the slashes are backslashes - does your OS support backslash as a file separator? Also note that it's an absolute path - is that intended? If all else is OK, does the file exist? Edit: if not, you should download it here ( http://github.com/tiendung/ruby-nlp/blob/master/left3words-wsj-0-18.tagger ), place it in the path that the system is specifying, and see what happens.
