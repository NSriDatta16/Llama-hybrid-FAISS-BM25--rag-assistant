You have two options here: Get rid of the margin on the p tag: p { margin: 0; } Use a div instead of a p Most browsers render paragraph tags with a 1em top and bottom margin. As for your problems with the rtl property, what browser are you using? As far as I'm aware it should work fine with as long as the element is inline and you're using the correct unicode characters.
