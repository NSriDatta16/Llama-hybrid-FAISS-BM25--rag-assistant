For organizational purposes namespaces have been added to php 5.3+. Static methods have access to protected/private members of an object of that class. e.g. class Foo { protected function xyz() { return 'abc'; } public static function bar(Foo $f) { echo $f->xyz(); } } $f = new Foo; Foo::bar($f); And you can use the same visibility options to share static properties between different methods of the same class but protect them from the "outside". E.g. class Foo { protected static $abc; public static function set($x) { self::$abc = $x*2; } public static function get() { return self::$abc; } } Foo::set(1); echo Foo::get();
