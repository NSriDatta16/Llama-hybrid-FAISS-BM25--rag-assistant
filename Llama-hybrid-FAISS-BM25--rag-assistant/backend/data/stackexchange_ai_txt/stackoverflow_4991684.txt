Paperclip is now deprecated . The library authors recommend switching to ActiveStorage . They have provided a migration guide as well. Paperclip is intended as an easy file attachment library for ActiveRecord. The intent behind it was to keep setup as easy as possible and to treat files as much like other attributes as possible. For more information: Paperclip repository Paperclip documentation
