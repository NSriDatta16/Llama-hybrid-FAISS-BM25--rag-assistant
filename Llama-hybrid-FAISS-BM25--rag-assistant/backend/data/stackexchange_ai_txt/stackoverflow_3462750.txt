HAML's elegant solution is helpers class ApplicationHelper... def body_for_user(user, &blk) return content_tag(:body, :'data-account' => user.account, &blk) if user content_tag(:body, &blk) end end The ternary operators described above are more than sufficient for this particular situation, but for more complex things, break out the helpers. Oh, to use this, in your view change %body(...) to = body_for_user @user do
