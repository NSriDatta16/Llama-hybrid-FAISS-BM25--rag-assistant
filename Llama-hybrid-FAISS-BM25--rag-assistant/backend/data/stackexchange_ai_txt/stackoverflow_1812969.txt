This process should be a two commit process. Create the destination folder in the root folder of your working copy. Commit the changes (ie. Adding new folder). Select all other files you want to move into the new folder. Right click and hold the selected group of files. Drag the your cursor to the new folder, release. In the menu that results, select SVN Move. Commit your changes
