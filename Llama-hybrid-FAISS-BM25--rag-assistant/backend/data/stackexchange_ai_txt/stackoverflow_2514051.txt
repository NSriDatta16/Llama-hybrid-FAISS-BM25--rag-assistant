NSHTTPURLResponse is a subclass of NSURLResponse so you can just ask it for its URL . This returns an NSURL object from which you can get the URL components such as the query, parameter string or fragment. In your case you want the fragment component: NSURL* url = [response URL]; NSString* fragment = [url fragment]; //fragment should be: 0;sid=865a84f0212a3a35d8e9d5f68398e535
