Since you are using Windows Application, your best bet is isolated storage. [Old Answer] In case you are using a web application: If you want to do save the string till the user is logged in then you can store it in the Session state, if you want to store it for the lifetime of the web application in IIS then you can use Application State Of course you can store the same in a database if required.
