All you need to do is invoke dragtable.makeDraggable(tableElement);​​​ on a table dom node after it has been injected. Consider the following example code: HTML ​ JavaScript (the last line here is the crucial one) var tableStr = ' Name Date Color Dan 1984-07-12 Blue Alice 1980-07-22 Green Ryan 1990-09-23 Orange Bob 1966-04-21 Red ', tableDiv = document.getElementById('tableDiv'), table; tableDiv.innerHTML = tableStr; table = document.getElementById('table'); dragtable.makeDraggable(table);​​​ Check out a working demo of this code here .
