For C# public static void cacheOutput(Page page, double minutes) { page.Response.Cache.SetExpires(DateTime.Now.AddMinutes(minutes)); page.Response.Cache.SetCacheability(HttpCacheability.Server); page.Response.Cache.SetValidUntilExpires(true); } It's better to take a double for minutes.
