in Permission to write to the SD card : You're right that the SD Card directory is /sdcard but you shouldn't be hard coding it. Instead, make a call to Environment.getExternalStorageDirectory() to get the directory: File sdDir = Environment.getExternalStorageDirectory(); If you haven't done so already, you will need to give you app the correct permission to write to the SD Card by adding this to your Manifest: Just change it to read (instead of write)
