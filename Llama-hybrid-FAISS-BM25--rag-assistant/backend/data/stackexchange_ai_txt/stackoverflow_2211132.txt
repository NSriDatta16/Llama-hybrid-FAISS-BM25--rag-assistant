I wouldn't do it via browser sniffing (i.e. either directly or via a Javascript framework), because a User Agent string could easily be forged (and, in these cases, is depending on JavaScript, which could be turned off). In this case (IE or not), I would use conditional comments in your HTML. They'll always work, whether JavaScript is enabled or not.
