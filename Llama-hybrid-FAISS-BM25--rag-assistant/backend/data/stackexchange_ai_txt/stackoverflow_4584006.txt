OpenMP version (non-ordered of course): #include #include int main(int argc, char** argv) { #pragma omp parallel num_threads(1000) { #pragma omp critical { std::cout (Does not work with VS2010 OpenMP runtime (restricted to 64 threads), works however on linux with, e.g., the Intel compiler) Here's an ordered version too: #include #include int main(int argc, char *argv[]) { int i = 1; #pragma omp parallel num_threads(1000) #pragma omp critical printf("%d ", i++); return 0; }
