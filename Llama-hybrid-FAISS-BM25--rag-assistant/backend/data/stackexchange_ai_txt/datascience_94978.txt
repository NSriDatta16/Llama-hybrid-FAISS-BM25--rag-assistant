You should try to frame as a prediction problem. Clearly define the target and then map that target to a technique: Mean time before failure (Survival analysis) Failure rates per million hours (Estimate Poisson distribution) Failure within a time window (Binary classification) Generally, Bayesian methods are used because the sparse available data can be used as a prior.
