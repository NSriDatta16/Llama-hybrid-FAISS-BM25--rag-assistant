Looks like it only happens when the overriding method also has the attribute . http://msdn.microsoft.com/en-us/library/a19191fh.aspx However, you can override attributes of the same type or apply additional attributes to the derived component. The following code fragment shows a custom control that overrides the Text property inherited from Control by overriding the BrowsableAttribute attribute applied in the base class. Visual Basic Public Class MyControl Inherits Control ' The base class has [Browsable(true)] applied to the Text property. _ Public Overrides Property [Text]() As String ... End Property ... End Class
