I would iterate over each row, and the parse each for the label and input value I cared about. Note that in the event you have duplicate label names, the object storage will lose values. var arr_storage = []; // var to store as array var obj_storage = {}; // var to store as object $('#metric_summary tr').each(function(){ var key = $('td label:first', this).text() var value = $('td input:first', this).val(); arr_storage.push([ key, value ]); obj_storage[key] = value; });
