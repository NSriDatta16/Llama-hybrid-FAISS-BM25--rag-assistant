Use CAST(text AS XML) to obtain an XML typed column that can be manipulated at the server (use value, nodes and query xml methods on it). SELECT ID, CAST(Request AS XML) As RequestXml, CAST(Request AS XML) As ResponsetXml, MerchantId, Amount FROM If you only need the XML in the client, then you can just return the text and then use whatever is your client side XML technology of choice (XmlDocument, XPathDocument, Linq 2 xml) they all allow you to cosntruct a fragment from a string.
