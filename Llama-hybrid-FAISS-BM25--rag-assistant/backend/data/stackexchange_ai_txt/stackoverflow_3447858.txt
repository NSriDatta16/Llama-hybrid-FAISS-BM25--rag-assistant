A solution could be to start an EnterFrame when you start drag first MC. You will stop the EnterFrame event when you stop dragging firstMc. function onEnterFrame(e:Event):void{ secondMc.x=firstMc.x; } You could also override the x setter of the first MC... override public function set x(value:Number):void{ super.x=value; secondMc.x=value; }
