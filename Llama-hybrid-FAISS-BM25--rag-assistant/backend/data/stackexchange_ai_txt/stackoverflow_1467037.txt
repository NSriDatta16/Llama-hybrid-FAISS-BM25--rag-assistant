What you need to do is to have a test to see if a particular address lives inside a particular address range as defined by CIDR So for instance, you need to be able to say is 192.168.1.5 inside 192.168.1.0/24 That function is easy to write, assuming you have some basic tools to do CIDR work. Assuming you are on a 32bit system, this class http://snipplr.com/view/15557/cidr-class-for-ipv4/ Pay attention to the IPisWithinCIDR function
