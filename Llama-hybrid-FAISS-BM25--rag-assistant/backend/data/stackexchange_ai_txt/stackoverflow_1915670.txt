You don't need to modify System.Data.SQLite if you want to use the latest version of SQLite, just use the ManagedOnly version of System.Data.SQLite and then only replace the sqlite3.dll with the latest version. For enabling foreign key support I simply execute a sql statement that enables foreign key support. e.g. string databasePath = "Your database path here"; string connectionString = "Data Source=" + databasePath; connection = new SQLiteConnection(connectionString); connection.Open(); const string sqlString = "PRAGMA foreign_keys = ON;"; SQLiteCommand command = new SQLiteCommand(sqlString, connection); command.ExecuteNonQuery();
