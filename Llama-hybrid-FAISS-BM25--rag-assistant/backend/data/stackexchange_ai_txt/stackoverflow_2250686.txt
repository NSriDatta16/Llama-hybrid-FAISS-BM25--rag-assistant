Something along the lines of from pcapy import open_offline from impacket.ImpactDecoder import EthDecoder from impacket.ImpactPacket import IP, TCP, UDP, ICMP decoder = EthDecoder() def callback(jdr, data): packet = decoder.decode(data) child = packet.child() if isinstance(child, IP): child = packet.child() if isinstance(child, TCP): if child.get_th_dport() == 80: print 'HTTP' pcap = open_offline('net.cap') pcap.loop(0, callback) using http://oss.coresecurity.com/projects/impacket.html
