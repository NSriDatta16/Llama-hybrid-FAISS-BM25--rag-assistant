The code below will save an int into the Isolated Storage, and then read it back out in the loadScore method. private void saveScore(int score) { IsolatedStorageSettings.ApplicationSettings[highScoreTag] = score; IsolatedStorageSettings.ApplicationSettings.Save(); } private int loadScore() { if (IsolatedStorageSettings.ApplicationSettings.Contains(highScoreTag)) { return (int)IsolatedStorageSettings.ApplicationSettings[highScoreTag]; } else return 0; } Also, check out Nick Gravelyn's blog .
