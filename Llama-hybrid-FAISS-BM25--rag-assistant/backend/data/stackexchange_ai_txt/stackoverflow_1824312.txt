From the The Pragmatic Programmer's Guide : Multiple parameters passed to a yield are converted to an array if the block has just one argument. For example: > fields = {:a => "foo", :b => "bar"} > fields.map { |a| a } # => [[:a, "foo"], [:b, "bar"]] So your code could be simplified like this: > fields.map{ |a| a.join('=') }.join('&') # => "a=foo&b=bar"
