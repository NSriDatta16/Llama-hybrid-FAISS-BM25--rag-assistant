As it was stated here, you should check size of block. You do this with stat family functions. In struct stat this information is located in field st_blksize. Second thing is function posix_fadvise(), which gives advice to OS about paging. You tell system how you're going to use file (or even fragment of a file). You'll find more on manual page.
