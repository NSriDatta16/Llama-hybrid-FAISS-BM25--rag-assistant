How about something like this: [ { "heading": "Foobar Example" }, { "paragraph": [ "This is normal text, followed by... ", { "bold": "some bold text" }, "etc." ] } ] That is: use a string for plain text without formatting or other mark-up; use an array whenever you want to indicate an ordered sequence of certain text elements; use an object where the key indicates the mark-up and the value the text element to which the formatting is applied.
