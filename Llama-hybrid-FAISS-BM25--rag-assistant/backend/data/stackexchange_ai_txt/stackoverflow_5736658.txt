A safe (more restrictive) alternative to using eval is ast.literal_eval : import ast pwm = raw_input("enter PWM: ") y = list(ast.literal_eval(pwm)) print(y) Running % test.py enter PWM: (7), (2), (1), (1), (6), (2), (1), (2), (1), (2), (2), (6) yields [7, 2, 1, 1, 6, 2, 1, 2, 1, 2, 2, 6] Note doing it this way avoids the fragile (and hard-to-read) replace('),(',',') , which only works when there is no space after the comma. E.g. (7),(2) instead of (7), (2) .
