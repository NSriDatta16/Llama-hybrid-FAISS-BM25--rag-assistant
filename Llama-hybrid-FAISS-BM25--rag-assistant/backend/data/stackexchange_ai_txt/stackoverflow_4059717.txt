nethogs On Ubuntu: sudo apt-get install nethogs sudo nethogs nethogs uses libpcap. To get the sources on Ubuntu and learn from that: apt-get source nethogs In particular the decpcap.c file is probably relevant. And these links might be helpful: http://www.gsp.com/cgi-bin/man.cgi?section=3&topic=pcap http://www.tcpdump.org/pcap.html http://libpcap.darwinports.com/ (Mac pcap install instructions)
