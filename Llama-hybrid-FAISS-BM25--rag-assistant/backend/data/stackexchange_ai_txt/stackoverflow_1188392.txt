Most likely the jQuery UI library's sortable functionality isn't that configurable (it's very rudimentary, meant for sorting lists one level deep). You'd be better off simply using one of the tree plug-ins that support drag and drop. The jQuery UI tree widget is in development , but you can use one of the following plugins: http://www.jstree.com/ http://abeautifulsite.net/notebook/58 http://news.kg/wp-content/uploads/tree/ http://bassistance.de/jquery-plugins/jquery-plugin-treeview/
