mvn dependency:resolve seems to be what you are looking for. Put following plugin configuration to your POM file: org.apache.maven.plugins maven-dependency-plugin list-dependencies prepare-package resolve dependencies.txt It will produce file dependencies.txt with content similar to: The following files have been resolved: am:amagent:jar:1.0:system am:amclientsdk:jar:1.0:system aopalliance:aopalliance:jar:1.0:compile asm:asm:jar:2.2.3:compile com.sun.jdmk:jmxtools:jar:1.2.1:compile com.sun.jmx:jmxri:jar:1.2.1:compile com.sun.xml.bind:jaxb-impl:jar:2.1.12:compile com.sun.xml.fastinfoset:FastInfoset:jar:1.2.2:compile com.sun.xml.messaging.saaj:saaj-impl:jar:1.3.2:compile commons-lang:commons-lang:jar:2.3:compile commons-logging:commons-logging:jar:1.1:compile dom4j:dom4j:jar:1.6.1:compile javax.activation:activation:jar:1.1:provided javax.jms:jms:jar:1.1:compile javax.mail:mail:jar:1.4:compile javax.xml.bind:jaxb-api:jar:2.1:compile javax.xml.soap:saaj-api:jar:1.3:compile junit:junit:jar:4.4:test log4j:log4j:jar:1.2.15:compile
