Just use PHP's parse_url function to do something like this: $url = 'http://username:password@hostname/path?arg=value#anchor'; //Replace the URL in your case with "http://".$_SERVER["SERVER_NAME"].$_SERVER["REQUEST_URI"] print_r(parse_url($url)); echo parse_url($url, PHP_URL_PATH) Which will return: Array ( [scheme] => http [host] => hostname [user] => username [pass] => password [path] => /path [query] => arg=value [fragment] => anchor ) /path In your case it would be more like: Array ( [scheme] => http [host] => localhost [path] => /final/home.php [query] => topic=jquery [fragment] => page2 )
