I'll add one more approach to the two above -- use a block to do the body of your iteration. __block NSUInteger longestWordSize = -1; // Make sure at least one object will be longer. __block NSUInteger longestWordIndex; [nameArray enumerateObjectsUsingBlock:^(id currentWord, NSUInteger index, BOOL *stop) { if ([currentWord length] > longestWordSize) { longestWordSize = [currentWord length]; longestWordIndex = index; } }]; NSLog("Longest word is %@ with size of :%d", [nameArray objectAtIndex:longestWordIndex], longestWordSize); Edit: The max and index have to be of storage type __block so they can be changed from inside the block.
