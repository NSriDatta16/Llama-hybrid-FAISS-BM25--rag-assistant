One approach in the Java world is to use the OpenSymphony Cache . This is a distributed cache (like memcached) that lets you store any Java objects and set expiration policies. If you use iBatis as your Java persistence manager, you can leverage its integration with OSCache. The Hibernate ORM also has an OSCache integration .
