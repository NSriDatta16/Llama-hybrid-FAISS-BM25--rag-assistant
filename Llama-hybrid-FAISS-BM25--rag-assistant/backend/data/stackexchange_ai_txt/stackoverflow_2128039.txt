First, using scriptlets (embedding raw Java code in JSP using the old fashioned things) is not the same as using taglibs (e.g. JSTL ) and EL (the ${} things). The performance penalty of taglibs and EL is really negligible and only noticeable during compilation and does by far not outweigh the advantages of using taglibs and EL instead of scriptlets .
