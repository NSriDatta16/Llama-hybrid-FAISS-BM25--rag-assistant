Subitems are only ordered by column index unluckily. So you'd have to access them like: protected override void OnItemDrag(ItemDragEventArgs e) { base.OnItemDrag(e); ListViewItem item = e.Item as ListViewItem; string val = item.SubItems[0].ToString(); }
