As of 2022, Hyndman is using fable::ARIMA, but in his otherwise excellent time series regression guide ( https://otexts.com/fpp3/ ) is still only showing how to do a linear regression with arima errors. However, it appears that as the fable package lets you specify the model via a formula, then you can do the following, which I think is a ARMAX(1,0,1) model (seems to work): ARIMA(y~lag(y)+x+pdq(p=0,d=0,q=1)
