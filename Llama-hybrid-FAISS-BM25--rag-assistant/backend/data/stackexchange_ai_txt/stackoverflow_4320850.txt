Try something like this: var query = @"select (select sum(pu.Amount) from Purchase pu where pu.People.Id = ppl.Id), (select sum(pa.Amount) from Payments pa where pa.People.Id = ppl.Id) from People ppl"; var results = session .CreateQuery(query) .List(); Or perhaps using ad-hoc mapping : public class BulkReport { public double Purchases { get; set; } public double Payments { get; set; } } var results = session .CreateQuery(query) .SetResultTransformer(Transformers.AliasToBean(typeof(BulkReport))) .List (); Another option would be using MultiQueries .
