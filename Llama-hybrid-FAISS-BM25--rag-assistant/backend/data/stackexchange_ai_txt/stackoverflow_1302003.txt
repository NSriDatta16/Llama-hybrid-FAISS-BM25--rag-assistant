First, I would use a tag rather than a division tag with a paragraph element inside of it. The tag is more suffice in what you are actually trying to do. Second, you are using the jQuery .css function incorrectly. According to the jQuery CSS Documents , the script should be like so: jQuery(document).ready(function($) { $("#mainButton").hover( // On hover over function() { $(this).css('background-color', '#FF0 '); }, // On hover out function() { $(this).css('background-color', '#333'); } ); });
