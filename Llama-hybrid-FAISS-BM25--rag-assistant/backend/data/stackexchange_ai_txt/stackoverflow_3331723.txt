You can get the values with .css() like this: var left = $(".soft_add_wrapper.ui-draggable").css('left'); var top = $(".soft_add_wrapper.ui-draggable").css('top'); If it's relative to the entire document and you want the overall position, you might consider using .offset() , or if you want relative to the parent, .position() . Note that the selector is .soft_add_wrapper.ui-draggable since you want to check both class selectors on the same element, there's no space.
