Supporting dynamic language changes is a tricky problem. The simplest solution is to organise your UI in such a way that you can recreate any visible panels. That avoids the need to register or update components, when the language changes you simply recreate the view. Obviously you lose the state of all visible components which may be an issue - but that is usually the same in a web app when the page is refreshed.
