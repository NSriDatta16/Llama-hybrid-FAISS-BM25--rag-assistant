Event handling in WP7 is generally handled on async callbacks. Accessing the accelerometer is no exception. You'll need to direct any code that results in a UI update to the dispatcher. In c# this can be done like Dispatcher.BeginInvoke( () => { /* your UI code */ } ); The approach for sending results to the Dispatcher used in this post may also be helpful for you in f# as it's more of a functional style than imperative as a result of using Rx. WP7 Code: Using the Accelerometer API - Dragos Manolescu's (work) blog
