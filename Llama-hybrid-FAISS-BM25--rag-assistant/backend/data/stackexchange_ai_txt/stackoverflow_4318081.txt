You can use a RegExp to replace the last 3 characters with a space plus the same 3 characters: $('#postcode').blur(function(){ $(this).val($(this).val().replace(/...$/, ' $&')); }); Try it: http://jsfiddle.net/PJMDp/1/ If you really must count 3 characters, you should use slice , not substring : $('#postcode').blur(function(){ var postcode = $(this), val = postcode.val(); if(val.length > 3) postcode.val(val.slice(0, -3)+' '+val.slice(-3)); }); Try it: http://jsfiddle.net/PJMDp/2/
