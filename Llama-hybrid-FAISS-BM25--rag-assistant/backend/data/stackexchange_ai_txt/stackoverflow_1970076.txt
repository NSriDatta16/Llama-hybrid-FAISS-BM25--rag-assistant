Use something like this: $( document ).ready ( function () { $.ui.dialog.defaults.bgiframe = true; $( "#box" ).dialog ( { modal: true, autoOpen: false, draggable: true, width: 600 } ); $( '#Button' ).click ( function () { $( '#box' ).dialog ( 'open' ); } ); } ); The catch is that you only init the dialog once, and then call .dialog ( 'open' ), when you actually want to open the dialog
