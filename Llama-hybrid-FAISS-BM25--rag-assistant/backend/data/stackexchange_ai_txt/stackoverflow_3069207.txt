I've found the comment by McDowell to be true - which MANIFEST.MF file gets picked up depends on the classpath and might not be the one wanted. I use this String cp = PCAS.class.getResource(PCAS.class.getSimpleName() + ".class").toString(); cp = cp.substring(0, cp.indexOf(PCAS.class.getPackage().getName())) + "META-INF/MANIFEST.MF"; Manifest mf = new Manifest((new URL(cp)).openStream()); which I adapted from link text
