Updated to address comment To bypass all certificate checks you can set the agent up to not verify the certificate. $agent->ssl_opts(verify_hostname => 0); The agent will also pass settings down to the SSL socket implementation being used. For example, with IO::Socket::SSL you can set SSL_verify_mode to 0x00 . $agent->ssl_opts(SSL_verify_mode => 0x00);
