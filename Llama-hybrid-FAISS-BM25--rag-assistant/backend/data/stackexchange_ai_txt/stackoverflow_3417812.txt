You can enable Drag Drop on the WebBrowser control via the "AllowWebBrowserDrop" property, i.e. webBrowser1.AllowWebBrowserDrop = true; Then you can intercept the "Navigating" event to read the URL that was dropped: private void webBrowser1_Navigating(object sender, WebBrowserNavigatingEventArgs e) { MessageBox.Show(e.Url.ToString()); }
