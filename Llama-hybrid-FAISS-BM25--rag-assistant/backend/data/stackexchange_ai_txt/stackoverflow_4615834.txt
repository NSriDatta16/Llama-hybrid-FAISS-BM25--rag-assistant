If the order is always the same, the following will reassign the values to the variables you want: $parts = preg_split("/'?\(|\)'?/", $original_values); $sub1 = explode(',', $parts[1]); $codes = trim(str_replace("'", '', $sub1[0])) ; $first_value = trim(str_replace("'", '', $sub1[1])); $second_value = trim(str_replace("'", '', $sub1[2])); $list_values = "($parts[2])"; $sub2 = explode(',', $parts[3]); $action = trim(str_replace("'", '', $sub2[1])); $timer = trim(str_replace("'", '', $sub2[2])); preg_match('/(\D+)(\d+)/', $parts[4], $matches); $case = $matches[1]; $status = $matches[2]; There might be a more elegant way to do it, but the original value is a string, rather than an array.
