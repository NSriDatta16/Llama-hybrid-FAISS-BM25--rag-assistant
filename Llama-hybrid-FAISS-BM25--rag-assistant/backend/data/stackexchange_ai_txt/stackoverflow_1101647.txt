From the calibre download page : Before trying to use the command line tools, you must run the app at least once. This will ask you for you password and then setup the symbolic links for the command line tools. The app cannot be run from within the dmg. You must drag it to a folder on your filesystem (The Desktop, Applications, wherever). In order for localization of the user interface in your language, select your language in the preferences (by pressing u8+P) and select your language. Have you done all three of these?
