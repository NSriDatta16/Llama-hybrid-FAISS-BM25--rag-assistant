You'll need to leverage Notes formulas instead of script to get your answer. Use the NotesSession.Evaluate method to call one of the following Notes @formulas: The @IsDocTruncated formula will return 1 if it is truncated and 0 if not. The @DocOmittedLength formula will return the approximate number of bytes that have been omitted.
