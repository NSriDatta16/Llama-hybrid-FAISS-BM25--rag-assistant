If you do build the integration code using .NET, you may want to consider leveraging the WCF Line-of-Business SDK: http://www.microsoft.com/biztalk/technologies/wcflobadaptersdk.mspx It's not a BizTalk-only technology, despite its categorization. The SDK is designed to make it easier to create a WCF channel to a LOB application, which can be consumed from almost any other platform.
