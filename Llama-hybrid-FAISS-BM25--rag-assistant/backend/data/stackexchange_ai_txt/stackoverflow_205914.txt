Something like this works, but it is just not elegant: Dim txtBox as TextBox = Ctype(parentControl.Controls(Index), System.Web.UI.Controls.TextBox) stringVariable = txtBox.Text The correct way to do it is to implement properties for your parentControl that access child's controls properties. Public Property AddressField() as string Set(byval value as string) txtAddressField.Text = value End Set Get Return txtAddressField.Text End Get End Property
