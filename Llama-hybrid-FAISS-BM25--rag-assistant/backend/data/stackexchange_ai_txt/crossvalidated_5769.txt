With respect to loess the BioPython project has a lowess() function. As for ARIMA model fitting, PyIMSL Studio contains a number of very useful time series analysis functions, including ones for automatically fitting the best ARIMA model and identifying outliers from it. These same functions are available in the other IMSL libraries if you choose to instead code in C, C#, Java, or Fortran.
