This is a nasty workaround. First you create QSettings , then get its location. QSettings cfg(QSettings::IniFormat, QSettings::UserScope, "organization", "application"); QString config_dir = QFileInfo(cfg.fileName()).absolutePath() + "/"; Credits go to the Qt Centre forum. QSettings stores the default config in the user AppData directory. See documentation for QSettings . Also this code instructs to store the config in the Ini file format.
