Use a program like FireBug for FireFox to see the request header. Then open up your web browser and go to your application. Turn on the Net tab on FireBug to view the request values. For example, on this web site I can see: Host stackoverflow.com User-Agent Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.0.5) Gecko/2008120122 Firefox/3.0.5 (.NET CLR 3.5.30729) Accept text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8 ... Thus I would expect that Request.Url.Host will be "stackoverflow.com". As a side note, you should definitely look into using the ~/ absolute path option, a brief example: string finalUrl = "~/home.aspx"; Response.Redirect(finalUrl);
