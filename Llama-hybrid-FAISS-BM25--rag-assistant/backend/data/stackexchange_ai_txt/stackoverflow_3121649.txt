Note the caveat (emphasis added) in the documentation : If the C functions fgetpos and fsetpos are available, then FileHandle::getpos returns an opaque value that represents the current position of the FileHandle , and FileHandle::setpos uses that value to return to a previously visited position. Opaque means you shouldn't pay attention the value: use it only as a parameter in future requests from the module.
