Unlike the answer from @Arne Jonas Warnke , I see no need to restrict attention to non-parametric estimators for nonlinear quantile regression. Simply use whatever form of nonlinear function of the parameter vector $\beta$ , namely $f(\beta)$, you have in place of $X\beta$ in https://en.wikipedia.org/wiki/Quantile_regression#Conditional_quantile_and_quantile_regression . Of course, if $f(\beta)$ is a nonlinear function of the parameter vector $\beta$, the quantile regression problem will not be a Linear Programming problem, but it could still be solved with an appropriate nonlinear optimization solver.
