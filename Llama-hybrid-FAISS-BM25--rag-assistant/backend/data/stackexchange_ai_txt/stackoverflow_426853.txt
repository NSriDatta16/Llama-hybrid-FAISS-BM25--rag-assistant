This may not be an answer to your question - but it is a handy article that does touch on style sheet organisation. In particular, it mentions having a few stylesheets, targetted at different platforms. In general, I'd have a global style sheet for layout and design, and then separate stylesheets if absolutely required. I can't think of a use case where you'd need a css file per page. Progressive Enhancement With CSS - A List Apart I encourage all web developers to read this article though.
