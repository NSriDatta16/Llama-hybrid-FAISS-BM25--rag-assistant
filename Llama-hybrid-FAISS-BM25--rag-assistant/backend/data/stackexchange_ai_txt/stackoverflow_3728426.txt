While not the most elegant solution, this should work: static constraints = { prop(validator: { val, obj -> if(obj.a == null) { return !myModel.findWhere(b: obj.b, prop: val) } else { return !myModel.findWhere(a: obj.a, b: obj.b, prop: val) } }) } I don't believe there's a way to conditionally validate uniqueness based on property values without manually performing the query.
