You can give them each the same parameter name. http://www.example.com/search?query=cars&query=phones The average server side HTTP API is able to obtain them as an array . As per your question history, you're using JSP/Servlet, so you can use HttpServletRequest#getParameterValues() for this. String[] queries = request.getParameterValues("query");
