Why don't you do it on the client side using javascript? The code for a given node n would be: function reverse(n) { // Reverses the order of the children of Node n var f = document.createDocumentFragment( ); // Get an empty DocumentFragment while(n.lastChild) // Loop backward through the children, f.appendChild(n.lastChild); // moving each one to the DocumentFragment n.appendChild(f); // Then move them back (in their new order) }
