Cluster validity is a fragile concept. There have been a dozen proposals, but none that is truly convincing. I recommend to use an application for evaluation . Only if the clustering helps you solve an actual problem then it was successful. It's meaningless to score high on some artificial metric if you can't apply it to real problems. So what's your use case scenario? Probably not streaming that dreadful poker hands data set or forest cover, like many of the "stream clustering" people do. Solving problems that do not exist on data that isn't a real stream...
