Try qualifying the type name of rs with the ADODB prefix to make sure it is not being defined as the built-in Access Recordset object type instead. Dim rs As ADODB.Recordset Edit: You will also need to use the ADO .Execute command instead of the DAO .OpenRecordset: Set rs = cnn.Execute("...")
