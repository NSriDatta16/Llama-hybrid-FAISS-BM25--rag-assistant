If you have source file on disk you can use this code: context.Response.AddFileDependency(pathImageSource); context.Response.Cache.SetETagFromFileDependencies(); context.Response.Cache.SetLastModifiedFromFileDependencies(); context.Response.Cache.SetCacheability(HttpCacheability.Public); Also, make sure that you test using IIS, not from Visual Studio. ASP.NET Development Server (aka Cassini) always sets Cache-Control to private. See also: Caching Tutorial for Web Authors and Webmasters
