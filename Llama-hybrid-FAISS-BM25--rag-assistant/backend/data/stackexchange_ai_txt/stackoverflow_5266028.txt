YetAnotherForum.NET (YAF) is an open-source discussion forum written in ASP.NET (C#) using SQL Server for its data storage. It is licenced under the GNU General Public License (GPL) Version 2 . The current version is 1.9.5.5 RTW released on March 4th, 2011. YAF runs on any web server that supports ASP.NET 2.0 or greater and SQL Server. Features of YetAnotherForum There is a full list of features available here but here are some of the main ones: 100% C# ASP.NET v3.5 Uses AJAX and jQuery for controls. Microsoft SQL Server 2005/2008 database support. 100% XHTML Validated. Isolated CSS theming system to avoid conflicts with an existing site. Dynamic plugin engine allowing module extensions. SEO features including dynamically generated meta tags on topic pages. IP banning and user "kill" system. Comes with many themes to choose from. See Also: YAF Wiki YAF Test Forum YAF on Codeplex
