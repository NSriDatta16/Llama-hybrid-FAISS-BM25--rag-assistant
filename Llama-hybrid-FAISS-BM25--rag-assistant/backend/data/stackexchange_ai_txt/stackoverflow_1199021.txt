Firefox has Firebug Opera has Dragonfly IE8 has developer tools Webkit browsers (Chrome/Safari) have web inspector Aside from these, you may want to look up the outline property which adds a border to elements, but without disrupting the page flow. This is useful for side-by-side comparisons. If you have some classes being applied with jQuery, simply use the outline property on them and it should appear when the class is added. .class1 { outline: 1px solid blue; } .class2 { outline: 1px solid red; }
