It depends on the type of ACF plot and the amount of autocorrelation. The $x$ -series and the $y$ -series generated below are both Markov chains. The $y$ -series is 'almost' equivalent to independent tosses of a fair coin. To my eyes, the ACF plot (from R) at left shows obvious autocorrelation, while the one at right does not. par(mfrow=c(1,2)) set.seed(1111) m = 200; x=numeric(m); x[1]=1; a=.3; b=.2 for(i in 2:m) { if (x[i-1]==0) x[i] = rbinom(1,1,a) else x[i] = rbinom(1,1,1-b) } acf(x) set.seed(1212) m = 2000; y=numeric(m); y[1]=1; a=.5; b=.502 for(i in 2:m) { if (y[i-1]==0) y[i] = rbinom(1,1,a) else y[i] = rbinom(1,1,1-b) } acf(y) par(mfrow=c(1,1))
