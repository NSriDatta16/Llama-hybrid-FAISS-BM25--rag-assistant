You can use open-uri to download the HTML code of one wiki page and then interpret with Regexp. Look: require 'open-uri' infobox = {} open('http://en.wikipedia.org/wiki/Wikipedia') do |page| page.read.scan(/ (.*?) . (.*?) /m) do |key, value| infobox[key.gsub(/ /, '').strip] = value.gsub(/ /, '').strip # Removes tags (as hyperlink) end end infobox["Slogan"] #=> "The free encyclopedia that anyone can edit." infobox["Available language(s)"] #=> "257 active editions (276 in total)" Should exist some better method. But this works.
