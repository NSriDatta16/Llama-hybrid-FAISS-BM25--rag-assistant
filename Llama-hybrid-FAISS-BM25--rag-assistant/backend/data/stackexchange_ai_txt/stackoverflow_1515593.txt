It could be a .VBS file as well I suppose. Try this: 'takes out (hides) empty columns in 1st worksheet 'drag and drop excel file into this .vbs set xl = createobject("excel.application") set wb = xl.workbooks.open(wscript.arguments(0)) set ws = wb.worksheets(1) xl.visible = true numCols = ws.usedrange.columns.count for col = numCols to 1 step -1 set r = ws.columns(col) if xl.worksheetfunction.counta(r) = 0 then 'column has no values, so hide it r.columnwidth = 0 'if you want to delete the column 'r.delete end if next
