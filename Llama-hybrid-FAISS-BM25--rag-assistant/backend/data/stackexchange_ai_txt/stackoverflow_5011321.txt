You need to use Javascript and modify param values. Browser info can be fetched using navigator object few of the methods navigator.appName navigator.appVersion navigator.userAgent and pass those values into applet tags dynamically,which are actual parameters of method in an applet. a small but good example can be found here on integrating javascript and java applets. http://www.ibiblio.org/java/course/week5/16.html after which you can set values using following snippet. document.getElementById("browserAgent").value=navigator.userAgent;
