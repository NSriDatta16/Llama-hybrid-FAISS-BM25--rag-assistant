On SQL Server varchar(100) = 100 ascii characters and 102 bytes nvarchar(100) = 100 unicode/ascii characters and 202 bytes of storage text and ntext go up to 2GB but are deprecated (from sql server 2005), you should use varchar(max) and nvarchar(max) which both also go up to 2 GB instead
