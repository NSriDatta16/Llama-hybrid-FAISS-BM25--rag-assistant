I wonder if I should write $('#C').val(doit(A,B)) instead, or is it acceptable to pass A.val(), B.val()? Passing A,B wouldn't work. Your solution looks reasonably elegant, you even cache the jQuery objects to get A,B in the closures you use. you could make it a little more concise by doing: function doit(A,B) { return A+B; }; $(function() { var A = $('#A'); var B = $('#B'); $('#A, #B').change(function() { $('#C').val(doit(A.val(),B.val())); }); });
