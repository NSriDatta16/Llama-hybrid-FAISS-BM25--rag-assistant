If you are using an outlook form you can publish it to the Organizational Forms Library and the get you clients to use that new custom form instead of the default out the box form. It a registry change that points it to a new form. There are tools to do this change for you. A good example http://www.petri.co.il/customizing_new_meeting_request_outlook_form.htm 76mel
