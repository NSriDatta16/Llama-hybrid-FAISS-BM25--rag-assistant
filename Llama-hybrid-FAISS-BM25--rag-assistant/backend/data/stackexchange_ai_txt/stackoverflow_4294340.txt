I would recommend using the ReportViewer control that is included is Visual Studio. The following resource site provides good information for embedding reports into your application: http://www.gotreportviewer.com/ The ReportViewer control supports rendering SQL Server Reports that can be designed using a number of WYSIWYG tools including Visual Studio. These reports are very robust and include many features in addition to the grouping and subtotal features that you need.
