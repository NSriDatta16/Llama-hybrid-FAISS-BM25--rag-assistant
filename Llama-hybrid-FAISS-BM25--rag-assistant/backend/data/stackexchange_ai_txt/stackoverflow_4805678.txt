You first create a dataset with a list of items which the user will select from. Select ID,Agentname from Agents Order by Agentname You then create the second dataset; eg. Select ID,ItemName from Orders where agentid=@agentid IN your report paramenter, you click on agentid parameter and then from a Query on available values. You select dataset1 and ID as value and AgentName as Label. Your last dataset will be the actual data. In this You create your statement like : Select Quantity, Amount From Orderdetails where OrderID=@orderid In reporting service, you go to parameter and the set Order to get value from Dataset2 So the Order of parameter should be agentid, orderid. Reporting Service will automatically prompt for parameter by order.
