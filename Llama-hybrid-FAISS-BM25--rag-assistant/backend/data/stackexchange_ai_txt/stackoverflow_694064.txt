SQL Injection Prevent malicious users from altering SQL queries via URL query strings. DoS Attacks Prevent users from the same IP address from accessing your site an excessive number of times in a small space of time. Password Strength When allowing users to create their own passwords, show a password strength indicator which encourages users to enter stronger passwords. Captcha Stop non-human users from submitting to forms by presenting a captcha image. You may also want to use this if password authentication is failed multiple times, to prevent robots from guessing passwords.
