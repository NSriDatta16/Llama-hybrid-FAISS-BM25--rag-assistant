Are the users using different connections to the SQLite DB? Try using the PRAGMA command to increase cache size. PRAGMA cache_size; PRAGMA cache_size = Number-of-pages; Query or change the suggested maximum number of database disk pages that SQLite will hold in memory at once per open database file. From documentation: http://www.sqlite.org/pragma.html#pragma_cache_size
