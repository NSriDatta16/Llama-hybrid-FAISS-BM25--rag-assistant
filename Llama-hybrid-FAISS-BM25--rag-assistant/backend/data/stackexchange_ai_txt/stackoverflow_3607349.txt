The link between the event firing and the listener is based on the event type. In your example: public class EventConsumer { public void afterMyEvent(@Observes MyEvent event) { // .. Insert event logic here } } This class will receive all fired events of type MyEvent, wherever they come from. Source: http://download-llnw.oracle.com/javaee/6/api/javax/enterprise/event/Observes.html
