I wouldn't call it the preferred way, but if you want to pursue your second option this should detect SVG support (from Raphaël 1.5.2 ): if(window.SVGAngle || document.implementation.hasFeature("http://www.w3.org/TR/SVG11/feature#BasicStructure", "1.1") { // supports SVG else { // no SVG } Raphaël uses this to determine if it should render using VML (IE) or SVG (everyone else). Out of curiosity, why SVG for your logo? If you already have a PNG version, this seems like a lot of work.
