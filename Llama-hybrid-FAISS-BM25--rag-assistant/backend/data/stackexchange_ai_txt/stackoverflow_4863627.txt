this what you are looking for? xa = [1, 2, 3, 10, 1383, 0, 12, 9229, 2, 494, 10, 49] xb = [1, 1, 4, 12, 1100, 43, 9, 4848, 2, 454, 6, 9] xc = [] for i in range(0, len(xa)-1): xc.append(xa[i] - xb[i]) print xc output: [0, 1, -1, -2, 283, -43, 3, 4381, 0, 40, 4] EDIT: why not take the percent difference of each value then average them all: from statlib import stats xa = [1, 2, 3, 10, 1383, 0, 12, 9229, 2, 494, 10, 49] xb = [1, 1, 4, 12, 1100, 43, 9, 4848, 2, 454, 6, 9] xc = [] for i in range(0, len(xa)-1): xc.append(abs(float(xa[i] - xb[i])/(xa[i] + xb[i])/2)) print stats.mean(xc)*100 if you don't have statlib you can get it here
