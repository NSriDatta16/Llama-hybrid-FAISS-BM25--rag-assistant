You can convert them numbers using LabelEncoder and then apply the machine learning algorithms. Example- column - city has data->ab,bf,cd,bf,gr,ab Now i can convert them into form-> 1,2,3,2,4,1 Use : from sklearn import preprocessing le = preprocessing.LabelEncoder() col= le.fit_transform(["ab","bf","cd","bf","gr","ab" ])
