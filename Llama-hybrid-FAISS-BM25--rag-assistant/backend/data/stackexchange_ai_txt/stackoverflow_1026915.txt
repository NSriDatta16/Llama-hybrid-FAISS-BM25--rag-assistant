& AFAIK in Windows you cannot do that using the Text widget easily. At drop you can insert text at the last cursor position or at the end of the text. You may consider using the StyledText widget which supports dropping at any position and 'caret hovering' as well. Both Text and StyledText support the SWT.MULTI style for being multi-line. You can find many great code snippets at the SWT snippets page including examples for drag and drop using the StyledText widget. Also there is a good introduction for DND in SWT .
