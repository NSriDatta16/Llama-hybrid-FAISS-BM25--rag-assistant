This will fix it: window_title.embedFonts = true; You also need to actually embed the font in the swf file, though. If you're using the Flash application, set the font in the properties window and then just check the embed box. If you're using mxmlc, you can embed the font with @Embed. See here: http://www.adobe.com/devnet/flash/quickstart/embedding_fonts/
