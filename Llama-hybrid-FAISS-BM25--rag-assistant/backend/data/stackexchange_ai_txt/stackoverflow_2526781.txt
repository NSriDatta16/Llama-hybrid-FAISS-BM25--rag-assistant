First of all, the header file for embedding or extending python is Python.h , with an uppercase P. Linux is case sensitive, and python.h won't work. On debian and debian deriviatives like ubuntu, you can use apt-file to find out what package to install, given a filename: $ apt-file search Python.h ... python2.6-dbg: /usr/include/python2.6_d/Python.h python2.6-dev: /usr/include/python2.6/Python.h ...
