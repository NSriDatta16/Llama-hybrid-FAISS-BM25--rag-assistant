In this case you need to go up to the parent .tier using .closest() then, to the .next() sibling, then .find() the .publication_date control inside, like this: $(".publication_date").livequery(function() { $(this).datepicker({ dateFormat: "dd M yy", changeYear: true, changeMonth: true, altFormat: 'yy-mm-dd', altField: $(this).next(), onSelect: function(dateText, inst) { console.log($(this).closest(".tier").next().find('.publication_date')); } }); }); Check out your HTML though, there are numerous errors in there, random JavaScript fragments will give you issues.
