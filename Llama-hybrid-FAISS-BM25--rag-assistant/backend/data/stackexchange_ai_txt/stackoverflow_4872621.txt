Time taken my $db->query() mysql_query() is almost the same. We do not use classes to get execute the code faster but to organise them. As your project gets bigger and bigger, you will have difficulty in managing your code. If you objects then you can instantiate that object and use them again again, and besides your code has less chances to get messed up with others. Object Oriented Way of coding is the easiest and best practice for managing your code elegantly and hence produce better results.
