You should take a look: ASP.NET Web Parts Overview and Web Parts Personalization Overview . I personally store webparts configuration by using WebPartStorage attribute on public properties from my webpart class. By default, sessions are disabled on Sharepoint. You should take extra care if you plan to enable it, including performance and stress tests.
