Postpone executing the Focus() method until after the event is finished executing. Elegantly done by using the Control.BeginInvoke() method. Like this: private void textBox2_Enter(object sender, EventArgs e) { this.BeginInvoke((MethodInvoker)delegate { textBox3.Focus(); }); }
