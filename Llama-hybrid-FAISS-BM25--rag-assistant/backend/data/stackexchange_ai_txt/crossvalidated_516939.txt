LinearDML in econml also accepts X=None. In that case a constant effect is estimated which can be recovered by calling: est.effect() With no arguments. Or effect_inference() if you want full inference info. Also even if you specify an X so as to allow for effect heterogeneity you can also recover the ate by calling: est.ate(X) And est.ate_inference(X) Where X contains all the samples over which you want to calculate an average effecy
