When **dict is used in a function call like that, the dictionary is expanded into keyword arguments, so the dictionary keys become the keywords and the dictionary values become the argument values. So in your case: output = model(**token) is expanded to: output = model(input_ids=token['input_ids'], attention_mask=token['attention_mask'])
