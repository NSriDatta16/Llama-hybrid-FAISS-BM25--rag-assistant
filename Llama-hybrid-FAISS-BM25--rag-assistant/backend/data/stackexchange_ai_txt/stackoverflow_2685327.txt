I have used Roget's thesaurus in the past. It has the synonymy information in plain text files. There is also some java code to help you parse the text. These pages provides links to a bunch of thesauri/lexical resources some of which are freely downloadable. http://www.w3.org/2001/sw/Europe/reports/thes/thes_links.html http://www-a2k.is.tokushima-u.ac.jp/member/kita/NLP/lex.html
