It probably expects the user agent to be a web browser. You can set this easily using cURL: curl_setopt($ch, CURLOPT_USERAGENT, $useragent); Where $useragent is the string you want to use for a user agent. Try it with some common ones for the major browsers and see if that helps. This page lists some common user agents .
