My solution so far: function start (e, ui) { var draggable = ui.helper.data('draggable'); draggable.helperProportions = { height: 10, width: 10 }; ui.helper.data('draggable', draggable); ui.helper.data('draggable')._setContainment(); } Makes me sad that I'm mucking around in draggable's internals, but it works.
