The comments on your question have already revealed the answer so here is an answer answer (as opposed to a comment answer): OrderAgencies is an EntitySet<> ( collection of entities) but in your binding you're trying to access COName which is a property of the OrderAgency entity class (a single entity). In your binding you'll need to reference a single OrderAgency entity to be able to access its COName property.
