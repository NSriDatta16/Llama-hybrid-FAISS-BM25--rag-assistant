There are a number good options for encoding categorical data with high cardinality. If using Python, the Category Encoders package has like a dozen options as of this writing. I wrote a guide to using some of them here . Category Encoders includes several Bayesian encoders, including Target encoder. Binary and Hashing encoders help reduce dimensionality, but might make it a bit harder for your model to pick up signal in your data.
