Why not consider using Jquery's UI Draggable/Sortable? It takes some of the brain damage out of the code and gives the user an intuitive way to interface with the reordering process: http://jqueryui.com/demos/draggable/#sortable You could then use the callbacks to update via Ajax, or set variables that could be dumped to PHP on submit.
