This is what I have to listen to BACK or MENU key events. Simply add this method, without implementing any Interface. I do this in my BaseActivity, from which every Activity inherits. public boolean onKeyDown(int keyCode, KeyEvent event) { Log.d(NAME, "Key pressed"); switch (keyCode) { case KeyEvent.KEYCODE_BACK: Log.d(NAME, "Back pressed"); // IGNORE back key!! return true; /* Muestra el Men√∫ de Opciones */ case KeyEvent.KEYCODE_MENU: Intent menu = new Intent(this, Menu.class); // start activity startActivity(menu); return true; } return super.onKeyDown(keyCode, event); } PS: I highly discourage ignoring the back key.
