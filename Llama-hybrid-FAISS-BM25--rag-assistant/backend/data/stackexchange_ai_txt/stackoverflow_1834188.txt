The only user agent setting in PHP is user_agent , but this setting is only used for HTTP requests. You can pass additional headers to the mail() function to accomplish this: mail( 'to@example.com', 'Subject', 'Message Body', "From: from@example.com\r\nX-Mailer: PHP" );
