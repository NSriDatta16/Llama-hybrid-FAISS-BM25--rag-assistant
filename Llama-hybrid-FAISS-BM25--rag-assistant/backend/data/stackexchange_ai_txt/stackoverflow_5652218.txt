Please note that instance_methods returns an array of String or Symbol depending on the Ruby version. Ruby 1.8 returns an Array of String, Ruby 1.9 an Array of Symbol. In Ruby 1.8 "".respond_to?(:upcase) # => true "".class.instance_methods.include?("upcase") # => false "".class.instance_methods.include?(:upcase) # => false In Ruby 1.9 "".respond_to?(:upcase) # => true "".class.instance_methods.include?("upcase") # => false "".class.instance_methods.include?(:upcase) # => true Also, instance_methods must be called on the class, not on the instance.
