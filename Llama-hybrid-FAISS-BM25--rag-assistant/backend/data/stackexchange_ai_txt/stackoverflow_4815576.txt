I would recommend creating a custom data storage class. You could call it PDPerson.h You'll also need the .m file. For each property, do something like this: In the .h : Declare each of your properties like so: @interface PDPerson : NSObject{ } @property(nonatomic, retain) NSString *firstName; @property(nonatomic, retain) NSString *lastName; @property(nonatomic, retain) NSString *class;//May want to consider renaming @property(nonatomic, retain) NSString *year; @property(nonatomic, retain) NSString *dept; @end Then in the .m: @implementation @synthesize firstName, lastName; @synthesize class, year dept; -(void)dealloc{ [firstName release]; [lastName release]; [class release]; [year release]; [dept release]; } Each time you want to create a new "Person" in your array, do this: PDPerson *person = [[PDPerson alloc]init]; You can then easily set the properties of the object like so: person.firstName = @"John"; person.lastName = @"Smith"; person.class = @"Math"; person.year = @"1995"; person.dept = @"Sciences"; And retrieve them: firstNameLabel.text = person.firstName; The nice thing about these objects is that all you have to do now is add the person object to your array: NSMutableArray *personArray = [[NSMutableArray alloc] init]; [personArray addObject:person];
