How about: require 'nokogiri' puts Nokogiri.HTML(' a ').to_html # # >> # >> a If you don't want Nokogiri to create a HTML document, then you can tell it to parse it as a document fragment: puts Nokogiri::HTML::DocumentFragment.parse(' a ').to_html # >> a In either case, the to_html method returns the HTML version of the document.
