I would suggest that instead of using an array for this purpose, you take advantage of the DOM-elements being objects: document.getElementById('something').info = {data: 'something', fn: function () { } }; but you may end up with some issues with memory leaks in certain older browsers (read: ie6). That is fixed if you use jQuery's data storage instead: $("#something").data("info", {data: 'something', fn: function () { } });
