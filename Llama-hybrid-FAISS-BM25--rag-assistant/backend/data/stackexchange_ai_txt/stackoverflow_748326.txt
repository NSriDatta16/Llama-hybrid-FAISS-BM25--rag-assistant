You can iterate through DOM elements, looking for a substring within them. Neither fast nor elegant, but for small HTML might work well enough. I'd try something recursive, like: (code not tested) findText(node, text) { if(node.childNodes.length==0) {//leaf node if(node.textContent.indexOf(text)== -1) return []; return [node]; } var matchingNodes = new Array(); for(child in node.childNodes) { matchingNodes.concat(findText(child, text)); } return matchingNodes; }
