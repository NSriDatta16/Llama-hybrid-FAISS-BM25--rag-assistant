I had this problem and it turned out the downloaded zip file had a new line inserted at the very beginning. Solved by using ob_clean and flush functions header("Pragma: public"); header("Expires: 0"); header("Cache-Control: must-revalidate, post-check=0, pre-check=0"); header("Cache-Control: private",false); header("Content-Type: application/octet-stream"); header("Content-Disposition: attachment; filename=".basename($archive_file_name)); header("Content-Transfer-Encoding: binary"); header("Content-Length: ".filesize($archive_file_name)); ob_clean(); flush(); echo readfile("$archive_file_name");
