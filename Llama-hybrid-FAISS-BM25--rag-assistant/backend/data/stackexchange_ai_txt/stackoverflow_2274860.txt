Quickest thing I can think of: Drag the WebService Connector component from the Components Panel to the Stage . Give it an instance name and set the operation in the Components Inspector . Trigger it via Actionscript: yourWebServiceConnector.trigger(); That should be it. A more comprehensive guide on as2 data components is available here . HTH, Georrge
