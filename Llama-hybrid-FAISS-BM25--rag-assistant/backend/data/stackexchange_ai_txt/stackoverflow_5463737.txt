A URI per RFC 3986 has the grammar: URI = scheme ":" hier-part [ "?" query ] [ "#" fragment ] So, yes, the scheme http as well as the separating : and the // prefix of the hier-part are part of the URI. news.bbc.co.uk is only the host .
