These instructions are for R: For (proportional odds) ordered probit and logit: library(MASS) ?polr and read the help page that appears. For (Bayesian) multinomial probit, first install the package MNP then: library(MNP) ?mnp However, this may be overengineered for your purposes. A simpler package is built in for multinomial logit. For multinomial logit: library(nnet) ?multinom There is some detailed description here on how to use it.
