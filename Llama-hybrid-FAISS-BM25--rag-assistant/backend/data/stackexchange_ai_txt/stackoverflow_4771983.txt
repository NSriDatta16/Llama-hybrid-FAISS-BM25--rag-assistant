I am trying to retrieve all of the possible Wikipedia supported languages and output them to a text file by traversing the tables on List_of_Wikipedias Your problem is that the element names in the document are in a default namespace. How to write XPath expressions that involve such element names is the most FAQ in XPath and has numerous good answer in the SO xpath tag. Just search for them. Here is a complete solution: Use : (//x:table)[1]/x:tr[not(x:th)]/x:td[2]//text() where you have registered the XHTML namespace ( "http://www.w3.org/1999/xhtml" ) bound to the prefix "x" . When I evaluated this XPath expression against the document obtained from: http://s23.org/wikistats/wikipedias_html I needed to add the following at the start of the document, because I was working locally and didn't have the DTD for XHTML -- maybe you will not need these: ]> The result of applying the above XPath expression to this document is : English German French Polish Italian Japanese
