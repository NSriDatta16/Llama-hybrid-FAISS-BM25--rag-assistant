In short, if you have set parseOnLoad as false, you need to parse the html fragment by yourself. And for your tab pages, if you clicked a tab, you may need to parse the tab page manually. For example: You can use dojo.parser.parse(dojo.byId("a")) to render the widget. Btw, you do not need to attach tundra class in every places of dijit widgets, it just need to be placed at the body tag,.
