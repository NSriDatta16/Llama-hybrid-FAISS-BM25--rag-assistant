If you're using GNU date you can use relative dates like this: $ doy=193 $ date -d "Jan 1 +$((doy -1)) days" +%U 28 This would give you a very simplistic answer, but doesn't rely on date : $ echo $((doy / 7)) which pays no attention to the day of week.
