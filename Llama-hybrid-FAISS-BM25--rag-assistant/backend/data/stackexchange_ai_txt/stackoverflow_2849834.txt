https://www.rfc-editor.org/rfc/rfc3986#section-3.5 : fragment = *( pchar / "/" / "?" ) and pchar = unreserved / pct-encoded / sub-delims / ":" / "@" unreserved = ALPHA / DIGIT / "-" / "." / "_" / "~" sub-delims = "!" / "$" / "&" / "'" / "(" / ")" / "*" / "+" / "," / ";" / "=" pct-encoded = "%" HEXDIG HEXDIG So, combined, the fragment cannot contain # , a raw % , ^ , [ , ] , { , } , \ , " , and > according to the RFC.
