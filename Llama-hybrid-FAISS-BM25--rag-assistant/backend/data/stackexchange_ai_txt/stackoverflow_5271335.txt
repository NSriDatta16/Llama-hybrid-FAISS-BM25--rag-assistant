They will all be run in the order they were registered. See the fifth paragraph in the documentation to bind() , which .ready() , .load() etc. are shortcuts of. When an event reaches an element, all handlers bound to that event type for the element are fired. If there are multiple handlers registered, they will always execute in the order in which they were bound. After all handlers have executed, the event continues along the normal event propagation path.
