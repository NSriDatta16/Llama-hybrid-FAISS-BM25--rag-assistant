The Post Office suggests ( http://www.postoffice.co.uk/portal/po/content1?catId=19100182&mediaId=19100267 ) 7 lines: Addressees Name Company/Organisation Building Name Number of building and name of thoroughfare Locality Name Post Town Post Code They then say you do not need to include a County name provided the Post Town and Postcode are used.
