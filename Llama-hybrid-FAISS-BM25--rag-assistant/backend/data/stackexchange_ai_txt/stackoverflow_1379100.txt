If your goal is to display the YUV image on the screen you can use the OpenGL module. Implement the conversion as a fragment shader. If you want to do something else like save the converted image to a file then it is not a good idea. Some links: http://doc.qt.digia.com/4.6/qglshaderprogram.html http://doc.qt.digia.com/4.6/qglshader.html http://www.fourcc.org/source/YUV420P-OpenGL-GLSLang.c (the conversion is in the FProgram string)
