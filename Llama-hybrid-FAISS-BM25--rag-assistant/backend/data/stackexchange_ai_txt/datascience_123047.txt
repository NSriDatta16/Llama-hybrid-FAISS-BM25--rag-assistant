You can easily combine embeddings by taking the element-wise mean of them. You can do more fancy things like building attention models but this is a good way to start. import numpy as np a = [1, 2, 3] b = [10, 20, 30] np.mean([a, b], axis=0) # [ 5.5, 11., 16.5]
