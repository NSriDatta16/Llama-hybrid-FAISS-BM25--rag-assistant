You're not linking all the libraries. Code::Blocks link all of 'em automatically, but not VS. When i create applications using OpenGL, i do this (linking via code, doesn't looks pretty, but "just works"): #pragma comment(lib, "sfml-system.lib") #pragma comment(lib, "sfml-window-s.lib") #pragma comment(lib, "sfml-graphics.lib")
