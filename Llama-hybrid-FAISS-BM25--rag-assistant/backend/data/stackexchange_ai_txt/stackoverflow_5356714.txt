If no initializer is specified for an object, the object is default-initialized; if no initialization is performed, an object with automatic or dynamic storage duration has indeterminate value. Par. 8.5, section 11 of a recent C++0x draft N3092.pdf, http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2010/
