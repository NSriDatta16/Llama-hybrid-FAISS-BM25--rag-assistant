MDB most often stands for "message-driven bean" or for Microsoft Jet DB, Access database standard (as the database file has ".mdb" extension). Despite attempts to resolve the conflict, there are numerous questions tagged with it related to both technologies. For further information, see discussions on meta: http://meta.stackexchange.com/questions/33146/ambiguous-tag-mdb http://meta.stackexchange.com/questions/33216/ms-access-or-mdb-or-access-database-engine-or-ms-jet-ace http://meta.stackexchange.com/questions/105117/proposal-to-discourage-the-mdb-tag
