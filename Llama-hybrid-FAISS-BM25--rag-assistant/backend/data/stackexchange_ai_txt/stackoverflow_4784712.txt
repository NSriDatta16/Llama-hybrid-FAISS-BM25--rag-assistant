If you're asking about an application running on the phone natively but written using HTML5 and Javascript, look into the WebWorks/Widget platform. It runs on Blackberry OS versions 5 and 6. If you want to have a web site/app that uses HTML5 offline storage, it will probably only work with Blackberry OS version 6. This guide can help in determining how much of the market is using those OS versions.
