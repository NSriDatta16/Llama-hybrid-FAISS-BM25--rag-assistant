If your servlet is doing something like calling a service every x minutes without a user interaction then you probably want to do something different. You can use a ServletContextListener to listen to when the web context is started and to then start whatever code you want. http://java.sun.com/javaee/5/docs/api/javax/servlet/ServletContextListener.html
