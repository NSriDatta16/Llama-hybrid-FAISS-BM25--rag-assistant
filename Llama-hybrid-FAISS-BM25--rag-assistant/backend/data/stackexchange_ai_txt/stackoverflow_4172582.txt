You may checkout the documentation . According to it the default value of the cache location is Any which means: The output cache can be located on the browser client (where the request originated), on a proxy server (or any other server) participating in the request, or on the server where the request was processed. This value corresponds to the HttpCacheability.Public enumeration value As far as caching the master you cannot put an OutputCache directive to a master page. You could do it programatically by enabling caching in all the content pages.
