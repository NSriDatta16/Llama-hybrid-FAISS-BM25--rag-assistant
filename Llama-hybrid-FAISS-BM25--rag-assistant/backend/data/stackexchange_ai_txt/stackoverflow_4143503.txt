Bash "coalesce" functionality

I'm trying to do the following on a single line: if [ -z "$foo" ] then source "$foo/subdir/yo.sh" else source "$bar/yo.sh" fi (note the subdir only present if foo is used). The simplest solution so far is: source "${foo:-$bar}${foo:+/subdir}/yo.sh" In other words, take either foo or bar, then append /subdir if foo exists. Is there a more elegant way to do this?
