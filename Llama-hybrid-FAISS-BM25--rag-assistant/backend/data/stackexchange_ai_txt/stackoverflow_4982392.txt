Use a ListView. With the ListView you will create a list of your SDcard´s media. ListView Example UPDATE :: yes Turtleboy you will save your media files inside a directory "/sdcard/mymediafiles/" then extract the file´s name to be displayed in your ListView! File sdCardRoot = Environment.getExternalStorageDirectory(); File myDirectory = new File(sdCardRoot, "/mymediafiles/"); for (File f : myDirectory.listFiles()) { if (f.isFile()) Log.i("Jorgesys","*** Media File :: " + f.getName()); }
