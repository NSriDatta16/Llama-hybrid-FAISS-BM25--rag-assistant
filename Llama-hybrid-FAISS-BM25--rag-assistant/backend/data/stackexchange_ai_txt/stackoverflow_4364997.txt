push an element to each array in a multidimentional array in php?

Here's my array $myArr = array(array('one', 'two'), array('one', 'two')); I would like add an element to every array inside the $myArr . I dont wanna loop over each array inside and add that element. Is there any quicker way to achieve this using array_map or array_walk .. preferably with one line of code ? Result array should be like $myArr = array(array('one', 'two','three'), array('one', 'two','three')); Thanks a bunch Got it, Kudos to @strager $myvar = 'Three'; $myArr = array_map(function ($subarray) { global $myvar; $subarray[] = $myvar; return $subarray;}, $myArr);
