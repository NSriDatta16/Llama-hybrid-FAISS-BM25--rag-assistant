You can try to use p2 API to query the installed feature. P2 is the manager of eclipse installation. // IProvisioningAgent is a OSGi service IProvisioningAgent agent = ...; IProfileRegistry profileRegistry = (IProfileRegistry) agent.getService(IProfileRegistry.SERVICE_NAME); IProfile profile = profileRegistry.getProfile(IProfileRegistry.SELF); IQueryResult rt = profile.query(QueryUtil.createIUPropertyQuery("org.eclipse.equinox.p2.eclipse.type", "feature"), null);
