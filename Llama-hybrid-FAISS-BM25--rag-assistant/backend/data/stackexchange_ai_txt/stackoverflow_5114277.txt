Use HashMap as storage with Keys as UserName . You can check if user exists by performing containsKey method of HashMap. Also, it is advisable to Check for NULL input Check for blank input Check for absurd name (like numeric names, if not allowed) Store keys in same case i.e. either in lower-case or in upper case. And while looking for duplicate make sure that your input has converted to that case. //this is your address-book with unique User-Name private static final HashMap addressBook = new HashMap (); ... ... boolean addEntry(){ boolean isNewEntry = true; //getNonBlankInput should check for valid name String name = getNonBlankInput("Enter Name: "); if(!addressBook.containsKey(name.toLowerCase())){ AddressBookEntry entry = new AddressBookEntry(); entry.setName(name); entry.setAdd(getNonBlankInput("Enter Address: ")); entry.setPhoneNo(getNonBlankInput("Enter Phone No.: ")); entry.setEmail(getNonBlankInput("Enter E-mail Address: ")); addressBook.put(name.toLowerCase(), entry); }else{ isNewEntry = false; } return isNewEntry; }
