You may either want to run the jnlp file with -verbose option via javaws command from the command line and/or check the logs. The logs location is java version dependant, In earlier windows releases default path was: c:\Documents and Settings\_user_\Application Data\Sun\Java\Deployment\log In Windows 7,default path is 'C:\Users\_user_\AppData\LocalLow\Sun\Java\Deployment\log'
