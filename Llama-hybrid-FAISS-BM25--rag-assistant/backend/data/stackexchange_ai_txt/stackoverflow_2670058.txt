Yes it is possible. You want to use $_SERVER['HTTP_USER_AGENT'] which holds information about the user's operating system and browser. You can use this resource to look up user agent strings (which are held in that variable). However, it is possible for browsers to spoof this information so you can't assume that this is reliable.
