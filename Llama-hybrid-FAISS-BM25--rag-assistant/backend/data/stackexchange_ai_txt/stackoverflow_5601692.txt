Even disregarding browser bugs you can't turn off the spell checking, all you can do is suggest to the user agent that it doesn't spell check stuff. If you look at the section of the spec where it describes the algorithm for determining what's spell checkable here's the first two steps: If the user has disabled the checking for this text, then the checking is disabled. Otherwise, if the user has forced the checking for this text to always be enabled, then the checking is enabled. User preferences always override the attributes.
