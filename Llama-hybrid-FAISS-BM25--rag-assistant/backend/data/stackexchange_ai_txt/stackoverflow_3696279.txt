You need quotes around the selector, like this: $(function(){ $("input[name='jsenabled']").val('1'); }); Also when checking the value, use parenthesis, like this: if($("input[name='jsenabled']").val() == '1') { alert('frf'); } .val() is a function, and called without any arguments gets the value. On a more broad note, pay attention to the console, as Firebug should be throwing your JavaScript errors here for each of these, the console (which shows errors) is your best resource here.
