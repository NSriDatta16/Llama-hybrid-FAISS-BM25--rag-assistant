If you could organize your data as in: | Moment | Student | NumberOfCorrectAnswers | |--------|---------|---------------------------| | Before | 1 | 3 | | Before | 2 | 4 | | Before | 3 | 3 | | ... | ... | ... | | After | 1 | 8 | | After | 2 | 7 | | ... | ... | | This could be analyzed with an one-way anova. So in R you would use: anova(NumberOfCorrectAnswers ~ Moment, data) where data is a data frame with the columns above.
