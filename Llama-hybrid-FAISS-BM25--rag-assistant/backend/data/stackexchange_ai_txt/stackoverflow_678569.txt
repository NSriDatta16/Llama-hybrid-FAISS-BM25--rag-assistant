Found RunScript.groovy which sets up the hibernate session for you, then runs the scripts you specify as arguments. I had to change my source from a Gant script (above) to simply: // Import.groovy def Channel = grailsApp.classLoader.loadClass("content.Channel") def c c = Channel.newInstance(title:"A Channel", slug:"a-channel", position:0).validate() c = Channel.newInstance().validate() println(c.errors) I'm able to run it like so: $> grails run-script scripts/Import.groovy
