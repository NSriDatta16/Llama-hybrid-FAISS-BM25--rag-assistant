You can add a Login control to your master page by adding something like this to the markup Or my dragging the Login control onto the master page's designer from the Toolbox. To hide the login control when the user is logged in, add this code to the master page code-behind: if (Page.User.Identity.IsAuthenticated) { MyLoginControl.Visible = false; }
