Yes, while setting up your XGBoost, you can make use of the parameter called "missing" inside the xgboost as follows below: Let's say you decide to fill up this missing values inside your dataset with (-99999) then parameter missing = -99999 and on the other hand if you decide not to fill up the missing values inside you dataset then set this missing parameter as missing = np.nan. You can see example below: dtrain = xgb.DMatrix(data=X_train, label=y_train, missing= -99999.0, enable_categorical=True) dval = xgb.DMatrix(data=X_val, label=y_val, missing= -99999.0, enable_categorical=True) dtest = xgb.DMatrix(data=comp_test, missing= -99999.0, enable_categorical=True) model = xgb.train(params_xgb, dtrain,\ evals=[(dtrain,'train'),(dval,'val')],\ verbose_eval=100, early_stopping_rounds=100, num_boost_round=10000)
