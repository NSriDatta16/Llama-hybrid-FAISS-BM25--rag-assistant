How to show the execution time before query result will display You can use "sys.dm_exec_requests" but it will support only a handful of operations listed below. Obviously it can't support normal DML/select queries. sys.dm_exec_requests (Transact-SQL) ALTER INDEX REORGANIZE AUTO_SHRINK option with ALTER DATABASE BACKUP DATABASE CREATE INDEX DBCC CHECKDB DBCC CHECKFILEGROUP DBCC CHECKTABLE DBCC INDEXDEFRAG DBCC SHRINKDATABASE DBCC SHRINKFILE KILL (Transact-SQL) RESTORE DATABASE, UPDATE STATISTICS.
