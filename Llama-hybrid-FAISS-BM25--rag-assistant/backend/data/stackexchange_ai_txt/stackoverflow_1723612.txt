How about using #pragma warning extension in VC++? http://msdn.microsoft.com/en-us/library/2c8f766e(VS.80).aspx #pragma warning (push, 2) // Temporarily setting warning level 2 #include #pragma warning (pop) // Restore back You may also try .. #pragma warning (disable: 4201) #include #pragma warning (default)
