In XCode's Organizer you have a list of Provisioning Profiles. Each provisioning profile has an associated profile identifier, which looks something like a GUID. When you build your app and XCode signs it you'll see that profile identifier displayed in the signing step: something like: ~/Library/MobileDevice/Provisioning Profiles/01234567-0123-0123-0123-012345678901.mobileprovision" -o ~/.../MyApp.app/embedded.mobileprovision (The line in the Build Results window will start with " ProcessingProductPackaging "...
