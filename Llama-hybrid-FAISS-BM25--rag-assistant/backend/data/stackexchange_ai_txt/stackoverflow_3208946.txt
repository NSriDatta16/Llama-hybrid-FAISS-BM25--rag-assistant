Not that I've seen it used before, but it is a valid URI-reference. From the grammar: URI-reference = URI | relative-ref relative-ref = relative-part [ "?" query ] [ "#" fragment ] relative-part = "//" authority path-abempty | path-absolute | path-noscheme | path-empty where as an absolute URI is: URI = scheme ":" hier-part [ "?" query ] [ "#" fragment ] For more information, feel free to read the rest of RFC3986
