The R rms package supports general simultaneous confidence intervals. The analyst can specify a series of comparisons, and simultaneous adjustment will be done over all the current set of comparisons. For more information see the rms contrast.rms function, along with Predict and other functions. Here's an example using ols. Substitute lrm for ols to do binary or proportional odds logistic regression. require(rms) set.seed(1) n
