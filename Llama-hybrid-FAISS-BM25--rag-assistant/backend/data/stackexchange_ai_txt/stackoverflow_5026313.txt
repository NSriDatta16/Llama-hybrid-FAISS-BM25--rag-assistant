As other answers suggested use draggable & droppable from the jQuery API. Here is an example how to actually do the post and save the layout. Server side saving is in the example done in php, but it can any other language also. There is a JavaScript function called updateWidgetData() that you might like to investigate. It collects all the information to the array and uses $.post to send the array.
