You cannot nest block-level elements, such as , inside in HTML, so Tumblr is behaving correctly. You should close your paragraph before inserting the pre-formatted text. If you want inline code-style text, try the tag. You can then supplement it with CSS if needed. I like to use the code tag in my HTML files. code { padding: 2px 4px; color: #000000; background-color: #eeeeee; border-radius: 3px; }
