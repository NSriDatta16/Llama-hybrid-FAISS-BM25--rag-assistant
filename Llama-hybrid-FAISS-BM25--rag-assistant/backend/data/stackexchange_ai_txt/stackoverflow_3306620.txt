Maybe this will help: IPackageFragmentRoot root = (IPackageFragmentRoot) element.getAncestor(IJavaElement.PACKAGE_FRAGMENT_ROOT); IClasspathEntry entry = root.getRawClasspathEntry(); Note, that this is not a one-to-one relation between a Java element and class-path entry, this code only returns the first entry.
