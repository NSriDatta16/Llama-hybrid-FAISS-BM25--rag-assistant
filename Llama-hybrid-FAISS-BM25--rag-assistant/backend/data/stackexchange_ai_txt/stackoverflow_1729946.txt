If you leverage abstract classes and methods, you can force subclasses to implement the missing methods. abstract class ParentClass { public function foo () { // do stuff $this->bar(); } abstract protected function bar(); } class Child extends ParentClass { protected function bar() { // does stuff } } Subclasses that don't implement bar() will generate a fatal error.
