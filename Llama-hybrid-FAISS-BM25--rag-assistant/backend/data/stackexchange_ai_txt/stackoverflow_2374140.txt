You've made some elements that are hardcoded in the HTML draggable by doing this: $('.seleccionable').draggable({ drag: function(e, ui) { $('#status_nombre').html($(this).attr('id')); $('#status_top').attr('value',$(this).css('top')); $('#status_left').attr('value',$(this).css('left')); } ,containment: '#layout' ,refreshPositions: true ,snap: true }); When you add a new element you need to call .draggable() for that one too. One way to do it would be to put that selector/draggable code into a function and call it after you've added a new element.
