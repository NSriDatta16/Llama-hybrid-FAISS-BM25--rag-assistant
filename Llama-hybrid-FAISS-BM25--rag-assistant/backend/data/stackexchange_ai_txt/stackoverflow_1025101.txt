The correct way to set the organization_id to the selected value is to include it in your data array in your controller. Eg function add($organization_id) if(!empty($this->data)) { if($this->Signature->save($this->data)) { $this->setFlash('Save successful') $this->redirect(array('action' => 'index')) } else { $this->setFlash('Please review the form for errors') } } if($organization_id) { $this->data['Signature']['organization_id'] = $organization_id; } } Then in your view simply put echo $form->create('Signature', array('action' => 'add')) echo $form->input('organization_id') and it will automatically insert the organization_id value from the the controllers data.
