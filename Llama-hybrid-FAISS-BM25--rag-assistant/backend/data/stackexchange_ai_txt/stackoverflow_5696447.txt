This is an interesting question. If you'd like to have the javascript pick up on parameters in the url, you can use the window.location.href parameter to pull it out. So, quite simply the following function: function CheckForHighlight(){ href = window.location.href; values = href.split('?')[1] // Remove the url highlight = values.split('=')[1]; // Grab the second parmeter highlightSpan(highlight); // Highlight it! } Please Note that this is a very basic example. I encourage learning through simplicity, and this function can be expanded to dynamically parse all url varaibles. Brain Exersise for you!
