I'm not aware of any method that returns self without taking any arguments (but I might be overlooking something), but there's no reason you couldn't monkeypatch one into the class you're using: irb(main)> class Object irb(main)> def ignore irb(main)> self irb(main)> end irb(main)> end => nil irb(main)> msg = nil => nil irb(main)> 'hi'.send(msg || :ignore).upcase => "HI" irb(main)> msg = :reverse => :reverse irb(main)> 'hi'.send(msg || :ignore).upcase => "IH"
