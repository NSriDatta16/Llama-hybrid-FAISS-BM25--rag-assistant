An alternative to Weston C 's answer would be using the hasOwnProperty function in the for in loop, as suggested by Chris Morgan : for(var i in obj) if(obj.hasOwnProperty(i)) { document.write(obj[i]); } And yes, it is slower than iterating using the traditional for loop.
