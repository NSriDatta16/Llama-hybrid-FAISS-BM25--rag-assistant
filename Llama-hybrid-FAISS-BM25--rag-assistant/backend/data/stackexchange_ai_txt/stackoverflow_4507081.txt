You need to bind to the drop event on the target list (ID "droplist"), like so: $("#droplist").bind( "drop", function(event){ $(this).append( event.dragTarget ); // or use prepend to put it at the top console.log("Dropped " + event.dragTarget.title); // trigger another event to do the work around checking if it is an unsaved addition }) To remove an item on drag out you might try binding to a container element around the lists. $("#background").bind( "drop", function(event){ event.dragTarget.remove(); // guessing here }) A code snippet would be handy to help any further.
