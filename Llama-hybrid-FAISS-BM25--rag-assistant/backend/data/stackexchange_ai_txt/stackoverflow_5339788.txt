ExifInterface problem on Droid

I'm using Android's Native Camera app to take a picture and save it on the sdcard. Intent cameraintent = new Intent(MediaStore.ACTION_IMAGE_CAPTURE); File file = new File(Environment.getExternalStorageDirectory(), "test.jpg"); cameraintent.putExtra(MediaStore.EXTRA_OUTPUT, Uri.fromFile(file)); outputFileUri = Uri.fromFile(file); startActivityForResult(cameraintent, CAMERA_REQUEST); Now after saving the image, it seems to lose all EXIF information. When I try to retrieve the exif info like this: ExifInterface exif = new ExifInterface(Environment.getExternalStorageDirectory()+"/test.jpg"); String exifOrientation = exif.getAttribute(ExifInterface.TAG_ORIENTATION); It returns exifOrientation = 0 (i.e ORIENTATION_UNDEFINED ), whereas it should return 6. However, this code works perfectly well on other android phones (apart from Droid).
