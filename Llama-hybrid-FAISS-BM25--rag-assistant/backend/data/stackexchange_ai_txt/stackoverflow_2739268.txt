The way to get the dropped element is to have two parameters to your drop method (generally event and ui) and get the "draggable" property from the ui parameter. $("#droparea").droppable({ drop: function(event, ui) { var $target = ui.draggable; //note: draggable is a jQuery object if($target.is("#flyer")) { alert("adasd"); } } });
