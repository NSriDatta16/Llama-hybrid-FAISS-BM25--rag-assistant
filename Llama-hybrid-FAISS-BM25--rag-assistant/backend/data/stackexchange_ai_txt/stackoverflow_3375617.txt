Whenever you want to select only a top row from a left table for each row in the right table you should consider using the APPLY operator instead of join, and move the join condition inside the left join: SELECT u.UserID, u.FirstName, u.LastName, hn.PhoneNumber AS HomePhone FROM Users u OUTER APPLY ( SELECT TOP(1) PhoneNumber FROM UserPhoneNumbers upn LEFT JOIN UserPhoneNumberTypes upt ON upn.UserPhoneNumberTypeID=upt.UserPhoneNumberTypeID WHERE upt.PhoneNumberType='Home' AND upn.UserID = u.UserID ORDER BY ...) as hn ...
