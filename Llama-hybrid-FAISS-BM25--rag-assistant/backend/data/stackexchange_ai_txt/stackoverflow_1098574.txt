One option, although less visually pleasing, is to only load images when the scrolling stops. Set a flag to disable image loading in: -scrollViewWillBeginDragging: Re-enable loading images when the scrolling stops using the: -scrollViewDidEndDragging:willDecelerate: UIScrollViewDelegate method. When the willDecelerate: parameter is NO , the movement has stopped.
