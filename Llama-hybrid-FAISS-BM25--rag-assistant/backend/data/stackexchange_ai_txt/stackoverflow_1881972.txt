Test the code you write. ActiveRecord has great test coverage including coverage for the validation class methods. Spec: require 'spec_helper' describe User do before(:each) do @user = User.new end it "should not be valid without an email" do @user.save.should be_false @user.should_not be_valid @user.email = "example@example.com" @user.should be_valid @user.save.should be_true end end To get that spec to pass you would need class User
