You don't need to define a route for this file, you need this file. We you create a new project, Rails stores Prototype library in your javascripts folder. create public/javascripts/prototype.js create public/javascripts/effects.js create public/javascripts/dragdrop.js create public/javascripts/controls.js create public/javascripts/application.js Make sure the file exists. If not, create a new project and copy the files from its /public/javascript folder.
