You are aware that .NET's file handling classes do this a lot more elegantly, right? For example in your last example, you could do: FileInfo fi = new FileInfo(@"somebucketname\MyFolder\MyFile.zip"); string nameOnly = fi.Name; The first example you could do: FileInfo fi = new FileInfo(@"C:\MyFolder\MyFile.zip"); string driveOnly = fi.Root.Name.Replace(@"\", "");
