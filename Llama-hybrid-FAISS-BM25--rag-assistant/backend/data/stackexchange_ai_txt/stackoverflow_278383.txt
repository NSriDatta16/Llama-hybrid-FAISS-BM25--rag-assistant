.RC files are simple text files. You can simply copy/paste the DIALOG text from one .RC file to the other. Make sure that you copy the corresponding RESOURCE.H entries as well. Alternatively, you might be able to open both .RC files and then simply drag-and-drop the dialog from one to the other.
