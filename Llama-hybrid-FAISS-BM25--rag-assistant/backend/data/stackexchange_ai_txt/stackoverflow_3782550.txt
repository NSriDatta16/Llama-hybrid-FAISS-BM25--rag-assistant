If you're interested in the .NET platform, consider: RavenDB for the data layer. WCF Data Services for your REST services layer. Here's a guide to WCF Data Services with some great video tutorials . If you need a web front-end, consider ASP.NET MVC or plain HTML as you see fit. jQuery and AJAX modules are well suited in both. Arguably, ASP.NET MVC IS a REST endpoint of sorts, so you could skip the WCF services. All the above can be leveraged with free tools, including: Visual Studio Express Raven DB's HTML interface at http://localhost:8080/
