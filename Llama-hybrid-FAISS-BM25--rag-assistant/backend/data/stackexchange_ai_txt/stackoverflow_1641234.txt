I've spent hours today on this site trying to do something very similar to what was posted in this question. What I have is Previous | Next links navigation doing through a series of divs, hiding and showing. Though what I ended up with was different than the answer here....this was the one that most got me where I needed to be. So, thanks. And in case anyone is interested, here's what I did: $(function() { $("#firstPanel").show(); }); $(function(){ $(".nextButton").click(function () { $(".panel:visible").next(".panel:hidden").show().prev(".panel:visible").hide(); }); }); $(function(){ $(".backButton").click(function () { $(".panel:visible").prev(".panel:hidden").show().next(".panel:visible").hide(); }); }); .defaultHidden { display: none; } .navigation { display: block; width: 700px; text-align: center; } #contentWrapper { margin-top: 20px !important; width: 700px; } .nextButton { cursor: pointer; } .backButton { cursor: pointer; } &nbsp; | &nbsp; Next >> Information Here Text for the paragraph Information Here Text for the paragraph Information Here Text for the paragraph Repeat ad naseum...
