JCL provides a Delphi interface unit to control Microsoft task schedule service. Its name is JclTask.pas. They also provide a demo application for adding/removing/showing Windows tasks in "jcl\examples\windows\tasks" folder. Also, if commercial components are OK with you, SiComponents provides VCL Scheduling Agent , which is a VCL wrapper for Windows Task Scheduler, and supports new interface provided in Windows Vista.
