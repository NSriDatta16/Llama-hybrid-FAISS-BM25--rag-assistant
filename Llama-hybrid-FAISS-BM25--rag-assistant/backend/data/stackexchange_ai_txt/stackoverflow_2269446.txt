You can call .net Assemblies directly since CF8. Example: or mathInstance = CreateObject(".NET", "mathClass", "C:/Net/Assemblies/math.dll"); myVar = mathInstance.multiply(1,2); CF8 Doc - Using Microsoft .NET Assemblies CF9 Doc - Using Microsoft .NET Assemblies JNBridgePro is used under the hood to make this possible.
