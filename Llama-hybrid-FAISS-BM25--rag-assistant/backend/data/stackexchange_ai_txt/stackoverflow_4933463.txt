Assuming that by "i am using ganymede" you are stating that you are using the "ganymede version of eclipse", do the following: Open the project properties. Click on "Java Build Path". Select the Libraries tab. Click the "Add Library" button. Choose junit. You should now be able to import junit classes into your project.
