I would set the appropriate HTTP headers to prevent caching in both the main page and the AJAX responses. Try: Cache-Control: no-cache Pragma: no-cache Also, if these are form fields that are stale, it could be IE's auto-complete causing the issue. Try adding autocomplete="off" to form fields.
