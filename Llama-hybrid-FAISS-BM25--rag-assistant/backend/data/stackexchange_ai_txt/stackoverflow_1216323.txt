From the plumber's mate : All MSMQ messages are written to memory mapped files in the Storage directory: R*.MQ files are for Express messages P*.MQ files are for Persistent (Recoverable) messages, including Transactional messages J*.MQ files are for Journal messages L*.MQ files are bitmap indexes to the Persistent and Journal files
