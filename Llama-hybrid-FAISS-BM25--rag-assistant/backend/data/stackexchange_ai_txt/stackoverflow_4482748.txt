This line: var elems = ' ' + ' ' + ' ' + ' '; ...has two extra quotes. It should be: var elems = ' ' + ' ' + ' ' + ' '; That breaks your selector for the click event. But note, that won't work anyway, since you are trying to hook an element that does not exist until the 'click me' link is pressed - so you need to use .live() or .delegate() for that anyway. Something like this might work: $("#drag").delegate("input[type=checkbox]", "change", function(){ var index = this.id.substring(8); $("#inputfield" + index).toggleClass("complete"); });
