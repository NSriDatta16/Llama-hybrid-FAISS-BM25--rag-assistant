Yes (VBScript) :- Dim oWinHTTP Set oWinHTTP = CreateObject("WinHttp.WinHttpRequest.5.1") oWinHTTP.Open "GET", "http://remoteserver/thing.ext", False oWinHTTP.SetRequestHeader "User-Agent", "My Agent String" oWinHTTP.Send Using WinHttp gives you the greatest level of control, you can use MSXML2.ServerXMLHTTP.3.0 if you want more effeciently access any XML DOM sent. Using the standard MSXML2.XMLHTTP.3.0 component goes throught WinINet at gives you the users proxy settings and cookie store etc, but reduces your control over the conversation.
