Cell ID can be used to give a very rough geographical area; there is an open source effort to get this info, and various companies sell the info, and if you dig hard enough you might find the unpublished APIs that Google maps uses. http://www.codeproject.com/KB/mobile/DeepCast.aspx , as linked in another post, demonstrates how to get the cell ID and send it to Google's unpublished API for resolution.
