So you have an array ( allSpecifics ) of NSManagedObjects , and these objects have a rate property? If that's correct then: To find the: minimum rate value: NSNumber *minimumRate = [allSpecifics valueForKeyPath:@"@min.rate"]; maximum rate value: NSNumber *minimumRate = [allSpecifics valueForKeyPath:@"@max.rate"]; average rate value: NSNumber *minimumRate = [allSpecifics valueForKeyPath:@"@avg.rate"]; summed rate value: NSNumber *minimumRate = [allSpecifics valueForKeyPath:@"@sum.rate"];
