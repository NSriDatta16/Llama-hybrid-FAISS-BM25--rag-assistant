It's quite easy, just access the row's rows. foreach(UltraGridRow childRow in e.Row.Rows) { // your code } Subsequently, you can access the children rows of those rows the same way childRow.Rows You can also access a specific row using its key UltraGridRow specificChildRow = e.Row.Rows.FromKey("ChildRowKey");
