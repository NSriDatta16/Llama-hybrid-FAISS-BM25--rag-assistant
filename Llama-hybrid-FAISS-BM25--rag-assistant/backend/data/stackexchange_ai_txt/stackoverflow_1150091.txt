I have multiple apps on Heroku using S3 for uploads. I would recommend using the attachment_fu or paperclip plugin over file_column. These plugins make it trivial to switch from the local file system to s3 (literally just ":storage => s3") S3 has been reliable, fast, and cheap for me.
