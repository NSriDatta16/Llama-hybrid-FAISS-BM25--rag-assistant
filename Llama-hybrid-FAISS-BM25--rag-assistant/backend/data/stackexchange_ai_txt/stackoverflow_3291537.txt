All you have to do is add this to your existing addListener call: if (latLng) { marker = new GMarker(latLng, {draggable:true}); map.addOverlay(marker); } See this article linked by Google in their API information for an even more advanced example that lets your users edit the map data. Update: Changed case of 'l' to 'L' in latLng.
