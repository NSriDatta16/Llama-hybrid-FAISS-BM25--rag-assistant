If this is an internal application only (i.e security is already provided by corporate infrastructure and certificates are needed to make the deployment process happy) then you can use temporary 'self signed' certificates. Visual studio can do this on the publish tab, or you can use makecert: makecert -r -pe -n "CN=My Organisation" -b 01/01/2000 -e 01/01/2099 -eku 1.3.6.1.5.5.7.3.3 -ss My
