Use the below steps to get better results: Using describe function you will get know the values of each column if it contains numbers. find the outliers and replace those with Mean or Median or Mode values. Identify the columns to know the impact on data set ex: heat maps, we will get know the columns which are key once. Use multiple models (Linear Regression, Random forest, SVM, etc.) with multiple parameters (change the parameter values in each model) for better results. Check the error with multiple models with multiple parameters and analyze the results.
