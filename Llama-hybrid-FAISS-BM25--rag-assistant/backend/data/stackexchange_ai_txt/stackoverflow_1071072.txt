How to separate unit test methods?

Imagine I have a method: void Method(bool parameter){ if(parameter){ // first case } else { // second case } } Which is your preferred unit test organization method? Option 1: void MethodTest(){ // test first case // test second case } or Option 2: void MethodTestFirstCase(){ // test first case } void MethodTestSecondCase(){ // test second case }
