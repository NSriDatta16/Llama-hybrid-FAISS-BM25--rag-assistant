Sorting huge Number of Integers from hard disk

Given 100 GB integer Data on Hard Disk with RAM amounting to 2 GB, how to sort the integers with minimal disk operation. Here fetching one number from disk is considered as one disk operation( though in reality a block of data can be fetched). We can use additional space on the disk for temporary storage and no need to consider the operations of cleaning up temporary spaces used.
