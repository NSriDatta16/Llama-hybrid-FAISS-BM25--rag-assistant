Perhaps this ... l = {{"1/12/2008", Bob, 123}, {"28/06/2007", Alice, 456}, {"19/08/2009", Charlie, 789}} SortBy[l, AbsoluteTime[{#[[1]], {"Day", "Month", "Year"}}] &] gives {{"28/06/2007", Alice, 456}, {"1/12/2008", Bob, 123}, {"19/08/2009", Charlie, 789}} HTH Edit Note that Sort[ ] compares using OrderedQ[ ] , and so it can compare lists. (Greater[ ], for example, can't). So, the following code also works: Sort@(Flatten@{DateList[{#[[1]],{"Day","Month","Year"}}], #[[2]], #[[3]]} & /@ l) or perhaps more elegant: Sort@(l/.{x_String, y__} :> Flatten@{DateList[{x, {"Day", "Month", "Year"}}], y})
