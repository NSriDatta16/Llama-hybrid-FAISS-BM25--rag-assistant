If your code uses tail recursion (that is, the function returns the result of every recursive call directly without any other processing) then it's possible to rewrite the function imperatively without a stack: function dosomething(a,b,c,d) { // manipulate a,b,c,d if (condition) return dosomething(a,b,c,d) else return something; } Into: function dosomething(a,b,c,d) { while (true) { // manipulate a,b,c,d if (condition) continue; else return something; } } Without tail recursion, using a stack (or a similar intermediary storage) is the only solution.
