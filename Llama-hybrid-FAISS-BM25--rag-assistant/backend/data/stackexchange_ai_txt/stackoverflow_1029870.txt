Given a list of numbers in A1 to A10, you can work out their 'Rank' relative to each other by using 'RANK'. e.g. RANK(A1,A1:A6,0) RANK(cell, list of cells to check against, order) For order, 0 = descending. From there you can work out which one is first pragmatically.
