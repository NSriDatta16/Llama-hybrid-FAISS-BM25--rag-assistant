ASP.Net Gridview Update event

I have created a custom event to handle how the data should be updated in my gridview. The problem is, it is not firing. I have used debug mode and not even the breakpoint comes into effect when I click the update button. Here is the event handler I have written: protected void DocView_RowUpdating(object sender, GridViewUpdateEventArgs e) { var rw = DocView.Rows[e.RowIndex] ; var Doc = (TextBox)rw.FindControl("DocTB"); var Num = (TextBox)rw.FindControl("NumberTB"); var Iss = (TextBox)rw.FindControl("IssuedTB"); var Exp = (TextBox)rw.FindControl("ExpiryTB"); var Stat = (TextBox)rw.FindControl("StatusTB"); var con = new LinqDBDataContext(); var doc = (from i in con.Documents where i.DocumentID == e.RowIndex select i).Single(); doc.DocumentType = Doc.Text; doc.Number = Num.Text; doc.Issued = DateTime.Parse(Iss.Text); doc.Expiry = DateTime.Parse(Exp.Text); doc.Status = Stat.Text; con.SubmitChanges(); DocView.EditIndex = -1; } The event just will not fire!
