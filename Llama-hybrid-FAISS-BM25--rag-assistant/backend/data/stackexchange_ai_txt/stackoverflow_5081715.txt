I was able to disable the tableDnD by calling the following function: function tableDragAndDropDisable(cssID){ $(cssID + " tr").addClass("nodrag nodrop"); } // sample usage tableDragAndDropDisable("#sortable_table"); This will add the "nodrag nodrop" classes to each row of the table with the given ID ( cssID ) - effectively disabling the table of its sorting ability.
