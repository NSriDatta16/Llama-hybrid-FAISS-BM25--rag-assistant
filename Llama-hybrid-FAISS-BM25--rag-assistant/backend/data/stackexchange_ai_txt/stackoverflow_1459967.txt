you have several options: store it as a long single string with no formatting, which will look bland on the screen. store it as a long single string with embedded html and css, which will be a bad choice if you ever want to make your site have a different look/feel. normalize it so you have tables to store books, chapters, paragraphs, etc. you could then format and style the text as you load it into the application.
